GNU Emacs é um editor de texto de código aberto para plataformas Linux, Windows e OS X.

Ele permite que você edite arquivos de texto simples, teste programas, gerencie arquivos e até mesmo jogue jogos , como o Tetris.

Emacs é um editor de texto simples que é frequentemente usado para editar o código-fonte, shells, listas de diretórios, páginas da web e mensagens de e-mail.

O programa opera por meio de comandos de pressionamento de tecla para ajudá-lo a editar e automatizar tarefas para um trabalho mais rápido e eficiente.

Além de editar texto, o Emacs funciona como um IDE, fornecendo ferramentas para compilar, executar e testar programas.

O Emacs pode também funciona como um gerenciador de arquivos, permitindo organizar, editar e excluir arquivos.

O programa permite que você jogue jogos, como Tetris e Torres de Hanói, dentro do programa.

Uma das características mais importantes do Emacs é que ele oferece a capacidade para personalizar o programa de acordo com suas necessidades.

Permite que você salve macros para automatizar suas tarefas repetitivas.

Você também pode alterar e estender quase todos os recursos do programa por meio do Emacs Lisp (Elisp ) idioma.

Você pode modificar as funções do Emacs e personalizar a aparência, como fontes e esquemas de cores.

O Emacs também fornece tutoriais úteis que o ajudarão a se aclimatar com seus comandos e recursos.

GNU Emacs é um editor de texto popular entre os usuários Unix e é semelhante aos editores de texto vi, como o gVim.

Se você tem conhecimento de programação e deseja personalizar seu editor de texto de forma mais rápida e eficiente funcionar, o GNU Emacs pode ser uma boa opção para você.
