Principais características

Inclui validação usando vários esquemas de validação
Oferece um editor WYSIWYG para criar ou editar documentos XML sem usar código
Utiliza preenchimento de conteúdo avançado com suporte para elementos, atributos e valores XML
Pode editar, validar e criar arquivos EPUB, PDF e HTML

oXygen XML Editor é uma ferramenta poderosa para desenvolver documentos XML.

Está disponível como um aplicativo independente ou como um plug-in para Eclipse.

O programa pode lidar com vários tipos de dados XML, incluindo bancos de dados XML, pipelines Xproc e serviços da Web.

oXygen está disponível para Windows, macOS, Linux e Solaris.

oXygen é a única ferramenta XML com suporte para todas as linguagens de esquema XML.

Ele também suporta Unicode completo e edição multilíngue.

oXygen reconhece Xquery e permite que você navegue, edite ou crie consultas personalizadas no idioma.

Você também pode usar seu depurador Xquery integrado avançado para solucionar problemas.

Com oXygen, você pode organizar seu trabalho em projetos que você pode usar para validação de lote e transformações.

Esses projetos também podem ser s com outros usuários.

O recurso de "controle de alterações" do programa torna possível mesclar as alterações feitas por vários usuários em um único documento.

o pacote abrangente de ferramentas de edição XML do oXygen XML Editor torna isso uma ótima opção para quem desenvolve projetos XML complexos.

O programa pode ter um preço alto, mas os desenvolvedores descobrirão que recebem o que pagaram.
