Principais características

Blackmagic Design Fusion, anteriormente Eyeon Fusion, é um aplicativo de composição avançado usado para criar efeitos visuais e gráficos de movimento.

O software fornece uma interface baseada em nó e uma ampla gama de recursos e efeitos.

Está disponível gratuitamente para Windows e OS X e para compra como Fusion Studio com mais recursos para Windows, OS X e Linux.

Fusion apresenta 2D e 3D integrados gráficos de movimento e recursos de composição de efeitos visuais.

O aplicativo oferece suporte a uma ampla variedade de modelos 3D, áudio e formatos de vídeo, como QuickTime Movies and Audio, JPEG, Photoshop, Alias ​​Pix, 3D Studio 3DS, AutoCAD DXF, SVG, Cineon, Autodesk FBX e Collada DAE.

O software oferece uma grande variedade de ferramentas, como animação, enquadramento avançado de teclas, titulação, aceleração de GPU e pintura.

O software é usado por efeitos visuais arti sts, animadores 3D e designers gráficos de transmissão e movimento para criar comerciais de TV, sequências de títulos e gráficos de transmissão.

Também foi usado para criar efeitos visuais em muitas produções de filmes de Hollywood, incluindo Gravity, Edge of Tomorrow, Avatar e Skyfall.

O aplicativo está disponível gratuitamente, o que é ótimo para testá-lo e ver se você gostaria de comprar (por um preço razoável) a versão de estúdio mais poderosa.

Blackmagic Design Fusion é uma excelente escolha para lidar com efeitos visuais e gráficos em movimento em sua próxima produção de grande sucesso.
