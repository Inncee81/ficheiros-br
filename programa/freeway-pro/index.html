Softpress Freeway Pro é um aplicativo WYSIWYG usado para criar sites sem precisar escrever código.

Embora seja utilizado principalmente para projetar sites, também pode ser usado para criar apresentações, boletins informativos e conteúdo de intranet.

O software foi descontinuado em 2016.

O aplicativo permite que você crie seu site do zero ou a partir de um modelo, seja um layout predefinido ou criado pelo usuário.

O Freeway Pro oferece recursos úteis, como páginas mestras, que permitem que você economize tempo na criação de layouts de página duplicados em seu site.

O Freeway Pro também pode gerar menus de navegação automaticamente com base nas páginas e pastas em seu site.

O programa permite que você crie sites responsivos e compatíveis com dispositivos móveis e verifique sua aparência em telas de telefones, tablets e desktops.

Se o Freeway Pro ainda não for profissional vide a funcionalidade de que você precisa, você pode adicionar seu próprio HTML, PHP, JavaScript ou CSS personalizado.

Além disso, o programa não se limita apenas ao desenvolvimento de sites, pois você pode desenvolver conteúdo para outros fins, como apresentações e boletins informativos.

O Freeway Pro é usado por usuários casuais que desejam publicar um site rapidamente e por usuários profissionais que desejam economizar tempo no processo de construção.

O Freeway Pro oferece algumas funcionalidades atraentes para a web desenvolvedores que desejam ignorar o código e se concentrar mais no design de seu site.

No entanto, como o software foi descontinuado, você deve procurar em outro lugar, possivelmente no Adobe Dreamweaver para projetar seu site.
