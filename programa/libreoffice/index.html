Principais características

Incluído com programas úteis de produtividade de escritório gratuitamente
Oferece muitas das mesmas funções do Microsoft Office
Oferece compatibilidade com muitos arquivos de produtividade de escritório diferentes
Apresenta uma interface de usuário limpa e fácil de aprender

LibreOffice é um pacote de escritório de código aberto, semelhante ao Microsoft Office (até as cores dos ícones do aplicativo correspondem), que permite criar vários tipos de documentos.

O LibreOffice possui um ambiente limpo e fácil de usar interface.

É parcialmente semelhante ao Microsoft Office, portanto, se você estiver saindo do Office, não deverá ter problemas para ajustar.

LibreOffice consiste nos seguintes programas:

  • Writer - Um processador de texto
  • Draw - Um editor de gráficos vetoriais
  • Calc - Um aplicativo de planilha
  • Impress - Um criador de apresentação
  • Math - Um criador de equação matemática
  • Base - Um aplicativo de gerenciamento de banco de dados

Depois de começar a criar um documento, você descobrirá que cada aplicativo do pacote oferece recursos familiares aos aplicativos correspondentes do Microsoft Office ativado.

Por exemplo, os documentos do Writer são um substituto para os documentos do Word e fornecem muitos dos estilos tradicionais e opções de formatação, como fontes, recuo de texto, inserção de tabela e o pincel de formato.

Calc, que é um substituto do Excel, apresenta uma planilha semelhante ao Excel, mas está visivelmente perdendo a funcionalidade macro.

.

Impress é a resposta do LibreOffice para PowerPoint e é ótimo para criar slides básicos apresentações.

O Draw é comparável ao Microsoft Visio e Publisher e é uma boa opção para criar diagramas, até mesmo apresentações.

O Base é semelhante ao Microsoft Access, com recursos semelhantes, como relatórios, SQL e formulários.

E, finalmente, o aplicativo Matemática do LibreOffice é uma ferramenta útil para criar fórmulas e incorporá-las a outros documentos do pacote LibreOffice criados pelo Writer e Calc.

Embora não ofereça todas as mesmas funções, o LibreOffice é uma alternativa atraente ao Microsoft Office.

É compatível com uma grande variedade de formatos de produtividade de escritório, incluindo arquivos do Microsoft Office, como .DOCX e .XLSX.

Se você está procurando uma solução de suíte de escritório barata (é grátis), o LibreOffice é uma ótima opção.
