Visão geral do software

Principais características

Abra mais de 290 formatos de mídia e escreva mais de 135 formatos de áudio
Crie seus próprios sons que são compatíveis com uma variedade de sintetizadores diferentes
Preserve a qualidade do áudio com suporte para amostras de dados de ponto flutuante de até 32 bits
Converta arquivos de áudio de sintetizadores, rastreadores e telefones celulares

FMJ-Software Awave Studio é uma ferramenta de áudio multifuncional disponível para Windows XP e posterior.

Pode reproduzir, editar, criar e converter uma variedade de formatos de áudio.

Como suporta muitos formatos de áudio, é comumente usado para converter arquivos de áudio.

Awave Studio suporta uma grande quantidade de formatos de áudio.

Pode ler mais de 290 formatos de arquivos de áudio e pode gravar mais de 135 formatos de áudio, incluindo formatos de sintetizadores, rastreadores e telefones celulares.

Este programa permite que você manipule arquivos enquanto mantém a qualidade do áudio com amostragem de dados de ponto flutuante de até 32 bits.

Ele também possui um sintetizador de software de alta qualidade integrado para criar seus próprios sons.

FMJ-Software Awave Studio é uma ótima ferramenta para entusiastas de áudio de computador.

Se desejar ser capaz de converter ou usar quase qualquer áudio formato, este programa é para você.
