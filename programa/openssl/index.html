Principais características

Software de código aberto disponível em várias plataformas
Implementa protocolos SSL e TLS para transmissões seguras de dados
Inclui uma biblioteca de funções criptográficas

OpenSSL é um kit de ferramentas de código aberto usado para implementar os protocolos Secure Sockets Layer (SSL) e Transport Layer Security (TLS).

O kit de ferramentas é gratuito para uso sob a licença OpenSSL e licença SSleay e está disponível para Windows, OS X e Linux.

O kit de ferramentas OpenSSL é usado por muitos sistemas de código aberto, como variantes Linux, e a maioria dos servidores web na Internet, como o Apache Servidor HTTP.

Os protocolos SSL e TLS, que OpenSSL implementa, permitem que as informações sejam enviadas com segurança pela Internet criptografando dados para que terceiros não possam acessar a transmissão de dados.

O kit de ferramentas também inclui uma biblioteca criptográfica com implementações de cifras simétricas, funções hash, algoritmos de chave pública e outros algoritmos criptográficos.

OpenSSL é um kit de ferramentas robusto usado em toda a Internet.

No entanto, como vulnerável ao susto Heartbleed, que fez com que uma grande quantidade de servidores ficasse suscetível a um ataque.

Em resposta ao Heartbleed, o código-fonte OpenSSL recebeu mais atenção de programadores que buscavam torná-lo mais seguro, o que é um bom presságio para a viabilidade do kit de ferramentas OpenSSL.
