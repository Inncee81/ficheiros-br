Principais características

Crie projetos em três formatos diferentes
Altamente compatível com Visual Basic
A hospedagem na nuvem oferece um ambiente seguro para testar aplicativos da web

Xojo é um IDE que permite aos usuários criar aplicativos de desktop, web e web-mobile.

O aplicativo é gratuito para baixar e usar, mas requer uma licença para implantar aplicativos.

O Xojo permite que os usuários desenvolvam aplicativos que variam de produtos de software comerciais a utilitários para uso pessoal.

O programa dá ao usuário a opção de criar projetos em três formatos diferentes: Projeto Xojo (texto), Xojo Binário e Xojo XML.

Os usuários devem ter uma licença para salvar seus projetos em qualquer um dos três formatos.

Se os usuários estiverem usando a versão gratuita do Xojo, eles só podem salvar no formato binário.

Uma vez que o desenvolvimento de um aplicativo é concluído, o Xojo fornece um recurso de construção rápida que converte o projeto em um executável autônomo.

O Xojo fornece um ambiente WYSIWYG que torna o desenvolvimento mais rápido, como arrastar e - soltar c capacidade de diferentes controles para criar a IU.

O aplicativo permite que os usuários criem aplicativos a partir de um conjunto de código-fonte, o que permite que seu aplicativo apareça e se comporte da maneira que você o projetou, independentemente da plataforma.

Xojo é altamente compatível com Visual Basic, permite aos usuários converter rapidamente projetos Visual Basic em projetos formatados em Xojo.

Xojo Cloud também está disponível como serviço de hospedagem em nuvem para aplicativos Xojo web do usuário.

A nuvem fornece um ambiente seguro para desenvolver e testar qualquer aplicativo da web.

Xojo é uma ferramenta útil, especialmente para programadores iniciantes, para desenvolver aplicativos de desktop, web e web-mobile.

Algumas desvantagens incluem a falta de recursos de desenvolvimento avançados e foco principal nos usuários do OS X, deixando os usuários Windows e Linux querendo mais.

Se você é um programa iniciante r trabalhando no OS X, o Xojo pode ser uma ótima opção para você, caso contrário, procure um IDE diferente.
