Principais características

Crie aplicativos shockwave para a web
Utiliza uma linguagem de script para desenvolver interação com os usuários
Altamente extensível com muitos plug-ins para melhorar seu uso
Suporte para uma ampla variedade de formatos de imagem, áudio e vídeo

Adobe Director é um programa usado para criar aplicativos multimídia.

Foi originalmente projetado para criar sequências de animação, mas agora é comumente usado para criar aplicativos Shockwave, como jogos e cursos de e-learning.

A versão 12 do Adobe Director foi a última versão do software lançada antes de ser descontinuado em 2017.

O Director permite que você crie aplicativos multimídia usando uma interface e um fluxo de trabalho baseado na criação de filmes.

Ele usa uma linguagem de script chamada Lingo que permite que a interatividade seja adicionada aos aplicativos.

Também possui ferramentas de manipulação de imagem 2D e pode oferecer suporte à interatividade 3D com objetos de arquivo 3D Shockwave.

Adobe Flash é comumente em comparação com o Director porque eles compartilham muitas semelhanças em uso e finalidade.

Ambos os programas são usados ​​para criar aplicativos da web que são executados em plug-ins da Adobe.

A principal diferença entre os dois é que o Director é muito mais extensível.

Existem muitos plug-ins disponíveis que adicionam ferramentas e recursos ao Director.

Se você deseja desenvolver web aplicativos e prefere um programa mais extensível, o Director tem muito mais a oferecer do que o Flash.

Se você não deseja que seus usuários sejam obrigados a baixar um plugin para que seu aplicativo funcione, você pode estar interessado em usando tecnologias da web como JavaScript.
