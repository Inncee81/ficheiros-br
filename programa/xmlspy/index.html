Principais características

Suporta Xpath, XSLT e XQuery
Gera exemplos de instâncias XML com todas as opções de esquema
Inclui ferramentas gráficas de edição de esquema XML
Processa e valida o código rapidamente com RaptorXML

Altova XMLSpy é um editor de XML e IDE usado para criar aplicativos baseados em XML para desktop e web.

É um produto licenciado comercialmente disponível apenas para Windows.

Existem muitos recursos que o XMLSpy fornece que tornam o desenvolvimento mais simples.

Ele tem uma variedade de visualizações diferentes, como uma visualização de texto e grade para edição XML, uma visualização de esquema XML e uma visualização WSDL.

O IDE também inclui editores gráficos para XML, Esquema XML e Taxonomia XBRL.

XMLSpy tem muitos outros recursos, como documentação e edição de instância XML, ferramentas de desenvolvimento e depuração XQuery, validadores, ferramentas de análise e Geração de código livre de royalties.

Se você está procurando uma ferramenta de edição XML poderosa que pode ajudar a gerenciar projetos complexos, XMLSpy é uma ótima opção.

No entanto, a licença do programa é bastante cara para aqueles em um orçamento apertado.

Se você não quer gastar muito dinheiro, existem muitos editores XML gratuitos recomendados, como EditiX Lite ou CAM XML Editor.
