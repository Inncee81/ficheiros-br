Bare Bones TextWrangler é um editor de texto livre usado principalmente para editar o código-fonte.

É considerado o "irmão mais novo" do BBEdit, pois é semelhante, mas oferece menos recursos.

TextWrangler fornece muitos dos recursos básicos de edição de texto, como verificação ortográfica e funcionalidade de localizar e substituir, mas também apresenta recursos de programação úteis.

O editor de texto oferece destaque de sintaxe e navegação de função para uma variedade de linguagens de programa, incluindo HTML, PHP, JavaScript, XML, Perl, Python, C ++ e Ruby.

O programa também fornece "Módulos de linguagem sem código" para produzir seus próprios módulos de linguagem TextWrangler e pode funcionar como um editor externo integrado com Xcode.

Embora o TextWrangler possa ser usado simplesmente para compor e editar arquivos de texto simples, ele é voltado para usuários que precisam ler e escrever código, como programadores e web designers.

O editor de texto fornece uma variedade de recursos úteis, uma interface limpa e integração conveniente com a plataforma OS X.

Bare Bones TextWrangler é uma boa escolha para usuários do OS X que procuram uma fonte gratuita editor de código.
