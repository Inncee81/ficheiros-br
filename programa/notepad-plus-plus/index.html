Visão geral do software

Principais características

Suporte para muitas linguagens de programação, como C ++ e Java
Destaque de sintaxe e dobradura
Interface de documento com guias e edição em tela dividida
Funcionalidade estendida por meio de plug-ins

Notepad ++ é um editor de texto semelhante ao Notepad, mas fornece recursos adicionais, como ferramentas avançadas de edição de código-fonte.

É distribuído gratuitamente, não vem junto com o Windows.

O editor de texto é composto de uma interface de documento com guias e edição de tela dividida, enquanto fornece a funcionalidade de arrastar e soltar.

Embora possa ser usado como um editor de texto, o programa é usado principalmente para edição de código-fonte .

Suporta nativamente uma variedade de linguagens de programação, como C ++, HTML, Pascal, Java e YAML.

O editor também apresenta muitos recursos básicos de edição de código-fonte, como destaque de sintaxe , Gravação e execução de macro e mapeamento de teclas de atalho personalizáveis.

.

O Notepad ++ também oferece suporte a plug-ins para funcionalidade adicional.

Você pode criar seu próprio ou instalar um plug-in usando o Plu do programa Gin Manager.

O Plugin Manager ajuda a instalar, atualizar e remover plug-ins compatíveis existentes.

Se você precisar de um editor de texto que o ajudará a codificar com mais eficiência, o Notepad ++ fará o trabalho feito.

Ele fornece uma interface eficiente, suporte para uma ampla gama de linguagens de programação e funcionalidade estendida por meio de plug-ins.

Notepad ++ é um editor de texto e código fonte muito capaz para usuários do Windows .
