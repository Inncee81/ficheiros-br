Visão geral do software

Principais características

Suporta todas as principais linguagens de programação
Permite extensões do editor por meio de pacotes
Pode gravar macros para ajudar a automatizar o trabalho tedioso
Inclui recursos como dobramento de código-fonte e localização e substituição de projeto global

MacroMates TextMate é um editor de texto robusto que permite várias tarefas, como edição de código-fonte, escrever roteiros, executar consultas SQL e criar entradas de blog.

O programa suporta todas as principais linguagens de programação e é executado no Mac plataforma.

A partir da versão 2, MacroMates disponibilizou o código-fonte TextMate para desenvolvedores que desejam fazer modificações personalizadas no editor.

TextMate permite que você use comandos, macros , modelos e snippets para ajudar a organizar e criar o código-fonte.

TextMate oferece um recurso de localização e substituição que localiza e substitui rapidamente o texto com várias opções de filtro.

O editor de texto possui a capacidade de destaque e edite vários blocos de código usando cursores adicionais.

O TextMate também oferece um preenchimento inteligente, que prevê o que você vai digitar para ajudá-lo a codificar mais rapidamente.

TextMate permite estender o editor com Bundles.

Com Bundles, você pode adicionar gramáticas de idioma, bem como combinar macros, snippets de código, comandos, modelos e outros atalhos em um Bundle para o editor.

Cada pacote oferece teclas de atalho que podem ser acionadas para inserção em seu código, o que ajuda a desenvolver o código-fonte com mais eficiência.

Você pode criar seus próprios pacotes personalizados, dependendo de suas necessidades de desenvolvimento .

MacroMates TextMate é uma ferramenta valiosa para qualquer desenvolvedor.

Suporta todas as principais linguagens de programação, atalhos de código-chave e pacotes para estender o editor.

Qualquer Mac designer ou desenvolvedor trabalhando em código deve dar uma chance a este editor.
