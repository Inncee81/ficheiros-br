Visão geral do software

Principais características

Faz e grava imagens de CD / DVD
Converte formatos de imagem de disco populares para o formato de imagem ISO
Compacta imagens ISO em seu formato ISZ

EZB Systems UltraISO é um programa Windows usado para criar, modificar e converter arquivos de imagem .ISO para gravar em um DVD ou CD.

É um produto comercial, mas também disponível como um teste gratuito.

UltraISO executa uma variedade de funções com o formato de imagem ISO.

O programa permite que você crie imagens ISO de arquivos em um disco rígido ou CD / DVD- ROM.

Permite duplicar CDs e DVDs em uma imagem ISO.

UltraISO permite criar CD / DVDs inicializáveis ​​e flash USB ou discos rígidos a partir de uma imagem de disco.

O programa também pode ser usado para editar arquivos ISO e adicionar ou extrair arquivos e pastas de ou para o arquivo.

.

UltraISO pode ser usado para converter vários arquivos de imagem de disco, como como .BIN, .IMG, .CIF, .DMG e .NRG, para uma imagem ISO.

UltraISO usa um formato ISZ proprietário para compactar arquivos ISO.

UltraISO é um programa sólido para gerenciar arquivos ISO.

EZB Systems UltraISO é útil quando você precisa compactar, converter e montar imagens ISO em drives virtuais ou criar e gravar CDs / DVDs.

.
