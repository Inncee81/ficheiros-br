Principais características

Dicionário e calendário integrados
Work Task Launcher para fácil seleção de tarefas
Gerencia contatos, listas de tarefas e programações pessoais
Compatível com formatos de arquivo do Microsoft Office

Microsoft Works é um pacote de produtividade projetado para usuários domésticos.

É frequentemente visto como uma versão reduzida do Microsoft Office, pois é usado principalmente em ambientes domésticos para tarefas pessoais e domésticas.

O Works não está mais desenvolvido.

O Works inclui os seguintes aplicativos:

  • Processador de Texto do Works : usado para criar documentos, brochuras, currículos, cartas, cartões, e outros artigos de papelaria; inclui vários modelos predefinidos, um dicionário e um dicionário de sinônimos.
  • Planilha de trabalho : permite que os usuários registrem e analisem dados pessoais, como finanças e inventários domésticos.
  • Banco de Dados do Works : usado para armazenar informações estruturadas, como contatos pessoais.
  • Calendário do Works : Permite que os usuários gerenciem agendas pessoais e listas de tarefas; pode combinar vários calendários em um único arquivo.
Embora o Works use seus próprios formatos de arquivo proprietários, o software é compatível com os formatos padrão do Office para Microsoft Word e Excel.

Por exemplo, os usuários podem salvar arquivos do Processador de Texto do Works ( .WPS ) como arquivos do Word ( .DOCX ) e Planilha do Works ( .WKS ) como arquivos do Excel (). ) documentos.

O Works também pode abrir a maioria dos formatos do Office diretamente, embora alguns objetos específicos do Office possam não ser suportados.

O Microsoft Works é uma excelente alternativa ao Office Suite para casa usuários.

Ele tem muitos aplicativos úteis para controlar suas finanças, gerenciar sua agenda e criar documentos.

Se você está procurando alguma das funcionalidades do Microsoft Office, mas não precisa de todas as ferramentas extras que vem com, o Works é um gr opção de comer para você.
