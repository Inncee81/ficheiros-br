Visão geral do software

Principais características

Crie gráficos de diagrama a partir de uma variedade de ferramentas de design
Utilize um dos muitos modelos pré-construídos
Exporte seu desenho para vários formatos, como BMP e JPEG
Aprenda com manuais online e tutoriais em vídeo

O Omni Group OmniGraffle é um aplicativo OS X usado para criar vários tipos de gráficos, como árvores genealógicas, projetos de sistemas elétricos e wireframes de sites.

O aplicativo também está disponível para iOS para on- usuários móveis.

.

OmniGraffle oferece uma variedade de ferramentas de design que incluem estênceis, inspetores, formas e objetos.

O aplicativo também vem com modelos de vários categorias, como unidades imperiais, unidades métricas, design de software e planejamento de espaço, que fornecem uma vantagem inicial em seus designs.

OmniGraffle apresenta vários formatos proprietários para salvar conteúdo.

O GRAFFLE arquivo é usado para armazenar seu desenho, GTEMPLATE é usado para armazenar modelos, GDIAGRAMSTYLE é usado para armazenar estilos de diagrama e GSTENCIL é usado para armazenar estênceis.

Você pode exportar seu desenho para BMP, EPS, GIF, JPEG, PNG , SVG, TIFF, PSD e formatos VDX.

Além disso, OmniGraffle é compatível com versões anteriores, permitindo que você importe recursos de versões anteriores do software.

.

O a interface é limpa, fácil de navegar e personalizável.

Você pode organizar os inspetores como uma barra lateral, uma janela flutuante ou paletas individuais.

Se você é novo no programa ou está apenas preso em um recurso, OmniGraffle fornece ajuda online, que inclui manuais e tutoriais em vídeo.

O aplicativo é projetado principalmente para criar gráficos relacionados a diagramas e os usuários são comumente artistas, designers ou mapeadores de dados.

O aplicativo vem com uma variedade de ferramentas de design e modelos que permitem que você crie diagramas de aparência profissional.

O Omni Group OmniGraffle é um pouco caro, mas ainda é uma ótima escolha para criar diagramas de aparência profissional.
