Visão geral do software

Principais características

Recursos de OLTP na memória para velocidade transacional rápida e rendimento de seu aplicativo
Backup criptografado de seus dados, seja no local ou no Microsoft Azure
Assistente de implantação para implantação de bancos de dados SQL Server no local no Microsoft Azure

Microsoft SQL Server é um aplicativo de banco de dados relacional para a plataforma Windows.

O aplicativo é usado principalmente para armazenar e recuperar dados, sejam de aplicativos de software no mesmo computador ou através de uma rede.

O SQL Server fornece um conjunto de ferramentas que permitem criar, implantar e gerenciar bancos de dados, esteja você no local ou na plataforma de nuvem Azure da Microsoft.

Os recursos de software baseados em colunas armazenamento de dados e processamento de consulta para armazenamento e gerenciamento de dados, o que funciona especialmente bem para cargas de trabalho de data warehousing.

O núcleo do SQL Server oferece recursos de processamento de transações online (OLTP) na memória para velocidades transacionais rápidas e taxa de transferência.

Além disso, o SQL Server oferece tranquilidade, pois oferece suporte à criptografia para backups de banco de dados, independentemente de eles ocorrerem no local ou por meio do Microsoft Azure.

SQL O servidor está disponível em uma variedade de edições voltadas para diferentes públicos, no entanto, o software padrão da indústria é usado principalmente por empresas.

O aplicativo fornece ferramentas de primeira linha que permitem criar, implantar e gerenciar bancos de dados, o que torna o Microsoft SQL Server uma ótima escolha para sua empresa.
