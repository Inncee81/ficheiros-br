JRiver Media Center é um aplicativo multimídia.

Está disponível para Windows e OS X.

.

O aplicativo de mídia permite que você ouça, assista e veja uma variedade de formatos de áudio, vídeo e imagem, que inclui MPEG, OGG, DIVX, QuickTime, JPEG e PNG.

O aplicativo também permite que você importe Windows Media, RealMedia, e formatos de lista de reprodução do iTunes.

Se você ficar entediado com a aparência e os recursos do reprodutor de mídia, pode aplicar diferentes skins à interface e instalar plug-ins para estender sua funcionalidade.

O Media Center é um aplicativo útil que suporta uma ampla gama de formatos de mídia, ao mesmo tempo que fornece uma interface personalizável e suporte para plug-ins.

Para usuários de Windows e OS X que procuram uma alternativa para o Windows Media Player ou iTunes, JRiver Media Center é uma opção decente.
