Integrated Genome Browser (IGB) é um programa de código aberto usado para visualizar e explorar grandes conjuntos de dados genômicos.

Você pode visualizar vários formatos de dados, interagir com seus dados, criar gráficos, salvar suas informações como imagens, e compartilhe seus dados com colaboradores.

O IGB permite que você visualize dados de anotações e sequências alinhadas.

O programa suporta uma variedade de formatos, como FASTA, BAM, BEDGRAPH, BigWIG, Cytoband, e GenBank.

Você pode pegar dados e transformá-los em uma variedade de formas de gráficos, incluindo gráficos de incompatibilidade para destacar diferenças de dados e gráficos de profundidade para mostrar cobertura.

Você também pode interagir com seus dados por meio Animações de zoom e panorâmica do IGB.

O IGB permite que você acesse e carregue genomas de sites QuickLoad, que são sistemas simples de diretórios, sequências e outros arquivos de dados para fins de visualização.

O IGB fornece funcionalidade avançada com suporte a plug-ins para adicionar funcionalidade e automação do fluxo de trabalho por meio do uso de scripts.

O IGB também oferece recursos avançados para compartilhar seus dados, seja publicando-os como JPEG de alta qualidade, PNG ou imagem SVG ou armazenamento de dados no Dropbox para dar aos seus colaboradores acesso ao seu projeto.

IGB é usado principalmente por pesquisadores universitários, biólogos e bioinformáticos.

Ele fornece ferramentas avançadas para abrir um variedade de formatos de dados do genoma, visualizando dados como gráficos diferentes e compartilhando dados com colaboradores.

Infelizmente, o programa depende de concessões para desenvolvimento contínuo, o que significa que o suporte para o aplicativo pode cessar a qualquer momento.

No entanto, Integrated Genome Browser ainda é um programa capaz e personalizável para usuários que procuram explorar e analisar seus dados de genoma.
