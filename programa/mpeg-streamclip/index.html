Squared 5 MPEG Streamclip é um reprodutor de vídeo gratuito com recursos de edição e conversão.

É comumente usado para baixar e converter streams de vídeo da Internet.

MPEG Streamclip é bastante simples programa que é simples de usar.

Ele pode ser usado para reproduzir, editar e converter arquivos de vídeo com todas as ferramentas acessíveis na barra de menu.

O programa também tem a capacidade de realizar operações em lote.

Se você está procurando um programa simples para assistir a um arquivo de vídeo, fazer edições simples nele ou convertê-lo para um formato diferente, MPEG Streamclip é uma boa opção gratuita.

Para recursos de edição mais avançados, você provavelmente vai querer usar um programa diferente, como Windows Moviemaker ou iMovie.
