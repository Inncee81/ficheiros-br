Microsoft Windows é um sistema operacional desenvolvido pela Microsoft.

Cada versão inclui uma interface gráfica do usuário, com uma área de trabalho que permite visualizar arquivos e pastas em janelas diferentes.

Os concorrentes de Windows incluem a plataforma OS X da Apple e sistemas operacionais baseados em Linux, como Linux Mint e Ubuntu.

Windows está incluído em muitas compras de computador, incluindo da Dell, Sony, HP, Lenovo e outros fabricantes.

Os usuários de PCs feitos em casa também costumam instalar o Windows ao contrário de outros sistemas operacionais.

A Microsoft adicionou suporte para dispositivos touchscreen com Windows 7, e portanto, o software agora é executado em muitos tablets também.

O lançamento do Windows 10 introduziu o Cortana, um assistente virtual nos moldes do Siri da Apple que foi projetado para ajudá-lo a encontrar arquivos, gerenciar seu calendário, e realizar o outras tarefas.

Cortana está integrada com Edge , Windows 'navegador da web padrão, que substituiu Internet Explorer .

O navegador vem com ferramentas de navegação familiares junto com recursos exclusivos como Web visualização de Notas e Leitura.

Outro novo aplicativo introduzido em Windows 10 é o Groove Music , o novo reprodutor de música padrão para o Windows.

Microsoft Windows é o sistema operacional mais comumente usado para computadores pessoais e corporativos.

É compatível com uma ampla gama de especificações de hardware e tem a maior comunidade de desenvolvedores de software para software pessoal e de produtividade.

O sistema operacional é adequado para usuários domésticos e profissionais.
