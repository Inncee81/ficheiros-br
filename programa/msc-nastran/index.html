Principais características

Praticamente protótipos de projetos estruturais
Corrija problemas estruturais que ocorrem durante o serviço de um produto
Otimize o desempenho de projetos existentes

O MSC Nastran é um aplicativo profissional de análise de elemento finito (FEA) usado para analisar estruturas.

Foi originalmente desenvolvido para a NASA na década de 1960, mas desde então evoluiu e foi disponibilizado através do software MSC.

O programa está disponível para Windows e Linux.

Nastran oferece uma variedade de ferramentas para projetar, testar, corrigir e otimizar projetos estruturais.

Você pode virtualmente protótipo em vez de protótipo físico para economizar dinheiro, otimizar o desempenho de projetos existentes e corrigir problemas estruturais durante o serviço de um produto para reduzir o tempo de inatividade.

O aplicativo também permite que você execute análises de fadiga usando suas tecnologias de fadiga incorporadas que ajudam a prevenir flambagem, tensões excessivas e deformações prejudiciais.

Nastran fornece algoritmos de otimização que automatizam a otimização estrutural pro cess.

Estes algoritmos podem melhorar a forma ou o perfil de membros estruturais, determinar padrões ideais de cordão ou estampa para peças de chapa metálica, encontrar a espessura ideal da camada laminada composta e otimizar vários modelos simultaneamente.

Nastran é usado principalmente por engenheiros aeroespaciais e automotivos para otimizar a integridade estrutural de sistemas, como trem de pouso, airbags e pás do ventilador.

O aplicativo permite que você execute uma variedade de funções com relativa facilidade, pois vem com clusters de vários núcleos e vários nós com tecnologias de paralelização que permitem resolver grandes problemas rapidamente.

MSC Nastran é uma escolha de alta qualidade para engenheiros que procuram executar a estrutura do sistema análise.
