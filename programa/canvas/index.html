Canvas é um programa de desenho desenvolvido para criar e editar ilustrações técnicas e visualizações especializadas.

É usado principalmente para editar imagens baseadas em vetor, mas também suporta imagens raster.

O Canvas a interface do usuário parece semelhante a uma tela de desenho real para editar desenhos.

Ela contém muitas paletas de ferramentas diferentes que podem ser exibidas e organizadas para personalizar o espaço de trabalho.

Um recurso de grade da tela ajuda a alinhar e uniformemente distribuir gráficos vetoriais para precisão.

O programa oferece suporte a mais de 100 formatos de arquivo de entrada, como AutoCAD ( .DWG e .DXF ), Photoshop (). PSD e .PDD ), Illustrator ( .AI ), formato de documento portátil ( .PDF ), CorelDRAW ( 70LTmCDR3702. >), PowerPoint ( .PPT ), DICOM (. DCM ), imagens RAW e outros arquivos de imagem raster, como arquivos .PNG , .JPG e .GIF

Os desenhos podem ser salvos no formato proprietário do Canvas ( .CVX ) ou podem ser exportados para um dos vários tipos de arquivo diferentes.

O Canvas é usado para uma ampla variedade de desenhos técnicos e pode ser usado para criar visualizações especializadas.

Por exemplo, para aplicações médicas e geológicas, o programa pode registrar dados numéricos de imagens de pesquisas médicas e fornecer ferramentas de imagem para exibir os dados dentro do ambiente de visualização.

Seu motor CGM (Computer Graphic Metafile) proprietário é compatível com os padrões da Aviation Transportation Authority (ATA) e do Petroleum Industry Protocol (PIP).

O suporte do programa para visualização científica também o torna uma ferramenta útil para geológico e médico profissionais de pesquisa.

O Canvas inclui uma paleta de Traços Sísmicos, que ajuda você a estudar as visualizações de dados sísmicos.

O aplicativo também inclui ferramentas para aprimorar imagens e visualizar formatos de dados científicos.

Para profissionais de ilustração técnica, o Canvas é um programa de desenho impressionante.

O suporte do Canvas a uma ampla variedade de formatos de arquivo da indústria, bem como suas ferramentas de edição profissional, o torna uma ferramenta valiosa.

Para o desempenho que você obterá do aplicativo, o preço vale a pena.
