Visão geral do software

Principais características

Reproduz arquivos mod, como MOD e XM
Baixa arquivos de sites HTTP e FTP
Oferece listas de reprodução de arrastar e soltar
Converte arquivos mod em arquivos WAV

O CocoModX de Sven Janssen é um reprodutor de áudio gratuito.

O programa foi desenvolvido apenas para Mac OS X, mas foi descontinuado.

O CocoModX pode reproduzir uma variedade de arquivos de módulo de áudio, como sites FTP e HTTP.

Os usuários podem então criar listas de reprodução a partir da variedade de arquivos de módulo suportados, arrastando e soltando-os nas listas de reprodução.

O CocoModX oferece aos usuários recursos úteis, como a capacidade de alternar entre mono e estéreo, alterar a interface do usuário entre as aparências de metal escovado e pinestripe e converter qualquer arquivo mod para o formato WAVE.

CocoModX é um aplicativo decente que pode reproduzir uma ampla variedade de arquivos mod.

No entanto, o player foi descontinuado e não é mais compatível com o desenvolvedor.

Portanto, os usuários devem procurar em outro lugar, como o reprodutor de mídia VideoLAN VLC, por um aplicativo para reproduzir seus arquivos de módulo de áudio.
