Visão geral do software

Principais características

Abra e edite arquivos STL
Especifique o material e a espessura
Ajuste uma ampla variedade de preferências de impressão por camada
Exportar arquivo fatiado para cartão SD para impressão

Spark Studio é o software proprietário necessário para usar uma impressora 3D SparkMaker.

O programa abre .STL arquivos de modelo 3D e os prepara para impressão.

.

O usuário pode fazer ajustes no ângulo e adicionar suporte.

Muitas preferências de impressão podem ser especificadas por camada.

Assim que as configurações forem finalizadas, o Spark Studio pode gerar um Cartão SD a ser transferido para uma impressora SparkMaker.

O arquivo desejado deve ser nomeado print.wow.

A impressora SparkMaker FHD apresenta velocidades de impressão rápidas e alta resolução.

Combinado com o software Spark Studio fácil de usar, o SparkMaker oferece uma opção de impressão 3D relativamente acessível para artistas, designers e amadores.
