Visão geral do software

Principais características

TeXworks permite que você edite arquivos fonte LaTeX
Yap permite que você visualize a saída do TeX
O gerenciador de pacotes MiKTeX ajuda você a baixar pacotes LaTeX

MiKTeX é uma implementação de código aberto de TeX / LaTeX comumente usada para escrever artigos, livros e outras publicações técnicas.

Está disponível para ser executado em Windows ou como uma edição portátil que você pode traga com você em um dispositivo de armazenamento onde quer que você vá.

MiKTeX fornece ferramentas que permitem criar publicações técnicas usando a linguagem de marcação TeX / LaTeX.

O pacote vem com TeXworks, que é usado para editar arquivos fonte LaTeX e Yap, um visualizador de saída TeX que otimiza o ciclo de edição, compilação e visualização.

Além disso, MiKTeX possui um gerenciador de pacotes para ajudá-lo a baixar plug-ins LaTeX.

Há uma variedade de distribuições TeX disponíveis para Windows, como XEmTeX, proTeXt e W32TeX.

Mas MiKTeX provou ser uma das melhores distribuições TeX disponíveis como vários cientistas, engenheiros elétricos, e físicos h ave atestada.

MiKTeX é uma boa escolha para a composição de suas publicações técnicas.
