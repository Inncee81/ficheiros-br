Visão geral do software

Principais características

Verificação em tempo real de malware
Firewall avançado para impedir a entrada de hackers
Proteção do navegador contra sites perigosos
Painel dos pais para controlar o conteúdo acessível

AVG Internet Security é um programa Windows usado para proteger contra ameaças de segurança.

Ele vem com uma variedade de recursos para proteger contra muitos tipos de malware, incluindo vírus, ransomware, trojans, spyware e rootkits.

O Internet Security verifica seu computador em tempo real em busca de malware e usa IA para detectar amostras de malware que ainda não foram identificadas pela equipe do AVG ThreatLabs.

O software possui um firewall avançado para impedir a entrada de hackers, análise instantânea de qualquer comportamento suspeito de aplicativos em execução e um destruidor de arquivos para exclusão segura de arquivos confidenciais.

O Internet Security protege seu computador enquanto você navega, faz compras e realiza operações bancárias online.

Ajuda a evitar sites perigosos, verificando se há ameaças em links clicados, bloqueia anexos de e-mail e spam perigosos e verifica aplicativos que podem ter sido baixados sem o seu conhecimento.

O programa fornece um painel para controlar o conteúdo acessível em diferentes dispositivos, o que é útil para os pais.

Além disso, o Internet Security apresenta a ferramenta WiFi Guard para protegê-lo de pontos de acesso maliciosos que podem ser usados ​​por hackers. .

O AVG Internet Security apresenta inteligência artificial para identificar ameaças em tempo real, um firewall avançado e recursos de segurança dos pais.

É uma escolha sólida para proteger PCs domésticos.
