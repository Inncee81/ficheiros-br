Principais características

Apresenta a interface de desktop do Finder que permite organizar seus aplicativos, arquivos e downloads
Incluído com uma variedade de aplicativos, incluindo iTunes, Safari, Fotos e Livros
Usado em ambientes pessoais, educacionais e profissionais

OS X, ou macOS, é um sistema operacional (OS) desenvolvido pela Apple e projetado para ser executado em computadores Macintosh, como o iMac e MacBook.

O macOS foi originalmente lançado em 2001 e é baseado em NeXTSTEP, um sistema operacional desenvolvido pela NeXT, que a Apple adquiriu quando Steve Jobs voltou para a Apple em 1997.

Ambos NeXTSTEP e macOS são baseados em Unix e usam o mesmo kernel Mach.

O sistema operacional foi chamado de "Mac OS X" até a versão OS X 10.8, quando a Apple retirou "Mac" do nome.

Então, "OS X" mudou para "macOS" com o lançamento do macOS 10.12 Sierra em 2016.

O sistema operacional vem com uma variedade de aplicativos, como iTunes, Safari, iMovie, Fotos, Livros, QuickTime Player e TextEdit, que podem ser usados ​​para uma variedade de finalidades, como ouvir música, navegar na web, ver fotos e ler e-books.

O Finder é o desktop interface operacional para macOS e inclui vários recursos padrão.

macOS inclui uma barra de menus, que é fixada na parte superior da tela.

O Finder também inclui um Dock, que é exibido por padrão na parte inferior da tela, que permite organizar todos os seus aplicativos, arquivos, documentos, downloads, filmes, músicas e imagens.

Depois de Windows, o macOS é o sistema operacional de desktop mais usado e é popular entre usuários pessoais, educacionais e profissionais. O sistema operacional oferece uma variedade de recursos que a Apple atualiza de forma consistente, com novas versões lançadas anualmente, normalmente em setembro ou outubro.

E embora o sistema operacional venha com Mac computadores que geralmente são mais caros do que os que executam Windows, o macOS ainda é um sistema operacional impressionante.
