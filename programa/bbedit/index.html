Principais características

Suporta linguagens de programação populares, como Ruby, Java e C
Fornece ferramentas úteis, incluindo coloração de sintaxe para linguagens de código-fonte
Apresenta projetos BBEdit para organizar arquivos
Permite a personalização de teclas de menu, funções e coloração de sintaxe

Bare Bones BBEdit é um editor de texto, código e marcação para Mac OS X.

Ele é projetado especificamente para web designers e desenvolvedores de software.

BBEdit oferece suporte a uma ampla variedade de linguagens de computador, como C ++, Fortran, Ruby, Python, JavaScript, Tex, PHP e HTML.

O editor possui uma interface intuitiva com ferramentas de marcação HTML e CSS que reduzem os erros de codificação e garantem a estrutura adequada do código .

O programa fornece recursos que incluem a capacidade de pesquisar e substituir vários arquivos, sintaxe de cores em várias linguagens de código-fonte e texto e código completos.

O aplicativo ajuda você a se manter organizado como você trabalhar em vários arquivos ao mesmo tempo.

Você pode adicionar arquivos a um Projeto BBEdit e salvá-los com a extensão .BBPROJECT .

O BBEdit também permite que você personalize o aplicação para refle ct como você trabalha.

Você pode personalizar as teclas de menu, criar suas próprias funções, modificar o comportamento dos menus integrados e criar sua própria coloração de sintaxe.

O BBEdit oferece ótimos recursos para ajudar você projeta e desenvolve.

Embora tenha um custo financeiro maior do que editores gratuitos, como o TextWrangler, o programa oferece recursos de codificação úteis.

Se você estiver disposto a pagar, Bare Bones O BBEdit provará valer a pena.
