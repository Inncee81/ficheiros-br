Principais características

Acesse sua música rapidamente com uma interface gráfica de usuário rápida
Use o programa em qualquer lugar sem a necessidade de instalação
Personalize sua experiência com um layout de teclado totalmente configurável

OpenMPT (Open ModPlug Tracker) é um rastreador de módulo de áudio gratuito e de código aberto que é usado para criar e tocar música.

É baseado no ModPlug Tracker que foi originalmente desenvolvido por Olivier Lapicque.

Com o OpenMPT você pode acessar e editar músicas MIDI de forma rápida e fácil.

Ele usa uma GUI nativa rápida Windows ao contrário de muitos outros rastreadores populares que ainda usam interfaces baseadas em DOS.

Além disso, ele não precisa ser instalado para ser executado.

Isso permite que você use o aplicativo em muitos computadores diferentes diretamente do arquivo.

OpenMPT tem muitos recursos que você pode usar para editar sua música.

Suporta muitos formatos diferentes de amostra e áudio de instrumento, como WAV, PAT e RAW.

O OpenMPT também permite que você edite amostras com efeitos como normalização, fase inversão, ampliação e remova DC offset l.

Você pode até editar um módulo em várias janelas abrangendo vários monitores.

Se você está procurando um ótimo rastreador de módulo totalmente gratuito, o OpenMPT é uma ótima escolha.

Muitos profissionais usam este programa para complementar suas outras ferramentas profissionais.
