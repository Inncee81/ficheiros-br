Apple Terminal é um aplicativo empacotado com Mac OS X.

É um emulador de terminal com uma interface de linha de comando que permite aos usuários digitar comandos Unix para obter mais funcionalidade de seu Mac.

O utilitário pode ser usado para configurar um Mac e fazer um script e programar várias operações do computador.

Você pode alterar os períodos de backup padrão no Time Machine, desabilitar o recurso de restauração automática, mostre arquivos ocultos no Finder, amplie ícones de encaixe e muito mais.

O terminal permite que você personalize uma variedade de preferências.

Você pode definir janelas, shells e comportamentos de guia.

A interface também é personalizável, pois você pode escolher aplicar diferentes perfis à interface, que incluem a imagem ou cor de fundo, cores do texto, tamanho da janela, aparência da guia e ações do teclado.

Desde o sistema operacional X fornece um gráfico impressionante al interface do usuário, a maioria dos usuários casuais do OS X nunca usará o Terminal.

No entanto, pode ser uma ferramenta útil para personalizar elementos, executar tarefas que podem exigir software adicional e solucionar problemas em seu sistema operacional.

Apple Terminal é um utilitário de linha de comando útil para Mac usuários do OS X.
