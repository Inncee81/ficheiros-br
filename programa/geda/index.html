gEDA é um kit de ferramentas de código aberto de ferramentas de Electronic Design Automation (EDA).

O conjunto pode ser usado para captura esquemática, projeto de circuito elétrico, prototipagem, simulação e produção.

O pacote gEDA oferece uma variedade de aplicativos que permitem executar diferentes tarefas, como gerenciamento de atributos, geração de lista de materiais e simulação digital e analógica.

A coleção de ferramentas inclui :.

gEDA é usado principalmente para projetar layouts de placas de circuito impresso (PCB).

As ferramentas fornecidas no gEDA permitem criar um PCB com até 8 camadas e uma quantidade ilimitada de redes e componentes.

O kit de ferramentas também inclui um visualizador Gerber, que permite abrir arquivos PCB, como .GBR e .GBO , que são salvos no formato Gerber popular.

gEDA é um formato útil coleção de ferramentas que permite projetar PCBs, realizar simulações e netlist em mais de 20 formatos de lista de rede.

Embora o gEDA não tenha tantos recursos de ponta quanto outros pacotes de EDA, ainda é um conjunto útil de ferramentas para usuários do OS X e Linux.
