Apache OpenOffice é um pacote de escritório de código aberto semelhante ao Microsoft Office.

O programa descendeu do OpenOffice.org e está intimamente relacionado ao LibreOffice.

Ele oferece vários programas de produtividade de escritório semelhantes Programas do Microsoft Office.

Programas incluídos no pacote Open Office:

  • Writer - Um processador de texto
  • Calc - Um aplicativo de planilha
  • Draw - Um editor de gráficos vetoriais
  • Impress - Um criador de apresentação
  • Base - Um aplicativo de gerenciamento de banco de dados
  • Matemática - Um criador de equação matemática

OpenOffice é usado para criar documentos que auxiliam na produtividade e organização de escritório.

Os aplicativos são semelhantes ao Microsoft Office e são usados ​​para criar apresentações, planilhas, desenhos s, diagramas, equações, documentos de texto e outros documentos de escritório.

Os arquivos podem ser salvos no formato de arquivo proprietário do OpenOffice ou em formatos de arquivo do Microsoft Office, como .DOC , .PPT e .XLS.

Cada um dos arquivos OpenOffice também pode ser exportado como um arquivo .P24236 para fácil compartilhamento. >

Apache OpenOffice é uma suíte de escritório sólida.

Oferece funcionalidade semelhante ao Microsoft Office e é compatível com muitos formatos de arquivo padrão da indústria.

Se você está procurando uma alternativa gratuita ao Microsoft Office, o Apache OpenOffice é uma boa escolha.
