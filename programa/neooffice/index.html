Visão geral do software

Principais características

Suporta os principais formatos de arquivo de produtividade de escritório, como .DOCX e .XLSX
Possui forte integração com os recursos do Mac OS X, como Gatekeeper e Versões.
Fornece suporte para vários idiomas, como árabe, espanhol, alemão e russo.

Planamesa NeoOffice é um pacote de escritório de código aberto semelhante ao Microsoft Office, mas desenvolvido especificamente para Mac OS X.

O programa descendeu do OpenOffice.org e está intimamente relacionado ao LibreOffice e ao Apache OpenOffice.

O pacote NeoOffice oferece vários aplicativos que permitem criar e editar documentos de texto, desenhos, diagramas, apresentações e planilhas.

Suporta uma ampla variedade de formatos de arquivo de produtividade de escritório.

Mais notavelmente, o pacote pode ler e gravar a maioria dos formatos de arquivo XML do Microsoft Office, como .DOCX , .XLSX e .PPTX . >

.

Como mencionado anteriormente, o NeoOffice é semelhante ao LibreOffice e ao OpenOffice, mas foi desenvolvido especificamente para Mac OS X.

NeoOffice inclui o uso do Mac Recurso X “Gatekeeper” para proteger impede que você instale software malicioso, Mac recurso “Versões” do OS X para permitir a restauração de versões anteriores do seu documento e Mac comportamento de realce de texto do OS X para oferecer a mesma sensação de outros aplicativos do Mac OS X .

Planamesa NeoOffice é um pacote de escritório competente para Mac OS X.

É compatível com muitos formatos de arquivo padrão da indústria e é executado nativamente no Mac OS X.

No entanto, o NeoOffice não é mais oferecido como um download gratuito (requer uma doação) e o Apache OpenOffice e o LibreOffice agora podem ser executados nativamente no Mac OS X.

Então, se você estiver procurando para uma alternativa gratuita ao Microsoft Office, OpenOffice ou LibreOffice são as melhores opções.
