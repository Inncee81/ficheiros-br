Principais características

Módulos de contatos, calendário, tarefas e notas
Visualize, edite, encontre e faça backup de dados em seu dispositivo Palm OS
Baixe e transfira jogos para o seu dispositivo

Palm Desktop é um aplicativo Windows usado para gerenciar dispositivos portáteis Palm OS.

O software foi descontinuado em 2008, mas ainda está disponível para Windows, mas não para OS X.

O programa consiste em 4 módulos principais que são Contatos, Calendário, Tarefas e Observações.

O Palm Desktop permite que você visualize, edite, encontre e faça backup de dados em seu dispositivo Palm OS.

Você também pode baixar jogos e transferi-los para o seu dispositivo.

O Palm Desktop vem com todos os dispositivos Palm, o que permite a você gerenciar o dispositivo.

O programa é fácil de usar e oferece a funcionalidade de visualizar, editar, localizar e fazer backup de dados.

Se você possui um dispositivo Palm, deve usar o aplicativo Palm Desktop.
