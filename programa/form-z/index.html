AutoDesSys Form·Z é um programa CAD usado para projetar espaços e formulários 3D para muitos fins diferentes.

É comumente usado para modelagem 3D, desenho arquitetônico, animação e renderização.

Form·Z vem com um grande conjunto de ferramentas para manipulação de formulários 2D e 3D.

Os formulários 3D podem ser criados com uma interface visual ou por fórmulas que você projeta.

As ferramentas de criação visual permitem que você crie formas, incluindo primitivas analíticas, objetos esféricos, metaballs e metaformz, e formas personalizadas.

Existem também folhas de layout que você pode usar para criar diferentes vistas de um modelo e incluir notas e dimensões.

Existem também muitos recursos fornecidos pelo Form·Z que ajudam a agilizar seu fluxo de trabalho.

Possui uma interface inteligente que determina o que você está tentando realizar e ajuda a evitar ações indesejadas.

Form·Z também simplifica seu fluxo de trabalho agrupando ferramentas relacionadas em áreas de trabalho, para que você sempre possa encontrar o que precisa.

Form·Z é uma boa ferramenta de modelagem para design 3D, especialmente para arquitetura.

Se você é um arquiteto que está procurando um bom programa CAD, o Form·Z é uma ótima opção.

Se você está interessado em criar modelos para jogos, pode não ser a melhor solução.
