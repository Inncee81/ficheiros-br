Principais características

Desenvolvimento rápido de aplicativos GUI
Interface de desenvolvimento de formulário de arrastar e soltar
Criação de arquivos EXE, controles ActiveX e arquivos DLL

Microsoft Visual Basic é uma linguagem de programação orientada a objeto e ambiente de desenvolvimento integrado usado para criar programas para o sistema operacional Windows.

Não é mais compatível e foi substituído pelo Visual Basic .NET.

Visual Basic foi um dos primeiros sistemas de desenvolvimento de computador a incluir uma linguagem e um ambiente de desenvolvimento integrado (IDE).

O IDE permite desenvolver interfaces de usuário com um simples arrastar e soltar abordagem.

É uma boa ferramenta para Rapid Application Development (RAD) devido à sua capacidade de construir rapidamente protótipos de programas.

Como o Visual Basic não é mais suportado, não é um desenvolvimento recomendado solução.

Se você gostou do ambiente de desenvolvimento e do código do VB, pode estar interessado em olhar para o Visual Basic .NET.
