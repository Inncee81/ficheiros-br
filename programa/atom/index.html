Principais características

Suporta vários formatos de arquivo, como arquivos HTML, XML, PHP e CSS
Vem com autocompleção inteligente, encontrar e substituir, e destaque de sintaxe
Oferece um navegador de sistema de arquivos e teletipo para colaboração com outros
Fornece pacotes para instalação de funcionalidades adicionais
Permite personalizar o tema e o estilo da interface do usuário

GitHub Atom é um editor de texto de código aberto de plataforma.

Ele pode ser usado para abrir vários formatos de arquivo de texto, fornece recursos úteis de edição e permite que os usuários personalizem o editor de texto instalando pacotes para adicionar funcionalidade.

O Átomo suporta vários formatos de arquivo, incluindo .HTML, .CSS, .PHP e .XML.

O editor de texto apresenta uma autocompletação inteligente para escrever código mais rapidamente e encontrar e substituir para encontrar, substituir e visualizar texto em um único arquivo ou em projetos.

Permite que os usuários compartilhem seu espaço de trabalho com outros e colaborem em tempo real com o Teletype.

Ele também vem com um navegador de sistema de arquivos para trabalhar com projetos e vários painéis para comparar e editar código.

Uma das melhores características do editor de texto é o seu gerenciador de pacotes embutido.

Você pode pesquisar e instalar ou criar seus próprios pacotes para adicionar funcionalidade ao editor.

Esses pacotes podem ser qualquer coisa, desde melhorias de destaque de sintaxe até suporte a novas linguagens de programação até um relógio personalizável na barra de status.

Atom é um editor de texto que vem com muitas das mesmas características dos editores de texto básicos.

No entanto, sua extensibilidade, através do uso de pacotes, é o que realmente o diferencia de outros editores.

Se você está procurando um editor de texto completamente personalizável, o GitHub Atom é uma ótima opção que rivaliza com os melhores editores de texto.
