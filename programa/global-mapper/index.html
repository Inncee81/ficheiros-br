Visão geral do software

Principais características

Suporte para mais de 250 tipos diferentes de dados raster, vetoriais e de elevação
Suporte de banco de dados espacial para SQL Server, MySQL Spatial e outros bancos de dados
Suporte a plug-ins para estender a funcionalidade
Capacidade de script para automatizar tarefas recorrentes
Processamento em lote que permite que vários arquivos selecionados sejam convertidos, reprojetados, colocados em grade e renomeados

Global Mapper é um aplicativo GIS usado para visualizar e converter dados vetoriais, raster e de elevação.

Foi desenvolvido para o sistema operacional Windows.

O aplicativo suporta mais de 250 diferentes tipos de dados raster, vetoriais e de elevação, como 3DS Max, Blender, OBJ, COLLADA 3D, ERMapper Compressed Wavelet, ESRI Shapefiles, LandXML e GeoJSON.

Você também pode importar e exportar bancos de dados espaciais, incluindo MS SQL Server, Oracle Spatial e ESRI ArcSDE.

.

Para processamento de dados, o Global Mapper fornece processamento em lote que permite que vários arquivos selecionados sejam convertidos, reprojetados, colocados em grade e renomeados.

Também fornece a ferramenta de extração de recurso que pode ser usada para analisar áreas que correspondem a uma cor ou intervalo de cores que estão dentro de uma determinada elevação ou têm uma camada de inclinação especificada.

Mapeador global é usado por iniciantes e profissionais e compete diretamente com outros aplicativos GIS desenvolvidos pela ESRI, GeoMedia e MapInfo.

A ferramenta GIS suporta uma grande variedade de tipos de dados e fornece ferramentas úteis, como extração de recursos e processamento em lote.

Embora o Global Mapper não ofereça tantos recursos quanto os aplicativos mais caros, como ERDAS IMAGINE, ainda é uma escolha de qualidade.
