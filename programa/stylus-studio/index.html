Visão geral do software

Principais características

Numerosos editores XML com texto, árvore e visualizações de grade
Ferramentas de consulta que suportam edição, mapeamento e depuração visual de XQuery
Ferramentas EDI para converter, mapear e implantar aplicativos de integração de dados XML
Ferramentas de publicação XML para criar relatórios XML para aplicativos de publicação XML
Ferramentas de design Java e HTML / XHTML para construir aplicativos e sites Java

Progress Stylus Studio é um IDE que permite aos usuários editar e transformar documentos XML usando uma variedade de ferramentas e designers visuais.

Está disponível apenas para a plataforma Windows.

O Stylus Studio vem com vários editores XML com texto, árvore e visualizações de grade que suportam detecção de código, coloração de sintaxe, preenchimento automático orientado por esquema e dobramento de código.

O IDE permite que você importe dados XML de CSV, separados por tabulação , EDI e formatos de dados binários, bem como outros utilitários de importação e exportação de dados para XML e RDBMS.

O IDE também oferece ferramentas Java e HTML / XHTML para construir aplicativos Java habilitados para XML e dados XML sites orientados.

Outras ferramentas XML que vêm com o pacote incluem um editor de esquema XML, um designer de pipeline XML, ferramentas EDI, ferramentas XPath, um editor de definição de tipo de documento e ferramentas de mapeamento XML.

Stylus Studio é uma suíte impressionante de Ferramentas XML usadas para trabalhar com XML XQuery, publicação XML, serviços da Web e outras tecnologias XML.

O IDE oferece vários recursos avançados e suporte a formatos voltados para programadores XML especializados que podem ser opressores para iniciantes.

Editores como o Bloco de notas XML da Microsoft podem ser uma opção melhor para usuários novatos.

No entanto, se você tiver experiência, o Progress Stylus Studio está à altura da tarefa.
