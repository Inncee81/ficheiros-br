O GNU Compiler Collection (GCC) é um conjunto de compiladores de código aberto mais comumente usado para compilar programas C e C ++.

GCC também oferece suporte a várias outras linguagens de programação, incluindo Objective-C, Fortran, Java , Ada, Go e BRIG (HSAIL).

Os compiladores são mais comumente usados ​​em sistemas baseados em Unix, mas uma porta Windows, chamada MiniGW, pode ser usada também.

O GCC inclui vários compiladores diferentes:

  • gcc: Um compilador C
  • g++: Um compilador de C++ e Objective-C
  • gfortran: Compilador do Fortran
  • gcj: Um compilador Java
  • GNAT: Um compilador Ada
  • gccgo: Um compilador GO
O design modular do GCC também permite que plug-ins sejam instalados para estender a função lidade do software.

Funcionalidades adicionais incluem suporte para linguagens de programação que não vêm com o GCC.

Um grande recurso do GCC é sua capacidade de compilar programas para diferentes plataformas.

Isso significa que, embora um desenvolvedor use o GCC em uma plataforma, ele pode ser compilado para outra plataforma de destino.

GCC é uma das escolhas mais comuns para compilar programas C e C ++ no Unix- sistemas baseados.

Se você estiver desenvolvendo para Linux, o GCC é uma boa escolha.

Ele também pode ser usado para Windows, mas outros compiladores são normalmente preferidos , como o compilador incluído com Microsoft Visual Studio .
