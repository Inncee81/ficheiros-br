Visão geral do software

Principais características

Reproduz os arquivos de áudio e vídeo mais comuns
Fornece uma interface de usuário limpa com controles de reprodução simples
Oferece vídeo HD usando compressão H.264
Permite que os usuários gravem seu próprio áudio e vídeo

O Apple QuickTime Player é um reprodutor de mídia gratuito incluído no macOS.

Ele permite que os usuários reproduzam os arquivos de áudio e vídeo mais comuns, incluindo .MOV , .MP4 , .MP3 e .WAV arquivos.

Usando o QuickTime Player, você pode abrir e reproduzir quase todos os arquivos de mídia que você encontrar, seja seu áudio, vídeo ou até mesmo uma série de imagens fixas.

O player usa compressão de vídeo H.264 para reproduzir vídeos em HD vibrante e sua interface de usuário simples torna a reprodução e o compartilhamento de mídia uma brisa.

O QuickTime Player também permite que os usuários gravem seu próprio áudio e vídeo.

Você pode gravar áudio usando um microfone, gravar vídeo usando a câmera do computador e até mesmo criar uma gravação de tela usando o QuickTime Player.

Se você é um usuário Apple, QuickTime Player é uma das suas melhores opções ns para reprodução de áudio e vídeo.

Suporta uma seleção impressionante de arquivos de mídia e, graças à sua interface de usuário elegante, é bastante fácil de usar.

O fato de QuickTime Player vir pré- instalado em Macs fecha o negócio.

NOTA: QuickTime Player estava disponível anteriormente para Windows, mas a Apple não oferece mais suporte para as versões Windows do QuickTime Player.
