Principais características

Importar arquivos de modelo 3D para uma biblioteca habilitada para nuvem
Renomear, editar e adicionar instruções de impressão
Monitore as impressões por meio de uma câmera integrada
Conecte-se a uma impressora MakerBot Replicator 3D via Wi-Fi, Ethernet ou USB

MakerBot Desktop é um aplicativo usado para gerenciar e compartilhar impressões 3D.

Está disponível para plataformas Windows e OS X.

O aplicativo permite que você importe . STL , .OBJ e .THING Arquivos de modelo 3D em uma biblioteca habilitada para nuvem.

Na biblioteca, você pode renomear, editar e adicionar instruções para arquivos pessoais, layouts e coleções.

Você também pode visualizar impressões 3D.

O MakerBot Desktop pode se conectar à impressora 3D MakerBot Replicator via Wi-Fi, Ethernet ou USB.

O MakerBot Desktop oferece ferramentas para monitorar e controlar suas impressões.

Oferece acesso rápido a comandos como suspender, alterar filamento ou cancelar.

Você pode monitorar suas impressões por meio de uma câmera integrada, tire fotos de suas impressões e carregue-as diretamente no Thingiverse, um comunidade de design 3D online.
