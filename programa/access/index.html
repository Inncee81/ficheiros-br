Visão geral do software

Principais características

Sistema de gerenciamento de banco de dados relacional robusto (RDBMS) para o ambiente de desktop
Fornece ferramentas visuais para gerenciar esquemas de banco de dados, registros, relatórios, consultas e formulários de entrada
Fornece modelos de banco de dados pré-configurados e pode usar modelos compartilhados
Suporta importar e exportar vários formatos de dados diferentes

O Microsoft Access é um sistema relacional de gerenciamento de banco de dados (RDBMS) para o ambiente de desktop.

O software foi projetado principalmente para usuários únicos, mas fornece muitos recursos encontrados em aplicativos RDBMS em larga escala, como o Microsoft SQL Server e o Oracle Database.

Ao contrário da maioria dos outros aplicativos Microsoft Office, o Access só está disponível para Windows.

O Access oferece ferramentas visuais para criar e preencher bancos de dados.

Inclui ferramentas de fluxo de trabalho para fazer aplicativos de Web on-line orientados a banco de dados, que são úteis para a entrada colaborativa de dados e gerenciamento de dados.

Depois de criar e desenvolver um aplicativo Web no Access, você pode publicá-lo em um servidor Microsoft SharePoint ou em um site do Office 365.

O Microsoft Access também inclui um conjunto completo de ferramentas para gerenciamento de bancos de dados, permitindo que você desenvolva bancos de dados simples e complexos.

Por exemplo, você pode gerar esquemas, criar formulários de entrada e criar relatórios sem ter que escrever nenhum código de banco de dados.

Os bancos de dados de acesso completo são encapsulados como arquivos .ACCDB (ou .MDB arquivos para versões mais antigas), que podem ser facilmente transferidos para outros computadores – algo que não é facilmente realizado com a maioria dos aplicativos RDBMS em larga escala.

O acesso é comumente usado em ambientes profissionais de pequeno e médio porte para rastrear estoques, informações de clientes e outros dados relacionados a negócios.

Sua acessibilidade e facilidade de uso fazem dele uma boa opção para rastrear seus contatos ou bens pessoais em casa também.
