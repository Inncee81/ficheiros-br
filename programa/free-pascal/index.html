Visão geral do software

Principais características

Escreva, compile e depure programas escritos em Pascal
Vem com uma biblioteca de tempo de execução e biblioteca de componentes Freepascal
Fornece propriedades globais, sobrecarga de função e sobrecarga de operador
Oferece uma comunidade de usuários ativa

Free Pascal, anteriormente conhecido como FPK Pascal, é um programa de código aberto usado para compilar uma variedade de dialetos Pascal.

O IDE pode ter como alvo uma variedade de arquiteturas de multiprocessador, como Intel x86, AMD64 / x86 -64, PowerPC e ARM e sistemas operacionais, como Windows, OS X, Linux, FreeBSD e Nintendo Wii.

Free Pascal permite escrever, compilar e depurar programas e é fonte compatível com Turbo Pascal (TP) e Delphi.

O pacote vem com uma Run-Time-Library (RTL) que fornece funções básicas de programação e a Freepascal-Components-Library, que contém classes úteis.

Alguns outros recursos do Free Pascal incluem sobrecarga de operador, propriedades globais e sobrecarga de função.

Além disso, o IDE vem com uma comunidade de usuários ativa que pode ajudar a responder a perguntas de programação gerais e específicas.

Free Pascal é um profissional popular grama para usuários que procuram compilar Pascal e Object Pascal.

O IDE também é conhecido por alimentar o IDE Lazarus, que é compatível com Delphi.

Pascal livre não é para todos os programadores, mas será interessante para você se quiser aprender apenas uma linguagem de programação e tiver uma grande base de código Pascal / Object Pascal.
