Visão geral do software

Principais características

Compactar e descompactar uma ampla gama de formatos de arquivo
Arquivos abertos que têm caracteres não ingleses no nome do arquivo
Configure e use facilmente o programa como um substituto para Mac Archive Utility

O Unarchiver é um programa de descompressão de dados gratuito para Mac que suporta muitos formatos compactados diferentes.

Foi desenvolvido para substituir o Mac Archive Utility integrado que não suporta muitos formatos.

O Unarchiver foi projetado para ser fácil de configurar e usar.

Depois de instalá-lo, tudo o que você precisa fazer é defini-lo como seu programa padrão para arquivos compactados (que podem ser feito facilmente dentro do programa).

Ele também é projetado para fluir melhor com o layout do Finder para ajudar a fornecer uma experiência de computação mais consistente.

O Unarchiver é um grande avanço em relação Mac Archive Utility.

Se você está procurando uma alternativa gratuita para compactar e descompactar arquivos e arquivos no Mac, é uma ótima opção.
