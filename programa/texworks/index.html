Visão geral do software

Principais características

GUI para o sistema de composição TeX
Interface simples e familiar baseada em aplicativos comuns
Visualizador de PDF integrado para visualizar a saída do projeto
Sincronização de fonte / visualização para encontrar rapidamente a localização no código-fonte da visualização de PDF

TeXworks é um aplicativo de código aberto criado como uma GUI para TeX, um sistema de composição que é o padrão para a criação de publicações técnicas.

Sem uma interface de usuário instalada, o TeX deve ser acessado por meio de uma interface de linha de comando .

TeXworks requer uma instalação do TeX para rodar.

A interface gráfica do usuário fornecida pelo TeXworks simplifica o processo de criação de documentos TeX.

Seu design é baseado em outro ferramentas de plataforma cruzada de código aberto para fornecer uma experiência familiar ao usuário.

TeXworks fornece um visualizador de PDF integrado para visualizar facilmente a saída de seu projeto com sincronização de origem / visualização.

Com código-fonte / visualização da sincronização você pode clicar com o botão direito do mouse em um ponto na visualização do PDF e ver onde ele está localizado na fonte.

TeXworks torna o trabalho com o TeX mais acessível do que usar um prompt de comando.

Se você está procurando uma maneira fácil e gratuita de criar documentos TeX, TeXworks é uma boa escolha.

Se você deseja mais recursos como gerenciamento de projeto, pode procurar outro lugar.
