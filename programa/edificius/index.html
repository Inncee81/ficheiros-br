Visão geral do software

Principais características

Ferramentas de design arquitetônico, interior e paisagístico
Modelagem de sistemas MEP
Integração com Google Maps para modelagem de terreno
Projeto 4D GANTT Cronogramas de tempo para gerenciar a linha do tempo dos projetos
Renderizações fotorrealistas, edição de vídeo e tours de realidade virtual envolvente
Suporte para uma variedade de formatos 2D e 3D, como DXF, DWG, OBJ e SKP

ACCA Edificius é um aplicativo de Modelagem de Informação de Construção (BIM) para Windows usado por profissionais de Arquitetura, Engenharia e Construção (AEC) para criar projetos arquitetônicos 2D e 3D.

É um "all in" um "pacote de software que fornece ferramentas para todo o processo BIM, que inclui planejamento, concepção, construção e gerenciamento do projeto.

Edificius fornece uma variedade de ferramentas úteis para planejar e criar um projeto, que inclui a arquitetura e design de interiores do edifício.

O programa fornece modelagem de sistemas de Engenharia Mecânica, Elétrica e de Encanamento (MEP) para configurar as instalações técnicas do edifício, como linhas de água e fiação elétrica.

Você também pode integrar recursos do terreno com o Google Maps para projetos de paisagismo externo, como parques, jardins e pátios.

Depois de terminar de projetar seu projeto ect, você pode compartilhar e apresentá-lo com a Edificius de várias maneiras.

Alguns dos recursos de apresentação profissional incluem vídeos, renderizações fotorrealistas e tours de realidade virtual imersivos.

Além disso, ao começar construção no projeto real, você pode gerenciar a linha do tempo em gráficos 4D GANTT para cada tarefa do início à conclusão e estimar o custo de cada detalhe do projeto.

Edificius salva projetos como .EDF arquivos e pode importar vários formatos 2D e 3D, como SketchUp ( .SKP ), AutoCAD Drawing Database ( .DWG ), Autodesk Drawing Exchange ( .DXF ), Industry Foundation Classes ( .IFC ), 3D Studio Scene ( .3DS ) e Autodesk ASCII Scene Export ( .ASE ).

O programa também pode exportar projetos para o Autodesk Drawing Exchange (DXF), AutoCAD Drawin g Database (DWG), Wavefront 3D Object ( .OBJ ), SketchUp (SKP), COLLADA ( .DAE ), Estereolitografia ( .STL ), Polygon Formatos de modelo ( .PLY ) e Industry Foundation Classes (IFC).

ACCA Edificius é um pacote completo para projetar casas, prédios de escritórios, parques e outros projetos arquitetônicos.

O software apresenta uma interface intuitiva que vem com uma variedade de ferramentas úteis de planejamento, design, gerenciamento e apresentação.

ACCA Edificius é uma ótima ferramenta para profissionais de AEC que procuram assumir um projeto complexo desde o início para terminar em um aplicativo capaz.
