Principais características

IDE popular de videogame para uma variedade de plataformas
Suporte para formatos de gráficos 3D populares, como Maya e 3ds Max
Ferramentas profissionais para a criação de iluminação, efeitos especiais, áudio, materiais, física e terrenos
Assets Store que fornece ativos, como efeitos sonoros e modelos, para ajudá-lo a economizar tempo

Unity Technologies Unity é um mecanismo de jogo 3D para Windows e Mac OS X.

É usado para desenvolver videogames de console, celular, desktop e plug-in da Web.

O Unity apresenta um IDE integrado ao aplicativo que oferece ferramentas e fluxos de trabalho para a criação de conteúdo 2D e 3D.

O IDE permite que você crie animações realistas, com iluminação avançada, terrenos, materiais e recursos de criação física.

O software vem com uma grande seleção de sombreadores simples e avançados, você também pode criar seus próprios.

O Unity permite que você crie cenas vivas e dinâmicas com geração automática de NavMeshes.

Este recurso permite que você determine os limites do espaço navegável para seus personagens de IA em seu jogo e são usados ​​para encontrar caminhos em tempo de execução.

O aplicativo suporta formatos gráficos 3D populares, como Maya, 3ds Max, e Blende r, que permite importar modelos e animações.

A Unity Technologies também fornece uma loja de ativos para Unity que permite adicionar ativos prontos, como armas ou efeitos sonoros de jogos de futebol, ao seu jogo para ajudar você economiza tempo.

O Unity oferece uma variedade impressionante de recursos líderes do setor para ajudá-lo a criar videogames para dispositivos móveis, consoles e plataformas de desktop.

Seja você um desenvolvedor independente ou parte de um grande estúdio, Unity Technologies Unity é uma excelente escolha para desenvolver seu videogame.
