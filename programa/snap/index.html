Snap! é uma linguagem de programação gráfica de arrastar e soltar educacional usada para ensinar ciência da computação a uma ampla faixa de idades.

Era originalmente chamada de Build Your Own Blocks (BYOB) e foi inspirada no Scratch, um programa semelhante.

Ao contrário do BYOB, Snap! fica sem um navegador da Web.

Em vez de exigir que você escreva código para programas, o Snap! fornece uma interface visual.

Você pode criar elementos gráficos chamados sprites e então programá-los para ter comportamentos diferentes.

Para programar seus sprites, tudo que você precisa fazer é arrastar blocos contendo comandos de um script aba e solte-os no sprite alvo.

Os comandos incluem controle de movimento, sons e aparência, bem como eventos, variáveis ​​e declarações lógicas.

Se você estiver interessado em aprender mais sobre os conceitos de programação de computadores ou o ensino desses princípios para outras pessoas, Snap! é uma ótima escolha.

É especialmente útil para pessoas que podem achar intimidante ir direto para o código.
