GDevelop é um aplicativo de plataforma cruzada de código aberto usado para criar jogos 2D sem nenhum conhecimento de programação.

Você pode criar jogos para uma variedade de plataformas, como Windows, OS X, Android , iOS e a web.

O programa foi originalmente denominado Game Develop pelo criador por Florian Rival, mas depois mudou para GDevelop.

GDevelop permite que você crie jogos usando cenas, objetos, e eventos.

O editor de cena do programa permite que você edite visualmente seus níveis e suporta múltiplas camadas para criar efeitos de paralaxe.

Você pode então adicionar vários objetos, como sprites com várias animações, objetos de texto , formas personalizadas, luzes e sombras dinâmicas e caixas 3D e aplicam comportamentos a elas.

GDevelop também fornece eventos para determinar a lógica do seu jogo, definindo as condições e ações para sequências.

Como você bu ild seu jogo, você pode visualizar seu progresso para experimentar a jogabilidade.

Quando terminar o processo de criação, você pode exportá-lo como um jogo para web, celular ou sistema operacional nativo.

GDevelop também fornece uma ampla quantidade de recursos para ajudá-lo a começar e se aclimatar com seu ambiente.

GDevelop é uma ferramenta gratuita projetada para todos os tipos de usuários para criar jogos para diversão ou para venda sem nenhum conhecimento de programação.

No entanto, o aplicativo não se destina a programadores avançados e não deve ser confundido com aplicativos de desenvolvimento de jogos mais profissionais.

Se você está procurando uma ferramenta de criação de jogos como Stencyl ou Construct 2, o GDevelop é ótimo opção.
