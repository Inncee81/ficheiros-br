Principais características

Código fonte e editor de texto simples
Web ferramentas de desenvolvedor
Cliente FTP / SFTP
Conversor de texto para HTML
Validador XML, HTML, JavaScript e CSS

EditRocket é uma ferramenta de desenvolvimento de software usada principalmente como um código-fonte e editor de texto.

Está disponível para Windows, OS X e Linux.

O código-fonte O editor suporta mais de 20 linguagens de programação, incluindo Groovy, SQL, HTML, CSS, C ++, JavaScript e PHP.

Possui construtores de código para pesquisar API de linguagem de programação e informações de biblioteca e inserções de código para inserir rapidamente a linguagem de programação comum constrói.

EditRocket oferece suporte a autocompletar código para uma variedade de linguagens, incluindo Java, JavaScript, Python, Ruby e HTML.

EditRocket também apresenta ferramentas de desenvolvimento web, como uma tag HTML e estilo CSS construtor, navegadores de tag e integração com servidores Web locais.

Funciona como um cliente FTP e SFTP, permitindo que você navegue, faça upload, baixe, renomeie e exclua arquivos e diretórios.

EditRocket é um editor usado por desenvolvedores da Web.

Alguns de seus recursos incluem um navegador de sistema de arquivos integrado, validadores de código e um cliente FTP / SFTP.

Richardson EditRocket é um dos mais editores de código-fonte completos disponíveis, mas também exigem compra.

Portanto, se você precisar de um editor de código-fonte gratuito, procure Notepad ++ ou TextWrangler em vez disso.
