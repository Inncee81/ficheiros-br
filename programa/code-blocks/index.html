Visão geral do software

Principais características

Orientado para as linguagens C, C ++ e Fortran
Funcionalidade altamente extensível
Suporte para vários compiladores, como Microsoft Visual C
Recursos úteis de depuração, como troca de thread e despejo de memória personalizado

Code Blocks é um C, C ++ e Fortran IDE.

Foi desenvolvido para oferecer extensibilidade por meio da instalação ou codificação de plug-ins.

Code Blocks é escrito em C ++ e suporta C, C ++ e linguagens Fortran fora da caixa.

É altamente extensível por meio de uma quantidade infinita de plug-ins que fornecem suporte e funcionalidade de linguagem de programa adicional.

Fornece suporte para vários compiladores, como o GCC , Microsoft Visual C ++, Digital Mars e muito mais.

Os Code Blocks também apresentam um sistema de criação personalizado rápido que armazena seus arquivos de projeto baseados em XML com a extensão .CBP .

O depurador do aplicativo tem suporte completo para ponto de interrupção, incluindo pontos de interrupção de código e dados.

O aplicativo também exibe argumentos e símbolos de função local e fornece relógios definidos pelo usuário e despejo de memória personalizado para depuração de programa.

Além desses elementos, o Code Blocks apresenta uma interface com guias, destaque de sintaxe e autocompletar código para ajudá-lo a trabalhar com mais eficiência.

Code Blocks é um IDE competente que pode ser estendido por meio de plug-ins para fornecem funcionalidade adicional.

É um aplicativo leve que oferece recursos úteis de compilação, depuração e IU.

Code Blocks é um aplicativo confiável, independentemente da plataforma.
