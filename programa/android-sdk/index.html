O Google Android SDK é um kit de desenvolvimento de software usado para criar aplicativos para o sistema operacional Android.

Ele trabalha com uma rede de desenvolvimento integrada (IDE) e pode ser baixado como um pacote com muitos IDEs.

Android aplicativos são salvos no formato .APK.

Existem muitas ferramentas diferentes que vêm com o pacote de partida SDK que são úteis e necessárias para o desenvolvimento.

Uma ferramenta importante é o Android, que permite criar, excluir e visualizar Android Dispositivos Virtuais (AVDs).

Também é usado para criar novos projetos de Android e fazer atualizações para eles.

Ao desenvolver o layout do seu aplicativo, você pode usar o Hierarchy Viewer para monitorar elementos de Interface do Usuário e, em seguida, usar o layout para otimizar seus layouts e hierarquias de layout.

Android SDK também possui ferramentas para testar e depurar sua aplicação durante o desenvolvimento.

Inclui um emulador de dispositivos móveis que permite testar aplicativos de Android móveis a partir do seu computador.

Para realizar testes completos de estresse, ele também tem uma ferramenta chamada Monkey que é executada em seu emulador ou dispositivo e automatiza eventos do usuário, incluindo cliques, toques e deslizes, e pode executar eventos em nível de sistema também.

Android SDK também vem com um depurador chamado Dalvik Debug Monitor Server que fornece muitos métodos diferentes para você resolver problemas com o seu aplicativo.

Se você estiver interessado em desenvolver software para dispositivos Android, Android SDK é uma ferramenta muito necessária para ter em seu arsenal.

A instalação do kit de desenvolvimento é muito personalizável, permitindo que você trabalhe com o pacote básico de ossos nus ou com quantas ferramentas extras desejar.
