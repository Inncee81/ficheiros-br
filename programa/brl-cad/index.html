Visão geral do software

Principais características

Suporta vários formatos de arquivo, como NASTRAN, IGES e DXF
Composto por mais de 400 bibliotecas, utilitários e ferramentas
Fornece uma interface de script incorporada
Inclui rastreamento de caminho e mapeamento de fótons para síntese de imagem realista

BRL-CAD é um sistema de modelagem sólida de geometria sólida construtiva (CSG) de plataforma cruzada.

BRL significa Laboratório de Pesquisa Balística, que agora é o Laboratório de Pesquisa do Exército.

O pacote de software está disponível em código-fonte e formato binário como software de código aberto gratuito (FOSS).

O software suporta vários tipos de arquivo, como DXF, IGES, NASTRAN, TANKILL, PDF, VRML e EUCLID .

O BRL-CAD é composto por mais de 400 utilitários, bibliotecas e ferramentas que permitem traçado de raio de alto desempenho para análise geométrica e de renderização e edição de geometria interativa.

Também vem com conjunto de benchmark de análise de desempenho do sistema, ferramentas de processamento de imagem e sinal e uma interface de script incorporada.

BRL-CAD é usado principalmente por US

aplicações militares, industriais e acadêmicas para projetar e analisar parte mecânica s, arquitetura e veículos.

O pacote de software BRL-CAD existe desde 1983, o que proporciona uma vida útil impressionante e ainda é útil até hoje, fornecendo uma opção rápida e estável para modelagem de sólidos.
