Visão geral do software

Principais características

Suporte para uma variedade de tipos de dados, como ESRI 2D e 3D Shapefiles e TIFF
Fluxo de trabalho consolidado que o orienta intuitivamente através das tecnologias geoespaciais integradas
Processamento inteligente para análise de dados
Modelagem dinâmica para processamento em lote multi-core e distribuído

ERDAS IMAGINE é um sistema de criação de dados geoespaciais.

Ele combina sensoriamento remoto, recursos de sistema de informações geográficas (GIS) e processamento e análise de imagens geoespaciais.

O software está incluído com o PRODUCER SUITE do Power Portfolio e está disponível nas edições Essentials, Advantage e Professional.

O aplicativo fornece ferramentas para modelagem espacial, produção de mapas, mosaicos, processamento em lote paralelo e detecção de alterações.

É usado principalmente para criar imagens 2D e 3D, filmes 3D e composições de mapas de qualidade cartográfica a partir de dados geoespaciais.

ERDAS IMAGINE suporta uma grande variedade de formatos de dados, como ESRI 2D e 3D Shapefiles, TIFF , MrSID (Geração 3 - Geo Express), ESRI GRID e GRID Stack.

O software apresenta um visualizador que exibe, combina, analisa e apresenta dados geográficos.

ERD AS IMAGINE também inclui a capacidade de adicionar recursos, como compatibilidade de formato, conjuntos de ferramentas de produção, geração automática de terreno e processamento de radar.

ERDAS IMAGINE é um sistema de criação de dados geoespaciais popular amplamente usado por arqueólogos, engenheiros , biólogos, geólogos, inspetores e cientistas do solo.

É embalado com recursos, como processamento e análise de imagens geoespaciais, recursos de GIS e sensoriamento remoto.

ERDAS IMAGINE é impressionante e escolha profissional para suas necessidades de fotogrametria, GIS e sensoriamento remoto.
