Principais características

Suporta muitos formatos de imagem de disco populares, incluindo ISO, BIN, MDS / MDF e MDX
Converte imagens em MDF / MDS, ISO e MDX
Emula até 32 dispositivos DT e SCSI
Cria 16 dispositivos HDD virtuais
Fornece uma interface de usuário personalizável

DT Soft DAEMON Tools é um utilitário Windows e OS X usado para emular unidades virtuais e criar imagens de disco de CD e DVD.

O software está disponível em diferentes edições, como Ultra, Lite, Pro Standard, Pro Advanced, Net e Tools for Mac.

A edição Pro Advanced do aplicativo oferece todos os recursos do DAEMON Tools (DT).

Você pode montar uma variedade de formatos de imagem de disco, como MDX (formato proprietário), ISO, BIN, ISZ e MDS / MDF, para uma unidade virtual, junto com arquivos VMDK, VHD, ZIP e TrueCrypt.

Você também pode converter imagens dos formatos suportados para os formatos MDX, ISO ou MDF / MDS e você pode fazer discos CD, DVD e Blu-ray de imagens MDF / MDS, ISO ou MDX.

.

O programa permite a você criar imagens de disco compactadas ou dividir uma imagem em vários arquivos.

Se você tiver dados importantes que precisam de informações avançadas Curity, o aplicativo permite que você crie imagens TrueCrypt protegidas por senha.

No que diz respeito à virtualização, o software permite que você crie até 16 dispositivos HDD virtuais, emule até 32 dispositivos DT e SCSI e use 4 dispositivos IDE virtuais.

DAEMON Tools é um utilitário usado por usuários iniciantes e avançados.

Dependendo da edição que você está usando, o software fornece ferramentas avançadas para emulação de unidade virtual e gerenciamento de disco formatos de imagem.

Para usuários Windows, o DT Soft DAEMON Tools oferece um grande conjunto de ferramentas, mas para usuários do OS X, os recursos não estão nem perto de serem comparáveis ​​e eles devem procurar em outro lugar.
