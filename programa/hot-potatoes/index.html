O Half-Baked Hot Potatoes é um conjunto de aplicativos usado para criar questionários interativos.

Foi originalmente criado para compra, mas foi lançado como freeware em 2009 com a versão 6.3, quando o suporte técnico foi encerrado.

Ele está disponível para Windows e será executado no OS X e Linux por meio da Java Virtual Machine.

Hot Potatoes consiste em seis aplicativos que permitem que você crie diferentes tipos de exercícios interativos:

  • JCloze - Cria exercícios preenchidos por lacunas
  • JQuiz - Cria questionários baseados em perguntas
  • JCross - Cria quebra-cabeças de palavras cruzadas
  • JMix - Cria exercícios de frases misturadas
  • JMatch - Cria exercícios de correspondência ou ordenação
  • Masher - Cria uma unidade completa de vários exercícios

O software permite que você insira imagens , links, tabelas HTML e objetos de mídia em seus exercícios para exercícios mais dinâmicos do que apenas texto.

Depois de terminar de criar seu exercício, o Hot Potatoes permite exportá-lo como um .HTM arquivo para compartilhamento na web, como um pacote .ZIP para compartilhamento com outras pessoas e para impressão para criar uma cópia física do exercício.

Hot Potatoes é útil para qualquer tipo de usuário, especialmente professores, que procuram criar exercícios interativos gratuitamente.

O software permite que você crie palavras cruzadas, exercícios de correspondência, perguntas para preencher as lacunas, questionários com várias respostas e muito mais.

E embora não tenha mais suporte técnico, é simples de usar, o que torna o Half-Baked Hot Potatoes uma opção razoável para a criação de exercícios interativos.
