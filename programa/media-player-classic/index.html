Visão geral do software

Principais características

Parece semelhante a Windows Media Player 6.4
Oferece reprodução de VCD, SVCD e DVD sem software ou codecs adicionais
Suporta legendas AVI
Interface personalizável, como a barra de ferramentas

O Media Player Classic é um programa de código aberto que se parece com o Windows Media Player 6.4.

O principal fork do Media Player Classic é o Home Cinema, que oferece opções e recursos adicionais.

.

O player fornece reprodução de SVCD, VCD e DVD prontos para uso.

Ele suporta muitos formatos populares de vídeo e áudio, como H.264, MPEG-4 ASP, OGG e FLV.

O programa oferece recursos como a capacidade de remover tearing e suporte para renderização aprimorada de vídeo.

O Media Player Classic é personalizável, o que permite configurar diferentes aspectos como a barra de ferramentas de reprodução.

Além disso, o player é leve e usa pouca memória, o que permite usá-lo em computadores mais antigos.

O Media Player Classic é um aplicativo para usuários que desejam aparência do Windows Media Player 6.4.

No entanto, além dos propósitos nostálgicos, não há muitos motivos para escolher este aplicativo em vez de outros Windows media players.
