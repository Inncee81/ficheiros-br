Visão geral do software

Principais características

Recursos padrão de processamento de palavras, como formatação de texto
Suporte para arquivos populares de produtividade de escritório, incluindo DOCX
Colabore no mesmo documento ao mesmo tempo com outros através de AbiCollab.net

AbiSource AbiWord é um processador de texto, semelhante ao Microsoft Word, que está disponível para Windows e Linux.

O programa permite criar documentos e colaborar com outros documentos ao mesmo tempo.

AbiWord permite que você execute muitas das funções padrão de processamento de palavras, como formatação de texto e verificação ortomóbico.

O programa tem um layout de fita semelhante a outros processadores de texto, com muitas das opções de formatação frequentemente usadas prontamente disponíveis.

O aplicativo permite salvar documentos em um formato AbiWord proprietário com a extensão .ABW.

Também suporta formatos populares, como .DOC, .DOCX, .RTF, .TXT, .ODT e .HTML.

O processador de texto facilita os projetos da equipe também.

Ele fornece um recurso de colaboração útil que permite que você trabalhe no mesmo documento que outra pessoa ao mesmo tempo através de AbiCollab.net.

Esta aplicação é uma escolha sólida para suas necessidades de processamento de palavras.

Ele pode lidar com formatos de escritório populares e facilitar a colaboração de documentos.

Pelo preço do grado, é difícil não dar uma chance ao AbiSource AbiWord.
