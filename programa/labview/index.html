Principais características

Usa uma linguagem de programação gráfica para ajudar os usuários a criar sistemas de medição e controle
Fornece projetos de amostra que podem ser personalizados de acordo com suas necessidades
Integra-se com uma infinidade de dispositivos LabVIEW

National Instruments LabVIEW é uma plataforma usada para construir sistemas de medição e controle.

É um ambiente de desenvolvimento completo voltado para cientistas e engenheiros realizando medições.

O software está disponível para Windows, OS X e Linux.

O LabVIEW usa uma linguagem de programação visual, conhecida como "G", para ajudar os usuários a criar visualmente seus sistemas.

Possui recursos de análise, telas de UI para dados científicos e de engenharia e a capacidade de incorporar diferentes abordagens de programação.

O aplicativo também fornece exemplos de projetos que podem ser personalizados de acordo com suas necessidades para que você não precise começar a construir seu sistema do zero.

Existem mais de 10.000 drivers de instrumentos diferentes disponíveis para uso com o LabVIEW, o que permite que você use qualquer dispositivo em seu sistema LabVIEW.

O software pode ser integrado com o Measu de baixo custo rementar e adquirir dispositivos de aquisição de dados, construir sistemas de medição autônomos construídos em uma plataforma de E / S reconfigurável e usar módulos de E / S para construir sistemas de controle embarcados.

O software fornece uma grande quantidade de recursos de desenvolvimento que podem ser adicionados por meio de sua linguagem de programação gráfica.

Além disso, há muitas ferramentas e recursos para aprender o software, se você quiser participar aulas em sua área ou aprender com tutoriais online.

Como cientista ou engenheiro da National Instruments LabVIEW, é uma opção de qualidade para construir seus sistemas de medição e controle.
