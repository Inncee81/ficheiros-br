Visão geral do software

Principais características

Suporta uma variedade de formatos de dados, incluindo ESRI ArcInfo e LAS Well Data
Fornece uma série de recursos de visualização 3D, como Isosurfaces, linhas de fluxo e seu módulo FaceRender
Grava vídeo para que você possa compartilhar as alterações feitas nos mapas com outros usuários

Golden Software é um aplicativo de modelagem e dados de poço 3D.

É um dos principais programas para mapeamento de superfície 3D, visualização de dados geológicos 3D e modelagem de poço 3D.

O aplicativo suporta uma grande variedade de formatos de dados brutos, GIS, geológicos, de poços e de nuvem de pontos, como ESRI ArcInfo, banco de dados de acesso, Office Open XML XLSX e LAS Well Data.

Você pode então visualizar seus dados por usando o aplicativo para gerar modelos 3D e imagens 3D.

Voxler fornece uma série de recursos que compõem um ambiente visual 3D poderoso, mas amigável, incluindo linhas de fluxo, Isosurfaces, renderização de volume e recursos de geração de mapa de contorno .

Quando você estiver pronto para compartilhar seu modelo ou imagem, a Voxler fornece ferramentas de impressão úteis, que incluem uma visualização de impressão que torna a impressão em várias páginas mais precisa.

O aplicativo inclui v Recursos de gravação de vídeo que permitem compartilhar o vídeo das alterações feitas nos mapas com outros usuários.

Você também pode exportar seu modelo ou imagem para formatos populares, como PDF, JPEG, ESRI Float Grid, BMP, GIF e PNG.

Voxler é um programa popular usado em uma variedade de ambientes profissionais de geólogos que estudam dados sísmicos 3D a meteorologistas compilando estudos atmosféricos.

Ele pode importar muitos formatos de dados diferentes e fornece as ferramentas necessárias para visualizar dados com modelos e imagens 3D.

Golden Software Voxler é uma ótima escolha para profissionais que precisam visualizar seus dados brutos, de sondagem, GIS e geológicos.
