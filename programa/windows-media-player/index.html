Visão geral do software

Principais características

Suporta formatos populares de vídeo e áudio, como WMV e MPEG
Permite que você personalize a organização de sua mídia
Ativa o streaming entre diferentes computadores e dispositivos na mesma rede
Rasga e grava áudio em CDs

Microsoft Windows Media Player é um aplicativo de biblioteca de mídia usado para reproduzir vídeo e áudio e para visualizar imagens.

O aplicativo vem com o sistema operacional Windows.

O Windows Media Player oferece suporte a vários formatos populares de áudio e vídeo, como WMV, MOV, MPEG, WAV, ASF, MIDI e AIFF.

O player também oferece suporte a H.264, Xvid e vídeo DivX.

Há uma grande quantidade de plug-ins gratuitos disponíveis que estendem as capacidades do Windows Media Player, também.

No Windows Media Player, você também pode transmitir mídia de ou para outro dispositivos em sua rede, sincronize sua mídia com dispositivos portáteis e grave seu áudio em um CD.

Windows O Media Player permite que você personalize sua experiência de mídia.

Ele permite que você configure o painel de navegação, classifique sua mídia por categorias, como gênero e proporção ng e edite a capa do player.

Windows O Media Player também permite que você crie suas próprias listas de reprodução de mídia e pode gerar listas de reprodução com base em seus critérios.

Se você for um usuário do Windows, Windows Media Player é uma boa opção para organizar e reproduzir sua mídia.

Suporta muitos formatos populares de áudio e vídeo, permite personalizar a organização de sua mídia e permite gravar seu áudio para um CD.

O Microsoft Windows Media Player é uma solução confiável para suas necessidades de reprodução de mídia.
