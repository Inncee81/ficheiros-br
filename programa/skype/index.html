Principais características

Converse por vídeo com pessoas em qualquer lugar da Internet
Utilize o compartilhamento de tela para colaborar com outras pessoas
Ligue para números de telefone com uma assinatura ou usando créditos
Envie comunicação de texto por meio da interface de chat integrada

Skype é um programa de telecomunicações usado principalmente para bate-papo com vídeo.

Foi originalmente desenvolvido para computadores, mas agora está disponível para smartphones e tablets conectados à Internet.

O o aplicativo é gratuito, mas os créditos podem ser adquiridos para fazer chamadas para telefones fixos ou celulares.

O Skype tem muitos recursos que ajudam você a se conectar com outras pessoas.

O bate-papo por vídeo é o recurso mais popular do programa mas também pode ser usado para chamadas de voz e bate-papo de texto.

Também oferece a capacidade de transferir arquivos facilmente pela Internet para outro usuário no Skype.

Se você estiver procurando por um aplicativo para se comunicar com outras pessoas, o Skype é uma boa opção.

Se você deseja um chat de vídeo ou simplesmente um chat de voz, você tem tudo para você.
