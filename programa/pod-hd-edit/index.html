Principais características

Crie seu próprio amplificador, pré-amplificador e predefinições de efeitos para usar na sua guitarra
Editar predefinições de efeitos de guitarra já existentes
Gerencie suas predefinições facilmente a partir de uma interface de usuário simples, mas atraente

Line 6 POD HD Edit é um aplicativo usado com o modelador de amplificador de guitarra POD HD para editar presets de efeitos de guitarra.

Está disponível para download gratuito no site da Line 6.

O POD HD Edit permite que você organize, edite e crie facilmente presets para parâmetros de modelo de amplificador, pré-amplificador e FX.

Ao criar ou editar um preset, você tem várias opções, como mixar e combinar modelos FX, ajustando a reverberação e tom e selecionando diferentes amplificadores.

Depois de terminar, você pode salvar seu preset no computador e abri-lo com o software Line 6. POD.

Se você tiver um POD HD, obter o POD HD Edit é uma ótima ideia.

É gratuito e adiciona uma nova profundidade para tocar guitarra em seu POD HD, tornando as possibilidades praticamente infinitas.
