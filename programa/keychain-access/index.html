Principais características

Recuperar ou remover senhas

Apple Keychain Access é um sistema de gerenciamento de senha empacotado com macOS (OS X).

O utilitário foi introduzido em Mac OS 8.6 e é usado para armazenar senhas com segurança.

Keychain O Access permite que você acesse o recurso keychain e modifique seu conteúdo, que inclui senhas para sites, e-mail, servidores, redes Wi-Fi, compartilhamentos de rede e imagens de disco criptografadas.

O programa também pode armazenar chaves privadas, certificados e notas seguras.

Algumas outras funções que o Keychain Access fornece é a capacidade de recuperar senhas esquecidas, remover senhas de sites indesejadas e compartilhar seu keychain com outras máquinas macOS usando o .KEYCHAIN ​​ arquivo.

Sempre que você salva uma senha, ela é armazenada nas chaves do Mac, o que torna o Keychain Access um programa importante.

O utilitário permite que você configure as chaves e executa uma variedade de outras funções, como recuperar ou remover senhas.

O Apple Keychain Access é uma ferramenta útil para todos os usuários do macOS.
