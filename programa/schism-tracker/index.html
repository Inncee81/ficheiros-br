Schism Tracker é um programa gratuito usado para criar música.

O programa é uma implementação do Impulse Tracker e está disponível para Windows, OS X e Linux.

O aplicativo parece e funciona de forma semelhante ao programa Impulse Tracker.

Schism Tracker é baseado em uma versão modificada do mecanismo Modplug, que permite que você toque uma variedade de formatos de música do Tracker e fornece melhor reprodução do Impulse Tracker ( .IT ) arquivos.

Você também pode usar o programa para reproduzir e editar outros arquivos de música do Tracker, como .XM , .S3M e .MOD.

Schism Tracker é útil para reproduzir e editar músicas do Tracker.

É projetado para usuários que não podem comprar equipamentos caros e , infelizmente, parece isso.

Embora alguns usuários possam achar que é útil ferramenta para criar música, existem muitas outras opções melhores disponíveis.
