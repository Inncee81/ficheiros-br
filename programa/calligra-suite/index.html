Principais características

Calligra Suite é uma suíte de escritório gratuita, semelhante ao Microsoft Office, que permite criar vários tipos de documentos.

A suíte foi desenvolvida pelo KDE e bifurcada do KOffice em 2010.

Está disponível para instalação em Windows e OS X, mas é desenvolvido principalmente para o ambiente Linux.

Calligra Suite consiste nos seguintes aplicativos:

  • Braindump - uma ferramenta para organização de ideias e facilitação de sessões de brainstorming
  • Fluxo - Um aplicativo de diagramação e fluxograma
  • Karbon - Um aplicativo de desenho vetorial
  • Kexi - Um criador de aplicativos de banco de dados visual 2738934723 <775269344723 Krita - Um aplicativo de desenho e pintura
  • Plano - Um aplicativo de gerenciamento de projeto
  • Estágio - Um criador de apresentação
  • Planilhas - Uma ferramenta de cálculo e planilha
  • Palavras - Um processador de texto
  • Autor - Um aplicativo de criação de e-book

Os aplicativos usam o formato OpenDocument como o formato de arquivo padrão, mas suportam outros formatos relacionados ao escritório, como como Microsoft Office Open XML e PDF.

.

Junto com o LibreOffice, Calligra Suite é um dos pacotes de escritório mais populares para usuários Linux.

Ele fornece um conjunto robusto de aplicativos que irão ajudá-lo com quase todas as necessidades relacionadas ao escritório e oferece suporte a uma variedade de formatos de arquivo de produtividade de escritório.

O Calligra Suite é uma boa escolha para Windows e usuários do OS X e uma ótima escolha para Linux usuários.
