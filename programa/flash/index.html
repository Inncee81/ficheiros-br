Visão geral do software

Principais características

Ambiente de criação líder do setor para conteúdo rico da Internet
Animação baseada em objetos com caminhos de movimento
Linguagem de script ActionScript
Tradução 3D e ferramentas de rotação
Suporte para muitos formatos de arquivo de imagem, vídeo e áudio
Capacidades de desenho integradas

Adobe Flash é um ambiente de autoria para a criação de conteúdo multimídia e interativo.

Animações em Flash são comumente desenvolvidas para sites e podem ser usadas para anúncios, menus de navegação, interfaces de reprodução de vídeo ou jogos online.

O Flash oferece suporte a gráficos vetoriais e raster e pode ser programado usando a linguagem de programação ActionScript.

O Flash tornou-se Adobe Animate em 2016.

O Flash oferece várias visualizações para vídeo, gráficos e fonte edição de código.

A janela do documento principal inclui uma linha do tempo e um editor de movimento, que é usado para criar e editar sequências de animação.

As paletas de ferramentas permitem que os usuários acessem ativos do projeto (por exemplo, vídeo, arquivos de imagem e áudio), modificar objetos e alterar as configurações do projeto.

O Flash fornece ferramentas de desenvolvimento para a criação de código ActionScript personalizado, que pode ser usado para criar inter conteúdo ativo.

O editor de código suporta destaque de sintaxe e ajuda a reduzir o tempo de desenvolvimento com recolhimento de código, tags de comentário e recursos de navegação de erro de código.

O Flash inclui uma interface de usuário visual para edição de script que permite conclusão automática da sintaxe.

Ele também fornece um depurador avançado para percorrer manualmente o código e identificar erros de script.

O Flash oferece aos usuários controle completo sobre o movimento do objeto.

O editor de movimento suporta animação baseada em objeto, que pode aplicar interpolação (quadros de transição interpolados) a objetos em vez de quadros-chave.

O Flash também oferece suporte a splines de Bezier para adicionar animações de objeto baseadas em curva a atributos de objetos individuais e inclui recursos de movimento avançados para animar Objetos 2D por meio do espaço 3D.

Os usuários também podem criar efeitos de animação em cadeia, vinculando objetos em uma série.

As animações também podem ser salvas como arquivos independentes .SWF para reprodução no Flash Player.

Adobe Flash é uma boa ferramenta para desenvolvedores da web que crie anúncios em sites e vídeos online.

Também é útil para produtores de multimídia que criam jogos online e outros aplicativos de Internet ricos em mídia.

Se você se enquadrar em uma dessas categorias, experimente o Adobe Flash.
