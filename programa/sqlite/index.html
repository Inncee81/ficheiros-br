Principais características

Sistema de gerenciamento de banco de dados relacional independente e leve
Facilmente incorporado em programas de software
Suporta propriedades de banco de dados ACID

SQLite é um sistema de gerenciamento de banco de dados relacional leve (RDBMS) que é frequentemente incorporado em aplicativos de software por causa de sua simplicidade e pegada pequena.

SQLite implementa ACID (Atomicidade, Consistência, Isolamento, Durabilidade), tornando o mecanismo de banco de dados robusto e totalmente transacional também.

Uma das principais diferenças entre o SQLite e outros mecanismos de banco de dados é que ele é independente.

Isso significa que requer pouco ou nenhum suporte de bibliotecas externas ou o sistema operacional a ser executado.

O banco de dados é escrito em ANSI C e pode ser compilado por qualquer compilador C, tornando-o adequado para uma variedade de aplicativos e tipos de sistema de destino.

SQLite é um dos sistemas de banco de dados mais populares e amplamente usados ​​e está incluído na maioria dos dispositivos móveis.

Se você é um desenvolvedor que precisa de uma solução simples, mas eficaz para um RDBMS, o SQLite é um ob vious escolha.
