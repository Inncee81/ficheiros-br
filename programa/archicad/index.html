GRAPHISOFT ArchiCAD é um programa de CAD arquitetônico para Windows e OS X.

É uma das principais ferramentas de modelagem de informações de construção (BIM) disponíveis.

O ArchiCAD pode importar e exportar uma variedade de formatos CAD, como DWG, IFC e DXF, juntamente com seus próprios formatos proprietários.

O aplicativo fornece muitas ferramentas para projetar estruturas, incluindo ferramentas de desenho 2D, uma interface de modelagem 3D, recursos de renderização arquitetônica e uma ferramenta de gerenciamento de documentos com um servidor de armazenamento de dados.

O ArchiCAD também permite adicionar funcionalidade ao aplicativo através de extensões comerciais e gratuitas.

O aplicativo fornece recursos úteis de colaboração para os membros da equipe trabalharem juntos.

O BIM Server da GRAPHISOFT é um ambiente de colaboração em tempo real e o BIMcloud, que permite que os fluxos de trabalho BIM sejam acessíveis em qualquer configuração de software, hardware ou rede.

.

O ArchiCAD é usado principalmente para projetar e projetar edifícios, áreas urbanas, interiores e outros ambientes estruturais.

Ele oferece uma grande quantidade de recursos para BIM, visualizações 3D, gerenciamento de projetos, trabalho com equipes e integração de engenharia.

GRAPHISOFT ArchiCAD é uma ótima ferramenta BIM CAD que os arquitetos acharão útil.
