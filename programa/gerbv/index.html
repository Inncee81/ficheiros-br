Principais características

Suporta arquivos Gerber RS-274X
Exporta uma imagem de camada para formatos populares como PDF e PNG
Permite que você execute operações de camadas complexas

Gerbv é um programa de código aberto usado para visualizar arquivos Gerber.

Gerber está disponível para as plataformas Windows e Linux e como parte do conjunto de aplicativos gEDA.

O aplicativo permite que você visualize arquivos Gerber RS-274X, bem como arquivos de perfuração Excellon e arquivos CSV.

Gerbv também pode exportar uma imagem de camada para formatos populares, como PDF, PNG e SVG.

Junto com seu suporte de arquivo, o programa permite que você execute operações de camada complexas, defina cores de camada, arraste e solte camadas para alterar sua ordem e alterar a polaridade da camada.

O aplicativo é usado principalmente para visualizar Gerber arquivos no formato RS-274X, que são enviados aos fabricantes de placas de circuito impresso (PCB).

Gerbv é confiável e permite que os usuários executem uma variedade de ações, o que o torna uma boa escolha para trabalhar com arquivos Gerber .
