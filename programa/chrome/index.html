Principais características

Desfrute de navegação na web de alta velocidade em uma interface de usuário minimalista e fácil de usar

O Google Chrome é o navegador da web mais usado no mundo.

Ele começou como um projeto de código aberto chamado Chromium e ainda é baseado na construção original.

O navegador era originalmente desenvolvido para computadores desktop, mas agora tem versões móveis para Android e iOS.

O Chrome é um navegador muito eficiente com muitos recursos para melhorar a conveniência do usuário na web.

Ele deixa seus concorrentes trabalhando em muitos testes de desempenho diferentes e também tem a classificação de segurança mais alta dos navegadores convencionais.

O Adobe Flash vem empacotado com o Chrome quando você faz o download e também é atualizado automaticamente com o navegador para sua conveniência.

O Chrome também tem tradução automática de páginas da web para que você possa verificar sites criados em diferentes idiomas.

A única desvantagem do Google Chrome é que ele rastreia algumas coisas s que você faz enquanto o usa.

Muitos dos dados que o Google coleta não são intrusivos, como sucesso na instalação do Chrome, local de download, dados de falha e frequência de uso.

Eles também coletam tudo o que você digita na barra de endereço antes mesmo de clicar em Enter, o que pode ser considerado mais intrusivo.

Se você não gostar dessas informações coletadas, pode desativar a maioria delas nas configurações do Chrome.

O Google Chrome é um navegador da web muito estável e eficiente que derrotou os navegadores concorrentes em muitos testes de benchmark diferentes.

Se você está procurando um navegador que o leve onde você precisa ir na Internet rapidamente e com eficiência, não procure mais.

Se você não se sentir confortável com o rastreamento de usuários do Google, tudo que você precisa fazer é desativar esses recursos.

NOTA: O Chrome se atualiza automaticamente para garantir que você sempre ays tem a versão mais recente.

Não há configuração no programa para desativar as atualizações automáticas.
