Principais características

Editor de mapas WYSIWYG
Ferramentas que permitem colocar nós, unir estradas e definir propriedades de estradas ou objetos
Exportação de dados do mapa para .OSM, .PDF, .GPX e .SVG
Capacidade de upload para contribuir com dados de mapa para o projeto colaborativo OpenStreetMap

Merkaartor é um editor OpenStreetMap (OSM) gratuito disponível para Windows, Mac OS X e Linux.

O programa permite que você baixe dados de mapa, edite seu conteúdo e então carregue os dados do mapa para sua conta OSM para compartilhar com outros usuários.

O aplicativo permite que você baixe áreas de mapa personalizadas e altere-as em um editor WYSIWYG.

A interface Merkaartor consiste em widgets, como como encaixe de camadas, encaixe de recursos e encaixe de estilos, que estão prontamente disponíveis para seu uso.

No programa, você pode colocar nós em seu mapa, unir ou separar estradas, definir propriedades de objeto, criar áreas de interação e aplicar estilos ao seu mapa.

.

Você pode salvar todas as alterações feitas em seu projeto de mapa como um documento Merkaartor com a extensão de arquivo .MDC .

Você também pode exportar seus dados de mapa para .OSM, .GPX , .KML , .PDF e .SVG

Merkaartor permite que você carregue seu mapa para sua conta OSM, permitindo que você contribua com correções e atualizações para o projeto colaborativo do OpenStreetMap.

O editor de mapas apresenta ferramentas úteis e widgets, ou docas, que fazem edição e navegação conveniente e eficiente.

O aplicativo também permite exportar dados de mapas para formatos de arquivo populares, como PDF.

Merkaartor é uma ferramenta de qualidade para quem deseja contribuir com o projeto colaborativo OpenStreetMap.
