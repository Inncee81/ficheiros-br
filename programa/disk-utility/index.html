Apple Disk Utility é uma ferramenta do sistema incluída no Mac OS X.

É usado pelo sistema para realizar várias tarefas em discos e volumes de disco, mas também pode ser acessado diretamente com o comando linha.

Sempre que você executa qualquer tipo de tarefa que acessa qualquer forma de disco em um Mac, o sistema usa o Utilitário de Disco.

Isso inclui tudo, desde a montagem de um volume de disco ou imagem até verificar a integridade de um disco e formatar um disco.

Você pode acessar o utilitário diretamente com a linha de comando usando os comandos hdiutil e diskutil.

Se você usa um computador baseado em OS X, você é mais do que provavelmente já usando o Utilitário de Disco.

Os usuários mais avançados que se sentem confortáveis ​​com uma interface de linha de comando podem estar interessados ​​em interagir com o utilitário diretamente usando o terminal.
