Principais características

Apresenta um navegador da web, cliente de e-mail, feeds de notícias, cliente de bate-papo e ferramentas de desenvolvedor da web
Fornece capacidade adicional para estender a funcionalidade do navegador e e-mail
Permite que você edite HTML com um editor WYSIWYG e inspecione a estrutura do documento da página da web

Mozilla SeaMonkey é um pacote de aplicativos de Internet de código aberto que inclui um navegador da web, cliente de e-mail, cliente de feed de notícias, cliente IRC Chat e editor HTML.

Está disponível para Windows, Mac OS Plataformas X e Linux e é altamente extensível.

O navegador SeaMonkey é baseado no mesmo código-fonte Mozilla em que o Firefox é construído, com o mesmo mecanismo de renderização.

A web o navegador possui navegação por guias, a capacidade de restaurar sessões e um bloqueador de pop-up.

O SeaMonkey também fornece gerenciamento de complementos, que permite instalar funções adicionais criadas por outros usuários, como temas do navegador.

O cliente de e-mail SeaMonkey suporta várias contas, filtros de mensagens, um dicionário, catálogo de endereços e add-ons para funcionalidade estendida.

O pacote também oferece um cliente de bate-papo personalizável e feeds para notícias e blogs.

SeaMonkey inclui ferramentas de desenvolvimento web como Composer, um editor HTML WYSIWYG com suporte para camadas posicionadas, tabela dinâmica e redimensionamento de imagem e suporte CSS.

O pacote também vem com um Inspetor DOM que permite para examinar a estrutura da página da web e um depurador de JavaScript, que ajuda a depurar o código JavaScript.

Se você é um desenvolvedor da web em busca de um pacote de Internet inclusivo, o SeaMonkey é uma ferramenta útil.

Com sua flexibilidade de código aberto permite que você personalize sua experiência de navegação, envio de e-mail, bate-papo e desenvolvimento de páginas da Web.

No entanto, se você está apenas procurando um navegador autônomo, o Chrome ou FireFox é o caminho a percorrer .
