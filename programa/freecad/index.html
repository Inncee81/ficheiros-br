O FreeCAD é um modelador 3D paramétrico usado para projetar objetos da vida real.

O programa remonta a 2001 e é mantido e desenvolvido por uma comunidade de desenvolvedores.

O programa suporta um grande quantidade de formatos padrão da indústria, como IGES, STEP, OBJ, DXF, SVG, STL e DAE.

O FreeCAD oferece vários módulos, incluindo um módulo de simulação de robô para estudar movimentos robóticos, um módulo de folhas de desenho para colocar 2D visualizações de um modelo 3D em uma folha e um módulo de renderização para exportar objetos 3D para renderização com renderizadores externos.

O programa também apresenta Sketcher, uma ferramenta que utiliza componentes 2D para esboçar formas 2D e criar desenhos de produção 2D a partir de um modelo 3D.

O FreeCAD é usado principalmente por engenheiros mecânicos, arquitetos e designers de produto.

Suporta muitos dos formatos mais proeminentes da indústria e oferece qualidade et de ferramentas.

FreeCAD é um aplicativo gratuito poderoso que será útil para usuários de CAD, especialmente aqueles na área de engenharia mecânica.
