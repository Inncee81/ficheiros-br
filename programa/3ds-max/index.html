Autodesk 3ds Max é um programa de modelagem gráfica 3D usado para modelagem, animação, simulação e renderização de gráficos 3D.

É mais comumente usado para criar modelos 3D para videogames, mas também é usado em filmes e gráficos de movimento.

3ds Max fornece uma extensa seleção de ferramentas para criar modelos 3D.

Oferece muitos métodos de desenvolvimento de texturas criativas, incluindo revestimento, rotação, desfoque, alongamento UV e relaxamento.

Também tem recursos para povoar automaticamente multidões de personagens que se comportam de forma realista.

Existem também muitas ferramentas que você pode usar para criar animações sofisticadas em 3ds Max.

Character Studio permite personalizar a forma como seus personagens biped se moverão ajustando as configurações do esqueleto.

3ds Max também possui um mecanismo integrado de simulação de pano que lida com o comportamento do tecido em sua cena.

Autodesk 3ds Max é um poderoso programa de modelagem 3D adequado para uma ampla gama de projetos.

Junto com Maya (também pela Autodesk), é o padrão da indústria para software de modelagem 3D.

Se você está interessado na indústria de design 3D, você não pode errar com o 3ds Max.
