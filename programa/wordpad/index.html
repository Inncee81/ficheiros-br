Visão geral do software

Principais características

Fornece uma interface de faixa de opções semelhante ao Microsoft Word
Permite formatação avançada e inserção de gráficos
Suporta o formato Office Open XML

Microsoft WordPad é um editor de texto fornecido com o Microsoft Windows.

É semelhante ao Microsoft Word , mas não oferece tantos recursos.

No entanto, ele oferece muito mais recursos do que o Microsoft Notepad , que é outro editor de texto junto com o Windows.

O WordPad permite criar e editar documentos de texto.

O programa permite a formatação avançada, onde você pode personalizar como o texto é organizado, escolher fontes diferentes, definir espaçamento entre linhas e muito mais.

O editor de texto também permite vincular ou incorporar objetos, como imagens ou desenhos do Microsoft Paint em o documento.

Ele oferece vários recursos convenientes, como a capacidade de enviar rapidamente seu documento por e-mail.

Outro recurso útil é que o WordPad oferece suporte ao Office Open XML .DOCX formato .

O Microsoft WordPad é ideal para trabalhar com documentos que não exigem uso intenso de gráficos, como anotações.

É mais poderoso do que o Bloco de notas, mas não chega nem perto do Microsoft Word.

Apesar de sua falta de poder, o editor de texto ainda será útil para usuários do Microsoft Windows.
