Microsoft XML Notepad é um aplicativo de código aberto que permite aos usuários navegar e editar documentos XML.

Está disponível para download gratuito apenas para Windows.

XML Notepad apresenta um interface simples junto com uma variedade de recursos úteis.

O programa fornece uma visualização em árvore que divide um documento XML em classes, tags e valores-chave.

Ele também vem com um visualizador de HTML para exibindo resultados de transformação XSLT.

Alguns outros recursos incluem suporte para arrastar e soltar, fontes e cores configuráveis, validação de esquema XML instantânea e operações infinitas de desfazer e refazer editar.

Bloco de notas XML é usado por iniciantes e programadores XML especializados.

Ele fornece recursos avançados com uma interface simples que não sobrecarrega os iniciantes.

Existem editores de XML que fornecem mais recursos, como o Liquid XML Studio, Stylus Studio e EditiX XML Editor, mas o Microsoft XML Notepad ainda é um programa muito capaz de navegar e editar documentos XML.
