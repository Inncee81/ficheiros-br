Principais características

Visualize os links, documentos, imagens e contatos contidos em uma página da web
Exporte endereços de e-mail, feeds RSS, links e tabelas de dados para os formatos HTML, SQL e CSV
Organize dados em tabelas e listas
Configure funções automatizadas para economizar tempo

Outwit Hub é um programa usado para explorar a web e extrair e organizar dados de fontes online.

Está disponível como um aplicativo independente e como um add-on para o navegador Mozilla Firefox.

Outwit Hub fornece recursos para explorar a web e coletar dados.

Você pode digitar a URL de qualquer site e Outwit Hub exibe a página na janela do aplicativo junto com as especificações da página, como o número de links na página e tempo de carregamento.

O aplicativo apresenta uma barra lateral esquerda que permite que você visualize cada link, imagem, documento e outros dados que aparecem na página.

Você pode até extrair links, tabelas de dados, notícias RSS, endereços de e-mail e muito mais da página e salvar os dados nos formatos CSV, Excel, SQL e HTML.

Você também pode extrair imagens e salvá-las em seu computador.

.

Outwit Hub fornece funcionalidade automatizada que pode ajudá-lo a economizar tempo.

Você pode especificar como os dados não estruturados devem ser extraídos de uma página, definir tarefas e configurar macros.

Outwit Hub é projetado principalmente para usuários que desejam coletar dados da web sem conhecimento de programação.

É direto e fácil de usar, no entanto, não possui a funcionalidade de raspagem avançada que alguns usuários experientes podem estar procurando.

Mas isso não deve impedi-lo de experimentar o Outwit Hub, porque ele ainda fornece uma funcionalidade valiosa por um preço baixo, quer você esteja usando-o como um aplicativo independente ou um complemento do Firefox.
