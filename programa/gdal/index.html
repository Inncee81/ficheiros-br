Principais características

A licença de código aberto permite modificação gratuita e redistribuição do código-fonte
Disponível como código-fonte ou executáveis ​​desenvolvidos por terceiros
Suporte para formatos de dados geoespaciais rasterizados populares

GDAL (Geospatial Data Abstraction Library) é uma biblioteca de tradução de código aberto para formatos de dados geoespaciais raster.

Está disponível apenas como código-fonte para download, mas vários esforços diferentes fornecem executáveis ​​prontos para uso para download .

A biblioteca GDAL suporta muitos formatos de dados geoespaciais raster populares, como NASA ELAS, ERMapper, Intergraph Raster, SGI Image, NITF e Idrisi Raster.

A licença de código aberto GDAL permite que você para fazer o que quiser com o código-fonte, incluindo modificar e redistribuir o código como se fosse seu.

GDAL é usado por muitos programas geoespaciais, como ArcGIS, Geoconcept, Vectorworks e Google Earth, para ler e escrever Formatos GIS.

Esses programas usam os diferentes dados formatados para criar mapas, gráficos e modelos 3D da superfície da Terra.

GDAL é uma biblioteca robusta que permite modificar e redistribuir te seu código-fonte como desejar.

Você também pode baixar executáveis ​​prontos para uso de GDAL fornecidos por desenvolvedores de terceiros em vez de compilar diretamente do código-fonte.

Se você usou uma ferramenta GIS antes, provavelmente você já experimentou os recursos úteis da biblioteca GDAL.
