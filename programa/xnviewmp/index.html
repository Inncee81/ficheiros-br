Visão geral do software

Principais características

Suporta mais de 500 formatos de imagem
Exporta imagens para mais de 70 formatos diferentes
Fornece um navegador de imagens semelhante à ferramenta exploradora de arquivos do seu sistema operacional

XnSoft XnViewMP é um aplicativo gratuito e versátil que pode abrir, editar e converter mais de 500 formatos de arquivo de imagem diferentes.

O programa é gratuito para usuários particulares ou educacionais, mas requer uma licença para uso comercial.

XnViewMP suporta muitos formatos de imagem padrão, bem como muitos formatos proprietários e obscuros.

Além de imagens, ele também suporta um número limitado de formatos de multimídia, incluindo Windows Mídia ( .WMV ) e MPEG.

O software oferece uma interface de navegação intuitiva Windows semelhante ao Explorer e se integra ao menu de contexto do sistema operacional, facilitando a abertura e visualização de arquivos.

XnViewMP permite que você visualize imagens de várias maneiras.

Por exemplo, o programa oferece modos de visualização em miniatura, tela inteira e tira de filme.

Você também pode editar suas imagens usando funções su corte, contraste, modificação de cor, filtros e efeitos.

O programa também fornece uma função de captura de tela para tirar capturas de tela de sua área de trabalho.

XnSoft XnViewMP é um programa multimídia muito útil para visualizar e converter arquivos de imagem.

Com sua interface de visualização de arquivo, também pode funcionar como uma ferramenta de navegação e exploração de arquivo.

Se você trabalha com imagens, especialmente imagens que têm imagens mais obscuras formatos, XnViewMP é uma ferramenta útil para manter por perto.
