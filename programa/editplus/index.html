Principais características

Permite que você edite arquivos de texto, HTML, PHP e Java
Suporta destaque de sintaxe de CSS, PHP, C / C ++, Perl, HTML e Python
Permite que você visualize páginas HTML como um navegador da web
Fornece uma barra de ferramentas HTML para inserção rápida de tags HTML

ES-Computing EditPlus é um editor de texto para Windows.

Você pode baixá-lo como uma versão de avaliação, mas quando o período terminar, você deve comprar uma licença.

EditPlus funciona como um editor de texto, bem como um editor de PHP, editor de HTML, editor de Java e visualizador Hex.

Suporta realce de sintaxe para uma variedade de linguagens, como HTML, CSS, Java, PHP, C / C ++ , Perl e Python, e pode funcionar como um navegador da web para visualizar páginas HTML.

O EditPlus também fornece alguns outros recursos úteis, como preenchimento automático, dobramento de código, verificação ortográfica, pesquisa e substituição, atalhos de teclado personalizáveis e uma barra de ferramentas HTML para inserção rápida de tags HTML.

EditPlus é um editor de texto sólido que fornece recursos úteis para programadores, como destaque de sintaxe, dobramento de código e uma barra de ferramentas HTML.

Não é tão popular como Notepad ++ ou Sublime Text, que são gratuitos e, mas se você estiver disposto a pagar, o ES-Computing EditPlus provará ser um assistente valioso para editar o código-fonte.
