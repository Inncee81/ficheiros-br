MAME OS X é uma porta OS X do emulador MAME (Multiple Arcade Machine Emulator).

É usado para emular jogos de arcade antigos, como Pacman, NBA Jam, Street Fighter, X-Men e 1942.

Para jogar com o MAME OS X, você precisa dos ROMs dos jogos de arcade originais.

O MAME OS X também emula o hardware do jogo, incluindo o microprocessador, chips de som e gráficos.

O MAME OS X oferece uma variedade de recursos úteis, como a capacidade de salvar seus jogos favoritos, ativar efeitos visuais para acomodar a jogabilidade e uma interface simples e limpa para jogar jogos .

MAME OS X é usado principalmente por pessoas que desejam jogar jogos clássicos de arcade.

O emulador é simples e funciona, embora tenha falhas e exija que você tenha o jogo de arcade original no para emular a ROM.

MAME OS X é semelhante ar para MacMAME , que é outra porta OS X do MAME.

Além disso, ambos os projetos não foram atualizados desde os anos 2000.

Mas para todas essas falhas não há outras opções melhores para emular jogos de arcade clássicos no OS X, o que significa que você terá que trabalhar com os bugs que vêm com o MAME OS X.
