Principais características

Emula vários milhares de jogos de arcade
Salva o progresso do jogo e replays
Configura mapeamentos de teclado e teclas de atalho para uso em um jogo

MAME é um emulador de máquina de arcade multiplataforma gratuito.

É desenvolvido principalmente para Windows, mas OS X e Linux são suportados por meio do projeto SDLMAME.

o software está disponível como um download binário ou de código-fonte.

MAME significa emulador de máquina de arcade múltiplo.

MAME pode emular vários milhares de jogos de arcade que datam dos anos 1970 até hoje.

O emulador requer a ROM do jogo de arcade original e imagens de dados de disco para reproduzir o jogo.

O MAME vem com vários recursos, incluindo a capacidade de salvar o progresso do jogo, repetições do jogo, layouts de objeto em um jogo e pontuações altas.

MAME é projetado principalmente para preservar os jogos de arcade em vez de realmente jogá-los.

O emulador não se destina a infringir direitos autorais ou patentes e requer que você possua os ROMs e CDs originais , ou discos rígidos das máquinas de arcade.

Se você possui esses elementos, o MAME é uma ótima escolha para emular clássicos de arcade.
