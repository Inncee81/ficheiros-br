Visão geral do software

Principais características

RDBMS de código aberto
Rápido, confiável e escalonável para ambientes cliente-servidor
Grande comunidade de suporte e disponibilidade de ferramentas GUI de terceiros

MySQL é um popular sistema de gerenciamento de banco de dados relacional de código aberto (RDBMS).

É um mecanismo de banco de dados confiável e de alto desempenho e é popularmente usado em servidores da Web e em software de código aberto e comercial.

A popularidade do MySQL aumentou com sua inclusão em pacotes "AMP" (Apache, MySQL, Perl / PHP / Python), como LAMP, WAMP e MAMP.

O MySQL funciona principalmente em cliente-servidor ambientes e é um sistema de banco de dados escalonável e muito rápido.

Também pode ser usado como um banco de dados embutido, mas outras soluções, como SQLite, são preferidas para bancos de dados embutidos porque são menores em área ocupada.

Existem muitos produtos RDBMS de código aberto disponíveis, mas o MySQL claramente tem uma grande vantagem no mercado cliente-servidor devido à sua história como um sistema de banco de dados rápido e confiável.

Se você estiver procurando por criar um aplicativo que usa um back end com dados estruturados, o MySQL deve ser um dos primeiros lugares onde você procura por sua solução de banco de dados.
