Principais características

Execute várias versões dos sistemas Windows, Linux, Android e macOS
Transfira arquivos e copie e cole dados entre Mac e sistemas operacionais VM
Controle os programas do VM OS como aplicativos nativos Mac

O Parallels Desktop para Mac é um programa de virtualização de sistema operacional que permite que Windows e outros sistemas operacionais (SO) sejam executados em um Mac baseado em Intel .

O programa oferece suporte a plug-and -exibir dispositivos USB, incluindo impressoras, unidades flash, câmeras digitais e sistemas de navegação GPS.

Também oferece suporte à transferência de dados entre o macOS e os sistemas operacionais executados no Parallels, como mover arquivos e copiar e colar informações.

Parallels Desktop para Mac é executado como um aplicativo dentro do ambiente de desktop macOS e permite que os usuários instalem uma versão do Windows, macOS ou baixem uma versão do Linux ou Android.

Após a instalação, o macOS e o SO na máquina virtual (VM) são perfeitamente integrados pelo Parallels Desktop para Mac.

Você pode optar por executar o software de virtualização no modo de tela inteira para mergulhar no sistema operacional, cohe modo rence para usar os aplicativos VM OS lado a lado com aplicativos macOS ou modo de modalidade para executar a VM como uma janela no macOS.

O Parallels Desktop para Mac permite compartilhar arquivos e texto copiado entre sistemas operacionais usando a funcionalidade arrastar e soltar.

Você também pode atribuir dispositivos USB plug-and-play a um sistema operacional host ou alternar entre sistemas operacionais.

O software de virtualização oferece suporte a milhares de aplicativos em mais de 60 sistemas operacionais, incluindo Windows 10, 8.1, 8, 7, Vista, XP e 2000.

O Parallels Desktop também suporta a execução de macOS 10.15 Catalina, 10.14 Mojave, 10.13 High Sierra, 10.12 Sierra, 10.11 El Capitan, 10.10 Yosemite, 10.9 Mavericks, 10.8 Mountain Lion e 10.7 Lion como SOs convidados.

Outros SOs suportados incluem Ubuntu Linux, Fedora Linux, CentOS Linux SUSE, Red Hat, Debian, Fedora Core, Mandriva, FreeBSD, Debian GNU / Linux, Mint Linux, Solaris, openBSD, FreeBSD, ReactOS e Android OS.

Ambos os sistemas operacionais de 32 bits e 64 bits são suportados.

O programa oferece controle completo de suas VMs, permitindo que você pause, suspenda, desligue ou reinicie os sistemas operacionais.

Você também pode tirar instantâneos de diferentes estados das VMs, o que permite reverter para uma configuração salva.

O programa de virtualização também oferece o Parallels Toolbox com Parallels Desktop para Mac, que fornece funcionalidade útil e conveniente para simplificar tarefas comuns.

Com apenas alguns cliques, você pode executar uma variedade de funções, como arquivar arquivos, gravar sua tela, fazer capturas de tela, gravar áudio, converter vídeo, desligar a comunicação sem fio no modo Avião, definir alarmes e temporizadores de notificação, criar GIFs, entrar no modo Apresentação e alterar a resolução do VM.

Parallels Desktop para Mac é ótimo solução para casa ou usuários de negócios que preferem o ambiente macOS, mas ainda precisam executar tarefas nos ambientes Windows, Linux e Android.

Na verdade, o software de virtualização é a escolha mais popular dos usuários do macOS para executar os sistemas operacionais Windows e Linux em suas máquinas Mac.

Se precisar usar um desses sistemas operacionais em seu Mac, experimente o Parallels Desktop para Mac.
