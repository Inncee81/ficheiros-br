ColdFusion Builder é um IDE comercial usado para criar aplicativos de Internet do lado do servidor.

É construído na plataforma de código aberto Eclipse, o mesmo software que alimenta o aplicativo Flash Builder da Adobe.

.

A interface do ColdFusion Builder pode ser personalizada em diferentes layouts de espaço de trabalho, chamados de “perspectivas”. Cada perspectiva permite que você personalize o IDE para um propósito de desenvolvimento específico, como Web design de página, programação ou implantação.

ColdFusion Builder fornece ferramentas para gerenciar servidores ColdFusion locais e remotos.

Um painel embutido exibe os servidores disponíveis e inclui controles para iniciar e parar cada servidor.

O painel também permite que você visualize os logs do servidor ColdFusion e fornece opções de menu para iniciar monitores de servidor e páginas de administração.

Se necessário, você pode configurar vários servidores para um único projeto.

Por exemplo, um projeto Web pode ser configurado para usar servidores de desenvolvimento e produção.

ColdFusion O IDE (Ambiente de Desenvolvimento Integrado) do Builder inclui várias ferramentas de produtividade para desenvolver aplicativos ColdFusion.

O editor de código-fonte fornece destaque de sintaxe, sm pop-ups de arte, conclusão de código e outros recursos de inteligência integrados.

A visualização do navegador Web exibe visualizações do aplicativo atual enquanto ele está sendo desenvolvido.

Além disso, vários assistentes estão incluídos para gerar módulos de programa, como código Object to Relational Mapping (ORM) e outros componentes de banco de dados.

Como o ColdFusion é baseado em Eclipse, você pode estender o IDE com plug-ins de terceiros (ou "extensões") disponíveis para a comunidade Eclipse.

A Adobe também inclui uma estrutura que permite criar plug-ins usando ColdFusion Markup Language (CFML), embora os plug-ins Eclipse sejam originalmente escritos em Java.

Se você estiver um desenvolvedor da web ou programador que cria aplicativos de servidor ColdFusion, ColdFusion Builder é um programa muito útil.

Ele oferece uma grande quantidade de funcionalidade e o ajudará a escrever um código mais limpo mais rápido.

Depois de experimentar por si mesmo, você descobrirá imediatamente os benefícios.
