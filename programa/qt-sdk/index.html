Digia Qt SDK é um kit de desenvolvimento de software (SDK) usado para criar aplicativos de plataforma cruzada.

O SDK foi desenvolvido pela Nokia antes de ser adquirido pela Digia, que desenvolve o software junto com o Projeto Qt.

Qt SDK é projetado para C ++ e Qt Meta Language (QML) e comumente usado para desenvolver aplicativos móveis, embora você também possa criar aplicativos de desktop e incorporados.

Junto com o suporte para Windows, OS X e Linux, o SDK também fornece cadeias de ferramentas para desenvolver aplicativos Android e iOS.

.

Qt SDK vem com diferentes utilitários para cada um estágio do ciclo de desenvolvimento.

O utilitário Qt Creator é um IDE, que é especialmente útil para novos usuários Qt que permite que eles comecem a desenvolver mais rapidamente.

Qt Designer é uma ferramenta WYSIWYG usada para construção e design de GUIs.

O Simulador QT é usado para testar seu aplicativo e o Qt Assistant permite que você visualize a documentação online no formato de arquivo de ajuda do Qt.

Finalmente, o Qt Linguist é usado para traduzir os aplicativos Qt, C ++ e Qt Quick para o local idiomas.

.

Qt SDK é um SDK competente para o desenvolvimento de aplicativos para desktop, incorporados e móveis.

Embora ofereça utilitários de desenvolvimento úteis e plataforma cruzada suporte, Digia Qt SDK bloqueia você na linguagem QML menos conhecida, tornando-a uma escolha impraticável para desenvolver seus aplicativos.
