Principais características

Crie diagramas manual ou automaticamente a partir de conjuntos de dados complexos
Importe dados e exporte diagramas por meio de uma variedade de formatos de arquivo
Organize os dados usando uma variedade de layouts, como hierárquico e árvore

yWorks yEd Graph Editor é um aplicativo gratuito de plataforma cruzada usado para gerar diagramas.

O aplicativo permite criar diagramas manual ou automaticamente importando dados.

O editor oferece uma variedade de recursos para tornar a criação de diagramas simples, sem sacrificar o poder.

O aplicativo pode importar dados de vários formatos, incluindo GraphML, Excel XLS, GEDCOM e GML.

Ao criar seu gráfico, você pode escolher entre uma variedade de elementos para personalizar seu diagrama.

Você pode usar nós de forma, nós de grupo, elementos UML para diagramas de classe, símbolos BPMN para diagramas de processo de negócios e símbolos de fluxograma.

O aplicativo pode organizar automaticamente os elementos do diagrama para conjuntos de dados complexos ou guiá-lo através do design manual do layout.

Alguns layouts disponíveis incluem Hierárquico, Ortogonal , Orgânico, Árvore e Circular.

Conforme você trabalha, o programa permite que você salve seu documento de diagrama em um formato de arquivo com a extensão .GRAPHML .

Quando terminar, você pode exportar seu diagrama para vários formatos de arquivo, como .PDF , .SWF , .SVG , .JPEG e .PNG.

Ou, se quiser atingir um público mais amplo, pode publicar seu conteúdo na web para visualização interativa.

yEd Editor de gráfico oferece uma ampla variedade de recursos, mas permanece fácil de usar.

O aplicativo pode importar e exportar dados usando diferentes formatos de arquivo e pode criar automaticamente seu diagrama ou permitir que você personalize manualmente seu diagrama.

yWorks yEd Graph Editor é uma escolha sólida para mostrar seus dados.
