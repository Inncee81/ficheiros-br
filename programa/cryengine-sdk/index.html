Principais características

Mergulhe em seus designs com o editor de nível WYSIWYG Sandbox
Conecte-se, aprenda e colabore com outros desenvolvedores por meio da útil comunidade online da Crytek
Modifique os elementos do jogo, incluindo terreno, vegetação, personagens e veículos

Crytek CryENGINE SDK é um kit de desenvolvimento de software (SDK) usado para criar e modificar jogos CryENGINE.

Está disponível gratuitamente para uso não comercial, caso contrário, os usuários podem pagar uma taxa de assinatura mensal para acessar todas as ferramentas de desenvolvimento da CryENGINE.

O SDK permite que você modifique qualquer recurso do CryENGINE seja terreno, vegetação, veículos, personagens ou música.

O kit fornece uma variedade de ferramentas de modding, incluindo o editor de nível Sandbox, que pode ser usado para criar e editar níveis de jogo em terreno aberto.

Ao editar um nível, você pode inserir-se instantaneamente no ambiente para ver suas criações de diferentes perspectivas.

Junto com os vários recursos do pacote de software, a Crytek também promove uma comunidade online saudável, fornecendo fóruns oficiais para que os desenvolvedores da CryENGINE se conectem e colaborem uns com os outros.

Este recurso e ajuda os iniciantes a se familiarizarem com o SDK, pois pode ser intimidante para usuários novatos.

O SDK está ganhando popularidade e é usado para desenvolver jogos 3D para PC e sistemas de console.

Se você usar a versão gratuita do SDK, você pode criar projetos, como jogos não comerciais, projetos escolares e filmes.

Embora não seja tão popular quanto o SDK do Unreal Engine da Epic Games, o SDK da Crytek CryENGINE ainda fornece um mecanismo impressionante e uma variedade de ferramentas para desenvolver ou modificar jogos.
