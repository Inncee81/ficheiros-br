Visão geral do software

Principais características

Acesse a maioria dos mecanismos de banco de dados modernos
Execute cálculos com 34 dígitos de precisão
Crie instruções SQL complexas com uma interface de arrastar e soltar
Encontre o que você precisa com o Enhanced Project Navigation

dBase é um sistema de gerenciamento de banco de dados que apareceu pela primeira vez em 1979.

Desde então, tornou-se uma ferramenta poderosa e hoje pode oferecer suporte à maioria dos mecanismos de banco de dados modernos, como MySQL, SQL Server, Access e FoxPro.

dBase fornece muitas ferramentas úteis para gerenciar seu banco de dados e apoiar o desenvolvimento rápido de aplicativos.

Ele fornece um SQL Builder útil que permite construir instruções SQL complexas com uma interface simples de arrastar e soltar .

Também possui Project Explorer e Enhanced Project Navigation que facilitam a localização da janela ou recurso de que você precisa.

Além de ferramentas para facilitar o gerenciamento do banco de dados, o dBase também possui outros recursos. para melhorar sua funcionalidade.

dBase suporta ActiveX Dynamic Objects (ADO) para melhorar sua compatibilidade e inclui vários componentes ADO personalizados que o tornam fácil de usar.

dBa se também pode realizar matemática de alta precisão com números de ponto flutuante que suportam até 34 dígitos de precisão, aumentando sua precisão matemática ainda mais.

Existem muitos sistemas de gerenciamento de banco de dados diferentes disponíveis hoje.

Se você deseja uma interface intuitiva que simplifique o gerenciamento do banco de dados, então o dBase pode ser a opção para você.
