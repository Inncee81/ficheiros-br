Adobe Captivate é um aplicativo usado para criar conteúdo de eLearning.

Foi originalmente desenvolvido apenas para Windows, mas foi expandido para macOS com o lançamento da versão 5.

O Captivate permite para criar conteúdo de aprendizagem eletrônico, como demonstrações de vídeo, jogos, questionários, screencasts, podcasts, vídeos de treinamento e simulações de software.

O Captivate usa seu próprio formato de arquivo proprietário para salvar projetos e modelos e pode importar vários vídeos, formatos de áudio e imagem, como MPEG e WMV.

O aplicativo pode até mesmo converter apresentações do Microsoft PowerPoint para o formato Flash.

O Captivate permite que você crie conteúdo responsivo personalizado para desktop, tablet, e tamanhos de tela do celular.

O aplicativo vem com uma ampla seleção de temas, permitindo que você crie seus próprios temas personalizados.

Também fornece HT Recursos de publicação de ML5, junto com suporte nativo para importação de animações HTML5.

Captivate é um aplicativo que fornece uma abundância de ferramentas para a criação de conteúdo original e envolvente de eLearning.

No entanto, você não tem para ser um especialista em usar essas ferramentas, já que o aplicativo apresenta uma vasta biblioteca de conteúdo predefinido e tutoriais úteis.

O Adobe Captivate é uma boa escolha para qualquer educador que deseja criar conteúdo interativo que ensinará outras pessoas.
