Principais características

Grave música e conteúdo de vídeo em CDs, DVDs e discos Blu-ray
Converta, importe e exporte uma grande variedade de formatos de arquivo de vídeo
Capture vídeo e áudio de vários dispositivos simultaneamente

Roxio Toast é um programa Mac popular para gravar vídeos e gravar CDs, DVDs e discos Blu-ray.

Embora seja mais conhecido como um utilitário de gravação de disco, o programa também inclui ferramentas para produzir filmes, reproduzir e converter arquivos de mídia, capturar streams da Internet e fazer backup de dados.

Está disponível nas edições Titanium e Pro.

O Toast se integra a vários reprodutores de mídia pessoais e oferece suporte a um grande variedade de formatos de arquivo de mídia.

Pode exportar clipes de vídeo para vários dispositivos, incluindo Apple TV, iPhone, iPad, iPod, PlayStation 3, Sony PSP, Xbox 360, BlackBerry e Palm Treo.

Arquivos de mídia convertidos podem ser carregados diretamente na biblioteca do iTunes para fácil organização.

Os usuários podem salvar um arquivo de imagem de disco .TOAST ou .ISO ou um .DISC arquivo de projeto em vez de gravação g o projeto em um disco.

O Toast também permite que os usuários criem vídeos e apresentações de slides em alta definição.

Ele suporta vídeo de filmadoras AVCHD, TiVo DVRs e EyeTV, e pode importar vários vídeos e formatos de arquivo de imagem.

O Toast também pode capturar a programação de TV na Internet e enviar streams de vídeo em tempo real para um DVR TiVo.

O Toast também pode ser usado para gravar vídeo e áudio de vários dispositivos em ao mesmo tempo.

As ferramentas de edição integradas para Toast podem ser usadas para filtrar ruídos indesejados e cortar segmentos de vídeo.

O vídeo capturado e editado pode então ser enviado para dispositivos como iPads , iPhones e consoles de jogos.

Toast inclui vários programas utilitários para gerenciamento de discos e dados:

  • DiscCatalogMaker : Um programa que permite aos usuários criar um banco de dados pesquisável de todos dados do disco que foram copiados ou criado.
  • Obter backup : Um título do Fujiwara Software que permite aos usuários fazer backup e restaurar dados multimídia.
  • Capa do disco : Um software BeLight título usado para criar e imprimir etiquetas de disco.
  • Spin Doctor : um utilitário para gravar e digitalizar áudio analógico de discos ou fitas de vinil.
  • Mac2Tivo e TiVo Transfer : dois utilitários usados ​​para gerenciar dados multimídia com dispositivos TiVo.

O amplo conjunto de recursos do Roxio Toast são úteis para usuários domésticos e profissionais.

Por exemplo, os usuários domésticos podem usar o Toast para criar DVDs pessoais para compartilhar com amigos e familiares, enquanto os usuários profissionais podem usar o Toast para fazer backup de dados importantes e criar apresentações de slides para os clientes.

O Toast também está crescendo em popularidade entre usuários domésticos para inteiro classificação de sistemas de entretenimento doméstico com TV via Internet e DVRs.

Seja para uso doméstico ou profissional, Roxio Toast é um dos melhores programas disponíveis para gravar vídeo, música e fotos.
