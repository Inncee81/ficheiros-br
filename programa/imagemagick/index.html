Visão geral do software

Principais características

Suporta uma ampla variedade de formatos de imagem, como JPEG, PNG e BMP
Fornece ferramentas de alteração de imagem, como forma e inserção de texto e efeitos
Cria sequências de animação GIF

ImageMagick é um programa de plataforma cruzada de código aberto usado para visualizar, editar e converter uma variedade de formatos de imagem.

O programa está disponível para download como uma distribuição binária ou como código-fonte.

ImageMagick oferece suporte a uma grande variedade de formatos de imagem, como JPEG, PNG, BMP, SVG, TIFF, GIF e PDF, que podem ser editados e convertidos.

O editor de imagens fornece ferramentas para executar uma variedade de funções.

Você pode alterar imagens artisticamente, desenhando formas ou colocando texto em uma imagem.

O aplicativo permite que você aplique bordas ou quadros a uma imagem, junto com uma variedade de efeitos especiais, como desfoque ou nitidez de imagem.

Você também pode criar sequências de animação GIF usando um grupo de imagens.

ImageMagick é comumente usado por meio de uma interface de linha de comando para cada visualização de imagem ou utilitário de manipulação.

No entanto r, diferentes interfaces de programa estão disponíveis para download, como MagickCore, PerlMagick e MagickWand, por meio do site oficial.
