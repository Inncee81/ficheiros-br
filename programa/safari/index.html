Apple Safari é um navegador desenvolvido pela Apple e o navegador padrão para macOS (OS X) e sistemas operacionais iOS.

Foi desenvolvido para Windows até a versão 5.1.7, mas foi descontinuado .

O Safari permite que você execute muitas das funções padrão incluídas em navegadores da web populares, como guias, verificação ortográfica integrada, favoritos, navegação privada, histórico de página da web e gerenciamento de senha.

Por padrão, o Safari oferece um design minimalista, mas você pode personalizá-lo escolhendo o que aparece na barra de ferramentas, mostrando a barra de favoritos, mostrando a barra lateral e muito mais.

Uma das funções mais convenientes do Safari's A barra de ferramentas é o botão Compartilhar, que permite que você compartilhe rapidamente sua página da web por e-mail, Mensagens, AirDrop ou sites sociais como Facebook e Twitter.

O Safari também usa o iCloud para permitir uma navegação contínua em todos os seus dispositivos.

Você pode acessar o histórico de navegação de seus outros dispositivos habilitados para iCloud e se você alterar seus favoritos em um de seus dispositivos, a alteração será atualizada para todos os outros dispositivos.

Para desenvolvedores, o Safari oferece o inspetor da Web, editor de snippets e criador de extensões que permitem examinar o código-fonte, editar trechos e criar e gerenciar as extensões do navegador.

O Safari é uma boa, mas não ótima opção para usuários do macOS.

Parece bom, oferece recursos de navegação padrão e possui funções úteis da barra de ferramentas.

Se você estiver procurando por um navegador da web que forneça navegação contínua entre seus dispositivos iCloud, use o Apple Safari, caso contrário use Chrome ou Firefox.
