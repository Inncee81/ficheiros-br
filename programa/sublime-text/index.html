Visão geral do software

Principais características

Suporte para uma variedade de linguagens de programação, como HTML, Java, CSS e Ruby
Interface de usuário personalizável
Edição dividida de arquivos lado a lado
O recurso Goto Anything permite que você pule rapidamente para diferentes arquivos, linhas e símbolos
Extensível por meio de plug-ins que fornecem funcionalidade adicional

Um editor de texto de plataforma cruzada usado principalmente para editar o código-fonte.

Pode ser avaliado gratuitamente e não há limite para a avaliação, mas o pacote completo requer compra.

O editor de texto oferece suporte a uma ampla variedade de linguagens de programação diferentes, como HTML, SQL, Java, CSS e Perl.

Ele apresenta um layout altamente personalizável, edição dividida de arquivos lado a lado e a capacidade de alternar instantaneamente entre os projetos sem salvar prompts.

O aplicativo ajuda a codificar de forma mais eficaz com o recurso Goto Anything que permite abrir arquivos diferentes, pesquisar em um arquivo, ir para um número de linha ou ir para um símbolo com apenas alguns pressionamentos de tecla.

E quando você realmente precisa se concentrar, o texto Sublime permite que você entre no Modo Livre de distração, que é a edição em tela inteira com apenas o seu texto na tela.

O aplicativo também é altamente extensível.

Você pode instalar plug-ins que fornecem funcionalidade adicional ao editor, independentemente de eles alterarem o tema da interface ou oferecerem suporte a linguagens de programação adicionais.

Fornece suporte para uma infinidade de linguagens de programação e recursos de edição úteis.

Se você editar o código-fonte, Sublime Text é uma de suas melhores opções.
