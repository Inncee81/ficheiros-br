Visão geral do software

Principais características

Suporte para formatos de imagem populares, como JPEG, BMP e PNG
Interface simples de organização de fotos
Ferramentas de edição de fotos
Espaço de armazenamento OneDrive para upload automático e acesso móvel

Microsoft Windows Photos (também conhecido como Microsoft Photos) é um organizador de fotos que permite tirar, editar e compartilhar fotos.

É fornecido com o sistema operacional Windows.

Windows Fotos permite que você use sua câmera embutida ou webcam conectada para tirar fotos e panorâmicas através do aplicativo Câmera.

Você também pode importar fotos de câmeras, telefones, unidades flash USB, etc.

Você pode ver suas fotos de várias maneiras, se quiser vê-las em aplicativos ou na tela de bloqueio.

Depois de tirar ou importar suas fotos, o Windows Photos oferece ferramentas para editá-las.

Você pode ajustar a cor e a iluminação da foto, cortar e girar a foto, retocar qualquer mancha e aplicar efeitos diferentes.

Quando estiver satisfeito com suas fotos (ou pelo menos com algum conteúdo), você pode armazenar seu ph otos no OneDrive (anteriormente SkyDrive).

Este espaço de armazenamento permite fácil compartilhamento e acesso de praticamente qualquer lugar que você vá.

Você também pode fazer upload de fotos e vídeos automaticamente de sua câmera ou telefone para OneDrive.

Windows Fotos também permite que você compartilhe suas fotos diretamente do aplicativo.

Você pode compartilhar suas fotos postando-as online, enviando-as por e-mail para amigos e familiares ou imprimindo-as .

O aplicativo permite que você use aplicativos da Loja Windows para postar em redes sociais e sites de compartilhamento de fotos, como Facebook e Flickr, também.

Embora Microsoft Windows Fotos não oferecem recursos tão avançados quanto alguns outros organizadores de fotos, são muito úteis para o usuário Windows básico.

Oferece uma interface simples de organização de fotos, ferramentas de edição úteis e espaço de armazenamento conveniente.

O st O espaço de armazenamento é especialmente útil se você usar vários dispositivos que executam Windows, como um PC, telefone e tablet.

Se você for um usuário Windows que deseja um organizador de fotos simples, mas útil, você deve experimentar o Microsoft Windows Photos.
