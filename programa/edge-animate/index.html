Principais características

O dimensionamento responsivo permite que as animações caibam em telas de dispositivos móveis e desktop
Editor de linha do tempo para criar animações precisas
Web seletor de fonte para adicionar tipografia rica
Filtros CSS, como desfoques e sombras projetadas
Recursos úteis, como manuais, vídeos e modelos, para usuários novatos

Adobe Edge Animate é uma ferramenta de desenvolvimento da web usada para criar animações HTML interativas.

É usado principalmente para desenvolver rich media para vários dispositivos.

Em novembro de 2015, o Edge Animate é não está mais sendo desenvolvido ativamente.

O aplicativo oferece uma variedade de recursos para projetar e desenvolver animações com recursos completos de JavaScript e HTML.

O editor de cronograma está disponível para criar animações precisas.

Os filtros CSS podem ser usados ​​para efeitos como sombras projetadas e desfocados, e o seletor de fontes da web permite adicionar tipografia rica.

Você também pode adicionar áudio à sua composição com reprodução de animação e interações do usuário.

.

A Adobe fornece recursos úteis por meio do aplicativo Edge Animate e online.

Alguns dos recursos fornecidos incluem manuais, tutorial vídeos e o Edge Animate Showcase, que permite baixar modelos reutilizáveis ​​de sites criados com o Edge Animate.

O Edge Animate é usado por artistas gráficos de movimento, designers e desenvolvedores para criar animações usando padrões da web, como HTML5 , JavaScript e CSS3.

O software apresenta uma série de ferramentas e recursos de design poderosos para colocar os usuários novatos em funcionamento.

Adobe Edge Animate é um aplicativo fácil de usar que ajuda designers e desenvolvedores da web criam animações interativas e responsivas.
