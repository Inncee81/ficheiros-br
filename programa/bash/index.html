GNU Bash é um shell Unix criado para o Projeto GNU.

Bash significa Bourne Again SHell e é um substituto de software livre para o shell Bourne.

É amplamente usado como um shell para o sistema operacional GNU.

O Bash usa a sintaxe de comando que é um superconjunto da sintaxe de comando do shell Bourne e pode executar muitos dos scripts de shell Bourne.

O shell vem com uma variedade de recursos do shell C (csh) e do shell Korn (ksh).

O Bash oferece edição interativa de linha de comando, funções e aliases do shell, matrizes indexadas de tamanho ilimitado e controle de tarefas.

O shell também oferece suporte à substituição de comando, globbing de nome de arquivo, tubulação e estruturas de controle e variáveis ​​para iteração e teste de condição.

Bash é amplamente utilizado como shell para o sistema operacional GNU.

Ele vem com uma variedade de recursos e é ativamente aprimorado.

No geral, GNU Bash é uma grande melhoria em relação ao shell Bourne que integra recursos dos shells csh e ash também.
