Visão geral do software

Principais características

Depurador de código incluído com o editor
Preenchimento automático de código IntelliSense
Suporte integrado para controle de versão Git
Extensível e personalizável

O Microsoft Visual Studio Code, comumente chamado de VS Code, é um editor de código de plataforma cruzada de código aberto.

Ele fornece aos programadores ferramentas de depuração, comandos Git integrados, preenchimento de código IntelliSense e suporte para extensões que podem adicionar recursos ao software.

Embora o programa pareça semelhante, não deve ser confundido com IDE).

.

O Visual Studio Code permite que você escrever e editar uma ampla variedade de arquivos de código-fonte, como .HTML , .JS ., .CSS , .PHP , .JAVA e .PY.

Além dos recursos comuns, como destaque de sintaxe, o Visual Studio Code oferece várias conveniências para desenvolvedores.

O programa vem com suporte integrado para Git e permite adicionar outros sistemas de controle de versão instalando extensões f do VS Code Marketplace.

Ferramentas úteis de depuração exibem erros e sugestões de código e permitem que você defina pontos de interrupção de código.

O IntelliSense também vem de fábrica, completando o código de forma inteligente e oferecendo referências rápidas de código.

A interface do Visual Studio Code consiste em um painel de edição com guias com um layout configurável.

Vários arquivos podem ser visualizados simultaneamente lado a lado ou em uma das várias opções de visualização em grade.

Um painel de explorador de arquivos permite uma navegação rápida nos arquivos e pastas que compõem o espaço de trabalho.

Um terminal opcional pode ser usado dentro do software, permitindo que os desenvolvedores interajam com ferramentas de linha de comando, como tarefas runners e gerenciadores de pacotes, sem ter que alternar entre janelas.

O Visual Studio Code se tornou um dos ambientes de desenvolvimento mais populares entre os desenvolvedores de software e web.

A comunidade saudável de usuários gerou uma grande variedade de extensões disponíveis para adicionar funcionalidades adicionais, bem como muitas opções para personalizar a aparência e o comportamento do software.

Se você está procurando um novo editor de texto para suas necessidades de programação, o Microsoft Visual Studio Code é uma ótima escolha.
