Cal3DViewer é um aplicativo de código aberto usado para carregar e visualizar arquivos Cal3D.

Cal3D é uma biblioteca de animação de personagens 3D escrita em C ++ que foi originalmente desenvolvida como um cliente 3D para Worldforge, mas acabou se tornando um programa separado.

O programa permite abrir arquivos Cal3D, que incluem CFG, CMF, XRF, XMF, CRF, CAF, XSF, CSF e XAF.

O aplicativo possui uma interface simples com um barra de ferramentas de controles básicos.

Você pode visualizar seus personagens 3D no modo sólido, com linhas texturizadas, linhas brancas, pontos brancos com osso, pontos brancos e de diferentes ângulos.

O aplicativo também permite que você visualize mais informações sobre o modelo e reproduza a animação.

Cal3DViewer é um aplicativo leve que fornece uma experiência direta para a visualização de arquivos Cal3D.

E não não vem com um toneladas de recursos, mas faz o que promete, que é visualizar arquivos Cal3D.
