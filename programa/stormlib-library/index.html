StormLib Library é uma biblioteca de código aberto usada para ler e gravar arquivos de e para arquivos MPQ, que são usados ​​por jogos da Blizzard Entertainment.

O programa foi originalmente desenvolvido para extrair os arquivos de música e som de o jogo de RPG de ação Diablo.

A Biblioteca StormLib é ótima em servir ao seu propósito.

Foi projetada simplesmente para ler e gravar arquivos MPQ e é isso que ela faz.

O uso é gratuito e nenhuma licença é necessária para incluí-lo em seus projetos.
