Principais características

Navegue rapidamente pelos arquivos de imagem em uma pasta
Veja as imagens individualmente ou em uma apresentação de slides
Girar imagens em incrementos de 90 graus

Windows Visualizador de fotos é um programa de visualização de imagens incluído no Windows 7 e posterior.

Ele substitui o Windows Visualizador de imagens e fax, que foi incluído com Windows XP e Windows Server 2003.

O visualizador de fotos fornece um método fácil de visualizar arquivos de imagem rapidamente.

Se for o programa padrão para abrir arquivos de imagem, tudo o que você precisa fazer é clicar duas vezes no arquivo desejado para visualizar.

Permite girar as imagens em incrementos de 90 graus e salva automaticamente as alterações.

Se você estiver trabalhando em um computador Windows e precisar visualizar um arquivo de imagem, o visualizador de fotos do Windows é uma opção rápida e fácil.

Se você precisar de recursos avançados de edição de imagem, existem outras ferramentas gratuitas que podem atender melhor às suas necessidades.
