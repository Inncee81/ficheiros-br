Visão geral do software

Principais características

Vários filtros para limpeza de malha, colorização e remeshing
Renderização baseada em Shader OpenGL personalizável
Cor de malha interativa e pintura de seleção
Suporte para muitos formatos de arquivo 3D

MeshLab é um aplicativo de processamento de malha 3D de código aberto usado por especialistas em desenvolvimento 3D.

É usado para processar e editar malhas triangulares 3D não estruturadas em projetos 3D.

MeshLab é livre para uso sob a GNU General Public License, desde que a citação adequada seja feita.

Projetado para ajudar no processamento de modelos não estruturados criados durante a digitalização 3D, o MeshLab fornece muitas ferramentas para limpar essas malhas e convertê-las em 3D utilizáveis modelos.

Possui filtros que permitem remover faces nulas e outros pequenos componentes não utilizados, bem como preencher furos automaticamente.

Filtros de remesagem podem pegar bordas de alta qualidade e recolhê-las em formas simplificadas enquanto preservando as coordenadas de textura.

Este programa também fornece ferramentas para pintura interativa de cores de malha e seleções, bem como suavização.

MeshLab i s uma ferramenta avançada recomendada para usuários experientes na indústria de desenvolvimento 3D que estão envolvidos com digitalização 3D.

Se você está procurando um programa de modelagem 3D para criar modelos do zero, este não é o programa para você .

Se você trabalha com digitalização 3D e acha que precisa de uma boa ferramenta para limpar suas digitalizações, o MeshLab é um ótimo programa.
