H&R Block Tax Software (anteriormente denominado "At Home") é um aplicativo usado para preparar e apresentar declarações de impostos federais e estaduais nos Estados Unidos.

H&R Block lança uma nova edição do Tax Software a cada ano para estar em conformidade com os códigos fiscais para a atual temporada de impostos.

Portanto, o software instalado para o ano fiscal de 2018 não é válido para preencher declarações de impostos para o ano fiscal de 2019.

H&R Block Tax está disponível nas seguintes versões:

  • Básico - Projetado para situações fiscais simples (compatível com Windows e macOS).
  • Deluxe - Projetado para proprietários e investidores (compatível com Windows e macOS). .
  • Premium - Projetado para proprietários autônomos ou de aluguel (compatível com Windows e macOS).
  • Premium e empresarial - Projetado para proprietários de pequenas empresas (compatível com Windows) .

H&R Block's Tax Software apresenta instruções passo a passo que orientam os usuários através do processo de declaração de impostos.

Este modelo permite que usuários comuns apresentem declarações de impostos sem a necessidade de ajuda fiscal profissional.

O programa executa verificações de erros e validação de impostos, o que dá aos usuários a confiança de que as declarações estão em conformidade com as regulamentações fiscais.

Tax Software fornece experiência integrada para pesquisar o reembolso máximo para qualquer tipo de usuário, como situações fiscais simples, proprietários e investidores, autônomos, proprietários de aluguel e proprietários de pequenas empresas.

Cada versão do Tax Software adiciona novos recursos para atender às necessidades crescentes de cada grupo de usuários.

Imposto O software torna a entrada de dados mais conveniente para usuários anteriores do software, permitindo que importem informações fiscais de versões anteriores do software fiscal.

Os usuários também podem importar do TurboT ax, Microsoft Money, Quicken ou qualquer pacote de software financeiro que exporte para arquivos .TXF (Tax Exchange Format).

Depois que o usuário importa os arquivos, o Tax Software automaticamente insere os dados no formulários fiscais do usuário com as informações fornecidas.

Depois que o usuário conclui a devolução, ele tem a opção de imprimir e enviar seus formulários fiscais ou arquivar a declaração eletronicamente online.

Software fiscal é um assistente útil que conduz pessoas solteiras, famílias e empresas através do labirinto complicado da temporada de impostos.

O aplicativo fornece um processo passo a passo, especialistas experientes disponíveis através de bate-papo em tempo real ou telefonema, e conveniente importe recursos para obter o reembolso máximo.

Embora o TurboTax seja um programa tributário mais popular, o H&R Block Tax Software também é uma ótima opção para usuários que precisam preencher seus impostos federais e estaduais es nos Estados Unidos.
