Principais características

Permite a inserção de objetos como rochas, árvores, plantas e até dinossauros no ambiente
Permite que você crie ou carregue atmosferas para dar ao seu ambiente a sensação certa
Fornece Assistente de Animação para ajudá-lo a animar os objetos em seu ambiente
Suporta outros formatos gráficos 3D populares, como 3D Studio e Collada

E-on Vue é um aplicativo usado para criar, animar e renderizar ambientes 3D.

Está disponível para Windows e OS X.

Vue fornece uma série de ferramentas para ajudá-lo a criar qualquer ambiente natural que você possa imaginar.

Você pode criar objetos como plantas, rochas e planetas, com efeitos de iluminação personalizados.

Você pode modificar a atmosfera inserindo nuvens ou carregando diferentes atmosferas.

O programa fornece ferramentas para renderizar e animar seu ambiente 3D, com um assistente de animação útil que fornece conselhos passo a passo para animar seus objetos.

Vue é amigável para outros aplicativos 3D, pois permite importar formatos gráficos de programas populares, como 3D Studio, Collada, WaveFront e Poser.

O aplicativo também é extensível, permitindo a instalação de recursos adicionais, como como objetos ou atm ospheres.

Finalmente, Vue fornece Cornucopia3D, uma comunidade online cheia de recursos que você pode comprar, tutoriais para ensinar novas habilidades e galerias onde você pode postar sua arte.

Vue é um ótimo ferramenta para artistas profissionais que procuram criar ambientes 3D impressionantes, ela tem sido usada em filmes populares de Hollywood, como Avatar, Shrek e How to Train Your Dragon.

E-on Vue irá ajudá-lo a levar seu público para um mundo totalmente novo.
