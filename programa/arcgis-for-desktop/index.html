Visão geral do software

Principais características

Suporta mais de 70 formatos de dados diferentes
Fornece um conjunto abrangente de ferramentas de análise espacial
Integra-se com plataformas móveis para acessibilidade remota

ESRI ArcGIS for Desktop é um aplicativo de mapeamento para a plataforma Windows.

Permite criar, compartilhar e gerenciar mapas geográficos, dados e modelos analíticos.

ArcGIS for Desktop consiste em vários aplicativos integrados, incluindo ArcCatalog, ArcToolbox e ArcMap.

Nesses aplicativos, o ArcGIS for Desktop oferece uma infinidade de ferramentas de análise espacial para gerar análises espaciais, como o número de casas a uma distância especificada de uma planície de inundação.

O aplicativo também oferece análise de localização que permite analisar dados com base na localização física.

Isso é especialmente útil para empresas que desejam conhecer dados do cliente, como onde o cliente mora.

O ArcGIS for Desktop também é um aplicativo flexível, pois suporta mais de 70 formatos de dados que podem ser integrados para análise e visualização.

Outra característica do ArcGIS for Desktop é que ele é um aplicativo altamente extensível.

Existem inúmeras extensões que podem ser baixadas para ampliar os recursos de análise e produtividade do ArcGIS para desktop, como análise 3D de seus dados ou um gerenciador de fluxo de trabalho para ajudá-lo a gerenciar tarefas e recursos gis.

O software também permite o compartilhamento conveniente de dados com o uso do ArcGIS Online.

Você pode procurar dados, compartilhar informações com outras pessoas e importar mapas básicos de um local.

O ArcGIS for Desktop também fornece integração com plataformas móveis, o que torna mapas e dados acessíveis em qualquer lugar.

Por exemplo, uma equipe de bombeiros pôde acessar as plantas do prédio no local antes de entrar no prédio.

ESRI ArcGIS for Desktop é um poderoso aplicativo GIS.

Ele suporta muitos formatos de dados diferentes, fornece uma ampla gama de ferramentas e é ótimo para colaboração.

Se você é um profissional do SIG, você deve dar uma chance ao ESRI ArcGIS for Desktop.
