Adobe Audition é uma estação de trabalho de áudio digital projetada para edição profissional de áudio.

A audição para a Creative Cloud é a primeira versão de 64 bits do aplicativo.

Foi originalmente lançado pela Syntrillium como Cool Edit, mas mais tarde foi comprado pela Adobe e renomeado para Audition.

Você pode usar o Audito para gravar faixas de áudio separadas e combiná-las em uma composição musical completa.

O programa apresenta edição multi-faixa não destrutiva, mas também tem uma exibição de edição de forma de onda para edição destrutiva.

As faixas de áudio separadas também podem ser ajustadas usando ferramentas como correção de tom e dobra de tom.

Além da composição musical, Audition também é comumente usada para edição de áudio para produções de vídeo.

Sua integração com Premiere Pro permite remover rapidamente o ruído de fundo indesejado com a ferramenta Remoção de som e, em seguida, saltar de volta para o Premiere Pro para continuar a edição de vídeo.

Você pode até mesmo abrir projetos Premiere Pro diretamente do navegador de mídia em Audição e selecionar sequências específicas para modificar.

A Adobe Audition é uma ótima editora de som e uma estação de trabalho de áudio digital completa que tanto profissionais quanto hobistas acharão útil.

Com suas muitas ferramentas diferentes e fácil integração com outros produtos da Adobe, é uma ótima opção para edição de áudio.
