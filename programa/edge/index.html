Principais características

Excelente desempenho de navegação

Sincronizar dados de navegação em todos os seus dispositivos conectados por meio do Edge
Recursos de segurança como navegação InPrivate e detecção de sites prejudiciais
Opções de layout de página de guia inspiradoras, informativas e focadas

Microsoft Edge é um navegador da web para Windows 10, 8 e 7, bem como macOS, iOS e Android.

Ele fornece dispositivos familiares recursos de navegação disponíveis no Internet Explorer (que substituiu), Google Chrome e Mozilla Firefox e introduz alguns novos recursos também.

O Edge é desenvolvido a partir do Chromium, que é um projeto de navegador de código aberto a partir do qual o Chrome também é construído.

Ele fornece 3 novas opções de layout de página de guia (inspirador, informativo e focado) junto com a opção de personalizar o layout.

Cada layout de página inclui uma barra de pesquisa desenvolvida pelo Bing e um feed de notícias e informações personalizável.

O Edge oferece muitos dos mesmos recursos de navegação do Chrome, como uma barra de favoritos, histórico da web , lista de downloads, navegação privada, ferramentas de desenvolvedor e extensão s que podem ser adicionados por meio da Microsoft Store.

O navegador possui vários recursos de privacidade, que incluem navegação InPrivate, bloqueio de rastreador de dados e detecção de malware ou sites de phishing.

O Edge também permite para sincronizar os dados do navegador, como senhas salvas, favoritos e configurações em todos os seus dispositivos conectados por meio de sua conta da Microsoft Store.

Todos os novos recursos e desempenho aprimorado tornam o Edge uma escolha atraente para navegação a web.

Ele oferece uma sensação muito semelhante e muitos dos mesmos recursos do Chrome, até mesmo com suporte a muitas extensões do Chrome.

No entanto, uma vez que oferece uma experiência semelhante para usuários do Chrome, não oferece muitos incentivos para que os usuários do Chrome façam a troca.

Independentemente disso, aproximar-se do mesmo desempenho e experiência do navegador da web líder é uma coisa boa, e os Micros baseados em Chromium frequentemente o Edge está indo na direção certa.
