O que é um arquivo .3DC? (Arquivo de cena 3DCrafter)

Cena tridimensional criada pelo 3DCrafter, um programa de modelagem e animação 3D anteriormente chamado 3D Canvas; inclui a geometria da cena e inclui formas, materiais e efeitos de iluminação; usado para criar imagens e animações 3D, e pode ser exportado para um arquivo .AVI.

O que é um arquivo .3DC? (Imagem 3DCT 3D)

Imagem tridimensional criada usando o software de compactação de imagens 3DCT 3D ou o plug-in do software de modelagem 3DCT; contém uma imagem 3D compactada; usada para reduzir o tamanho do arquivo de imagens formatadas padrão 3D e compartilhá-las por e-mail, o que é útil para revisões de design.

Mais Informações

3DCT pode criar arquivos 3DC de vários formatos, incluindo .STL, .PLY, .OBJ, .3DS, .LWO, .DXF, .VRML e .OFF. Ele também pode exportar arquivos .3DM Rhino se o plugin for instalado com Rhino.

NOTA: Os arquivos 3DC podem ser visualizados com o programa gratuito 3DCT Viewer.

O que é um arquivo .3DC? (arquivo de construção iGO 3D)

Arquivo de dados GPS usado pelo software iGO; contém dados de construção 3D e metadados correspondentes; salvo no diretório \iGO\content\building\ do cartão de armazenamento da unidade de navegação.

O que é um arquivo .3DC? (Arquivo de curso RacerMate 3D)

Arquivo de curso de treinamento usado pela Computrainer 3D, um software de simulação para computrainer e ergômetros/treinadores eletrônicos de bicicletas velotron/ contém uma pista de corrida, incluindo características de estrada e elevação; usado para armazenar cursos reais, bem como cursos de reabilitação para fisioterapia.

Mais Informações

O software RacerMate interage com uma bicicleta de exercício estacionária e responde ao movimento enquanto o usuário pedala e dirige a bicicleta. Desta forma, os ciclistas podem treinar para eventos de ciclismo usando simulação.

Arquivo de configuração de células RoboCell

Desenvolvedor Intelitek
Categoria CAD
Formato Não Disponível 

O que é um arquivo .3DC? (Arquivo de configuração de células RoboCell)

Arquivo de projeto criado pelo RoboCell, um programa usado para modelagem e simulação de robôs SCORBOT; armazena uma configuração de componentes de robôs 3D em uma "workcell;" permite que um programa SCORBASE seja executado visualmente em um ambiente virtual 3D.

Mais Informações

RoboCell é usado para integrar software de controle robótico SCOREBASE com recursos de modelagem e simulação 3D. É usado em ambientes educativos.

NOTA: Robocell foi descontinuado.
