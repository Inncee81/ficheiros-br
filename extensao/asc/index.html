O que é um arquivo .asc? (Arquivo de comunicação ActionScript)

Script escrito no ActionScript, uma linguagem de programação usada para controlar aplicativos baseados em Flash; contém código executado pelo software de hospedagem Flash Media Server; usado para controlar eventos de aplicativos, como carregar aplicativos Flash, se comunicar com outros servidores, acessar recursos e transmitir conteúdo multimídia.

Mais Informações

Os arquivos ASC podem ser gerados pelo Adobe Media Encoder (incluído no Creative Suite da Adobe) ou pelo Codificador Ao Vivo do Adobe Flash Media. Eles podem ser usados pelo Flash Media Streaming Server (para aplicativos simples) ou pelo Flash Media Interactive Server (para aplicativos que requerem comunicação multiway).

Por padrão, o Flash Media Server executa o arquivo main.asc quando executa um aplicativo.

O que é um arquivo .asc? (Arquivo blindado PGP ASCII)

Um arquivo ASC é um arquivo ASCII blindado usado pela Pretty Good Privacy (PGP), um programa de criptografia utilizado para comunicação segura. Ele contém uma mensagem assinada digitalmente e pode armazenar informações escritas por texto simples, bem como informações binárias codificadas como texto. Os arquivos ASC também incluem uma chave como texto assinado por limpar, que pode ser verificada usando métodos de autenticação PGP.

Mais Informações

O arquivo ASC é comumente usado para postar avisos de segurança on-line, bem como transmitir mensagens com segurança por e-mail e texto. Uma vez que os arquivos ASC podem conter representações textuais de informações binárias, eles podem ser maiores em tamanho do que o arquivo pré-codificado original. No entanto, essa compensação geralmente vale a sobrecarga para verificar a origem e o conteúdo das informações transmitidas.

NOTA: Em 2010, a Symantec adquiriu a PGP e foi integrada ao Grupo de Segurança Empresarial da empresa.
