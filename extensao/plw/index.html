O que é um arquivo .plw? (Módulo de plugin Windows IDA de 32 bits)

32 bits Windows módulo de plugin usado pelo IDA (The Interactive Disassembler), um depurador e desmontagem multiprocessador; contém dados escritos em C++ que ampliam a funcionalidade da ferramenta IDA.

Mais Informações

Um arquivo PLW é uma biblioteca carregável, semelhante ao .DLL, que amplia os recursos de desmontagem do IDA. Exemplos de funções adicionais que podem ser instaladas incluem a capacidade de exportar informações de disasm do BID ou de permitir o suporte integrado de navegação de salto e descompilação. O plugin deve ser colocado no diretório de plugins do aplicativo.

NOTA: Diferentes extensões de arquivo são usadas para diferentes plugins IDA. A extensão .P64 é usada para Windows extensão de 64 bits, a extensão .PLX é usada por Linux de 32 bits, e a extensão .PLX64 é usada por plugins de Linux de 64 bits.

O que é um arquivo .plw? (Arquivo de dados picolog)

Formato de dados usado pelo software de registro de dados PicoLog; normalmente gerado a partir de um dispositivo de hardware de aquisição de dados do Pico ou de aquisição de dados, como um osciloscópio; inclui um cabeçalho de comprimento fixo, seguido de um registro de dados para cada amostra.

Mais Informações

Os dados registrados pelos dispositivos Pico podem ser coletados, analisados e exibidos em planilha e formato gráfico usando o software PicoLog.
