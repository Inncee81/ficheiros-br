O que é um arquivo .moho? (Projeto de Animação Moho)

Um arquivo MOHO é um projeto de animação criado por Moho (antigo Anime Studio), um programa de animação usado para criar filmes de anime. Ele contém um projeto, que inclui camadas, objetos e uma linha do tempo. Os arquivos MOHO também podem armazenar estilos e efeitos que podem ser aplicados a objetos usados no vídeo.

Captura de tela de um arquivo moho em Smith Micro Moho Pro 12

Desde que os arquivos MOHO armazenam projetos de animação, eles são os arquivos mais conhecidos entre os usuários do Moho. Você pode criar um arquivo MOHO selecionando File → New ou New From Template, em seguida, salvando-o selecionando File → Save ou Save As.... Para abrir um arquivo MOHO, basta selecionar File → Open... ou Open Recent... e selecionar o arquivo MOHO.

ARQUIVOS MOHO foram originalmente criados pelas primeiras versões de Moho. No entanto, o arquivo foi substituído pelo arquivo .ANME quando Moho se tornou Anime Studio. Eventualmente, na versão 12, Anime Studio foi renomeado como Moho, que usa o arquivo MOHO para armazenar projetos. As versões atuais do Moho ainda suportam arquivos ANME.
