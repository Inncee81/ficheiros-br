O que é um arquivo .lma? (Logotipo! Arquivo UDF)

Arquivo criado pelo Logo! Soft Comfort, um aplicativo de automação usado para criar programas de circuito; contém um "O diagrama de bloco de função definido pelo usuário (UDF)" que contém todas as partes do programa de circuito; inclui temporizadores, texto rico, contadores, gatilhos e outras funções.

Mais Informações

O Logotipo! O programa Soft Comfort possui uma interface "drawing board" que organiza o posicionamento e a vinculação de diferentes elementos do programa para criar e testar programas complexos de controle. O programa oferece inúmeros recursos, como edição de diagramas, formatação de texto e ferramentas de simulação.

Para criar um arquivo LMA, selecione File → New → UDF diagram (UDF), construa seu diagrama, selecione File → Save or Save As..., escolha o local de salvamento (ele é padrão para a pasta "Program"), nomeie o arquivo e clique em Salve.

Para abrir um arquivo LMA, basta clicar duas vezes no arquivo ou selecionar File → Open... no aplicativo, navegar para o seu arquivo e clicar em Abrir.

Para editar as propriedades do seu UDF em seu arquivo LMA, selecione Edit → Edit UDF properties..., faça suas edições (crie uma senha, altere parâmetros, etc) e clique em OK.

Existem algumas restrições com UDFs em um arquivo LMA:

  • No máximo, um UDF pode ter 8 conectores de entrada e 4 de saída.
  • Os registros de turno, os controladores pi, o Registro de Dados, textos de mensagens e entradas e saídas digitais e analógicas não podem ser integrados a um UDF.
