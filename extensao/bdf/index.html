O que é um arquivo .bdf? (Arquivo de dados binário)

Formato de dados binários legível por programas de análise de dados forenses; usado para identificar dados brutos recuperados de blocos de espaço não alocado em um disco rígido.

Mais Informações

BDF arquivos são reconhecidos por E-mail Detective, um programa de software forense que pode extrair texto e imagens de arquivos de e-mail aoL.

O que é um arquivo .bdf? (Arquivo de patch BSDiff)

Formato patch usado para atualizar ROMs de videogames; também pode ser usado para produzir patches para arquivos binários.

Mais Informações

Patching é comumente feito para ROMs de videogame por uma das duas razões. Uma das razões é que o usuário quer jogar um jogo em inglês que não foi originalmente criado para usuários ingleses. A outra razão é que o usuário quer alterar a jogabilidade, como a ordem de nível ou o diálogo de caracteres.

As ferramentas bsdiff/bspatch podem ser usadas para criar e aplicar arquivos de patch BDF:

  • Para criar um arquivo de patch BDF que faça o goodfile.123 desejado do seu badfile.123 original, digite bsdiff badfile.123 goodfile.123 patch.bdf.
  • Para aplicar um arquivo de patch BDF ao seu badfile.123, que cria posteriormente o seu goodfile.123, digite bspatch badfile.123 goodfile.123 patch.bdf.

NOTA: A extensão do arquivo .BSDIFF também pode ser usada em vez da extensão do BDF, mas é menos comum.

O que é um arquivo .bdf? (Arquivo de entrada de dados em massa)

Arquivo de entrada usado pelo MSC Nastran, um programa que analisa estruturas como aeronaves, edifícios e espaçonaves; contém definições de dados em massa para serem inseridas no MSC Nastran para analisar os níveis de estresse e tensão das estruturas do modelo; inclui informações como o nome dos dados em massa, o número de grades e os locais de nó dos elementos que compõem os modelos; os dados aparecem na linguagem de programação fortran, que foi originalmente desenvolvida pela IBM, e significa Formula Translating System.

Mais Informações

Arquivos BDF são separados em 5 seções:

  1. Declaração nastran
  2. Gerenciamento de Arquivos
  3. Controle Executivo
  5. >Controle de casos
  6. Dados em massa

NOTA: O arquivo BDF é semelhante aos arquivos .NAS e .DAT, que são usados para inserir dados pelo MSC Nastran.

O que é um arquivo .bdf? (Formato de distribuição de bitmap glifo)

Formato de fonte baseado em ASCII usado em sistemas Macintosh e Windows; armazena fontes de tela da Adobe como dados legívels por humanos; deve ser colocado na pasta Fontes do sistema para ser reconhecido pelos aplicativos.

Mais Informações

Os arquivos BDF são referenciados por programas que acessam fontes do sistema.

O que é um arquivo .bdf? (Arquivo de imagem de disco MIDAS BDF)

Formato de imagem em disco usado pelo software ESO-MIDAS ( European Southern Observatory-Munich Image Data Analysis System); usado principalmente para armazenar dados de astronomia, como fotometria estelar e superficial.

Mais Informações

Pode ser lido e analisado em sistemas Unix e Open VMS usando o software ESO-MIDAS.
