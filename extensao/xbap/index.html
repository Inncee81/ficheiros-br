O que é um arquivo .xbap? (Arquivo de aplicativo do navegador XAML)

Arquivo de aplicativo de Internet usado pela tecnologia XAML Browser Application (XBAP), que permite que aplicativos ricos da Internet seja executado em navegadores Web; contém um arquivo de programa compilado, semelhante a um arquivo de .EXE, mas para o navegador Web; executado dentro de uma caixa de areia para fornecer segurança.

Mais Informações

Arquivos XBAP podem ser compilados usando o Microsoft Visual Studio. Eles exigem que a estrutura NET da Microsoft seja instalada para ser executada. Eles podem ser implantados a partir de um servidor Web ou como um aplicativo autônomo e podem ser iniciados no Internet Explorer ou Firefox.

NOTA: Os aplicativos de navegador XAML são Windows programas da Presentation Foundation (WPF) que são executados no navegador.
