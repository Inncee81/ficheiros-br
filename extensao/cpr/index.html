O que é um arquivo .cpr? (Adobe Captivate Preferences File)

Um arquivo CPR é um arquivo de preferências criado pelo Adobe Captivate, um aplicativo de gravação de tela. Ele contém configurações como a qualidade e o formato do vídeo de saída e é salvo em um formato XML. Os arquivos de RCP são usados para transferir preferências entre projetos.

O que é um arquivo .cpr? (Cartucho CPC Plus)

Arquivo cartucho criado por NO$CART para imitar jogos CPC em computadores 464plus usando as ROMs CPC 6128 incluídas no NO$CART; convertidas a partir de uma imagem de disco CPC.

Mais Informações

O CPC (Color Personal Computer) foi desenvolvido pela Amstrad em meados da década de 1980. Os desenvolvimentos tecnológicos posteriores da Amstrad incluíram o computador 464plus sem disco e o console de jogos GX4000 que usava cartuchos.

NO$CART converte os arquivos .DSK usados pelo computador CPC em cartuchos CPC Plus/GX 4000. NO$CARTROM (incluído no download no$CART) contém uma cópia das ROMs DE BIOS, BASIC e AMSDOS do CPC 6128 que são copiadas para os primeiros bancos de cartuchos, e a imagem do disco é então copiada para os bancos restantes do cartucho. O uso da ROM 6128 original permite que os jogos usem quaisquer funções de BIOS.

NOTA: Criado para ser acessado em computadores CPC 464plus sem disco.

O que é um arquivo .cpr? (Projeto Cubase)

Projeto de produção de áudio criado pela Cubase, um aplicativo DAW (Digital Audio Workstation, estação de áudio digital); armazena dados do projeto, como faixas de áudio e MIDI, instrumentos de software, efeitos de áudio e configurações de mixagem; também contém referências a ativos de arquivos de áudio.

Mais Informações

Os arquivos de RCP são criados pela versão 4 ou posterior da Cubase, bem como as edições Cubase LE e SX/SL.

NOTA: Os arquivos do projeto cubase não contêm os arquivos de áudio reais do projeto. Portanto, se você copiar ou mover um projeto cubase, você também deve transferir todos os arquivos referenciados também.

O que é um arquivo .cpr? (Arquivo de configurações de exibição de telefone Windows)

Exibir arquivo de configurações usado pelo Windows Phone (e Windows Mobile), um sistema operacional para dispositivos móveis; salva o layout dos painéis de configurações e outros painéis na interface Windows Phone; salvo em um formato XML e pode ser personalizado.

Mais Informações

Os arquivos de RCP são salvos no diretório \windows\ do sistema de arquivos do dispositivo Windows Phone.
