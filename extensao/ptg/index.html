O que é um arquivo .ptg? (Pintura artrage)

Projeto de pintura com o ArtRage, um programa de edição e pintura de imagens; contém todos os dados de imagem do projeto salvos no ArtRage; pode ser exportado para outros tipos de arquivos de imagem, incluindo .PNG, .JPG e .PSD; pode ser exibido como uma miniatura em Windows e Mac OS X.

Mais Informações

ArtRage está disponível nas edições standard e Studio Pro.

O que é um arquivo .ptg? (Arquivo de configuração do PowerTeacher Gradebook)

Arquivo de configuração usado pelo PowerTeacher Gradebook, um aplicativo de gerenciamento de sala de aula baseado na Web; contém informações de configuração de livro de séries que são referenciadas quando o aplicativo se conecta ao servidor; inclui identificação do usuário, identificação de sessão, número da escola, identificação do professor, credenciais e idioma.
