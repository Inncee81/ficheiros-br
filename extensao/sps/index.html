O que é um arquivo .sps? (Arquivo de definição do alto-falante do motor Quake 3)

Arquivo de sons de mapa usado por jogos desenvolvidos com o motor Quake III (id Tech 3); contém definições de alto-falantes de texto simples para sons de mapa ambiente; especifica o arquivo de áudio para reproduzir, volume, alcance e x, y, z origem dentro do mapa; também pode especificar se o som é looped, transmitido ou direcionado a uma entidade específica.

Mais Informações

As definições de alto-falantes SPS são declaradas dentro de speakerDef { . blocos, que contêm as propriedades para o som. Os arquivos SPS são tipicamente localizados no diretório de /sound/maps/ de um .PK3 arquivo de jogos.

NOTA: Uma vez que os arquivos SPS contêm texto simples, eles podem ser criados ou modificados com qualquer editor de texto. Eles também às vezes podem ser criados com editores no jogo, como o Editor de Alto-Falantes dentro de Wolfenstein: Enemy Territory.

O que é um arquivo .sps? (Sharkport PS2 Salvar arquivo)

Arquivo criado pela Sharkport, um acessório playstation 2 (PS2) usado para transferir jogos salvos para o PC; contém uma salvação de cartão de memória criada pelo dispositivo Sharkport; pode ser compartilhado com outros usuários do Sharkport para carregar o mesmo jogo salvo.

Mais Informações

Arquivos SPS podem ser convertidos em outros formatos de salvamento usando mymc, um programa de conversão de salvamento PS2. Os tipos de conversão de destino incluem arquivos max .MAX e arquivos .PSU EMS.

NOTA: Outras versões do Sharkport salvam jogos com as extensões .MD e .PS2.

O que é um arquivo .sps? (Arquivo do programa SPSS)

Arquivo de programa utilizado pelo SPSS, um aplicativo de análise estatística; salvo em formato de texto simples e contém instruções escritas usando a sintaxe SPSS; geralmente desenvolvida com o SPSS Syntax Editor; usado para manipular conjuntos de dados e automatizar análises estatísticas.

Mais Informações

Para executar um arquivo SPS, escolha File → Open → Syntax..., selecione seu arquivo SPS e escolha Run → All depois que o arquivo for aberto. Você também pode passar por arquivos SPS usando a opção Run → Step Through ou executar uma seleção do arquivo escolhendo Run → Selection.

O que é um arquivo .sps? (Arquivo de configurações de sincronização)

Arquivo de configurações de backup criado pelo SyncBack, um programa de backup e sincronização; salvo usando .ZIP compactação e contém configurações como diretórios de origem e destino e o cronograma de backup para o perfil; usado para salvar trabalhos de backup.

Mais Informações

Os arquivos SPS são salvos usando .ZIP compactação. O SyncBack está disponível nas edições Pro, SE e Freeware.
