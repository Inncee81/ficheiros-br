O que é um arquivo .MMM? (Arquivo MiraMon Maps)

Arquivo de mapas não comprimidos criado com o MiraMon, um programa de Sistemas de Informação Geográfica (SIG) desenvolvido cooperativamente pelo Centro de Pesquisa Ecológica e Aplicações Florestais (CREAF) e pela Universidade de Barcelona (UAB); contém dados de mapas gráficos em formatos vetoriais e rasteres; também pode incluir documentos anexados ou arquivos de mídia.

Mais Informações

Os Mapas miramon podem ser compactados em .MMZ arquivos para torná-los mais fáceis de distribuir. Para acessar camadas individuais do arquivo do mapa, ele deve ser visualizado no formato não compactado.

O que é um arquivo .MMM? (Arquivo do gerente de dinheiro)

Arquivo de dados introduzido com a versão 7 do Money Manager, um aplicativo de finanças pessoais e empresariais; armazena dados financeiros para contas bancárias, pedidos e transações; usado para rastrear finanças pessoais ou da empresa e pode ser usado para gerar relatórios no software.

Mais Informações

NOTA: O Money Manager versão 6 usa arquivos .MM6.

Desenvolvedor N/A
Categoria Dados
Formato Binário 

O que é um arquivo .MMM? (Arquivo de dados métricos mestre múltiplos)

Arquivo de recursos criado para várias fontes mestres, que são programas de fonte tipo 1 que incluem duas ou mais fontes "master" dentro de um único arquivo de fonte; contém dados métricos referenciados por .PFM e .PFB arquivos no Ambiente Windows; não devem ser abertos manualmente.

Mais Informações

Os dados mantidos nos arquivos MMM determinam os eixos de fonte, que representam a largura, o peso e o tamanho óptico da fonte.

NOTA: Deve ser usado com arquivos de fontes PFM e PFB.

O que é um arquivo .MMM? (Arquivo de arranjo do criador de música)

Um arquivo MMM é um projeto de áudio criado pelo MAGIX Music Maker, uma estação de trabalho de áudio digital (DAW). Ele contém um arranjo multi-faixa, que inclui loops instrumentais, clipes de áudio, transições e efeitos.

Mais Informações

Captura de tela de um arquivo mmm no MAGIX Music Maker 2019

Arquivos MMM permitem que você salve, feche e reabra um projeto de arranjo para mais edição. Quando você terminar de ajustar seu arranjo, você pode exportar seu arranjo para vários formatos de áudio diferentes, incluindo .WAV, .MP3, .OGG, .WMA e .MOV.

Você pode abrir um arquivo MMM no Music Maker selecionando File → Load project... ou clicando no ícone da pasta no menu. Você pode salvar um arquivo MMM no Music Maker selecionando File → Save project... ou clicando no ícone do disquete no menu.
