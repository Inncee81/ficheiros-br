O que é um arquivo .pro6? (Documento ProPresenter 6)

Um arquivo PRO6 contém uma apresentação criada com o ProPresenter 6, que é um programa usado para a montagem de produções para eventos esportivos, cultos à igreja e conferências. Ele armazena informações sobre uma apresentação, que inclui o número de slides, o layout de cada slide, transições e tempo entre slides. Os arquivos PRO6 também contêm referências a áudio ou vídeo na apresentação.

Captura de tela de um arquivo pro6 em Visão Renovada ProPresenter 6

Você provavelmente só encontrará arquivos PRO6 se você for o especialista em vídeo e usar o ProPresenter para exibir apresentações complexas para arenas esportivas, locais de shows ou casas de adoração. Os arquivos PRO6 são criados pelo ProPresenter quando um usuário salva uma nova apresentação, que pode ser uma música com vários slides para versos e refrões, uma apresentação de slides com vários slides de imagens e vídeos, ou outro tipo de apresentação personalizada destinada a ser exibida no ProPresenter.

Todos os documentos do ProPresenter 6 são armazenados na Biblioteca, onde os usuários podem pesquisar suas diferentes apresentações e exibi-las. Você pode ir para o local onde o arquivo PRO6 é armazenado clicando com o botão direito do mouse no ProPresenter e selecionando "Open File Location" em Windows ou "Reveal in Finder" no macOS.

PRO6 arquivos não incluem os arquivos multimídia (por exemplo, .MP3, .MOV ou .WMV arquivos) incluídos na apresentação. Se você deseja incluir arquivos multimídia com o documento, exporte seu documento como um arquivo .PRO6X escolhendo File → Export Document → As Bundle.... Você também pode clicar com o botão direito do mouse em um documento existente e selecionar "Export ProPresenter Bundle"
