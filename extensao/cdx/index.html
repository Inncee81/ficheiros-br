O que é um arquivo .cdx? (Arquivo de índice de tabela Alfa Cinco)

Um arquivo CDX é um arquivo de índice utilizado pela Alpha Five, um IDE para desenvolvedores de aplicativos web. Ele contém dados de índice de tabela referenciados pela Alpha Five para encontrar registros de tabela e é criado quando os campos de índice na tabela são definidos e destinados a encomendar listas em seu arquivo de dados. Os arquivos CDX só podem conter nomes de índices de até dez caracteres sem espaçamento e são semelhantes a .DDX, .ALX e .ASX arquivos de índice Alpha Five.

Mais Informações

Ao fazer backup dos seus dados, verifique se o arquivo CDX também está sendo salvo. Caso contrário, quando os dados de arquivos .DBF e .FPT são apenas restaurados, o arquivo CDX manterá listas de dados antigos e o Alpha Five terá que criar novas listas para os dados restaurados, o que pode levar tempo.

NOTA: Em 2013, a Alpha Five tornou-se Alpha Anywhere.

O que é um arquivo .cdx? (Arquivo de troca de chemdraw)

Arquivo de informações químicas criado pela ChemDraw, um conjunto de programas de edição de moléculas para plataformas Macintosh e Windows; armazena dados moleculares em um formato binário marcado; usado para armazenar desenhos químicos precisos.

Mais Informações

Arquivos CDX armazenam informações como átomos, ligações, fragmentos, setas e texto.

O formato CDX é o formato nativo usado pela ChemDraw. Outro formato chamado CDXML (.CDXML arquivos) é usado para exportar e trocar dados CDX em um formato XML.

O que é um arquivo .cdx? (Arquivo de índice composto)

Um arquivo CDX contém um índice de arquivos ou outros dados armazenados em um banco de dados. É semelhante a um arquivo de índice compacto (.IDX), mas os nódulos de folha no nível mais baixo de um índice composto apontam para uma das tags do índice. Os arquivos CDX foram introduzidos na versão 2 do Microsoft Visual FoxPro, que é um programa de gerenciamento de banco de dados para Windows.

Mais Informações

O índice composto é o formato de índice de banco de dados nativo usado pela Visual FoxPro. Existem dois tipos de arquivos CDX:

  1. Arquivo CDX estrutural - Aberto e mantido automaticamente pela Visual FoxPro quando a tabela é acessada e requer uso exclusivo da tabela.
  2. Arquivo CDX não estrutural - Não aberto ou mantido automaticamente pela Visual FoxPro quando a tabela é acessada e novos registros são adicionados, mas não requer uso exclusivo da tabela.

NOTA: Visual FoxPro foi descontinuada em 2007.

O que é um arquivo .cdx? (Documento ConceptDraw PRO XML)

XML arquivo diagrama criado pela ConceptDraw PRO, um programa de desenho técnico; armazena a versão XML de um arquivo binário de .CDD; usado para transferir informações de desenho em um formato de troca XML, que é mais fácil de ler do que a versão binária.

O que é um arquivo .cdx? (Arquivo compactado CorelDRAW)

Arquivo criado pelo CorelDRAW, um programa de edição vetorial-gráficos incluído no pacote de software CorelDRAW Graphics Suite; contém um arquivo de desenho .CDR compactado; usado para reduzir o espaço de armazenamento necessário para um desenho.

Mais Informações

Arquivos CDX são úteis para enviar arquivos menores por e-mail ou para arquivar desenhos em mídia externa.
