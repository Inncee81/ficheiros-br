O que é um arquivo .pbk? (Arquivo dial-up da reserva telefônica)

Arquivo de dados usado pelo utilitário Network Connections no Microsoft Windows, que também é chamado de Phonebook de acesso remoto; armazena configurações de conexão dial-up salvas, que incluem números de telefone, logins e senhas; podem salvar dados para múltiplas conexões.

O que é um arquivo .pbk? (Arquivo do kernel Pixel Bender)

Arquivo de programa criado pelo Pixel Bender Toolkit, um aplicativo da Adobe usado para criar filtros de imagem personalizados para programas como Photoshop e After Effects; contém código-fonte escrito na linguagem do kernel Pixel Bender, que usa sintaxe semelhante a C.

Mais Informações

O Pixel Bender Toolkit foi projetado para criar algoritmos de processamento de imagem de baixo nível que compilam eficientemente para execução em sistemas multiprocessadores de GPU e CPU. A linguagem Pixel Bender permite que os desenvolvedores escrevam algoritmos altamente otimizados de manipulação de imagens.

NOTA: Os arquivos PBK podem ser compilados em .PBJ arquivos que são filtros que são utilizáveis pelo Adobe Flash Player.
