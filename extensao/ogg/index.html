O que é um arquivo .OGG? (Arquivo de áudio Ogg Vorbis)

Um arquivo OGG é um arquivo de áudio compactado que usa compactação de áudio Ogg Vorbis gratuita e não patenteada. É semelhante a um metadados, como informações de artistas e dados de trilha. Os arquivos OGG são suportados por muitos jogadores de música de software e alguns toca-músicas portáteis.

Mais Informações

O formato Ogg é um contêiner usado para armazenar dados de áudio. O termo "Vorbis" refere-se a um esquema de codificação específico fornecido pela XiphOrg, os desenvolvedores do Ogg. Os arquivos OGG também podem incluir outros tipos de compactação de áudio, incluindo FLAC e Speex. No entanto, esses arquivos não podem ser oficialmente referidos como arquivos "Vorbis", uma vez que eles usam um esquema de compressão diferente.

O que é um arquivo .OGG? (Arquivo gráfico de origem)

Um arquivo OGG contém um gráfico criado pela Origin, um aplicativo de análise e gráfico de dados usado por vários cientistas e engenheiros. Armazena um gráfico gerado a partir de dados importados, como uma coluna, barra, torta, linha, símbolo, cachoeira, 3D, polar ou gráfico estatístico. Os arquivos OGG também armazenam metadados sobre o gráfico, como o título e as cores.

Mais Informações

Você pode abrir arquivos OGG com Origin e Origin Viewer. O Origin só está disponível para Windows e é o programa principal usado para criar, abrir e editar arquivos OGG. Quando aberto na Origem, o gráfico aparece como uma janela em um projeto aberto (.OPJ ou .OPJU arquivo). O Origin Viewer também é comumente usado, pois é uma alternativa gratuita ao Origin e pode ser usado em Windows e macOS.

Para abrir um arquivo OGG no Origin, selecione File → Open..., selecione o tipo de arquivo "Origin Window ogg", navegue até o local do arquivo OGW e clique em Abra.

Você pode salvar gráficos origin como modelos, a fim de preservar configurações a serem usadas com outros gráficos que você cria. Para fazer isso, selecione File → Save Template As... e a Origin criará um arquivo .OTP para armazenar o modelo.

NOTA: O lançamento do Origin 2018 incluiu um formato unicode atualizado para salvar livros de trabalho (.OGGU), que apresenta vantagens de desempenho sobre o formato de arquivo OGG mais antigo e não-unicode.
