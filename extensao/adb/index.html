O que é um arquivo .adb? (Arquivo de banco de dados de capacidade)

Arquivo de banco de dados criado pelo Ability Database, um programa de banco de dados relacional de desktop incluído no Ability Office; salva tabelas, formulários, consultas, relatórios e macros para o banco de dados; usado para armazenar livros de endereços, inventários, informações do cliente e outros dados estruturados.

Mais Informações

Os arquivos de banco de dados de capacidade são compatíveis com .MDB arquivos criados pelo Microsoft Access e podem ser exportados para o formato MDB.

O que é um arquivo .adb? (Arquivo do corpo de Ada)

Arquivo de programação contendo código-fonte escrito na linguagem de programação Ada; armazena o corpo de um programa Ada; normalmente salvo junto com um arquivo "Spec" correspondente que tem uma extensão .ADS.

O que é um arquivo .adb? (Arquivo de banco de dados Alpha Five)

Arquivo usado pela Alpha Five, um IDE que permite que os usuários desenvolvam aplicativos Web; contém nomes de arquivos para todas as tabelas e bibliotecas que fazem parte do banco de dados; armazenados como parte de um banco de dados global para desenvolvedores Alpha Five.

Mais Informações

Os arquivos ADB podem ser reutilizados por outros programas Alpha Five.

NOTA: Em 2013, a Alpha Five tornou-se Alpha Anywhere.

Desenvolvedor Now Software

Formato Não Disponível 

O que é um arquivo .adb? (Agora, o modelo do catálogo de endereços de contato)

Modelo de livro de endereços criado pelo Now Contact, um aplicativo que organiza informações de contato; contém um layout de impressão de modelo para seu catálogo de endereços; as opções pré-definidas incluem "Portfólio A4"," "DayRunner Classic," "Organizador - 2 Coluna," e "Portfólio - Basic"

Mais Informações

Para criar o arquivo ADB, selecione Define → Print Templates → Address Books..., selecione o botão " " no canto superior direito, nomeie o modelo e clique em OK. Em seguida, você pode editar o layout do modelo e suas dimensões. Clique em OK quando terminar.

Para selecionar o modelo que deseja imprimir, selecione File → Print Templates..., escolha seu modelo e clique em OK.
