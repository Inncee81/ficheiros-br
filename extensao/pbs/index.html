O que é um arquivo .pbs? (PaintShop Pro Brush Strokes)

Um arquivo PBS é um arquivo predefinido usado pelo Corel PaintShop Pro, um programa usado para editar imagens e fotos digitais. Contém uma coleção de pinceladas usadas pela ferramenta Brush no PaintShop Pro. Os arquivos PBS incluem forma, textura, dinâmica e outros dados para cada estilo de escova.

Mais Informações

O PaintShop Pro permite que você escolha entre diferentes estilos de pincel ao usar sua ferramenta de pincel na tela. As diferentes predefinições para os pincéis são armazenadas no arquivo PBS e são usadas para simular a textura de vários tipos de pincéis. Embora você use as predefinições de escova através da interface do programa, você provavelmente não se deparará com um arquivo PBS real a menos que você baixe ou receba novos pincéis de um site ou outro usuário do PaintShop Pro.
