O que é um arquivo .ilogicvb? (Arquivo ILogic VB)

Um arquivo ILOGICVB é um arquivo que contém código de regra iLogic que é pré-processado pela iLogic, um navegador incluído no Autodesk Inventor que é usado para padronizar e automatizar processos de design. Ele contém o código para um programa Visual Basic (VBNET) que é usado para controlar ou monitorar outros recursos, componentes ou parâmetros do Inventor.

Mais Informações

O navegador iLogic possui regras, formulários, formulários globais e regras externas da iLogic. As regras são as regras incluídas no documento, enquanto as regras externas são as regras armazenadas fora do documento e inventor. Essas regras externas são normalmente armazenadas em arquivos ILOGICVB. Embora, eles também podem ser armazenados em arquivos .VB ou .TXT.

Para criar uma regra, clique na guia "Gerenciar" na fita e selecione o painel "iLogic" e clique em "Adicionar regra", nomeie a regra e selecione "OK". To create an external rule, open a rule, copy the code, and then paste the code in an external rule created from the iLogic Browser "External Regras". Em seguida, você pode salvar a regra externa como um arquivo ILOGICVB. Para editar uma regra externa, basta abrir a caixa de diálogo "Editar regra" e clicar duas vezes na regra na árvore.
