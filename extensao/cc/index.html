O que é um arquivo .cc? (Arquivo de código-fonte C++)

Código-fonte para um programa C++; pode incluir todo o código para um único programa ou pode ser um dos vários arquivos de código-fonte referenciados por um projeto de programação; contém funções que são compiladas em código de máquina, que o sistema operacional do computador pode entender.

Mais Informações

Pode ser editado com um editor de texto básico, embora não mostre destaque de sintaxe, que está disponível na maioria dos programas de desenvolvimento de software.
