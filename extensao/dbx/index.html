O que é um arquivo .dbx? (Arquivo de extensão do banco de dados AutoCAD)

Arquivo de extensão usado pelo AutoCAD, um aplicativo CAD de engenharia e design; contém informações personalizadas usadas para ler e escrever informações de desenho; escrito usando o API ObjectDBX da Autodesk, que permite aos desenvolvedores acessar programáticamente componentes de desenho.

Mais Informações

Os desenvolvedores terceirizados podem ler .DWG arquivos criando extensões DBX. Por exemplo, algumas versões do Adobe Acrobat empacotam vários arquivos DBX para ler desenhos CAD do Autodesk.

O que é um arquivo .dbx? (Arquivo de banco de dados criptografado do Dropbox)

Um arquivo DBX contém um banco de dados criptografado usado pelo Dropbox, um serviço de hospedagem de arquivos on-line. Ele armazena informações criptografadas sobre o Dropbox de um usuário em .SQLITE formato de registro. Os arquivos DBX podem incluir arquivos modificados recentemente, o caminho local para uma pasta Dropbox e o endereço de e-mail do usuário.

Mais Informações

Os arquivos DBX são usados pelo Dropbox para armazenar vários tipos de dados e são semelhantes a .DB arquivos de banco de dados, mas são criptografados. O nome do arquivo indica que tipo de informação está contida no arquivo, como filecache.dbx, notifications.dbx, photo.dbx ou config.dbx.

Os arquivos DBX são armazenados em várias pastas nos diretórios de instalação do Dropbox. O config.dbx está armazenado nos seguintes locais:

Windows XP:
C:\Documents and Settings\%USERNAME%\Application Data\Dropbox\
C:\Documents and Settings\%USERNAME%\Local Settings\Application Data\Dropbox\

Windows Vista, 7, 8, 81, e 10:
C:\Users\%USERNAME%\AppData\Local\Dropbox\
C:\Users\%USERNAME%\AppData\Roaming\Dropbox\

macOS:
/Users/$USER/.dropbox/

Linux:
/home/$USER/.dropbox/

O que é um arquivo .dbx? (Pasta de e-mail do Outlook Express)

Um arquivo DBX é uma pasta criada pelo Outlook Express, um programa de e-mail da Microsoft. Contém mensagens de e-mail para uma caixa de correio específica. Os arquivos DBX são normalmente salvos no diretório de usuários Documents and Settings e podem ser copiados para outra pasta, a fim de fazer backup de mensagens de e-mail.

Mais Informações

As pastas DBX têm nomes correspondentes à caixa de correio de e-mail. Por exemplo, nomes comuns são Inbox.dbx, Sent Items.dbx, Drafts.dbx e Deleted Items.dbx. Folders.dbx armazena o índice mestre para todas as caixas de correio. Se faltar, o Outlook Express reexaminará as caixas de correio e irá recriá-la.

DBX pastas de e-mail podem ser importadas para Windows Mail (incluído com Windows Vista) selecionando File → Import → Messages, escolhendo Import mail from an OE6 store directory e selecionando a pasta DBX que você gostaria de importar.

As pastas DBX podem ser importadas para Windows Live Mail (incluído com Windows 7) selecionando File → Import → Messages → Microsoft Outlook Express 6 e localizando a pasta DBX apropriada.

O programa Stellar DBX to PST Converter pode ser usado para converter arquivos Outlook Express DBX para o formato .PST Do Outlook. Em seguida, você pode salvar arquivos convertidos para formatos Office 365, HTML, RTF e PDF.

NOTA: As pastas DBX só devem ser abertas no Outlook Express ou importadas usando outro programa. Se abertos ou editados manualmente, eles podem ficar corrompidos e fazer com que o Outlook Express não funcione corretamente.
