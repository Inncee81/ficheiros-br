O que é um arquivo .rap? (Arquivo flowchart raptor)

Um arquivo RAP é um fluxograma criado pela RAPTOR, um ambiente de programação baseado em fluxograma. Contém um fluxograma, que é uma coleção de símbolos conectados que representam instruções a serem executadas e quando devem ser executadas.

Mais Informações

Captura de tela de um arquivo de rap no Raptor 2015

O arquivo RAP é o principal tipo de arquivo associado ao Raptor. Ele é criado para armazenar um fluxograma, que é usado para projetar um programa Raptor. Quando você cria inicialmente um fluxograma, ele contém apenas os símbolos "Iniciar" e "End". Você pode adicionar símbolos entre esses símbolos, como "Atribuição", "Call", "Entrada",", "Output", "Seleção" e "Loops".

Você pode criar um arquivo RAP selecionando File → New e salvá-lo selecionando File → Save ou Save As.

Você pode abrir um arquivo RAP selecionando File → Open.

O programa RAPTOR foi projetado para ajudar os alunos a visualizar algoritmos.

O que é um arquivo .rap? (Arquivo criptografado do Scarab Ransomware)

Um arquivo com uma extensão .rap é um arquivo que foi criptografado por uma variante do ransomware Scarab, um tipo de malware utilizado por cibercriminosos. Ele é criptografado com criptografia AES e RSA 2048 bits para que não seja possível abrir o arquivo simplesmente alterando a extensão do arquivo.

Mais Informações

O ransomware Scarab renomeia arquivos padrão que são importantes para a vítima com uma extensão .rap. Os arquivos são normalmente documentos, imagens, vídeos e arquivos de backup, como backup de seus arquivos, você pode executar uma restauração do sistema.
