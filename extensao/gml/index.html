O que é um arquivo .gml? (Arquivo de idioma gamemaker)

Arquivo desenvolvedor usado pelo GameMaker, um programa usado para criar jogos de computador; contém código de programa escrito na Linguagem GameMaker (GML) que é executado pelo intérprete gamemaker.

Mais Informações

Os arquivos GML são usados para criar scripts, funções ou módulos de jogo.

O que é um arquivo .gml? (Arquivo de camada gerber mill)

Um arquivo GML é um arquivo de saída de dados criado por ferramentas CAD de placa de circuito impresso (PCB), como o Autodesk EAGLE. Ele contém dados de layout para a camada que especifica os locais de furo que precisam ser moídos.

Mais Informações

Arquivos de camada gerber, como o GML, são usados por máquinas de fabricação de PCB para criar placas PCB. O arquivo GML é uma versão exportada do layout PCB e é referenciado a furos de moinho na placa.

NOTA: O formato Gerber foi originalmente desenvolvido pela Gerber Systems, mas agora é mantido pela Ucamco.

O que é um arquivo .gml? (Arquivo de linguagem de modelagem de gráfico)

Arquivo criado no formato GML (Graph Modeling Language), um formato de texto ASCII usado para descrever estruturas de gráficos; descreve a estrutura do gráfico em uma hierarquia baseada em texto e inclui os nós, bordas e atributos que compõem o gráfico.

Mais Informações

A linguagem de modelagem gráfica pode descrever árvores genealógicas, fluxogramas, redes e outras estruturas de gráficos.
