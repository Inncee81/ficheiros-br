O que é um arquivo .req? (Arquivo de solicitação de cartão Canon SD)

Arquivo vazio às vezes armazenado em cartões SD de câmera digital Canon; salvo com o nome de arquivo ver.req e usado para relatar a versão do firmware na câmera quando o usuário segura o botão FUNC/SET e pressiona o botão DISP.

Mais Informações

Você pode criar manualmente o arquivo ver.req vazio salvando um documento em branco no Microsoft Notepad e renomeando o nome do arquivo para ver.req.

O que é um arquivo .req? (Arquivo de solicitação de certificado SSL)

Um arquivo REQ é um arquivo usado no processo de criação de um certificado digital. Armazena uma Solicitação de Assinatura de Certificado (RSE), que inclui informações que identificam exclusivamente o iniciador da solicitação. Os arquivos REQ são enviados a entidades confiáveis (autoridades de certificado) que utilizam as solicitações para gerar certificados digitais válidos para os solicitantes.

Mais Informações

Os CSRs são usados no processo de criptografia de chave pública, que é usada para validar identidades na Internet. Os certificados digitais validam a autenticidade de uma empresa ou comerciante on-line para que usuários ou clientes possam saber que o site é confiável e seguro.
