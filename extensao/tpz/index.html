O que é um arquivo .tpz? (Arquivo de modelo de exportação do motor Godot)

Um arquivo TPZ é um arquivo comprimido zip usado pela Godot Engine, um mecanismo de jogo de código aberto e gratuito. É usado para distribuir export templates que, uma vez instalado na Godot Engine, permitem que os desenvolvedores exportem seu projeto como um pacote que pode ser executado em outras plataformas sem a Instalação godot engine.

Mais Informações

Quando a Godot Engine foi originalmente lançada, ela não veio com nenhum dispositivo de exportação para que os desenvolvedores tivessem que construir e compilar manualmente os binários apropriados para suas plataformas-alvo. Como correção para a solução, os desenvolvedores que usam o Godot Engine devem baixar o export templates antes de poderem exportar seus projetos para o formato de pacote apropriado para as plataformas Windows, macOS, Linux, Android e HTML5.

As export templates são versões otimizadas do mecanismo do jogo sem as ferramentas de edição pré-compiladas para cada plataforma. Eles consistem em pastas e arquivos em um arquivo TPZ que é simplesmente um arquivo .ZIP renomeado.

O arquivo TPZ pode ser baixado em um navegador da Web clicando no link "EXPORT TEMPLATES" da página "Download" no site da Godot. Para instalar o arquivo TPZ no Godot Engine, selecione Settings → Install Export Templates e selecione o arquivo TPZ. O arquivo também pode ser baixado diretamente para o Godot Engine selecionando Editor → Manage Export Templates e, em seguida, clicando em Download.

Uma vez que os arquivos TPZ são renomeados arquivos ZIP, você pode extrair e visualizar o conteúdo no arquivo renomeando a extensão .tpz para .zip. Em seguida, extraia o conteúdo usando um programa de descompressão ZIP, como o Microsoft File Explorer, o Apple Archive Utility ou o Corel WinZip.

O que é um arquivo .tpz? (Kindle Topaz eBook File)

arquivo eBook usado pelos dispositivos Amazon Kindle e pelo software de desktop Amazon Kindle; contém um eBook que foi salvo de um download da Internet para o computador; suporta fontes embarcadas avançadas, bem como informações auxiliares, como marcadores e anotações, que são armazenadas em arquivos de metadados separados.

Mais Informações

Os arquivos TPZ são salvos em um formato proprietário, mas ainda podem ser abertos pela Calibre. No entanto, se eles contêm proteção DRM, então calibre pode não abri-los.
