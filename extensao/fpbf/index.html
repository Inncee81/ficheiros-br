O que é um arquivo .FPBF? (Mac pasta de queima do OS X)

Um arquivo FPBF é uma pasta de gravação Mac OS X que contém arquivos ou codinomes de arquivos a serem gravados como um CD ou DVD de dados. É usado para armazenar dados que serão queimados em um ou mais discos. Os arquivos FPBF podem ser criados a partir de Mac OS X 104 ou posterior e normalmente aparece na área de trabalho.

Mais Informações

Para criar uma pasta de queima no Mac OS X, basta clicar com o botão direito do mouse na área de trabalho e selecionar "Nova pasta de queima" no menu contextual. Você também pode selecionar "Nova pasta de queima" no menu Arquivo finder.

Uma vez criada a pasta de queima, você pode arrastar arquivos e pastas para ela. O tamanho total da pasta aparecerá na parte inferior da janela. Quando estiver pronto para gravar o conteúdo da pasta em um disco, insira um CD ou DVD e clique no botão Gravar perto do canto superior direito da janela.

NOTA: Os computadores da Apple não incluem mais uma unidade de disco de DVD. Portanto, os arquivos FPBF são menos comumente usados do que com modelos de computador anteriores que incluíam uma unidade de disco.
