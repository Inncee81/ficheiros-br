O que é um arquivo .df1? (Arquivo de dados nativo omnis)

Formato de dados proprietário usado pelo software de desenvolvimento Omnis, que é usado para criar aplicativos de vários níveis e Web; armazena dados em um formato de banco de dados e pode ser referenciado por programas Omnis.

Mais Informações

NOTA: Os arquivos DF1 usados pelo Omnis Studio são formatados de forma diferente dos arquivos Omnis 7. Portanto, arquivos DF1 antigos não podem ser usados com o Omnis Studio. No entanto, novas versões de arquivos DF1 estão incluídas no Omnis Studio. Por exemplo, o arquivo de dados de ícone "Userpicdf1" usado pelo Omnis 7 é substituído por "Omnispicdf1" no Omnis Studio. Outros arquivos DF1 podem ser convertidos ou atualizados dentro do programa.

Arquivos de dados omnis também podem terminar em "df2", "df3", etc.
