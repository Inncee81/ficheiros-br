O que é um arquivo .MSM? (Arquivo de design de circuito multisim)

Projeto de circuito criado pela Multisim, um programa de simulação com ênfase em circuitos integrados (SPICE) usado para ensinar teoria eletrônica e circuitos de design e protótipos; contém um design de uma placa de circuito elétrico, que inclui diodos de lugar, transistores, indicadores e componentes de energia.

Mais Informações

Para criar o arquivo MSM, selecione File → Save As....

O arquivo MSM é usado pela versão 2001 do Multisim. À medida que novas versões do Multisim são lançadas, novas extensões de arquivo são usadas para salvar os designs do circuito. Por exemplo, a versão Multisim 7 usa a extensão de arquivo .MS7.

O que é um arquivo .MSM? (Módulo de mesclagem do instalador Windows)

Banco de dados de um "módulo de mesclagem" que contém configurações de instalação e propriedades para um instalador de software específico; inclui arquivos, recursos, entradas de registro e informações de configuração; muitas vezes contidos em um arquivo .MSI.

Mais Informações

Os arquivos MSM também são chamados "mesclam arquivos de pacotes" Como eles geralmente fazem parte de um pacote de instalações, os arquivos MSM não devem ser abertos manualmente.
