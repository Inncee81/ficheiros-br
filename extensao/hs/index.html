O que é um arquivo .hs? (Roteiro de Haskell)

Código fonte de um programa escrito em Haskell, uma linguagem de programação puramente funcional, o que significa que o código é baseado estritamente em funções; também usa semântica não estrita, que permite ao usuário definir funções e estruturas de dados; semelhante a um arquivo .LHS.

Mais Informações

Deve ser compilado por um compilador Haskell para executar.

O que é um arquivo .hs? (Arquivo Java HelpSet)

Define a documentação de ajuda referenciada pelo sistema JavaHelp para um determinado aplicativo; inclui um arquivo de mapa, informações de exibição, título, ID residencial e dados de apresentação; também pode conter subconjuntos opcionais e informações de implementação; formatados em XML.

O que é um arquivo .hs? (Arquivo de pontuações altas do LaserTank)

Arquivo de jogo criado pelo LaserTank, um jogo de quebra-cabeças onde os jogadores resolvem diferentes níveis; economiza a pontuação alta para um nível e pode ser submetido à lista de altas pontuações do LaserTank on-line.

Desenvolvedor Motorola
Categoria Diversos

O que é um arquivo .hs? (Motorola Flash Flex File)

Arquivo de atualização de firmware para um celular Motorola; usado para "flash" a memória do telefone e atualizar o software; muitas vezes usado para alterar ou "mod" a interface do telefone.

Mais Informações

Pode ser aplicado a um celular conectando o telefone a um PC via USB e executando os programas PST ou RSD.
