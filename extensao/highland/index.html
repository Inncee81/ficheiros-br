O que é um arquivo .highland? (Documento highland)

Um arquivo HIGHLAND é um documento criado pela Highland, um programa de roteiro macOS. Contém texto formatado como roteiro, peça teatral, graphic novel, manuscrito, romance, fala, boletim, artigo, tratamento ou relatório MLA.

Captura de tela de um arquivo highland em Quote-Unquote Apps Highland 2

Você provavelmente só encontrará um arquivo HIGHLAND se você for um roteirista usando o aplicativo Highland. Você só pode abrir um arquivo HIGHLAND com highland selecionando File → Open....

Os documentos de highland são salvos no formato de arquivo Highland, que é uma versão compactada do formato TextBundle projetado para tornar a troca de arquivos de texto simples mais perfeita entre os aplicativos. Um arquivo HIGHLAND é na verdade um arquivo comprimido por Zip que armazena o seguinte conteúdo:

  • Um ou mais metadados.
  • Um arquivo .FOUNTAIN, .MARKDOWN ou .TXT - O arquivo armazena o texto no documento.
  • "assets" pasta - A pasta armazena imagens em destaque no documento.

Enquanto os arquivos HIGHLAND são feitos para serem abertos e editados com highland, você também pode extrair o conteúdo em um arquivo HIGHLAND. Para isso, renomeie a extensão de arquivo .highland para .zip e descomprima o arquivo como um arquivo TEXTBUNDLE. Em seguida, você pode abrir o arquivo com um programa que suporta o formato TextBundle ou renomear a extensão de arquivo .textbundle para .zip e descompactar o arquivo como uma pasta onde você pode editar o conteúdo.
