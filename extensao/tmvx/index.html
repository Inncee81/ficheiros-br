O que é um arquivo .tmvx? (Modelo de documento textmaker)

Um arquivo TMVX é um modelo de documento criado pelo TextMaker, um programa de processamento de espadas incluído no Softmaker Office. Ele pode armazenar texto, objetos desenhados, tabelas, imagens e informações de formatação de página. Os arquivos TMVX são usados para criar documentos .TMD e .TMDX com a mesma formatação ou conteúdo.

Mais Informações

Captura de tela de um arquivo tmvx no SoftMaker Office TextMaker 2018

Os arquivos TMVX são salvos em um formato proprietário e são semelhantes aos modelos do Microsoft Word (.DOT ou .DOTX arquivos). Eles podem ser convertidos em outros formatos em TextMaker, incluindo .PDF, .DOC, .DOCX, DOT, DOT, DOTX, TMD, TMDX, TMV, .ODT, .PSW e .RTF.

Você pode criar um arquivo TMVX selecionando File → Save ou Save as e, em seguida, escolhendo o formato TMVX. Você pode abrir um arquivo TMVX selecionando File → Open e navegando para o arquivo TMVX. Você também pode escolher File → New e selecionar o modelo TMVX do qual deseja criar seu novo documento. Os arquivos TMVX são salvos por padrão na pasta "Modelos [versão]" da pasta "SoftMaker" no caminho da pasta "Documents".

NOTA: Os arquivos TMVX substituíram .TMV arquivos como o tipo de arquivo principal usado para armazenar modelos.
