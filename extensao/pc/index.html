O que é um arquivo .pc? (Arquivo de compositor pessoal)

Partitura Musical criada com Personal Composer, uma notação de música, MIDI, sequenciamento e programa de publicação; pode ser criada usando a interface de software ou gravada a partir de um instrumento MIDI.

Mais Informações

Arquivos de Compositor Pessoal podem incluir notação de música, letras, símbolos, títulos e outras informações.

O que é um arquivo .pc? (Arquivo de código-fonte Pro*C)

Arquivo de código-fonte escrito no Pro*C, uma linguagem SQL incorporada usada com o software de banco de dados Oracle RDBMS; pode conter tanto as declarações de código-fonte C/C++quanto o código SQL; usado para escrever aplicativos de banco de dados que combinam o poder tanto do SQL quanto do C/C++.

Mais Informações

Pro*C permite que o SQL e o C/C++ sejam gravados nos mesmos arquivos de código-fonte e compilados em um programa. O pré-compilador Pro*C transforma as instruções SQL em código C/C++ e, em seguida, compila-as com o código C/C++já existente.
