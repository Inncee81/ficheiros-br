O que é um arquivo .pd4? (Arquivo primer do gerenciador de clones)

Arquivo Primer criado pelo Clone Manager, um programa usado para simulação de clonagem, operações de enzimas, desenho de mapa gráfico; contém uma sequência de primer, informações de moléculas de origem, um nome de primer, critérios de primer, uma descrição e informações de notas.

Mais Informações

NOTA: Os primers podem ser adicionados aos arquivos de coleta de primer .PX5 para um compartilhamento mais conveniente de vários primers entre usuários.

O que é um arquivo .pd4? (arquivo de informações da galeria de fotos ao vivo Windows)

Arquivo de banco de dados usado pelo Windows Live Photo Gallery, um programa que permite que os usuários gerenciem fotos e filmes caseiros; contém metadados descrevendo imagens e vídeos pessoais, incluindo o nome do arquivo, propriedades, informações de caminho e miniatura; usado para rastrear informações de arquivos multimídia.

Mais Informações

Por padrão, os arquivos PD4 usam o nome Pictures.pd4 e são armazenados no seguinte diretório:
%USERPROFILE%\AppData\Local\Microsoft\Windows Live Photo Gallery

Se um banco de dados de arquivos PD4 estiver corrompido, você poderá renomear o arquivo PD4 existente para OLD_Pictures.pd4 e Windows Live Photo Gallery recriará o banco de dados.

NOTA: A Microsoft Windows Live Photo Gallery não é mais desenvolvida.
