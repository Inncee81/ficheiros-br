O que é um arquivo .pls? (Arquivo de lista de reprodução multimídia)

Um arquivo PLS contém uma lista de reprodução, que é uma lista de arquivos multimídia que podem ser reproduzidos com um media player. Armazena referências às localizações dos arquivos multimídia, juntamente com metadados sobre os itens da lista. Os arquivos PLS não armazenam dados reais de áudio e vídeo, apenas informações em texto simples sobre os arquivos de áudio e vídeo.

Mais Informações

Muitos media players podem abrir listas de reprodução no formato PLS. Alguns deles incluem Apple iTunes, Nullsoft Winamp, Clementine e VideoLAN VLC media player. Os editores de texto também podem abrir arquivos PLS, uma vez que são salvos em texto simples.

Existem vários tipos de arquivos usados para armazenar listas de reprodução para reprodudores multimídia. O tipo de arquivo de lista de reprodução mais comum é .M3U, que é criado e aberto por muitos media players. Alguns outros arquivos de lista de reprodução incluem arquivos .ASX, .WPL e .XSPF.

O que é um arquivo .pls? (Arquivo de dados contábeis MYOB)

Arquivo de dados financeiro criado pelo software contábil MYOB, como AccountEdge e Accounting Plus; contém informações financeiras da empresa, como receitas e despesas; usadas para orçamento e rastreamento, bem como relatórios de impressão.

Mais Informações

A tecnologia MYOB não oferece mais alguns produtos que criam arquivos PLS. No entanto, você normalmente pode convertê-los usando produtos de software MYOB mais recentes.

O que é um arquivo .pls? (Arquivo de configurações do PicoLog)

Armazena configurações de captura de dados para o PicoLog, um programa de aquisição de dados incluído com dispositivos de data logger do Pico; pode ser salvo manualmente selecionando "File→Save As" dentro da sessão atual.

Mais Informações

O arquivo recorderpls contém informações sobre a sessão de gravação atual, enquanto playerpls contém informações sobre a sessão atual do jogador.

Os dados contidos no arquivo de registros também são escritos até o final de cada arquivo de dados relacionado .PLW.

O que é um arquivo .pls? (Procedimento armazenado PL/SQL)

Um arquivo PLS contém um procedimento armazenado, que é um conjunto de instruções PL/SQL que executam uma ação para acessar ou modificar dados em um banco de dados. Os arquivos PLS são usados pelo software de banco de dados Oracle PL/SQL, que é usado para vários fins, incluindo armazenamento de dados e processamento de transações on-line.

Mais Informações

PL/SQL é uma extensão de linguagem processual para linguagem de consulta estruturada (SQL). Ele está disponível no Banco de Dados Oracle (desde a versão 6), IBM DB2 (desde a versão 97) e no banco de dados timesTen em memória (desde a versão 1121).
