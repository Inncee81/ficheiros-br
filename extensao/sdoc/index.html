O que é um arquivo .sdoc? (Memorando da Samsung)

Um arquivo SDOC contém uma nota criada pelo Samsung Notes, um aplicativo de anotações comumente usado por dispositivos móveis Samsung selecionados, como S Note e Galaxy S7. Pode armazenar texto, notas manuscritas, imagens, esboços e áudio.

Mais Informações

Os arquivos SDOC são normalmente abertos pelo Samsung Notes. No entanto, uma vez que o arquivo é compactado com compactação zip, você pode renomear a extensão "sdoc" para "zip" e extrair o conteúdo do arquivo. Em seguida, você pode abrir manualmente os arquivos que compõem a nota.

O que é um arquivo .sdoc? (Documento de Satra Khmer)

Documento criado por Satra Khmer, um programa usado para proteger texto de documento; contém texto que foi transposto de texto regular para texto satra usando o sistema de transposição proprietário; usado para armazenar informações confidenciais.

Mais Informações

NOTA: A interface de usuário de Satra Khmer é exibida na língua Khmer, que também é conhecida como Cambojana. O aplicativo não está mais disponível para download.

  Driver-TC group Satra Khmer

O que é um arquivo .sdoc? (Documento de Palavra Selado)

"Documento de palavra selado" (arquivo .DOC) criado pelo software de proteção de documentos IRM da Oracle; bloqueia o documento para que ele só possa ser visualizado por um usuário com direitos suficientes; usado para garantir a distribuição de documentos comerciais e anexos de e-mail.

Mais Informações

Os arquivos SDOC podem ser visualizados usando o Oracle IRM Desktop, um programa que "deslacra"," os documentos para visualização. No entanto, os documentos só podem ser visualizados se o usuário tiver os direitos necessários. O software IRM também às vezes se integra às instalações corporativas do Word e fornece uma opção para "selar" um documento após editá-lo.

NOTA: O software Oracle IRM também cria vários outros tipos de arquivos selados, como arquivos de .PDF lacrados (extensão .SPDF).
