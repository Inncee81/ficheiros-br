O que é um arquivo .rbw? (Arquivo de janela do estúdio real)

Arquivo de janela criado pelo Xojo, um IDE usado para criar aplicativos; contém uma janela exportada para um aplicativo de desktop que pode ser compartilhado entre projetos Xojo; inclui informações descrevendo a janela, como tamanho máximo da janela, cor de fundo, comportamento e botões; substituído pela extensão de arquivo .XOJO_BINARY_WINDOW.

Mais Informações

Antes de ser substituído, você pode criar um arquivo RBW clicando na janela que deseja exportar, selecionando File → Export [window name]..., escolhendo o local de salvar, nomeando o arquivo e clicando em Save.

Para importar o arquivo RBW, selecione File → Import, navegue até a localização do arquivo e clique em Import.

NOTA: REAL Software Real Studio tornou-se Xojo, Inc. Xojo em 2013.

O que é um arquivo .rbw? (Roteiro de Ruby)

Arquivo de código fonte escrito em Ruby, uma linguagem de script orientada a objetos projetada para ser intuitiva e fácil de ler; também pode usar a extensão .RB, exceto que scripts Ruby que usam a extensão "rbw" são tipicamente destinados a código de interface gráfica.

Mais Informações

NOTA: Usar extensões "rb" e "rbw" durante o desenvolvimento permite definir diferentes associações de arquivos para executar diferentes tarefas padrão. Por exemplo, você pode atribuir a extensão "rb" a um intérprete Ruby enquanto define a associação de extensão "rbw" para outro. Alternativamente, você pode definir um para um intérprete enquanto define outro para abrir em um editor de texto.
