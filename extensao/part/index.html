O que é um arquivo .part? (Arquivo parcialmente baixado)

Um arquivo PART é um arquivo parcialmente baixado da Internet usado para downloads que estão em andamento ou foram interrompidos. Alguns arquivos PART podem ser retomados posteriormente usando o mesmo programa que iniciou o download. Arquivos PART são normalmente usados pelo Mozilla Firefox e vários programas de transferência de arquivos, como Go! Zilla, eMule e o programa eDonkey descontinuado.

Mais Informações

Ao baixar um arquivo com o Firefox, o navegador da Web cria um arquivo PART na pasta "Downloads" no seu computador. Você pode ver o arquivo PART se o arquivo está baixando no momento ou se o download foi interrompido antes da conclusão. Uma interrupção de download pode ser devido a uma conexão perdida com a Internet, o navegador do Firefox ou o computador travando, ou para você pausar o download.

À medida que o Firefox baixa o arquivo, o programa cria um arquivo PART para armazenar os dados que estão sendo baixados. O Firefox substitui o nome do arquivo por uma sequência de caracteres e anexa a extensão .part à extensão do arquivo. Por exemplo, o arquivo example.mov teria o nome ZT5KR1EL.mov.part quando ele está sendo baixado. Uma vez que o download esteja concluído, o Firefox remove a extensão .part e o arquivo está pronto para ser usado.

Você também pode encontrar arquivos PART ao usar o Skype em uma Mac. Se um arquivo PART continuar reaparecendo enquanto usa o programa, mova o arquivo para o lixo e certifique-se de que é o único arquivo no lixo. Em seguida, clique com o botão direito do mouse no ícone "Lixo",.segure a tecla "Command", selecione Secure Empty Trash e verifique se o lixo está vazio. Agora, quando você reabrir o Skype, o arquivo PART não deve mais reaparecer
