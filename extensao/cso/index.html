O que é um arquivo .cso? (Arquivo de objeto Shader compilado)

Arquivo criado por arquivos .HLSL compilados escritos em High Level Shader Language (HLSL), que é uma linguagem de sombreamento desenvolvida pela Microsoft; contém um objeto shader compilado, que deriva de um arquivo HLSL; mantém o mesmo nome do arquivo HLSL, mas aparece como um arquivo CSO após a compilação no Microsoft Visual Studio.

Mais Informações

No SDK OpenGL, os arquivos CSO podem ser compilados usando o método glCompileShader. glCompileShader compila strings HLSL no objeto shader. Informações sobre se uma compilação foi ou não bem sucedida podem ser obtidas através do registro de informações do objeto shader ligando para glGetShaderInfoLog.

O que é um arquivo .cso? (Imagem compactada do disco ISO)

Arquivo de imagem .ISO em disco compactado usado para o arquivamento de jogos do PlayStation Portable (PSP) pode incorporar até nove camadas de compressão; criado por programas como Prometeus e YACC (Ainda outro CSO Compressor).

Mais Informações

NOTA: Níveis maiores de compressão podem fazer com que as imagens de disco CSO carreguem lentamente.
