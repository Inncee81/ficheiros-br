O que é um arquivo .swp? (Arquivo de troca)

Trocar arquivo usado pelo componente de memória virtual de um sistema operacional; contém dados que foram trocados da memória para o disco rígido; ajuda a aumentar a quantidade de memória disponível para um computador; também ajuda a tornar o acesso a dados comumente usados mais eficiente.

Mais Informações

Os arquivos SWP podem ser pré-alocados em uma seção de um disco rígido durante uma instalação do sistema operacional, ou podem ser criados sob demanda. Para pré-alocação, o sistema operacional pode aumentar a quantidade de memória virtual pelo tamanho do arquivo swap.

Muitos sistemas operacionais implementam um esquema de memória virtual chamado Demand Paging. Esse esquema armazena o arquivo de troca em "páginas," que são trocados do disco rígido para a memória sob demanda. A paginação da demanda também pode tentar antecipar quais segmentos de memória provavelmente serão usados em seguida e prefetch-los.

O que é um arquivo .swp? (Arquivo Vi Swap)

Arquivo de troca criado pelo editor de texto Vi ou uma de suas variantes como Vim (Vi iMproved) e gVim; armazena a versão de recuperação de um arquivo que está sendo editado no programa; também serve como um arquivo de bloqueio para que nenhuma outra sessão de edição vi possa ser simultaneamente escrita ao arquivo.

Mais Informações

Os arquivos SWP são criados imediatamente quando uma sessão de edição de texto Vi é iniciada e são salvos no mesmo diretório que seu arquivo original. Por exemplo, um arquivo de texto com o nome example.txt teria um arquivo de troca chamado .example.txt.swp.

Se uma sessão vi terminar devido a um programa de morte ou acidente, o arquivo SWP ainda permanece. Isso permite que os usuários recuperem os dados usando a opção "(R)ecover" ao abrir o arquivo original. No entanto, se você não precisar recuperar dados, você pode excluir manualmente o arquivo SWP para que a mensagem de aviso pare de aparecer.

Ao tentar abrir um arquivo que tenha um arquivo de swap, Vi fornece a seguinte mensagem: "Trocar arquivo .example.txt.swp já existe!" (onde example.txt é o arquivo sendo aberto). A partir deste diálogo, você pode escolher estas opções: 1) [O]pen Read-Only, 2) (E)dit, 3) (R)ecover, 4) (Q)uit, ou 4) (A)bort.

NOTA: O editor original da Vi foi desenvolvido por Bill Joy, co-fundador da Sun Microsystems.
