O que é um arquivo .fft? (Arquivo de impressão de ruído da Audição da Adobe)

Filtro de áudio usado pelo Adobe Audition, um aplicativo de produção de áudio; salva configurações que limitam as frequências de áudio para que certas partes da forma de onda possam ser isoladas ou pronunciadas; permite que os engenheiros ajustem o som dos dados de áudio.

O que é um arquivo .fft? (Arquivo de texto de formulário final)

Formato de documento de texto de formulário final (FFT) criado pela IBM para formatar texto que não pode ser modificado; inclui texto de documento, bem como dados gráficos; usado para corrigir o layout de páginas de documentos, cabeçalhos, rodapés e texto.

Mais Informações

Os arquivos FFT foram comumente criados pelo DisplayWrite, um programa ibm outrora popular usado para processamento de texto.
