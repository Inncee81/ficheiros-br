O que é um arquivo .sidd? (Arquivo de dados de autorização do iTunes)

Arquivo de dados usado pelo iTunes, um aplicativo de reprodução de música e vídeo; contém dados que autorizam o computador e o usuário a reproduzir músicas e vídeos comprados com a conta do usuário; semelhante a .SIDB e .SIDN arquivos; não devem ser abertos manualmente pelo usuário.

Mais Informações

Em Windows 7 e 8, o iTunes armazena arquivos SIDD para o seguinte diretório:

C:\ProgramData\Apple Computer\iTunes\SC Info\

iTunes para Windows XP usa este local:

C:\Documents and Settings\All Users\Application Data\Apple Computer\iTunes\SC Info\

NOTA: Se o iTunes não abrir depois de instalar uma nova versão em Windows, você pode corrigir isso excluindo o SC Info.sidb, SC Info.sidd, e SC Info.sidn arquivos dos diretórios listados acima. Após a exclusão, reinicie o computador e reabra o iTunes.
