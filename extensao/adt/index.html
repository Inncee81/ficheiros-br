O que é um arquivo .ADT? (ACT! Modelo de documento)

Um arquivo ADT é um modelo para criar um novo documento de processamento de palavras no software de gerenciamento de contatos ACT!. É usado para criar vários documentos com um layout semelhante.

Mais Informações

Os arquivos ADT podem ser criados com um processador de texto de terceiros, como o Microsoft Word, que funciona em conjunto com o ACT!. O local padrão para armazenar arquivos ADT é "C:Program FilesACTTemplates"

NOTA: Sage ACT! tornou-se Swiftpage Act! em 2013.

O que é um arquivo .ADT? (Arquivo de áudio ADTS)

Arquivo de áudio codificado no formato ADTS (Audio Data Transport Stream, fluxo de transporte de dados de áudio); contém uma série de quadros de áudio, cada um dos quais inclui uma parte dos dados de áudio globais a serem transferidos; comumente usados para streaming de áudio .AAC; também vistos com a extensão .ADTS.

O que é um arquivo .ADT? (Log de auditoria autocad)

Arquivo de registro gerado pelo AutoCAD, um programa 2D comum e CAD 3D; registra a atividade que ocorre durante uma auditoria de desenho.

Mais Informações

Para criar um arquivo ADT dentro do AutoCAD, siga estas etapas.

  1. Inicie uma auditoria de desenho (Menu de Arquivos→Serviços de Desenho→Auditoria).
  2. Na linha de comando, digite auditctl.
  3. Digite 1 (em vez do valor padrão de 0).
  4. Pressione a tecla Enter.
O registro de auditoria será criado no mesmo diretório do desenho atual com o nome de arquivo "drawingadt"

O que é um arquivo .ADT? (Arquivo de mapa do World of Warcraft)

Arquivo mapa usado por World of Warcraft (WoW), um jogo de RPG online massivamente multiplayer (MMORPG); contém um azulejo de mapa para um continente de jogo ou masmorra; usado para carregar terrenos e objetos do mapa enquanto joga o jogo.

Mais Informações

Os arquivos ADT estão incluídos dentro de arquivos .MPQ, que são usados para dados e atualizações de instalação de jogos. As atualizações do WoW podem incluir arquivos ADT para alterar o conteúdo do mapa.

Exemplos de nomes de arquivos ADT incluem Northrend_33_18.adt, Azeroth_29_47.adt e IcecrownCitadel_25_28.adt.
