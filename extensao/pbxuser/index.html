O que é um arquivo .pbxuser? (Arquivo de dados do usuário do projeto Xcode)

Arquivo criado pela Xcode, um aplicativo de desenvolvimento de software para Mac OS X; salva as configurações do usuário para o projeto de desenvolvimento, incluindo os arquivos abertos atualmente, configurações de janelas, preferências de compilação, diretórios de projetos e muitas outras propriedades; usado para lembrar as configurações locais.

Mais Informações

Como os arquivos PBXUSER contêm configurações de usuários locais, eles geralmente não são verificados em sistemas de controle de código-fonte para projetos de desenvolvimento colaborativo. No entanto, às vezes, um arquivo chamado default.pbxuser é verificado no repositório no início do projeto para que todos os desenvolvedores comecem com as mesmas preferências comuns em vez de ter que redefini-los cada um individualmente.

NOTA: O arquivo PBXUSER para um projeto Xcode é armazenado dentro do arquivo do projeto .XCODEPROJ. Para acessar este arquivo manualmente, remova a extensão "xcodeproj" do arquivo XCODEPROJ e abra a pasta resultante.
