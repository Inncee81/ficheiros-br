O que é um arquivo .pjpg? (Imagem JPEG progressiva)

Um arquivo PJPG é uma imagem raster salva no formato JPEG Progressivo. Ele contém dados codificados do bitmap que permitem que a imagem seja renderizada em várias varreduras de uma imagem embaçada para uma imagem nítida à medida que é baixada. Os arquivos PJPG permitem que programas de visualização de imagens mostrem visualizações antes que os dados finais da imagem seja recebido.

Mais Informações

O formato JPEG progressivo foi usado por alguns navegadores Web mais antigos, como versões mais antigas do Microsoft Internet Explorer. No entanto, agora o formato raramente é usado e suportado, uma vez que os links de comunicação estão se tornando muito mais rápidos.

Você pode converter a imagem PJPG em uma imagem .JPG renomeando a extensão "pjpg" para "jpg" e depois abrindo-a em um programa que suporta imagens JPG, como o Microsoft Photos em Windows ou o Apple Preview no macOS. Você pode visualizar a imagem e exportá-la para um formato, como JPEG ou PNG, que é mais amplamente suportado.
