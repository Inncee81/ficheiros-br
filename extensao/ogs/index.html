O que é um arquivo .ogs? (Arquivo de script Origin LabTalk)

Um arquivo OGS contém script LabTalk usado pela Origin, um aplicativo de análise e gráfico de dados usado por vários cientistas e engenheiros. Armazena comandos em texto simples que podem ser executados para realizar análise e grafia de dados. Os arquivos OGS podem ser usados para vários propósitos, incluindo o cálculo de problemas matemáticos simples e equações complexas de gráficos.

Captura de tela de um arquivo ogs no OriginLab Origin 2018

Você pode criar arquivos OGS com qualquer editor de texto. No entanto, os arquivos OGS são mais frequentemente criados usando Code Builder, que é uma ferramenta incluída com o Origin. Você pode acessar o Code Builder clicando no ícone de engrenagem amarela na barra de ferramentas padrão. Você também pode abrir arquivos OGS no Code Builder selecionando File → Open....

Outra maneira de abrir arquivos OGS é através da Janela de Script. Para acessar a janela de script, selecione Window → Script Window e selecione File(Text) → Open... na janela do script.

NOTA: Os arquivos OGS definidos pelo usuário são normalmente salvos na pasta "Arquivos do Usuário" na pasta do programa Origin.

O que é um arquivo .ogs? (Arquivo de dados do filme Origons)

Arquivo de dados usado pela Origons, um serviço online que permite que os usuários compartilhem listas de filmes entre si; pode conter informações sobre DVDs, vídeo DivX, fitas VHS e outros filmes de propriedade do usuário; permite que as informações do filme sejam baixadas e enviadas com o serviço.
