O que é um arquivo .csf? (Arquivo de configurações de cores do Adobe)

Um arquivo CSF é um arquivo de configurações de gerenciamento de cores usado pelo Adobe Photoshop, InDesign e Illustrator. Ele contém políticas para imagens coloridas, como definições de cores gráficas da Web, cores ColorSync, cores de prepress para diferentes países e outras configurações de cores. Os arquivos CSF são usados para obter consistência de cores entre imagens e impressões.

Mais Informações

NOTA: Os arquivos CSF podem ser compartilhados entre Photoshop, InDesign e Illustrator. Eles também podem ser gerenciados usando o aplicativo Adobe Bridge.

O que é um arquivo .csf? (Arquivo esqueleto binário Cal3D)

3D arquivo esqueleto criado usando software de animação de caracteres de código aberto Cal3D; armazena uma lista de ossos na forma de uma relação pai-filho; usado pelo componente de animação para criar movimentos de cabeça, tronco, perna, braço e mão.

Mais Informações

Arquivos esqueletos Cal3D também podem ser expressos em um formulário XML usando arquivos .XSF.

O que é um arquivo .csf? (Arquivo do sistema de coordenadas GeoMedia)

Coordenar arquivo do sistema usado pelo GeoMedia, um aplicativo GIS usado para analisar informações geográficas e produzir mapas; contém o sistema de coordenadas para um arquivo de dados de saída que não tem sistema em vigor; inclui o tipo de armazenamento base, que é geográfico, projeção ou geocêntrica, do sistema de coordenadas.

Mais Informações

Um arquivo CSF é usado para definir o sistema de coordenadas de dados de entrada sem sistema de coordenadas especificado. Ele é criado quando os dados de entrada são exportados para um formato de dados de saída, como o formato MicroStation V7. Quando é exportado, o arquivo CSF é gerado no mesmo local que os arquivos de saída.

NOTA: Para criar ou editar um arquivo CSF, use o utilitário "Definir arquivo do sistema de coordenadas" incluído no software GeoMedia.
