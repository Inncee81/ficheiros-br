O que é um arquivo .kcb? (Kindle Criar arquivo de livro)

Um arquivo KCB é um arquivo de projeto criado pelo Amazon Kindle Create, um programa usado para criar livros de qualidade profissional a serem publicados na loja Kindle. Contém informações sobre um livro, que inclui o conteúdo do livro, a tabela de conteúdo e informações de formatação de livros. Os arquivos KCB são usados para formatar livros a serem publicados na loja Kindle através da plataforma Kindle Direct Publishing.

Captura de tela de um arquivo kcb no Amazon Kindle Create 1

Kindle Create pode ser usado para publicar vários tipos de livros, como romances, ensaios, livros didáticos, guias de viagem, livros de receitas e coleções de poemas. Os arquivos KCB permitem que você salve livros, feche-os, eles os reassum para mais edição.

Para criar um arquivo KCB no Kindle Create, selecione o tipo de livro e, em seguida, importe o livro que você gostaria de publicar, que deve ser um arquivo .PDF, .DOCX ou .DOC. Para salvar um arquivo KCB no Kindle Create, clique em Save ou selecione File → Save Project ou Save Project As. abra o arquivo de trabalho do KCB que armazena seu livro e clique em Publique. Para abrir um arquivo KCB no Kindle Create, selecione File → Open Project.

Quando estiver pronto para publicar seu livro, clique em Publique ou selecione File → Publish para criar um arquivo .KPF publicável. Em seguida, carregue o arquivo KPF para o Kindle Direct Publishing para ser publicado na loja Kindle.
