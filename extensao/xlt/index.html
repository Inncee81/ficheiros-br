O que é um arquivo .xlt? (Modelo excel)

Um arquivo XLT é um modelo criado pelo Microsoft Excel, um aplicativo de planilha incluído no pacote Microsoft Office. Ele contém formatação padrão e dados para uma planilha e é usado como base para criar novos arquivos .XLS. Os arquivos XLT são salvos no Formato de Arquivo Binário excel, que era o formato principal até que o formato Office Open XML (.XLTX extensão de arquivo) o substituiu no Excel 2007 por Windows e 2008 por Mac.

XLT arquivos vêm instalados com o Excel permitindo que você crie documentos como planejadores de orçamento, calendários e faturas a partir de modelos com formatação pré-definida. Você também pode criar seus próprios modelos personalizados para reproduzir livros de trabalho ou planilhas específicas de suas necessidades.

Excel é o aplicativo de planilha mais utilizado no mundo. Ele vem com todas as edições do pacote Microsoft Office e é usado em ambientes domésticos, acadêmicos e de negócios.

NOTA: Embora os arquivos XLT tenham sido substituídos pelo arquivo XLTX, eles ainda são suportados pelo Excel 2007 ou posterior.
