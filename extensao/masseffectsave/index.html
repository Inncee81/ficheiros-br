O que é um arquivo .masseffectsave? (Jogo salvo de efeito em massa)

Arquivo de jogo salvo criado pelo Mass Effect, um jogo de RPG (sci-fi RPG) onde os jogadores completam missões como capitão de uma nave estelar; contém um estado de jogo salvo armazenado como um pacote Zip; usado para salvar e carregar o progresso no jogo; salvo para o diretório BioWare\Mass Effect\Save de uma instalação do Mass Effect.

Mais Informações

OS arquivos MASSEFFECTSAVE podem ser salvos automaticamente ou manualmente. Salvamentos automáticos usam a convenção de nomeação de arquivos profileXX_AutoSave.MassEffectSave, onde "profile" é o nome do perfil do jogador, e "XX" é o número da salvação. Da mesma forma, salvamentos rápidos normalmente usam a convenção profileXX_QuickSave.MassEffectSave.

Para visualizar manualmente o conteúdo de um arquivo MASSEFFECTSAVE, renomeie a extensão do arquivo para "zip" e descomprima-a com uma ferramenta de descompressão, como 7zip. Os dois arquivos descomprimidos resultantes, player.sav e state.sav, podem ser usados para modificar ("mod") conteúdo de jogo salvo.

NOTA: Os arquivos MASSEFFECTSAVE podem ser importados para o Mass Effect 2, a sequência do Mass Effect.
