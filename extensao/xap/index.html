O que é um arquivo .xap? (Pacote de aplicação Silverlight)

Pacote de aplicativos originalmente criado para o Microsoft Silverlight (um plug-in Web navegador que permite conteúdo de mídia rico) mas agora também usado para Windows aplicativos móveis Phone 7; inclui um arquivo .XAML chamado AppManifest.xaml e pode incluir .DLL referência de arquivos pelo software; armazenados no formato .ZIP.

Mais Informações

Arquivos XAP são comumente usados para distribuir aplicativos Windows Phone 7.

O que é um arquivo .xap? (Projeto XACT)

Projeto criado com a Ferramenta de Criação de Áudio Multiplataforma (XACT) da Microsoft); salvo em formato de texto e contém a forma intermediária das informações produzidas pela XACT; inclui descrições completas de bancos de ondas (arquivos .XWB), bancos de som (arquivos .XSB) e configurações globais (.XGS arquivos).

Mais Informações

Arquivos XAP podem ser abertos e visualizados em um editor de texto, mas não devem ser editados manualmente. Em vez disso, os arquivos do projeto só devem ser modificados com o XACT.
