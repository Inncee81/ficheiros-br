O que é um arquivo .udcx? (Arquivo universal de conexão de dados)

Arquivo de conexão de origem de dados originalmente usado pelo Microsoft FrontPage, mas agora adaptado para vários programas da Microsoft, incluindo Excel e InfoPath; contém propriedades para se conectar a uma ou mais fontes de dados; inclui uma string de conexão e informações de autenticação.

Mais Informações

Os arquivos UDCX são usados para armazenar e compartilhar informações de conexão de origem de dados em ambientes de negócios colaborativos. Por exemplo, os arquivos UDCX podem ser armazenados em um servidor Microsoft SharePoint da empresa para acessar fontes de dados comuns.

O formato Universal Data Connection (UDC) estende a especificação em arquivos .ODC para incluir suporte para vários tipos diferentes de fontes de dados, bem como tipos de conexão arbitrária. Os arquivos UDCX são salvos em um formato XML.

NOTA: A Microsoft atualizou o formato de arquivo UDC para a versão 20 para uso com aplicativos do Microsoft Office 2010, como o InfoPath, um aplicativo de coleta de informações e colaboração para empresas.
