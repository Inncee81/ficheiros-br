O que é um arquivo .pub? (Arquivo de chave pública)

Arquivo contendo texto criptografado que é usado em criptografia de chave pública; usado por uma entidade remota para criptografar dados que apenas uma entidade local pode descriptografar usando a chave privada correspondente; também pode ser usado para descriptografar dados criptografados com uma chave privada.

Mais Informações

Um arquivo PUB comum é id_rsapub. Os arquivos PUB normalmente podem ser encontrados sob o diretório oculto "SSH" sob o diretor doméstico de um usuário Linux.

O que é um arquivo .pub? (Documento do Editor)

Um arquivo PUB é um documento criado pelo Microsoft Publisher, um programa de publicação de desktop usado para criar documentos profissionais e materiais de marketing. Contém imagens de texto, raster e vetor e objetos desenhados formatados em um documento.

Mais Informações

Captura de tela de um arquivo de pub no Microsoft Publisher 2016

Publisher é usado por usuários casuais e profissionais para projetar layouts de páginas, como boletins informativos, folhetos e pôsteres. O programa só vem com edições do Office 365 do Microsoft Office, enquanto as edições básicas incluem apenas Word, Excel, PowerPoint e OneNote. Quando você salva um documento no Publisher, o programa cria um arquivo PUB para armazenar o conteúdo do documento. O arquivo permite que você salve edições em seu documento e permite que você feche o arquivo e abra-o novamente para mais edição. Os arquivos PUB não são usados apenas para layouts de página, eles também podem ser usados para integrar e armazenar fontes de dados para mesclagens de e-mail e gerar folhas de dados.

Se você tiver uma grande quantidade de texto em seu documento PUB que deseja reutilizar para outros fins, você pode exportá-lo para um Microsoft Word. A formatação de texto e fonte será salva, mas quaisquer gráficos ou colunas em seu documento não serão preservados. Para salvar seu arquivo PUB como um arquivo DOCX selecionar File → Save As, escolha o formato de documento do Word que deseja salvar o documento como e clique em Save.

ARQUIVOS PUB também podem ser abertos por vários outros aplicativos. Alguns notáveis são o LibreOffice Draw, que é embalado em LibreOffice e CorelDRAW, que faz parte do CorelDRAW Graphics Suite. Os arquivos PUB também podem ser abertos no Adobe InDesign, um dos programas de publicação de desktop mais populares, com o plugin PUB2ID instalado.
