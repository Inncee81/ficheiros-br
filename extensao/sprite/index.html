O que é um arquivo .SPRITE? (Arquivo Scratch Sprite)

Arquivo gráfico usado pela Scratch, um programa educacional usado para desenvolver habilidades criativas de programação; salva uma imagem bidimensional, bem como configurações de animação para a imagem; comumente usado para personagens e objetos de jogo.

Mais Informações

Arquivos SPRITE também podem ser usados por variantes scratch, como BYOB (Build Your Own Blocks).

NOTA: A liberação do Scratch 20 introduziu o arquivo .SPRITE2, que substituiu o arquivo SPRITE.

O que é um arquivo .SPRITE? (Arquivo SuperTux Sprite)

Sprite referenciado pelo SuperTux, um jogo de videogame sidecroller semelhante ao Super Mario Bros; formatado como um arquivo de texto e contém um conjunto de ações para um personagem, incluindo referências a arquivos de imagens que compõem o sprite.

Mais Informações

Sprites são usados para exibir personagens e suas animações associadas. As ações dentro do arquivo sprite são executadas quando o usuário insere entrada (como andar para a frente ou virar à direita) ou quando certos eventos acontecem durante a jogabilidade (como o jogador sendo atingido pelo caractere do computador). Usando sprites, imagens simples de bitmap podem ser usadas para executar todas as animações dentro do jogo.
