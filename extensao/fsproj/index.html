O que é um arquivo .fsproj? (Arquivo de projeto Visual F#)

F# arquivo de projeto de desenvolvedor criado pela Visual F#, um componente do kit de desenvolvimento de software Visual Studio da Microsoft; armazena configurações de projeto em um formato XML; contém referências a arquivos de código-fonte, bibliotecas incluídas e outros ativos do programa.

Mais Informações

Os arquivos FSPROJ são usados para armazenar projetos F# à medida que são desenvolvidos. Eles podem ser usados para construir código-fonte do programa e ativos em aplicativos executáveis.
