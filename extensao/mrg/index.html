O que é um arquivo .MRG? (Arquivo de mesclagem MySQL)

Arquivo de banco de dados usado pelo MySQL, um programa de banco de dados relacional gratuito de código aberto; contém os nomes das tabelas que são usadas como uma única tabela em um esquema de tabela de mesclagem MySQL; criado pelo mecanismo de armazenamento MySQL MERGE.

Mais Informações

Os arquivos MRG são salvos com um arquivo .FRM que contém a estrutura para as tabelas na tabela de mesclagem.

O que é um arquivo .MRG? (Agora, o modelo de mesclagem de contato)

Mesclar modelo criado pelo Now Contact, um aplicativo que organiza informações de contato e atividades do dia-a-dia; contém um modelo de layout de impressão para o documento da carta; as opções pré-carregadas incluem "Carta de Fantasia", "Carta de Flores," ""Carta de Swoosh", "Carta de Navio"

Mais Informações

Para criar o arquivo MRG, selecione Define → Print Templates → Faxes..., selecione o botão "+" no canto superior direito, nomeie o modelo e clique em OK. Em seguida, você poderá editar o layout do modelo e suas dimensões. Clique em OK quando terminar.

Ao criar um novo documento de letra .NWP, você pode selecionar um modelo MRG para trabalhar fora, escolhendo um modelo no menu suspenso "Use Template".

Para selecionar o modelo que deseja imprimir, selecione File → Print Templates..., escolha seu modelo no menu "Tipo de modelo"," e clique em OK.
