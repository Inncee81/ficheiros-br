O que é um arquivo .har? (ARQUIVO HTTP FIle)

Um arquivo HAR é um arquivo salvo no formato HTTP Archive (HAR), que é um formato JSON usado para salvar dados de desempenho HTTP coletados. Ele contém informações sobre páginas da Web que um navegador da Web carregou, o que inclui as páginas da Web rastreadas, os tempos de resposta e a versão do navegador da Web. Os arquivos HAR também armazenam metadados, como a versão do formato HAR e o criador do arquivo.

O objetivo do formato HAR é registrar o desempenho dos navegadores da Web, o que permite que os desenvolvedores analisem o quão bem os navegadores estão carregando páginas da Web. Os arquivos HAR podem ser exportados por várias ferramentas relacionadas ao HTTP, como Electron HAR, HttpWatch, HTTP Toolkit, mas normalmente são exportados por navegadores da Web. A maioria dos navegadores suporta o formato, incluindo Google Chrome, Mozilla Firefox, Apple Safari e Microsoft Edge e Internet Explorer.

Você pode abrir arquivos HAR com vários programas, incluindo a ferramenta HAR Viewer on-line e o open source, multiplataforma HTTP Toolkit. Uma vez que os arquivos HAR são salvos no formato JSON, você também pode abri-los usando um editor JSON ou um editor de texto simples, como Microsoft Notepad ou Apple TextEdit.

Se você abrir um arquivo HAR em um editor de texto, você verá o conteúdo no formato JSON organizado nos seguintes tipos de objeto:

  • log - Inclui informações sobre a versão do formato HAR, criador de log, versão do navegador e lista de páginas da Web rastreadas.
  • creator - Inclui informações sobre o criador do registro.
  • browser - Inclui informações sobre o navegador.
  • pages - Inclui informações sobre a lista de páginas da web exportadas.
  • pageTimings - Inclui informações sobre os eventos disparados durante a carga de página em milissegundos.
  • entries - Inclui informações sobre todas as solicitações HTTP.
  • request - Inclui informações sobre solicitações realizadas, incluindo o método de solicitação, objetos de cookie e cabeçalho e objetos de parâmetro de consulta.
  • response - Inclui informações sobre o status de resposta, descrição do status, versão HTTP e objetos de cookie e cabeçalho.
  • cookies - Inclui informações sobre todos os cookies request e response objetos.
  • headers - Inclui informações sobre todos os cabeçalhos nos objetos request e response.
  • queryString - Inclui informações sobre os parâmetros e valores analisados a partir de uma sequência de consulta.
  • postData - Inclui informações sobre quaisquer dados publicados no objeto request.
  • params - Inclui informações sobre quaisquer parâmetros publicados no objeto postData.
  • content - Inclui informações sobre conteúdo de resposta no objeto response.
  • cache - Inclui informações sobre uma solicitação do cache do navegador.
  • timings - Inclui informações sobre os tempos de solicitação e resposta em milissegundos.

NOTA: O formato HAR é desenvolvido pelo Grupo de Trabalho de Desempenho Web do Consórcio Web Mundial (W3C).
