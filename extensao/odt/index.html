O que é um arquivo .odt? (Arquivo de tabela de dados OOMMF)

Um arquivo ODT contém dados no texto ASCII e é criado pela OOMMF (Object Oriented Micromagnetic Framework), que é um projeto desenvolvido pelo Instituto Nacional de Padrões e Tecnologia (NIST). Contém comentários e dados micromagnéticos organizados em tabelas, com cada linha de não-dados começando com #.

Mais Informações

Você pode abrir arquivos ODT no OOMMF, que exibe os dados nas tabelas. Você também pode usar um editor de texto para abrir e editar o arquivo, uma vez que ele é salvo em texto simples.

O que é um arquivo .odt? (Documento de texto openDocument)

Um arquivo ODT é um documento de texto criado por vários processadores de texto, como o programa Writer incluído no Apache OpenOffice e no LibreOffice. Contém diferentes elementos, como texto, imagens, objetos desenhados e estilos. Os arquivos ODT são formatados usando o padrão baseado em OASIS OpenDocument XML.

Mais Informações

Captura de tela de um arquivo odt em Apache OpenOffice Writer 413

Os arquivos ODT são comumente criados e abertos pelo processador de texto Writer, que é um dos vários programas disponíveis nas suítes OpenOffice e LibreOffice. É semelhante ao programa do Word disponível no pacote Microsoft Office. O arquivo ODT é o arquivo principal usado pelo Writer para salvar documentos da mesma forma que o Word salva documentos no arquivo .DOCX.

Arquivos de texto OpenDocument podem ser abertos e editados com qualquer programa compatível com OpenOffice, incluindo: NeoOffice (Mac), AbiWord (Mac & Windows) e KWord (Unix). Eles também podem ser importados para o Google Docs, processador de texto baseado na Web do Google incluído no Google Drive. Os arquivos ODT também podem ser abertos no Microsoft Word e salvos como arquivos DOCX.

NOTA: O Apache OpenOffice era anteriormente conhecido como OpenOfficeorg.

Desenvolvedor OriginLab
Categoria Configurações
Formato Não Disponível 

O que é um arquivo .odt? (Arquivo tema de diálogo de origem)

Um arquivo ODT é um arquivo temático usado pela Origin, um aplicativo de análise e gráfico de dados usado por vários cientistas e engenheiros. Ele contém configurações em formato XML que personalizam a aparência de uma ou mais janelas de diálogo que aparecem no Origin.

Mais Informações

Os arquivos temáticos do ODT não devem ser abertos por um usuário. Em vez disso, as configurações de arquivos que são referenciadas pela Origin para compor a aparência de janelas de diálogo no software.

NOTA: Os arquivos ODT estão tipicamente localizados na pasta "Diálogo" da instalação de C:\Program Files\OriginLab\Origin<version>\Themes.
