O que é um arquivo .IFF? (Formato de arquivo de intercâmbio)

Um arquivo IFF é um formato de arquivo universal criado pela Electronic Arts. Ele pode conter dados de imagem, texto ou áudio e é usado para intercodestrar diferentes tipos de dados entre aplicativos e plataformas. Os arquivos IFF são suportados por muitos programas e são usados como base para vários outros formatos de arquivo, incluindo AIFF.

Mais Informações

Os arquivos IFF são compostos por seções de dados chamadas "chunks"," que são definidas por IDs de quatro letras. Existem três tipos principais de pedaços, cada um dos quais pode conter texto, dados numéricos ou dados brutos:

  • FORM: especifica o formato do arquivo
  • LIST: inclui as propriedades do arquivo
  • CAT: contém o resto dos dados
O formato IFF também é conhecido como "EA IFF 1985" desde que a Electronic Arts projetou o formato de arquivo em 1985.

NOTA: Os arquivos IFF usados pelos Sims e The Sims Online podem ser editados usando o IFF Snooper para a plataforma Mac ou IFF Pencil for Windows.
