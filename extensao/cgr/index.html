O que é um arquivo .cgr? (Arquivo de representação gráfica CATIA)

ARQUIVO CAD criado pelo CATIA, um aplicativo CAD usado para design de produto; salva a visualização de um design (.CATPART ou .CATPRODUCT), mas armazena apenas as informações geométricas necessárias para visualizar o modelo em vez de todos os dados CAD.

Mais Informações

Arquivos CGR são usados para enviar rascunhos de projeto para revisão. Eles também podem ser inseridos em documentos .PDF da Adobe usando a versão Pro Extended do Adobe Acrobat.

O que é um arquivo .cgr? (Arquivo do grupo de canal Quest3D)

Arquivo criado pelo Quest3D, um mecanismo usado para criar programas 3D que agora está descontinuado; contém um grupo de "canais," ou propriedades para o fluxo e exibição de objetos, modelos, superfícies, texturas, propriedades e animações; usado para armazenar visualizações de produtos, jogos, tutoriais, animações, interfaces de usuário e ambientes imersivos.

Mais Informações

Os arquivos CGR são sempre armazenados com complementação de arquivos .IGR, que contêm a localização de canais dentro do painel de layout do Gráfico de Canal Quest3D. Se os arquivos IGR estiverem faltando, os canais usarão um layout padrão e podem ser difíceis de visualizar.

NOTA: Arquivos CGR são usados em jogos como Audiosurf, Leo e The Endless Forest.
