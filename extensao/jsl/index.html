O que é um arquivo .jsl? (Biblioteca de formas Corel PaintShop Pro Shape)

Um arquivo JSL contém uma biblioteca de moldes criada pela versão 7 do PaintShop Pro, um programa usado para edição de imagens e fotos digitais. Ele armazena uma ou mais formas de predefinição vetorial, como um círculo, quadrado, triângulo, bolha de bate-papo ou borboleta.

Mais Informações

Os arquivos JSL foram substituídos por arquivos PSPSHAPE na versão 8 do PaintShop Pro. No entanto, eles ainda são suportados pela versão 8 e 9 do PaintShop Pro. Para importar uma biblioteca de formas JSL no PaintShop Pro 7, selecione File → Import → Shape. Para exportar uma biblioteca de formas JSL no PaintShop Pro 7, selecione File → Export → Shape ou clique com o botão direito de clicar em uma imagem e escolha Export → Shape no menu de contexto.

O que é um arquivo .jsl? (Roteiro JMP)

Um arquivo JSL é um script criado pelo SAS JMP, um software de análise de dados frequentemente usado por engenheiros e cientistas. Ele contém JMP Scripting Language (JSL) que pode ser executado para recriar resultados de dados, estender a funcionalidade do software e automatizar análises. Os arquivos JSL também podem incluir metadados, como o autor, informações de contato do autor, data criada e o propósito do script.

Mais Informações

Captura de tela de um arquivo jsl em SAS JMP 143

Para abrir um script JSL no JMP selecione File → Open..., navegue até a localização do arquivo JSL e clique em Abra. Para editar um script JSL no JMP, clique à direita no script depois de abri-lo no painel esquerdo da interface do programa e selecione Edit Script. Para executar um script JSL em JMP, clique à direita no script depois de abri-lo no painel esquerdo da interface do programa e selecione Run Script.

Desde que os arquivos JSL armazenam a linguagem de script em formato de texto simples, eles podem ser abertos e editados com editores de texto. Alguns exemplos de programas que podem ser usados para abrir e editar arquivos JSL incluem o Microsoft Notepad e o Apple TextEdit.

Você provavelmente só encontrará um arquivo JSL se você trabalhar em um campo que requer análise de dados. O software JMP vem com uma variedade de scripts JSL pré-fabricados que podem ser modificados. Os scripts JSL de amostra incluídos com JMP podem ser encontrados na pasta "Scripts" na pasta "Samples" incluída no diretório de instalação do JMP.

Os usuários também podem criar seus próprios scripts do zero. Alguns exemplos de usos dos scripts incluem conectar-se a bancos de dados para recuperar dados ou automatizar a modificação de campos de dados em uma tabela.
