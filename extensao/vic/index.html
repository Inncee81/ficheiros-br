O que é um arquivo .vic? (Imagem do VIGÁRIO)

Imagem criada no formato VICAR (Video Image Access and Retrieval), que foi originalmente desenvolvido para dados de imagem de missão planetária; contém um cabeçalho de texto ASCII e uma ou mais imagens salvas como registros de dados binários; mais conhecido por armazenar imagens capturadas de sistemas de missão planetária.

Mais Informações

As imagens do VICAR foram projetadas para a espaçonave planetária não tripulada do Jet Propulsion Laboratory (JPL). Eles foram usados durante as 1970 e 1980 para as seguintes missões: Mariner 9 para Marte, Mariner 10 para Mercúrio e Vênus, Voyager para os planetas exteriores.

As imagens do VIGÁRIO também são usadas para aplicações de cartografia, biomédica, geologia e astronomia.

O que é um arquivo .vic? (Arquivo de mapeamentos de importação de código vizigen)

Contém mapeamentos de importação usados pela ViziGen, um programa de desenvolvimento de software para geração de arquivos de código-fonte para qualquer linguagem de programação; controla como as informações de uma ferramenta de modelagem UML são importadas para o ViziGen.
