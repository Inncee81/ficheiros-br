O que é um arquivo .pmo? (Arquivo de meta objeto de impressão broderbund)

Arquivo criado pelo PrintMaster, um programa usado para criar cartões de saudação, folhetos, etiquetas de CD e outras impressões; contém todas as informações do projeto, incluindo gráficos, layouts e outras configurações; usado para transferir projetos entre programas dentro do conjunto de criatividade Broderbund.

Mais Informações

Arquivos PMO podem ser exportados no PrintMaster selecionando File → Export As.

O que é um arquivo .pmo? (Arquivo de mensagem salvo da Pegasus)

Arquivo criado pelo Pegasus Mail, um programa de e-mail; contém uma mensagem de e-mail salva em formato de texto simples; usada para mensagens de e-mail ou rascunhos de mensagens manuais e salvas automaticamente.
