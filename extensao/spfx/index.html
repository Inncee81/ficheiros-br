O que é um arquivo .spfx? (Arquivo Predefinições de espremer)

Arquivo De configurações usado pelo Squeeze, um aplicativo de transcodificação de vídeo; salva valores predefinidos para realizar uma conversão de vídeo; muitas vezes usado para compartilhar predefinições com outros através do site da Sorenson Media Presets Exchange.

Mais Informações

Presets permitem que os usuários salvem configurações para conversões comuns. Por exemplo, uma predefinição pode incluir os parâmetros necessários para converter qualquer vídeo importado em um formato adequado para o YouTube, um iPhone, um PlayStation 3 ou um Xbox 360.

As predefinições SPFX podem ser importadas selecionando Preset → Import... no menu de aplicativos Squeeze. As predefinições podem ser exportadas escolhendo Preset → Export. Os arquivos são salvos usando formatação XML, para que você possa visualizá-los em qualquer editor de texto.

Você pode ir diretamente ao site da Sorenson Media Presets Exchange escolhendo Preset → Get More Presets no menu do aplicativo Squeeze.

NOTA: O Sorenson Squeeze foi descontinuado em 2018.
