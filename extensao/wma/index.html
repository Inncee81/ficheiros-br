O que é um arquivo .wma? (Arquivo de áudio de mídia Windows)

Um arquivo WMA é um arquivo de áudio salvo no formato proprietário DO Advanced Systems Format (ASF) desenvolvido pela Microsoft. Contém Windows Media Audio e objetos de metadados, como o título, artista, álbum e gênero da faixa. Os arquivos WMA são semelhantes aos arquivos .MP3 e usados principalmente para streaming de música da web.

ASF é o formato de contêiner de áudio e vídeo digital proprietário da Microsoft. Se ele contiver Windows Media Video em vez de Windows Media Audio, ele usará a extensão .WMV. Os arquivos WMA e WMV são os mesmos de .ASF arquivos, mas são mais comumente usados.

O Microsoft Windows Media Player 9 foi a última versão do Windows Media Player desenvolvido para Mac OS X. No entanto, Mac usuários podem usar o Flip4Mac WMV (também conhecido como Microsoft Windows Media Components for QuickTime) para reproduzir arquivos WMA.
