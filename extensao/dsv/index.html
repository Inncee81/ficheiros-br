O que é um arquivo .dsv? (Arquivo de mapeamento UV do Estúdio DAZ)

Arquivo criado pelo DAZ Studio, um aplicativo de modelagem de caracteres 3D; contém dados de mapa UV para objetos na cena; referenciado pelo arquivo de cena 3D .DAZ e usado com arquivos de .DSO e .DSD para compor a cena.

Mais Informações

NOTA: O mapeamento UV é o processo pelo qual texturas, ou gráficos 2D, são projetadas em superfícies 3D. Este processo é usado na aplicação de gráficos em malhas esqueléticas para que as figuras assumam a aparência de uma superfície sólida pintada.

O que é um arquivo .dsv? (Arquivo de salvamento DeSmuME)

Jogo salvo criado com DeSmuME, um emulador do Nintendo DS; salva o progresso do jogo em um ponto específico dentro do jogo para que ele possa ser retomado posteriormente; salvo pelo emulador, o que significa que o arquivo de salvamento pode ser criado a qualquer momento, não apenas nos pontos de salvamento dentro de cada jogo.

Mais Informações

NOTA: Alterar a extensão do arquivo de "dsv" para "sav" pode permitir que um arquivo DSV seja aberto por outros emuladores do Nintendo DS.
