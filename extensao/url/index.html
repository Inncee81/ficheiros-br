O que é um arquivo .url? (Atalho da Internet)

Um arquivo URL é um arquivo de atalho referenciado por navegadores da Web, como Microsoft Internet Explorer (MSIE), Apple Safari e Google Chrome. Ele contém uma URL da Web e também pode armazenar uma referência ao arquivo favicon.ico ícone, que é exibido como o ícone para o arquivo de atalho.

Mais Informações

Arquivos de URL abrem um local da Internet, como um endereço web, no navegador padrão do usuário. Normalmente, o local do atalho pode ser visualizado abrindo o arquivo em um editor de texto, como Microsoft Notepad ou Apple TextEdit.

Para criar um arquivo URL em Windows, basta arrastar o endereço URL da janela do seu navegador para a área de trabalho. Ele criará um arquivo URL que abrirá o endereço quando clicado duas vezes. Se você arrastar o endereço URL para sua área de trabalho em uma Mac, ele criará um arquivo .WEBLOC.

Aqui está um exemplo de dados contidos em um arquivo URL:
[{000214A0-0000-0000-C000-0000000000046}]
Prop3=19,2
[InternetShortcut]
URL=[Endereço de URL]
IDList=

NOTA: O microsoft Windows não exibe a extensão do arquivo "url" mesmo que exista no nome do arquivo. Portanto, os arquivos de URL salvos usando Windows navegadores da Web aparecerão apenas com o prefixo de nome de arquivo.
