O que é um arquivo .deskthemepack? (Windows arquivo de pacote temático de desktop de 8 de Windows)

Um arquivo DESKTHEMEPACK é um arquivo temático de desktop introduzido com o sistema operacional Windows 8. Ele é salvo no formato .CAB e contém imagens de fundo (.JPG arquivos) bem como um arquivo .THEME, que contém cores, sons e outras configurações usadas para personalizar o Windows desktop. Os arquivos DESKTHEMEPACK podem ser baixados e compartilhados com outros usuários Windows.

Mais Informações

Você pode instalar um arquivo DESKTHEMEPACK em Windows 8 clicando duas vezes nele. Você também pode criar um tema personalizado clicando com o botão direito do mouse em um dos arquivos "Meus Temas" no painel de controle de personalização Windows e selecionando "Salvar o tema para compartilhar"

DESKTHEMEPACK arquivos são específicos para Windows 8 e não são compatíveis com Windows 7, que usa .THEMEPACK arquivos para temas de desktop. No entanto, você pode usar as imagens contidas em um arquivo DESKTHEMEPACK como seu fundo de desktop em Windows 7 seguindo estas etapas:

  1. Altere a extensão do arquivo "deskthemepack" para "cab"
  2. Clique duas vezes no arquivo "cab" para visualizar seu conteúdo.
  3. Copie os arquivos "jpg" da pasta para outro local no computador.
  4. Abra a janela De personalização clicando com o botão direito do mouse na área de trabalho Windows e selecionando "Personalize" no menu suspenso.
  5. Na janela Personalização, escolha o link "Plano de Fundo de Trabalho".
  6. Em "Escolha seu plano de fundo da área de trabalho," clique no botão Browse. e selecione a pasta onde salvou os arquivos de imagem "jpg" .
  7. Clique no botão Salve alterações e as imagens serão aplicadas no seu desktop.

NOTA: Com o lançamento de Windows 8, a Microsoft atualizou a Galeria de Personalização online para fornecer arquivos DESKTHEMEPACK, além de arquivos THEMEPACK. Você pode ver esta galeria clicando em "Obtenha mais temas online" da janela Personalização em Windows 7 e Windows 8.
