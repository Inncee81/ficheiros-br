O que é um arquivo .lang? (Arquivo de definição de idioma gtkSourceView)

Um arquivo LANG é um arquivo de configuração salvo no formato XML de definição de linguagem. Ele é usado para adicionar suporte de destaque de sintaxe para uma linguagem específica no widget GtkTextView incluído no kit de ferramentas GTK, que é um conjunto de widgets usados para criar interfaces gráficas de usuário (GUIs). Os arquivos LANG contêm informações no formato XML que normalmente incluem definições, estilos e metadados.

Mais Informações

Captura de tela de um arquivo lang no Microsoft Visual Studio Code 1

GTK é uma ferramenta multiplataforma de código aberto usada para criar GUIs para programas. Consiste em vários widgets que são usados para projetar a GUI. Um dos widgets é o GtkTextView, que é usado para exibir e editar texto multiline na GUI. Os recursos do widget GtkTextView podem ser estendidos pela biblioteca GtkSourceView. Alguns recursos que podem ser estendidos pela biblioteca incluem carregamento e salvamento de arquivos, pesquisa e substituição e recursos de destaque de sintaxe.

O suporte para o destaque da sintaxe pode ser adicionado para idiomas específicos usando arquivos LANG na biblioteca GtkSourceView. Algumas linguagens que podem ser adicionadas incluem linguagens de programação C, C++, Java, Fortran, Logtalk e Axe Parser, linguagens de marcação XML, HTML e LaTex, e linguagens de scripts PHP, Python e Perl.

Você provavelmente só encontrará um arquivo LANG se você for um desenvolvedor GTK, já que eles não são usados por outros programas. Se você quiser visualizar ou editar um arquivo LANG, você pode usar um texto simples ou editor XML, como Microsoft Notepad, Apple TextEdit ou Microsoft Visual Studio Code.

NOTA: GTK era anteriormente conhecido como GIMP Toolkit e GTK+.

O que é um arquivo .lang? (Arquivo de linguagem minecraft)

Um arquivo LANG é um arquivo de recursos de jogo usado pelo Minecraft, um jogo aberto que permite que os jogadores criem e explorem mundos 3D. Ele salva uma lista de pares de strings de valor-chave que mapeiam menus de jogos, conquistas e outros objetos de jogo para nomes em uma determinada linguagem. Os arquivos LANG são usados para localização.

Mais Informações

Os arquivos DO MINECRAFT LANG estão localizados dentro do arquivo minecraft.jar, que está localizado no seguinte diretório:

[user]\AppData\Roaming\.minecraft\bin\

OS arquivos LANG são armazenados em um formato de texto simples, para que você possa editá-los com um editor de texto. No entanto, isso pode potencialmente fazer com que minecraft não seja executado corretamente, então você deve fazer backup minecraft.jar antes de prosseguir. Para editar arquivos LANG, siga estas etapas:

  1. Explorar minecraft.jar com 7-Zip ou outra ferramenta que possa explorar arquivos JAR.
  2. Navegue até a pasta \lang\.
  3. Extraia en_US.lang e salve-a em um local em seu computador.
  4. Edite o arquivo LANG com um editor de texto como o Bloco de Notas.
  5. Coloque o arquivo LANG editado de volta no arquivo minecraft.jaf, substituindo o anterior.
  6. Exclua a pasta \META-INF\ no diretório raiz de minecraft.jar.
  7. Reinicie Minecraft.

O que é um arquivo .lang? (Arquivo de idioma skype)

Arquivo de idioma usado pelo Skype, um popular programa de comunicação online; contém informações de idioma que alteram a linguagem da interface do usuário do Skype.

Mais Informações

Os arquivos LANG podem ser carregados no Skype selecionando a opção "Ferramentas→Alterar language". Eles também podem ser editados selecionando "Tools→Alterar language→Editar Arquivo de Linguagem do Skype"
