O que é um arquivo .pbf? (Arquivo de backup paragon)

Arquivo de backup criado por um dos aplicativos de backup e recuperação da Paragon; armazena arquivos de backup em um formato proprietário; pode ser navegado dentro do software, que permite que os usuários restaurem arquivos individuais ou todo um sistema de arquivos.

Mais Informações

Além dos backups do sistema, os arquivos PBF são frequentemente usados para migrar sistemas de arquivos Windows entre versões de sistemas operacionais Windows.

Arquivo de dados PowerBackup

Desenvolvedor CyberLink
Categoria Cópia de Segurança
Formato Binário 

O que é um arquivo .pbf? (Arquivo de dados PowerBackup)

Arquivo de backup criado pelo CyberLink PowerBackup, um aplicativo de backup e recuperação; armazena uma coleção de arquivos e pastas em backup; pode ser usado para estratégias completas, diferenciais ou incrementais de backup; usado para restaurar arquivos através do aplicativo PowerBackup.

Mais Informações

O PowerBackup pode criar backups periodicamente através de seu componente de agendador de trabalho, que usa .PBJ arquivos de trabalho.

Programas que abrem o arquivo .pbf (Arquivo de dados PowerBackup)

  Windows
  CyberLink PowerBackup
