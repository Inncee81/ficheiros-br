O que é um arquivo .SPUB? (Arquivo de capa de disco)

Etiqueta de CD/DVD ou inserção de caixa de joias criada com software de capa de disco; permite que capas e etiquetas para CDs e DVDs sejam criados de forma fácil e precisa; Os arquivos SPUB também podem ser modelos de rótulo ou capa incluídos no programa de software.

O que é um arquivo .SPUB? (Documento do Editor Swift)

Um arquivo SPUB contém um documento criado pelo Swift Publisher, um programa de layout de página para macOS. Ele armazena informações sobre o documento em um formato proprietário, que inclui as configurações do layout da página e elementos de documentos, como texto, imagens e formas no documento. Os arquivos SPUB podem ser usados para salvar vários tipos de documentos, incluindo panfletos, boletins informativos, anúncios, folhetos, calendários, pôsteres e cartões de visita.

Captura de tela de um arquivo spub em BeLight Swift Publisher 5

Para criar documentos do Swift Publisher, você pode começar com um dos muitos modelos fornecidos com o software ou criar um do zero. Para criar um arquivo SPUB no Swift Publisher, selecione File → New... e escolha um documento em branco ou um modelo para criar. Depois de modificar o documento, selecione File → Save para criar o arquivo SPUB.

Para abrir um arquivo SPUB no Swift Publisher, selecione File → Open... ou Open Recent. Para converter um arquivo SPUB em outro formato, como .PDF, .EPS, .TIFF, .PNG ou .JPG, selecione File → Export... no Swift Publisher.
