O que é um arquivo .mkd? (Arquivo de documentação de marcação)

Um arquivo MKD é um arquivo de texto criado usando um dos vários dialetos possíveis do idioma Markdown. Ele é salvo em formato de texto simples, mas inclui símbolos de texto inline que definem como formatar o texto (por exemplo, negrito, recuos, cabeçalhos, formatação de tabela). Os arquivos MKD também podem ser salvos à medida que .MD, .MARKDOWN ou .MDOWN arquivos

Mais Informações

Arquivo MKD são usados para autor de documentação de texto simples que pode ser facilmente convertida em HTML. No entanto, eles não são apenas úteis para sistemas de documentação HTML, mas também para o controle de versão de código-fonte. Os arquivos MKD podem ser comparados com revisões históricas em textos legívels por humanos, enquanto um formato binário não pode ser facilmente comparado.
