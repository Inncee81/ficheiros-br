O que é um arquivo .api? (Arquivo adobe photoshop inks)

Configurações do arquivo usado pelo Photoshop, um programa de edição de gráficos raster; armazena configurações de cores para cores de tinta que são usadas para imprimir imagens; permite que os usuários salvem e carreguem configurações de cores calibradas, o que é útil ao estabelecer propriedades de impressão para uma impressora específica de jato de tinta ou laser.
