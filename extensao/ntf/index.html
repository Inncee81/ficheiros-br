O que é um arquivo .ntf? (Modelo de nota lotus)

Arquivo usado pela Lotus Notes, um programa de software de colaboração de negócios frequentemente usado por grandes empresas; criado por desenvolvedores do Lotus Notes e usado para armazenar módulos personalizados do Lotus Notes que podem se integrar à maior infraestrutura do Lotus Notes.

Mais Informações

Arquivos NTF podem conter formulários, código-fonte e outros dados usados pelo aplicativo.

O que é um arquivo .ntf? (Arquivo de etiqueta de CD MediaFACE II)

Selo de CD criado pelo MediaFACE 2, um programa usado para criar etiquetas de disco imprimíveis; armazena informações de texto, gráficos e formatação; pode ser impresso em um CD ou DVD com uma impressora compatível.

Mais Informações

NOTA: Os arquivos NTF não são usados por versões posteriores do MediaFACE.
