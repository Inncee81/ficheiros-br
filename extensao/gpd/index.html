O que é um arquivo .gpd? (Arquivo de dados de perfil gamer)

Um arquivo GPD é um arquivo de dados de perfil gamer usado pelo Xbox 360 para armazenar informações sobre um usuário, incluindo suas configurações, jogos jogados, registros de conquistas para cada jogo jogado e quaisquer imagens de realização. Ele é salvo no Formato de Banco de Dados Xbox (XDBF), que é um formato de contêiner para armazenar arquivos e registros do Xbox. Os arquivos GPD são gerados a partir de arquivos De Estatísticas, Presença e Realização (SPA) incorporados em pacotes de jogos quando um usuário joga o jogo.

Mais Informações

Você pode mod um jogo editando o arquivo GPD associado ao jogo, comumente Black Ops ou Borderlands 2. Isso permite que você desbloqueie recursos de jogabilidade que você anteriormente não tinha. Para substituir ou injetar um arquivo GPD, mova seu perfil de um HDD Xbox 360 para uma unidade flash USB e conecte a unidade a um computador em execução Windows com as ferramentas de modding Horizon ou Modio instaladas:

  • Se você estiver usando o Horizon, selecione "Device Explorer" e navegue até a pasta Perfis Gamer na unidade flash. Em seguida, você pode clicar com o botão direito do mouse no seu perfil, selecionar Extract File... e escolher o local de salvamento. Em seguida, arraste e solte seu perfil na área cinza aberta, que abre a ferramenta "Gerenciador de pacotes". Clique na guia "Conteúdo" para exibir todos os arquivos GPD dentro do seu perfil.

  • Se estiver usando o Modio, clique no logotipo "M" no canto superior esquerdo e selecione "Explorar meu dispositivo" Na janela Explorador de dispositivos selecione File → Open/Close Drive e navegue a árvore de arquivos até o seu perfil. Você pode arrastar e soltar o perfil na sua área de trabalho, em seguida, arrastá-lo de volta para Modio e deixá-lo na área aberta. A janela "Profile" será aberta onde você pode selecionar a guia "Conteúdo de arquivo" para exibir todos os arquivos GPD dentro do seu perfil.

Cada jogo tem um ID de título exclusivo usado para nomear seu arquivo GPD correlacionando. O nome do arquivo é uma sequência de números e letras. Você pode abrir o arquivo GPD com um editor hex, como HxD ou HEXplorer, e editar o código hexam.

O que é um arquivo .gpd? (Arquivo de descrição da impressora genérica)

Arquivo de texto descrevendo uma impressora para o sistema operacional Windows ao imprimir com o Unidrv (Universal Printer Driver); usa o idioma GPD para descrever uma impressora que fornece vários tipos de informações, incluindo: atributos de impressora, comandos de impressora, recursos de impressora, opções de impressora, descrições de fontes da impressora e instruções condicionais.

Mais Informações

Todas as entradas de arquivos GPD são semelhantes a esta:

* EntradaNo nome: EntryValue {GPD_FileEntry, GPD_FileEntry, }

EntryName segue o asterisco e é sempre uma palavra-chave predefinida que o analisador GPD da Unidrv reconhece.

Um exemplo de um arquivo GPD se parece com este:

*Recurso: Orientação
{
*Nome: "Orientação"
*DefaultOption: Portrait
*Opção: Portrait
{
*Nome: "Retrato"
*Comando: CmdSelect
   {
*Ordem: DOC_SETUP7
*Cmd: "<1B>&l0O"
}
}
}

O que é um arquivo .gpd? (Arquivo de imagem PhotoDefiner gráfico)

Formato de imagem desenvolvido pela RGB Light para uso com o PhotoDefiner; incorpora compactação de imagem sem perdas que usa a "nova organização de referência de luz na abordagem teórico RGB" trigêmea; pode comprimir imagens mais do que os formatos .GIF ou .PNG.
