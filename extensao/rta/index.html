O que é um arquivo .rta? (Arquivo de dados RoboHelp RTA)

Project data file criado pela RoboHelp, um sistema de autoria de ajuda multiplataforma; salvo como um dos vários arquivos de um projeto RoboHelp e contém informações usadas internamente pela RoboHelp ao abrir e salvar projetos.

Mais Informações

NOTA: Os arquivos RTA não devem ser abertos manualmente. Em vez disso, eles são usados pela RoboHelp internamente ao editar projetos.

O que é um arquivo .rta? (Arquivo do Projeto TrueRTA)

Arquivo de projeto criado pela TrueRTA, um analisador de software usado para testar sistemas de áudio; salva configurações que são usadas para analisar entradas ou sons gerados; permite que as configurações do projeto sejam gravadas e usadas posteriormente.
