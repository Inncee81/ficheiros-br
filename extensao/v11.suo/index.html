O que é um arquivo .v11.suo? (Arquivo de opções de usuário da solução Visual Studio 2012)

Arquivo de informações do usuário criado pelo Visual Studio 2012, uma ferramenta de desenvolvimento de software usada para criar programas de Windows e aplicativos de Web; contém informações do usuário para um projeto como nós expandidos, janelas de ferramentas abertas e tarefas do usuário; estruturadas em formato binário.

Mais Informações

O arquivo é usado para manter a continuidade da aparência a partir de quando você fechou o projeto e, em seguida, quando você reabri-lo. O arquivo é referenciado por VSPackages criados com o Visual Studio, aberto pelo Método LoadUserOptions e modificado pelos métodos WriteUserOptions e SaveUserOptions.

A extensão composta sobe a cada nova versão do Visual Studio. Visual Studio 2013 usa a extensão .V12.SUO.

NOTA: O arquivo V11SUO pode ser excluído para aumentar o desempenho do Visual Studio, mas você perderá os dados da solução do usuário contidos no arquivo.
