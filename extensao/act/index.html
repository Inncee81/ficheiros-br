O que é um arquivo .act? (Arquivo de tabela de cores do Adobe)

Arquivo de tabela de pesquisa de cores (CLUT) usado pelo Adobe Photoshop, um aplicativo usado para editar gráficos raster; armazena uma coleção de cores pré-definidas; usado para aplicar um CLUT a uma imagem na caixa de diálogo "Salve para Web" no Photoshop.

Mais Informações

Os arquivos ACT são usados para otimizar imagens para Web publicação. Adicionando ou reduzindo o número de cores no CLUT, os usuários podem examinar a qualidade da imagem versus o tamanho do arquivo resultante antes de salvá-la.

As tabelas de cores da Adobe podem ser carregadas na caixa de diálogo Salvar para Web, selecionando o menu Tabela de Cores e escolhendo Load Color Table.... Vários arquivos ACT predefinidos estão incluídos com a instalação do Photoshop no diretório /Presets/en_US/Save for Web Settings/Color Tables/.

NOTA: OS CLUTs são usados por alguns formatos de imagem no diálogo Salvar para Web, como .GIF. No entanto, eles não são usados por .JPG e .PNG arquivos.

O que é um arquivo .act? (Arquivo de ação do criador de jogos DS)

Arquivo usado pelo DS Game Maker, um aplicativo usado para criar jogos Nintendo DS; contém uma ação de jogo escrita em C; usada para exibir gráficos, reproduzir sons, criar transições e controlar a lógica do jogo, bem como outras ações.

Mais Informações

Ações personalizadas podem ser criadas a partir de modelos fornecidos na interface DS Game Maker Action Editor. Os arquivos ACT são armazenados no diretório \Actions\ da instalação do DS Game Maker.

OS arquivos ACT são salvos com um arquivo .ACTX correspondente, que contém uma descrição para a ação.

NOTA: Invisionsoft DS Game Maker não é mais desenvolvido.

O que é um arquivo .act? (Arquivo de ator genesis3D)

Personagem tridimensional, ou "ator," salvo no formato Gênesis 3D; pode incluir movimentos de objeto, skins e materiais; arquivos de ator podem ser criados diretamente dentro do Genesis3D ou convertendo outros formatos 3D para o formato ACT usando o Genesis3D Actor Studio.

Mais Informações

Genesis3D é um motor de renderização 3D em tempo real de código aberto usado principalmente para criar jogos 3D. O pacote de motores Genesis 3D inclui o Actor Studio, Actor Viewer, e outros arquivos. Mais informações estão disponíveis no site do Genesis3D.
