O que é um arquivo .scskin? (Arquivo de pele de survivalcraft)

Um arquivo SCSKIN é um arquivo de pele usado pelo Survivalcraft, um jogo de sobrevivência que apresenta jogabilidade e gráficos semelhantes ao Minecraft. Contém uma .PNG pele para personagens em Survivalcraft que inclui cor da pele, características faciais e roupas. Os arquivos SCSKIN são simplesmente renomeados arquivos PNG.

Mais Informações

Arquivos SCSKIN são frequentemente compartilhados entre diferentes usuários do Survivalcraft para modificar os personagens do jogo. Você pode colocar o arquivo no diretório de jogos Survivalcraft com os outros arquivos SCSKIN para usá-lo no jogo. Você também pode renomear a extensão "scskin" para "png" e visualizá-la em um editor de imagem que suporta o formato PNG.

Arquivos SCSKIN são usados para todos os tipos de personagens, como homens, mulheres, monstros, esbeltos e palhaços. Eles são comumente 64 x 64 pixels de tamanho.
