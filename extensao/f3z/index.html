O que é um arquivo .f3z? (Fusion 360 Arquivo de Projeto Compartilhado)

Um arquivo F3Z é um arquivo comprimido zip usado pelo Autodesk Fusion 360, uma ferramenta de design industrial e mecânico 3D CAD. Ele contém um ou mais .F3D arquivos de design Fusion 3D, juntamente com vários arquivos .JSON contendo informações sobre os projetos e manifesto. Os arquivos F3Z são criados para compartilhar projetos do Fusion 360 que fazem referência a outros arquivos de design.

Mais Informações

Você pode abrir arquivos F3Z usando o Fusion 360. Para abrir o arquivo, selecione "Upload" no painel Dados fusion 360 e escolha o arquivo F3Z. O conteúdo do arquivo será carregado no Fusion 360, exibindo o conjunto principal no painel com o design F3D, permitindo que você visualize e edite-os.

Você pode criar um arquivo F3Z no Fusion 360 selecionando File → Export... e escolhendo o formato de arquivo F3Z em Type.

Como os arquivos F3Z são compactados com compactação zip, eles podem ser descomprimidos usando um programa de descompressão zip, como Corel WinZip, 7-Zip, WinRAR ou Apple Archive Utility. Basta renomear a extensão .f3z para .zip e descompactá-la. Você pode então extrair manualmente o conteúdo do arquivo F3Z, que normalmente inclui um arquivo de design F3D Fusion 3D, um arquivo JSON descrevendo o design e um arquivo JSON contendo o manifesto F3Z. Quando terminar, você pode comprimi-lo com compactação Zip novamente usando um utilitário Zip e, em seguida, renomear a extensão .zip para .f3z.

NOTA: O Autodesk Fusion 360 fornece vários recursos baseados em nuvem e é frequentemente usado para colaboração.
