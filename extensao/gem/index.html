O que é um arquivo .gem? (GEM Metafile)

Formato de imagem vetorial usado pelo GEM Draw, um programa básico de desenho para PCs; GEM significa "Graphical Environment Manager"," e era um sistema de janelas usado por computadores Atari ST e alguns computadores baseados no Windows.

O que é um arquivo .gem? (Pacote RubyGems)

Pacote que pode conter programas ruby e bibliotecas; salvo em um formato independente chamado "gem;" pode ser instalado usando o aplicativo de gerenciador de pacotes RubyGems incluído no software Ruby.

Mais Informações

Os arquivos Ruby GEM podem ser instalados usando o comando "gem instalar GEMNAME [opções]". Você também pode usar o comando "gem list -r -d" para listar as gemas instaladas em uma rede.

O que é um arquivo .gem? (Documento do Editor Ventura)

Formato de metafile baseado em vetores usado pela Ventura Publisher, um programa de publicação de desktop orientado para negócios; originalmente desenvolvido pela Ventura Software, mas adquirido pela Corel em 1993.

Mais Informações

NOTA: Corel Ventura não está mais desenvolvido.
