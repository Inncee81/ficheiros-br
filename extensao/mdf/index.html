O que é um arquivo .mdf? (Arquivo de formato de dados de medição)

Formato de arquivo padrão usado pela indústria automotiva para armazenar dados de medição; contém dados de medição bruto de uma medição e os metadados necessários para interpretar os dados brutos.

Mais Informações

O que é um arquivo .mdf? (Arquivo de imagem do disco de mídia)

Um arquivo MDF é uma imagem de disco salva por um programa de autoria de discos como Álcool 120%. Ele armazena os dados reais do disco de CD ou DVD, enquanto as informações de cabeçalho e faixa são armazenadas em um arquivo .MDS correspondente. Os arquivos MDF são semelhantes aos arquivos .ISO, mas são salvos em um formato diferente.

Mais Informações

Os arquivos MDF também podem ser acompanhados por metadados sobre os arquivos MDF. eles incluem informações, como onde as camadas de disco começam e terminam e informações de direitos autorais.

O que é um arquivo .mdf? (Arquivo de banco de dados do servidor SQL)

Formato de arquivo de banco de dados primário usado pelo Microsoft SQL Server, um programa de banco de dados de nível corporativo; usado para arquivos de banco de dados instalados com SQL Server e complementos relacionados; arquivos MDF personalizados também podem ser criados pelo usuário.

O que é um arquivo .mdf? (Arquivo de achado de dados de memória de tradução)

Suporte a arquivo para uma memória de tradução usada pelo software de tradução Trados; permite procurar rapidamente dados dentro da memória de tradução.

Mais Informações

Uma memória de tradução (TM) inclui cinco tipos de arquivos, incluindo arquivos MDF, .TMW, .IIX, .MTF e .MWF.
