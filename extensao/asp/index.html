O que é um arquivo .asp? (Página do servidor ativo)

Um arquivo ASP é uma página web criada por um desenvolvedor web que reside em um servidor web e pode conter scripts escritos em VBScript ou JavaScript. Ele é analisado no servidor, o que gera HTML que é enviado para o navegador do cliente. Os arquivos ASP são comumente usados por sites e sites ASPNET hospedados em servidores Windows executando o Microsoft IIS.

Os arquivos ASP são semelhantes aos que analisam o arquivo ASP e geram HTML simples que é enviado para o navegador do cliente. Como as páginas ASP são processadas no servidor, os visitantes do site não vêem o código ASP real, apenas o HTML gerado a partir dos scripts dentro da página. As páginas ASP normalmente usam a extensão "asp" em vez de "html"

Desde o lançamento do ASP 10 em 1996, houve apenas duas outras versões, ASP 20 em 1997 e ASP 30 em 2000. Em 2002, a Microsoft lançou a ASPNET, que substituiu o ASP. Agora ASP é comumente referido como Classic ASP ou ASP Classic. Devido à ampla implementação do framework ASPNET, as páginas ASP foram largamente substituídas por páginas .ASPX. Algumas vantagens do ASPNET sobre ASP é o uso de mais idiomas como C#, VBNET, J#e DelphiNET, bem como melhor desempenho com código compilado, depuração mais avançada e melhor manuseio de erros de tempo de execução.

O que é um arquivo .asp? (Arquivo de configuração de separação de cores do Adobe)

Contém opções de separação de cores que podem ser carregadas por programas da Adobe, como o Photoshop; usado para definir preferências para separações de cores quando você imprime, exporta para PDF ou exporta seu documento para outro programa; inclui tipos de cor de tinta, tipo de separação e limite de tinta; referenciado para imagens usando modos de cores como RGB e CMYK; semelhante à extensão de arquivo .AST Adobe.
