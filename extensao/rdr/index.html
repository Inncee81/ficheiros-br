O que é um arquivo .RDR? (Arquivo de configurações de relatório do gerenciador de desktop remoto)

Um arquivo RDR é um relatório criado pelo Remote Desktop Manager, software usado para centralizar conexões remotas, credenciais e senhas. Ele contém configurações para criar relatórios sobre entradas, logs, credenciais e informações de segurança de um usuário.

Mais Informações

Um arquivo RDR é criado pelo Remote Desktop Manager quando as configurações do relatório são exportadas. Para criar o arquivo RDR, selecione seu relatório em Administrations - Report e clique em Export Settings.

Você pode gerar um relatório usando configurações de um arquivo RDR inserindo a seguinte linha de comando no Prompt de comando Windows:

C:***/DataSource:***/report:***/reportoutput:"***"/reportsettings:"***rdr"/ closeapp
