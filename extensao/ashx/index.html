O que é um arquivo .ashx? (Arquivo manipulador de Web ASPNET)

Um arquivo ASHX é uma página web que faz parte de um aplicativo de servidor web ASPNET. Ele contém referências a outras páginas hospedadas no servidor web que são enviadas para o navegador da Web do usuário. Os arquivos ASHX são processados pelo manipulador ASPNET HTTP do servidor quando um navegador cliente solicita a página.

Mais Informações

Às vezes, os arquivos ASHX são usados para enviar arquivos .PDF para o navegador do usuário, já que o arquivo ASHX tem uma referência ao documento PDF. No entanto, quando o arquivo é baixado, a extensão do arquivo não é renomeada para .pdf e permanece com a extensão de arquivo .ashx. Neste caso, você pode renomear a extensão do arquivo para .pdf e visualizá-la com o Adobe Reader ou outro visualizador PDF. Às vezes, você pode corrigir esse problema certificando-se de que o navegador da Web tenha o plug-in do Adobe Reader instalado. Se instalado, o arquivo PDF pode então abrir diretamente no navegador.

NOTA: Muitas páginas ASHX têm apenas uma pequena quantidade de código, mesmo tão curta quanto uma única linha. As funções da página estão escritas na linguagem de programação C#.
