O que é um arquivo .SCM? (Roteiro Compilado cleo)

Roteiro compilado escrito para CLEO, um mecanismo de script de terceiros para Grand Theft Auto: San Andreas (GTA SA); permite que os jogadores expandam os recursos do jogo usando funções personalizadas chamadas "opcodes;" pode ser criado usando Sanny Builder, um utilitário de modding GTA.

Mais Informações

Para usar um arquivo SCM com GTA SA, ele deve ser colocado no diretório de \CLEO\ na pasta de instalação do jogo.

O que é um arquivo .SCM? (Roteiro GIMP Script-Fu)

Script escrito para o programa de edição de imagens GIMP; usado para automatizar ações ou adicionar efeitos artísticos às imagens abertas no editor GIMP; semelhante a um plugin do Photoshop, mas salvo como um arquivo de script.

Mais Informações

Os scripts do GIMP devem ser colocados nos seguintes locais para diferentes sistemas operacionais:

Mac: $HOME/Library/Application Support/Gimp/scripts/

Windows: %UserProfile%\.gimp-2.x\scripts\

Unix: $HOME/.gimp-2.x/scripts/

O que é um arquivo .SCM? (Arquivo de lista de canais do Samsung)

Arquivo exportado de algumas TVs Samsung usando uma conexão USB; contém uma lista de canais salvos na televisão, incluindo estações de cabo e satélite; usados para salvar listas de canais exportados.

O que é um arquivo .SCM? (Arquivo de esquema)

Arquivo descritor que descreve outro documento, como um arquivo XML; normalmente especifica a estrutura e formatação do documento, bem como que tipo de dados ele pode conter.

Mais Informações

Arquivos SCM são frequentemente encontrados em sistemas Unix, mas também podem ser usados em outras plataformas. Outros tipos de arquivos de esquema XML incluem arquivos de definição de tipo de documento (.DTD) e arquivos W3C XML Schema (.XSD).

Arquivo de código fonte do esquema

Desenvolvedor N/A

O que é um arquivo .SCM? (Arquivo de código fonte do esquema)

Um arquivo SCM é um arquivo de código fonte escrito em Scheme, uma linguagem de programação desenvolvida no Massachusetts Institute of Technology (MIT). Ele é normalmente editado com um editor de texto e, em seguida, executado usando um intérprete de Esquema, que lê a sintaxe e executa o programa. Os arquivos SCM podem ser editados com qualquer editor de texto, mas são melhor modificados usando um editor que suporta o destaque da sintaxe do Esquema.

Mais Informações

O esquema faz parte da família Lisp de linguagens de programação. É mais frequentemente usado em ambientes acadêmicos para ensinar conceitos de programação funcional.

O que é um arquivo .SCM? (Arquivo de mapa starcraft)

Mapa de jogo usado pelo StarCraft, um jogo de estratégia em tempo real da Blizzard Entertainment; inclui vários tipos de terreno e recursos; pode ser criado em diferentes tamanhos, incluindo 96x96, 128x128, 192x192 e 256x256.

O que é um arquivo .SCM? (Arquivo de mídia super cadeia)

Arquivo de áudio ou vídeo codificado com o codec SCM; inclui drm de proteção de direitos autorais digitais, que garante a transmissão válida de mídia pela Internet; desenvolvido pela SuperMV, uma empresa de pesquisa interativa de áudio e vídeo.

Mais Informações

Arquivos SCM podem ser reproduzidos em vários media players se o codec SuperMV SCM estiver instalado.
