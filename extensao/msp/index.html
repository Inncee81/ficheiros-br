O que é um arquivo .msp? (Arquivo do dicionário ClarisWorks)

Arquivo usado pelos produtos de software do pacote de escritórios ClarisWorks e AppleWorks; contém uma lista de palavras usada pelo verificador ortomân escrito do software; pode suportar vários idiomas e dialetos.

Mais Informações

Exemplos de arquivos MSP incluem usenglsh.msp e ukenglsh.msp. Esses arquivos armazenam os dicionários para a língua inglesa dos Estados Unidos e a língua inglesa do Reino Unido, respectivamente.

NOTA: Tanto a ClarisWorks quanto a AppleWorks foram descontinuadas.

O que é um arquivo .msp? (Arquivo de página MaxScript)

Script baseado na Web criado para MaxScript, um intérprete xBase que executa scripts do lado do servidor; contém código HTML e comandos xBase ou Clipper; armazenado em um formato de texto ASCII; pode ser editado com qualquer editor de texto.

Mais Informações

MaxScript interfaces diretamente com fontes de dados, como conexões ODBC, para recuperar conteúdo dinâmico para páginas Web.

MaxScript é usado por MAXSIS DBmax e DBfree.

O que é um arquivo .msp? (Imagem do Bitmap do Microsoft Paint)

Arquivo de imagem monocromático (preto e branco) criado pelo Microsoft Paint, um dos primeiros programas de pintura lançados pela Microsoft; também usado para armazenar imagens de clip art mais antigas.

O que é um arquivo .msp? (Patch instalador de Windows)

Arquivo usado para atualizar o sistema operacional Windows e outros programas da Microsoft; normalmente incluído com atualizações de segurança ou hotfixes da Microsoft; executado pelos programas Windows Installer Hotfix.exe e Update.exe.
