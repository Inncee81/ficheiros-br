O que é um arquivo .PDB? (Arquivo de banco de dados palm desktop)

O formato de dados padrão usado por vários programas diferentes em dispositivos PDA baseados em Palm; pode armazenar vários tipos de dados, como eBooks, documentos de processamento de texto, planilhas, bancos de dados, mapas e arquivos de imagem.

Mais Informações

Para determinar qual programa criou um arquivo PDB, abra o arquivo em um editor de texto. Os códigos Type and Creator devem estar localizados no cabeçalho do arquivo perto da parte superior do documento.

O que é um arquivo .PDB? (Banco de dados do programa)

Um arquivo PDB é um arquivo de banco de dados usado por vários aplicativos, incluindo pegasus, Quicken, MS Visual Studio e Palm Pilot software. Armazena dados em um formato estruturado e normalmente é instalado com o aplicativo correspondente.

Mais Informações

Arquivos PDB usados por diferentes programas geralmente são salvos em um formato proprietário, o que significa que eles só podem ser abertos pelo programa com o que foram instalados. Por exemplo, um arquivo PDB Intuit Quicken não é reconhecido pelo Microsoft Visual Studio.

O Visual Studio usa arquivos PDB para armazenar informações de depuração sobre um programa, incluindo informações de símbolos, que é muito grande para ser armazenado no próprio programa. As informações do símbolo consistem em uma lista de todos os símbolos em um módulo de programa com endereços, o nome do arquivo e a linha em que o símbolo foi declarado. Você pode usar o Visual Studio para carregar informações do arquivo PDB para depurar um programa.

O que é um arquivo .PDB? (Arquivo do Banco de Dados de Proteína)

Arquivo de dados criado no formato Do Banco de Dados de Proteínas (PDB); armazena coordenadas 3D especificando a estrutura de átomos dentro de uma proteína ou outro composto químico; usado como formato padrão para estudar estruturas proteicas; também utilizado pela Pesquisa Colaboradora de Bioinformática Estrutural (RCSB) para o estudo de macromoléculas biológicas.

Mais Informações

Muitos arquivos do Protein Data Bank são disponibilizados através do Worldwide Protein Data Bank (wwPDB), uma organização que arquiva macromoléculas e as distribui livremente.

NOTA: Os arquivos PDB são frequentemente vistos dentro de um applet Java que usa jmol, um pacote de visualização de moléculas de código aberto. Os arquivos PDB também são suportados por alguns plugins do navegador de visualização molecular.

O que é um arquivo .PDB? (Arquivo de construtor de demonstração de Tanida)

Arquivo de projeto de filme criado e usado pelo Demo Builder, um programa usado para criar tutoriais, apresentações e demonstrações para mostrar como funcionam os softwares e sistemas; contém cenas de tela gravadas, juntamente com vídeo e imagens importados.

Mais Informações

Um projeto de filme de Demo Builder pode ser exportado para uma variedade de formatos, dependendo de como você deseja apresentar seu tutorial. Os formatos suportados incluem Flash, HTML, Windows Media Video, MPEG4 e Word Document.
