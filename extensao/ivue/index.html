O que é um arquivo .ivue? (Imagem ao vivo IVUE Imagem)

Imagem raster criada pelo Live Picture, um aplicativo de edição de imagens para Mac OS Classic que agora está descontinuado; usa o formato IVUE, que é um formato de imagem de multi resolução usado como base para FlashPix (.FPX arquivos).

Mais Informações

Ao editar uma imagem IVUE na Live Picture, as camadas de ajuste podem ser aplicadas para modificar subseções da imagem. Embora a imagem IVUE em si não seja modificada, as instruções para os ajustes são armazenadas em arquivos FITS (Funcional Interpolating Transformation System) ou FPX. Quando uma imagem IVUE manipulada está pronta para saída em Live Picture, a imagem IVUE e os arquivos FITS ou FPX são processados em conjunto e incorporados em um formato mais padrão, como .JPG, .TIFF ou .PSD. As imagens finais podem ser construídas com cálculos em profundidade de 48 bits e para baixo amostradas a profundidade de 24 bits. As visualizações localizadas também podem ser selecionadas em Live Picture e saída como arquivos de imagem padrão.

NOTA: O formato IVUE não é mais suportado. Live Picture estava disponível anteriormente para Mac OS Classic e também ofereceu um plug-in do Photoshop para ler o formato IVUE. Live Picture competiu com o Adobe Photoshop na década de 1990 como um dos principais editores de imagem para Mac.
