O que é um arquivo .ifc? (Arquivo de classes da Fundação da Indústria)

Um arquivo IFC é um arquivo modelo criado no formato IFC (Industry Foundation Classes, classes de base do setor), que é um formato de arquivo aberto usado pelos programas BIM (Building Information Modeling, modelagem de informações de construção). Contém um modelo de construção ou instalação, incluindo elementos espaciais, materiais e formas.

O formato de arquivo IFC é um formato neutro de plataforma usado para permitir a interoperabilidade entre programas BIM da indústria. Como resultado, a maioria dos programas BIM, seja proprietário ou de código aberto, pode abrir um arquivo IFC.
