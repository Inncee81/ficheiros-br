O que é um arquivo .fasta? (Arquivo de sequência FASTA)

Formato de dados científicos usado para armazenar sequências de ácido nucleico (como sequências de DNA) ou sequências proteicas; pode conter múltiplas sequências e, portanto, às vezes é referido como o formato de banco de dados FASTA; pode ser visto e analisado usando software de análise de DNA.

Mais Informações

Arquivos FASTA geralmente começam com uma linha de cabeçalho que pode conter comentários ou outras informações. O resto do arquivo contém dados de sequência. Cada sequência começa com um símbolo ">" seguido pelo nome da sequência. O resto da linha descreve a sequência e as linhas restantes contêm a sequência em si.
