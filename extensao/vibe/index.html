O que é um arquivo .vibe? (Arquivo Haptic do VibePlayer)

Um arquivo VIBE é um arquivo de dados criado pelo SensationSoft VibePlayer, um player de áudio e vídeo para Android que permite que os usuários sintam vibrações do dispositivo enquanto a mídia é reproduzida. Ele contém dados hápticos que sincronizam com um arquivo de áudio ou vídeo, como um arquivo .MP3 ou .MP4, de modo que quando a mídia é reproduzida no aplicativo o usuário pode sentir as vibrações físicas através do dispositivo. Os arquivos VIBE são usados para melhorar a experiência de um usuário assistindo ou ouvindo mídia no VibePlayer.

Mais Informações

O VibePlayer permite que os usuários criem arquivos VIBE que sincronizam com arquivos de áudio ou vídeo, que, quando reproduzidos, vibram nos momentos apropriados. Por exemplo, um usuário pode tocar no aplicativo VibePlayer quando certas notas são reproduzidas em uma música ou efeitos sonoros são reproduzidos em um filme para criar vibrações que são salvas no arquivo VIBE. Depois que o arquivo é criado, ele é colocado na mesma pasta do arquivo de mídia ao qual está associado. Quando um usuário reproduz o arquivo de mídia no VibePlayer, o aplicativo também carrega o arquivo VIBE para reproduzir ao lado do arquivo de mídia.

Um arquivo VIBE tem o mesmo nome do arquivo de mídia com o qual está associado, mas com a extensão .vibe anexada a ele. Por exemplo, se o arquivo VIBE estiver associado a example.mkv, o arquivo será nomeado example.mkv.vibe.

NOTA: Enquanto os arquivos VIBE são destinados a ser reproduzidos pelo VibePlayer, eles podem ser editados por um editor de texto, uma vez que os dados são armazenados em texto simples.
