O que é um arquivo .settingcontent-ms? (Arquivo de configurações de Windows)

Arquivo criado para o sistema operacional Windows 81; contém a definição de conteúdo para funções Windows, como como as atualizações são instaladas ou quais aplicativos padrão devem ser usados para abrir determinados tipos de arquivos; o conteúdo contido é escrito na linguagem XML.

Mais Informações

Você pode criar um atalho para o arquivo CONFIGURAÇÃOCONTENT-MS, o que facilita a personalização de suas configurações:

  1. Clique com o botão direito do mouse em um espaço vazio na área de trabalho e selecione New → Shortcut.
  2. Copie e cole o seguinte texto na barra da área de localização do item e clique em Next:
    %windir%explorerexe %localappdata%Packageswindows
    immersivecontrolpanel_cw5n1h2txyewyLocalState
    IndexadoConfiguraçõesen-USAAA_SystemSettings_Update
    _ChangeSettingssettingcontent-ms
  3. Nomear o atalho "Configurações de atualização de Windows" e clique em Finish.

NOTA: Para o texto da etapa 2, se você estiver usando um idioma diferente do inglês, você terá que alterar en-US no texto. Além disso, este texto é para um atalho para Windows configurações de atualização, consulte a seção "Nomes de arquivos comuns" abaixo para outros arquivos CONFIGURAÇÃOCONTENT-MS.
