O que é um arquivo .studio3? (Documento do estúdio silhueta 3)

Um arquivo STUDIO3 é um projeto criado pela Silhouette Studio versão 3, um programa usado para projetar itens que podem ser cortados com uma máquina de corte eletrônico Silhueta. Ele contém um design, que pode incluir texto, formas, linhas, imagens, cores, gradientes, padrões e efeitos. Os arquivos STUDIO3 também armazenam configurações de página de design, configurações de corte e marcas de registro.

Captura de tela de um arquivo studio3 em Silhouette Studio 3

O arquivo STUDIO3 é o arquivo principal associado à versão 3 do Silhouette Studio. Quando você cria e salva um projeto de design, o arquivo STUDIO3 é o formato padrão usado para armazenar o design. No entanto, você também pode optar por salvar o design como um arquivo .STUDIO ou .GSP. O arquivo STUDIO3 substituiu o arquivo STUDIO pela versão 3. O arquivo STUDIO foi usado pelas versões 1 e 2 do Silhouette Studio.

Uma vez terminado o seu projeto, você pode verificar as configurações de corte, relevo, deboss, grade, marcas de registro, pontilha e corte. Em seguida, você pode enviá-lo para um dispositivo de silhueta conectado para corte.
