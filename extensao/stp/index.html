O que é um arquivo .STP? (Arquivo de informações do projeto do estúdio de análise)

Arquivo criado pelo Analysis Studio, um programa usado para mineração de dados e análise estatística; contém o esquema de projeto e objetos de projeto em formato XML; mantém as ações, transformações de dados e variáveis que permitem que um conjunto de dados seja reconstruído a partir de um conjunto de dados original.

Mais Informações

Quando o Analysis Studio está no modo online, os dados podem ser baixados e reconstruídos a partir de um servidor de análise remoto. Quando no modo offline, .DATA arquivos armazenados na máquina local são usados.

NOTA: Appricon não desenvolve mais o Analysis Studio.

O que é um arquivo .STP? (Arquivo da lista de parada robohelp)

Arquivo de texto simples criado pela RoboHelp, um programa de criação de ajuda Windows; salva uma lista de stop que contém palavras-chave que o assistente do Smart Index ignora ao criar o índice de pesquisa de documentação de ajuda; só pode ter 512 caracteres de comprimento; criado como parte de um projeto RoboHelp (.XPJ).

Mais Informações

Listas de parada geralmente contêm palavras como "o," " ""ou" e "a" Essas são palavras que normalmente não são úteis para indexar como palavras-chave para pesquisas de documentação de ajuda.

O que é um arquivo .STP? (Modelo do SharePoint)

Modelo de site criado para o Microsoft SharePoint, um programa de colaboração baseado na Web; contém um site pré-assinado que pode ser carregado da galeria de modelos SharePoint; usado como ponto de partida para a criação de novos sites.

Mais Informações

Para criar um arquivo STP, clique em "Configurações do site" Em seguida, em "Administração," escolha "Vá para a Administração do Site" Sob Gerenciamento e Estatísticas, selecione "Salvar site como modelo"

NOTA: Os arquivos do SharePoint STP contêm apenas informações sobre o que é personalizado no site. Eles não incluem a definição completa do site.

O que é um arquivo .STP? (Arquivo CAD STEP 3D)

Um arquivo STP é um arquivo de montagem 3D suportado por vários programas de design mecânico, como TurboCAD e Fusion 360. Ele contém dados tridimensionais de objeto salvos no formato Padrão para a Troca de Dados do Produto (STEP). Os arquivos STP são salvos em um formato ASCII baseado em texto que está em conformidade com os protocolos de aplicação STEP ISO 10303-2xx.

Mais Informações

Os arquivos STP são usados principalmente para transferir gráficos 3D entre diferentes sistemas CAD e CAM. Muitos fabricantes usam o arquivo STP para projetar e precificar moldes ou peças fabricadas.

NOTA: Os arquivos STP também podem usar a extensão menos comum .STEP.
