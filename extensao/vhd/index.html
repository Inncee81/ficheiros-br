O que é um arquivo .vhd? (Arquivo de origem VHDL)

Arquivo de origem escrito em VHDL, que significa Very High Speed Integrated Circuits (VHSIC) Hardware Description Language (HDL); frequentemente usado para automatizar o design de circuitos elétricos e componentes.

Mais Informações

Engenheiros eletrônicos podem usar vHDL para descrever componentes de hardware em um formato baseado em texto. Esta descrição pode então ser usada para fabricar um circuito físico, como uma placa-mãe, processador ou módulo de memória.

O que é um arquivo .vhd? (Disco rígido virtual do PC virtual)

Um arquivo VHD contém uma imagem de disco rígido virtual usada pela Microsoft Windows Virtual PC, um programa de virtualização Windows. Ele armazena o conteúdo de um disco rígido de uma máquina virtual (VM), que pode incluir partições de disco, um sistema de arquivos, arquivos e pastas. Os arquivos VHD podem ser usados para instalar vários sistemas operacionais em um único computador, programas de software de teste ou executar aplicativos mais antigos.

Mais Informações

A Especificação do Formato de Imagem VHD está disponível como uma licença livre de royalties desde junho de 2005. O formato VHDX foi introduzido em 2012 como uma extensão do formato VHD. Discos rígidos virtuais salvos no formato VHDX são armazenados em .VHDX arquivos.

NOTA: Você pode examinar o conteúdo de um arquivo VHD manualmente usando 7-Zip.
