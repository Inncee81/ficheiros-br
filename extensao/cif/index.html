O que é um arquivo .cif? (Arquivo de informações cristalográficas)

Formato de troca de dados cristalográfico desenvolvido pela União Internacional de Cristalografia (IUCr); contém categorias de dados cristalográficos utilizados para arquivamento e distribuição de finalidades.

Mais Informações

O formato foi adotado em 1990 como uma estrutura de arquivo padrão e é regularmente utilizado para relatar determinações de estrutura de cristal. O arquivo CIF contém categorias de dados, como _audit_, _atom_, _cell_>, _database_, e _refine_>. Essas categorias contêm informações, incluindo a criação do CIF, propriedades do tipo átomo, parâmetros celulares, dados especificados pelos gerentes de banco de dados e parâmetros de refinamento da estrutura.

O que é um arquivo .cif? (Imagem fácil do disco do criador de CD)

Formato de imagem de disco de CD criado pelo Corel e usado por Roxio Easy CD Creator (agora Roxio Creator); é montado como um CD físico em Windows e é viável em "Meu Computador;" muitas vezes usado para criar cópias de backup de CDs.
