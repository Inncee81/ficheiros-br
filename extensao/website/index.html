O que é um arquivo .website? (Atalho do site do Internet Explorer)

Um arquivo SITE contém um atalho de site criado pelo Internet Explorer (IE), um navegador da Web para Windows. Ele armazena uma referência de endereço web (uma URL) e um arquivo de ícone, que faz referência à favicon.ico do site e é usado como o ícone exibido para o atalho. Os arquivos SITE são normalmente usados para salvar e abrir marcadores e sites favoritos.

Mais Informações

Para criar um arquivo SITE no IE, clique e arraste a URL na barra de endereços ou no "favicon" do site para a sua área de trabalho. Você também pode arrastar o site para a área de Lançamento Rápido em Windows Vista ou para a Barra de Tarefas nos Windows 7 e Windows 8. Dessa forma, você pode ter acesso rápido aos seus sites favoritos.

OS arquivos do SITE foram introduzidos na versão 9 do IE e não são reconhecidos por computadores Windows com o Internet Explorer 8 (IE8) ou instalados anteriormente. No entanto, você pode renomear a extensão .website para .url e abrir o arquivo como um arquivo URL. Versões anteriores do IE criaram a chave de API, enquanto os arquivos URL requerem apenas os valores de URL e URL favicon. Você ainda pode criar arquivos url na versão 9 ou posterior do IE segurando a tecla SHIFT antes de arrastar o favicon da barra de endereços para a área de trabalho.

Para abrir a localização da Web de um arquivo SITE em outro navegador da Web, abra o arquivo com um editor de texto, como Microsoft Notepad ou WordPad. Uma vez que os arquivos do SITE são salvos em texto simples, você pode navegá-los com o editor de texto até localizar a propriedade "URL". Copie a URL e cole-a na barra de endereços de outro navegador da Web, como o Google Chrome ou o Mozilla Firefox.

Windows não exibe a extensão de arquivo .website para atalhos de site presos. Para visualizar a extensão, abra a janela de propriedades do arquivo clicando com o botão direito do mouse no arquivo SITE e selecionando Properties da lista de retirada.
