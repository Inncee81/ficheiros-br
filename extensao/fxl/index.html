O que é um arquivo .fxl? (Biblioteca de Expressão Facial CryENGINE)

Arquivo criado com o CryENGINE SDK, um kit de ferramentas de programa usado para o desenvolvimento de jogos 3D; salva uma biblioteca de expressões faciais para um personagem 3D; pode incluir expressões falantes ou olhares de preocupação, felicidade, raiva e outros.

Mais Informações

Arquivos FXL armazenam expressões faciais para um modelo de caractere (.CHR arquivo). O arquivo FXL para carregar para um determinado caractere é especificado no arquivo .CHRPARAMS.

O que é um arquivo .fxl? (Arquivo de idioma FaceFX)

Script escrito usando o FaceFX, um aplicativo usado para construir animações faciais para videogames; salva comandos que executam instruções de aplicativos, como construir um gráfico facial, publicar um ator ou configurar uma animação; útil para executar operações repetitivas.

Mais Informações

ARQUIVOS FXL salvam uma série de instruções que podem ser digitar manualmente individualmente na linha de comando FaceFX.
