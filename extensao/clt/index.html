O que é um arquivo .clt? (Arquivo de transcrição legal de clareza)

Arquivo criado pela Clarity Legal DepoSmart, um programa usado para criar transcrições eletrônicas certificadas; contém dados de transcrição usados por empresas de relatórios judiciais ao criar transcrições legais para advogados; criado para reduzir o tamanho do arquivo de transcrições enviadas por e-mail.

O que é um arquivo .clt? (Arquivo da lista de trapaça visualBoyAdvance)

Arquivo de jogo usado pelo VisualBoyAdvance, um emulador para o sistema de console portátil Gamboy Advance; salva uma lista de trapaças de jogo que podem ser aplicadas a um jogo emulado.

Mais Informações

Para abrir um arquivo CLT no VisualBoyAdvance, selecione Cheats → Load cheat list... no menu do programa após o carregamento de um ROM (arquivo .GBA).
