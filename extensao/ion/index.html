O que é um arquivo .ion? (Arquivo de descrição do arquivo)

Arquivo baseado em texto usado para descrever o conteúdo de outro arquivo ou arquivo; pode especificar o nome de arquivo, extensão de arquivo, metainfo e outras informações de outros arquivos; muitas vezes nomeado descript.ion; usado por gerentes de sistemas de arquivos e programas de software para organizar e ler informações de arquivos.

Mais Informações

O arquivo descript.ion (que diz "description" mas com "ion" como extensão de três letras) tem sido usado em vários contextos, incluindo com sistemas de quadro de avisos mais antigos (BBSs), 4DOS (um intérprete alternativo de linha de comando MS-DOS) e com extensões ao software Drupal CMS, como Filebrowser e File Tree. O XnView também suporta arquivos ION para exportar e importar metadados no programa.

NOTA: Como descript.ion arquivos foram usados em diferentes contextos, o formato pode variar entre os diferentes programas de software que os utilizam.
