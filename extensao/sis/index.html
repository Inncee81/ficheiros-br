O que é um arquivo .sis? (Arquivo arivis ImageCore)

Um arquivo SIS é um arquivo de dados de imagem salvo no formato ImageCore, que é comumente usado pelo software de ciência de imagem Arivis Vision4D. Ele contém dados de imagem, que podem incluir dados de dimensão de plano espacial, canal e quadro. Os arquivos SIS suportam 8 bits, 16 bits, 32 bits e 32 bits (flutuam) profundidades de dados e também armazenam metadados e configurações de cores.

Mais Informações

arivis Vision4D é um aplicativo de imagem 2D/3D/4D que permite anotar, visualizar e analisar imagens biológicas no nível microscópico. Você pode visualizar imagens 2D e 3D, bem como imagens 4D, que é uma imagem 3D que inclui dados de tempo. O formato ImageCore foi desenvolvido para armazenar a grande quantidade de dados de imagem necessários para que o software arivis renderizasse as imagens.

Para abrir um arquivo SIS no Vision4D, basta clicar em File → Open... e selecionar o arquivo SIS que deseja abrir. Você também pode importar uma imagem salva em outro formato para Vision4D selecionando File → Import. A imagem será automaticamente convertida para o formato ImageCore.

NOTA: O formato ImageCore foi introduzido em 1997.

O que é um arquivo .sis? (Arquivo de banco de dados do Sisma)

Arquivo criado pelo Sisma, um gerenciador de senhas para Windows; contém informações da conta e senhas, que são criptografadas usando criptografia AES de 256 bits; usadas para armazenar dados confidenciais que só podem ser recuperados com uma única senha mestre.

Mais Informações

O Sisma pode armazenar datas de validade de senhas em arquivos SIS para que as senhas sejam invalidadas após um período de tempo. O software também pode auxiliar os usuários criando senhas automáticas aleatórias que são pronunciáveis.

NOTA: O Sisma é gratuito para uso pessoal, mas requer uma licença adquirida para uso não pessoal.

Categoria Cópia de Segurança
Formato Texto 

O que é um arquivo .sis? (Arquivo de informações de backup do jogo steam)

Arquivo de informações usado pelo Steam, um serviço de jogos que permite aos usuários baixar e gerenciar jogos da Valve; contém marcações de texto simples, que descrevem propriedades de um backup de jogo; usado no processo de restauração de backups de jogos.

Mais Informações

Os arquivos SIS são backup com um arquivo .SIM e um ou mais arquivos de dados .SID.

O que é um arquivo .sis? (Arquivo de instalação symbian)

Arquivo instalador para Symbian OS, um sistema operacional usado para dispositivos móveis, como os telefones Palm PDAs e Nokia, Ericsson e Motorola; criado usando a ferramenta de linha de comando "makesis" da Symbian, que é empacotado com o programa CreateSIS; frequentemente usado para instalações temáticas de telefone S60.

Mais Informações

NOTA: Para instalar um arquivo SIS em um telefone compatível, use o software de comunicação do telefone para transferir o arquivo para o telefone. Em seguida, abra o arquivo SIS no telefone para executar o instalador.
