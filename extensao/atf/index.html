O que é um arquivo .atf? (Arquivo de textura alternativa)

Arquivo criado pelo AlternativaPlayer, um visualizador on-line modelo 3D; contém uma textura exportada do programa on-line sobreposto no modelo 3D; localizado dentro da pasta "modelo" da pasta no arquivo .ZIP criado pelo programa.

Mais Informações

Para criar o arquivo ATF:

  1. Selecione Exporte para A3D no canto superior esquerdo da janela.
  2. Na caixa de diálogo "Export to A3D", selecione Export.
  3. Na próxima caixa de diálogo "Export to A3D", selecione Salve o arquivo.
  4. Na próxima caixa de diálogo, escolha o nome do arquivo, salve a localização e clique em Salve.
  5. O arquivo é mantido junto com o arquivo .A3D na pasta "model" no arquivo ZIP.

Para carregar e visualizar o arquivo ATF:

  1. Selecione Carregar modelo no canto superior esquerdo da janela.
  2. Na próxima caixa de diálogo, navegue até a localização do seu arquivo A3D, selecione Abra.
  3. Quando a caixa de diálogo "Texturas ausentes" for abrir, selecione Escolha texturas.
  4. Escolha seu arquivo ATF, selecione Abra.

O que é um arquivo .atf? (Arquivo de texto Axon)

Formato de dados usado pelo software de análise de microarray GenePix; usa um texto simples, formato delimitado por guias e contém registros, cada um com um ou mais campos; pode ser usado intercambiavelmente com software GenePix, editores de texto e programas de planilha, como o Microsoft Excel.

Mais Informações

O formato ATF é usado para outros formatos de arquivos GenePix, incluindo .GAL e arquivos .GPR.

NOTA: AXon Instruments, desenvolvedora original do formato ATF, foi adquirida pela Molecular Devices, que utiliza os formatos ATF, GAL e GPR com seu software GenePix.

O que é um arquivo .atf? (Arquivo de função de transferência do Photoshop)

Armazena informações de transferência usadas para compensar o ganho/perda de pontos quando uma imagem é transferida para filme; pode adicionar ou subtrair percentagens de pontos, o que permite que as imagens sejam impressas com cores consistentes.

Mais Informações

As configurações de transferência podem ser criadas clicando no botão "Transferir" na caixa de diálogo Photoshop Print. As configurações podem ser salvas como um arquivo ATF clicando em "Salvar" na caixa de diálogo resultante.
