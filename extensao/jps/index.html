O que é um arquivo .jps? (Arquivo de Backup Akeeba)

Formato de arquivamento de backup criado pelo software Akeeba, que é usado para fazer backup e proteger sites; armazena um backup completo de um site; inclui o banco de dados, o script de restauração e quaisquer outros arquivos relacionados.

Mais Informações

O formato JPS foi projetado para criar arquivos de grandes árvores de diretórios usando apenas o código PHP. JPS é semelhante ao formato .JPA, mas é mais seguro. O formato JPS divide o descritor de arquivos e os dados de arquivos para blocos de 64Kb criptografados usando AES-128 no modo CBC. Além disso, o formato usa Deflate (ZLib) para comprimir todos os arquivos.

A extensão JPS pode ser usada para arquivos de backup Akeeba em várias partes. Se o backup de um site for dividido em vários arquivos, um arquivo JPA e .J01 arquivo serão criados com um arquivo JPS criado como o último arquivo no conjunto de várias partes.

O que é um arquivo .jps? (Imagem estéreo JPEG)

Um arquivo JPS é uma imagem JPEG estereoscópica usada para criar efeitos 3D a partir de imagens 2D. Contém duas imagens estáticas, uma para o olho esquerdo e outra para o olho direito que são codificadas como duas imagens lado a lado em um único arquivo .JPG.

Mais Informações

As imagens estéreo contêm duas cópias da mesma imagem, cada uma com pequenas variações, ou perspectivas, da outra. Desta forma, um espectador pode ver um efeito 3D das imagens 2D ao visualizar a imagem de uma das três maneiras:

  • Ver os olhos cruzados: O espectador olha para o lado estático-por lado imagem usando um olhar "cross-eye"
  • Anaglyph View: As duas imagens são sobrepostas umas sobre as outras e o espectador usa óculos 3D.
  • 3D Rendering: Um aplicativo torna as imagens 2D em uma imagem 3D resultante.
NOTA: Os arquivos JPS podem ser salvos por câmeras estéreo, que têm duas ou mais lentes.
