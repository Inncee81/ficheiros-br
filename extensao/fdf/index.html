O que é um arquivo .fdf? (Acrobat forma formato de dados)

Formato de arquivo de texto usado para dados exportados de campos de formulário .PDF; tipicamente menor que arquivos PDF, porque eles contêm apenas dados de campo de formulário, não todo o formulário; podem ser criados no Acrobat selecionando Advanced→Forms→Export Forms Data na barra de menu.

Mais Informações

Os arquivos FDF são comumente processados em servidores Web por scripts .CGI.

O que é um arquivo .fdf? (Arquivo de definição da função de montagem de origem)

Um arquivo FDF é um arquivo de definição de função de ajuste matemático usado pela Origin, um aplicativo de análise e gráfico de dados usado por vários cientistas e engenheiros. Contém todas as propriedades sobre uma função de montagem, como o tipo de função, restrições, parâmetros de montagem, uma expressão de função, variáveis e uma descrição. Os arquivos FDF podem vir incluídos com o Origin ou serem definidos pelo usuário.

Mais Informações

Os arquivos FDF definidos pelo usuário são criados usando o Construtor de Função de Montagem na Origem. Você pode acessar a ferramenta selecionando Tools → Fitting Function Builder.... Escolha "Criar uma nova função" para criar um arquivo FDF ou "Editar uma função definida pelo usuário" para editar um arquivo FDF. Você também pode visualizar seus arquivos FDF selecionando Tools → Fitting Function Organizer.... Você também pode abrir arquivos FDF com um editor de texto, uma vez que a definição da função de montagem é armazenada em texto simples.

FDF arquivos que vêm embalados com o programa estão localizados na subpaste "FitFunc" da pasta do programa Origin. No entanto, os arquivos FDF definidos pelo usuário são normalmente salvos na subpastos "FitFunc" da pasta "Arquivos do usuário" na pasta programa Origin.
