O que é um arquivo .BA6? (TaxAct 2016 Tax Return Backup File)

Um arquivo BA6 é um arquivo de backup de uma declaração de imposto de renda de 2016 criada com a versão de 2016 do TaxAct, um programa de preparação fiscal. Ele contém dados fiscais para o ano fiscal de 2016, que é usado para fins de arquivamento e no caso de o arquivo .TA6 for perdido ou se tornar corrupto. Os arquivos BA6 incluem dados federais da Receita Federal e formulários fiscais estaduais, juntamente com as informações básicas do usuário.

Captura de tela de um arquivo ba6 no TaxAct 2016

Os usuários do TaxAct 2016 provavelmente usarão apenas o arquivo TA6 para salvar, imprimir ou carregar seus impostos. No entanto, o arquivo BA6 é valioso para os usuários preocupados com seu arquivo TA6 se tornar corrupto ou se perder.

Para salvar um backup da sua declaração de imposto de renda selecione File → Backup... e nomeie seu arquivo. Você deve salvar sua declaração de imposto como um arquivo TA6 antes de criar seu backup.

Para restaurar um backup no TaxAct, selecione File → Restore Backup... e navegue até a localização do seu arquivo BA6.

Como as declarações fiscais anuais são arquivadas no ano anterior, o arquivo BA6 é provavelmente criado em 2017, do início de janeiro a 15 de abril. Uma vez que você tenha concluído uma declaração de imposto, você pode imprimir e enviar nos formulários fiscais ou arquivar eletronicamente.
