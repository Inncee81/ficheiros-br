O que é um arquivo .hdd? (Paralelamente arquivo disco rígido da área de trabalho)

Disco rígido virtual criado pelo software de virtualização Parallels Desktop; armazena os dados para o disco rígido de uma máquina virtual (você pode pensar nele contendo a unidade de Windows C:\ para VMs Windows); usado para carregar os arquivos do VM quando o VM é iniciado.

Mais Informações

Os arquivos HDD são armazenados dentro de arquivos HDD de montagem manualmente com o Parallels Mounter, um programa incluído no Parallels Desktop (localizado em /Library/Parallels/). Em seguida, clique com o botão direito do mouse no arquivo HDD e escolha Open With → Parallels Mounter.app.

NOTA: Versões mais antigas de arquivos HDD da versão Paralelas Desktop 2 podem ser abertas pelo Oracle VM VirtualBox.
