O que é um arquivo .SPJ? (Projeto Panorama do Microsoft ICE)

Projeto de imagem panorama criado pelo Microsoft Image Composite Editor (ICE), um costurador de imagem panorâmico avançado; salva o resultado de costurar várias fotos ou quadros de vídeo juntos; muitas vezes usado para compor grandes cenas de natureza.

Mais Informações

Os projetos do ICE Panorama podem ser exportados para formatos padrão de bitmap, ou podem ser salvos em formatos avançados de multi-resolução usados pelos espectadores do panorama. Eles também podem ser carregados no site PhotoSynth da Microsoft para compartilhar com outros.

NOTA: O ICE é um programa gratuito desenvolvido pela Microsoft Research.

O que é um arquivo .SPJ? (Projeto de apresentação de slides do PhotoStage)

Um arquivo SPJ é um arquivo de projeto criado pelo software de apresentação de slides do NCH PhotoStage. Ele contém uma apresentação multimídia sequenciada que pode incluir texto, formas, imagens, clipes de vídeo, música, narração, efeitos e transições. Os arquivos SPJ armazenam referências a arquivos de mídia importados, como áudio, vídeo e imagens, mas não armazena os arquivos em si.

Mais Informações

PhotoStage Os arquivos do projeto SPJ só são suportados pelo software PhotoStage, que é usado para criar slideshows pessoais e compartilhá-los com familiares e amigos. Você pode abrir projetos SPJ no PhotoStage selecionando File → Open.

Se você quiser compartilhar um projeto com alguém que não possui o PhotoStage, você pode gravar o projeto em um DVD, enviá-lo para o YouTube, Vimeo ou Flick, ou exportá-lo em outro formato de arquivo. Alguns formatos multimídia você pode exportar o projeto para incluir .MP4, .MOV e .AVI.

Se você está movendo um projeto SPJ para outro computador ou fazendo backup, você também precisa copiar os arquivos de mídia reais importados para o projeto, uma vez que eles são referenciados no projeto. Se você selecionar File → Back Up Project Files To Folder..., você pode copiar todos os arquivos referenciados em um único local para backup ou transferência.

O que é um arquivo .SPJ? (Arquivo de trabalho de produção spss)

Arquivo de trabalho criado pela mineração de dados SPSS e software de análise preditiva; faz referência a uma ou mais .SPS de sintaxe de comando que executam o procedimento; permite que as funções SPSS sejam automatizadas e possam ser executadas em horários agendados; pode suprimir o modo interativo para que nenhum popup apareça durante a execução.

Mais Informações

Você pode criar e executar arquivos de trabalho usando a caixa de diálogo Trabalho de Produção, que pode ser aberta selecionando Utilities → Production Job....

NOTA: Os trabalhos de produção criados usando SPSS 15 ou anteriormente (arquivos .SPP) não podem ser executados em SPSS 16 ou posterior.
