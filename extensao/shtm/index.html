O que é um arquivo .shtm? (O lado do servidor HTML inclui arquivo)

Web página que pode fazer referência a outros arquivos no servidor Web; semelhante a um documento HTML padrão, mas pode acessar outros arquivos no servidor quando carregado; permite que um arquivo HTML padrão seja um pouco dinâmico.
