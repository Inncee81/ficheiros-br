O que é um arquivo .mad? (Atalho do módulo de acesso)

Arquivo de atalho usado pelo Microsoft Access, um aplicativo de banco de dados relacional para o ambiente de desktop Windows; links para um módulo de banco de dados que contém procedimentos escritos na linguagem Visual Basic for Applications (VBA); usado em versões mais antigas do Microsoft Access como um atalho para a área de trabalho para abrir o módulo.

Mais Informações

Os arquivos MAD são atalhos para módulos e não incluem o conteúdo adicional em si.

O que é um arquivo .mad? (Arquivo de filme Madcow da Electronic Arts)

Arquivo de vídeo usado por uma variedade de jogos desenvolvidos pela Electronic Arts, como a série Need for Speed e selecionar jogos Sims; contém dados de áudio e vídeo usados para reproduzir sequências de título e logotipo, filmes de abertura, cutscenes no jogo e outros filmes.

Mais Informações

O formato MAD usa pedaços MADk, MADm e MADe, ou conjuntos de dados, que correspondem a diferentes tipos de quadros de vídeo usados para reproduzir vídeo e áudio.

NOTA: Media Player Classic só pode reproduzir o áudio em arquivos MAD e não no vídeo.
