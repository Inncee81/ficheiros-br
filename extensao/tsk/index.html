O que é um arquivo .tsk? (Arquivo top silkscreen)

Um arquivo TSK é um arquivo de dados criado por ferramentas CAD de placa de circuito impresso (PCB) como autodesk EAGLE. Contém dados de layout para a camada superior de serigrafia do PCB. Os arquivos TSK são salvos no formato RS-274X, que é o formato Gerber mais comum e contemporâneo.

Mais Informações

O arquivo TSK permite que você salve a tela de seda impressa em um PCB. Isso inclui as informações baseadas em texto impressas em cima de um PCB que podem ser lidas por humanos. As informações podem ser pontos de teste, logotipos da empresa, configurações de switch, identificadores de componentes ou números de peças.

NOTA: O formato Gerber foi originalmente desenvolvido pela Gerber Systems, mas agora é mantido pela Ucamco.

O que é um arquivo .tsk? (Pele de TTPod)

Um arquivo TSK contém uma pele usada pelo TTPod, um music player para dispositivos Android. Ele armazena uma skin para o jogador que configura a aparência do TTPod player. Os arquivos TSK são armazenados no diretório /SDCard/TTPod/Skin.

Mais Informações

Os arquivos TSK são usados para personalizar o aplicativo TTPod. O arquivo é usado para armazenar skins ou skins padrão criadas por usuários de TTPod. Uma pele inclui a aparência do fundo, menus, texto, botões, linhas e margens.
