O que é um arquivo .ftm? (Fabricante de árvores genealógicas para arquivo DOS)

Árvore genealógica criada com o Family Tree Maker for DOS, uma versão mais antiga do popular programa de criação de árvores genealógicas; contém dados de genealogia que podem ser vistos como uma árvore genealógica.

Mais Informações

Family Tree Maker for Windows salva árvores genealógicas à medida que .FTW arquivos.

O que é um arquivo .ftm? (Módulo FamiTracker)

Módulo rastreador de áudio criado com o FamiTracker, um programa de áudio usado para produzir música para sistemas nintendo (NES); inclui amostras curtas de áudio e uma série de notas que compõem a melodia.

Mais Informações

Os arquivos FTM são salvos em um formato proprietário que só pode ser aberto e editado com o FamiTracker. No entanto, as músicas criadas com o FamiTracker podem ser exportadas para arquivos de .WAV padrão ou .NSF arquivos de áudio.

NOTA: O nome FamiTracker vem de "Famicom," que era o nome japonês para o sistema original da Nintendo.

O que é um arquivo .ftm? (Arquivo de modelo final)

Modelo criado pelo Finale, um programa de notação de música; contém um modelo com configurações pré-carregadas ou personalizadas e notação de música.

Mais Informações

Para criar um arquivo FTM basta selecionar File → Save As... com o documento das configurações desejadas aberto, escolha seu local de salvamento (um local útil é a pasta Finale/Modelos), nomeie o arquivo, selecione o tipo de arquivo FTM e clique em Salve.

Para abrir um arquivo FTM, selecione File → New → Document from Template e clique duas vezes no arquivo FTM.

NOTA: Após o lançamento do Finale 2014, o programa começou a usar arquivos .FTMX.
