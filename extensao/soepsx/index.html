O que é um arquivo .soepsx? (Arquivo de item do player studio viewer)

Um arquivo SOEPSX é um arquivo de item criado pelo Player Studio Viewer (PSV), um aplicativo distribuído com jogos daybreak como jogos EverQuest, EverQuest II, Landmark e PlanetSide 2. Contém um item como capacetes, decalques ou armaduras, mapas de textura e arquivos de geometria. Os arquivos SOEPSX são salvos no formato SONY Online Entertainment Player Studio (SOEPSX).

Mais Informações

Player Studio permite que você crie e edite itens no jogo. Esses itens podem ser carregados no mercado daybreak games para serem vendidos desde que você tenha uma conta válida do Daybreak. Os itens também podem ser criados apenas para importar para o jogo.

Você pode visualizar e editar seus itens com o PSV antes de submetê-los. O aplicativo PSV pode ser encontrado na subpasa "PlayerStudio" do diretório de instalação do jogo. Cada PSV é único para o jogo com o que é empacotado. O PSV é usado para preparar pacotes de submissão e salvar o item no formato SOEPSX. Uma vez que você esteja pronto para enviar seu item para o mercado, você deve tirar uma captura de tela de sua submissão em .JPG formato e nomear a captura de tela para corresponder à submissão.

NOTA: Daybreak Game Company é uma desenvolvedora de jogos americana que anteriormente era conhecida como Sony Online Entertainment.
