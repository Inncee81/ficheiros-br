O que é um arquivo .dex? (Arquivo executável dalvik)

Um arquivo DEX é um arquivo executável salvo em um formato que contém código compilado escrito para Android, a plataforma de telefone celular baseada em Linux do Google. É tecnicamente referido como um "Dalvik Executable"," e pode ser interpretado pela máquina virtual Dalvik.

Mais Informações

OS arquivos DEX podem ser criados manualmente ou traduzindo automaticamente programas Java compilados. Vários arquivos DEX são fechados em um único pacote de .APK, que serve como o arquivo final de Android de aplicativos.

O que é um arquivo .dex? (Arquivo de raios-X Digital DEXIS)

Imagem de raio-X salva no formato de raio-X proprietário; usada pelo sistema de raios-X Digital DEXIS para imagem dentária; captura radiografias dentárias e as salva em formato digital.

Mais Informações

DEXIS foi adquirido pela Danaher Corporation em 2005.

O que é um arquivo .dex? (Arquivo de projeto Disketch)

Projeto de etiqueta de disco criado pelo software de design de etiqueta Disketch; salva o layout de um CD ou DVD e pode incluir texto e imagens importadas; pode ser impresso com uma impressora padrão ou uma impressora que suporta etiquetas de disco.

Mais Informações

O Disketch agora usa a extensão .DEPROJ para projetos de etiquetas de disco. O formato DEX é preterido, mas ainda pode ser aberto pelas versões atuais do Disketch.

Programas que abrem o arquivo .dex (Arquivo de projeto Disketch)

  Windows
  NCH Disketch

O que é um arquivo .dex? (Planilha excel)

Planilha ou grupo de planilhas criadas pelo MS Excel; contém dados formatados em células, que são organizados em linhas e colunas.

Mais Informações

A extensão .XLS é muito mais comumente usada para arquivos Excel.
