O que é um arquivo .lxsproj? (Arquivo de projeto de estúdio XML líquido)

Arquivo de projeto de desenvolvedor criado pelo Liquid XML Studio, um ambiente de desenvolvimento XML; salvo em um formato XML e armazena o nome do projeto, referências a arquivos de projeto e grupos de arquivos; usado para organizar código-fonte para projetos XML.
