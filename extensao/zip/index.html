O que é um arquivo .zip? (MAME Jogo ROM)

Arquivo de jogo usado pelos emuladores de jogos de arcade MAME; pode conter todo o conteúdo dos dados do jogo de arcade, ou pode ser complementado por .CHD arquivo de disco rígido arcade, que é necessário para jogar o jogo; armazenado em um formato .ZIP comprimido.

Mais Informações

MAME ROMs às vezes são difíceis de carregar adequadamente devido ao formato do arquivo e diferenças de software de emulação. Portanto, alguns emuladores são distribuídos com uma ferramenta que verifica se uma ROM pode ser carregada ou não.

O que é um arquivo .zip? (Arquivo com zíper)

Um arquivo ZIP é um arquivo que contém um ou mais arquivos compactados ou "fechados" usando compactação zip. Ele armazena arquivos separadamente um do outro permitindo que os arquivos sejam compactados usando diferentes métodos e extraídos sem compactar ou descomprimir todo o arquivo.

Mais Informações

Captura de tela de um arquivo zip no Microsoft Windows 10 File Explorer

O formato Zip foi criado em 1989 por Phil Katz e foi usado no utilitário PKZIP desenvolvido pela PKWARE, Inc. O formato cresceu em popularidade e agora é suportado pela maioria dos programas de compressão/descompressão de arquivos.

Você pode encontrar arquivos ZIP de várias maneiras, como receber arquivos compactados por e-mail de um amigo ou colega de trabalho, baixar arquivos on-line ou explorar arquivos de backup que foram compactados com compactação zip. Tanto Windows quanto macOS vêm com programas que podem comprimir e descomprimir arquivos ZIP.

Como abro um arquivo ZIP?

Em Windows, você pode abrir um arquivo ZIP com o File Explorer, também conhecido como "Windows Explorer", " simplesmente clicando duas vezes no arquivo. Em seguida, você pode escolher um, múltiplo ou todos os arquivos para extrair do arquivo Zip. No macOS, você pode abrir um arquivo ZIP com o Apple Archive Utility clicando duas vezes no arquivo.

Como criar um arquivo ZIP?

Windows usuários podem criar arquivos ZIP clicando com o botão direito do mouse em um arquivo e selecionando "Enviar para → Compactados (com zíper) Pasta" Os usuários do macOS podem criar arquivos ZIP clicando com o botão direito de clicar em um arquivo e selecionar "Compactar [nome de arquivo]"
