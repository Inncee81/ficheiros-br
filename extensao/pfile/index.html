O que é um arquivo .pfile? (Arquivo protegido para gerenciamento de direitos)

Um arquivo PFILE é um arquivo protegido pelo Microsoft Rights Management (RMS), um aplicativo usado para proteger as informações digitais de um usuário. Ele contém um arquivo, como um arquivo .VSDX ou .DOC, que foi criptografado com arquivos RMS. PFILE são genericamente protegidos enquanto .PPDF, PTXT e arquivos PJPG são protegidos apenas como leitura.

Mais Informações

Se você encontrar um arquivo PFILE em seu computador ou receber um arquivo PFILE como um anexo de e-mail, você deve instalar o RMS para abrir o arquivo. Quando o RMS é instalado, você pode simplesmente clicar duas vezes no arquivo para abri-lo. Você será solicitado a confirmar que está autorizado a visualizar o arquivo com um nome de usuário e senha.

NOTA: O Microsoft Rights Management também é conhecido como Azure Information Protection.
