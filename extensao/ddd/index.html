O que é um arquivo .ddd? (Arquivo do dicionário de dados Alpha Five)

Arquivo dicionário criado por Alpha Five, um IDE que permite que os usuários desenvolvam aplicativos Web, desktop e móveis; contém uma lista de todos os elementos adicionados ao arquivo .DBF; os elementos incluem navegaçãos, relatórios, operações salvas, letras e designs de formulários; funções semelhantes ao formato de arquivo DBF, pois é uma tabela vinculada a um arquivo de memorando .DDM como o arquivo DBF está vinculado ao arquivo de memorando .FPT.

Mais Informações

Um exemplo da relação entre a DBF, DDD e arquivos DDM: Digamos que criamos uma regra de campo no arquivo DBF chamada "Rule" Um registro é então criado no arquivo DDD com Regra no campo de nome, FRUL no campo de tipo (porque é uma regra de campo), e código no campo de memorando que faz referência ao código real no arquivo DDM.

Para cada tabela criada em Alpha Five, será criado um arquivo DDM, DDD e .DDX. Esses arquivos compõem o dicionário de dados de tabela para um arquivo DBF. O dicionário armazena consultas, configurações de utilidade salvas e layouts. É referenciado por Alpha Five para superar limitações do arquivo DBF. Os arquivos DDD devem ser mantidos com os arquivos do Dicionário de Dados DDM e DDX, a fim de que alpha encontre todos os elementos e suas peças.

NOTA: Em 2013, o Alpha Five tornou-se Alfa Anywhere.

O que é um arquivo .ddd? (Arquivo de dados GLBasic 3D)

Arquivo usado pelo GLBasic, um aplicativo usado para desenvolver software cross platform com a linguagem de programação BASIC; salva os vértices de um objeto 3D e pode ser renderizado como parte de uma animação 3D ou como um objeto estático em um programa GLBasic.
