O que é um arquivo .sb2? (Arquivo de projeto scratch 20)

Um arquivo SB2 é um programa criado com o Scratch, uma plataforma de desenvolvimento de aplicativos criada no Massachusetts Institute of Technology (MIT). Ele contém código de linguagem de programação Scratch usado para criar histórias, pequenos jogos e animações. Os arquivos SB2 substituíram os arquivos originais do Scratch .SB.

Captura de tela de um arquivo sb2 no MIT Scratch 2

Os arquivos SB2 são salvos no formato de arquivo Scratch 20. O arquivo SB2 é simplesmente um sprites, scripts, figurinos e arquivos de som. Você pode extrair o conteúdo de um arquivo SB2 com um utilitário zip-compression, como Microsoft File Explorer, Apple Archive Utility, 7-Zip, Corel WinZip ou RARLAB WinRAR.

Para criar um arquivo SB2:

  1. Abra o "Scratch Project Editor" na página do MIT Scratch.
  2. Crie suas animações e selecione File → Download to your computer na janela do navegador.
  3. Nomeie seu arquivo, escolha o local de salvamento e clique em Salve.

NOTA: Um projeto Scratch pode ser compilado como um arquivo Java (arquivo .JAR), que pode ser executado em um navegador da Web.
