O que é um arquivo .fce? (Arquivo de elemento personalizado foreUI)

Arquivo usado pela ForeUI, uma ferramenta de prototipagem de interface de usuário; contém um elemento de interface de usuário personalizado criado por um desenvolvedor; usado para estender os elementos padrão fornecidos pelo aplicativo.

Mais Informações

Os elementos personalizados da ForeUI podem ser direcionados para tipos específicos de interfaces, como iPhones ou Web janelas do navegador.

O que é um arquivo .fce? (Necessidade de arquivo de malha de velocidade)

Um arquivo FCE é um modelo 3D usado pela Need for Speed 1-3 jogos de corrida de carros. Pode conter a malha de um carro ou o painel do carro, que define a forma do carro ou do painel. Os arquivos FCE são armazenados em arquivos .VIV, que são usados pelo jogo para carregar carros de corrida.

Mais Informações

Os arquivos FCE não devem ser abertos diretamente pelos usuários do Need for Speed. No entanto, alguns usuários que gostam de modificar arquivos FCE extraidores de jogabilidade de arquivos VIV, a fim de editar a aparência de carros e painéis no jogo. O utilitário NFS Wizard pode ser usado para extrair arquivos FCE de arquivos VIV e visualizá-los e editá-los.
