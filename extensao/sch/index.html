O que é um arquivo .SCH? (Arquivo de esquemas EAGLE)

Um arquivo SCH contém um esquema, ou diagrama visual, de um PCB (placa de circuito impresso), criado com software de design EAGLE PCB. Armazena símbolos (para portões lógicos) conectados por linhas, que representam ônibus. Os arquivos SCH são usados para criar arquivos reais de placa de circuito (.BRD).

Mais Informações

NOTA: Autodesk EAGLE era anteriormente conhecido como CadSoft EAGLE. O software foi adquirido pela Autodesk em 2016. EAGLE significa "Editor de layout gráfico facilmente aplicável"

O que é um arquivo .SCH? (Arquivo de esquemas ExpressSCH)

Um arquivo SCH contém um esquema, ou diagrama visual, de um PCB de placa de circuito impresso criado com o software de design ExpressSCH. Armazena um esquema eletrônico como está disposto na tela de desenho do programa e inclui símbolos (para portões lógicos) conectados por linhas, que representam ônibus.

Mais Informações

Captura de tela de um arquivo sch no ExpressSCH 7

O tipo de arquivo ExpressSCH Schematics (SCH) é um dos vários tipos de arquivos esquemáticos que usam a extensão de arquivo .sch. O tipo de arquivo ExpressSCH é um formato de arquivo proprietário que não é suportado por outros programas de Esquemas. Você só pode abrir um arquivo ExpressSCH com o software ExpressSCH.

Para abrir um arquivo SCH com o ExpressSCH, selecione File → Open.... Para exportar um arquivo SCH para um arquivo .BMP com o ExpressSCH, selecione File → Export schematic image....

Uma vez que um usuário termina de projetar um esquema eletrônico, ele passa para o processo de design do PCB. Após a criação do PCB, ele é repassado ao processo de fabricação para ser montado e colocado em uso em diversos produtos eletrônicos.

NOTA: O ExpressSCH está incluído no pacote de software ExpressPCB.

O que é um arquivo .SCH? (arquivo esquemagático gEDA)

Arquivo esquemático criado e usado pelo gEDA, uma coleção de ferramentas usadas para projetar layouts PCB e esquemas eletrônicos; contém dados que compõem o esquema; inclui a versão em formato gEDA/gaf, imagens e coordenadas XY para redes, ônibus e componentes.

Mais Informações

Os esquemas eletrônicos consistem em uma coleção de símbolos, ou componentes, redes e ônibus. Eles são usados durante o processo de captura de esquemas no design do circuito e, uma vez criado, ele pode então ser passado para o processo de design da placa de circuito impresso.

O que é um arquivo .SCH? (Arquivo esquemático kicad)

Arquivo criado pelo programa Shematic Capture Editor incluído no KiCad, um conjunto de ferramentas usado para projetar layouts pcb e esquemas eletrônicos; salva um esquema eletrônico à medida que está disposto na tela de desenho do programa.

Mais Informações

Esquemas eletrônicos são usados durante o processo de captura de esquemas no design do circuito. Uma vez criado o esquema eletrônico, ele pode então ser passado para o processo de design da placa de circuito impresso.

Formato Binário 

O que é um arquivo .SCH? (Arquivo de esquema strater)

Arquivo usado pelo Strater, um programa usado para visualizar furos subsuperfícies e dados de registro de poços; especifica a nomenclatura (palavras-chave, intervalo de dados, litologia, etc) e as propriedades de exibição (linha, texto, símbolo, preenchimento, etc) para elementos de design que usam a nomenclatura; permite que os projetos sejam criados de acordo com as normas.

Mais Informações

Os esquemas Strater podem ser criados e editados no Editor de Esquemas, que é acessado selecionando Tools → Scheme Editor... no menu do aplicativo.
