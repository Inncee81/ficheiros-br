O que é um arquivo .rpy? (Python Script)

Script escrito em Python, uma linguagem de programação interpretada, orientada a objetos e extensível; pode conter tanto a sintaxe Python quanto outras sintaxe implementadas para estender python com recursos específicos de aplicativo ou framework; comumente usado para armazenar código-fonte de aplicativo que é executado em servidores Web.

O que é um arquivo .rpy? (Roteiro de Ren'Py)

Roteiro escrito para Ren'Py, um motor de jogo visual usado para criar histórias digitais interativas; pode conter tanto a sintaxe python quanto o Ren'Py Script, que é uma linguagem simples usada para definir cenas novas, transições, texto de diálogo, sons, música e outros elementos da história.

Mais Informações

Os scripts RPY são usados pelo motor Ren'Py para gerar arquivos .RPYC compilados, que o motor usa para executar o jogo.

O que é um arquivo .rpy? (Arquivo de repetição do projeto Touhou)
