O que é um arquivo .gsp? (Arquivo de bloco de esboço do Geometer)

Arquivo usado pelo Sketchpad do Geometer, um programa de software educacional que ensina matemática através de esboços e desenhos; contém um esboço geométrico criado por um aluno.

Mais Informações

Os arquivos GSP são usados pelos alunos em salas de aula para armazenar formas geométricas criativas, desenhos e funções desenvolvidas durante a exploração e aprendizagem.

O que é um arquivo .gsp? (Página do servidor Groovy)

Um arquivo GSP é uma página da Web criada pelo Graals, uma estrutura de aplicativo web baseada em código aberto para a plataforma Java. É semelhante a um analisado em um servidor web que torna o código em HTML e é enviado para o computador de um usuário.

Mais Informações

Como o código Groovy é analisado no servidor web, o usuário final nunca vê o código GSP. O usuário final só verá o HTML gerado pelo código Groovy na página.

Os arquivos GSP são usados para armazenar modelos e layouts para páginas da Web. Os modelos permitem que os usuários criem seções de código que podem ser reutilizadas em várias "visualizações," ou páginas da Web, e os layouts permitem que os usuários criem a estrutura de uma página da Web, que inclui o título da página, o conteúdo da tag principal e o conteúdo da tag corporal. Todos os arquivos GSP estão localizados no diretório grails-app/views.

NOTA: Graals era anteriormente conhecido como "Groovy on Rails"

O que é um arquivo .gsp? (Arquivo de mensagem de erro do iMail)

File sent by IMail Server, an email server program used by small to medium-sized businesses; contains an error message or flag indicating a problem with the email delivery; typically generated by the mail server postmaster.

Mais Informações

Os arquivos GSP são dados em resposta ao remetente de um e-mail quando há um erro.

O que é um arquivo .gsp? (Documento do cartão SD do Estúdio silhueta)

Um arquivo GSP é um design criado pela versão 3 do Silhouette Studio, um programa usado para projetar itens que podem ser cortados com uma máquina de corte eletrônico Silhueta. Ele contém um design, que pode incluir texto, formas, linhas, cores, padrões, gradientes, imagens e efeitos. Os arquivos GSP também armazenam configurações para especificar parâmetros e estilos de corte.

Mais Informações

O arquivo GSP é semelhante ao studio e ao cartão SD. Depois de ser salvo no cartão SD, o cartão pode ser colocado diretamente no leitor de slot SD do cortador Silhueta CAMEO. Isso permite que os designs sejam cortados por um cortador que não está conectado ao seu computador executando o Silhouette Studio.

Para salvar seu design como um arquivo GSP, selecione File → Save As → Save to Hard Drive.... Você também pode salvar seu design como um arquivo GSP diretamente para o seu cartão SD bby selecionando File → Save As → Save to SD Card....

NOTA: Embora o arquivo GSP seja criado pelo Silhouette Studio, ele não pode ser aberto pelo programa. Ele foi projetado para ser usado apenas pela ferramenta de corte CAMEO.
