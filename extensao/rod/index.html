O que é um arquivo .rod? (Arquivo de design de objeto de relatório de atuação)

Projeto de relatório criado com software Actuate, como Actuate eReport Designer; inclui layouts de formulário, campos, formulários e outras informações; usado para personalizar a saída de relatórios gerados com o software Actuate.

Mais Informações

Os relatórios executáveis são salvos à medida que .ROX arquivos.

O que é um arquivo .rod? (Arquivo de banco de dados genealógico de ancestralidade)

Banco de dados usado pela Ancestry, um programa usado para acompanhar informações genealógicas; contém dados sobre a linha genealógica do usuário, que inclui nomes de família, relacionamentos, datas de nascimento, fotos e aniversários.

Mais Informações

Ancestry, que originalmente se chamava Rodokmen, foi projetado para ajudá-lo a organizar suas informações genealógicas e gerar árvores genealógicas. O arquivo ROD é usado para armazenar e transferir suas informações genealógicas.

NOTA: A extensão do arquivo ROD foi substituída pela extensão do arquivo .RODX, que foi então substituída pela extensão .RODZ.

Arquivo de banco de dados do escritório RIB

Desenvolvedor RIB Software
Categoria Base de Dados
Formato Binário 

O que é um arquivo .rod? (Arquivo de banco de dados do escritório RIB)

Arquivo de banco de dados usado pelo escritório RIB, um aplicativo de gerenciamento de projetos para planejadores e engenheiros em construção civil, engenharia civil e construção de estradas; armazena todos os dados de escritório no formato de banco de dados Progress ObjectStore; usado como o formato backend para salvar dados de escritórios da RIB.

Mais Informações

Os arquivos ROD podem conter informações como dados de gerenciamento de tempo, desenhos, informações dos funcionários, contatos, dados de contratos e faturas.

NOTA: O escritório da RIB não está mais disponível.
