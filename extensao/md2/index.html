O que é um arquivo .md2? (Segmento de imagem do disco de álcool 2)

O terceiro segmento de uma imagem de CD ou DVD copiado pelo software Alcohol 120% ; parte de uma sequência de segmentos de imagem de disco (cada parte termina em .MD0, .MD1, depois MD2, etc); deve ser juntado aos outros segmentos para recuperar a imagem completa do disco.

Mais Informações

Os arquivos MD2 são frequentemente salvos com um arquivo .MDF ou um arquivo .MDS, que define as faixas e a estrutura de dados usadas pelo disco.
