O que é um arquivo .exe4j? (Arquivo de configuração Exe4j)

Arquivo criado pelo Exe4j, um programa usado para criar arquivos Windows .EXE usando arquivos .JAR; contém configurações de configuração de aplicativos, como a tela de respingos, opções de compilação, opções de invocação java e informações executáveis; permite que os aplicativos Java sejam gerados em Windows arquivos executáveis.

Mais Informações

Os aplicativos Exe4j podem ser criados usando o assistente Exe4j ou o programa de linha de comando exe4jc.exe, que é encontrado no diretório bin da instalação exe4j.

NOTA: Os arquivos EXE criados pela Exe4j exigem que uma instalação java Runtime Environment seja executada corretamente.
