O que é um arquivo .car? (Arquivo de recurso de aplicativo de componentes brew)

Arquivo CAR (Component Application Resource, recurso de aplicativo componente) usado por aplicativos desenvolvidos para a Brew Mobile Platform (MP); descreve ativos disponíveis para um aplicativo de execução; compilados em um arquivo binário de .BAR carregado quando um aplicativo é executado.

Mais Informações

Os recursos de aplicação nos programas de desenvolvimento BREW 3x e 4x mais antigos utilizam a extensão .BRX. No entanto, os arquivos BRX foram preteridos com o lançamento do Brew MP. Você pode converter arquivos BRX em arquivos CAR usando o Brew MP SDK.

NOTA: O Brew MP SDK é na verdade um conjunto de programas e plug-ins. Por exemplo, o SDK inclui plug-ins de desenvolvimento Brew MP para Microsoft Visual Studio, Adobe Flash e Eclipse.

O que é um arquivo .car? (Arquivo de dados do Explorador de Carros)

Arquivo de dados usado pelo Car Explorer, um programa projetado para visualizar informações sobre diferentes modelos de carros; salvo no formato eXtensible Car Description (XCD), uma especificação padronizada para armazenamento de dados de automóveis; inclui informações como make e modelo, ano, preço, dimensões físicas e imagens relacionadas.

Mais Informações

Por design, os arquivos CAR armazenam informações sobre todos os modelos para uma make (ou empresa) a partir de um único ano. Arquivos de carros são frequentemente distribuídos como coleções, ou um arquivo que contém vários arquivos CAR em um formato compactado.

NOTA: O aplicativo Car Explorer não está mais disponível.

O que é um arquivo .car? (Arquivo do carro gary gadget)

Arquivo criado por Gary Gadget: Building Cars, um jogo infantil que permite aos usuários projetar, construir e dirigir carros; contém números de peças de carro para o carro salvo construído pelo usuário; inclui quaisquer medalhas conquistadas com o carro.

Mais Informações

NOTA: Os arquivos CAR estão localizados na Autos pasta do diretório do jogo. Infelizmente, o programa Viva Media Gary Gadget: Building Cars não está mais disponível para download.
