O que é um arquivo .info? (Arquivo de informações do módulo Drupal)

Arquivo usado pelo Drupal, um sistema de gerenciamento de conteúdo de site; armazena informações sobre um módulo Drupal (arquivo .MODULE), que é um plugin que adiciona recursos à instalação base do Drupal; contém o nome do módulo, descrição, a versão drupal central suportada pelo módulo e quaisquer dependências exigidas pelo módulo.

Mais Informações

Um arquivo INFO com o mesmo prefixo de nome de arquivo é necessário para que o Drupal reconheça o módulo. Por exemplo, mymodule.module e mymodule.info são nomes válidos. Ambos os arquivos devem ser instalados no seguinte diretório, onde mysite é o seu site (que também pode ser o diretório all):

sites/mysite/modules/

O que é um arquivo .info? (Arquivo de informações genéricas)

Um arquivo INFO é um arquivo de informações de texto genérico que pode ser incluído com vários programas de software ou downloads de arquivos. Ele contém texto simples não formado que descreve o conteúdo do software ou download. Os arquivos INFO são frequentemente usados para documentar informações de versão, autoria e informações de direitos autorais.

Mais Informações

Uma vez que os arquivos INFO são salvos em um formato de texto, você pode usar um simples editor de texto para abrir um arquivo INFO. Microsoft Notepad ou WordPad podem ser usados em Windows, Apple TextEdit pode ser usado no macOS, e gedit pode ser usado em Linux.

O que é um arquivo .info? (Arquivo de informações do Minecraft Forge)

Um arquivo INFO é um arquivo de dados usado pelo Minecraft Forge, uma ferramenta que permite que os usuários instalem mods minecraft. Ele contém informações de texto simples no formato JSON que descreve um ou mais mods e suas propriedades.

Mais Informações

NOTA: O arquivo é armazenado em um .JAR ou .ZIP arquivo que pode ser descompactado por programas como 7-Zip e WinRAR. Uma vez que o arquivo INFO contém texto simples, ele pode ser editado por qualquer editor de texto.

O que é um arquivo .info? (Documento texinfo)

Um arquivo INFO é um documento de informação formatado criado pela Texinfo, uma ferramenta de geração de documentos do Projeto GNU. Armazena documentação em formato de livro com capítulos, seções e subseções e pode conter até quatro camadas de profundidade.

Mais Informações

Documentos INFO são usados para armazenar manuais de ajuda de software e livros técnicos. Eles podem ser vistos com um visualizador de informações, como o leitor de informações da Texinfo ou o GNU Emacs.

O que é um arquivo .info? (Arquivo de índice de imagem ZoomBrowser)

Armazena informações sobre imagens salvas em um álbum de fotos do Zoom Browser; inclui dados de imagem em miniatura, que permite a navegação rápida de imagens no álbum; criados quando um ou mais arquivos de imagem são abertos pelo ZoomBrowser.

Mais Informações

O arquivo de índice ZoomBrowser é tipicamente chamado "Zbthumbnailinfo" e é salvo no mesmo diretório das imagens a que corresponde. O arquivo Zbthumbnailinfo geralmente está oculto, mas pode aparecer dentro da pasta do álbum, dependendo das configurações de visualização de arquivos do computador.

NOTA: A Canon ZoomBrowser EX não está mais disponível para download.
