O que é um arquivo .blz? (Imagem de bitmap compactado)

Gráfico raster que usa um algoritmo de compressão para reduzir o tamanho do arquivo de imagem; semelhante a um arquivo padrão .BMP, mas compactado; às vezes usado para empacotar imagens em videogames.

O que é um arquivo .blz? (Arquivo de jogo de vídeo móvel N-Gage)

Arquivo de contêiner de videogame que armazena um jogo desenvolvido pela N-Gage, um serviço de distribuição de jogos para celulares Nokia; contém todos os dados embalados para o jogo; pode ser instalado após transferir o arquivo para o telefone com o software Nokia PC Suite.

Mais Informações

Uma vez que os arquivos BLZ tenham sido transferidos para o celular, eles podem ser instalados usando o aplicativo BLZ Installer, que pode ser carregado e instalado em um telefone Nokia. Os arquivos BLZ também podem ser extraídos no seu PC usando o utilitário BLZ Extractor.

N-Gage jogos também são distribuídos usando a extensão de arquivo .N-GAGE.

NOTA: Os arquivos BLZ às vezes são carregados em outros dispositivos móveis baseados no Sistema Operacional Symbian para jogar jogos. No entanto, esse processo não é oficial e não é suportado pela Nokia.

O que é um arquivo .blz? (Arquivo de sessão TeamViewer Blizz)

Um arquivo BLZ é um arquivo de vídeo criado pelo TeamViewer Blizz, um programa usado para colaborar com outros usuários através de vídeo, voz, mensagens instantâneas e compartilhamento de tela. Contém uma gravação de vídeo de uma sessão. Os arquivos BLZ são semelhantes aos arquivos .TVS criados pelo TeamViewer.

Mais Informações

Os arquivos BLZ só são suportados pelo TeamViewer Blizz. Para compartilhar uma gravação criada em Blizz com usuários não-Blizz, salve a gravação como um arquivo de .AVI em Blizz. Você também pode carregar um arquivo BLZ para o programa on-line MPS para MP4 e converter o vídeo gravado em um arquivo .MP4.
