O que é um arquivo .tko? (Arquivo do módulo de software AMX)

Arquivo usado pelo software AMX ou Netlinx, que é usado para desenvolver e controlar displays de museus; contém um módulo que pode ser adicionado ao software; usado para atualizar ou atualizar componentes.

O que é um arquivo .tko? (Arquivo win32/oficla trojan)

Arquivo de vírus de Trojan gerado por um dos trojans da família Win32/Oficla; contém código executável que pode prejudicar um sistema de computador; tenta baixar um arquivo .DLL e instalar em um sistema de Windows.

Mais Informações

O arquivo pful.tko é conhecido por abrigar o trojan Win32/Oficla. Depois de infectar um sistema, o trojan se copia para a pasta do sistema Windows com este nome ou um dos muitos outros nomes gerados aleatoriamente.
