O que é um arquivo .lld? (Logotipo! Arquivo SOFT Comfort LAD)

Arquivo criado pelo Logo! Soft Comfort, um aplicativo de automação usado para criar programas de circuito; contém um "Diagrama de escada (LAD)" que exibe as diferentes partes do programa de circuito; inclui entradas, texto rico, saídas, temporizadores, contadores e outras funções.

Mais Informações

O Logotipo! O programa Soft Comfort possui uma interface "drawing board" que organiza o posicionamento e a vinculação de diferentes elementos do programa para criar e testar programas complexos de controle. O programa oferece inúmeros recursos, como formatação de texto, edição de diagramas e ferramentas de simulação.

Para criar um arquivo LLD, selecione File → New → Ladder diagram (LAD), construa seu diagrama, selecione File → Save or Save As..., escolha o local de salvamento (ele é padrão para a pasta "Program"), nomeie o arquivo e clique em Salve.

Para abrir um arquivo LLD, basta clicar duas vezes no arquivo ou selecionar File → Open... no aplicativo, navegar para o seu arquivo e clicar em Abrir.

Para converter seu tipo de diagrama de Escada em Bloco de Função, selecione File → Convert to FBD. Isso mudará o layout do seu diagrama e você salvará o arquivo como um arquivo .LSC.

NOTA: O arquivo LLD é semelhante a um arquivo LSC, que contém um diagrama de bloco de função. Se você está acostumado a trabalhar com caixas lógicas de álgebra booleana, você deve usar o diagrama de bloco de função.
