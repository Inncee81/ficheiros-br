O que é um arquivo .dmg? (Imagem de disco do Apple)

Um arquivo DMG é uma imagem de disco montável usada para distribuir software para o sistema operacional macOS. O arquivo é normalmente baixado da Internet por um usuário Mac e, em seguida, clicado duas vezes para instalar um aplicativo no computador.

Mais Informações

Hoje, os desenvolvedores de software normalmente constroem e distribuem aplicativos para usuários Mac como DMG, monta um disco virtual na área de trabalho e em uma janela do Apple Finder como se fosse um CD ou uma unidade flash USB que foi apenas inserido no computador.

Na janela Finder, o usuário Mac pode clicar duas vezes no ícone do programa, que é um arquivo APP, para executar o aplicativo. Ou o Mac usuário pode arrastar o ícone do programa para o ícone da pasta "Aplicativos" para movê-lo para o diretório "Applications" no macOS.

Depois que o software armazenado no arquivo DMG é instalado, o arquivo DMG não é mais necessário. Portanto, os arquivos DMG podem ser excluídos após a instalação do software.

Enquanto os arquivos DMG são específicos do Mac e não são destinados a uso em Windows, existem vários programas Windows que podem extrair seus conteúdos ou convertê-los em outros formatos. Alguns programas incluem 7-Zip, PeaZip e DMG2IMG.

NOTA: O formato DMG substituiu o formato de arquivo .IMG, que antes era usado em Mac OS Classic.
