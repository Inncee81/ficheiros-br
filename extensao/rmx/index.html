O que é um arquivo .rmx? (Formato RealJukebox)

Arquivo de áudio protegido por cópias usado pelo RealPlayer, um programa de reprodução e streaming de música; armazena com segurança os dados de áudio para que eles não possam ser reproduzidos a menos que a chave de segurança seja encontrada; possam ser vinculados a um determinado computador ou chave.

Mais Informações

Os arquivos RMX são usados pelo RealPlayer 10 ou posterior.

O que é um arquivo .rmx? (Arquivo de índice do gerente de referência)

Arquivo usado pelo Reference Manager, um programa usado para a criação de biografias; contém um índice dos dados armazenados na biografia e requer um arquivo .RMD separado, que contém os dados para as referências; permite que as informações sejam pesquisadas.
