O que é um arquivo .am? (Arquivo de mídia Anark)

Arquivo criado pelo Anark Media Studio, um programa usado para publicar apresentações multimídia interativas; usado como um formato de publicação que pode ser implantado em uma página da Web e visualizado com um navegador Web que tem o plugin Anark Client instalado; também pode ser usado para reprodução autônoma com o Anark Media Player.

O que é um arquivo .am? (Modelo de makefile de arquivo automático)

Arquivo usado pela Automake, um programa usado para gerar Makefiles; contém um modelo Makefile, que inclui as informações necessárias para gerar sintaxe Makefile válida; ajuda os programadores a definir um Makefile usando uma linguagem de ordem mais alta em vez de ter que escrever sintaxe Makefile de baixo nível.

Mais Informações

NOTA: A Makefile declara como compilar e vincular um programa e é executado usando o comando Unix make. Makefiles tem o nome do arquivo Makefile.
