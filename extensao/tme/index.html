O que é um arquivo .tme? (Arquivo hora de Bryce)

Arquivo de animação usado por Bryce, um programa de modelagem 3D; contém configurações de movimento para a animação de um modelo 3D; usado para armazenar curvas de tempo personalizadas padrão ou definidas pelo usuário.

Mais Informações

As curvas de tempo definidas pelo usuário são armazenadas no Accel Engine.tme. As curvas de tempo padrão incluídas no aplicativo são armazenadas no arquivo Def Accel Engine.tme. Ambos os arquivos estão localizados no diretório /Presets/ da instalação bryce.
