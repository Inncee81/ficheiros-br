O que é um arquivo .ras? (Arquivo do sistema de arquivo de remédio)

Arquivo de recursos de jogo usado por videogames Max Payne; contém informações padrão referenciadas pelo jogo, como texturas, modelos, dados de nível e música; semelhantes a arquivos .MPM.

Mais Informações

NOTA: Um arquivo RAS que contém música (pode aparecer como x_music.ras) pode ser aberto pela Audacity. Para fazer isso, selecione File → Import → Raw Data... e escolha o arquivo RAS.

Gráfico de Sun Raster

Desenvolvedor Sun Microsystems
Categoria Imagens Rasterizadas
Formato Binário 

O que é um arquivo .ras? (Gráfico de Sun Raster)

Formato de imagem raster usado pelos computadores Sun Microsystems, tipicamente criado em uma estação de trabalho Unix; suporta 1, 8, 24 e 32 bits por pixel; pode ser descomprimido ou comprimido usando compactação RLE; reconhecido pela maioria dos programas de visualização de imagens.
