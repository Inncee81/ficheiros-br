O que é um arquivo .lvix? (Arquivo de índice de vídeo lightworks)

Arquivo criado pela Lightworks, um programa que permite editar vídeo; contém metadados de vídeo e áudio sobre .MP4 e .MOV arquivos de mídia usados pela Lightworks; referenciado pela Lightworks para refinar o fluxo de trabalho de edição; semelhante ao arquivo .MXF.

Mais Informações

Depois que um vídeo é importado para o Lightworks, o arquivo LVIX é criado. Ele estará localizado na pasta Lightworks "Material" com o mesmo nome do vídeo importado, mas com a extensão LVIX anexada ao final. Se o vídeo importado for nomeado V00P00J.mp4, o novo arquivo seria V00P00J.mp4.lvix.

NOTA: .VIX arquivos são os mesmos que arquivos LVIX, mas são usados para .MTS arquivos de mídia.
