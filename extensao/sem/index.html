O que é um arquivo .sem? (Arquivo de memorando do dicionário de dados Alpha Five Set)

Arquivo de memorando criado pela Alpha Five, um IDE que permite que os usuários desenvolvam aplicativos Web, desktop e móveis; contém dados no formato de memorando, que é o mesmo formato usado por arquivos .FPT e .DDM; inclui informações de design e código para quaisquer objetos, relatórios e formulários anexados ao conjunto; criados juntamente com .SET e .SEX arquivos de conjunto vinculados.

Mais Informações

Quando duas ou mais tabelas são ligadas juntas em Alpha Five, um conjunto, que consiste em um arquivo SEM, SET e SEX é criado. Esses arquivos compõem o dicionário de dados definido para as duas ou mais tabelas ligadas. O dicionário armazena relatórios, formulários e quaisquer outros objetos ligados ao conjunto. Os arquivos SEM devem ser mantidos com os arquivos SET e SEX Set Data Dictionary para que Alpha encontre todos os elementos e suas peças.

SET arquivos são semelhantes aos arquivos .DBF porque eles têm uma limitação de largura para quantos caracteres podem ser mantidos dentro de uma coluna. Assim, o Alpha Five cria um campo de dez caracteres em arquivos SET que faz referência ao local do memorando no arquivo SEM que contém as informações reais.
