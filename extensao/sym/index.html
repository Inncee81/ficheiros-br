O que é um arquivo .sym? (arquivo símbolo gEDA)

Arquivo símbolo criado e usado pelo gEDA, um conjunto CAD usado para projetar layouts PCB e esquemas eletrônicos; armazena dados de símbolos, que inclui a versão do formato gEDA/gaf e coordenadas XY para linhas, caixas, círculos, arcos, texto e pinos.

Mais Informações

O formato de arquivo SYM é semelhante ao arquivo .SCH usado pelo gEDA. Ambos armazenam a versão do formato, que é a primeira linha do arquivo, e contêm objetos, com dados relacionados. Ambos os arquivos SYM e SCH são usados pelo gEDA para criar o layout de circuitos eletrônicos.

O que é um arquivo .sym? (Arquivo símbolos)

Arquivo desenvolvedor usado por aplicativos de desenvolvimento de software; salva informações não essenciais para os arquivos executáveis binários do programa real, mas contém símbolos que podem ser usados para coletar informações do programa durante a depuração; podem incluir nomes de função e variáveis e outras informações úteis de depuração.

O que é um arquivo .sym? (Arquivo de símbolos TatukGIS)

Arquivo usado pelo software de mapeamento TatukGIS; salva um símbolo, que é uma forma ou objeto que pode ser colocado no mapa e dimensionado de acordo com as coordenadas do sistema de mapas; usado para marcar limites e pontos de interesse.
