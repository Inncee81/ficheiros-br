O que é um arquivo .cmmtpl? (Modelo do fabricante de menus camtasia)

Modelo de produção de vídeo usado pelo Camtasia MenuMaker, um programa incluído no software de gravação de tela do Camtasia Studio; contém as configurações padrão para criar um projeto .CMMP MenuMaker; pode ser usado para criar vários menus de vídeo com o mesmo visual e sensação.
