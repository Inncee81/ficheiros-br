O que é um arquivo .emlx? (Mensagem do Apple Mail)

Um arquivo EMLX contém uma mensagem de e-mail individual criada pelo Apple Mail, um programa de e-mail empacotado com o macOS. Ele armazena o conteúdo de uma mensagem, que inclui o assunto, remetente e destinatário. Os arquivos EMLX não armazenam anexos de e-mail, apenas as informações da mensagem.

Mais Informações

Captura de tela de um arquivo emlx no Apple Mail 8

Os arquivos EMLX criados pelo Apple Mail não devem ser abertos manualmente. No entanto, se você encontrar um arquivo EMLX, você pode clicar duas vezes nele para abri-lo, desde que o Apple Mail seja definido como o programa padrão.

Uma vez que os arquivos EMLX são salvos em texto simples, você também pode abrir o arquivo para visualizar o conteúdo em um editor de texto, como Apple TextEdit ou Microsoft Notepad. Você pode precisar alterar a extensão do arquivo .emlx para .txt.

Quando um arquivo EMLX é criado pelo Apple Mail, o arquivo é normalmente salvo em uma das seguintes pastas:

  • ~user/Library/Mail/Mailboxes/[mailbox]/Messages/
  • ~user/Library/Mail/[account]/INBOX.mbox/Messages/
