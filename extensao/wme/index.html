O que é um arquivo .wme? (Arquivo de sessão de codificador de mídia Windows)

Um arquivo WME é um perfil de sessão criado pela Microsoft Windows Media Encoder 9 Series, uma ferramenta de produção usada para converter áudio e vídeo que é gravado ao vivo ou pré-gravado em arquivos de mídia Windows (.WMV ou .WMA arquivos) ou streams. Ele contém configurações para uma sessão de codificação que pode ser usada para outras sessões.

Mais Informações

Se você se deparar com o arquivo WME no seu computador, provavelmente já usou a Série Microsoft Windows Media Encoder 9 ou pelo menos tê-lo instalado. Normalmente, um arquivo WME é criado quando um usuário seleciona File → Save... na interface do programa. O programa vem com um Assistente de Sessão Nova, que guia você através das etapas de configuração de uma sessão de codificação.

No entanto, o programa também vem com arquivos WME de início rápido que contêm configurações para cenários comuns de codificação, a fim de que você comece mais rápido. Esses arquivos de início rápido podem ser considerados arquivos WME de modelo.

NOTA: A série Microsoft Windows Media Encoder 9 não é mais desenvolvida. Foi substituído pelo Microsoft Expression Encoder.
