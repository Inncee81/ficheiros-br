O que é um arquivo .nsf? (Banco de dados de notas lotus)

Arquivo de banco de dados usado pelo software de negócios IBM Lotus; salvo no formato NSF (Notes Storage Facility, centro de armazenamento de notas), que armazena dados em unidades de documentos chamadas "notes;" usado para armazenar informações de negócios colaborativas, como dados de correio e calendário.

Mais Informações

Os arquivos NSF são frequentemente hospedados por um servidor Lotus Domino, e podem ser baixados pelos clientes da Lotus Notes para trabalhar com um banco de dados totalmente funcional no modo offline. Se o cliente fizer alterações no banco de dados, a Domino inclui as facilidades para sincronizar as alterações de volta ao banco de dados mestre. Os arquivos NSF também podem ser usados para o processo de replicação de dados.

NOTA: Os arquivos NSF também podem servir como um aplicativo Domino Web hospedado em um servidor Web, onde a URL tem uma sequência "nsf". Esses arquivos incluem o Domino Web Service Design Element, que permite Web acesso.

O que é um arquivo .nsf? (Arquivo de formato de som NES)

Arquivo de música de um jogo original da Nintendo (NES/Famicom); permite que a música dos jogos da Nintendo seja jogada sem carregar o jogo real; pode ser aberta por uma variedade de jogadores da NSF.
