O que é um arquivo .trk? (Arquivo de trilha terrestre compeGPS)

Arquivo criado pelo CompeGPS Land, um programa usado para interagir com dispositivos GPS CompeGPS; contém uma série de locais que são sobrepostos em um mapa; usados para planejar rotas e carregá-las em dispositivos CompeGPS.

O que é um arquivo .trk? (Arquivo de mapeamento de faixa final)

Arquivo criado pelo Finale, um aplicativo de notação de música; contém dados de configuração da equipe sequencer track/channel-to-Finale que é referenciado ao abrir um arquivo .MID no Finale; os dados de configuração ajudam a traduzir as faixas do arquivo MID para finale.

Mais Informações

Para criar um arquivo TRK, abra o arquivo MID desejado selecionando File → Open... e clicando duas vezes no arquivo MID, clique em Defina a lista de faixas para funcionários, personalize a configuração da faixa/equipe e clique em Save.

Para abrir um arquivo TRK, primeiro abra seu arquivo MID selecionando File → Open..., selecione "Arquivo MIDI padrão" no menu suspenso, clique duas vezes no arquivo MID, clique em Lista de acompanhamento para funcionário, clique em Loade clique duas vezes no arquivo TRK que deseja abrir.

O que é um arquivo .trk? (Arquivo garmin track)

Arquivo criado por dispositivos GPS Garmin, incluindo os modelos PCX5, PCX5AVD e PC150; contém uma série de locais, ou pontos de rota, que são conectados em tempo real enquanto um usuário navega com uma unidade GPS; inclui versão de software, sistema de coordenadas, latitude, longitude, data e altitude.

Mais Informações

Arquivos TRK são usados para gravar registros de viagem. Eles são armazenados em um formato de texto ASCII.

O que é um arquivo .trk? (Arquivo de faixa offline do Pacific Fighters)

Arquivo criado pelo Pacific Fighters, um jogo de simulador de voo; contém uma gravação de uma missão de voo no modo "offline"; criado clicando no botão Save Track quando uma missão está completa; pode ser reproduzido no Pacific Fighter.

Mais Informações

As faixas gravadas no modo "online" são armazenadas em .NTRK arquivos. Ao contrário dos arquivos NTRK, os arquivos TRK armazenam apenas posições iniciais e entradas do jogador em vez de todas as posições do objeto durante uma missão.

NOTA: Os arquivos TRK não contêm dados de vídeo. Eles só armazenam as posições iniciais e as entradas de controle do jogador.
