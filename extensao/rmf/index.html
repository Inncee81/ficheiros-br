O que é um arquivo .rmf? (Documento de gerenciamento de direitos da Adobe)

Licenciar arquivo de tecla usado por produtos da Adobe, como Acrobat e Reader; salva um arquivo de licença para um documento de .PDF bloqueado e usa o mesmo prefixo de nome de arquivo que o PDF; permite que o PDF seja lido pelo comprador do documento.

Mais Informações

Os arquivos RMF às vezes são distribuídos com compras de eBooks e revistas digitais compradas em lojas online. Você precisará do arquivo RMF para abrir o documento PDF correspondente.

O que é um arquivo .rmf? (Formato do mapa rico em martelo)

Arquivo de mapa criado pelo Valve Hammer Editor, uma ferramenta de mapeamento incluída no SDK de origem da válvula; contém dados hammer, como formas arbitrárias de escova e VisGroups (Grupos de visibilidade ajudam você a gerenciar objetos dando-lhe controle do que você vê na tela); pode ser compilado em .MAP e .BSP arquivos.

Mais Informações

Para criar um arquivo RMF, basta selecionar File → Save selecionar o formato RMF e nomear seu arquivo. Você pode escolher o local de salvamento do seu arquivo RMF, mas o local de salvamento padrão dos arquivos RMF criados pelo Hammer Editor está na pasta "Maps" dentro da pasta "Valve Hammer Editor".

Categoria Áudio
Formato Binário 

O que é um arquivo .rmf? (Arquivo de áudio de formato de música rico)

Arquivo de áudio criado no Rich Music Format, um formato de áudio criado pela Beatnik; usado como formato de contêiner para arquivos MIDI e outros formatos de áudio padrão; pode ser criptografado para proteger a música original; usado por alguns videogames mais antigos.

Mais Informações

O Editor Beatnik, editor original da RMF lançado pela Beatnik, não está mais disponível. No entanto, o formato RMF ainda é suportado pelo Java Sound e pelo Java Media Framework.
