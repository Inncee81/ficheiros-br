O que é um arquivo .icma? (Arquivo de atribuição de incopy)

Um arquivo ICMA é um arquivo de atribuição usado pelo InCopy, um processador de texto que se integra ao Adobe InDesign. Ele contém links para um subconjunto de um documento pai InDesign (arquivo .INDD), bem como permissões de edição para um usuário específico. Os arquivos ICMA são usados para atribuir uma parte de um documento a um editor.

Mais Informações

Os arquivos ICMA são criados usando o InDesign e, em seguida, atribuídos a um editor específico, que tem permissão para editar o conteúdo usando o InCopy. Uma vez terminado, o editor notifica o remetente. Esse fluxo de trabalho baseado em atribuição tem vários benefícios:

  1. Separando o fluxo de trabalho do layout da página do fluxo de trabalho de desenvolvimento de conteúdo.
  2. Dividindo tarefas de edição entre vários editores.
  3. Impedir o acesso indesejado ao conteúdo de páginas protegidas, como o layout mestre.
  4. Reduzindo o tamanho do arquivo e do projeto enviados aos editores.
  5. Habilitando um sistema de check-in/check-out para que apenas um editor possa acessar o conteúdo da página designada por vez.
  6. Acompanhar os trabalhadores que modificaram o documento.
Os arquivos ICMA são exportados com um ou mais arquivos .ICML, que contêm o texto, gráficos e quadros que podem ser modificados. Quando os usuários "edit" um arquivo de atribuição ICMA, eles estão realmente modificando arquivos ICML nos bastidores. Os arquivos IMCL são normalmente armazenados e acessados em um arquivo de rede.

NOTA: Os arquivos DO ICMA substituíram .INCA arquivos no InDesign e no InCopy CS4 e posteriormente.
