O que é um arquivo .SCX? (Arquivo de cenário de jogo)

Arquivo de cenário contendo dados de nível ou missão para um videogame específico; pode incluir mapas, caracteres, objetivos e outros dados; normalmente instalado dentro do mesmo diretório do próprio jogo; semelhante a um arquivo de missão (.MIS) usado por outros jogos.

Mais Informações

Arquivos SCX são usados por Maxis Sim City 2000, LucasArts Star Wars Galactic Battlegrounds e o pacote de expansão Blizzard StarCraft Brood War.

O que é um arquivo .SCX? (Arquivo de cenário de jogo da Microsoft)

Arquivo de cenário usado por jogos de estratégia da Microsoft em tempo real; contém dados de nível ou campanha para missões específicas; normalmente instalado dentro do mesmo diretório do próprio jogo.

Mais Informações

Arquivos SCX são usados pela Microsoft Rise of Nations, Age of Empires II e Age of Mythology.

O que é um arquivo .SCX? (Projeto de Ajuda do SuperCard)

Arquivo de projeto criado pelo SuperCard, uma ferramenta de autoria visual para Mac OS X; armazena dados para um projeto de ajuda, que contém conteúdo de ajuda indexável; semelhante a um arquivo .SCH, mas também inclui funcionalidades adicionais fornecidas por recursos de código compilados com o Apple Xcode para a interface de comando/função externa do SuperCard.

Mais Informações

Arquivos SCX são usados para estender tanto o SuperCard quanto o Sistema de Ajuda ao SuperCard.
