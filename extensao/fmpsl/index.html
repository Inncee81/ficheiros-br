O que é um arquivo .fmpsl? (FileMaker Pro 12 Snapshot Link)

Arquivo instantâneo de banco de dados criado pelo software de desenvolvimento de banco de dados FileMaker Pro 12; salva o estado da exibição atual do banco de dados; inclui os registros selecionados, o layout visual e outras informações visíveis; muitas vezes usado para enviar relatórios.

Mais Informações

Arquivos FMPSL permitem que instantâneos sejam reconstruídos em outra cópia do FileMaker, ou com FileMaker Go, um aplicativo para usuários de iPhone e iPad. Links de snapshot podem ser criados com FileMaker Pro e FileMaker Go. Para criar no FileMaker Pro, escolha File → Save/Send Records As → Snapshot Link....

NOTA: Links instantâneos foram introduzidos com o FileMaker Pro 11, que usa a extensão .FPSL em vez da extensão FMPSL, que é usada na versão 12.
