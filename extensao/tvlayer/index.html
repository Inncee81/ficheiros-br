O que é um arquivo .tvlayer? (mimoLive TV Layer)

Um arquivo TVLAYER é uma camada usada pela BoinxTV, um aplicativo que permite que os usuários produzam transmissões como palestras, sermões, noticiários, podcasts e muito mais. Ele armazena informações de camadas, como duração de transição, que configura como os gráficos são exibidos na tela durante uma transmissão. Os arquivos TVLAYER podem ser exportados à medida que .QTZ arquivos.

Mais Informações

mimoLive permite que você coloque camadas sobre seu vídeo para criar uma apresentação mais envolvente e interativa. O programa vem com várias camadas pré-construídas, que incluem clima, gráficos 3D, alertas de notícias de última hora, relógios, iOS demos de dispositivos, inserções de overshoulder, Twitter, Facebook, Twitch e legendas. Cada camada é armazenada no arquivo TVLAYER. Você pode importar novas camadas selecionando File → Import Layer Compositions....

mimoLive era anteriormente conhecido como BoinxTV. O aplicativo é usado principalmente para transmitir vídeo ao vivo para sites como YouTube, Facebook e Twitchtv. Ele também é usado para projetar vídeo para uma tela externa via VGA, DVI ou HDMI. Você pode usar vídeo do seu computador, webcams, câmeras de vídeo conectadas e dispositivos móveis. O software também vem com ferramentas de produção úteis que incluem suporte para múltiplos ângulos de câmera, gráficos, recursos de chaveamento e integração de mídia social.
