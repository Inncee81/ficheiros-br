O que é um arquivo .cos? (Capturar um arquivo de configurações)

Arquivo de configurações de imagem criado pelo Capture One, um programa usado para editar e organizar fotos digitais; salva ajustes de imagem (exposição, temperatura de cor, balanço de branco, etc) e outras edições não destrutivas para uma foto.

Mais Informações

Quando os ajustes de imagem são feitos, o Capture One salva um arquivo COS que tem o mesmo nome da imagem original, mas com a extensão de "cos" adicionada. Por exemplo, se você fizer ajustes em myphoto.jpg, capture One salvará um arquivo myphoto.jpg.cos. Quando os ajustes finais são feitos, você pode processar, ou "bake," as configurações e foto juntos em uma imagem de saída com todos os ajustes aplicados.

O que é um arquivo .cos? (Arquivo CRIATURAS CAOS)

Script usado por vários dos jogos nas Criaturas a série de jogos de vida artificial; salvo no formato Creatures Object Source Code (CAOS) e usado por jogos como Creatures 1-3, Creatures Playground e Creatures Adventures.

Mais Informações

COS arquivos são usados para definir os comportamentos dos "agentes"," no mundo. Eles descrevem ações de agentes a serem tomadas com base em certos eventos. Os arquivos COS geralmente estão localizados dentro do diretório de instalação de jogos \Bootstrap\.

Os jogadores às vezes modificam ou "injetam," arquivos COS para criar conteúdo personalizado do jogo. No entanto, se você decidir modificar um script COS, você deve fazê-lo com cautela, já que pode causar o mau funcionamento do jogo.

NOTA: Criaturas não estão mais desenvolvidas e não estão disponíveis para download.
