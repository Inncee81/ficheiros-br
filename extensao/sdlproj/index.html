O que é um arquivo .sdlproj? (Projeto SDL Trados Studio)

Um arquivo SDLPROJ é um projeto criado pelo SDL Trados Studio, um programa usado pelas empresas para editar, revisar e gerenciar projetos de tradução. Ele contém um ou mais arquivos adicionados por um usuário que precisam ser traduzidos para uma linguagem diferente ou revisados por outro usuário. Os arquivos SDLPROJ também incluem detalhes do projeto, como descrição, usuários autorizados, data de vencimento, idioma de origem e idioma-alvo.

Captura de tela de um arquivo sdlproj no SDL Trados Studio 2017

Os arquivos SDLPROJ são o principal tipo de arquivo usado pelo Trados Studio. Os projetos contêm todos os arquivos, como .DOC, .DOCX ou arquivos .XML, necessários para serem traduzidos e revisados. Você pode abrir um SDLPROJ selecionando File → Open Project. Você pode criar um SDLPROJ selecionando File → New → New Project e optar por começar do zero ou modelo (.SDLTPL arquivo).

Os arquivos SDLPROJ podem ser salvos à medida que .SDLPPX pacotes para uma melhor organização e compartilhamento mais fácil com outros membros da equipe. Você pode criar um arquivo SDLPPX usando o assistente "Criar pacote de projeto" no SDL Trados Studio. Para isso, selecione Create Project Package na barra de ferramentas. Você também pode clicar com o botão direito dos projetos que deseja no pacote e selecionar Create Project Package.
