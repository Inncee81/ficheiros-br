O que é um arquivo .adv? (Arquivo predefinido do dispositivo Ableton)

Configurações do arquivo criado pelo Ableton Live, um programa de produção de áudio frequentemente usado para criar mixes eletrônicos; salva predefinições de dispositivos, que atuam como um filtro de áudio para gerar sons exclusivos; pode ser aplicado a um "conjunto ao vivo" (.ALS arquivo) para criar efeitos sonoros baixos, deformado ou outros efeitos sonoros.

O que é um arquivo .adv? (Arquivo do driver de áudio)

Arquivo de driver usado por alguns jogos mais antigos para fornecer compatibilidade de jogo com placas de som; muitas vezes usado para gerar sons de alto-falantes MIDI ou PC.

Mais Informações

Um jogo de exemplo que usa arquivos ADV é Dune from Virgin Interactive, que foi lançado em 1993.

O que é um arquivo .adv? (Arquivo Doppler Velocimeter Acústico Nortek)

Arquivo de dados que armazena dados do Doppler Velocimeter Acústico (ADV), que é usado para capturar medições de velocidade de água, partículas e outras matérias; usado em pesquisas científicas para estudar flutuações rápidas de corrente no oceano ou no laboratório.

Mais Informações
