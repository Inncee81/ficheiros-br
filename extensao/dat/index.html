O que é um arquivo .dat? (Arquivo de dados)

Um arquivo DAT é um arquivo de dados genérico criado por um aplicativo específico. Ele pode conter dados em formato binário ou de texto (arquivos DAT baseados em texto podem ser visualizados em um editor de texto). Os arquivos DAT são normalmente acessados apenas pelo aplicativo que os criou.

Mais Informações

Muitos programas criam, abrem ou referenciam arquivos DAT. Além disso, muitos arquivos DAT são usados apenas para suporte a aplicativos e não devem ser abertos manualmente pelo usuário.

O que é um arquivo .dat? (Trocar anexo de e-mail)

Os arquivos winmaildat contêm informações de formatação de mensagens RTF (Rich Text, texto rico) enviadas pelo Microsoft Exchange Server. Eles são adicionados ao fim dos e-mails quando o cliente de e-mail de um destinatário não suporta mensagens RTF. O arquivo Winmaildat não é útil para usuários que não usam o Outlook ou o Exchange Server para recuperar suas correspondências.

O que é um arquivo .dat? (Pacote de dados de áudio do jogo gamemaker studio)

Um arquivo DAT é um pacote de dados de jogos criado pelo GameMaker Studio, um mecanismo de jogo multiplataforma. Ele contém um grupo de .WAV arquivos de efeito sonoro que são apresentados em um jogo desenvolvido pelo GameMaker Studio. Os arquivos DAT são distribuídos com o jogo para o qual contêm ativos de áudio.

Mais Informações

Os arquivos DAT da GameMaker são criados quando um desenvolvedor de jogos opta por salvar o áudio apresentado em um jogo em um pacote separado do pacote principal de ativos do jogo. Os arquivos DAT não devem ser abertos por um usuário. No entanto, os usuários que gostam de modificar a jogabilidade podem abrir arquivos DAT com UndertaleModTool.

NOTA: Os usuários não devem modificar jogos que estão legalmente proibidos de modificar.

Formato Binário 

O que é um arquivo .dat? (Inno Setup Desinstaller Data File)

Arquivo criado pela Inno Setup, um programa usado para tornar Windows instaladores e desinstaladores de programa; tipicamente chamado unins000.dat (ou um nome semelhante) e está localizado no diretório de instalação do programa; contém dados que ajudam o Inno Setup a desinstalar o programa quando o usuário optar por fazê-lo a partir do Painel de Controle Windows.

Mais Informações

Inno Setup "dat" tem um arquivo .EXE correspondente chamado unins000.exe (ou um nome semelhante). Quando o executável é executado, ele usa as informações em unins000.dat para desinstalar o programa a partir do computador Windows.

O que é um arquivo .dat? (Arquivo de dados minecraft)

Um arquivo DAT é um arquivo de dados usado pelo Minecraft, um jogo de exploração e construção 3D. Ele é salvo em um formato Gzip comprimido e pode salvar vários tipos de dados, incluindo inventário do jogador, posição do jogador e informações mundiais. Os arquivos DAT são usados para salvar informações do estado mundial, bem como para rastrear o progresso do jogador.

Mais Informações

Os arquivos DAT também foram usados para armazenar o nível "chunks," que são pequenas peças de mapa que são carregadas sob demanda à medida que um jogador navega pelo mundo. Este formato foi introduzido começando com a versão Minecraft Infdev, e os pedaços usaram uma dimensão de mapa 16 x 16 x 128. Começando com a versão beta de Minecraft 13, foi introduzido um novo formato chamado "Region" que armazena pedaços com o programa Java que funciona como uma interface de linha de comando.

O que é um arquivo .dat? (Arquivo DAT Piriform)

Um arquivo DAT é um arquivo de dados usado por aplicativos Piriform, como CCleaner, Speccy, Defraggler e Recuva. Ele funciona como uma bandeira, dizendo a um aplicativo Piriform para operar no modo portátil, que é usado quando os usuários querem executar um aplicativo Piriform a partir de uma unidade USB, ou quando os usuários não querem deixar rastros em um sistema de computador.

Mais Informações

Para executar um aplicativo Piriform no modo portátil, crie um arquivo em branco chamado piriform.dat e salve-o no diretório de instalação do aplicativo. Por exemplo, para CCleaner, o arquivo pode ser salvo para C:\Program Files\CCleaner\. Depois que o arquivo for salvo, o CCleaner armazenará as configurações do programa em um arquivo chamado CCleaner.ini, em vez de usar outros locais de armazenamento, como o registro de Windows.

NOTA: O conteúdo do arquivo piriform.dat não importa. Portanto, um arquivo vazio ou um arquivo não vazio que contenha dados produzirá o mesmo resultado.

O que é um arquivo .dat? (Arquivo de contêiner Porteus Save)

Arquivo de salvamento de sistema usado pelo Porteus, um sistema operacional portátil Linux (OS); contém alterações salvas que foram feitas no Porteus OS; inclui alterações nas configurações do sistema, marcadores, arquivos baixados e histórico do navegador.

Mais Informações

O arquivo DAT é usado como um contêiner que contém alterações que você fez ao Porteus, o que permite que você mantenha seu sistema operacional o mesmo ao instalá-lo em outros dispositivos. É um arquivo de contêiner necessário se você estiver instalando o Porteus em dispositivos formatados com um sistema de arquivos Windows. O arquivo é usado para reter dados, como symlinks e Linux permissões, para executar seu sistema corretamente.

Para criar um contêiner de arquivos DAT para salvar suas personalizações, inicializa o Porteus nas interfaces de desktop KDE, Xfce ou LXDE. Após a inicialização ser concluída, abra o menu e selecione System → Porteus save.dat manager. Em seguida, siga as instruções para terminar de criar seu arquivo DAT.

Desenvolvedor Microsoft

Formato Binário 

O que é um arquivo .dat? (Arquivo colmeia de registro de Windows)

O registro de Windows da Microsoft usa arquivos DAT para armazenar as configurações do usuário. Exemplos incluem senhas, configurações de conta e associações de arquivos. Certos arquivos DAT são carregados quando um usuário faz login em Windows.

Mais Informações

Um arquivo de colmeia de registro comum é NTUSER.DAT, que armazena informações de registro em nível de usuário. Está localizado no diretório a seguir:

C:\Users\[user]\

>NOTA: Os arquivos NTUSER contêm configurações importantes do usuário e não devem ser excluídos.

Outro arquivo DAT comum é SCHEMA.DAT, que é um arquivo de backup e restauração de registro. Está localizado em:

C:\Windows\System32\SMI\Store\Machine\
