O que é um arquivo .xlsx? (Planilha Microsoft Excel Open XML)

Um arquivo XLSX é uma planilha do Excel criada pelo Microsoft Excel ou outro programa de planilha, como OpenOffice Calc ou Apple Numbers. Armazena dados em planilhas, que contêm células dispostas em uma grade de linhas e colunas, e também pode conter gráficos, funções matemáticas, estilos e formatação. Os arquivos XLSX são comumente usados para armazenar dados financeiros e para salvar modelos matemáticos simples ou complexos.

Captura de tela de um arquivo xlsx no Microsoft Excel 2019

O formato de Planilha XML Do Microsoft Excel Open foi introduzido com o lançamento do Microsoft Excel 2007. Desde a introdução do formato, os arquivos XLSX tornaram-se um dos tipos de arquivos mais comuns para salvar dados em planilhas.

Com o lançamento do Excel 2007, os arquivos XLSX substituíram o pacote Zip. O arquivo inclui o arquivo [Content_Types].xml, que descreve a planilha, e um arquivo .XML para cada planilha dentro da planilha.

Cada célula em uma planilha XLSX pode ser formatada individualmente. Por exemplo, as células podem ter um número, texto, moeda ou outro formato, bem como cor, estilo de borda, fonte personalizada e tamanho. As células também podem fazer referência a outras células para calcular um valor resultante.

NOTA: Para explorar o conteúdo de um arquivo XLSX, renomeie a extensão .xlsx para .zip e, em seguida, descomprima o arquivo resultante com qualquer utilitário de descompressão Zip.
