O que é um arquivo .xpj? (Arquivo de projeto Micro Planner X-Pert)

Projeto criado pela X-Pert, um programa de gerenciamento de projetos desenvolvido pela Decisive Tools; usado pelos gestores para projetar e acompanhar projetos individuais ou integrar vários projetos em conjunto; suporta até 10.000 operações por projeto.

Mais Informações

X-Pert pode mesclar e importar dados de vários arquivos XPJ, bem como arquivos do Microsoft Project (.MPX).

O que é um arquivo .xpj? (Arquivo de projeto ROBOHelp HTML)

Ajuda arquivo de projeto criado pela RoboHelp, um aplicativo de autoria de conteúdo de ajuda; salvo usando formatação XML e contém referências a arquivos de ajuda HTML e outros ativos de conteúdo de ajuda; também contém informações de navegação, tabela de conteúdo e indexação de pesquisa.

Mais Informações

Os projetos xpj são frequentemente usados para desenvolver manuais de ajuda e bases de conhecimento para aplicativos de software.
