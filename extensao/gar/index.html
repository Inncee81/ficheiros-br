O que é um arquivo .gar? (Arquivo do Projeto Ghidra)

Um arquivo GAR é um arquivo comprimido zip usado pela Ghidra, um desmontagem de software de código aberto desenvolvido pela Agência Nacional de Segurança (NSA). Ele contém um projeto, que pode incluir um ou mais programas de .EXE importados, e é comprimido para reduzir o tamanho do projeto. Os arquivos GAR são normalmente usados para fazer backup de projetos ou transferi-los para outro local.

Mais Informações

Ghidra consiste em um conjunto de ferramentas desenvolvidas pela Diretoria de Pesquisa em apoio à missão de Cibersegurança da NSA. Foi usado internamente por vários anos, mas depois tornado público como software de código aberto em 2019.

O software foi projetado para desarmar programas executáveis em código de montagem. Ele é usado principalmente para investigar malware para analisar vulnerabilidades que foram criadas para atacar.

Os usuários do Ghidra podem importar programas em um projeto para análise selecionando File → Import File... ou arrastando e soltando o programa para a pasta do projeto no visor do Project Manager. Para salvar um projeto como um arquivo GAR em Ghidra, selecione File → Archive Current Project.... Para importar um arquivo GAR em Ghidra, selecione File → Restore Project....

Arquivo GridGain

Desenvolvedor GridGain Systems
Categoria Desenvolvedor
Formato Binário 

O que é um arquivo .gar? (Arquivo GridGain)

Pacote implantável criado para o GridGain, um aplicativo de computação em nuvem de alto desempenho; salva um arquivo compactado de arquivos que podem ser executados ou lidos na nuvem; pode ser uma unidade de processamento menor que é executada com muitas outras unidades desse tipo na nuvem.

Mais Informações

Os arquivos gridGain incluem executáveis Java na forma de .JAR ou arquivos .CLASS. Eles também contêm metadados sobre o arquivo. A estrutura do arquivo é semelhante ao formato .WAR.
