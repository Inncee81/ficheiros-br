O que é um arquivo .qmtf? (Arquivo de transferência de Mac quicken)

Um arquivo QMTF é um arquivo de transferência criado pela Quicken for Mac, um programa de gestão financeira. Ele contém dados financeiros do usuário, como informações pessoais, transações e relatórios, em formato de texto. Os arquivos QMTF são usados para transferir dados do Quicken para outros softwares financeiros.

Mais Informações

Arquivos QMTF são criados quando você exporta dados do Quicken. Para exportar o arquivo QMTF selecione File → Export e escolha "Export to Quicken 2007" ou "Quicken Mac 2007 Transfer File (QMTF)" Você pode selecionar a faixa de data dos dados que deseja incluídos no arquivo QMTF.

NOTA: Os arquivos QMTF são semelhantes aos arquivos .QIF, que também são usados para transferir dados financeiros em formato de texto.
