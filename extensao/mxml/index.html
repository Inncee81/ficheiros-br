O que é um arquivo .mxml? (Componente Flex MXML)

Componente usado por um aplicativo Adobe Flex; escrito na linguagem de marcação MXML, que é semelhante ao XML, mas foi desenvolvido pela Macromedia; usado para definir componentes da interface do usuário no aplicativo.

Mais Informações

A maioria dos aplicativos Flex inclui vários arquivos MXML e ActionScript (.AS).
