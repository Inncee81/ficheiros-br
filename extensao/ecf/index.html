O que é um arquivo .ecf? (Complemento do Outlook)

Um arquivo ECF é um complemento usado pelo Microsoft Outlook, um cliente de e-mail. Ele contém informações que adicionam recursos extras, como segurança reforçada ou aumento de produtividade, ao Outlook.

Mais Informações

Os complementos são integrações desenvolvidas para melhorar a funcionalidade do Outlook. Eles são desenvolvidos por terceiros e podem ser acessados através da loja do Office ou outros sites. Os complementos podem estar disponíveis para uma variedade de propósitos, como fornecer acesso ao mercado do Google no Outlook, combater spam, gerenciamento avançado de trechos de e-mail ou adicionar ferramentas avançadas de impressão ao Outlook.

Os complementos são instalados e gerenciados de forma diferente, dependendo da versão do Outlook:

  • Outlook 2003 e mais antigo - Tools → Options → Other → Advanced Options → Add-In Manager....
  • Outlook 2007 - Tools → Trust Center → Add-Ins.
  • Outlook 2010 - File → Options → Trust Center → Trust Center Settings → Add-ins.
  • Outlook 2013 & 2016 - File → Options → Add-ins.

O que é um arquivo .ecf? (Arquivo de componente SageCRM)

Contém elementos componentes criados pelo software SageCRM Customer Relationship Management; gerados quando os arquivos de script .ES associados são criados; podem incluir campos, guias, botões e outros elementos que são usados dentro do programa.

Mais Informações

ECF os arquivos são chamados "COMPONENTECF" por padrão e são salvos no diretório "CRM<installname >inf" juntamente com os arquivos de script ES.
