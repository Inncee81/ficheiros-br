O que é um arquivo .bup? (Arquivo de backup)

Arquivo de backup criado automaticamente por vários aplicativos. Os arquivos BUP são gerados no caso de o programa falhar ou o arquivo original ficar corrompido e precisar ser restaurado.

Mais Informações

A extensão "bup" é frequentemente adicionada após a extensão original do arquivo. Para abrir o arquivo com o programa correto, a extensão "bup" pode precisar ser removida.

O que é um arquivo .bup? (Arquivo de backup DVD IFO)

Um arquivo BUP é um backup de um arquivo .IFO de um disco de vídeo DVD. Ele armazena informações sobre o conteúdo do DVD e pode ser acessado se o arquivo VIDEO_TSIFO não puder ser lido. Os arquivos BUP não são criptografados.

Mais Informações

ARQUIVOS BUP são úteis caso o arquivo IFO esteja corrompido. Os usuários podem então restaurar o arquivo para o DVD.

O que é um arquivo .bup? (Arquivo em quarentena da McAfee)

Um arquivo BUP é um arquivo em quarentena criado por produtos antivírus da McAfee. Ele salva um backup de um arquivo que foi movido de outro local no computador porque um vírus ou outro malware foi detectado.

Mais Informações

OS arquivos BUP são salvos no seguinte diretório em Windows Vista e Windows 7:

C:\ProgramData\McAfee\VirusScan\Quarantine\

O diretório de Quarantine no computador local pode se tornar muito grande em tamanho à medida que mais arquivos em quarentena são adicionados. Se você quiser remover os arquivos manualmente, siga estas etapas:

  1. Vá ao Centro de Segurança → Navegação → Configurações e Alertas Gerais → Proteção de acesso.
  2. Desmarque a opção "Use proteção de acesso".
  3. Excluir os arquivos do diretório Quarantine.

Você pode restaurar arquivos de um arquivo em quarentena usando o Gerenciador de Quarentena. No entanto, pode haver casos em que o arquivo já foi excluído pelo produto McAfee. Neste caso, você pode restaurar manualmente os arquivos seguindo as etapas deste artigo base de conhecimento da McAfee:
