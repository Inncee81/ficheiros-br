O que é um arquivo .edg? (Arquivo diagramador de borda)

Diagrama criado pelo Edge Diagrammer, uma ferramenta usada para criar diagramas técnicos, de apresentação e outros tipos; armazena o layout do desenho, bem como blocos, setas e etiquetas de texto; pode ser usado para vários tipos de diagramas, incluindo fluxogramas, gráficos organizacionais, diagramas de rede e diagramas de blocos.

Mais Informações

O Diagramador de Borda pode exportar desenhos para vários formatos de arquivos diferentes, incluindo .WMF, .EMF, .BMP, .JPG, .PNG, .TIF e .GIF.
