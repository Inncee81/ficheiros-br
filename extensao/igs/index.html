O que é um arquivo .igs? (Arquivo de desenho IGES)

Um arquivo IGS é um arquivo gráfico salvo em um formato vetorial 2D/3D com base na Especificação inicial de troca de gráficos (IGES). Ele pode armazenar modelos de wireframe, representações de objetos superficiais ou sólidos, diagramas de circuito e outros objetos. Os arquivos IGS são usados por muitos programas CAD como um formato padrão baseado em texto ASCII para salvar e exportar dados vetoriais.

Captura de tela de um arquivo igs no CadeX CAD Exchanger 37

IGES é um padrão para a transferência de modelos tridimensionais entre programas CAD. Você provavelmente só encontrará um arquivo IGS se você trabalhar com software CAD em um campo que envolve fabricação, como as indústrias automotiva ou aeroespacial.

O formato IGES foi oficialmente introduzido em 1980 e posteriormente foi adotado por muitos programas CAD. Nos últimos anos, no entanto, foi substituído pelo novo formato STEP (.STP).

Você pode abrir arquivos IGS com vários programas CAD de desktop, que inclui Autodesk Fusion 360, Dassault Systemes CATIA, Nemetschek VectorWorks e CADEX CAD Exchanger. Você também pode abrir arquivos IGS enviando-os para um programa on-line, como CADEX CAD Exchanger Cloud ou ShareCADorg.

Desde que os arquivos IGS armazenam informações de texto simples, eles podem ser abertos com um editor de texto. No entanto, ao visualizar os arquivos com um editor de texto, o conteúdo aparecerá como texto em vez de uma representação visual de um modelo, como quando visualizado com software CAD. Você pode usar o Microsoft Wordpad, que é empacotado com Windows, ou Apple TextEdit, que é empacotado com macOS, para visualizar o texto simples em arquivos IGS.

NOTA: O arquivo IGS também pode ser salvo como .IGES arquivos.

O que é um arquivo .igs? (Arquivo de cena do renderizador Indigo)

3D cena usada pela Indigo Renderer, um aplicativo usado para gerar imagens fotorealistas; muitas vezes salvo como uma exportação de outro pacote de modelagem 3D que tem o plug-in Indigo Renderer, incluindo Blender, 3ds Max, Cinema 4D, Maya, SketchUp ou Revit.

Mais Informações

Uma vez que uma cena 3D tenha sido importada para o Indigo Renderer, ela pode ser transformada em uma imagem final. O software suporta aceleração de GPU para que as imagens possam ser geradas mais rapidamente.
