O que é um arquivo .rox? (Arquivo executável do objeto do relatório de atuação)

Relatório executável criado com software de emissão de relatórios Actuate; executa relatórios com base nos parâmetros salvos no arquivo do relatório; compilado a partir de um arquivo de design de objeto de relatório (.ROD) e um ou mais arquivos de fonte Básica (.BAS).

O que é um arquivo .rox? (Arquivo do Projeto Roxio)

Projeto de compilação de discos criado pelo Roxio Creator, um aplicativo de gravação de disco, edição de vídeo e aprimoramento de fotos; armazena as informações para uma compilação de áudio, vídeo ou disco de dados; usado para gravar CDs ou DVDs no software.

Mais Informações

ARQUIVOS ROX são salvos usando formatação XML. Eles contêm referências aos arquivos a serem queimados, bem como às informações do disco, como o título, comprimento e informações de faixa.

Roxio Creator gera um nome de arquivo padrão para arquivos ROX que inclui o tipo de disco e uma sequência de data e hora. Por exemplo, uma compilação de áudio pode ter o nome BurnAudioCDs_01-01-2010_123456.rox. O usuário pode optar por criar seu próprio nome personalizado.
