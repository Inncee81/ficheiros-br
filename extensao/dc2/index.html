O que é um arquivo .dc2? (Desenho 2D ASCII DesignCAD)

2D desenho criado pelo DesignCAD, um aplicativo de design e elaboração CAD; codifica o desenho em texto ASCII e inclui todos os elementos desenhados; usado como uma alternativa de texto simples ao formato de desenho .DCD.

Programas que abrem o arquivo .dc2 (Desenho 2D ASCII DesignCAD)

  IMSI DesignCAD
  IMSI DesignCAD 3D Max

O que é um arquivo .dc2? (Documento Pro de DevCad Cam)

Documento CAD criado pela DevCad Cam Professional, um programa usado para criar projetos CAD; suporta todos os recursos do DevCad Pro .DC1 arquivos, mas também pode armazenar instruções da máquina de fresagem CNC para fresagem automatizada.

Mais Informações

DevCad Cam Pro pode simular visualmente os caminhos de ferramentas armazenados em arquivos DC2.

  Windows
  DevCad Cam Pro

Desenvolvedor Kodak
Categoria Imagens Rasterizadas
Formato Binário 

O que é um arquivo .dc2? (Arquivo kodak foto-enhancer)

Arquivo RAW câmera criado por algumas câmeras digitais Kodak; armazena a imagem em um formato não comprimido exatamente como foi capturada na câmera; muitas vezes importada em software de edição de fotos para retoques e publicação.

Programas que abrem o arquivo .dc2 (Arquivo kodak foto-enhancer)

  Adobe Photoshop with LaserSoft Imaging SilverFast DC plug-in
  Windows
  FileStream Image Broadway 
  Adobe Photoshop with LaserSoft Imaging SilverFast DC plug-in
