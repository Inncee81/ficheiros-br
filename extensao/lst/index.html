O que é um arquivo .lst? (Lista de dados)

Um arquivo com uma extensão .lst é provavelmente um arquivo de texto que contém uma lista de dados. Os arquivos LST são usados por muitos programas diferentes, incluindo o Microsoft Visual Studio e vários editores de texto.

Mais Informações

Um arquivo LST típico contém uma lista de informações, separadas por quebras de linha. Exemplos incluem uma lista de arquivos a serem instalados por um instalador de software, dados exportados de uma planilha ou dados de banco de dados despejados de outro programa.

Uma vez que os arquivos LST são salvos em um formato de texto simples, eles podem ser visualizados e editados com um editor de texto básico. Em raras ocasiões, um arquivo LST pode ser salvo em um formato binário, que não exibirá informações reconhecíveis em um editor de texto. Se este for o caso, você precisará abrir o arquivo com o programa que originalmente o criou.

O que é um arquivo .lst? (Arquivo da lista de inicialização GRUB)

Arquivo de sistema usado pelo GRUB (GRand Unified Bootloader), um carregador de inicialização multi-inicialização usado para executar vários sistemas operacionais no mesmo computador; salvo como o nome de arquivo menu.lst e contém opções de menu para exibir o menu de tela do carregador de inicialização na inicialização do sistema.

Mais Informações

Os arquivos LST especificam propriedades como os nomes do sistema operacional e a partição do disco rígido para o sistema operacional. Eles também podem especificar o sistema operacional padrão e um tempo limite que seleciona o padrão se nenhuma entrada for especificada.

Um exemplo de um arquivo de menu.lst GRUB seria:
timeout 10
título Fatdog64
kernel/boot/vmlinuz
initrd/boot/initrd
boot

Qualquer extensão usada pelo sistema operacional Linux Tiny Core deve ser listada em um arquivo LST, com apenas uma extensão por linha.

O local padrão do arquivo menu.lst está /boot/grub/.

O que é um arquivo .lst? (Modelo de etiqueta do LightScribe)

Modelo de etiqueta de disco usado pelo LightScribe Template Labeler, um programa de criação de etiquetas de disco; contém um design de etiqueta que serve como ponto de partida para a criação de designs personalizados; o design final da etiqueta pode ser gravado na parte superior de um CD ou DVD usando uma unidade óptica LightScribe.

Mais Informações

Os designs de etiquetas LightScribe padrão são salvos com uma extensão .LSL.

O que é um arquivo .lst? (Agora modelo de lista de contato)

Modelo de lista criado pelo Now Contact, um organizador de atividades do dia-a-dia e organizador de informações de contato; contém um layout de impressão de modelo para uma lista; as opções pré-carregadas são "Por Empresa," "Por Listagem," e "Por Nome"

Mais Informações

Para adicionar ou editar os modelos LST, selecione o menu "List Layouts" e selecione Define Layout..., faça suas edições e clique em OK quando terminar.

Para selecionar o modelo que deseja imprimir, selecione File → Print Templates..., selecione seu modelo no menu "Tipo de modelo"," e clique em OK.
