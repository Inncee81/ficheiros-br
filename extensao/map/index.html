Arquivo do mapa de fallout

Desenvolvedor Bethesda Softworks
Categoria Jogos
Formato Binário 

O que é um arquivo .map? (Arquivo do mapa de fallout)

Arquivo usado por Fallout 1 e 2, que são jogos de rpg pós-apocalípticos e futuristas; contém dados de mapa que descrevem o ambiente de jogabilidade, incluindo limites de grade, paredes, cenários, animais e objetos.

Mais Informações

O arquivo MAP é composto por 5 seções:

  1. O cabeçalho do arquivo MAP.
  2. As variáveis globais e locais usadas pelos scripts do mapa.
  3. Os dados de ladrilhos que compõem o telhado e o piso.
  4. Os scripts MAP que armazenam informações sobre scripts conectados aos objetos no mapa.
  5. Os objetos MAP, que guardam as paredes, item, cenário, etc. que aparecem no mapa.

O que é um arquivo .map? (Mapjects Server Webparts File)

Arquivo usado pela Mapjects GRYD, uma plataforma de visualização de operações para gerenciamento de energia, gerenciamento de segurança e logística; armazena uma configuração de um ou mais "Webparts," que são widgets que podem ser amassados juntos para produzir visualizações integradas; também inclui os metadados GIS necessários para exibir os dados.

Mais Informações

ARQUIVOS MAP são arquivos de dados do lado do servidor e incluem os arquivos de dados e programas para o pacote de software. Um arquivo .MAPX correspondente é usado para exibir as informações no arquivo MAP para o usuário através de uma página Web.

NOTA: Mapject GRYD renderiza HTML5, Silverlight, Flash e PHP, bem como código do motor de visualização ASPNET Razor.

Arquivo do mapa do motor de terremoto

Desenvolvedor id Software
Categoria Jogos
Formato Texto 

O que é um arquivo .map? (Arquivo do mapa do motor de terremoto)

Um arquivo MAP é um mapa criado para jogos desenvolvidos com um dos motores Quake. Ele é salvo em um formato baseado em texto legível por humanos. Os arquivos MAP não são compilados, mas podem ser compilados em um arquivo de mapa binário .BSP para uso em jogos como Half-Life, Quake, Quake 2 e Quake 3.

Mais Informações

Arquivos MAP podem ser construídos usando vários editores de mapas, incluindo o Valve Hammer Editor, que também compila o mapa no formato BSP de destino. Outra ferramenta popular que usa arquivos MAP é o TorquePoweredcom's Torque Constructor, que pode transformar mapas no formato .DIF para desenvolvimento. Os arquivos de mapa resultantes são usados para jogos torque, que visam as plataformas pc, mobile e console.

Muitos motores de jogo usam o formato MAP, incluindo os motores Quake, Torque Engine, Goldsource Engine, Dark Engine e Unreal Engine.

NOTA: O Valve Hammer Editor 4, que é usado para Half-Life, salva arquivos de mapas no formato .VMF mais novo. O programa era anteriormente chamado de "Worldcraft",", mas agora é comumente referido como "Hammer"

Arquivo do mapa de guerra de Super Mario

Desenvolvedor Michael Schaffer and Florian Hufsky
Categoria Jogos
Formato Não Disponível 

O que é um arquivo .map? (Arquivo do mapa de guerra de Super Mario)

Um arquivo MAP é um mapa de jogos usado por Super Mario War, um jogo de código aberto semelhante a Super Mario. Ele contém o layout de um nível no jogo, que inclui a localização de blocos, itens e sprites. Os arquivos MAP estão localizados na pasta "Maps" do diretório de instalação do jogo.

Mais Informações

Arquivos MAP são carregados pelo jogo quando um usuário seleciona o nível. Você também pode criar e modificar os níveis de MAP usando a ferramenta de editor de nível (Leveleditor.exe) incluída no download de Super Mario War.

Super Mario War apresenta muitos dos mesmos elementos de outros jogos da série Super Mario. O principal elemento da jogabilidade é os jogadores pulando na cabeça de outros jogadores para derrotá-los. Os jogadores também podem usar itens colocados em cada nível para derrotar outros jogadores.

Arquivo do mapa terraria

Desenvolvedor Re-Logic
Categoria Jogos
Formato Binário 

O que é um arquivo .map? (Arquivo do mapa terraria)

Um arquivo MAP é um arquivo de dados de jogo usado pela Re-Logic Terraria, um jogo de construção de ação e caixa de areia 2D. Contém um mapa da área explorada por um personagem para um determinado mundo, que pode incluir terreno, água, monstros, itens e outros objetos.

Mais Informações

Você provavelmente só encontrará um arquivo Terraria MAP se você tiver a Terraria instalada em seu computador e estiver navegando nos arquivos de dados para caracteres no diretório Steam\userdata. Os arquivos MAP não devem ser abertos manualmente por um usuário. Em vez disso, eles são referenciados pela Terraria para carregar informações para um personagem específico que detalha a área que o personagem explorou em um determinado mundo.
