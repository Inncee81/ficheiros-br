O que é um arquivo .wlx? (Arquivo de link de Web MapSource)

ARQUIVO GIS criado pela MapSource, um software de planejamento de viagem incluído nas compras de produtos do mapa Garmin; salvo usando formatação XML e armazena uma localização mundial, bem como uma referência a um site de mapa externo como o Google Maps; atua como um atalho de link e permite que locais sejam abertos rapidamente em sites de terceiros.

Mais Informações

Arquivos WLX criam links entre o software MapSource e um site externo. Isso permite que os usuários visitem outras fontes de informação para saber mais sobre um local.

Se você receber um arquivo WLX, você pode instalá-lo no MapSource selecionando Edit → Preferences → Web Links e, em seguida, escolhendo "Adicionar Web Link"

O que é um arquivo .wlx? (Windows arquivo de sincronização de malha ao vivo)

Arquivo temporário criado pelo Live Mesh, um aplicativo da Microsoft usado para sincronizar arquivos entre computadores, Windows Live SkyDrives e outros locais de armazenamento; marca um arquivo que ainda precisa ser baixado de um local de armazenamento remoto para o local de armazenamento local; atua como um espaço reservado até que o arquivo real seja baixado.

Mais Informações

Os nomes de arquivos WLX são criados anexando o sufixo "wlx" aos arquivos que precisam ser sincronizados. Por exemplo, se myimage.png precisa ser sincronizado a partir de um local remoto, o Live Mesh criará um arquivo temporário chamado myimage.png.wlx no diretório local. Quando a sincronização estiver concluída (quando o arquivo é baixado), o Live Mesh remove o arquivo WLX.

NOTA: A Live Mesh foi descontinuada em 2013 e agora é a Microsoft OneDrive.

O que é um arquivo .wlx? (Arquivo do protocolo de análise winlist)

Arquivo de protocolo de análise usado pelo WinList, um programa de análise de citometria de fluxo; contém uma descrição completa da configuração de análise de citometria de fluxo que o usuário criou; inclui informações de tela, portões, regiões e configuração da paleta.

Mais Informações

Um protocolo permite que você salve uma configuração de sua análise, que permite que você feche e abra uma configuração para retomar seu trabalho.

NOTA: Quando você precisa transferir arquivos WLX, o WinList permite criar pacotes de protocolo que usam o formato compatível com Zip.
