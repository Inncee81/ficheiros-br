O que é um arquivo .atlas? (Arquivo gerador atlas)

Um arquivo ATLAS é um arquivo de configurações criado pelo Phaser Editor, um IDE usado para desenvolver jogos HTML5. É usado para gerar um atlas de textura, que é um sprites que são usados para texturas de objetos e personagens em um jogo. Os arquivos ATLAS contêm as configurações de um atlas de textura e os caminhos para cada sprite a serem embalados na imagem PNG quando o jogo é construído.

Mais Informações

Você pode abrir e editar arquivos ATLAS no Phaser Editor. Quando você terminar de editar o arquivo ATLAS, você pode gerar o atlas de textura. O atlas de textura inclui uma imagem PNG com texturas sprite e uma estrutura, como Photon Storm Phaser e Apple SpriteKit. Alguns jogos que usam atlas de textura, incluem Iron Armada, Surge Breaker, Goodnight e Protolife.

Embora o Phaser Editor use arquivos ATLAS para gerar atlas de textura, outras frameworks, como Photon Storm Phaser e Apple SpriteKit, não usam um arquivo ATLAS. Em vez disso, os desenvolvedores que usam essas estruturas colocam sprites dentro de uma pasta com a extensão .atlas e a estrutura gera o atlas de textura PNG e O arquivo JSON ou .XML quando o jogo é construído.

O que é um arquivo .atlas? (Arquivo de pele do Atlas de textura)

Um arquivo ATLAS é um arquivo de configuração de pele usado pelo libGDX, uma estrutura de desenvolvimento de jogos de código aberto. Ele é usado para configurar a aparência de elementos de interface do usuário, ou widgets, em um jogo, como fontes, botões, rótulos e imagens. Os arquivos ATLAS contêm a posição e o tamanho das imagens para os elementos de interface do usuário, que são embalados em um arquivo .PNG para diminuir o tempo de carregamento do jogo.

Mais Informações

O arquivo ATLAS é um dos 3 arquivos que compõem um atlas de textura, que é usado para reduzir o número de arquivos de imagem separados usados para os elementos de interface do usuário no jogo. Os três arquivos a seguir são necessários no libGDX para criar uma pele para elementos de interface do usuário em um jogo:

  1. PNG file - Este arquivo contém todas as imagens usadas para compor a aparência dos elementos de interface do usuário. As imagens são embaladas em um arquivo para reduzir o uso da memória.
  2. ARQUIVO ATLAS - Este arquivo contém as propriedades de cada imagem armazenada no arquivo PNG. Ele inclui o nome e o tamanho do arquivo PNG embalado e o comportamento e os locais xy de cada imagem no arquivo PNG embalado.
  3. JSON file - Este arquivo contém informações que mapeiam as imagens especificadas no arquivo ATLAS para cada elemento de interface do usuário.

NOTA: Os arquivos ATLAS, PNG e .JSON que compõem um atlas de textura devem ser localizados na mesma pasta dos ativos do jogo com o mesmo nome.
