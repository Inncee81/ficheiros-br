O que é um arquivo .a2theme? (Arquivo temático Aston 2)

Windows tema usado pela Aston, um programa que substitui o Windows desktop e shell por aparências personalizadas; armazena configurações visuais para o menu iniciar, desktop e painéis; salvo em um formato de .ZIP comprimido.

Mais Informações

O arquivo compactado A2THEME contém gráficos bitmap, bem como arquivos XML que definem o layout dos elementos temáticos.

NOTA: Se você baixar um arquivo A2THEME, você pode clicar duas vezes no arquivo para instalar o tema, já que a Aston registra automaticamente a associação de arquivos "a2theme" quando instalada. No entanto, como os arquivos A2THEME são salvos no formato ZIP, o navegador Web pode atribuir automaticamente a extensão "zip" ao arquivo baixado. Portanto, primeiro você precisa alterar a extensão para "a2theme" e, em seguida, clicar duas vezes no arquivo para instalar o tema.
