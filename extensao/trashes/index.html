O que é um arquivo .trashes? (pasta de lixo da unidade flash USB do macOS)

Uma pasta com uma unidade flash USB que foi conectada a um computador que executa o macOS. Os arquivos TRASHES podem ser excluídos para liberar espaço nas unidades flash USB em que estão localizados.

Mais Informações

A maioria dos usuários não se deparará com um arquivo TRASHES, uma vez que está oculto e só aparece em unidades flash USB que foram conectadas a um computador que executa o macOS. O arquivo TRASHES é tecnicamente uma pasta e é criado em uma unidade flash USB quando um usuário move arquivos da unidade flash para o lixo no macOS, mas ainda não esvaziou o lixo, que exclui permanentemente os arquivos.

Embora o arquivo TRASHES seja normalmente visto no File Explorer quando a unidade flash USB que estava conectada ao computador que executa o macOS está conectada a Windows. Um usuário pode simplesmente clicar duas vezes no arquivo TRASHES para abri-lo no File Explorer da mesma forma que abriria uma pasta. Ele pode então recuperar quaisquer arquivos que ainda não foram excluídos permanentemente quando a unidade flash USB foi conectada ao computador que executa macOS.

Se um usuário precisar liberar espaço em sua unidade flash USB, ele pode excluir o arquivo TRASHES ou simplesmente esvaziar a lixeira no macOS onde os arquivos do arquivo TRASHES foram movidos para.
