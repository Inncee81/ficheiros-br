O que é um arquivo .afd? (Arquivo de peça de imagem de disco AFF)

Parte de um arquivo dividido criado a partir de uma imagem de disco .AFF (Advanced Forensics Format) ; permite que grandes arquivos AFF sejam divididos em segmentos menores para que possam ser gerenciados com mais facilidade; ajuda com sistemas de arquivos de 32 bits onde o limite de tamanho do arquivo é de 2GB; também ajuda com limitações de armazenamento com backups de DVD.

O que é um arquivo .afd? (Desenho de chama alfacam)

ARQUIVO CAD criado pela Alphacam, um aplicativo de marcenaria CAD; contém um desenho de chama, que especifica um design 3D que será cortado usando uma ferramenta de chama; pode ser usado para cortar peças simples ou componentes 3D complexos de cinco eixos.
