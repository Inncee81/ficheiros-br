O que é um arquivo .C3D? (Arquivo modelo C3D)

Um arquivo C3D é um modelo 3D salvo no formato binário C3D Labs, que é um formato CAD usado em indústrias de construção de máquinas e arquitetura. Contém dados geométricos de objetos para um modelo 3D, como curvas, pontos, triangulação, conjuntos e corpos. Os arquivos C3D são os arquivos principais usados para trocar modelos 3D usando o software C3D Labs.

Captura de tela de um arquivo c3d no C3D Labs C3D Viewer 2017

Se você receber um arquivo C3D, você pode abri-lo com o C3D Viewer, que está disponível para download gratuito. O programa permite que você amplie e saia, pan, gire e orbite o modelo. Você pode salvar diferentes visualizações do modelo nos formatos JPEG, PNG, BMP e TIFF, o que permite compartilhar seu modelo em um formato mais amplamente suportado. O programa também pode ser usado para visualizar outros formatos, como .STEP, .IGES e .STL.

NOTA: A C3D Labs foi formada em 2012 a partir da ASCON (uma das primeiras desenvolvedoras de CAD russas fundada em 1989) após anos de desenvolvimento do kernel C3D. Eles estão localizados no centro de inovação Skolkovo fora de Moscou.

O que é um arquivo .C3D? (Coordenar arquivo 3D)

Um arquivo C3D é um arquivo de dados salvo no formato Coordinate 3D (C3D), que contém informações biomecânicas. Armazena dados tridimensionais de captura de movimento em uma estrutura binária compacta e é um formato padrão para armazenamento de dados de biomecânica.

Mais Informações

Captura de tela de um arquivo c3d em Mokka 06

O 3D coordinate foi projetado para armazenamento de dados flexível e extensível. Permite que os usuários armazenem e modifiquem parâmetros personalizados, além dos parâmetros de biomecânica padrão e dados brutos.

O formato C3D foi originalmente desenvolvido pelo Dr. Andrew Dainis na ADTech no início da década de 1980 e continua sendo o padrão para dados de marcha biomecânica e movimento 3D. Ele permite que dados 3D sejam compartilhados em indústrias clínicas, de pesquisa e de filmes. Os detalhes completos do formato estão disponíveis no site do C3Dorg.
