O que é um arquivo .dec? (Arquivo de Declaração)

Um arquivo DEC é um arquivo de declaração baseado em texto usado pelos programas ADL (Application Development Language). Ele contém variáveis globais e constantes que podem ser referenciadas por outros arquivos dentro de um projeto de programação.

Mais Informações

Os arquivos DEC são normalmente salvos no formato EDK II Declaration (DEC). O EDK II é uma base de código de desenvolvimento usada para criar drivers UEFI (Unified Extensible Firmware Interface, interface de firmware extensível) unificados, imagens de firmware e aplicativos. O objetivo do formato DEC é auxiliar na construção de componentes EDK e módulos EDK II e criar aplicativos UEFI e imagens de firmware.

O que é um arquivo .dec? (Magic: O arquivo de deck on-line de coleta)

Um arquivo DEC contém um baralho de cartas para Magic: The Gathering Online (MTGO), um jogo de jogo de cartas online. Ele armazena uma coleção de cartas que compõem um baralho jogável, que inclui o número de cartas e os nomes de cada carta no baralho. Os arquivos DEC também podem conter os nomes dos cartões e notas do aparador.

Mais Informações

Captura de tela de um arquivo dec em TJ Soft Decked Builder 125

Os arquivos DEC são salvos em texto simples, com cada linha começando com o número de cartões seguido pelo nome do cartão. Algumas linhas também podem começar com SB:, que indica que são cartões de aparador, ou //, que indica uma nota.

Uma vez que os arquivos são salvos em texto simples, você pode facilmente criar, modificar e compartilhar decks jogáveis. Você pode abrir e exibir arquivos DEC com qualquer editor de texto, como Bloco de Notas, WordPad e TextEdit.

No entanto, você só deve modificar e salvar arquivos DEC com editores de texto simples para evitar salvar informações extras de formatação, que não podem ser lidas. Além disso, você pode ter que alterar a extensão do arquivo de um deck salvo usando um editor de texto de .txt para .dec.

NOTA: Os decks jogáveis também podem ser salvos à medida que .DEK arquivos.
