O que é um arquivo .indt? (Modelo adobe indesign)

Um arquivo INDT é um modelo de layout de página para projetar páginas no Adobe InDesign, um programa de publicação de desktop. É normalmente usado ao criar várias páginas que têm um formato semelhante. Os arquivos INDT podem ser incluídos no software InDesign ou criados pelo usuário.

Mais Informações

Captura de tela de um arquivo indt no Adobe InDesign CC 2017

Os arquivos INDT são usados para criar vários documentos .INDD com os mesmos estilos e formatação. Quando você salva um projeto de layout de página, o arquivo INDD é criado pelo InDesign para armazenar o projeto.

Você pode criar um arquivo INDT no InDesign selecionando File → Save As... e escolher o modelo "InDesign CC 2017" da lista suspensa "Format".

Para abrir um arquivo INDT no Creative Cloud, basta clicar duas vezes no arquivo INDT ou selecionar File → Open..., navegar até o local do arquivo, selecionar o arquivo e clicar em Abrir.
