O que é um arquivo .sii? (18 rodas de aço: Arquivo de definição de caminhão de transporte)

Arquivo de dados usado por 18 Rodas de Aço: Haulin', um jogo de condução de caminhão; contém propriedades para um caminhão, como a classe do motor; salvo em um formato de texto simples e muitas vezes alterado para criar configurações personalizadas de caminhões.

Mais Informações

Os arquivos SII também podem conter uma lista de arquivos de caminhão referenciados para o jogo. Por exemplo, o arquivo truck_storage.sii contém a lista de arquivos de definição de caminhão e pode ser estendido para adicionar novos caminhões. Ele é salvo em um formato de texto mais simples.
