O que é um arquivo .rec? (fsck Arquivo de recuperação de dados)

Um arquivo REC é um arquivo de recuperação de dados criado pelo fsck, um utilitário usado para verificar a integridade de um sistema de arquivos em sistemas operacionais semelhantes ao Unix e Unix. Ele contém um registro de "inconsistências" em um sistema de arquivos que foram detectados durante uma verificação fsck que pode ser analisada pelos usuários para investigar problemas em um sistema de arquivos. Os arquivos REC são semelhantes aos arquivos .chk criados pela ferramenta de sistema "CHKDSK" para Windows.

Mais Informações

A ferramenta fsck é normalmente usada em sistemas operacionais semelhantes ao Unix, como Linux, macOS, Free BSD. O utilitário normalmente é executado durante o inicialização do sistema operacional para verificar a consistência do sistema de arquivos. Se o utilitário encontrar algum problema, ele os registra em um ou mais arquivos REC a serem analisados.

Quando fsck cria arquivos REC, ele os nomeia como FSCK####.REC com o #### subindo de 0000 para cada arquivo de recuperação de dados criado. Por exemplo, o primeiro arquivo REC é FSCK0000.REC, o segundo é FSCK0001.REC, o terceiro é FSCK0002.REC, e assim por diante.

ARQUIVOS REC armazenam dados em texto simples e podem ser abertos com um editor de texto, como o Apple TextEdit ou o gedit. Os arquivos também podem ser excluídos sem afetar o desempenho do sistema, mas podem ser regenerados durante o processo de inicialização do sistema operacional.

NOTA: fsck é abreviação de File System Consistency Check.

O que é um arquivo .rec? (Gravação pvr topfield)

Um arquivo REC contém vídeo criado por um gravador de vídeo digital Topfield PVR, um dispositivo usado para gravar programação de televisão padrão e de alta definição. Ele é salvo em um formato compactado proprietário que pode ser visto em um dispositivo TOPfield PVR ou com vários jogadores de vídeo de terceiros.

Mais Informações

Você provavelmente só encontrará um arquivo REC se você possui um gravador de vídeo digital Topfield PVR e gravou programação de TV. Você também pode receber um arquivo de vídeo REC de alguém que possui um gravador de vídeo digital Topfield PVR.

Os arquivos REC podem ser abertos com vários reprodutos de vídeo, incluindo o reprodutograma squared 5 MPEG Streamclip e o videolan VLC media player. O software do reprodução de mídia VLC também pode ser usado para converter arquivos REC em formatos de vídeo mais amplamente suportados.

NOTA: Os arquivos REC também são conhecidos como arquivos Toppy, uma vez que toppy é abreviação de Topfield.
