O que é um arquivo .swift? (Arquivo de código fonte swift)

Um arquivo SWIFT é um arquivo de código fonte escrito em Swift, uma linguagem de programação introduzida pela Apple com iOS 8 e OS X Yosemite. Ele foi projetado para escrever e compilar aplicativos e pode ser usado com Xcode começando com Xcode 6.

Mais Informações

Swift usa um estilo familiar para iOS e desenvolvedores OS X Objective-C, mas torna a programação mais fácil e mais como programação em uma linguagem de script. A Swift oferece recursos como gerenciamento automático de memória e integração com as estruturas de Cacau existentes.
