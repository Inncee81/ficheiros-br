O que é um arquivo .p5d? (Projeto Planner 5D)

Um arquivo P5D é um projeto criado pela Planner 5D, um programa de design de casa 2D e 3D e decoração de interiores. Contém um projeto de design residencial, que inclui o layout da casa, materiais de construção, móveis e eletrodomésticos. Os arquivos P5D também podem armazenar objetos localizados ao redor do exterior da casa, como caminhos, árvores, plantas e piscinas.

Captura de tela de um arquivo p5d no Planner 5D

Se você se deparar com um arquivo P5D em seu computador, você pode abri-lo com o programa Planner 5D, que está disponível para Windows, Android e iOS e como um programa web. O arquivo P5D é usado pelo programa para armazenar todos os conteúdos do projeto.

O Planner 5D permite que você visualize o design do seu projeto como um projeto 2D ou como um modelo 3D. Você pode mover paredes, e colocar portas, janelas, escadas, colunas e outros elementos de construção. O programa também permite que você organize peças de móveis, tapetes, cortinas, luminárias e eletrodomésticos. Ao projetar seu layout, você pode tirar capturas de tela e compartilhá-las via correio ou para o aplicativo OneNote. Projetos P5D podem ser iniciados do zero ou de um modelo para lhe dar uma vantagem.

NOTA: O Planner 5D permite que você inicie um projeto gratuitamente, mas requer a compra para salvá-lo (o que cria o arquivo P5D). A compra também oferece mais itens internos para escolher, suporte técnico e instantâneos HD.
