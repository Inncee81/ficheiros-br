O que é um arquivo .gifv? (Arquivo de vídeo GIF)

Um arquivo GIFV é um arquivo de vídeo salvo em um formato desenvolvido pela Imgur que melhora o formato GIF. Contém vídeo compactado no formato H264 e armazenado dentro de um recipiente de arquivo .MP4.

Mais Informações

GIF é um formato popular para compartilhar loops de animação curtos na web, que inclui sites como Twitter e Imgur. O formato GIFV faz parte da iniciativa "Project GIFV" da Imgur para melhorar o formato GIF. Uma das principais atualizações do formato é que os arquivos GIFV são significativamente menores do que os arquivos GIF, o que permite velocidades de carregamento mais rápidas.

NOTA: O site Imgur converte arquivos GIF carregados para o formato GIFV.
