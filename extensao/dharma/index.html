O que é um arquivo .dharma? (Arquivo criptografado do Dharma Ransomware)

Um arquivo com uma extensão .dharma é um arquivo que foi criptografado pelo vírus Dharma, que é uma variante do ransomware Crysis. Ele é criptografado para que não seja possível abrir o arquivo simplesmente alterando a extensão de arquivo .dharma.

Mais Informações

O ransomware Dharma é um tipo de malware utilizado por cibercriminosos que criptografa os arquivos. Depois que o ransomware toma os arquivos como reféns, ele força a vítima a pagar o criminoso para desbloquear os arquivos. Ele pode ser introduzido no computador da vítima através de e-mails de spam com links maliciosos ou anexos de arquivos, sites gratuitos de hospedagem de arquivos e redes Peer to Peer (P2P).

Quando o ransomware é executado no computador de um usuário, ele criptografa arquivos no computador e adiciona a extensão .dharma aos nomes dos arquivos. Ele também anexa um endereço de e-mail [bitcoin143@india.com] ou [worm01@india.com] com a extensão. Os tipos de arquivos tipicamente direcionados incluem planilhas, documentos, imagens, vídeos e arquivos de backup, como backup de seus arquivos, ele pode executar uma restauração do sistema para remover o vírus, mas quaisquer alterações feitas nos arquivos após o backup foi feita serão perdidas.
