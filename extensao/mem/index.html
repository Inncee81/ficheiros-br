O que é um arquivo .mem? (Arquivo de memórias GreekToMe)

Um arquivo MEM é um arquivo de dados criado pelo GreekToMe, um programa de estudo do Novo Testamento grego multiplataforma. Ele salva o estado das diferentes opções na interface do usuário, que podem incluir o livro, verso, palavra, palavra léxica, autor e listas gramaticais. Os arquivos MEM permitem que os dados de memória sejam salvos e restaurados posteriormente.

Mais Informações

Memórias podem ser exportadas selecionando File → Export Memories... no menu do aplicativo. Você pode importar memórias escolhendo File → Import Memories....

NOTA: Versões futuras do GreekToMe podem não ser capazes de importar arquivos de memória mais antigos, por isso os usuários devem considerá-los dispensáveis.

Paralelamente arquivo de despejo de memória da área de trabalho

Desenvolvedor Parallels
Categoria Sistema
Formato Binário 

O que é um arquivo .mem? (Paralelamente arquivo de despejo de memória da área de trabalho)

Arquivo criado pela Parallels Desktop, um programa usado para executar Mac os sistemas operacionais OS X e Windows usando um computador Mac; contém um despejo de memória da Máquina Virtual paralela (VM); salvo quando o VM é suspenso.

Mais Informações

OS arquivos MEM podem ser salvos com outros arquivos de estado salvos por VM para recuperar o estado exato do VM em um determinado momento.

Programas que abrem o arquivo .mem (Paralelamente arquivo de despejo de memória da área de trabalho)

O que é um arquivo .mem? (Arquivo de memória da Estação de Jogo Virtual)

Jogo para a Virtual Game Station (VGS), um programa que permite que jogos de PlayStation sejam jogados em Mac ou Windows computadores; normalmente contém o conteúdo de um jogo de PlayStation que pode ser carregado pelo emulador VGS.

Mais Informações

Virtual Game Station foi lançado pela Connectix em 1999. Mais tarde foi comprado pela Sony, que acabou com o desenvolvimento do produto. Connectix foi adquirido pela Microsoft em 2003.
