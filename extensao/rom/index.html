O que é um arquivo .rom? (Arquivo ROM do jogo N64)

Um arquivo ROM é um arquivo de jogo nintendo 64 (N64) jogável com um emulador N64. Ele armazena o conteúdo de um cartucho de jogo N64 em um formato de arquivo somente leitura que pode ser aberto e jogado usando um programa de emulação do Nintendo 64.

O que é um arquivo .rom? (Leia somente imagem de memória)

Um arquivo ROM é um arquivo de dados que contém uma cópia do chip ROM de memória somente leitura para um dispositivo de hardware, normalmente a partir de firmware de computador ou um cartucho de videogame. É comumente usado para emulação de software ou para atualizar o firmware em um determinado dispositivo.

Mais Informações

Se o arquivo ROM for usado para emulação, ele normalmente é usado para permitir que o software de sistema operacional mais antigo (OS) seja executado em sistemas operacionais mais recentes Windows, Mac e Linux. Ou pode ser usado para jogar videogames mais antigos, como Nintendo ou Sega Genesis em um computador usando um emulador de videogame.

NOTA: Uma vez que os arquivos ROM são usados para diferentes propósitos, eles podem usar diferentes formatação de arquivos. Portanto, uma ROM usada por um programa pode não ser compatível com outro programa.
