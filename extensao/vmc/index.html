O que é um arquivo .vmc? (Windows arquivo de configuração da máquina virtual)

Arquivo de configuração usado pelo Windows Virtual PC, um componente do Windows 7 que permite aos usuários emular diferentes versões de Windows como máquinas virtuais (VMs); salvos em um formato XML e inclui parâmetros que descrevem o VM; carrega o VM especificado quando aberto.

Mais Informações

Os arquivos VMC incluem configurações como o nome da máquina virtual, memória alocada, arquivos em disco rígido (arquivos .VHD), o arquivo de estado VM atual (arquivo .VSV), o arquivo de disco desfaçante (arquivo .VUD e configurações de rede). Eles não armazenam os dados reais do usuário vm e do nível do sistema. Em vez disso, os dados são armazenados em disco rígido virtual (arquivos VDH) e instantâneos VM (arquivos VSV).

Windows PC Virtual é o sucessor do Microsoft Virtual PC 2007 e 2004. Ele só está disponível para Windows 7 Professional e Ultimate. O software é frequentemente usado para "Windows XP Mode," que é a tecnologia que permite que Windows XP seja executada dentro Windows 7.

Windows O PC Virtual também cria arquivos de atalho para arquivos VMC que usam a extensão .VMCX. Esses arquivos também podem ser usados para carregar o VM especificado.

NOTA: Os arquivos VMC também são usados pelo Microsoft Virtual Server.
