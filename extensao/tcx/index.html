O que é um arquivo .tcx? (Pacote de extensão de script testComplete)

Um arquivo TCX é um pacote de extensão de script usado pelo SmartBear TestComplete, um programa usado para automatizar os testes da interface do usuário de um aplicativo. Ele contém arquivos usados por uma extensão de script, que incluem um formato Zip e pode armazenar arquivos para várias extensões de script.

Mais Informações

As extensões de script adicionam funcionalidades personalizadas ao TestComplete de forma semelhante aos plugins, mas são mais fáceis de criar no TestComplete. Eles permitem que você personalize testes automatizados que podem ser realizados em seu aplicativo. As extensões de script podem ser usadas para criar ações personalizadas de tempo de registro, objetos de script, ações em tempo de projeto, operações de teste de palavras-chave e operações em resultados de teste.

TCX normalmente armazena os seguintes arquivos:

  • Um arquivo description.xml que contém metadados sobre as extensões do script incluídas no pacote TCX.
  • Arquivos de script VBS ou JS que contenham as ações, operações e objetos que compõem a extensão.
  • Arquivos AQFRM que contêm informações de formulário para formulários usados pela extensão.
  • Um arquivo de ajuda CHM que é colocado na mesma pasta que a extensão quando instalado no TestComplete.
  • Arquivos PNG ou BMP que contêm ícones para as operações e ações.

Os arquivos TCX são abertos principalmente no TestComplete quando um usuário o importa para o aplicativo. No entanto, você também pode abrir manualmente arquivos TCX, uma vez que eles são renomeados arquivos ZIP.

Para abrir manualmente um arquivo TCX, renomeie a extensão de arquivo "tcx" para "zip" e expanda o arquivo com um programa de descompressão zip, como Corel WinZip, Apple Archive Utility ou 7-Zip.

Para criar um arquivo TCX, basta reverter o processo acima mencionado. Comprima seus arquivos de extensão como um arquivo ZIP com um programa de compactação zip e, em seguida, renomeie a extensão "zip" para "tcx".

O que é um arquivo .tcx? (Arquivo XML do Centro de Treinamento)

Um arquivo TCX é um arquivo XML usado para transferir dados de aptidão entre dispositivos de fitness. Contém dados de aptidão, como o tipo de esporte, tempo de volta, distância, calorias, intensidade e frequência cardíaca. Os arquivos TCX são tipicamente criados por dispositivos Garmin e Fitbit.

Mais Informações

Arquivos TCX podem ser abertos usando uma variedade de programas, como Garmin Training Center, Google Earth e Strava. Eles também são apoiados pela MotionBasedcom e pelo Garmin Communicator Plugin.

TCX também pode ser exportado do painel de conta de um usuário da Fitbit no fitbitcom. Para exportar dados de aptidão, selecione Log → Activities no painel, clique em Exibir detalhes no Histórico de Atividades, selecione os três pontos e escolha Export as a TCX file.

NOTA: O formato TCX está disponível nos esquemas TCX v1 e TCX v2.

O que é um arquivo .tcx? (Arquivo de texto modelo 3D TurboCAD)

Desenho TurboCAD salvo em formato de texto; pode conter dados de modelo 2D ou 3D; semelhante a um arquivo padrão de desenho TurboCAD (.TCW), mas salvo em um formato de texto que pode ser visualizado em um editor de texto.

Mais Informações

O formato TCX do TurboCAD é semelhante ao formato de arquivo .DXF AutoCAD.
