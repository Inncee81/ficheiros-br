O que é um arquivo .spp? (Arquivo de imagem Serif PhotoPlus)

Imagem salva no formato de imagem photoplus nativo; pode incluir camadas, efeitos de camada, transparência, texto editável e QuickShapes; também pode incluir dados de malha, que permitem que partes da imagem sejam distorcidas usando a ferramenta Mesh Warp.

Mais Informações

Arquivos SPP são semelhantes aos documentos do Photoshop (.PSD), mas são criados com o Serif PhotoPlus.

NOTA: Serif descontinuou o PhotoPlus e o substituiu por Affinity Photo em 2015.

O que é um arquivo .spp? (Projeto Pintor de Substâncias)

Um arquivo SPP é um projeto criado pela Allegorithmic Substance Painter, um programa de texturização 3D para Windows, macOS e Linux. Contém informações do projeto, que inclui malhas e texturas que compõem o modelo 3D. Os arquivos SPP também incluem metadados usados para configurar o projeto.

Mais Informações

Captura de tela de um arquivo spp em Allegorithmic Substance Painter 2018

O arquivo SPP é o principal tipo de arquivo associado ao Substance Painter. Você pode criar um arquivo SPP selecionando File → Save ou Save As.... Você pode abrir um arquivo SPP selecionando File → Open.... Você também pode exportar o arquivo SPP para outros formatos, como .BMP, .PSD, .JPEG, .HDR, .TIFF e .GIF, no menu File.

NOTA: O pintor de substâncias é usado para uma variedade de campos de animação 3D, como efeitos visuais, animações de recursos e videogames.
