O que é um arquivo .fzb? (Despejo de banco Casio FZ-1)

"Dump" de todos os dados do banco de som armazenados na memória de um Casio FZ-1, um teclado de amostragem digital lançado pela Casio por volta de 1990.

O que é um arquivo .fzb? (Arquivo de backup do projeto Form•Z)

Arquivo de backup criado por form•Z, um aplicativo de modelagem de sólidos e superfícies 3D; armazena um backup de um arquivo de projeto form•Z .FMZ; usado para fins de arquivamento e para restaurar o projeto a partir de backup.

Mais Informações

NOTA: Form•Z também cria arquivos de salvamento automático com a extensão .FZA que pode ser usada para restaurar um projeto se um arquivo de backup for mais antigo do que o último auto-save.

O que é um arquivo .fzb? (Arquivo Fritzing Bin)

Arquivo Bin usado pelo Fritzing, um programa que permite aos usuários criar projetos eletrônicos interativos; armazena informações sobre peças eletrônicas (.FZP arquivos), que podem fazer parte de um circuito eletrônico; inclui o nome da peça, o ID do módulo, o ícone .PNG para a peça e coordenadas geométricas.

Mais Informações

As lixeiras e peças armazenadas em cada caixa são exibidas no painel "Parts" no lado direito da interface Fritzing. As peças podem ser arrastadas de uma lixeira e pisadas em um esboço. Você também pode adicionar peças às suas próprias lixeiras personalizadas e carregá-las em Fritzing.
