O que é um arquivo .wdp? (Arquivo de projeto elétrico autocad)

Arquivo de projeto criado pela AutoCAD Electric, um programa usado para criar diagramas de fiação; contém uma lista de texto simples de arquivos de desenho de diagrama de fio que juntos compõem o diagrama geral; também contém configurações do projeto.

Mais Informações

Arquivos WDP podem conter referências de caminho absolutas ou relativas ao desenho de arquivos. Portanto, se você mover um arquivo WDP de caminho relativo para outro local, o AutoCAD Electric pode não ser capaz de localizar os arquivos de desenho, a menos que você os mova também.

NOTA: Os arquivos WDP contêm referências a desenhos, mas não aos próprios desenhos.

O que é um arquivo .wdp? (Arquivo do Projeto Windev)

Projeto de desenvolvedor criado pelo WinDev, um programa usado para a construção de aplicativos Windows e móveis; pode incluir várias janelas de programa (.WDW ou .WPW arquivos), código-fonte e construir informações para o aplicativo; pode ser compilado em um programa executável final quando concluído.

O que é um arquivo .wdp? (Arquivo de foto de mídia Windows)

Formato de imagem raster comprimido desenvolvido pela Microsoft; semelhante à compressão lossy e lossless; usado principalmente para fotografias digitais de alta resolução; pode ser usado com vários produtos da Microsoft, incluindo Windows Photo Viewer, Windows Live Movie Maker (WLMM) e Microsoft Word.

Mais Informações

NOTA: O formato Windows Media Photo passou para o formato HD Photo, que usa a extensão de arquivo .HDP.
