O que é um arquivo .dng? (Arquivo de imagem negativa digital)

Formato de imagem RAW universal para salvar fotos digitais em um formato não compactado; desenvolvido pela Adobe Systems como um formato de arquivamento de imagens RAW e é suportado por muitos programas de visualização de imagens e edição de fotos.

O que é um arquivo .dng? (Imagem de Dongle Virtual)

Imagem de Dongle virtual criada por programas, como EdgePro e Sentinel Dongle Dumper, então usada por emuladores dongle; contém uma cópia virtual de um dongle, que inclui informações de licença e age como uma chave que permite que o software seja executado.

Mais Informações

Um dongle é um dispositivo físico usado para ativar softwares caros. O dispositivo ajuda a proteger o software de ser roubado por hackers. Os arquivos DNG são projetados para que os usuários façaem backup ou recuperem chaves armazenadas nos dongles físicos perdidos ou quebrados.
