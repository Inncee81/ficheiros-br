O que é um arquivo .veg? (Projeto de Vídeo vegas)

Um arquivo VEG é um projeto de vídeo criado pelo MAGIX VEGAS, um aplicativo profissional de edição de vídeo e criação de discos. Ele contém uma linha do tempo com faixas de áudio e vídeo, transições, conteúdo de texto, efeitos especiais e referências às mídias do projeto, como arquivos de vídeo e áudio usados pelo projeto.

Mais Informações

Captura de tela de um arquivo veg em MAGIX VEGAS Pro 16

Os arquivos VEG são usados pelo VEGAS principalmente para armazenar projetos, mas também são usados para modelos e arquivos (o formato de arquivo VEG permanece o mesmo). O uso do arquivo VEG como modelo permite que os usuários mantenham as mesmas configurações em diferentes projetos. Usando um arquivo VEG como um arquivo, você pode salvar backups de projetos que você pode abrir mais tarde. Selecione File → Export → VEGAS Project Archive para salvar seu projeto para fins de arquivamento.

Projetos de VEGAS concluídos podem ser compartilhados online diretamente de VEGAS. Eles também podem ser exportados para uma variedade de formatos de vídeo digital, incluindo .AAF (Pro Tools ou Media Composer), .PRPROJ (Premiere ou After Effects) e .FCPXML (Final Cut Pro X).

NOTA: O software VEGAS foi desenvolvido anteriormente pela Sony antes de ser adquirido pela MAGIX em 2016.

O que é um arquivo .veg? (Arquivo de vegetação vue)

3D arquivo de cena usado pelo E-on Vue, um aplicativo de modelagem de ambientes e atmosferas; armazena modelos de plantas que podem ser incorporados em uma cena vue; podem ser usados para gerar fractals de plantas em terrenos ou outras paisagens.
