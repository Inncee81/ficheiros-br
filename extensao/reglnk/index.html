O que é um arquivo .reglnk? (Arquivo de atalho do editor de registro)

Um arquivo REGLNK contém um atalho para uma chave de registro. Ele armazena o atalho em formato .INI e permite que os usuários pulem rapidamente para a localização de uma chave de registro usando o Registry Editor, um programa empacotado com Windows que permite que os usuários editem chaves de registro.

Mais Informações

O registro Windows é um banco de dados que contém dados necessários para o funcionamento do sistema Windows e aplicativos executados no sistema. A estrutura do registro inclui chaves de registro que funcionam semelhantes às pastas, mas em vez de conter arquivos, elas contêm valores ou outras chaves de registro.

O arquivo REGLNK é semelhante ao arquivo .LNK, que contém um atalho para um arquivo ou pasta em um sistema. No entanto, o REGLNK se conecta a uma chave de registro, o que é útil para navegar rapidamente até a localização de uma chave de registro. Você pode abrir um arquivo REGLNK usando o Registry Editor, uma ferramenta incluída com Windows que permite que os usuários editem manualmente as chaves do registro.

Para abrir o Editor de Registro em Windows 10, digite regedit na caixa de pesquisa da barra de tarefas e selecione Editor de Registro. Ou, clique com o botão "Iniciar"," selecione Run, digite regedit no "Open:" caixa, e clique em OK.

NOTA: Tenha cuidado ao editar as chaves do registro, pois modificações incorretas podem desativar seu computador.
