O que é um arquivo .grd? (Arquivo gradiente do Photoshop)

Arquivo criado pelo Photoshop, um programa de edição de imagens; contém um preenchimento gradiente ou multicolorido, no qual as cores se transformam incrementalmente em outras cores; comumente aplicadas a objetos de desenho ou ladrilhos em padrões de fundo; usado para armazenar e compartilhar gradientes.

Mais Informações

Gradientes são projetados para repetir cores mesmo quando o espaço gradiente aumenta de tamanho. Portanto, os gradientes podem preencher suavemente objetos ou fundos desenhados sem bordas visuais abruptas. Eles podem usar formatos lineares, radiais, refletidos ou angulares.

O que é um arquivo .grd? (Arquivo de grade do surfista)

Arquivo salvo no formato de grade da Golden Software, que é usado para gerar mapas; contém rettices uniformes bidimensionais (matriz retangular regularmente espaçada) de dados XYZ (por exemplo, leste, norte e elevação); pode estar em um formato binário ou de texto, mas é mais frequentemente salvo no formato binário.
