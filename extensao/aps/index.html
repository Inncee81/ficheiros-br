O que é um arquivo .aps? (Arquivo do projeto do estúdio de cartão de saudação)

Arquivo de projeto criado pelo Greeting Card Studio, um programa usado para criar cartões de cumprimento; pode conter fotos digitais, gráficos e texto, bem como gradientes e clip art; usado para imprimir cartões e enviá-los para amigos e familiares.

Mais Informações

Os arquivos APS podem ser exportados para formatos de imagem bitmap, incluindo .BMP, .JPG, .GIF, .PNG e .TIF.

NOTA: O Greeting Card Studio também é outro produto de software desenvolvido pela Software Depot Online. Ele salva projetos de cartões de saudação usando arquivos .GCDP, que não são compatíveis com o Estúdio de Cartões de Saudação da AMS Software.

O que é um arquivo .aps? (Arquivo de recurso Visual C++)

Arquivo criado pelo Microsoft Visual C++, um aplicativo de desenvolvimento de software; armazena a representação binária de um recurso incluído no projeto; permite que o aplicativo carregue recursos mais rapidamente.
