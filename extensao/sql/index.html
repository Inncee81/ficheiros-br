O que é um arquivo .sql? (Arquivo de dados de linguagem de consulta estruturada)

Um arquivo SQL contém sql structured que contém linguagem de consulta estruturada (SQL), que é um idioma usado para acessar e modificar informações em um banco de dados. Ele armazena instruções SQL para criar ou modificar estruturas de banco de dados, inserções, atualizações, exclusões ou outras operações SQL.

Você provavelmente só encontrará arquivos SQL se desenvolver ou gerenciar bancos de dados da Web. A linguagem SQL apareceu pela primeira vez na década de 1970 na IBM e foi adotada pelos grupos padrão ANSI e ISO em 1986 como a linguagem padrão para sistemas relacionais de gerenciamento de banco de dados (RDBMS). Desde sua adoção, passou por várias revisões para suportar um conjunto mais amplo de recursos e gerou várias variações de linguagem de banco de dados, que incluem Mini SQL (mSQL) e MySQL.

SQL arquivos podem ser lidos por qualquer programa de banco de dados compatível com SQL, como MySQL e Richardson RazorSQL. Os arquivos também podem ser abertos e editados com vários editores de texto de origem, como gVim, Bare Bones BBEdit e MacroMates TextMate. Uma vez que os arquivos armazenam comandos SQL em formato de texto simples, eles também podem ser abertos e modificados com editores de texto simples, incluindo Microsoft Notepad e Apple TextEdit, que são empacotados com os sistemas operacionais Windows e macOS.

NOTA: SQL pode ser pronunciado como "S-Q-L" ou "Sequel"
