O que é um arquivo .fds? (Cópia segura do rascunho final)

Script criado com software de roteiro final draft e salvo como uma "Cópia Segura;" semelhante a um documento de rascunho final regular (.FDR arquivo), mas não pode ser editado.

Mais Informações

Arquivos FDS podem ser abertos pelo Rascunho Final 5 ou posterior, bem como pelo Espectador final do rascunho 503 ou posterior.

O que é um arquivo .fds? (Arquivo de formato personalizado FME)

ARQUIVO DE DADOS GIS criado pela FME Desktop, um conjunto de aplicativos espacial ETL (Extract, Transform, Load); armazena dados espaciais transformados em um formato personalizado; usado para exportar conjuntos de dados personalizados que podem ser importados através do componente Reader Gallery da FME Desktop.

Mais Informações

Arquivos FDS podem ser exportados na bancada de trabalho FME (um aplicativo incluído no FME Desktop) selecionando File → Export As Custom Format....

O que é um arquivo .fds? (Nintendo FDS ROM)

Arquivo de jogo usado pelo software de emulação para o Nintendo Family Computer Disk System (também chamado Famicom Disk System); salva uma cópia dos dados originais do cartucho do jogo; permite que o jogo seja jogado em um PC com emuladores de suporte.

Mais Informações

FCEUX pode jogar FDS ROMs se você tiver o arquivo FDS BIOS carregado (disksys.rom).
