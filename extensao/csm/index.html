O que é um arquivo .csm? (Arquivo marcador do estúdio de caracteres)

ARQUIVO DE TEXTO ASCII que contém marcadores de posição para um caractere 3D; usado pelo Character Studio, uma ferramenta de animação biped incluída no programa 3ds Max da Autodesk; salva a posição do biped no buffer de captura de movimento do software.

Mais Informações

Os dados de rotação de membros não são armazenados em arquivos CSM, mas são salvos .BVH arquivos relacionados.

O que é um arquivo .csm? (Arquivo de backup do jogo steam)

Arquivo de backup usado pelo Steam, um serviço de distribuição digital usado para acessar jogos e software; contém os dados de backup de um jogo jogado com o Steam; semelhante a um arquivo .CSD e .SIS.

Mais Informações

Para restaurar um arquivo de backup CSM, selecione Steam → Backup and Restore Games... e escolha o arquivo. O arquivo CSM está localizado com outros arquivos de backup CSD e SIS na pasta de jogo apropriada na pasta "Backups a vapor".
