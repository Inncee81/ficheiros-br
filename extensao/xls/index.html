O que é um arquivo .XLS? (Planilha excel)

Um arquivo XLS é um arquivo de planilha criado pelo Microsoft Excel ou outro programa de planilha, como OpenOffice Calc ou Apple Numbers. Ele contém uma ou mais planilhas, que armazenam e exibem dados em formato de tabela. Os arquivos XLS também podem armazenar funções matemáticas, gráficos, estilos e formatação.

Você pode encontrar arquivos XLS em ambientes domésticos, escolares e de negócios ao organizar, analisar, manipular ou visualizar dados. Cada planilha incluída em uma planilha consiste em uma tabela com linhas e colunas e as células em uma tabela podem conter dados inseridos manualmente ou os resultados computados a partir dos dados de outras células.

O formato binário do Microsoft Excel Planilha (XLS) foi introduzido em 1987 com o lançamento do Excel. Os arquivos XLS tornaram-se um dos tipos de arquivos mais comuns para salvar planilhas até o lançamento do Excel 2007. Com esta versão, a Microsoft introduziu o formato De Planilha XML Do Microsoft Excel Open e substituiu arquivos XLS por arquivos .XLSX como o arquivo padrão para salvar planilhas no Excel.
