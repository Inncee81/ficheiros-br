O que é um arquivo .afi? (Arquivo de backup aomei backup)

Formato de imagem de backup usado pelo AOMEI Backupper, um programa de backup e recuperação Windows; contém um backup de um arquivo que pode ser usado para restaurar um arquivo quando o arquivo está corrompido ou perdido.

Mais Informações

Para criar um backup de arquivoS AFI, selecione "Backup," e escolha "Backup de arquivo" Selecione o arquivo que deseja fazer backup e onde deseja salvar o arquivo AFI e clique em "Iniciar backup"

NOTA: O AOMEI Backupper usa o arquivo .ADI para salvar arquivos de backup de imagem em disco.
