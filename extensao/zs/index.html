O que é um arquivo .zs? (Arquivo ZenScript)

Um arquivo ZS contém a linguagem de programação ZenScript, que é usada por vários mods de Minecraft, como CraftTweaker e MineTweaker, para modificar o conteúdo do jogo. Ele possui uma sintaxe semelhante ao Java e foi projetado para ser simples para os jogadores de Minecraft que não têm amplo conhecimento de programação.

A linguagem de programação do ZenScript foi originalmente desenvolvida para uso com o CraftTweaker, um mod minecraft que permite aos jogadores modificar conteúdo, que inclui "receitas" Essas receitas permitem que os usuários criem novos itens com recursos que reúnem. A linguagem de programação ZenScript é usada principalmente por modders para adicionar, editar e excluir receitas no jogo.

Desde que os arquivos ZS armazenam comandos em texto simples, você pode abrir e editar os arquivos com qualquer editor de texto. No entanto, editores de texto selecionados, como Bloco de Notas++,Visual Studio Code e Sublime Text, são melhores opções para editar arquivos ZS, uma vez que eles suportam o destaque da sintaxe do idioma ZenScript ou plugins que suportam o idioma.

NOTA: Os arquivos ZS devem ser armazenados na pasta [gamedir]/scripts a ser carregada quando um jogador se junta a um mundo em Minecraft.
