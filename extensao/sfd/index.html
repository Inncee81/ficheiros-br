O que é um arquivo .sfd? (Arquivo de partituras do MagicScore Maestro)

Um arquivo SFD contém uma partitura criada pelo MagicScore Maestro, um programa de composição e notação musical. Ele armazena uma composição, que pode incluir instrumentos, assinaturas de tempo e chave, e notas musicais. Os arquivos SFD também contêm metadados, como o título e compositor da partitura e o layout da página da composição da partitura.

Mais Informações

Captura de tela de um arquivo sfd no MagicScore Maestro 8

Você pode abrir arquivos SFD em Windows usando o MagicScore Maestro. Para abrir um arquivo SFD, selecione File → Open, navegue até a localização do arquivo e clique em Abra. Para criar um arquivo SFD usando o MagicScore Maestro, selecione File → New, componha a pontuação e selecione File → Save ou Save as...

Os arquivos SFD são o principal tipo de arquivo usado pelo MagicScore Maestro para salvar pontuações. No entanto, você também pode salvar pontuações como .MIDI arquivos, arquivos .KAR karaoke e arquivos de .XML de música, escolhendo os tipos de arquivos do menu suspenso Save As. Você também pode exportar pontuações no MagicScore Maestro para uma imagem, áudio ou .PDF arquivo selecionando File → Export.

O que é um arquivo .sfd? (Filme Sofdec Dreamcast)

Arquivo de vídeo usado por videogames feitos para Sega Dreamcast, Nintendo GameCube, Microsoft Xbox e outros sistemas de consoles de videogame; salva o vídeo em um formato incomum "Sofdec" que não é jogável em jogadores de vídeo padrão; usado para jogar sequências cinematográficas de videogames.

Mais Informações

Os arquivos SFD podem ser convertidos em arquivos .MPG usando um utilitário chamado sfd2mpg, que faz parte do SFD Tool Pack da SWAT. No entanto, essa utilidade não foi amplamente distribuída e não está mais em desenvolvimento. Os arquivos SFD podem ser criados usando o aplicativo Movie Creator da Sega, mas também não estão mais sendo desenvolvidos e foi lançado pela última vez em 1999. O SFD Tool Pack da SWAT inclui este programa.

NOTA: Pro Evolution Soccer 2012 (PES 2012) Os arquivos SFD podem ser convertidos para o formato .MPG usando o PES Video & Audio Converter.

O que é um arquivo .sfd? (Arquivo de banco de dados de fonte spline)

Arquivo de fonte criado no formato Spline Font Database (SFD), que é o formato nativo usado pelo software de edição de fontes FontForge; descreve uma ou mais fontes em um formato de texto ASCII; usado para documentar e editar fontes.

Mais Informações

Arquivos SFD salvam informações descritivas, como a versão da fonte, nome, família, peso e codificação.

NOTA: O FontForge pode ser executado em Windows usando o Cygwin, um ambiente semelhante ao Linux para Windows.
