O que é um arquivo .xaf? (Arquivo de animação 3ds Max XML)

XML arquivo criado pelo software de modelagem e animação 3ds Max; contém informações de movimento para um biped, que é um modelo com dois pés; permite que os modeladores capturem com precisão os movimentos humanos e de criaturas para videogames, filmes e outras animações 3D.

Mais Informações

Arquivos XAF podem ser reutilizados em vários modelos biped. Eles também podem ser incluídos em uma mistura de 3ds Max Motion Mixer (.MIX arquivo), que permite aos animadores compor uma animação em uma linha do tempo de várias faixas.

O que é um arquivo .xaf? (Transcrição do depoimento de afirmação)

Arquivo criado pelo Editor de Transcrição de Afirmação, um programa usado para criar transcrições de depoimentos durante processos judiciais; armazena texto rico e contém o testemunho verbal de uma testemunha gravada fora do tribunal.

Mais Informações

Arquivos XAF permitem que advogados armazenem provas digitais durante o litígio.

NOTA: Web Depo Affirm Transcript Editor foi descontinuado e não está mais disponível para download.

O que é um arquivo .xaf? (Arquivo de animação Cal3D XML)

3D arquivo de animação usado pelo Cal3D, um kit de ferramentas de código aberto usado para animação de caracteres; armazena movimentos ósseos para o esqueleto do modelo, que é especificado em um arquivo .CSF ou .XSF; usado para criar movimentos humanos realistas.

Mais Informações

Arquivos XAF armazenam dados de animação em formato XML, mas os dados de animação Cal3D também podem ser armazenados em um formato binário usando arquivos .CAF.
