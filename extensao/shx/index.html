O que é um arquivo .shx? (Arquivo de forma compilado autocad)

Um arquivo SHX contém uma forma ou fonte compilada pelo Autodesk AutoCAD a partir de um arquivo de forma .SHP ou .PFB arquivo de fonte. Ele armazena definições de forma, bem como definições de fonte para exibir texto personalizado. Os arquivos SHX suportam muitas fontes PostScript tipo 1.

Mais Informações

As formas compiladas permitem que os usuários salvem blocos de construção menores e fontes que são frequentemente inseridos em desenhos. Isso permite que os usuários economizem tempo e não precisem recriar formas e fontes usadas com frequência.

Depois que um arquivo de forma SHP ou um arquivo de fonte PFB for compilado, o arquivo SHX resultante pode ser inserido em um desenho. O usuário pode então especificar a localização, escala e rotação da forma importada.

NOTA: Autodesk AutoCAD é um aplicativo de design e engenharia 2D e 3D.

O que é um arquivo .shx? (Motorola Flash Superfile)

Arquivo criptografado usado para "flash" o firmware de um celular Motorola, como um Razr ou Rokr; muitas vezes usado para personalizar ou "modding" a interface de celulares Motorola.

Mais Informações

Para mostrar a memória de um celular, você deve conectar o telefone a um PC via USB e executar um programa como PST ou RSD, que pode abrir o superfile flash.

SHX os arquivos são baseados no formato de arquivo flex .HS.

O que é um arquivo .shx? (Arquivo de índice de arquivo de forma)

Índice de arquivos de formas ESRI (arquivos de .SHP) que armazena informações de índice e compensação do shapefile principal; permite acesso direto aos registros no arquivo principal; consiste em um cabeçalho de 100 byte seguido de registros de comprimento fixo de 8 byte que definem os arquivos SHP referenciados pelo arquivo de índice; muitas vezes salvos junto com um arquivo de banco de dados dBASE (.DBF) correspondente.
