O que é um arquivo .pct? (Arquivo de imagem)

Um arquivo PCT é uma imagem salva no formato Macintosh PICT, que foi desenvolvido pela Apple em 1984 para armazenar imagens usando a tecnologia Apple QuickDraw. Contém dados de imagem em um dos dois formatos, PICT 1, o formato original que armazena 8 cores, ou PICT 2, um formato mais novo que permite milhares de cores (imagens de 24 e 32 bits). Os arquivos PCT suportam imagens raster e vetorial.

Mais Informações

Embora as imagens PICT geralmente usem a extensão de arquivo .PICT em computadores Macintosh, elas geralmente são vistas com uma extensão "pct" em Windows. O formato PICT foi em grande parte preterido para o formato de .PDF mais flexível em Mac OS X.

O que é um arquivo .pct? (Arquivo de texto composto puro)

Arquivo de dados químicos usado pela ChemSep, um aplicativo usado para simular operações de destilação, extração e absorção; contém dados de componentes, como nome, índice, número CAS, fórmula, estrutura, correlações T e valores moleculares.

Mais Informações

Você pode importar e exportar os dados de ou para um arquivo PCT no ChemSep através do gerenciador pcd. Para acessar o gerenciador de PCDs, selecione Databanks → Pure components data (PCD/PCT), clique Edit e selecione Import component from file ou Export component to file.

NOTA: Os dados em arquivos PCT são armazenados em formato de texto, enquanto .PCD arquivos armazenam dados em formato binário.
