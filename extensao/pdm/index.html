O que é um arquivo .pdm? (Projeto de fabricante de DVD fotográfico)

Um arquivo PDM é um projeto criado pela Anvsoft Photo DVD Maker, um programa Windows usado para criar slideshows de fotos e queimá-los em DVDs. Ele contém informações sobre um projeto de apresentação de slides, incluindo edições, o tema e caminhos para imagens na apresentação de slides. Arquivos PDM também armazenam metadados, como o nome do projeto e nomes de álbuns no projeto.

Mais Informações

Para abrir um arquivo PDM no Photo DVD Maker, selecione File → Open. Para criar um arquivo PDM no Photo DVD Maker, selecione File → Save ou Save As....

O arquivo PDM permite que você salve edições que você faz no projeto quando fechar o programa e, em seguida, reassuça o projeto para mais edição. Depois de fazer edições finais para o projeto PDM, você pode gravá-lo para um DVD, exportá-lo para um arquivo de vídeo Flash ou MPEG ou enviá-lo para a Internet.

NOTA: Como os arquivos PDM armazenam os locais de imagens e áudio carregados no projeto, não mova os arquivos de mídia após carregá-los ou você terá que carregar a mídia novamente.

O que é um arquivo .pdm? (Arquivo do banco de dados PowerDesigner)

Banco de dados criado com o Sybase PowerDesigner, um programa de modelagem e design corporativo; armazena informações de banco de dados, bem como referências a outros arquivos, como os arquivos de definição de arquivo (.XDB) de definição de banco de dados (.XEM .XDB).

Mais Informações

Arquivos PDM também podem ser abertos com o QDesigner da Quest Software, que foi descontinuado em 2006.

NOTA: O PowerDesigner foi originalmente desenvolvido pela Sybase, que foi comprado pela SAP.

O que é um arquivo .pdm? (ProntoDoc para modelo de documento de exibição móvel do Word)

Arquivo usado pelo ProntoDoc for Word (PDW), um plug-in do Microsoft Word usado para preenchimento automático e geração de modelos de documentos; armazena a estrutura de páginas do modelo, bem como campos vazios que são preenchidos por dados de um ou mais bancos de dados de origem (SQL Server, SharePoint e Oracle são suportados); usado para gerar conteúdo conforme o padrão HTML5, que pode ser visualizado em dispositivos móveis.

Mais Informações

ProntoDoc for Word é usado em ambientes de negócios para geração de relatórios e outros documentos comerciais.

O que é um arquivo .pdm? (Arquivo de informações do projeto VB)

Arquivo de texto contendo um ou mais scripts, juntamente com informações sobre a embalagem e implantação de um aplicativo Visual Basic (VB); inclui todos os arquivos necessários, locais de instalação e o nome do script(s).

Mais Informações

Quando pacotes adicionais de implantação são criados para um projeto. Novos scripts de pacote/implantação são escritos para o início do arquivo PDM do projeto.
