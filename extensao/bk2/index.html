O que é um arquivo .BK2? (Arquivo Bink Video 2)

Um arquivo BK2 é um arquivo de vídeo salvo no formato Bink Video 2, que é a segunda versão do formato de vídeo proprietário bink desenvolvido pela RAD Game Tools. Contém dados compactados com os codecs de vídeo e áudio bink 2. Os arquivos BK2 são usados para reproduzir clipes de filmes em videogames em PCs e sistemas de console.

Mais Informações

O formato Bink Video 2 foi lançado em 2013, substituindo o formato original bink (.BIK). O formato oferece um desempenho consideravelmente melhor do que o Bink, mantendo alta qualidade de vídeo. Os arquivos BK2 funcionam como recipientes para dados compactados com os codecs Bink Video 2 e áudio. Os dados compactados também podem estar contidos em arquivos .BIK2.

NOTA: O Bink Video 2 está disponível para as plataformas Windows, macOS e Linux, Android e iOS. Também está disponível para Xbox One, Xbox 360, Sony PS4, Sony PS3, Sony PSVita, Nintendo Switch, Nintendo Wii e Nintendo WiiU.

O que é um arquivo .BK2? (Arquivo de dados do serviço de indexação de Windows)

Arquivo de sistema usado pelo componente Serviço de Indexação da Microsoft Windows, que indexa informações do sistema de arquivos para pesquisa; muitas vezes criado no diretório C:\System Volume Information\catalog.wci\ e armazena dados usados para realizar pesquisas no sistema de arquivos.

Mais Informações

Assim como .BK1 arquivos, os arquivos BK2 são normalmente vistos com o nome de arquivo propstor.bk2.
