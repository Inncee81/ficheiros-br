O que é um arquivo .tif? (Imagem GeoTIFF)

Arquivo de imagem geoespacial compatível com TIFF criado no formato GeoTIFF; contém dados geográficos ou cartográficos incorporados como tags dentro do arquivo; permite que os usuários incorporem dados GIS no arquivo sem desviar do formato TIFF (Tagged Image File Format).

Mais Informações

Como os arquivos GeoTIFF contêm coordenadas, os aplicativos GIS podem mapear regiões específicas da imagem raster para coordenadas de latitude e longitude.

NOTA: O formato GeoTIFF usa as características extensíveis da especificação TIFF para adicionar metadados marcados à imagem.

O que é um arquivo .tif? (Arquivo de imagem marcado)

Um arquivo TIF contém uma imagem salva no TIFF (Tagged Image File Format, formato de arquivo de imagem tagged), um formato gráfico de alta qualidade. É frequentemente usado para armazenar imagens com muitas cores, fotos tipicamente digitais, e inclui suporte para camadas e várias páginas.

Mais Informações

O formato TIFF foi criado em meados da década de 1980 para ser um formato de imagem universal para diferentes plataformas de computador e scanners de documentos. O formato tornou-se popular entre os profissionais da indústria editorial, incluindo fotógrafos e artistas gráficos. A popularidade do formato deveu-se, em parte, à sua capacidade de ser editado e re-salvo sem perder a qualidade da imagem.

Os arquivos TIF podem ser salvos em um formato não compactado (sem perdas) ou podem incorporar compactação lossy. Eles também podem usar a compactação sem perdas LZW, o que reduz o tamanho do arquivo TIF, mas não reduz a qualidade da imagem.

Os arquivos TIF são suportados por uma variedade de programas de visualização e edição de imagens. Você pode usar o Microsoft Photos em Windows, que vem empacotado com o sistema operacional. Se você é um usuário de macOS, você pode usar o Apple Preview, que vem empacotado com o sistema.

NOTA: Os arquivos TIF também podem usar a extensão .TIFF.
