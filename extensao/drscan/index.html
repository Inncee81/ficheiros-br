O que é um arquivo .DRSCAN? (Arquivo de varreduras de resgate de dados)

Um arquivo DRSCAN contém varreduras criadas pelo Data Rescue, um programa usado para recuperar arquivos de discos rígidos acidentados ou infectados. Ele salva o resultado de uma ou mais varreduras de unidade, que detectam a estrutura do diretório existente e localizam arquivos. Os arquivos DRSCAN podem ser importados de volta para o Data Rescue, mas apenas se as especificações do sistema Mac OS X coincidirem com as quando a varredura foi realizada.

Mais Informações

Os arquivos DRSCAN permitem que os usuários façam backup das varreduras e, em seguida, removam-nas da interface de varredura do Data Rescue para reduzir o tamanho da lista.
