O que é um arquivo .bytes? (Arquivo de dados de ativos de texto binário)

Arquivo usado pela Unity, um aplicativo de desenvolvimento de jogos 3D; contém dados binários do ativo de texto (comumente usando, mas não se limitando ao arquivo .TXT extensão); carregado como um ativo de texto e acessado através da propriedade bytes; a extensão deve ser alterada manualmente para BYTES.

Mais Informações

Um exemplo da extensão BYTES em uso seria alterar uma extensão de .JPEG e colocá-la em sua pasta "Resources". Em seguida, usando o script a seguir:

///Carregar textura do disk
TextAsset bindata= ResourcesLoad("Texture") como TextAsset;
Texturizado tex = novo Texture2D(1,1);
texLoadImage(bindatabytes);
