O que é um arquivo .xnb? (Pacote binário do XNA Game Studio)

Arquivo de dados binário criado com o XNA Game Studio Express, um IDE projetado para desenvolver jogos Windows e Xbox 360; contém um ou mais arquivos em um único pacote.

Mais Informações

O formato XNB é considerado um "formato intermediário" uma vez que empacotar arquivos originais em um tipo de arquivo proprietário. Os arquivos XNB são referenciados por jogos criados com xna Game Studio Express e não devem ser abertos manualmente.

Um jogo de exemplo que usa arquivos XNB é Terraria, um jogo de ação e aventura 2D. A Terraria armazena gráficos, áudios e outros dados do jogo usando arquivos XNB.

NOTA: Enquanto a maioria dos arquivos XNB não pode ser aberta diretamente, arquivos de imagem, como imagens de .PNG, podem ser extraídos de um arquivo XNB usando um programa como gXView (parte do GameTools Suite).
