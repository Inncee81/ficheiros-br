O que é um arquivo .lay? (Arquivo de layout DO DVD Studio Pro)

Arquivo criado pelo DVD Studio Pro, um programa usado para autor de DVDs; contém metadados internos ao DVD Studio Pro e ao projeto de DVD; normalmente armazenados dentro do diretório VIDEO_TS de um projeto de DVD.

Mais Informações

NOTA: O DVD Studio Pro não grava arquivos LAY para discos ao queimá-los. Se você gravar o projeto DVD usando outro programa, você deve ter certeza de excluir os arquivos LAY.

O que é um arquivo .lay? (Arquivo de layout MAME)

Arquivo de layout usado pelo MAME (Multiple Arcade Machine Emulator), um programa usado para jogar jogos de arcade emulados; contém configurações de layout que descrevem como organizar diferentes objetos para um jogo.

Mais Informações

Os arquivos de layout são usados para colocar gráficos de arte, bem como a janela de jogo ativa dentro do MAME.

Programas que abrem o arquivo .lay (Arquivo de layout MAME)

  Windows
  MAME

O que é um arquivo .lay? (Arquivo de estado de camada 3D do Rinoceronte)

Arquivo criado pela Rhinocerous 3D, um programa usado para modelagem tridimensional; armazena o estado de uma camada, que contém um agrupamento de objetos 3D e suas propriedades; usado para salvar e carregar configurações específicas de camadas.

Mais Informações

Todos os objetos dentro de uma camada podem ser modificados juntos. Por exemplo, os usuários podem alterar o wireframe e a cor juntos, bem como ligar e desligar todos os objetos dentro de uma camada.

O que é um arquivo .lay? (Arquivo de layout tecplot)

Arquivo de layout diagrama criado pela Tecplot, um programa científico de computação e visualização; salva o layout visual de um plot (arquivo .PLT) no espaço de trabalho; contém instruções de como exibir o plot, bem como configurações que definem a aparência de cada quadro.

Mais Informações

ARQUIVOS LAY também incluem informações de layout da página e propriedades de espectro de cores.
