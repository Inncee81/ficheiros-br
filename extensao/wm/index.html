O que é um arquivo .wm? (Arquivo de mídia Windows)

Windows Arquivo de áudio ou vídeo de mídia; pode conter áudio, vídeo ou ambos os tipos de dados; formatado usando o codec Windows Media Player.

Mais Informações

A extensão WM é uma extensão simplificada de arquivos que pode ser um .WMA (Windows Audio) ou .WMV (Windows Video). Os arquivos WMA e WMV são mais específicos e são usados mais comumente do que a extensão WM.

Programas que abrem o arquivo .wm (Arquivo de mídia Windows)

  Windows
  Microsoft Windows Media Player
  Digital Cloud Technologies Media Player
