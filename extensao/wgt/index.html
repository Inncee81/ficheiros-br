O que é um arquivo .wgt? (Arquivo de Widget opera)

Widget que serve como um pequeno aplicativo de Web para o navegador Opera Web; salvo como um arquivo compactado contendo um arquivo de configuração de widget, documento de índice, imagens, arquivos JavaScript e folhas de estilo; pode ser instalado por download direto ou usando a opção "Adicionar Widget" no menu Opera Widget.

Mais Informações

Os arquivos widget são construídos por "zipping" o conteúdo do widget e, em seguida, renomeando a extensão de arquivo .ZIP para "wgt".

O que é um arquivo .wgt? (Arquivo de pesos sinápticos XNBC)

Registra os dados de pesos sinápticos para um projeto XNBC (Xwindow Neuro Bio Clusters); armazenados em formato binário; XNBC é uma estação de trabalho de simulação para neurobiólogos usado para fins de pesquisa
