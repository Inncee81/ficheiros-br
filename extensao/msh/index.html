O que é um arquivo .msh? (Arquivo de malha 3D godot engine)

Um arquivo MSH é um modelo 3D criado pela Godot Engine, um mecanismo de jogo de código aberto usado para criar jogos 2D e 3D. Ele armazena informações sobre um modelo 3D em um formato proprietário semelhante ao formato Wavefront 3D Object. Os arquivos MSH são criados quando um usuário importa um arquivo .OBJ no Godot Engine como um objeto ao desenvolver um jogo.

Mais Informações

Você pode importar um arquivo OBJ para o Godot Engine usando o menu Importação 3D Mesh ou simplesmente arrastando e soltando-o uma cena, em seguida, selecionando Import. Godot Engine só cria arquivos MSH quando os arquivos Wavefront 3D OBJ são importados para o projeto. Se um arquivo modelo .DAE, .GLTF ou .GLB for importado no Godot Engine, o aplicativo criará um arquivo .SCN.

NOTA: Os arquivos MSH são usados na versão 2 do Godot Engine, mas foram substituídos por arquivos .MESH na versão 3 do aplicativo.

O que é um arquivo .msh? (Arquivo de malha 3D orbitador)

Formato de malha de modelo 3D proprietário usado pela Orbiter, um programa de simulação de voo espacial gratuito; salva a estrutura tridimensional de um ônibus, estação ou outro objeto espacial; usa formatação de texto ASCII e especifica vértices e grupos de malha para diferentes partes do modelo, bem como as texturas (.DDS arquivos) e materiais para aplicar ao modelo.

Mais Informações

Os arquivos MSH são salvos na pasta \Meshes\ da instalação do software.

O que é um arquivo .msh? (Arquivo do projeto do comunicador visual)

Projeto de filme criado com o Visual Communicator; contém clipes de vídeo, arquivos de áudio, imagens e recortes e efeitos de texto personalizados; usado para criar vídeo de qualidade profissional; semelhante a um Windows Movie Maker Project (arquivo .MSWMM).

Mais Informações

Serious Magic, o desenvolvedor original da Visual Communicator, foi adquirido pela Adobe Systems em 2006.
