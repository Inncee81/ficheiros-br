O que é um arquivo .jmcp? (Arquivo criptografado JM-Crypt)

Arquivo criptografado pelo algoritmo de criptografia JM-Crypt, que é usado pelo software JM-Crypt; contém um arquivo protegido por um algoritmo de criptografia híbrida e senha.

Mais Informações

O arquivo JMCP é uma das várias extensões de arquivo criadas pelo software JM-Crypt. Outras extensões de arquivo incluem "jmcx", "jmck", "jmcr" e "jmce". Os arquivos criptografados são comumente usados para proteger arquivos pessoais confidenciais e arquivos de negócios confidenciais.

O algoritmo JM-Crypt usa o método de criptografia one-time pad (OTP) e criptografa duas vezes o arquivo. A primeira criptografia é com um extrato aleatório de uma longa chave aleatória gerada pelo rápido gerador de chaves Fourier Transform (FFT), que é exclusivo da JM-Crypt Security. A segunda criptografia inclui proteção por senha que depende do método selecionado pelo usuário, que pode ser totalmente automático (senha gerada automática aleatória usada pelo arquivo JMCX), semiautomática (modo projeto, que é utilizado pelo arquivo JMCP) ou manual (uma senha criada pelo usuário e apresentada no arquivo .JMC).

NOTA: Você precisa do software e algoritmo JM-Crypt que foram usados para criptografar originalmente o arquivo a fim de descriptografar o arquivo JMCP.
