O que é um arquivo .afzplug? (AfterShot Pro Plugin Archive)

Um arquivo AFZPLUG é um arquivo plugin usado pelo Corel AfterShot Pro, um editor de imagem profissional. Ele contém conteúdo que amplia a funcionalidade do aplicativo AfterShot, como efeitos de cor, inserção de texto, mistura de canais e recursos de geotagging.

Mais Informações

Você pode encontrar e instalar plugins com o Gerenciador de Plugins do AfterShot Pro. Os plugins estão disponíveis gratuitamente ou para compra. Basta clicar duas vezes no arquivo AFZPLUG para instalar o plugin. Você deve reiniciar o aplicativo AfterShot Pro para concluir a instalação.

NOTA: O arquivo .BZPLUG substituiu o arquivo AFZPLUG em Windows e OS X, mas a extensão "afzplug" ainda é usada em Linux e todas as versões Do AfterShot Pro 24 e anteriores.
