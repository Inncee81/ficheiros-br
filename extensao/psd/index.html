O que é um arquivo .psd? (Documento do Adobe Photoshop)

Um arquivo PSD é um arquivo de imagem criado pelo Adobe Photoshop, um programa profissional de edição de imagens frequentemente usado para melhorar fotos digitais e criar gráficos da Web. É o formato nativo usado para salvar arquivos no Photoshop. Os arquivos PSD podem incluir camadas de imagem, camadas de ajuste, máscaras de camada, anotações, informações de arquivos, palavras-chave e outros elementos específicos do Photoshop. Eles são comumente criados e compartilhados entre profissionais gráficos.

Você pode modificar camadas PSD e outros elementos, como imagens, formas, texto e efeitos. O formato PSD suporta modos RGB, CMYK, grayscale, monocromático, duotônico, cor indexada, cor lab e modos de cor multicanal O modo de cor pode ser alterado dentro do Image → Mode submenu. Quando você estiver pronto para publicar a imagem, você pode exportá-la para formatos de imagem padrão, como codec, como o codec Ardfry PSD, é instalado Exemplos incluem Windows Explorer, Windows Photo Viewer e Windows Media Center.

Qual é a estrutura de dados de um arquivo PSD?

O formato do arquivo do Photoshop é separado em cinco partes principais:

  1. File Header - contém as propriedades básicas da imagem, como a versão, número de canais na imagem, número de bits por canal, altura e largura de pixel da imagem e o modo de cor do arquivo (bitmap, escala de cinza, RGB, CMYK, Multicanal, indexado, duotônico e lab).
  2. Color Mode Data - especifica o comprimento dos dados de cor, que é definido para 0, a menos que o modo de cor seja indexado ou duotona conforme especificado no cabeçalho do arquivo Se este for o caso, os dados de cor para imagens de cor e duotônicos também são armazenados nesta seção.
  3. Recursos de imagem - especifica o comprimento da seção de recursos de imagem e os recursos de imagem, que é uma série de blocos cada um com um ID de recurso que indica o tipo de dados armazenados no bloco Esses blocos são usados para armazenar dados não pixels associados à imagem, como caminhos de caneta ou ferramenta de lápis.
  4. Informações sobre camadas e máscaras - contém informações sobre camadas e máscaras, que incluem o número de camadas, canais nas camadas, intervalos de mistura, teclas de camada de ajuste, camadas de efeitos e parâmetros de máscara.
  5. Image Data - contém os dados reais da imagem e o método de compressão de dados e os dados de pixels de imagem Diferentes métodos de compressão podem ser dados de imagem bruta, compressão RLE, Zip sem previsão ou Zip com previsão.
