O que é um arquivo .tar? (Arquivo de arquivo Unix consolidado)

Um arquivo TAR é um arquivo criado pelo tar, um utilitário baseado em Unix usado para empacotar arquivos para fins de backup ou distribuição. Ele contém vários arquivos armazenados em um formato não comprimido, juntamente com metadados sobre o arquivo. Os arquivos TAR são comumente compactados em arquivos .GZ com compactação do ZNU Zip.

Mais Informações

O formato TAR foi introduzido com o utilitário tar em 1979 para gravar dados em unidades de fita, que eram dispositivos de armazenamento de dados que leriam e gravariam dados em fita magnética. O nome TAR vem de "Arquivo de fita"

Na computação moderna, o formato TAR é normalmente usado em sistemas Unix e Linux para consolidar vários arquivos em um único arquivo. As tarballs são especialmente úteis para fazer backup de arquivos ou transportar vários arquivos para outro local.

Os usuários também podem comprimir arquivos TAR para reduzir seu tamanho. Os arquivos TAR comprimidos com a compressão do ZNU Zip podem se tornar GZ, instaladores.

NOTA: Os arquivos TAR são frequentemente referidos como "tarballs," uma vez que coletam todos os tipos de arquivos em um arquivo, semelhante a uma tarball pegajosa.
