O que é um arquivo .fx? (Arquivo de efeitos direct3D)

Arquivo de efeitos criado para Direct3D, uma parte do Microsoft DirectX que é usado para definir a aparência de objetos 3D; contém propriedades de efeitos de texto simples estruturadas em três seções: variáveis, funções e técnicas; escrita usando a linguagem shader de alto nível da Microsoft (veja também .HLSL arquivos).

Mais Informações

Arquivos FX são usados para especificar texturas, parâmetros de iluminação, sombreamento, propriedades de renderização e outros efeitos.

NOTA: A tecnologia Direct3D é usada apenas para programas de Windows.

O que é um arquivo .fx? (Arquivo de rastreamento ReadyBoot)

Arquivo de sistema usado pelo ReadyBoot, uma tecnologia Windows que otimiza o processo de inicialização, permitindo tempos de carregamento mais rápidos do sistema; salva um traço de arquivos acessados no sistema que Windows usa como plano de inicialização para a próxima vez que o sistema for iniciado.

Mais Informações

ReadyBoot Os arquivos FX são criados e atualizados em segundo plano após uma reinicialização do sistema. Eles gravam arquivos acessados das últimas botas do sistema. Essas informações históricas permitem que Windows calcule um plano de cache para a próxima reinicialização que ajude a aumentar o desempenho da inicialização do sistema.

Você pode encontrar arquivos FX (por exemplo, Trace1.fx, Trace2.fx, etc) no seguinte diretório:

C:\Windows\ReadyBoost\Prefetch\Readyboot\
