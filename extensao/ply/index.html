O que é um arquivo .ply? (Arquivo de reprodução finale)

Arquivo de música criado pelo software de notação de música Finale; salva dados de áudio para partituras que podem ser reproduzidos no software; usado para reproduzir composições musicais.

O que é um arquivo .ply? (Arquivo de modelo polygon)

Arquivo de imagem tridimensional criado usando o Polygon File Format, um padrão aberto que descreve objetos como uma coleção de polígonos; contém um cabeçalho seguido por uma lista de vértices que especificam a forma de cada polígono.

Mais Informações

Arquivos PLY são usados por muitos programas 3D diferentes. Eles podem ser salvos em um formato binário ou de texto simples.

NOTA: Desde que o formato de arquivo PLY foi desenvolvido na Universidade de Stanford, ele também é conhecido como o Formato do Triângulo de Stanford.
