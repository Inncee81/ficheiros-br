O que é um arquivo .tap? (Imagem da fita 64 do Comodoro)

Imagem em disco criada a partir de uma fita de armazenamento de dados formatada para os sistemas Commodore 64 (C64) ou ZX Spectrum; contém uma cópia exata dos dados da fita em um único arquivo; normalmente usada para jogos ou outros tipos de aplicativos.

Mais Informações

Os arquivos TAP podem ser abertos em computadores Windows usando um emulador C64 ou ZX Spectrum.

O que é um arquivo .tap? (Toque em 'Tocar ''' Arquivo de toque de vingança)

Tap file usado pelo Tap Tap Revenge, um jogo de música semelhante ao Guitar Hero para iPhone e iPod Touch; contém uma linha do tempo de "taps" que são exibidos quando uma música é reproduzida em Tap Tap Revenge.

Mais Informações

Os arquivos TAP podem ser criados e editados usando software de terceiros, mas só podem ser usados pelo Tap Tap Revenge no iPhone ou iPod Touch.
