O que é um arquivo .sim? (Arquivo de simulação LogikSim)

Um arquivo SIM contém uma simulação de circuito criada pelo LogikSim, um programa de código aberto usado para testar e projetar circuitos lógicos. Ele armazena um design de circuito, que inclui pontos de conexão de fio e vários componentes.

Mais Informações

Captura de tela de um arquivo sim no LogikSim 064

Os arquivos SIM são criados e abertos usando o LogikSim. Os arquivos são usados para armazenar todos os tipos de circuitos, como portões E simples ou sistemas de computação complexos. Não há outro programa que suporte o tipo de arquivo LogikSim SIM. Mas logikSim está disponível para download gratuito.

NOTA: O LogikSim é normalmente usado por estudantes do ensino médio e universitário para aprender sobre circuitos e como eles funcionam.

O que é um arquivo .sim? (Arquivo tutor SAP)

Tutorial criado pelo SAP Tutor, um programa usado para gravar, editar e reproduzir unidades interativas de autoaprendito; contém uma lição, que pode incluir imagens de captura de tela, instruções de texto e perguntas de avaliação; usada para treinar os funcionários como operar programas de software.

Mais Informações

Os arquivos SIM podem ser reproduzidos usando o SAP Tutor Player, que é empacotado com o pacote SAP Tutor ou disponível como um programa autônomo.

NOTA: O tamanho dos arquivos SIM pode variar dependendo de quantas imagens de captura de tela estão incluídas no tutorial.

O que é um arquivo .sim? (Arquivo de informações de backup do jogo steam)

Arquivo de informações de backup usado pelo serviço de backup do Steam, que permite aos usuários fazer backup e restaurar jogos usando DVDs ou outras mídias externas; contém configurações e outras informações para o backup, juntamente com um arquivo .SIS correspondente.

O que é um arquivo .sim? (O arquivo Sims 3 Sim)

Um arquivo SIM contém um "Sim", que é uma pessoa virtual criada para o The Sims 3, um jogo de simulação de vida aberto. Ele armazena informações sobre uma pessoa Sim pré-feita ou personalizada, que inclui sua personalidade e aparência. Os arquivos SIM são normalmente usados para compartilhar com outros usuários e para fazer backup de pessoas Sims.

Mais Informações

O arquivo .SIM é um dos três principais tipos de arquivos, juntamente com .PACKAGE e .SIMS3PACK, usados no The Sims 3 para conter conteúdo personalizado. Os arquivos SIM são normalmente compactados em um arquivo .ZIP ou .RAR e precisam ser descomprimidos antes de serem instalados. Existem muitos programas disponíveis para extrair arquivos, como Apple Archive Utility, Corel WinZip e PeaZip. Depois de extrair o arquivo SIM do arquivo ZIP ou RAR usando um programa de descompressão, coloque o arquivo no seguinte diretório:

  • Windows XP: C:\Documents and Settings\[username]\My Documents\Electronic Arts\The Sims 3\SavedSims
  • Windows Vista/7/7/8/10: C:\Users\[username]\Documents\Electronic Arts\The Sims 3\SavedSims
  • macOS: /Users/[username]/Documents/Electronic Arts/The Sims 3/SavedSims

NOTA: Se não houver pasta "SavedSims", você pode criar um.
