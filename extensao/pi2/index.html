O que é um arquivo .pi2? (Arquivo de imagem de resolução média de Degas)

Formato gráfico Bitmap usado pela DEGAS, editor de imagens para sistemas Atari ST; versão de resolução média (640x200) e suporta 4 cores; pode ser aberto com software Degas em sistemas Atari ou com vários programas de visualização de imagens para Mac e PC.

O que é um arquivo .pi2? (Foto de Inovações de Retrato)

Formato de foto digital de alta resolução usado pelos estúdios Portrait Innovations; salvo em um formato de imagem proprietário que só pode ser visto usando o software Portrait Innovations.

Mais Informações

PI2 arquivos podem ser incluídos em CDs de foto, juntamente com imagens de .JPG de baixa res. Os arquivos Portrait Innovations (PI2) são versões de alta res das imagens que são criptografadas para fins de direitos autorais.
