O que é um arquivo .img? (Arquivo de dados de imagem do disco)

Um arquivo IMG é um arquivo de imagem de disco criado por vários aplicativos de imagem de disco, como o HH Software Virtual CD. Ele armazena uma cópia exata dos dados em um CD ou DVD e é usado para backups e gravação de novos discos.

Mais Informações

Arquivos IMG criados por diferentes programas de imagem de disco podem usar diferentes formatos. Portanto, você deve verificar a compatibilidade antes de usar arquivos IMG entre programas.

NOTA: Os arquivos IMG criados pela CloneCD devem ser salvos no mesmo diretório que os arquivos .CCD e .SUB, que são referenciados ao queimar novos CDs. CloneCD não é mais desenvolvido.

Desenvolvedor ERDAS

Formato Binário 

O que é um arquivo .img? (Arquivo de imagem ERDAS IMAGINE)

Imagem raster criada pelo ERDAS IMAGINE, um aplicativo de autoria de dados geoespaciais; contém uma ou mais camadas, chamadas bandas, que incluem várias telhas de imagem que compõem a imagem; usadas para armazenar pequenos e muito grandes conjuntos de dados geográficos.

Mais Informações

Os arquivos IMG podem conter informações adicionais, como estatísticas, pontos de controle de solo, informações do mapa e metadados de imagem. Eles também suportam camadas rasterestradas subsampladas, chamadas camadas de pirâmide, para exibição de tela mais rápida dentro de aplicativos GIS.

IMAGINE que as imagens usem o Formato de Arquivo Hierarerárquico ERDAS (HFA) para armazenar dados rasteres.

O que é um arquivo .img? (Imagem de disco disquete)

Imagem em disco tipicamente de um disquete de 35"," que geralmente tem uma capacidade de 144MB; armazena o conteúdo do disco como um único arquivo que pode ser virtualmente montado em um computador em vez da unidade disquete.

Mais Informações

Os arquivos de imagem de disquete não são mais comumente usados, uma vez que as unidades de disquete não estão mais incluídas na maioria dos novos sistemas de computador.

Desenvolvedor Novell

O que é um arquivo .img? (Imagem GEM)

Arquivo de imagem salvo no formato GEM; contém um bitmap que permite imagens monocromáticas, de escala cinza e coloridas; originada da Digital Research, que agora pertence à Novell.

Mais Informações

Os arquivos GEM eram comumente usados em sistemas operacionais mais antigos, mas não são mais usados com frequência.

Desenvolvedor Rockstar Games

O que é um arquivo .img? (Arquivo de dados de vídeo de grande roubo)

Arquivo de dados usado por Grand Theft Auto (GTA) III e IV, uma série de jogos onde os jogadores completam missões em ambientes urbanos da cidade; contém vários dados de jogos, incluindo gráficos de carros e unidades, armas e outros itens do jogo; muitas vezes alterados para criar conteúdo personalizado do jogo.

Mais Informações

Os arquivos IMG têm três versões diferentes de formato de arquivo:

  1. GTA III e GTA: Vice City: armazenado com um arquivo separado "dir"
  2. GTA: San Andreas: combina os arquivos "img" e "dir" em um único arquivo.
  3. GTA IV: suporta dados de jogos criptografados.
GTA III, GTA: Vice City e GTA: San Andreas usa o arquivo gta3.img para salvar dados do jogo. Este tipo de arquivo pode ser "modded" usando a ferramenta IMG do programa. Os arquivos GTA IV IMG podem ser editados usando SparkIV ou OpenIV.

Imagem de disco Macintosh

Desenvolvedor Apple
Categoria Imagens de Disco
Formato Binário 

O que é um arquivo .img? (Imagem de disco Macintosh)

Um arquivo IMG é uma imagem de disco usada para montar um sistema de arquivos. Ele pode ser montado como uma unidade de disco (por exemplo, um disco rígido externo) ou um disco óptico (por exemplo, um CD ou DVD). Os arquivos IMG são armazenados no disco rígido, mas são reconhecidos como um disco ou disco rígido quando montados pelo computador.

Mais Informações

Os arquivos IMG foram usados por Mac OS 9 e versões anteriores do Mac OS. Eles ainda podem ser abertos no macOS, mas foram substituídos pelo mais novo formato de imagem em disco .DMG.

NOTA: Em alguns casos, renomear a extensão de um arquivo IMG de "img" para "iso" permite que outros programas abram o arquivo.
