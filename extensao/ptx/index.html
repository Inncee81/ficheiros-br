O que é um arquivo .PTX? (Arquivo de transcrição de e)

Arquivo criado pelo E-Transcript Manager, um programa usado para criar transcrições eletrônicas certificadas; contém dados de transcrição gravados em uma audiência no tribunal; mais comumente usado por empresas de relatórios judiciais ao criar transcrições legais para advogados; criado para reduzir o tamanho dos arquivos das transcrições enviadas por e-mail.

Mais Informações

E-Transcripts também pode usar a extensão "exe", que permite que a transcrição seja visualizada a partir de um arquivo independente. No entanto, os filtros de e-mail geralmente interrompem a entrega de arquivos "exe". Por essa razão, o formato de arquivo PTX foi criado como formato não executável para contornar mais facilmente os filtros do servidor de e-mail.

PTX E-Transcripts podem ser convertidas entre "exe" E-Transcripts.

NOTA: A RealLegal, desenvolvedora do formato PTX, foi adquirida pelo LiveNote e, em seguida, pela Thomson West. Tanto as marcas West quanto RealLegal são usadas para anunciar e vender aplicativos de software semelhantes. A versão Mac do RealLegal E-Transcript Viewer só está disponível para Mac OS Classic, o antecessor de Mac OS X.

O que é um arquivo .PTX? (Arquivo de textura Paint Shop Pro)

Arquivo predefinido de textura usado pela Paint Shop Pro, um programa de edição de imagens; armazena informações de textura de imagem usadas para aplicar texturas a formas e objetos dentro do Paint Shop Pro; muitas vezes salvo junto com um arquivo .BMP correspondente.

O que é um arquivo .PTX? (Pentax RAW Image File)

Arquivo de imagem RAW não comprimido capturado com uma câmera Pentax; salva dados de imagem exatamente como registrado pelo sensor da câmera; pode ser visto usando o software Pentax incluído com a câmera.

Mais Informações

A maioria dos arquivos de imagem Pentax RAW usa a extensão .PEF.

Desenvolvedor Avid Technology
Categoria Áudio
Formato Binário 

O que é um arquivo .PTX? (Arquivo de sessão pro tools)

Formato de arquivo de sessão introduzido com o software DAW (Digital Audio Workstation, estação de trabalho de áudio digital) Pro Tools 10; suporta faixas de instrumentos de áudio, MIDI e software, bem como efeitos, configurações de mixagem e outras edições; armazena referências a arquivos de áudio, mas não aos próprios arquivos de áudio.

Mais Informações

NOTA: As Ferramentas Pro 7-9 usam o formato .PTF e não podem abrir arquivos PTX. Mas, você pode converter arquivos PTX para o formato PTF com Pro Tools 10 e posteriormente na caixa de diálogo Save Session Copy. Observe que alguns recursos mais novos do Pro Tools 10 serão perdidos se você economizar para o formato mais antigo, como configurações de ganho de clipe.
