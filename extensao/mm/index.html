O que é um arquivo .MM? (Arquivo do mapa da mente)

Um arquivo MM contém um "mapa da mente"," que é um diagrama visual que consiste em texto, nódulos, ramos, formas e ícones que representam ideias. Ele pode ser criado por uma variedade de programas "mapeamento da mente", como FreeMind e Freeplane. Arquivos MM são frequentemente usados para organizar ideias ao brainstorming, pesquisar tópicos, rastrear projetos e tomar notas.

Mais Informações

Captura de tela de um arquivo mm no FreeMind 1

Quando você cria um mapa no FreeMind, o arquivo MM é criado para armazenar seu conteúdo. Você pode reabrir o arquivo para editar seu conteúdo ou pode exportar o arquivo para outros formatos, como .HTML, .XHTML, .PNG, .JPEG, .PDF e .SVG.

FreeMind foi projetado para ajudá-lo a organizar visualmente as ideias. Você pode criar mapas que consistem em ideias colocadas em nós que podem ser conectados a outros nós e imagens. FreeMind é um programa de código aberto e está disponível como download gratuito.

O que é um arquivo .MM? (Arquivo macro NeXtMidas)

Arquivo macro usado pelo NeXtMidas, uma estrutura de processamento de sinal MIDAS em rede; contém uma sequência de comandos usados para automatizar processos dentro do aplicativo NeXtMidas; pode incluir procedimentos e subrotinas.

Mais Informações

Macros são convenientes de usar, pois automatizam longas sequências de comandos que são executados com frequência. Existem dois tipos de macros usadas pela NeXtMidas. Um tipo é uma macro instalada com valores padrão atribuídos aos seus parâmetros. Esse tipo pode ser instalado adicionando-o ao dicionário de comando. O segundo tipo é uma macro de usuário, que é uma macro desinstalada. Esse tipo não pode ter valores de parâmetro padrão dados pelo sistema.

MIDAS significa Sistema de Análise e Desenvolvimento Interativo Multi-usuário. NeXtMidas é uma versão em rede da concha MIDAS escrita em Java. É usado para manipular sinais digitalizados e outros dados de sistemas de coleta e processamento de sinais. Seu principal modo de operação é responder aos comandos de texto, o que torna o uso de arquivos macro MM importante.

NOTA: NeXtMidas requer Java 6 ou posterior para ser executado.

O que é um arquivo .MM? (Arquivo de origem Objective-C++)

Arquivo de código-fonte escrito nas linguagens de programação Objective-C e C++; difere de um arquivo .M, que contém apenas código Objective-C; usado para incluir o código C++ em aplicativos baseados em Mac.
