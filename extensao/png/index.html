O que é um arquivo .PNG? (Gráfico de rede portátil)

Um arquivo PNG é uma imagem salva no formato PNG (Portable Network Graphic, gráfico de rede portátil). Ele contém um bitmap comprimido com compressão sem perdas semelhante a um arquivo .GIF. Os arquivos PNG são comumente usados para armazenar gráficos web, fotografias digitais e imagens com fundos transparentes.

Mais Informações

O formato PNG é amplamente utilizado, especialmente na web, para salvar imagens. Ele suporta imagens coloridas de RGB de 24 bits indexadas (baseadas em paletas) ou 32 bits de RGBA (RGB com um quarto canal alfa). O formato também suporta imagens RGB ou RGBA de cores completas e não indexadas e imagens em escala de cinza.

Muitos tipos de usuários de computador podem encontrar arquivos PNG, seja baixando uma imagem de um site, recebendo uma foto digital anexada a um e-mail ou tirando uma captura de tela em um computador. Os profissionais gráficos geralmente salvam gráficos como arquivos PNG para uso na web. Eles também usam arquivos PNG para salvar modelos de logotipo ou ícones com fundos transparentes que podem ser colocados sobre outras imagens.

Como abro um arquivo PNG?

Você pode abrir imagens PNG com um grande número de programas gratuitos e comerciais, incluindo a maioria dos editores de imagens, editores de vídeo e navegadores da Web. Windows e macOS também vêm empacotados com programas que suportam imagens PNG, como Microsoft Photos e Apple Preview.

Como faço para converter um arquivo PNG?

Muitos dos editores de imagens que podem abrir arquivos PNG também podem converter as imagens em outros formatos. Se você quiser usar um programa empacotado com Windows, você pode usar Paint ou Paint 3D. Se você é um usuário de macOS, você pode usar o Apple Preview. Todos esses programas podem converter imagens PNG em vários formatos, incluindo JPEG e TIFF.

Por que o formato PNG foi criado?

PNG foi criado em 1995 para ser uma alternativa gratuita ao formato GIF para transferência de imagens na web. Melhorou as limitações do formato GIF, que suportava um número limitado de cores e incluía uma licença de patente. Além disso, enquanto as imagens GIF suportam apenas pixels totalmente opacos ou totalmente transparentes, as imagens PNG podem incluir um canal de transparência de 8 bits, que permite que as cores da imagem desbotem de opacas para transparentes.

Enquanto o formato PNG se tornou mais amplamente utilizado do que o GIF, o PNG é um formato de imagem única, enquanto o GIF suporta várias imagens. O formato GIF permitiu que os usuários salvassem animações simples como arquivos GIF e levou ao ressurgimento do formato on-line. Em resposta a essa limitação, o PNG Development Group criou o Multiple-image Network Graphics (Mac OS X (macOS) 104 e posteriores. O Ubuntu também Linux armazena capturas de tela de impressão no formato PNG.
