O que é um arquivo .app? (Aplicativo gerado pelo FoxPro)

Aplicativo usado pela FoxPro para automatizar procedimentos (ou seja, criação de novos tamanhos de etiquetas, transformação de valores de dados, análise de código, etc); pode ser incluído na FoxPro ou personalizado construído pelo usuário a partir de um arquivo de projeto (.PJX) usando o comando BUILD APP

O que é um arquivo .app? (Aplicativo macOS)

Um arquivo APP é um pacote de aplicativos que é executado sob macOS, um sistema operacional (OS) desenvolvido pela Apple e projetado para ser executado em computadores Macintosh. Armazena vários diretórios, como Contents, Resources e Frameworks, que contêm vários arquivos que compõem o aplicativo. Os arquivos APP são arquivos executáveis semelhantes aos .EXE arquivos em Windows e executados quando clicados duas vezes.

Mais Informações

Como os arquivos APP são pacotes macOS, você pode clicar com o botão direito (ou clicar no controle) em um aplicativo e selecionar "Mostrar conteúdo do pacote" para visualizar os arquivos contidos no pacote. Estes incluem recursos usados pelo aplicativo, executáveis, arquivos de lista de propriedades (.PLIST), frameworks, plugins, ícones e outros arquivos.

Em Windows e Linux sistemas, pode ser possível visualizar o conteúdo de um aplicativo macOS como se fosse uma pasta. Um programa de descompressão como o WinRAR também pode ser capaz de extrair arquivos de um aplicativo macOS em outros sistemas operacionais.

NOTA: Os arquivos dentro de um pacote de aplicativos não devem ser editados ou movidos, uma vez que as alterações podem impedir que o programa seja executado corretamente.

O que é um arquivo .app? (Aplicativo Symbian OS)

Programa de software escrito para o Symbian OS, um sistema operacional para telefones celulares e smartphones; compreende um aplicativo Symbian juntamente com um arquivo de recursos .RSC; pode ser compilado usando um kit de desenvolvimento de software Symbian OS (SDK).

Mais Informações

O Symbian OS é usado por telefones da Nokia, Sony Ericsson, Samsung e FOMA.

O que é um arquivo .app? (Arquivo de dados do jogo Wii U)

Um arquivo APP contém dados criptografados despejados de um videogame Nintendo Wii U. Ele armazena uma seção dos dados do jogo, juntamente com outros arquivos APP, que quando descriptografados e reunidos, podem ser abertos e reproduzidos em um computador usando um emulador Wii U.

Mais Informações

Os arquivos APP são usados por jogadores que gostam de jogar e alterar jogos wii u em seus computadores. Como os arquivos APP contêm peças de um jogo Wii U para reduzir o tamanho dos arquivos, todos os arquivos APP precisam estar presentes para jogar o jogo Wii U.

Além disso, os arquivos Wii U APP são criptografados e devem ser descriptografados antes de serem abertos e reproduzidos. Um metadados de metadados de metadados de título e a chave de título para o jogo APP, é necessário para descriptografar o arquivo APP.

OS arquivos APP são nomeados de acordo com o número de arquivos para os quais os dados do jogo foram divididos. Os arquivos começam em 0 (00000000.app), depois aumentam de lá para 00000001.app, 00000002.app, etc.
