O que é um arquivo .pkl? (Arquivo Python Pickle)

Um arquivo PKL é um arquivo criado pelo pickle, um módulo Python que permite que objetos sejam serializados em arquivos em disco e desergados de volta ao programa no tempo de execução. Contém um fluxo byte que representa os objetos.

Mais Informações

O processo de serialização é chamado "pickling," e a deserialização é chamada "unpickling" Um arquivo PKL é conservado para economizar espaço ao ser armazenado ou transferido por uma rede e, em seguida, é despickled e carregado de volta na memória do programa durante o tempo de execução. O arquivo PKL é criado usando picles Python e o método dump() e é carregado usando picles Python e o método load() método.

Existem três protocolos usados para o pickling:

  • Versão 0 - O protocolo ASCII original, que é retrocompatível com versões python anteriores.
  • Versão 1 - O antigo formato binário, que é retrocompatível com versões python anteriores.
  • Versão 2 - Formato Python 23, que apresenta uma picape mais eficiente de novas classes de estilo.

NOTA: Os arquivos PKL também podem ter a extensão .PICKLE, mas mais comumente têm a extensão .P.
