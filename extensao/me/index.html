O que é um arquivo .me? (Arquivo de bandeira de partição do filhote Linux)

Arquivo criado pela Puppy Linux, uma variante de Linux fácil de usar; usa o nome de arquivo FSCK.me; armazenado como parte de uma partição e marca a partição como tendo estados de salvamento (por exemplo, .2FS arquivos), e, portanto, tem que ser digitalizado por verificação do sistema de arquivos (FSCK) na inicialização.

Mais Informações

NOTA: O arquivo FSCK.me é uma maneira inteligente de dizer "sistema de arquivos me verifique" para uma partição.

O que é um arquivo .me? (Arquivo de texto Readme)

Readme arquivo que contém informações sobre um aplicativo de software; tipicamente chamado READ.ME e pode ser criado durante o processo de instalação do software.

Mais Informações

Os arquivos Readme são arquivos de texto simples que podem ser visualizados em qualquer editor de texto.

O que é um arquivo .me? (Arquivo de exclusão temporária de mim)

Arquivo temporário chamado delete.me ("Delete-me") e marcado para exclusão do computador; pode ser excluído pelo usuário manualmente.

Mais Informações

Uma vez que alguns arquivadores de arquivos/não-arquivos ignoram pastas vazias ao criar e desempacotar arquivos, um arquivo de 0 byte, vazio delete.me pode ser colocado nas pastas vazias para que o programa não os ignore.
