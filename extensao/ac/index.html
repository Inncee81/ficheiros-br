O que é um arquivo .ac? (Arquivo de definição AC3D 3D)

Um arquivo AC contém informações sobre o aparecimento de um modelo 3D no formato AC3D. Armazena várias informações de aparência, como malhas, caminhos de textura, mapeamentos UV e definições de materiais. Os arquivos CA são tipicamente criados pelo AC3D, um aplicativo usado para criar modelos 3D para jogos, simulação de voo e realidade virtual.

Mais Informações

Enquanto o AC3D é um programa popular foo criando arquivos CA, os arquivos são mais comumente usados pelos simuladores X-Plane e FlightGear. Os arquivos CA podem até ser usados para criar arquivos X-Plane .OBJ em AC3D e usando o plugin X-Plane para exportar para o formato X-Plane OBJ.

Uma vez que os arquivos CA são salvos em texto simples, você pode abri-los e modificá-los usando um editor de texto. Para abrir um arquivo CA com um editor de texto, você pode precisar alterar a extensão de arquivo .ac para .txt. Depois de visualizar ou salvar alterações no arquivo, altere a extensão do arquivo de volta para .ac.

NOTA: Os arquivos AC usados pelo FlightGear são frequentemente acompanhados por um arquivo .XML, que contém informações que animam o objeto ou cenário no jogo.

O que é um arquivo .ac? (Autoconf Script)

Script de configuração automática produzido pela GNU Autoconf; configura automaticamente pacotes de código-fonte de software para se adaptar a vários tipos de sistemas semelhantes ao Posix; criados testando um pacote de software para cada recurso que o programa pode precisar para executar um sistema específico; Em seguida, o Autoconf constrói uma lista de recursos do sistema que o pacote requer.

Mais Informações

Os scripts de autoconf são tipicamente chamados "configureac" O script de configuração automática é criado executando o programa de pré-processador m4 com o arquivo configurante como entrada.
