O que é um arquivo .cat? (Banco de dados avançado do catálogo de discos)

Contém um catálogo de dados de vários tipos de mídia, incluindo discos rígidos, unidades de rede, disquetes, discos ópticos (DVDs, CDs, etc), discos ZIP e JAZ, etc; usados para manter o controle das informações armazenadas dentro de uma coleção de mídia

O que é um arquivo .cat? (Arquivo do catálogo Windows)

Arquivo usado pelo sistema operacional Windows; especifica que um grupo de arquivos são de uma fonte verificável; usados para fins de segurança e contém uma assinatura digital, a versão do catálogo e uma data efetiva; muitas vezes usados para verificar novos arquivos de atualização de software.

Mais Informações

Windows arquivos de catálogo ajudam a suprimir vários popups de diálogo de confiança para downloads de novos componentes de software. Como os arquivos CAT podem identificar uma fonte confiável, o usuário pode optar por "Sempre confiar no conteúdo" do distribuidor de software. Em seguida, os downloads futuros do distribuidor de software que usam um arquivo CAT são confiáveis e não requerem uma janela popup de confiança.
