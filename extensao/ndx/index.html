O que é um arquivo .ndx? (arquivo de índice dBASE)

Arquivo de índice que define a estrutura e os campos de um banco de dados; consiste em um cabeçalho seguido por uma árvore B, que contém um nó raiz, ramo e folha; usado principalmente pelo dBASE, mas pode ser reconhecido por outros programas de banco de dados.

O que é um arquivo .ndx? (Arquivo neat 5)

Arquivo criado pela Neat 5, uma organizadora de documentos digitais; contém uma cópia exportada de uma pasta Neat de documentos ou documento único, como um recibo ou receita, que pode ser compartilhado entre usuários do Neat.

Mais Informações

Para criar um arquivo NDX, selecione a pasta no gabinete que deseja exportar, clique em Export → To Neat File → Selected Folder... (para toda a pasta) ou Selected Item(s)... (para itens selecionados de uma pasta), escolha o local de salvamento e clique em Salve.

Para importar um arquivo NDX, selecione Import → Neat File..., clique em "Procurar," navegar para o seu arquivo NDX e clique em Abra.

NOTA: O arquivo NDX é semelhante aos arquivos .NRX, .NRT e .NRF, que são usados por versões mais antigas do Neat.
