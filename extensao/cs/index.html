O que é um arquivo .cs? (C# Arquivo de código fonte)

Um arquivo CS é um arquivo de código-fonte escrito em C# (pronuncia-se "C Sharp"), uma linguagem de programação orientada a objetos criada pela Microsoft para uso com o NET Framework. É usado para desenvolver uma gama de aplicativos, desde programas simples de desktop até aplicativos para ambientes distribuídos.

Mais Informações

Captura de tela de um arquivo cs no Adobe Dreamweaver CC 2019

CS os arquivos são frequentemente editados com Visual C#, um componente do Microsoft Visual Studio. Mas, eles também podem ser editados com uma variedade de outros editores de código-fonte que apresentam destaque de sintaxe.

Você pode até mesmo abrir arquivos CS usando um editor de texto empacotado com Windows ou macOS, como Micorosft Notepad++ ou Apple TextEdit. No entanto, editores de texto simples não são boas opções para editar os arquivos, uma vez que eles não suportam a estrutura de sintaxe.

NOTA: Quando os programas C# são compilados, eles podem então ser executados pelo Tempo de Execução da Língua Comum (CLR), que está incluído no NET Framework.
