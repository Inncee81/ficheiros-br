O que é um arquivo .mq5? (Arquivo de código fonte MQL5)

Um arquivo MQ5 é um programa escrito para metaTrader, uma plataforma de negociação online. Ele contém código-fonte escrito no MetaQuotes Language 5 (MQL5), que inclui classes, variáveis, funções e comentários que podem ser compilados em um arquivo .EX5. Os arquivos MQ5 substituíram .MQ4 arquivos pela versão do MetaTrader 5.

Mais Informações

MQ5 arquivos podem ser abertos e editados pela MetaEditor, editora para criação de robôs de negociação e indicadores técnicos e incluídas no MetaTrader.

MetaEditor também pode compilar arquivos MQ5 em arquivos EX5 Expert Advisor, que são programas para automatizar os processos de negociação e análise no MetaTrader. O processo de compilação consiste em múltiplas etapas, como digitalização léxica, análise semântica, geração de código e otimização de código. Para compilar um MQ5, abra o arquivo de código em "Navegador" e selecione File → Compile. Se houver algum erro, a compilação não será concluída nenhum arquivo EX5 será criado. Os erros aparecerão na guia "Erros" na janela "Caixa de ferramentas".

O software MetaTrader é usado para analisar e negociar Ações, Forex, Futuros e CFDs. Ele vem com ferramentas de análise técnica e permite o uso de sistemas de negociação automatizados e negociação de cópias.
