O que é um arquivo .lib4d? (Biblioteca predefinida cinema 4D)

Biblioteca criada pelo Cinema 4D, um aplicativo profissional que permite criar modelos e animações 3D; contém uma ou mais predefinições que podem ser inseridas em uma cena pelo usuário; inclui objetos pré-definidos ou personalizados, como imagens, marcas de lápis, céus, sombreadores de tijolos, gradientes de pintura corporal e balões 3D.

Mais Informações

LIB4D Preset Library arquivos que vêm com o aplicativo estão localizados na pasta "browser" dentro da pasta "biblioteca", que está localizada com o aplicativo. Se você clicar duas vezes no arquivo LIB4D, a janela "Navegador de conteúdo" será aberta. Aqui você pode ver uma hierarquia de seus diferentes arquivos LIB4D. Você pode criar uma nova biblioteca predefinida selecionando File → New Preset Library..., nomeando-a e clicando em OK.

Ao usar a interface do aplicativo, você pode inserir um predefinido de objeto a partir de sua biblioteca predefinida. Selecione File → Load Object Preset no menu no lado direito da interface, navegue até o objeto e selecione-o. Você também pode salvar um objeto predefinido em sua biblioteca a partir desta mesma área selecionando Save Object Preset... no mesmo menu.
