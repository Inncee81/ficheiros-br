O que é um arquivo .lpk? (Pacote de camadas ArcGIS)

ARQUIVO DE DADOS GIS usado pelo ArcGIS Desktop, um pacote de software de mapeamento e visualização; contém uma camada de mapa e seus dados que podem ser importados para outro mapa ou globo; usado para compartilhar camadas de mapa auto-contidas com outros; pode ser carregado no serviço ArcGIS Online para compartilhamento pela Internet.

Mais Informações

LPK arquivos podem ser usados tanto pelo ArcMap quanto pelo ArcGlobe, que estão incluídos como parte do software ArcGIS Desktop.

O que é um arquivo .lpk? (Pacote de layout tecplot)

Pacote de visualização criado pela Tecplot, um aplicativo de visualização científica; salva um layout de parcela (arquivo .LAY) bem como os dados necessários para gerar as visualizações; útil para catalogar imagens ou fazer documentos grandes.

Mais Informações

Você pode salvar layouts do Tecplot em arquivos LAY, mas esses arquivos apenas especificam como exibir dados e não incluem os dados em si. Portanto, você pode empacotar tanto o layout quanto os dados em um único arquivo visual usando o formato LPK.
