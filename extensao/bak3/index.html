O que é um arquivo .bak3? (Arquivo de backup)

Arquivo de backup criado por vários programas, incluindo jogos como Crytek Crysis; contém uma cópia do arquivo original, como uma cena; semelhante às extensões de arquivo .BAK e .BAK2.

Mais Informações

A extensão "bak3" pode ser usada para representar a terceira versão de backup de um arquivo. Toda vez que você salva uma cena no Vue, outro arquivo BAK é criado. A extensão "bak3" seria usada na terceira salvação. Você pode escolher a quantidade máxima de backups que deseja criar usando a configuração "Backups Max" no Vue.

Outro uso da extensão "bak3" é fazer backup de um arquivo BAK2 que faça backup .CRY arquivos de nível de mapa. Se o seu arquivo CRY estiver corrompido, você pode substituí-lo renomeando a extensão "bak3" do seu arquivo para "cry"

O que é um arquivo .bak3? (Arquivo de backup starbound)

Um arquivo BAK3 é um arquivo de backup usado pelo Starbound, um jogo de exploração espacial 2D aberto e sandbox. Ele armazena informações de backup para o jogo, que podem ser sobre uma nave espacial ou personagem do jogador. Os arquivos BAK3 são úteis para recuperar informações no caso de uma falha no computador ou arquivo SHIPWORLD ou PLAYER corrompido.

Mais Informações

BAK3 os arquivos são nomeados com a extensão "bak3" adicionada a um arquivo PLAYER ou SHIPWORLD. Por exemplo, se o arquivo estiver armazenando informações de backup de um arquivo SHIPWORLD, o nome do arquivo será ab 07704dbcb4162646d4d93bbeef34df.shipworld.bak3.

Você pode excluir a extensão "bak3" do nome do arquivo e abrir o arquivo como um arquivo SHIPWORLD no Starbound. No entanto, é importante que você não altere a sequência de caracteres que precedem as extensões.

BAK3 arquivos podem ser encontrados na pasta "player" na pasta "Starbound". Por exemplo, se você comprou o Starbound através do Steam, os arquivos BAK3 são armazenados no seguinte local em Windows:

C:/Program Files/Steam/SteamApps/common/Starbound/player

>NOTA: Starbound também usa arquivos .BAK1 e .BAK2.
