O que é um arquivo .STC? (Arquivo do catálogo da Celestia Star)

Arquivo de dados usado pela Celestia, um aplicativo multiplataforma usado para criar simulações 3D do universo; armazena instruções para desenhar estrelas no universo; compreende parte do banco de dados global do universo quando o programa é carregado.

Mais Informações

Celestia carrega automaticamente um arquivo STC na inicialização se o arquivo estiver no local correto. Os arquivos personalizados stc, .DSC e .SSC de catálogo devem estar localizados dentro do \extras-standard\ subdiretório da instalação Celestia.

O que é um arquivo .STC? (Arquivo de chave de conexão de contribuição)

Arquivo de conexão criado pelo Contribute, um programa da Adobe usado para desenvolvimento e publicação de sites; salva o host remoto, o protocolo de conexão (por exemplo, FTP, SFTP, etc) e o nome de usuário; permite que os usuários reutilizem informações de conexão e distribuam entre os desenvolvedores do site.

Mais Informações

Quando você clica duas vezes em um arquivo STC, o Contribute abre uma conexão com o servidor remoto e mostra os arquivos remotos.

O que é um arquivo .STC? (Sombra para Manga Studio File)

3D modelo exportado de um modelo Shade 3D para um formato reconhecido pelo Manga Studio; criado pelo plug-in "Shade to Manga Studio", que está incluído no Manga Studio EX; pode ser importado em quadrinhos projetado com o Manga Studio.

Mais Informações

STC geralmente incluem modelos de personagens usados em quadrinhos. Manga Studio tornou-se Clip Studio Paint em 2015. Também é conhecido como ComicStudio no Japão.

O que é um arquivo .STC? (Arquivo temático siemens)

Arquivo temático usado por celulares Siemens; armazenado dentro de um arquivo temático global .SDT e contém marcação XML que especifica como o tema parece e o que soa para reproduzir; pode ser visto com qualquer editor de texto.

Modelo de planilha staroffice calc

Desenvolvedor Oracle
Categoria Planilha
Formato Binário 

O que é um arquivo .STC? (Modelo de planilha staroffice calc)

Modelo para criar um novo arquivo de planilha; originalmente usado pelo software StarOffice mais antigo e ainda suportado pelo mais recente Apache OpenOffice; criado pelo programa Calc incluído no StarOffice.

Mais Informações

O Apache OpenOffice agora usa o formato .OTS OpenDocument para armazenar modelos de planilhas.

NOTA: O Apache OpenOffice era anteriormente conhecido como OpenOfficeorg.
