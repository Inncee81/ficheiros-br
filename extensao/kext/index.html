O que é um arquivo .kext? (Mac extensão do kernel OS X)

Mac extensão OS X que fornece funcionalidade adicional de SO para um dispositivo ou programa; comumente usado para drivers de dispositivos periféricos; normalmente encontrado no diretório /System/Library/Extensons/.

Mais Informações

Você não deve mover ou excluir manualmente arquivos KEXT, pois pode causar um pânico no kernel e Mac o OS X congelar.
