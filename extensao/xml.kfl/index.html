O que é um arquivo .xml.kfl? (Lista de feeds conhecida do Microsoft Outlook)

Um arquivo XMLKFL é um arquivo de configurações usado pela versão 2007 e 2010 do Microsoft Outlook, um cliente de e-mail e gerente de informações pessoais. Ele armazena informações sobre um ou mais feeds RSS que são sincronizados entre um perfil de e-mail do Outlook e o Internet Explorer através do recurso Common Feed List.

Mais Informações

O arquivo XMLKFL é referenciado pelo Outlook se o Internet Explorer (IE) também estiver sendo executado no computador. O arquivo é usado pelo recurso CFL (Common Feed List, lista de feed comum), que permite que os usuários do Outlook recebam feeds RSS no aplicativo de e-mail sincronizando os feeds RSS do IE com o Outlook.

Os arquivos XMLKFL são frequentemente armazenados no perfil do usuário do Outlook no seguinte local:

  • Windows XP - C:\Documents and Settings\<username>\Local Settings\Application Data\Microsoft\Outlook
  • Windows Vista, 7, 8 e 10 - C:\Users\<username>\AppData\Local\Microsoft\Outlook

Se o recurso CFL estiver ativado, o Outlook verifica o arquivo XMLKFL quando o programa for aberto. Para desativar o recurso CFL no Outlook selecione Tools → Options, clique na guia "Outras"," selecione "Opções avançadas"," e remova os feeds "Sync RSS para a lista de feeds comum".
