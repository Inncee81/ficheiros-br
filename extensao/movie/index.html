O que é um arquivo .MOVIE? (Arquivo de filme payday)

Um arquivo MOVIE é um arquivo de vídeo usado por jogos payday, como Payday 2 e Payday: The Heist. Ele contém um vídeo usado no Payday, como quando os personagens oferecem contratos ao jogador durante uma cutscene. Os arquivos MOVIE são renomeados .BIK arquivos.

Mais Informações

Você provavelmente só se deparará com um arquivo MOVIE se estiver modificando a jogabilidade do Payday. Caso contrário, o arquivo é referenciado pelo jogo durante a jogabilidade.

NOTA: Uma vez que os arquivos MOVIE são renomeados arquivos BIK, você pode alterar a extensão "movie" para "bik" para abri-los com jogadores de vídeo que suportam o formato de vídeo Bink.

O que é um arquivo .MOVIE? (Arquivo de filme QuickTime)

Filme salvo ou formatado no formato QuickTime proprietário da Apple; pode usar um dos muitos tipos de codecs para compactar o arquivo, incluindo o codec de vídeo H264, que usa um algoritmo de compressão de alta eficiência; O suporte ao H264 foi introduzido com o QuickTime 70, que foi lançado com Mac OS X 104 Tiger e também está disponível com o QuickTime para Windows.

Mais Informações

A maioria dos filmes do QuickTime pode ser reproduzido por versões anteriores do QuickTime Player, mas o QuickTime 7 ou posterior é necessário para reproduzir arquivos de vídeo codificados por H264.

Os filmes do QuickTime também podem usar a extensão .MOV, que é especialmente comum em computadores baseados no Windows.
