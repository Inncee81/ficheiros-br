O que é um arquivo .tga? (Gráfico targa)

Um arquivo TGA é um arquivo salvo no formato gráfico raster projetado pela Truevision. Suporta 8, 16, 24 ou 32 bits por pixel a um máximo de 24 bits para cores RGB e canal alfa de 8 bits. Os arquivos TGA são usados para vários tipos de imagens, como fotos digitais e texturas referenciadas por videogames 3D.

Captura de tela de um arquivo tga no Apple Preview 101

O formato Targa, que significa Truevision Advanced Raster Graphics Adapter, foi projetado pela Truevision (agora Avid Technology) em 1984 para uso com seus primeiros programas de software de vídeo. O formato original consistia em 2 áreas:

  1. TGA File Header
  2. Imagem/Color Map Data

Enquanto essas duas seções foram incluídas na especificação original de 1984 do formato Targa, "Área do desenvolvedor", " "Área de extensão," e "TGA File Footer" foram adicionadas em 1989. Esta especificação é conhecida como "Novo Formato TGA" e consiste nas seguintes 5 áreas:

  1. TGA File Header
  2. Image/ColorMap Data
  3. Área do Desenvolvedor
  4. Extensão
  5. TGA File Footer

Além das três novas seções do formato de imagem, A Truevision também introduziu várias novas extensões ao formato Targa em 1989 para atender às necessidades dos profissionais gráficos. Exemplos das extensões extras incluem:

  • Data e hora em que o arquivo foi criado
  • Nome do autor e comentários
  • Tipo de canal Alfa
    • >
  • Proporção de pixel
  • Valor gama
  • Uma cópia de selo postal reduzido da imagem (uma imagem "miniatura"
