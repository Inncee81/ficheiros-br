O que é um arquivo .DBM? (Arquivo do servidor ColdFusion)

Web página escrita no DBML (Database Markup Language), o antecessor do CFML (ColdFusion Markup Language) da Adobe); executado por um servidor web executando o ColdFusion, que gera HTML que é enviado para o computador de um usuário; semelhante a um arquivo .ASP ou .PHP, mas requer que o ColdFusion seja executado.

Mais Informações

A linguagem DBML foi usada pela ColdFusion 5 e anteriormente. Versões mais recentes do ColdFusion usam a extensão .CFM.

O que é um arquivo .DBM? (Arquivo material de tijolo DAZ)

Arquivo criado pelo DAZ Studio, um aplicativo de modelagem de caracteres 3D; armazena uma rede de mixer de sombreamento, que especifica como um material é sombreado; pode ser editado e visualizado com a tela aberta fornecida pela exibição Shader Mixer; salva em um formato XML.

Mais Informações

NOTA: Selecione Window → Tabs → Shader Mixer para abrir o Mixer Shader no DAZ Studio. Escolha File → Open Shader... dentro da exibição Do Misturador de Sombreador para abrir um arquivo DBM e File → Save Shader... para salvar um arquivo DBM.
