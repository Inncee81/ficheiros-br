O que é um arquivo .ufs? (Arquivo do sistema de arquivos FreeBSD Unix)

Arquivo do sistema de arquivos Unix criado para PC-BSD, um sistema operacional Unix baseado no FreeBSD; armazena todo o disco ou uma parte de um disco, incluindo arquivos, diretórios e a estrutura do sistema de arquivos; usado para clonar sistemas de arquivos para instalações de rede PC-BSD.

Mais Informações

Os arquivos UFS podem ser compactados em .UZIP arquivos que usam a extensão de vários arquivos .UFS.UZIP.
