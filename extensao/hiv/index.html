O que é um arquivo .hiv? (Arquivo colmeia de registro de Windows)

Arquivo de registro criado pelo reg.exe, um programa de Windows usado para salvar, carregar e excluir informações do registro; contém um registro "colmeia," que salva dados de valor-chave usados pelo sistema operacional e programas instalados; usado para backups ou para carregar grupos de chaves de registro; salvo em um formato binário.

Mais Informações

Você pode salvar colmeias de registro usando reg.exe digitando o seguinte no prompt de comando (retirado diretamente da reg.exe ajuda da linha de comando):

REG SAVE HKLMSoftwareMyCoMyApp AppBkUphiv

Você pode carregar colmeias de registro usando este comando (novamente retirado diretamente da linha de comando reg.exe ajuda):

REG LOAD HKLMTempHivehiv

Digite REG /? no prompt de comando Windows para maior ajuda com reg.exe comandos. O programa está localizado no diretório C:\Windows\System32\, mas pode ser executado a partir de qualquer diretório no Windows comando prompt.

NOTA: Tenha cuidado ao salvar e carregar informações cadastrais, uma vez que a corrupção do registro de Windows pode causar o mau funcionamento dos aplicativos ou de todo o sistema. O arquivo setupreg.hiv incluído em Windows discos de instalação não devem ser editados manualmente, uma vez que é usado pelo instalador Windows.
