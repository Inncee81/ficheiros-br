O que é um arquivo .id? (Lotus observa arquivo de identificação do usuário)

Um arquivo de identificação contém informações de identificação para um usuário do Domino Server e do Lotus Notes. Ele é criado pelo administrador do Domino e inclui o nome do usuário, número de licença notas, certificado seguro, chave pública e privada e uma senha.

Mais Informações

Os IDs do usuário são obrigados a fazer login no IBM Lotus Notes. Um ID de notas pode ser necessário para acessar um servidor Domino. Eles normalmente expiram após um ano e, portanto, devem ser renovados anualmente.

Você pode visualizar as informações armazenadas em seu arquivo de ID selecionando File → Security → User Security no programa Lotus Notes. Uma vez que os arquivos de ID armazenam informações confidenciais, eles devem ser armazenados em um local seguro e bloqueados.

Se seu ID estiver perdido ou você esquecer sua senha, você poderá recuperar seu ID, mas apenas se o administrador configurar informações de recuperação para o seu ID. Você pode verificar isso selecionando Mail Recovery ID no painel "Security Basics" da caixa de diálogo "Segurança do usuário". Se o botão não estiver presente, o administrador não configurou informações de recuperação para o seu ID e você não poderá recuperar seu ID.
