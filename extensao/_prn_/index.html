O que é um arquivo .prn? (Arquivo de texto formatado Lotus 1-2-3)

Um arquivo PRN é um arquivo de texto criado pelo software de planilha Lotus 1-2-3. Ele contém uma exportação da folha atualmente ativa e é salvo em um formato de texto simples. Os arquivos PRN podem ser usados para importar dados de planilhas para outros programas, como o Microsoft Excel.

O que é um arquivo .prn? (Arquivo imprimível)

Um arquivo PRN contém instruções para uma impressora, que inclui o conteúdo a ser impresso, o número de páginas para imprimir, o tamanho do papel e a bandeja da impressora para usar. Ele é criado escolhendo "Imprimir para Arquivar" dentro da caixa de diálogo Imprimir em um programa Windows ou macOS. Os arquivos PRN são semelhantes aos arquivos PostScript (.PS).

Mais Informações

PRN os arquivos podem ser impressos em Mac OS X arrastando o arquivo para o ícone da impressora desktop. Em Windows, eles podem ser impressos clicando com o botão direito do mouse no arquivo e selecionando "Microsoft Office Document Imaging" (apenas algumas versões do Microsoft Office Document Imaging suportam arquivos PRN). Você também pode abri-los através da linha de comando Windows. Para fazer isso, selecione "Run" no Menu Iniciar, digite "cmd," e execute as seguintes etapas:

Se a impressora estiver conectada via USB, digite:
"COPY /B [filenameprn] \\[Nome do computador]\[Nome da impressora compartilhada]"

Se a impressora estiver conectada por meio de uma conexão de porta paralela mais antiga, digite:
COPY /B [filenameprn] LPT1:"

NOTA: Não inclua os [suportes] ao digitar o comando real.

Você também pode renomear o arquivo PRN como um arquivo .XPS e abri-lo com vários programas, como Microsoft XPS Viewer, Pagemark XpsViewer e NiXPS View. Para renomear o arquivo, selecione o nome do arquivo e renomeie .prn para .xps.

Você pode abrir um arquivo PRN com um editor de texto, como Microsoft Notepad ou Apple TextEdit. O arquivo armazena o texto a ser impresso como texto simples, mas as instruções de impressão consistem em códigos de impressora, que não devem ser modificados ou então você encontrará erros na impressora.
