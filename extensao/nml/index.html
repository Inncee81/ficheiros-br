O que é um arquivo .nml? (Arquivo de nível exportado do editor NSMB)

Arquivo de jogo criado pela NSMBe (New Super Mario Bros. Editor), um programa usado para fazer níveis personalizados para o jogo Nintendo DS New Super Mario Bros; salva um nível exportado criado pelo usuário e pode incluir blocos, moedas, Koopas, itens e outros objetos.

Mais Informações

Os arquivos NML são usados para exportar níveis individuais dentro de um novo mundo super mario bros. Esses níveis podem ser compartilhados com outros e, em seguida, aplicados como um patch a uma ROM .NDS existente do jogo NSMB usando o botão Export Patch na guia "Ferramentas/Opções" na guia NSMBe.

Para importar arquivos NML e aplicá-los em um patch de jogo, primeiro você deve ter o arquivo ROM NDS para o jogo NSMB. Depois de abrir o arquivo NDS no NSMBe, você pode importar e exportar níveis a partir da guia "Level Listing" .

NOTA: NSMBe também é referido como NSMBe5.

O que é um arquivo .nml? (Arquivo de coleta traktor)

Arquivo usado pelo Traktor, um aplicativo de mixagem de DJ usado para criar mixagens de áudio; salva a lista de reprodução do usuário em um formato XML; inclui a lista de artistas, lançamentos, rótulos, gêneros, faixas, amostras e conjuntos de remixes; não inclui nenhum arquivo de amostra de áudio real, mas inclui referências a eles.

Mais Informações

Arquivos NML são úteis para alternar entre diferentes listas de reprodução ao criar diferentes mixagens.
