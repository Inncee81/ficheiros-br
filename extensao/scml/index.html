O que é um arquivo .scml? (Arquivo XML de configuração Spriter)

Um arquivo SCML contém informações no formato XML usado para configurar uma animação 2D que inclui vários sprites. É normalmente usado por videogames 2D, incluindo aqueles que usam o Exient XGS Engine, como Angry Birds Go! e Angry Birds Transformers. Os arquivos SCML são criados e abertos pelo BrashMonkey Spriter, que é um programa multiplataforma usado para criar animações 2D.

Captura de tela de um arquivo scml em BrashMonkey Spriter

Para abrir um arquivo SCML em BrashMonkey Spriter, siga estas instruções:

  1. Open BrashMonkey Spriter.
  2. Selecione File → Open Project....
  3. Navegue até o arquivo SCML, selecione-o e clique em Abra.

Para criar um arquivo SCML em BrashMonkey Spriter, siga estas instruções:

  1. Open BrashMonkey Spriter.
  2. Selecione File → New Project....
  3. Escolha ou crie a pasta raiz para você projetar que contenha apenas as imagens usadas na animação.
  4. Selecione File → Save Project... ou Save Project As....
  5. Escolha o local de salvamento, nomeie o arquivo e clique em Save.

Arquivos SCML são usados pelo BrashMonkey Spriter como arquivos de projeto que podem ser modificados, salvos e reabertos para posterior modificação. Os arquivos armazenam todas as informações de configuração da animação, que inclui os caminhos para os ativos, informações de pivô XY para cada ativo e a linha do tempo da animação. Quando embalado com um jogo, o jogo faz referência ao arquivo SCML para reproduzir a animação apropriada.

Como os arquivos SCML são salvos no formato XML, você também pode abri-los e modificá-los usando um editor de texto, como Microsoft Notepad ou Apple TextEdit. No entanto, tenha cuidado com o que você edita porque os arquivos SCML armazenam caminhos para ativos, como .PNG e .DB arquivos que compõem a animação. Se você quebrar esses caminhos, a animação não aparecerá mais corretamente no jogo.
