O que é um arquivo .prvx? (Modelo de apresentações)

Um arquivo PRVX é um modelo de apresentação criado pela Presentations, um programa de apresentações de slides incluído no pacote SoftMaker Office. Ele contém uma lista ordenada de um ou mais slides, que podem incluir texto, imagens, tabelas, animações, objetos desenhados e outros meios de comunicação. Os arquivos PRVX são normalmente usados para criar planilhas .PRD e .PRDX com a mesma formatação ou dados.

Captura de tela de um arquivo prvx em Apresentações do SoftMaker Office 2018

Os arquivos PRVX são salvos em um formato proprietário desenvolvido pela SoftMaker Software. Mas eles são semelhantes aos modelos .POTX criados pelo Microsoft PowerPoint, que é um programa popular de apresentação de slides incluído no Microsoft Office. Os arquivos PRVX podem ser convertidos em vários formatos em Apresentações, incluindo .PPT, .PPTX, .PDF, .HTML, .PNG, .JPG, .TIF, .BMP, .WMV, .MP4 e .RTF.

Você pode criar um arquivo PRVX selecionando File → Save ou Save as e, em seguida, escolhendo o formato PRVX. Você pode abrir um arquivo PRVX selecionando File → Open e navegando para o arquivo PRVX. Você também pode escolher File → New e selecionar o modelo do qual deseja criar seu novo documento. Os arquivos PRVX são salvos por padrão na pasta "Designs [versão]" da pasta "SoftMaker" no caminho da pasta "Documents".

NOTA: Os arquivos PRVX substituíram .PRV arquivos como o tipo de arquivo principal usado para armazenar modelos em Apresentações.
