O que é um arquivo .smc? (Arquivo de dados do cartão SmartMedia)

Um arquivo SMC contém dados despejados de um SmartMedia Card, um cartão de memória flash usado por vários dispositivos para armazenar informações. Armazena uma cópia exata dos dados transferidos do cartão. Os arquivos SMC são mais usados para conter dados para jogos GP32, que é um console portátil lançado em 2001.

Mais Informações

Os cartões SmartMedia são cartões finos que se parecem com disquetes e podem ser inseridos e removidos de dispositivos para armazenar e transferir informações. As cartas foram introduzidas na década de 1990 e foram comumente usadas nos anos 2000 com câmeras digitais, PDAs, reprodudores de áudio digital portáteis e consoles de jogos portáteis.

A capacidade de armazenamento dos cartões varia de 2MB a 128MB, o que significa que os arquivos SMC podem variar de 2MB a 128MB de tamanho, uma vez que armazenam uma cópia exata dos dados. À medida que a tecnologia melhorou, melhores dispositivos de armazenamento foram desenvolvidos e a produção de cartões SmartMedia cessou na década de 2010.

Os arquivos SMC do SmartMedia Card são tipicamente usados por entusiastas de jogos, especificamente jogadores nostálgicos que gostavam de jogar o console de jogos portátil GP32 da Coreia do Sul fabricado pelo Game Park. Os arquivos SMC que armazenam dados de jogos podem ser abertos e emulados em um computador usando MAME e GeePee32.

O que é um arquivo .smc? (Arquivo verificador de modelo Solibri)

Um arquivo SMC contém um modelo 3D salvo no formato Solibri Model Checker (SMC), que é um formato proprietário usado pelo software Solibri. Ele armazena um modelo de um edifício ou instalação ou seus componentes e inclui elementos espaciais, materiais e formas. Os arquivos SMC só são suportados pelo software Solibri.

Captura de tela de um arquivo smc no Solibri Office 9

Você provavelmente só encontrará um arquivo Smc Solibri se você for um profissional de arquitetura, engenharia ou construção (AEC) usando solibri Office, Solibri Site ou Solibri Anywhere. Você também pode receber um arquivo SMC de outro profissional da AEC que está compartilhando um design criado com o software Solibri como parte do processo DE Modelagem de Informações de Construção (BIM).

Para abrir um arquivo SMC com o Solibri Office, selecione "FILE," clique em Open Model... e escolha o arquivo SMC que deseja abrir. Para salvar um arquivo SMC com o Solibri Office, selecione "FILE," clique em Save Model ou Save Model as....

O que é um arquivo .smc? (Jogo de Super Nintendo ROM)

Game ROM usado por emuladores SNES como Snes9x e ZSNES; contém uma cópia dos dados do jogo armazenados no cartucho original do Super Nintendo; permite que os jogos SNES sejam jogados no PC com um emulador de suporte.
