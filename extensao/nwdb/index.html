O que é um arquivo .nwdb? (Arquivo de banco de dados puro)

Um arquivo NWDB é um arquivo de banco de dados criado pela Neat 5, uma organizadora de documentos digitais. Contém uma ou mais imagens de documentos, contatos ou recibos digitalizados. Os arquivos NWDB são criados automaticamente quando você digitaliza um documento.

Mais Informações

Você provavelmente só verá o arquivo NWDB se você pesquisar a pasta Dados Limpos na pasta Documentos. Deve haver pelo menos três arquivos na pasta Neat Data: um arquivo NEATworks NWDB, um arquivo NWDB de imagem e um arquivo NWCAB de gabinete.

Se você se deparar com uma mensagem de erro indicando que não pode abrir um banco de dados Neat, provavelmente significa que um arquivo em seu banco de dados está corrompido, bloqueado por outro programa ou desaparecido. Você deve verificar se todos os três arquivos estão na pasta Dados Limpos. Se um dos três arquivos está faltando, então ele causa a mensagem de erro.

NOTA: Um arquivo NWDB é na verdade um banco de dados SQL Server Compact Edition (SQLCE), mas em vez da extensão de arquivo .sdf, ele tem a extensão de arquivo .nwdb. Portanto, quaisquer programas que possam abrir .SDF arquivos de banco de dados compactos do SQL Server também podem abrir arquivos NWDB.
