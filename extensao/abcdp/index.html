O que é um arquivo .abcdp? (Arquivo de pessoa CoreData do catálogo de endereços)

Arquivo de metadados de backup criado pelo Catálogo de Endereços, um aplicativo de organização de contato empacotado com o OS X; contém uma lista de propriedades para uma entrada de cartão de dados do Catálogo de Endereços com informações sobre uma pessoa.

Mais Informações

Quando o Catálogo de Endereços é aberto, os arquivos ABCDP são usados para construir ou reconstruir o banco de dados SQLite, que é usado para pesquisa rápida. Os arquivos também permitem a pesquisa de endereços nos aplicativos Mail e iChat, que são integrados ao Catálogo de Endereços.

Arquivos ABCDP podem ser encontrados no seguinte diretório:/Library/Application Support/AddressBook/Metadata

NOTA: O formato ABCDP foi introduzido no OS X 106. No OS X 108, o aplicativo Catálogo de Endereços foi substituído pelo aplicativo Contatos.
