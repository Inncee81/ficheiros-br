O que é um arquivo .oem? (Arquivo de configuração OEM)

Arquivo OEM (Original Equipment Manufacturer) usado durante a configuração do modo de texto de Windows sistemas operacionais; contém uma lista de componentes de hardware, arquivos para copiar para cada componente de hardware e uma lista de chaves e valores de registro para criar para cada componente.

Mais Informações

Os arquivos OEM são comumente usados para inicialar Windows os sistemas operacionais carregam drivers específicos de dispositivos. Um arquivo OEM comum que é usado na configuração Windows é chamado txtsetupoem.

O que é um arquivo .oem? (Arquivo de salvamento OEMEdit)

Arquivo criado pelo OEMEdit, um programa usado para editar os dados do OEM sobre Windows Vista, Windows 7 e Windows 8 sistemas; salva informações do fabricante e do modelo do sistema, uma referência a um logotipo gráfico e informações de suporte opcionais (número de telefone, etc); pode ser usado para aplicar as mesmas informações do sistema em vários computadores.

Mais Informações

Você pode aplicar uma edição do OEM clicando no botão Save na interface OEMEdit. Você pode visualizar as informações atualizadas do OEM no Painel de Controle de Windows em Sistema e Segurança → Sistema.

NOTA: Os arquivos OEM salvos pelo OEMEdit não usam um prefixo de nome de arquivo e são nomeados .oem.
