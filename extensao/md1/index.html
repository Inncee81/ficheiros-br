O que é um arquivo .md1? (Segmento de imagem do disco de álcool 2)

O segundo segmento de uma imagem de CD ou DVD copiada pelo software Alcohol 120% ; parte de uma sequência de segmentos de imagem de disco (precedido por .MD0, seguido por .MD2, etc); deve ser acompanhado com os outros segmentos para recuperar a imagem completa do disco.

Mais Informações

Os arquivos MD1 são frequentemente salvos com um arquivo .MDF ou um arquivo .MDS, que define as faixas e a estrutura de dados usadas pelo disco.
