O que é um arquivo .mobirise? (Arquivo do projeto Mobirise)

Um arquivo MOBIRISE é um projeto criado pela Mobirise, um programa gratuito de Windows, macOS e Android que permite que os usuários construam sites sem qualquer codificação. Ele contém "blocos"," que compõem o site, que pode incluir um menu, cabeçalho, artigos, imagens, vídeos, controles deslizantes, galerias, recursos, tabelas de preços, formulários, tabelas e rodapés. Os arquivos MOBIRISE são salvos no formato JSON.

Captura de tela de um arquivo mobirise em Mobirise 4

O Mobirise permite que os usuários arrastem e solucionem "blocos"," em um site para personalizar a aparência e funcionalidade do site. Esses blocos são salvos no formato JSON no arquivo MOBIRISE. Quando o site é publicado por um usuário, .HTML, .CSS e .JS arquivos são criados para cada bloco, dependendo da funcionalidade do bloco.

Para criar um novo projeto no Mobirise, selecione "Sites" no menu principal, clique em "Criar novo site," selecione o tema do site, nomeie o site e clique em "OK" O arquivo MOBIRISE é criado para salvar o conteúdo do projeto. Se você não nomear o site, o nome do site será Site-MM-DD_TTTT, com MM representando o mês, DD representando o dia, e TTTT representando o tempo que o site foi criado.

Para abrir um site no Mobirise, selecione "Sites" no menu principal e clique no site que deseja abrir. Se o site ainda não estiver no Mobirise, selecione "Sites" no menu principal, clique em "Import Mobirise Site," e selecione o arquivo MOBIRISE que deseja importar no explorador.

Para exportar um site no Mobirise, selecione "Sites" no menu principal, clique no ícone de engrenagem "Configurações do site que deseja exportar", selecione "Export Site"," e escolha o local para o qual deseja salvar o site.
