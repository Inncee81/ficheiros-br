O que é um arquivo .csv? (Arquivo de valores separados de círgula)

Um arquivo CSV é um arquivo de valores separados de círia comumente usado por programas de planilha, como Microsoft Excel ou OpenOffice Calc. Ele contém conjuntos de dados de texto simples separados por írgulas com cada nova linha no arquivo CSV representando uma nova linha de banco de dados e cada linha de banco de dados composta por um ou mais campos separados por uma círgula. Os arquivos CSV são frequentemente abertos por programas de planilha para serem organizados em células ou usados para transferir dados entre bancos de dados.

O formato de troca de dados CSV é suportado por uma grande quantidade de programas pessoais, empresariais e científicos. Devido ao seu amplo suporte, o formato é especialmente útil na transferência de dados tabulares entre programas.

O arquivo também é útil para transferir dados salvos em um formato proprietário, como um arquivo .XLSX, para outro programa que não suporta o formato XLSX. Por exemplo, você pode exportar milhares de contatos de um serviço de e-mail para um arquivo CSV e, em seguida, importar o arquivo para um organizador de endereços ou outro serviço de e-mail.
