O que é um arquivo .ldb? (Arquivo de bloqueio de acesso do Microsoft)

Um arquivo LDB é um arquivo de informações de bloqueio que impede que um banco de dados access seja alterado por mais de um usuário por vez. Também ajuda a garantir que as atualizações não serão exageradas inesperadamente. Os arquivos LDB são criados e excluídos automaticamente pelo Access quando um banco de dados é aberto e fechado.

Mais Informações

Os arquivos LDB não devem ser abertos pelo usuário, eles são destinados apenas para bloquear informações em um banco de dados .MDB Access. Ao bloquear as informações no banco de dados, vários usuários não podem substituir as atualizações uns dos outros.

NOTA: Os arquivos LDB têm o mesmo nome do banco de dados aberto (o arquivo .MDB) e estão localizados na mesma pasta.

O que é um arquivo .ldb? (Arquivo de nível minecraft)

Um arquivo LDB contém um nível usado pelas versões Windows 10, Android e iOS do Minecraft, um popular jogo de construção de blocos de Minecraft. Ele armazena informações sobre um nível jogável, ou mundo, em formato de valor chave, que é baseado no formato LevelDB desenvolvido pelo Google. Os arquivos LDB são compactados com compactação Zlib.

Mais Informações

Os arquivos LDB são frequentemente associados a "Minecraft: Pocket Edition" e "Bedrock Engine" porque o formato LDB foi introduzido em Minecraft: Pocket Edition (o nome das primeiras versões móveis de Minecraft) que também é quando o jogo usou pela primeira vez o Bedrock Engine. No entanto, o nome "Pocket Edition" acabou sendo descartado com o lançamento do "Better Together Update" em 20 de setembro de 2017. As versões do Pocket Edition agora são simplesmente chamadas "Minecraft"

Em Windows 10, Android e iOS, os arquivos LDB estão localizados no caminho /db de uma pasta de nível no diretório de instalação do jogo. Os arquivos LDB são frequentemente localizados com arquivos .DAT e .LOG.
