O que é um arquivo .til? (Arquivo de textura da Revolução CTGP)

Um arquivo TIL é um arquivo que contém texturas usadas pelo CTGP Revolution (CTGP-R), um mod feito por fãs para Mario Kart Wii que adiciona recursos extras, como novas faixas e modos de jogo. Ele armazena uma coleção de gráficos referenciados no CTGP-R, que inclui botões, títulos de opções de menu e ícones.

Mais Informações

Você só encontrará arquivos TIL se baixar o mod CTGP-R e cavar os arquivos de recursos embalados com ele. O arquivo não deve ser aberto manualmente pelos usuários. Em vez disso, é referenciado pelo console Wii ou Wii U para carregar recursos para menus e gráficos no jogo.

No entanto, alguns usuários que procuram modificar a aparência do menu CTGP-R abrirão manualmente um arquivo TIL para alterar as texturas armazenadas dentro dele. Depois de fazer suas alterações, eles embalarão as texturas de volta no arquivo TIL no mod CTGP-R, transferirão-na para um cartão SD e inserirão o cartão SD no Wii ou Wii para você jogar o mod. O usuário deve ter uma cópia válida de Mario Kart Wii para jogar o mod.
