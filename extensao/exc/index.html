O que é um arquivo .exc? (Arquivo de zona preta e branca)

Arquivo de dados de jogos usado pela Black & White, um jogo de estratégia onde os jogadores agem como deuses e controlam aldeias de pessoas; armazena informações para uma zona, que é um nível para o jogo; localizado no diretório \Data\Zones\ dentro de uma instalação black and white.

Mais Informações

Arquivos EXC também são usados pela expansão Black & White Creature Isle.

NOTA: Lionhead Studios Black & White foi descontinuado.

O que é um arquivo .exc? (Arquivo de auto-extração de auto-extração do gerenciamento do eDataSecurity)

Arquivo auto-extrativo criado pelo eDataSecurity Management, um programa usado para criptografar documentos privados; contém exatamente os mesmos dados do arquivo de origem, mas é criptografado para proteger seu conteúdo; semelhante ao arquivo .ENX.

Mais Informações

O objetivo do arquivo EXC é permitir que os usuários compartilhem mais facilmente seu arquivo criptografado com outras pessoas. O formato é auto-extração, o que permite que uma pessoa abra o arquivo EXC sem usar o eDataSecurity Management. No entanto, o receptor do arquivo deve renomear a extensão "exc" para "exe" para abrir o arquivo criptografado.

NOTA: eDataSecurity Management foi descontinuado após Windows Vista.
