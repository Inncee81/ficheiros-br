O que é um arquivo .fxml? (Arquivo de código-fonte FXML)

Contém código escrito na linguagem de programação FXML desenvolvida pela Oracle Corporation; usada para criar a interface de usuário de um aplicativo JavaFX 20; com base no XML que usa a extensão de arquivo .XML.

Mais Informações

JavaFX é usado para desenvolver aplicativos de internet ricos e pode usar o FXML para desenvolver suas interfaces de usuário em vez de escrevê-lo usando código processual. Os arquivos FXML são usados juntamente com .JAVA arquivos para criar um aplicativo.
