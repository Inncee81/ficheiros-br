O que é um arquivo .imt? (Meu arquivo de dados de tempo)

Arquivo de dados criado pelo Intuit My Time, um aplicativo de rastreamento de tempo para Mac OS X; economiza informações da folha de tempo em um formato XML; pode ser importado para o QuickBooks 2010 por Mac.

Mais Informações

Os dados salvos em arquivos IMT também podem ser exportados pelo My Time para um formato de texto padrão que pode ser importado e editado por aplicativos de planilha.
