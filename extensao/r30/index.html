O que é um arquivo .r30? (WinRAR Split Archive Parte 30)

Arquivo usado como parte de um arquivo RAR compactado de vários volumes; contém conteúdo de arquivo compactado como parte de um arquivo maior; pode ser reconstruído com as outras partes compactadas de volta ao arquivo original.

Mais Informações

RXX, onde "XX" é um número de 00 a 99, geralmente são criados para salvar um grande arquivo em vários discos. Os arquivos de arquivos usando o formato de extensões de arquivo "rxx" podem ser criados usando RAR para Mac e Unix ou WinRAR para Windows.

NOTA: O instalador de software WinRAR registra associações de arquivos para as extensões de arquivos .R00 - R29, mas não registra associações para R30 e além. Portanto, você pode ver um ícone WinRAR para arquivos R00 - R29, mas não para arquivos R30, mesmo que o WinRAR ainda possa abrir arquivos R30 que fazem parte de um arquivo dividido.
