O que é um arquivo .PLN? (Adobe InDesign Plug-in)

Arquivo plug-in usado por versões mais antigas do software de layout de página profissional InDesign; adiciona recursos extras ao software, como novos recursos de estilo e edição; usados para plugins de terceiros.

O que é um arquivo .PLN? (Arquivo de projeto ArchiCAD)

Arquivo de projeto solo que contém um modelo 3D criado com software de design arquitetônico ArchiCAD; usado por arquitetos para projetar estruturas de construção, interiores, exteriores e paisagens; usa a tecnologia Archicad Virtual Building para simular a forma como um edifício real é construído.

Desenvolvedor Microsoft
Categoria Jogos
Formato XML 

O que é um arquivo .PLN? (Plano de voo simulador de voo)

Arquivo de configuração formatado em XML que armazena um plano de voo para o Simulador de Voo da Microsoft; especifica parâmetros para o Flight Planner, incluindo localização de partida, destino, tipo de rota, waypoints, altitude de cruzeiro e outras informações.

Mais Informações

Os arquivos PLN são usados pela versão 2000 do Flight Simulator e posteriores.

Programas que abrem o arquivo .PLN (Plano de voo simulador de voo)

  Windows
  Microsoft Flight Simulator X Steam Edition

Categoria Dados
Formato Não Disponível 

O que é um arquivo .PLN? (Arquivo Planograma do Astronauta)

Planogram criado com o software ACNielsen Spaceman; usado para aplicações de merchandising, como layouts de lojas de varejo; utilizado por varejistas e fornecedores para planejar e analisar a colocação de produtos.

Mais Informações

ARQUIVOS PLN podem ser criados com o Spaceman Stock Planner ou o Spaceman Merchandiser. Eles são salvos em um formato baseado em texto que pode ser visualizado com o Spaceman Viewer.

Programas que abrem o arquivo .PLN (Arquivo Planograma do Astronauta)
