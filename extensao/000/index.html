O que é um arquivo .000? (Arquivo compactado DoubleSpace)

Arquivo de arquivo compactado pelo Microsoft DoubleSpace, um utilitário de compactação de arquivos mais antigo incluído no MS-DOS; pode ser descomprimido e aberto em tempo real usando o MS-DOS com o subsistema de compactação DBLSPACEBIN instalado.

O que é um arquivo .000? (Arquivo de dados do serviço de indexação)

Um arquivo 000 é um arquivo de dados criado pelo Indexing Service, um componente de software do sistema operacional Windows que cria um índice de sistema de arquivos pesquisável a partir de arquivos no disco rígido. Os dados armazenados em um arquivo 000 ajudam a Windows realizar pesquisas rápidas.

Mais Informações

000 arquivos são salvos com vários outros tipos de arquivos que compõem o índice de pesquisa. Outros tipos podem incluir CPU, você deve considerar desativar a indexação ou desativar seletivamente a indexação no computador. Para fazer isso, clique em Iniciar ou na barra de pesquisa em Windows e digitar "indexando" no campo de pesquisa. Em seguida, clique em Opções de indexação e remova os locais em Windows que você não deseja mais ser indexado. Isso aumentará o tempo que leva Windows para pesquisar nesses locais, mas também melhorará o desempenho da CPU.

O que é um arquivo .000? (Arquivo de imagem de disco de CD virtual)

Arquivo de dados de imagem de disco criado pelo Virtual CD, um programa de imagem de disco; criado pelas versões 4 e posteriores do software; contém os dados do disco e é salvo ao lado de um arquivo .VC4; também pode ser salvo com um arquivo .VCX, que é o projeto do Editor de CD que foi usado para criar os arquivos 000 e VC4.

Mais Informações

000 arquivos podem ser gravados em um disco usando CD virtual. No entanto, você precisa abrir o arquivo VC4 correspondente (ou arquivo VCX se ele existir), o que permite que o software virtual de CD acesse a imagem do disco.

O que é um arquivo .000? (arquivo de dados de instalação Windows CE)

Índice de arquivos de gabinete (.CAB) incluídos em um pacote de instalação para um dispositivo ce Windows; referenciado pelo script instalador do programa e define os arquivos a serem instalados.

Mais Informações

Os arquivos de índice de gabinete também incluem o nome e o editor do aplicativo, o caminho do diretório designado para instalação, os requisitos de software WinCE, a arquitetura do gabinete e outras informações.
