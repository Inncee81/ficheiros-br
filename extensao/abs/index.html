O que é um arquivo .abs? (Jogo salvo de AB SuDoKu)

Arquivo de jogo salvo criado por AB SuDoKu, um jogo de quebra-cabeça Sudoku para Windows que foi descontinuado; salva o estado de um quebra-cabeça Sudoku (só suporta um tamanho de grade 24x24) e permite que o jogo seja continuado mais tarde.

O que é um arquivo .abs? (Arquivo de banco de dados absoluto)

Um arquivo ABS é um arquivo de banco de dados criado pelo Absolute Database, um mecanismo de banco de dados para o IDE de desenvolvimento de software Delphi. Armazena dados em formato estruturado e relacional e suporta criptografia e desempenho na memória. Os arquivos ABS são usados por desenvolvedores de software para armazenar dados com seus aplicativos.

Mais Informações

Você pode criar um banco de dados ABS usando o utilitário DBManager incluído no conjunto de instalação Absolute DB, que está localizado no seguinte diretório:

\AbsoluteDatabase\Ultils\Bin\

Você também pode criar um banco de dados ABS a partir do seu aplicativo usando o método CreateDatabase do componente TABSDatabase.

NOTA: O banco de dados absoluto é usado como alternativa ao Borland Database Engine (BDE).

O que é um arquivo .abs? (Arquivo de pele do navegador Avant)

Formato de pele mais antigo usado pelo Avant Browser, um navegador de Web personalizável; especifica a aparência da interface de usuário do navegador e inclui gráficos personalizados; substituído pelo formato .SKN nas versões mais recentes do navegador.

O que é um arquivo .abs? (Script de compacta automática TurboZIP)

Arquivo criado pelo TurboZIP, um utilitário de compactação e descompressão de arquivos; salva todos os critérios especificados pelo usuário para comprimir um grupo de arquivos e pode ser reutilizado para comprimir um grupo diferente de arquivos com as mesmas opções.

Mais Informações

Para criar um Script de compactação automática, escolha File → Auto Compress Script → New no menu de aplicativos TurboZIP.
