O que é um arquivo .pk? (Arquivo de pico de audição)

Contém a representação visual de uma forma de onda de áudio para um arquivo de áudio correspondente; gerado automaticamente por Audition e Cool Edit Pro quando um arquivo de áudio é aberto; salvo na mesma pasta do arquivo de áudio original.

Mais Informações

Por padrão, os arquivos de pico são gerados para todos os arquivos de áudio abertos na Audição 20. Isso acelera o processo de exibição de forma de onda, uma vez que a forma de onda só precisa ser gerada pela primeira vez. Você pode desativar a geração automática de arquivos de pico escolhendo "Edit→Preferences→Display" Em seguida, desmarcando "Save Peak Cache Files"

A Adobe adquiriu a Syntrillium em maio de 2003 e renomeou o programa Cool Edit Pro como Adobe Audition.

O que é um arquivo .pk? (Arquivo METAFONT embalado)

Arquivo de fonte comprimido criado pelo GFtoPK, um programa incluído com algumas distribuições de software TeX; contém a cópia compactada de um arquivo de fonte bitmap metafont .GF e é usado para manter tamanhos de fonte menores.
