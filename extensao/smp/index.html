O que é um arquivo .smp? (Arquivo de paleta IDRISI)

Arquivo de paleta de cores criado por IDRISI Taiga, um aplicativo usado para imagem GIS; armazena informações de cores para uma imagem rasterizada em um arquivo .RST; age como uma lenda para as cores exibidas quando a imagem é visualizada.

O que é um arquivo .smp? (Arquivo da galeria PhotoImpact)

Um arquivo SMP é um arquivo de galeria criado por Ulead PhotoImpact (agora Corel PhotoImpact), um programa usado para editar fotos e arte digital. Ele contém uma galeria, que pode incluir imagens, objetos, melhorias de imagem ou efeitos de texto/animação. Os arquivos SMP também incluem propriedades de galeria, como o número de grupos de guias, número de miniaturas e tamanho do arquivo.

Mais Informações

As galerias permitem que você armazene diferentes efeitos, melhorias ou imagens e, em seguida, aplique-os à sua obra de arte. O arquivo SMP é criado quando você seleciona Create do Gallery Manager em "EasyPalette", nomeie o arquivo, escolha o local de salvamento e clique em OK. Você também pode criar um arquivo SMP selecionando Export. Você pode carregar um arquivo SMP Gallery selecionando Load... no Gallery Manager.

O que é um arquivo .smp? (Arquivo ScrapeMap)

Um arquivo SMP contém um ScrapeMap criado pela skyLyn Applications ScrapeMate, um programa usado para raspar informações de sites e organizá-los em um banco de dados ou planilha. Armazena informações sobre os dados que estão sendo direcionados e como os dados devem ser organizados e armazenados após serem raspados.

Mais Informações

Você pode abrir um arquivo ScrapeMap SMP no ScrapeMate usando o painel DoCorte Explorer. Clique no ícone da pasta "Abra o ScrapeMap" e navegue até a localização do arquivo SMP.

Você pode criar um ScrapeMap em ScrapeMate usando o painel DoCoreMap Explorer. Primeiro, você escolhe o site para raspar, filtrar e classificar os dados que deseja segmentar, clicar no ícone "Criar ScrapeMap" e escolher o local de salvar do arquivo. O ScrapeMap é salvo como um arquivo SMP.

Você também será solicitado a escolher o formato de saída dos dados raspados do site. Você pode optar por exportar os dados para "Excel" (arquivo .XLSX), "CSV" (.CSV arquivo) ou "SQL" (.SQL arquivo).

O que é um arquivo .smp? (Arquivo de desempenho do SmartMusic)

Arquivo de música usado pelo SmartMusic, um aplicativo de prática musical para banda, orquestra e voz; salva um acompanhamento que pode ser usado para a prática de solos e outras partes musicais; inclui as informações de partituras e pode ser usado para avaliação; substituído pelo .SMPX no Finale 2014.

Mais Informações

NOTA: Os arquivos SMP podem ser exportados do software de notação musical Finale.

O que é um arquivo .smp? (Arquivo de dados do projeto SmilePlant)

Um arquivo SMP é um arquivo de dados criado pelo SmilePlant, um CMMS (Computerized Maintenance Management System, sistema de gerenciamento de manutenção computadorizada) que ajuda os administradores da planta com supervisão, gerenciamento e manutenção. Armazena informações sobre equipamentos de fábrica, funcionários e áreas de produção e armazenamento. Os arquivos SMP são salvos como um de um conjunto de arquivos na pasta de projeto SmilePlant.

Mais Informações

A SmilePlant foi descontinuada e não está mais mantida.
