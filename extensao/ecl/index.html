O que é um arquivo .ecl? (Lista de e-campanha de e-campanha de correio de massa de gerenciamento de ware)

Arquivo XML criado pelo Management-Ware Mass Mailing News, um aplicativo de e-mail em massa; contém uma lista de contatos por e-mail, bem como o status de cada contato; pode rastrear endereços de e-mail, nomes completos, títulos, números de telefone, a data adicionada e outras informações; usadas para salvar e carregar listas de contatos, que são referenciadas pelo aplicativo ao enviar e-mails em massa.

Mais Informações

Management-Ware Mass Mailing News também fornece ferramentas para a construção de listas de assinantes e para gerenciar as relações com os clientes por meio de campanhas de e-mail marketing. O software inclui um editor visual para criar mensagens personalizadas, como boletins informativos personalizados.

O que é um arquivo .ecl? (Roteiro de Palco de Touhou)

Arquivo de jogo usado por jogos de Touhou como Double Spoiler e Ten Desires; contém um script de palco, que fornece a localização de inimigos e a forma de "padrões danmaku"," que são os padrões de balas inimigas filmadas na tela (por exemplo, espirais, círculos, etc).

Mais Informações

Os jogos de Touhou são jogos de ação de rolagem vertical desenvolvidos pelo Time Shanghai Alice.

NOTA: Os scripts da ECL podem ser descompilados usando o Touhou Toolkit, um utilitário de linha de comando.
