O que é um arquivo .eop? (Arquivo de partituras de piano de todos)

Arquivo criado por Everyone Piano, um programa de ensino de piano; contém uma partitura musical pré-gravada ou gravada pelo usuário.

Mais Informações

Para criar o arquivo EOP:

  1. Regissão uma pontuação selecionando File → Record ou clicando no ícone de registro de círculo.
  2. Assim que terminar a gravação, selecione File → Save... ou clique no ícone do disquete.
  3. Escolha o local de salvamento, nomeie seu arquivo e selecione Salve.

Você pode abrir o arquivo EOP de 3 maneiras diferentes:

  1. Basta clicar duas vezes no arquivo.
  2. Clique no ícone da pasta, selecione seu arquivo e clique em Abra.
  3. Selecione File → Open..., escolha seu arquivo e clique em Abra.

Os arquivos EOP pré-gravados estão localizados no seguinte diretório:
C:\Program Files (x86)\EveryonePiano\Music
