O que é um arquivo .ged? (Arquivo de projeto do editor de jogos)

Projeto de desenvolvimento de jogos criado com o Game Editor, um programa de criação de jogos de plataforma cruzada; inclui dados e configurações do jogo; salvo junto com uma pasta de jogo que contém os recursos do jogo.

Mais Informações

Game Editor pode ser usado para desenvolver jogos para Windows 95, 98, Me, NT, 2000, 2003, XP e Vista, bem como Pocket PC, Linux e Windows sistemas operacionais baseados em dispositivos móveis.

O que é um arquivo .ged? (Arquivo de dados de genealogia GEDCOM)

Um arquivo GED é um arquivo de dados que armazena o histórico familiar e os dados de eventos genealógicos no formato padrão de genealogia GEDCOM. Ele contém dados em um formato de texto ASCII que pode ser aberto por diferentes programas de genealogia em vários sistemas de computador.

Mais Informações

GEDCOM é uma abreviação para "GEnealogical Data COMmunications"
