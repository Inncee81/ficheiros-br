O que é um arquivo .procreate? (Arquivo de arte procriar)

Um arquivo PROCREATE contém uma tela criada por Savage Procreate ou Savage Procreate Pocket, que são aplicativos usados para criar ilustrações e animações. É o formato nativo usado para salvar obras de arte com aplicativos Procreate e pode incluir camadas de imagem, camadas de ajuste, máscaras de camada, máscaras de recorte, informações de arquivos, palavras-chave e outros elementos.

Mais Informações

Você provavelmente só encontrará arquivos PROCREATE se usar o Procreate no seu iPad ou Procreate Pocket no seu iPhone para criar obras de arte. Você também pode receber um arquivo PROCREATE de um usuário do Procreate tentando compartilhar sua obra de arte.

O arquivo é salvo em um formato proprietário que só é suportado pelo Procreate e pelo Procreate Pocket. Você também pode visualizar, mas não editar arquivos PROCREATE com o plugin ProcreateViewer (desenvolvido por Jarom Vogel) para Apple Finder no macOS.

Se você não tiver acesso ao Procreate, Procreate Pocket ou ao plugin ProcreateViewer para Finder, você não poderá abrir o arquivo. Se esse for o caso, você pode pedir ao remetente para exportar a obra de arte com Procreate para um formato diferente, como .PSD, .PNG, .JPG, .GIF, .TIFF, .PDF ou .MP4 (se for uma animação).
