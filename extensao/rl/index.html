O que é um arquivo .rl? (Arquivo da lista Rviewer)

Arquivo criado pelo Rviewer, uma ferramenta usada para criar listas de arquivos e pastas em um disco rígido Windows; salva a lista de arquivos, bem como uma identificação de arquivos danificados e restaurados; permite que os usuários rastreiem arquivos corrompidos para reparo.

Mais Informações

Os dados em arquivos RL podem ser exportados para os formatos .TXT, .HTML e .CSV.

NOTA: A RLAB é uma empresa russa e a interface de usuário Rviewer está escrita em russo.

O que é um arquivo .rl? (O 7º Arquivo de Dados de Áudio do Convidado)

Arquivo usado pelo The 7th Guest, um jogo de quebra-cabeça interativo 3D mais antigo desenvolvido pela Trilobyte usando o motor Groovie; especifica o formato de dados de áudio armazenados em um arquivo .GJD correspondente; usado para reprodução de clipe de áudio do jogo.

Mais Informações

NOTA: Os arquivos RL também são usados pela 11ª Hora e podem ser extracidos usando o GROOVEX2, programa incluído no The 11th Hour Toolset.
