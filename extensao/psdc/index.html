O que é um arquivo .psdc? (Documento de Nuvem do Adobe Photoshop)

Um arquivo PSDC é um documento do Adobe Photoshop salvo no formato Adobe Photoshop Document Cloud (PSDC). Ele contém uma imagem no mesmo formato de um arquivo .PSD, mas é armazenado na nuvem usando através do sistema de documentos em nuvem da Adobe. O arquivo é atualizado automaticamente quando é aberto e modificado em diferentes dispositivos.

O formato PSDC foi introduzido em novembro de 2019 com o lançamento do Photoshop para iPad e Photoshop CC 2020 para Windows e macOS. O objetivo do formato é permitir que os usuários trabalhem no mesmo documento PSD em seu computador ou iPad e tenham suas modificações sincronizadas para que o documento esteja sempre atualizado.

O formato de arquivo PSDC é o mesmo do formato PSD, que inclui suporte para camadas de imagem, camadas de ajuste, máscaras de camada, anotações e outros elementos específicos do Photoshop. No entanto, o formato PSDC também suporta o cache de edições feitas em um documento quando um usuário está trabalhando nele offline. Em seguida, quando o usuário se conecta à Internet novamente, as alterações são sincronizadas.

Para criar um arquivo PSDC no Adobe Photoshop, selecione File → Save ou Save As.... Se você for solicitado a salvar o documento em seu computador, clique no botão Salvar documentos na nuvem, nomear o arquivo PSDC e, em seguida, clicar em Salvar. Se você for solicitado a salvar o documento no seu espaço de armazenamento de documentos em Nuvem, basta nomear o arquivo PSDC e, em seguida, clicar em Save.

💡 Se você já criou um arquivo PSD, então carregue-o do seu computador para o espaço de armazenamento Adobe Cloud, o arquivo PSD é automaticamente salvo como um arquivo PSDC. Por exemplo, um documento do Photoshop chamado example.psd torna-se example.psdc
