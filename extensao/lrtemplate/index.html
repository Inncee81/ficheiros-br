O que é um arquivo .lrtemplate? (Modelo adobe lightroom)

Um arquivo LRTEMPLATE é um arquivo de configurações usado pelo Adobe Lightroom, um aplicativo de edição e organização de fotos digitais. Ele é salvo em texto simples e contém predefinições usadas para visualizar e alterar fotos digitais. Os arquivos LRTEMPLATE incluem configurações para cor, balanço de branco, tom e efeitos, bem como outras propriedades de edição.

Mais Informações

Você provavelmente só encontrará um arquivo LRTEMPLATE se usar o Adobe Lightroom para editar fotos. Os arquivos podem ser criados pelo usuário, mas também vêm embalados com o aplicativo. Os arquivos LRTEMPLATE são criados quando um usuário salva configurações de edição de imagens para reumuí-los com outras fotos.

Você pode encontrar arquivos LRTEMPLATE nos seguintes locais:

  • Windows - C:\Users\[nome do usuário]\AppData\Roaming\Adobe\Lightroom\[Pasta de predefinições ou modelos]
  • macOS - /Users/[user name]/Library/Application Support/Adobe/Lightroom/[Preset or Template folder]

Os arquivos LRTEMPLATE só são suportados pelo Lightroom 6 e anteriores e lightroom Classic antes da versão 73. Para abrir os arquivos LRTEMPLATE nestas versões do Lightroom siga estas instruções:

  • Windows - Selecione Edit → Preferences..., clique na guia "Presets", clique em "Mostrar presets folder", copie o arquivo LRTEMPLATE na pasta e clique no botão "Restart Lightroom"
  • macOS - Selecione Lightroom → Preferences..., clique na guia "Presets", clique em "Mostrar pasta de predefinições de sala de luz", copie o arquivo LRTEMPLATE na pasta e clique no botão "Restart Lightroom".

NOTA: Os arquivos LRTEMPLATE foram substituídos por arquivos .XMP com o lançamento do Lightroom Classic CC v73 em abril de 2018. O formato XMP permitiu que os modelos fossem compatíveis com outros programas da Adobe e sincronizados via Adobe Cloud.
