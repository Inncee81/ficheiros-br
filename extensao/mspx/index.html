O que é um arquivo .mspx? (Página de Web do Microsoft ASPNET)

Web página desenvolvida usando a estrutura de apresentação da Microsoft para ASPNET; processada por um servidor Web microsoft Internet Information Services (Microsoft Internet Information Services) Web; gera dinamicamente conteúdo que é passado para o navegador Web do cliente.

Mais Informações

AS páginas de Web MSPX são semelhantes às páginas .ASPX, mas usam "mspx" como uma extensão alternativa.

Programas que abrem o arquivo .mspx (Página de Web do Microsoft ASPNET)

  Mac
  Abre com o mesmo programa que o criou.
  Windows
  Run with a Microsoft IIS Web server
