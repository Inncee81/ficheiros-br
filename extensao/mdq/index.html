O que é um arquivo .mdq? (Arquivo de definição do banco de dados do extensor de transformação)

Um arquivo MDQ é um arquivo de dados criado pelo Database Interface Designer, que é uma ferramenta incluída no IBM Transformation Extender. Contém definições para um ou mais bancos de dados, procedimentos armazenados, consultas e outras especificações. Os arquivos MDQ são usados para executar mapas do Extensor de Transformação, que são representações visuais de bancos de dados que incluem como os objetos devem ser transformados do formato de origem para o formato de destino.

Mais Informações

Para abrir um arquivo MDQ no Database Interface Designer, selecione File → Open. Para criar um arquivo MDQ no Database Interface Designer, selecione File → Save.

O IBM Transformation Extender é um programa que permite aos usuários integrar transações de clientes, parceiros de negócios e fornecedores em uma empresa. O Database Interface Designer é uma ferramenta incluída no Transformation Extender e usada para importar metadados sobre procedimentos, tabelas e consultas armazenados a partir de um banco de dados relacional.
