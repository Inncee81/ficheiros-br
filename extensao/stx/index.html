O que é um arquivo .stx? (Arquivo do projeto Do Pinnacle Studio)

Arquivo de projeto criado pelo software de edição de vídeo do Pinnacle Studio; contém configurações de projeto de vídeo e referências a mídia e outros arquivos de dados usados pelo projeto.

Mais Informações

O arquivo STX é normalmente salvo no mesmo diretório da pasta que contém os dados do projeto.

O que é um arquivo .stx? (Arquivo do kit de interface de música scream tracker)

Formato de arquivo do módulo de rastreador de áudio publicado pela Future Crew como um formato de transição entre os formatos Scream Tracker 2 (.STM) e Scream Tracker 3 (.S3M).
