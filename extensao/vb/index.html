O que é um arquivo .vb? (Arquivo de áudio de auto roubo de grande parte)

Arquivo de áudio usado por Grand Theft Auto (GTA) 3 e GTA: Vice City jogos para PlayStation 2; armazena vários clipes de áudio e inclui clipes de estação de rádio, clipes de notícias de rádio e áudio cutscene; usado para reprodução de áudio durante a jogabilidade.

Mais Informações

Os arquivos VB podem ser convertidos para o formato .WAV com o MFAudio definindo as seguintes opções de programa:

  • Frequência: 32000Hz ou 16000Hz para as estações de conversa (por exemplo, POLICE.VB, KCHAT.VB, e VCPR.VB)
  • Amostras: 16 bits
  • Canais: 2
  • Interleave: 2000 Bytes
  • >Offset: 0 Bytes
  • Formato de saída: WAVE
VB arquivos são salvos no diretório de \AUDIO\ no disco GTA do PlayStation 2.

NOTA: Arquivos de áudio VB também podem ser usados pelas versões Windows de GTA3 e GTA: Vice City com o plugin VBDec instalado.

O que é um arquivo .vb? (Arquivo ROM do garoto virtual)

Game ROM criado a partir de um videogame Nintendo Virtual Boy; contém dados de um cartucho Virtual Boy salvo em um arquivo somente leitura; pode ser jogado em um PC usando um programa de emulação do Virtual Boy, como o Red Dragon.

Mais Informações

Virtual Boy foi um console portátil desenvolvido pela Nintendo e lançado em 1995. Ele incluiu um fone de ouvido que proporcionou uma experiência 3D para os jogadores.

O que é um arquivo .vb? (Arquivo de item do projeto básico visual)

Um arquivo VB é um arquivo de item de projeto escrito na linguagem Visual Basic, uma linguagem de programação orientada a objetos criada pela Microsoft para uso com o NET Framework. Contém instruções Visuais Básicas em formato de texto ASCII.

Mais Informações

Um item do projeto pode se referir a uma variedade de componentes do projeto Visual Basic. Cada item do projeto disponível difere com base em se você está trabalhando em um projeto local ou web. Exemplos de itens do projeto incluem um formulário de Windows, formulário de login, tela de respingo, diálogo, classe, interface, arquivo de código, controle personalizado, controle personalizado da Web ou classe de instalador. A extensão "vb" é usada para todos esses itens.

Visual Basic é uma linguagem de programação implementada no framework NET. Visual Basic NET (VBNET) foi lançado em 2002 para substituir a linguagem Visual Basic original. O idioma VBNET é usado para criar aplicativos baseados no NET, como aplicativos web e serviços web XML. "NET" foi eventualmente descartado pela Microsoft em 2005 e o arquivo VB agora é usado pelo Microsoft Visual Studio.

NOTA: Arquivos VB também podem ser usados para conter vírus ou outros malwares. Portanto, não abra arquivos VB se eles forem um anexo de e-mail desconhecido.
