O que é um arquivo .ogm? (Arquivo de mídia Ogg)

Arquivo de vídeo compactado que usa uma forma alterada do formato de contêiner de arquivos Ogg do XiphOrg; suporta a reprodução de conteúdo de vídeo que incorpora filtros DirectShow; criado como uma extensão para a especificação Ogg para que Windows Media Player (versões mais antigas) e outros programas possam facilmente reproduzir arquivos de vídeo e áudio Ogg.

Mais Informações

NOTA: A XiphOrg, proprietária e desenvolvedora da especificação Ogg, não suporta o formato OGM porque está fora da especificação Ogg. O formato OGM é diferente do formato usado para .OGG e arquivos .OGV.

O que é um arquivo .ogm? (Arquivo matriz de origem)

Um arquivo OGM contém um livro de matriz criado pela Origin, uma aplicação científica e de análise de dados de engenharia e grafia. Ele armazena uma ou mais folhas de matriz, que incluem colunas e linhas de números, caracteres especiais ou espaços. Os arquivos OGM também armazenam outras informações sobre a matriz, como os nomes das folhas, funções matemáticas, dimensões de matriz, estilo e formatação e tipo de dados.

Mais Informações

Você pode abrir arquivos OGM com Origin e Origin Viewer. O Origin só está disponível para Windows e é o programa principal usado para criar, abrir e editar arquivos OGM. Quando aberta na Origem, a matriz aparece como uma janela em um projeto aberto (.OPJ ou .OPJU arquivo). O Origin Viewer também é comumente usado para visualizar arquivos OGM, uma vez que é gratuito e pode ser usado em Windows e macOS.

Para abrir um arquivo OGM no Origin, selecione File → Open..., selecione o tipo de arquivo "Origin Window ogm", navegue até o local do arquivo OGM e clique em Abra.

Você pode salvar matrizes origin como modelos, a fim de preservar configurações a serem usadas com outras matrizes que você cria. Para fazer isso, selecione File → Save Template As... e a Origin criará um arquivo .OTM para armazenar o modelo.

NOTA: O lançamento do Origin 2018 incluiu um formato de unicode atualizado para salvar matrizes (.OGMU), que apresenta vantagens de desempenho em relação ao formato de arquivo OGM mais antigo e não-unicode.
