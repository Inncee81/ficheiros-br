O que é um arquivo .ns1? (Nestopia Save State)

Save state criado e usado pela Nestopia, um emulador do Nintendo Entertainment System (NES); armazena um estado de jogo salvo que permite que você retome seu jogo a partir do ponto de salvamento em um momento posterior.

Mais Informações

Nestopia fornece 9 slots de salvamento rápido que usam as extensões "ns1" para "ns9" para salvar estados de um jogo. O estado de salvamento terá o mesmo nome do arquivo ROM .NES de onde foi criado. Se o arquivo NES for nomeado game.nes, então o arquivo NS1 será nomeado game.ns1.
