O que é um arquivo .vob? (Arquivo objeto de vídeo DVD)

Um arquivo VOB é um arquivo de dados de filme de um disco de DVD, normalmente armazenado na pasta VIDEO_TS na raiz do DVD. Ele contém a maioria dos dados armazenados no disco que inclui vídeo, áudio, legendas, menus de DVD e outros conteúdos de navegação. Os arquivos VOB geralmente são formatados como fluxos de sistema MPEG-2 e podem ser abertos por vários programas de reprodução de vídeo.

Mais Informações

Como a maioria dos DVDs comerciais usa algum tipo de proteção de direitos autorais, o arquivo VOB extraído do DVD pode ser criptografado, o que pode resultar em um erro ao tentar copiar o arquivo para outro local. Existem uma série de programas de descriptografia disponíveis para reproduzir o arquivo VOB sem as credenciais adequadas, no entanto, os programas podem ser ilegais em certos países.

O que é um arquivo .vob? (Arquivo de carro ao vivo para velocidade)

Arquivo usado pelo Live for Speed (LFS), um jogo de corrida de carros; armazena dados de carros 3D, como texturas de carros, rostos, malhas e materiais; usado para personalizar a exibição de corpos e peças de carros de corrida 3D.

Mais Informações

Os arquivos VOB contêm apenas metade dos dados de exibição do carro de corrida, uma vez que os carros em LFS são simétricos. A outra metade do carro é espelhada dentro do jogo.

Desenvolvedor E-on Software
Categoria Imagens 3D
Formato Binário 

O que é um arquivo .vob? (Arquivo objetos vue)

3D arquivo modelo criado pelo E-on Vue, um aplicativo de modelagem de ambiente 3D; contém objetos para o ambiente, que são armazenados como malhas poligonais; podem ser texturizados usando dados armazenados em arquivos E-on Vue .MAT.

Mais Informações

Arquivos VOB são usados para armazenar objetos como árvores, montanhas, oceanos, céus e outras paisagens.

NOTA: O E-on Vue está disponível em diversas edições diferentes para profissionais, artistas e entusiastas.
