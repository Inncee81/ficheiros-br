O que é um arquivo .j? (Arquivo JAR)

Arquivo de arquivo compactado criado com JAR, o sucessor do programa de compactação de arquivos ARJ (que criou .ARJ arquivos); Jar está disponível em uma versão dos de 16 bits e uma versão de Windows de 32 bits.

Mais Informações

Não está relacionado ao formato .JAR, que é um Arquivo Java.

O que é um arquivo .j? (Arquivo de origem Java)

Contém código-fonte do programa escrito na linguagem de programação Java; pode incluir o código-fonte de um programa completo ou pode ser um dos muitos arquivos referenciados por um projeto de programação Java.

Mais Informações

Os arquivos de origem Java geralmente têm uma extensão .JAVA.

O que é um arquivo .j? (Imagem JPEG)

Gráfico comprimido que usa .JPEG compressão de imagem; pode conter milhares de cores, o que o torna um bom formato para armazenar fotos digitais; pode usar diferentes níveis de compressão, mas quanto mais compressão for usada, menor será a qualidade da imagem resultante.

Mais Informações

A maioria dos arquivos JPEG tem uma extensão .JPG.
