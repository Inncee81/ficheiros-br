O que é um arquivo .OPF? (Arquivo FlipAlbum)

Álbum de fotos digital criado pela FlipAlbum, um programa que organiza imagens em álbuns de imagens; armazena a estrutura do álbum, configurações de aparência e referências a arquivos de imagem; pode ser visto com efeitos de inversão de página 3D.

Mais Informações

Os álbuns de fotos da FlipAlbum são comumente usados para compartilhar fotos com amigos e familiares. Fotógrafos profissionais também as usavam para organizar fotos para clientes.

Os arquivos OPF são salvos no formato FlipAlbum XML proprietário da E-Book Systems e não devem ser confundidos com o padrão Formato de Embalagem Aberta (OPF) padrão.

NOTA: A FlipAlbum foi descontinuada e não está mais disponível para download.

O que é um arquivo .OPF? (Arquivo FlipViewer FlipBook)

FlipBook, como uma revista digital ou outra publicação que pode ser lida com o FlipViewer; salvo no Formato OEB Package (OPF) que é formatado em XML; inclui metadados que descrevem o conteúdo e uma lista de páginas, imagens e texto referenciados pela publicação.

Mais Informações

Como os arquivos OPF são formatados em XML, eles podem ser criados e editados com um editor de texto básico.

O que é um arquivo .OPF? (Arquivo de formato de embalagem aberto)

Arquivo formatado de acordo com o padrão Formato de Embalagem Aberta (OPF), que faz parte da especificação do eBook do EPUB; armazena informações do eBook, como o título, idioma, ID do livro, criador, ordem de leitura e formatação de exibição; usado como parte de um metadados.

Mais Informações

Os arquivos OPF são salvos em um formato XML. O padrão Formato de Embalagem Aberta também requer um segundo arquivo XML .NCX que armazena a tabela de conteúdo para o eBook EPUB.
