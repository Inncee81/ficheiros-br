O que é um arquivo .psy? (Arquivo de canção psycle)

Arquivo criado pelo Psycle Modular Music Creation Studio, um programa de música modular usado para gerar músicas sintetizadas; armazena blocos de música "modular"" como padrões de entrada e efeitos organizados no sequenciador; usado para gerar música semelhante a MIDI.

Mais Informações

Arquivos PSY podem ser exportados para .WAV arquivos ou para .XM canções fasttracker 2.

O que é um arquivo .psy? (O arquivo de configurações do protetor de tela psicodélica)

Arquivo de configurações usado pelo The Psychedelic Screen Saver, um programa usado para exibir protetores de tela coloridos; salvo em texto simples e contém propriedades para cores, formas, rotações e outros efeitos para o protetor de tela; usado para gerar padrões visuais.

Mais Informações

Os parâmetros salvos nos arquivos PSY são todos carregados no mesmo arquivo protetor de tela psicodélica, que tem o nome de arquivo PSYCH.SCR. Os valores simplesmente alteram as propriedades do protetor de tela para que ele seja exibido de forma diferente para diferentes parâmetros.
