O que é um arquivo .cpt? (Arquivo criptografado Ccrypt)

Arquivo criptografado com ccrypt, um utilitário para criptografar e descriptografar arquivos e fluxos; com base na cifra Rijndael e usa um algoritmo de criptografia assimétrica; requer uma senha para abrir.

Mais Informações

O Ccrypt é desenvolvido como um programa de código aberto e está disponível para Mac, Windows e sistemas Unix.

O que é um arquivo .cpt? (Imagem corel FOTO-PAINT)

Um arquivo CPT é uma imagem bitmap criada pelo Corel PHOTO-PAINT, um programa de edição de fotos que faz parte do CorelDRAW Graphics Suite. Ele contém uma imagem que suporta camadas, canais alfa, transparência e outros efeitos de imagem. Os arquivos CPT são usados por fotógrafos amadores e profissionais e designers gráficos para armazenar fotos sobrestilhávidas.

Mais Informações

Captura de tela de um arquivo cpt em Corel PHOTO-PAINT 2020

O arquivo CPT é o principal tipo de arquivo criado pelo PHOTO-PAINT. Ao salvar sua imagem, o PHOTO-PAINT cria o arquivo CPT para armazenar a imagem e todas as edições que foram aplicadas. Os arquivos CPT são semelhantes aos arquivos .PSD do Adobe Photoshop mais populares desta forma. Você pode salvar um arquivo CPT selecionando File → Save.

O formato de arquivo CPT não é suportado por muitos editores de imagens. No entanto, você pode exportar o arquivo em PHOTO-PAINT para formatos de imagem mais amplamente suportados, como .JPG, .PNG, .GIF e .BMP. Para exportar o arquivo CPT selecione File → Export... e escolha o formato.

Além de Corel PHOTO-PAINT, as seguintes aplicações estão incluídas no CorelDRAW Graphics Suite 2020:

  • CorelDRAW - Um programa de ilustração vetorial e layout de página
  • Corel Font Manager - Um programa de gerenciamento de fontes
  • CorelDRAWapp - Um programa de ilustração de vetores online acessível via navegador da Web
  • Captura Corel - Um utilitário de captura de tela de um clique único (disponível apenas com Windows versão)

O que é um arquivo .cpt? (Arquivo de áudio compacto DTS)

Arquivo de áudio usado por programas de autoria de DVD; contém dados de áudio, timecode e cabeçalho; usado em conjunto com dados de vídeo para criar DVDs.

Mais Informações

NOTA: Os arquivos CPT não podem ser usados para criar discos Blu-ray. O formato .DTSHD pode ser usado em vez disso.
