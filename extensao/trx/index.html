O que é um arquivo .trx? (Arquivo de imagem de firmware)

Imagem de firmware usada por vários dispositivos de computador incorporados, incluindo a série roteador Linksys WT54G; contém o conteúdo da imagem do firmware, bem como um cabeçalho que fornece o comprimento do arquivo, um cheque e os deslocamentos de partição dentro do arquivo; usado para recuperar ou atualizar o firmware em um dispositivo.

Mais Informações

TRX também podem ser fornecidos em formato .BIN, que normalmente é uma versão específica do fornecedor de um arquivo TRX.

O que é um arquivo .trx? (Arquivo da lista de tradução PASSOLO)

Lista de tradução criada pela PASSOLO, um programa usado para tradução de idiomas de interfaces de usuário; contém uma coleção de strings em vários idiomas; usado dentro de um programa específico para reconhecer frases-chave e traduzi-las de um idioma para outro.

Mais Informações

NOTA: Uma vez que os arquivos TRX tenham sido exportados, eles são bloqueados e não podem ser modificados novamente em PASSOLO.

O que é um arquivo .trx? (Arquivo de resultados de teste do estúdio visual)

Arquivo de resultados de teste criado pelo Visual Studio, um aplicativo da Microsoft usado para desenvolver software Windows; contém resultados de teste armazenados em um formato XML; usado para exibir resultados de testes no Visual Studio, bem como salvar resultados históricos de casos de teste.

Mais Informações

TRX também pode ser criado usando o Microsoft MSTest, um utilitário de linha de comando que executa testes de unidade criados no Visual Studio 2005 e posteriormente. O arquivo executável do programa MSTest.exe tem várias opções de linha de comando que podem ser usadas para personalizar a saída de testes.

NOTA: Os resultados dos testes permanecem na memória no Visual Studio até que sejam explicitamente escritos em disco em um arquivo TRX. Eles podem ser salvos usando a opção "Export Test Run Results" no Visual Studio.

O que é um arquivo .trx? (Arquivo em lote do WebAuthorize)

Arquivo em lote criado pelo WebAuthorize, um programa que processa transações com cartão de crédito; armazena uma coleção de onze campos de transações para ID de transação, número do cartão de crédito, expiração do cartão de crédito, nome do auth do cartão de crédito, AVS, CEP, valor, ID do cliente e RG de fatura.

Mais Informações

Os arquivos TRX são usados para processar um lote de transações de cartão de crédito através de um sistema automatizado e normalmente são compostos por um dia inteiro de transações comerciais. Eles são comumente usados no e-commerce.

TRX os arquivos são nomeados pelo identificador do comerciante, um tilde, o número do lote e a extensão do arquivo "trx". Por exemplo, "merchant1~3245trx".

Tellan Software, os fabricantes do PC Authorize, o antecessor do WebAuthorize, foram comprados pela CyberCash em 1999, que foi comprada pela VeriSign e, posteriormente, payPal.
