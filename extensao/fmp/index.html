O que é um arquivo .fmp? (Arquivo do mapa de fonte autoCAD)

Arquivo de configurações de fonte usado pelo AutoCAD, um programa CAD usado para desenhar e documentar projetos 2D e 3D; armazena um mapeamento entre uma lista de fontes e seus substitutos; geralmente inclui referências a .PFB, .SHX e arquivos .TTF; aberto pelo AutoCAD quando uma fonte padrão não pode ser encontrada para localizar a fonte de substituição a ser usada.

Mais Informações

Arquivos FMP são salvos em texto simples e podem ser editados com um editor de texto. Eles armazenam substituições separadas por quebras de linha no seguinte formato:

original_font;replacement_font_file

O arquivo padrão do mapa de fonte AutoCAD tem o nome acad.fmp para AutoCAD e acadlt.fmp para AutoCAD LT.

O que é um arquivo .fmp? (Banco de dados FileMaker Pro)

Banco de dados criado pelo FileMaker Pro, um programa de software de banco de dados para Windows e o Mac OS; FMP é uma extensão genérica para um banco de dados FileMaker em vez de .FP4 para bancos de dados da versão 4, .FP5 para bancos de dados da versão 5, etc.

Mais Informações

FMP também é comumente usado como abreviação para "FileMaker Pro"
