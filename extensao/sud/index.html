O que é um arquivo .sud? (Arquivo de quebra-cabeça Sudoku)

Puzzle criado para o software Sudoku; contém números organizados em grades; pode ser aberto por vários jogos sudoku e programas de resolução de quebra-cabeças Sudoku; também usado por alguns applets Sudoku baseados em Java disponíveis na Web.

O que é um arquivo .sud? (Arquivo do analisador de super projeto)

Projeto salvo pelo Super Project Analyzer, um programa que analisa projetos de desenvolvimento de software; contém informações sobre arquivos, procedimentos, constantes, variáveis, tipos de dados e enums.

Mais Informações

Arquivos SUD rastreiam o uso de arquivos dentro de um projeto. O programa determina quais arquivos são necessários e quais não são. Os elementos do projeto são então rotulados como "Incluídos," "Compartilhados," "Usados," ou "Morto"
