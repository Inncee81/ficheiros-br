O que é um arquivo .wdf? (Arquivo de disco Wiimm)

Imagem de disco criada pela Wiimms WBFS Tool, um programa usado para gerenciar imagens de discos Wii; contém dados de jogo que foram extraídos de um disco Wii; usado para armazenar o tamanho reduzido de uma imagem de jogo Wii quando salvo no disco rígido de um computador.

Mais Informações

Os arquivos de disco Wiimm podem ser muito menores do que o tamanho de um arquivo de imagem de disco Wii, uma vez que eles não armazenam o espaço de preenchimento não uso, ou "buracos," que são encontrados em discos Wii. Isso pode reduzir significativamente o tamanho dos jogos Wii quando salvos em arquivos WDF.

Arquivos WDF também podem ser convertidos em arquivos do Sistema de Arquivos de Backup wii (.WBFS), que são usados para armazenar jogos Wii em dispositivos externos.

O que é um arquivo .wdf? (Arquivo da Fundação do Motorista Windows)

Arquivo de driver usado pelos componentes Windows Kernel-Mode Driver Framework (KMDF) e UMDF (User-Mode Driver Framework) e UmDF (User-Mode Driver Framework) e UmDF (User-Mode Driver Framework) e Nadf (User-Mode Driver Framework) e Uso-Mode Driver Framework (UMDF); normalmente contém um arquivo vazio que serve como marcador de versão do driver; usado pela ferramenta OCA (Online Crash Analysis, análise de acidentes online) da Microsoft para fornecer informações do driver.

Mais Informações

Arquivos WDF são tipicamente encontrados no diretório %windir%\system32\drivers. Exemplos de arquivos WDF incluem MsftWdf_user_01_09_00.Wdf, MsftWdf_Kernel_01009_Coinstaller_Critical.Wdf e Msft_Kernel_WdfRamdisk_01009.Wdf.

A Windows Driver Foundation (WDF) é um modelo conceitual de driver que inclui os componentes Windows KMDF e UMDF, bem como ferramentas de verificação de condutores.

O que é um arquivo .wdf? (Arquivo de Genealogia WinGenea)

Arquivo de árvore genealógica criado pela WinGenea, um programa usado para armazenar e imprimir genealogias; armazena até 32.000 pessoas e como eles se relacionam uns com os outros; pode conter nomes, datas, locais e fontes; usado para armazenar informações de ancestralidade familiar.

O que é um arquivo .wdf? (Arquivo de desenho Wintech)

Um arquivo WDF é um arquivo de dados salvo em um formato desenvolvido pela Wintech Engineering e usado para fabricação auxiliada por computador (CAD). Ele contém um ou mais objetos e formas salvas na ordem de serem cortados. Os arquivos WDF são usados por máquinas cericamente controladas por computador (CNC) usadas para cortar projetos de diferentes materiais.

Mais Informações

Os arquivos WDF podem ser abertos pelas versões 5 a 9 do Wintech Profiler. O tipo de arquivo WDF é usado por Profiler 5 (macOS), Profiler 6 (Windows), Profiler 7 (Windows), Profiler 8 (Windows) e o Profiler 9.

O que é um arquivo .wdf? (Arquivo de mídia Wonderland Adventures)

Arquivo de mídia usado pela Wonderland Adventures, um jogo desenvolvido pela Midnight Synergy; geralmente um arquivo gráfico ou de áudio renomeado.

Mais Informações

Arquivos WDF podem conter efeitos sonoros para o jogo ou imagens usadas para gráficos, texturas e skins modelo. Eles podem ser renomeados e abertos como um tipo de arquivo de imagem diferente (como um arquivo .BMP).

NOTA: Os arquivos WDF podem ser abertos em Windows Media Player e Windows Photo Viewer. No entanto, você pode precisar alterar a extensão do arquivo para que Windows Photo Viewer abra o arquivo.
