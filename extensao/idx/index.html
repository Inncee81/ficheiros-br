O que é um arquivo .idx? (Arquivo de índice)

Armazena um índice de arquivos usados por vários programas; usado por funções de pesquisa para acelerar as pesquisas; uma vez que um disco rígido, diretório de arquivos ou banco de dados é indexado, os dados podem ser pesquisados e classificados de forma mais eficiente.

Mais Informações

Usado por Microsoft FoxPro, Corel QuickFinder, ESRI ArcView, ICQ e outros programas.

  ESRI ArcGIS Pro
  ICQ

O que é um arquivo .idx? (Arquivo de legendas do filme)

Um arquivo IDX é um arquivo de dados usado por vários programas de reprodução de DVD para exibir legendas. Ele contém o texto da legenda, bem como pontos de sinalização para quando cada legenda deve ser exibida durante a linha do tempo do filme. Os arquivos IDX são salvos com um arquivo de vídeo correspondente, como um arquivo .AVI ou .MKV.

Mais Informações

Os arquivos IDX são semelhantes aos arquivos .SUB, que também são usados para armazenar legendas para vídeos. Um arquivo IDX é normalmente acompanhado por um arquivo SUB e o arquivo de vídeo. Eles também terão o mesmo nome de arquivo.

O que é um arquivo .idx? (Arquivo POI de navegação)

Contém pontos de interesse (POI) para um dispositivo de navegação portátil VDO Dayton GPS (PND); inclui um cabeçalho e um conjunto de POIs; cada POI consiste em um índice de byte de 3, depois 8 bytes para as coordenadas e 46 bytes para nome e tipo.

Mais Informações

Arquivos IDX podem ser transferidos para PNDs VDO Dayton via USB.

Programas que abrem o arquivo .idx (Arquivo POI de navegação)

  Windows
  Transfer to a VDO Dayton device

O que é um arquivo .idx? (Arquivo do índice Outlook Express Mailbox)

Armazena um índice de mensagens de e-mail armazenadas em uma caixa de correio do Outlook Express (arquivo .MBX); deve ser localizado na mesma pasta que os arquivos da caixa de correio; necessário para importar caixas de correio antigas do Outlook Express para o Outlook Express 5 ou posterior.

  Microsoft Outlook Express
