O que é um arquivo .fbk? (Arquivo de livro de campo 3D civil autocad)

Arquivo usado pelo AutoCAD Civil 3D, um programa BIM (Building Information Modeling) utilizado para o projeto de engenharia civil; salvo em um formato de texto simples e armazena dados de ponto observados; pode ser importado em um desenho.

Mais Informações

FBK arquivos podem ser criados a partir de arquivos de pesquisa bruto no AutoCAD Civil 3D. Os dados convertidos podem então ser importados para um desenho ou desenho.

O que é um arquivo .fbk? (Fabricante de árvore genealógica para Windows arquivo de backup)

Backup comprimido de um arquivo de árvore genealógica (.FTW) criado com o Family Tree Maker for Windows; pode ser criado como um arquivo de backup no caso de o arquivo original da árvore genealógica ser danificado ou excluído; pode ser restaurado selecionando "Arquivo→Restaurar de Backup" dentro do Family Tree Maker.

Mais Informações

Family Tree Maker for Windows usa a extensão de arquivo "fbk", enquanto o Family Tree Maker for DOS salva backups à medida que .FBC arquivos.

  Windows
  Software MacKiev Family Tree Maker

O que é um arquivo .fbk? (Arquivo de backup MICROSOFT Dynamics NAV)

Backup de um banco de dados criado com o Microsoft Dynamics NAV; contém dados financeiros usados para gerenciamento e análise de negócios; pode ser usado para restaurar dados se o arquivo principal de banco de dados (.FDB) ficar corrompido ou for excluído.

Mais Informações

O Microsoft Dynamics NAV foi anteriormente chamado de Microsoft Navision.

  Microsoft Dynamics NAV or Navision
