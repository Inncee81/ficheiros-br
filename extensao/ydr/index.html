O que é um arquivo .ydr? (Projeto de Desenho YouiDraw)

Um arquivo YDR contém um desenho criado pelo YouiDraw Drawing, um programa de desenho vetorial online. Ele é salvo em um formato proprietário que armazena informações sobre o desenho, como propriedades de camada, texto, formas, padrões, imagens e caneta, lápis e pinceladas.

Captura de tela de um arquivo ydr no Desenho YouiDraw

Você pode criar um arquivo YDR no Desenho do YouiDraw selecionando File → Download to Local Disk. O desenho é então salvo no arquivo YDR e baixado pelo navegador da Web.

Você pode abrir arquivos YDR no YouiDraw Drawing selecionando File → Open from Local Disk. Você pode salvar arquivos YDR selecionando File → Save ou Save As.

NOTA: O YouiDraw fornece três aplicativos web relacionados ao design, que incluem Desenho, Criador de Logotipo e Pintor.

O que é um arquivo .ydr? (Yu-Gi-Oh! Arquivo de repetição do Poder do Caos)

Gravação de um duelo de "Yu-Gi-Oh! Poder do Caos", que pode ser jogado de volta mais tarde; pode ser salvo por um dos três jogos power of chaos: Yugi the Destiny, Kaiba the Revenge, e Joey the Passion.
