O que é um arquivo .smm? (Ami Pro Macro)

Macro originalmente usado pelo programa de processamento de palavras Ami Pro, que mais tarde foi convertido para Lotus Word Pro; automatiza funções dentro dos programas Ami Pro e Lotus Word Pro.

O que é um arquivo .smm? (Arquivo de esquema do gerenciador de mapas)

Arquivo usado pelo Map Manager, um programa que permite que conjuntos de dados espaciais baixados do EDINA Digimap sejam convertidos em arquivos de forma; contém dados que controlam como os dados de entrada são convertidos para o arquivo de forma de saída; usado no processo de plotagem de dados da Ordnance Survey em um mapa.

Mais Informações

NOTA: O Gerente de Mapas EDINA não está mais desenvolvido.

  Windows
  EDINA Map Manager

Desenvolvedor Samsung
Categoria Dados
Formato Binário 

O que é um arquivo .smm? (Arquivo de memorando da Samsung)

Arquivo de dados criado por celulares Samsung Galaxy; salva os memorandos do usuário que foram criados no telefone; pode incorporar notas e gráficos manuscritas; pode ser sincronizado e ressarcido até o computador usando o software Samsung Kies.

Mais Informações

NOTA: Embora o Samsung Kies possa ser usado para fazer backup e restaurar arquivos SMM, ele não pode abri-los ou lê-los.
