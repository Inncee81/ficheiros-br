O que é um arquivo .PAC? (Arquivo proxy auto-config)

Um arquivo PAC é um arquivo de configuração usado por navegadores da Web para selecionar um servidor proxy apropriado. Ele contém uma ou mais funções JavaScript com uma delas chamada FindProxyForURL, que define regras para escolher um servidor proxy com base em uma determinada URL. Os arquivos PAC são usados para controlar e equilibrar o tráfego da Internet, especialmente em ambientes corporativos.

Mais Informações

Os arquivos PAC são ativados em navegadores da Web inserindo a URL do arquivo PAC nas configurações de conexão proxy ou usando o protocolo Web Proxy Auto-Discovery (WPAD). Quando executado, um arquivo PAC retorna uma sequência com um ou mais URLs de servidor proxy. Os arquivos PAC são normalmente hospedados em um servidor web e são nomeados proxy.pac.

Desde que os arquivos PAC contêm a configuração em texto simples. Você pode usar um editor de texto, como o Microsoft Notepad ou o Apple TextEdit para visualizar ou modificar os arquivos.

NOTA: O formato do arquivo PAC foi originalmente lançado em 1996 para uso pelo Netscape Navigator 20.

O que é um arquivo .PAC? (Arquivo gráfico STAD)

Formato de imagem usado pelo programa STAD para sistemas Atari ST; salvo como uma imagem em preto e branco com resolução de 640x400 pixels; considerado um gráfico de alta resolução para computadores Atari.
