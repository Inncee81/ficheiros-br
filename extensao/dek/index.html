O que é um arquivo .dek? (Arquivo de lote de bisbilhoteiro)

Um arquivo DEK é um arquivo de Comando de Processamento em Lote que contém uma lista de comandos para O Eavesdropper, um programa de processamento de dados sísmico. Ele inclui uma lista de instruções que são usadas para processar arquivos de dados sísmicos (.DAT).

Mais Informações

Um comando de arquivo DEK de exemplo é: DSEIS EDKLdek EDKLlst

Este comando executa as instruções no arquivo DEK através do programa DSEIS e produz os resultados para um arquivo de diário chamado "EDKLlst"

O que é um arquivo .dek? (Arquivo de deck de cartas LackeyCCG)

Um arquivo DEK é um arquivo criado pelo LackeyCCG, uma ferramenta usada para criar baralhos de cartas e jogar jogos de cartas colecionáveis (CCGs) on-line ou off-line. Ele armazena um pool de cartas, que é feito de cartas de jogadores. Os arquivos DEK são usados para salvar decks para diferentes jogos, bem como para diferentes estilos de jogo.

Mais Informações

LackeyCCG tem sido usado para criar decks para vários jogos como Magic: The Gathering, Pokémon TCG, Yu-Gi-Oh!, e muitos outros.

O que é um arquivo .dek? (Magic: O arquivo do convés de coleta)

Um arquivo DEK é um arquivo de baralho de cartas usado por Magic: The Gathering Online (MTGO), um jogo de cartas colecionável online. Os arquivos DEK contêm uma lista de cartas que geralmente compõem um baralho Magic: The Gathering. Em alguns casos, os arquivos DEK listam uma coleção não jogável de cartas, como um subconjunto da coleção de um jogador, em vez de um baralho jogável.

Mais Informações

Captura de tela de um arquivo dek no Átomo

Além de usar arquivos DEK com MTGO, muitos jogadores do Magic usam arquivos DEK para importar decks ou exportar decks de aplicativos e sites de rastreamento de coletas e de construção de decks, como:

  • MTG Studio
  • CardHoarder

O que é um arquivo .dek? (Memorize-It Flashcard Deck)

Um arquivo DEK é um deck de flashcards que pode incluir texto, som e imagens. É usado pela Memorize-It para criar e administrar quizzes que podem ser usados para estudar e testar conhecimentos. Os arquivos DEK estão disponíveis para centenas de tópicos diferentes.

O que é um arquivo .dek? (Arquivo de deck de desktop virtual Yugioh)

Um arquivo DEK é um arquivo usado pela Yugioh Virtual Desktop (YVD), um jogo usado para jogar jogos de cartas Yu-Gi-Oh! online. Ele é salvo em texto simples e contém uma lista de cartas no baralho separadas por quebras de linha. Os arquivos DEK são usados para salvar decks para diferentes estilos de jogabilidade.
