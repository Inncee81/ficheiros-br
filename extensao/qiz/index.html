O que é um arquivo .qiz? (Quiz do construtor de exames)

Um arquivo QIZ contém um quiz criado pela Scientific Notebook ou Scientific WorkPlace, que são processadores de texto matemáticos e científicos. Armazena fórmulas e condições que são referenciadas toda vez que o arquivo QIZ é aberto, a fim de gerar diferentes perguntas de quiz para o usuário.

Mais Informações

Os arquivos QIZ são gerados a partir de documentos .TEX usando a ferramenta Process Builder oferecida pela Scientific Notebook e Scientific WorkPlace. A ferramenta Exame Construtor permite que instrutores de matemática e ciências desenhem diferentes tipos de testes que podem ser feitos e classificados online pelos alunos.

Toda vez que um aluno abre um arquivo QIZ no Scientific Notebook ou WorkPlace, as perguntas são reorganizadas, ou seja, toda vez que um aluno abre o arquivo QIZ para fazer o teste, ele encontrará diferentes perguntas. Isso permite que os instrutores construam testes mais desafiadores e dinâmicos do que um teste estático que exibe as mesmas perguntas e respostas toda vez que é aberto.
