O que é um arquivo .pod? (Arquivo de projeto OpenProj)

Arquivo de projeto criado com o OpenProj, um programa de gerenciamento de projetos de código aberto; inclua informações de agendamento de projetos, como fases e prazos; também inclui pessoal envolvido no projeto; dados salvos em um arquivo POD podem ser visualizados visualmente usando o software OpenProj.

Mais Informações

OpenProj foi projetado como uma alternativa ao Microsoft Project e está disponível para as plataformas Windows, Macintosh e Unix.

O que é um arquivo .pod? (Arquivo Perl POD)

Arquivo de texto usado para documentar código-fonte escrito na linguagem de programação Perl; usa sintaxe básica que pode ser facilmente analisada e lida em formato de código-fonte; POD significa "Documentação Velha simples"

Mais Informações

A documentação do POD pode ser salva como um arquivo POD, mas também pode ser escrita dentro do código-fonte perl. Os arquivos POD são usados para documentar módulos Perl lançados publicamente e para o manual de ajuda perl, também chamado de Manpages Perl.

O que é um arquivo .pod? (Arquivo de dados do jogo de realidade terminal)

Formato de contêiner usado pelos jogos Terminal Reality; pode armazenar vários tipos de dados de jogos, como modelos 3D, texturas, sons, mapas e a interface do usuário.

Mais Informações

Jogos que usam arquivos POD incluem Nocturne, Monster Truck Madness, 4x4 Evolution, Fury, Fly!, Bloodrayne, entre outros. Os arquivos POD podem ser salvos nos formatos POD1, POD2, POD3 ou POD5, mas todos eles usam a extensão "pod". O conteúdo de um arquivo POD pode ser extraído usando Dragon UnPACKer.
