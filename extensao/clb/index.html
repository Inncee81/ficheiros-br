O que é um arquivo .clb? (Arquivo de catálogo COM+)

Arquivo usado pelo Microsoft COM+, um serviço em sistemas operacionais Windows que permite que os componentes se comuniquem entre si; contém atributos de aplicativos, classe e nível de computador COM+; armazenar informações que fornecem consistência entre os serviços de computador.

Mais Informações

CLB arquivos são comumente encontrados no banco de dados de registro COM+, que está localizado na pasta %windows_directory%\registration. Eles usam a convenção de nomeação R00000000000X.clb, onde X é uma letra ou número.

O que é um arquivo .clb? (Lista de contatos do ICQ)

Arquivo criado pelo ICQ, um programa de bate-papo on-line; contém contatos do usuário, incluindo seu número exclusivo de ICQ, pseudônimos e outras informações de contato; usado para criar cópias de backup de contatos.

Mais Informações

NOTA: ICQ para Mac, que já foi disponibilizado, não é mais oferecido pela ICQcom.
