O que é um arquivo .xsx? (Arquivo de layout do esquema do estúdio visual XML)

Um arquivo XSX é um arquivo de propriedades XML usado pelo Microsoft Visual Studio, um Windows Ambiente de Desenvolvimento Integrado (IDE). Ele contém configurações de layout para um esquema XML, que é armazenado em um arquivo .XSD. Versões anteriores de arquivos XSX gerados automaticamente pelo Visual Studio quando os desenvolvedores projetaram layouts de esquema na interface XML Schema Designer.

Captura de tela de um arquivo xsx no Microsoft Visual Studio

O componente original do XML Schema Designer foi incluído no Visual Studio 2005, mas não foi incluído no Visual Studio 2008 e posteriormente. A Microsoft renovou e reintroduziu o XML Schema Designer no Visual Studio 2010. No entanto, o Visual Studio 2010 e versões posteriores não geram arquivos XSX.

NOTA: Se você acidentalmente excluir um arquivo XSX durante a edição de um esquema, você pode regenerar o arquivo fechando e reabrindo seu esquema.
