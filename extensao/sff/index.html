O que é um arquivo .sff? (Mugen. Arquivo Sprite)

Arquivo gráfico usado pela MUGEN, um motor de jogo de luta freeware 2D; armazena sprites, que são gráficos que armazenam os estados de animação para um lutador ou outro objeto animado; pode armazenar sprites para saltar, chutar, socar, correr ou outros movimentos.

Mais Informações

Os arquivos SFF são criados a partir de .PCX imagem bitmap usando o programa sprmaker incluído no mugen. game engine toolsuite.

O que é um arquivo .sff? (Arquivo de formato de fluxograma padrão)

Arquivo criado no Formato De Fluxo Padrão (SFF), um formato de arquivo binário usado para sequenciamento de DNA; salva a saída de uma máquina de sequenciamento de 454 Life Sciences (por exemplo, um GS Junior ou GS FLX System), que pega uma amostra de DNA de entrada e gera a sequência de DNA.

Mais Informações

Exemplos de programas que podem abrir e editar arquivos SFF incluem sff Workbench (anteriormente 454 SFF Tools) e MATLAB. Se usar o MATLAB, você deve usar a função sffread(), que retorna os dados SFF em uma matriz de estruturas MATLAB. 454 Life Sciences também fornece um kit de ferramentas para processamento e análise de arquivos SFF, chamado GS Data Analysis Software.

O que é um arquivo .sff? (Arquivo de fax estruturado)

Arquivo fax criado principalmente para documentos de fax do Grupo 3; inclui uma imagem de raster compactado e a estrutura de página do fax; armazena um cabeçalho seguido de informações de pixels linha por linha.

Mais Informações

Os arquivos SFF não são mais comumente usados, mas ainda podem ser abertos com vários programas de visualização de imagens.
