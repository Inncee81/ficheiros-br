O que é um arquivo .mrf? (Arquivo do gravador do mouse)

Um arquivo MRF é um arquivo de dados criado pela Bartels Media Mouse Recorder, um programa usado para gravar atividades de mouse e teclado que podem ser reproduzidos. Ele contém uma macro gravada, que pode incluir ações do mouse, entrada do teclado e tempo de espera entre as ações do mouse. Os arquivos MRF são usados para ajudar a automatizar as ações de um usuário de computador para economizar tempo ao executar tarefas repetitivas.

Captura de tela de um arquivo mrf em Bartels Media Mouse Recorder 1

O arquivo MRF é o tipo de arquivo principal associado ao Mouse Recorder. Depois de gravar a atividade do mouse no programa, um arquivo MRF é criado para armazenar as informações. Você pode abrir o arquivo MRF para reproduzir ações e editar a macro caso queira alterar qualquer atividade.

Para criar um arquivo MRF, selecione File → New e selecione File → Save. Para abrir um arquivo MRF, selecione File → Open e navegue até a localização do arquivo MRF.

Ao salvar um arquivo MRF, você pode escolher o nome ou deixá-lo como o nome padrão para ajudar a organizar seus arquivos MRF no momento em que eles foram criados. Os nomes padrão geralmente começam como "Macro" seguidos pela data e hora. Um nome de arquivo de exemplo é Macro_2_21_2018 10-20-23 AM.mrf.
