O que é um arquivo .ccp? (Arquivo de página do estúdio CodeCharge)

Arquivo XML criado pelo CodeCharge Studio, um programa usado para desenvolvimento rápido de aplicativos de Web; contém o layout de design de um componente de página web e pode ser reutilizado em páginas da Web; usado para formulários, layouts de relatório e outros componentes reutilizáveis.

Mais Informações

Arquivos CCP podem ser incluídos em um projeto do CodeCharge Studio (.CCS arquivo).

O que é um arquivo .ccp? (Arquivo CopperCube JavaScript)

Arquivo criado pelo CopperCube, um aplicativo 3D, aplicativo de edição de jogos e sites; contém um objeto pré-fabricado que pode ser inserido em sua cena 3D; também contém links para .JPG e .PNG arquivos gráficos que criam a aparência dos objetos; semelhante ao Arquivo .PREFAB Unity.

Mais Informações

CopperCube contém pré-fabricados ccp embutidos, mas você pode criar o seu próprio também. Selecione File → Export → Save current file as CopperCube Prefab..., nomeie seu arquivo, escolha o local de salvamento e clique em Salve.

Os arquivos CCP incorporados podem ser encontrados na seção "Prefabs" da Interface do Usuário CopperCube. Alguns exemplos incluem "balconista animado", " "soldado animado", " "mão de pistola", "fogo," "escadas de cobre," e "corredor"

Em um Mac, arquivos CCP embutidos estão localizados no aplicativo CopperCube. Clique com o botão direito do mouse no aplicativo na janela Finder e selecione Show Package Contents → Contents → Resources → bif_prefabs. Os arquivos CCP estão localizados nesta pasta entre outros arquivos .BMP.

Em Windows, os arquivos CCP incorporados estão localizados no seguinte diretório:
C:\Program Files (x86)\Ambiera\CopperCube 4.3\prefabs

O que é um arquivo .ccp? (Perfil do conversor de importação RIB)

Arquivo de dados usado pelo iTWO, um pacote de software usado para planejamento e engenharia de construção; armazena um perfil de conversão, que mapeia campos de dados entre uma fonte de dados externa (por exemplo, arquivo microsoft excel .XLS ou um arquivo de .TXT de texto simples) e o formato interno iTWO XML.

Mais Informações

Os arquivos CCP são usados por uma ferramenta no pacote de software iTWO chamado CONVERSOR de Importação RIB, que tem o nome de arquivo cconv.exe.
