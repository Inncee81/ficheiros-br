O que é um arquivo .mui? (Arquivo de interface de usuário multilíngue)

Arquivo de recursos usado pelo componente Interface de Usuário Multilíngue de Windows 2000 e posteriores; contém recursos que permitem que a interface Windows seja alterada para diferentes idiomas.

Mais Informações

Muitos arquivos MUI podem ser encontrados dentro do diretório Windows %SystemRoot%\ (por exemplo, %SystemRoot%\en-US\ em Windows 7). Eles também podem estar localizados dentro de instalações específicas do programa, como Windows Journal, Windows Defender, Windows Mail e outros programas de Windows da Microsoft, incluindo o Bloco de Notas (notepad.exe.mui).

Pacotes adicionais de idiomas podem ser instalados em Windows 7, selecionando Control Panel → Clock, Language, and Region → Change display language → Keyboards and Languages e clicando no botão Instalar/desinstalar idiomas. botão.

O que é um arquivo .mui? (Arquivo de instrumento de usuário myriad)

Um arquivo MUI é um arquivo de instrumentos de usuário usado por produtos de software de notação musical da Myriad, como o Harmony Assistant. Armazena sons para uma variedade de instrumentos como piano, cordas, sopros, latão. Os arquivos MUI são usados para gerar sons realistas.

Mais Informações

Captura de tela de um arquivo mui em Myriad Harmony Assistant

Existem muitos instrumentos padrão que vêm com o Harmony Assistant. Os instrumentos consistem em uma gravação de um instrumento tocando uma nota específica e parâmetros para dizer ao programa como ele deve ser reproduzido. O arquivo MUI permite que os usuários criem instrumentos diferentes dos padrão editando um instrumento e selecionando "Salvar instrumento predefinido" no menu contextual "Action". O usuário também pode definir os parâmetros dos instrumentos, como a faixa de nota, taxa de ataque de som, decadência, liberação de volume e liberação de loop.
