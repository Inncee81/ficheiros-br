O que é um arquivo .ddt? (Arquivo de texturas da Era da Mitologia)

Arquivo gráfico usado pela Age of Mythology (AOM), um jogo de estratégia em tempo real; geralmente empacotado dentro de um arquivo .BAR e contém texturas que são aplicadas a modelos de jogos 3D; usado para texturizar objetos como heróis, ícones, objetos de terreno e unidades do exército.

Mais Informações

DDT arquivos podem ser extraídos de um arquivo BAR e convertidos em um formato de .BMP usando AOM Edit, que também é chamado de AOMed. Uma vez extraídos, eles podem ser editados usando qualquer editor de imagem e, em seguida, convertidos de volta no formato DDT usando AOM Edit.

DDT suportam múltiplas resoluções, mas as dimensões do gráfico devem ser um poder de dois. Por exemplo, uma textura 100x100 pode usar os tamanhos 200x200 e 400x400, mas não 300x300.

O que é um arquivo .ddt? (Modelo de designer de diagrama)

Arquivo de modelo criado pelo Diagram Designer, um editor de diagramas freeware usado para criar fluxogramas e diagramas UML; contém objetos de diagrama, linhas de conexão e texto.

Mais Informações

Os arquivos DDT são usados para armazenar documentação técnica para código de programa orientado a objetos, processos de negócios e outros desenhos conceituais.

Programas que abrem o arquivo .ddt (Modelo de designer de diagrama)

  Diagram Designer

O que é um arquivo .ddt? (Gerente de futebol manter arquivo de dados do jogador)

Arquivo de dados usado pelo Football Manager, um jogo onde os jogadores gerenciam finanças, escalações e outros aspectos de times de futebol; contém instruções de texto simples que descrevem importações para jogadores de futebol e times.

Mais Informações

Os arquivos DDT são usados para estender o banco de dados padrão do jogo para reter e carregar jogadores personalizados em listas de equipes.

NOTA: Jogadores de times de ligas não ativas geralmente ficam de fora quando um novo jogo é criado no Football Manager. Use um arquivo DDT para reter esses jogadores.

Formato Binário 

O que é um arquivo .ddt? (Jill do Arquivo de Música da Selva)

Arquivo de música usado por Jill of the Jungle, um jogo onde os jogadores controlam Jill através de uma série de níveis baseados na selva; contém dados de áudio de jogo armazenados no Creative Music Format (CMF), que é uma versão reduzida do formato MIDI.

Mais Informações

Arquivos DDT são usados para reprodução de áudio durante a jogabilidade.
