O que é um arquivo .str? (arquivo objeto de lista de estrutura dBASE)

Arquivo de texto que define a estrutura de um banco de dados dBASE; contém uma lista de todos os objetos que podem ser incluídos no arquivo do banco de dados; inclui o número, nomes e larguras de cada campo no banco de dados.

Mais Informações

STR arquivos podem ser usados pelo dBASE para criar tabelas de índice de arquivo impresso ou para documentar a estrutura de cada tabela.

O que é um arquivo .str? (Arquivo de cordas Kingsoft)

Arquivo Strings usado pelo WPS Office (anteriormente conhecido como Kingsoft Office), um pacote de escritório que inclui os aplicativos Writer, Planilhas e Apresentação; contém informações usadas para configurar a interface do aplicativo, que inclui a colocação e aparência de formas, painéis e bordas; também define a aparência do texto que aparece nos títulos da janela, botões, textos do menu e outros elementos de interface.

Mais Informações

NOTA: O arquivo STR está localizado no seguinte diretório:/Kingsoft/WPS Office/office[version]/mui/default/tips

O que é um arquivo .str? (Arquivo de sequência de objeto x-plane)

Arquivo de sequência de objetos usado pelo X-Plane, um simulador de voo profissional; contém dados que definem a colocação de objetos no simulador.

Mais Informações

O arquivo STR usa o formato X-Plane padrão, que consiste em um cabeçalho e comandos que definem diferentes aspectos dos objetos e sua colocação na linha. O formato STR usa strings de objeto, que são objetos colocados ao longo de uma curva bezier. O arquivo foi projetado para colocar objetos no simulador X-Plane, mas é usado principalmente para construir luzes de taxiway do aeroporto.
