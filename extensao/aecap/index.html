O que é um arquivo .aecap? (Modelo de texto após efeitos)

Um arquivo AECAP contém um modelo de texto criado por After Effects, um programa de vídeo gráfico de movimento. Ele armazena um projeto compactado com todos os seus ativos, que inclui uma ou mais camadas de texto, efeitos e mídia. Os arquivos AECAP são normalmente usados para sobreposições de texto, como títulos, terços inferiores, legendas e legendas.

Mais Informações

Os modelos AECAP são criados por After Effects para uso no software de edição de vídeo Premiere Pro. Você pode importar um arquivo AECAP no Premiere Pro usando a ferramenta Media Browser incluída no Premiere Pro ou selecionando File → Import e escolhendo o modelo AECAP. Todas as edições que você fizer no modelo de texto no Premiere Pro serão salvas no arquivo AECAP.

Para exportar um arquivo AECAP em Efeitos posteriores, crie sua composição e selecione File → Export → Composition as Text Template ou Composition → Export Composition as Text Template. Depois de exportar o arquivo AECAP, não poderá editá-lo em After Effects.

Você pode importar um arquivo AECAP em After Effects de várias maneiras diferentes:

  • Selecione File → Import e escolha Adobe After Effects Text Templates (*.aecap).
  • Selecione File → Adobe Dynamic Link → Import After Effects Composition e clique no arquivo AECAP.
  • Use o Navegador de Mídia para escolher seu arquivo AECAP, selecione File → Import, clique em Import no menu de contexto do arquivo AECAP.

NOTA: Os modelos da AECAP foram introduzidos com o lançamento do Adobe Creative Cloud (CC) 2017 nas versões 140 e 141 de After Effects. No entanto, a Adobe logo substituiu os modelos AECAP em versões posteriores do After Effects por modelos .MOGRT. O Premiere Pro ainda suporta os modelos AECAP, mas eles não podem mais ser criados em After Effects.
