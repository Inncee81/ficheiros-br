O que é um arquivo .mcp? (Projeto CodeWarrior)

Projeto de desenvolvimento criado pela CodeWarrior, um IDE de desenvolvimento C e C++; salva referências a código-fonte do projeto e arquivos de recursos, bem como preferências específicas do projeto; originalmente usado pelo Metrowerks CodeWarrior (MCP), que agora é o CodeWarrior em escala livre.

O que é um arquivo .mcp? (Pacote Megacubo)

IPTV (Internet Protocol TV) arquivo de dados do canal usado pelo Megacubo, um aplicativo de streaming de TV online espanhol; contém vários arquivos .MEGA, cada um dos quais define um canal online que pode ser transmitido para o computador local.

Mais Informações

Os arquivos MCP são normalmente retirados do site do Megacubo. Uma vez instalado o software Megacubo, você pode clicar duas vezes em um arquivo MCP para instalar os canais.
