O que é um arquivo .lsp? (Arquivo de código fonte do programa Lisp)

Arquivo de código fonte escrito em Lisp, uma linguagem de programação que inclui variações como Common Lisp e Scheme; armazena código de programa de texto simples que pode ser executado com um intérprete lisp; pode ser um componente de aplicativo ou um programa inteiro.

Mais Informações

Os arquivos LISP podem ser visualizados e editados com qualquer editor de texto, mas são mais facilmente visualizados e editados com um editor que fornece destaque de sintaxe lisp.

Os arquivos de código fonte lisp são mais comumente vistos com a extensão .LISP.

NOTA: A linguagem de programação lisp tem sido amplamente utilizada nas áreas de inteligência artificial (IA), processamento simbólico, processamento de linguagem natural (PNL).
