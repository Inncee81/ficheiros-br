O que é um arquivo .gbr? (Arquivo Gameboy Tileset)

Arquivo criado pela Gameboy Tile Designer (GBTD), um programa que permite aos usuários projetar telhas para suas produções Gameboy; contém informações de azulejos e paletas, incluindo o conjunto de cores Gameboy, número de azulejos e tamanho de azulejos (8 x 8, 8 x 16, 16 x 16, ou 32 x 32).

Mais Informações

Para criar um arquivo GBR, projete seus azulejos, selecione File → Save or Save As..., nomeie o arquivo, escolha o local de salvamento e clique em Save.

O que é um arquivo .gbr? (Arquivo Gerber)

Um arquivo GBR contém dados de design de placa de circuito impresso (PCB) no formato Gerber, que é um formato de arquivo padrão do setor suportado por muitos sistemas de design PCB. Contém instruções para perfuração e fresagem da placa e inclui informações sobre as conexões elétricas do conselho. Os arquivos GBR são comumente usados por profissionais do PCB para trocar dados de imagem, broca e rout com segurança do software de design para máquinas de fabricação.

Mais Informações

O formato Gerber é simples e compacto, armazenando dados que descrevem precisamente uma imagem 2D de um PCB e metadados necessários para a fabricação de um PCB. O formato GBR foi originalmente desenvolvido pela Gerber Systems, mas agora é mantido pela Ucamco.

O que é um arquivo .gbr? (Arquivo de pincel GIMP)

Escova de imagem usada pelo GNU Image Manipulation Program (GIMP), um aplicativo de edição de imagens de código aberto; armazena a descrição, imagem do bitmap e o espaçamento para o pincel; usado para armazenar gráficos que podem ser pintados repetidamente em uma tela de imagem GIMP.

Mais Informações

Os arquivos GBR são usados para armazenar tanto os pincéis padrão do programa quanto os pincéis personalizados do usuário. Alguns pincéis padrão estão localizados no diretório /share/gimp/2.0/brushes/ de uma instalação GIMP.
