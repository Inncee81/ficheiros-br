O que é um arquivo .ps2? (Arquivo do índice do catálogo de pesquisa da Microsoft)

Arquivo de sistema criado pelo Indexing Service, um componente do Microsoft Windows usado para gerar automaticamente um índice de sistema de arquivos pesquisável; contém dados que ajudam a pesquisa Windows realizar pesquisas de arquivos mais rapidamente.

Mais Informações

Arquivos PS2, juntamente com outros arquivos de índice de pesquisa, são frequentemente criados no diretório C:\System Volume Information\catalog.wci.

O que é um arquivo .ps2? (Arquivo do cartão de memória PCSX2)

Arquivo usado pelo PCSX2, um emulador do PlayStation 2 (PS2); contém uma imagem de cartão de memória que pode incluir dados salvos do usuário; pode ser aberto com o programa mymc, a fim de extrair jogos salvos e convertê-los em formatos como .PSU.

Mais Informações

PCSX2 usa arquivos .P2S para estados salvos (ou arquivos sem uma extensão em versões anteriores do software).

NOTA: Sharkport, um acessório para PlayStation 2, também usa a extensão "ps2", mas para jogos salvos em vez de imagens de cartão de memória. Portanto, se você receber um arquivo PS2, você pode precisar testar para ambos os tipos de arquivos PCSX2 e Sharkport.

O que é um arquivo .ps2? (Jogo salvo sharkport)

Arquivo de jogo salvo criado pelo Sharkport, um acessório de hardware do PlayStation 2 (PS2) usado para transferir dados para um PC; contém um jogo salvo do PS2, mas no formato de arquivo Sharkport; usado para compartilhar jogos salvos com outros via e-mail e internet.

Mais Informações

A extensão "ps2" também é usada por imagens do cartão de memória PCSX2 (PlayStation 2). No entanto, os formatos Sharkport e PCSX2 "ps2" não são compatíveis entre si. Portanto, se você receber um arquivo PS2, você pode precisar testá-lo tanto com Sharkport quanto PCSX2. O PCSX2 usa a extensão .P2S para salvamentos.

Os arquivos PS2 do Sharkport podem ser convertidos para outros formatos de salvamento do PlayStation 2 usando o PS2 Save Builder. No entanto, primeiro você precisa alterar a extensão "ps2" para .MD, a extensão usada por versões anteriores do Sharkport.
