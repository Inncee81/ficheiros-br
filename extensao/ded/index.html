O que é um arquivo .ded? (Arquivo criptografado do DED Cryptor Ransomware)

Um arquivo com uma extensão .ded é um arquivo que foi criptografado pelo vírus DED Cryptor, que é o ransomware que se tornou predominante no final de 2019. Ele é criptografado para que não seja possível abrir o arquivo simplesmente alterando a extensão de arquivo .ded.

Mais Informações

Nbes ransomware é um tipo de malware utilizado por cibercriminosos que criptografa os arquivos em um computador com criptografia AES-256. Depois que o ransomware toma os arquivos como reféns, ele força a vítima a pagar o criminoso para desbloquear os arquivos. Na maioria das vezes é introduzido no computador de uma vítima através de e-mails de spam com um Bitcoins.

NOTA: Infelizmente, não há nenhum programa disponível para restaurar efetivamente arquivos infectados. Se o usuário tiver um backup recente de seus arquivos, ele pode executar uma restauração do sistema para remover o vírus, mas quaisquer alterações feitas nos arquivos após o backup foi feito serão perdidas.

O que é um arquivo .ded? (Dr. Engrave Documento)

Documento criado pelo Dr. Engrave, um programa usado para projetar gravuras para crachás, placas de identificação e prêmios; contém texto e gráficos, bem como as dimensões de layout de destino; suporta gráficos raster e vetor, bem como Windows fontes True Type; pode ser gravado usando um dispositivo de gravação Roland compatível.

Mais Informações

Dr. Engrave é compatível com várias máquinas de gravação Roland, incluindo o MPX-90, EGX-20, EGX-30, EGX-300, EGX-360 e EGX PRO. O software originalmente suportava Windows plataformas 95, 98 e NT, mas agora inclui suporte para Windows 2000, XP e Vista.
