O que é um arquivo .vtpr? (Projeto visualizador de loja de estúdio)

Um arquivo VTPR contém um projeto criado pela Esko Studio Store Visualizer, um módulo que pode ser adicionado ao Esko Studio para visualizar projetos de produtos em ambientes de lojas de varejo. Ele armazena um projeto, que inclui informações de design de produtos, bibliotecas, configurações de câmera e configurações de projeto.

Mais Informações

Você pode abrir um arquivo VTPR clicando com o botão direito do mouse na área de renderização 3D no Store Visualizer, que abre o menu contextual. Selecione File → Open e escolha o arquivo VTPR que deseja abrir. Você também pode usar a barra lateral do sistema para abrir um arquivo VTPR. Na barra lateral, clique no painel "Arquivo",selecione Project → Open e escolha o arquivo VTPR que deseja abrir.

Existem dois tipos de projetos que podem ser armazenados em arquivos VTPR:

  • Ambiente fotográfico esférico - Permite que você coloque seu produto em fotografias reais adquiridas de uma loja real.
  • Ambiente virtual 3D - Permite que você coloque seu design de produto em um ambiente de varejo virtual 3D.

Os arquivos VTPR são salvos por padrão no seguinte local:

~/Documents/Store Visualizer [version]/Projects

Os modelos VTPR estão localizados no seguinte local:

~/Documents/Store Visualizer [version]/Templates

Arquivos VTPR estão vinculados a arquivos .3DW, que contêm informações sobre o ambiente 3D no qual o design do produto está sendo visualizado. Quando um arquivo VTPR é aberto no Studio Store Visualizer, o programa carrega as informações no arquivo 3DW. O VTPR e o arquivo 3DW forrado devem estar localizados na mesma pasta. Além disso, se você compartilhar um arquivo VTPR com outro usuário, você deve incluir o arquivo 3DW vinculado e quaisquer arquivos de biblioteca .L3DW externos vinculados, ou então você não será capaz de visualizar o produto no ambiente 3D.
