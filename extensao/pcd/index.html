O que é um arquivo .pcd? (Arquivo de imagem kodak photo CD)

Formato proprietário da Kodak para armazenar imagens em CDs de fotos; usado para salvar fotografias digitalizadas usando hardware de processamento de fotos kodaks.

Mais Informações

As imagens de CD de fotos são tipicamente digitalizadas em uma resolução de 3072x2048 pixels, mas podem ser digitalizadas em 6144x4096 usando o serviço de CD Pro Photo da Kodak. Ambos os formatos usam cor de 24 bits.

Os arquivos PCD contêm cinco cópias da imagem em diferentes resoluções, e as imagens do CD Pro Photo contêm uma sexta imagem adicional que tem 6144x4096 pixels de tamanho. O menor tamanho de imagem é de 192x128 pixels, e esta imagem é usada como miniatura para navegar todas as fotos no CD de uma só vez.

O que é um arquivo .pcd? (Arquivo do cartão Pokémon Wonder)

Arquivo de jogo usado por jogos Pokémon para o dispositivo de jogo portátil Nintendo DS; armazena informações para um Wonder Card, que pode ser contido em um "presente misterioso" recebido do entregador no jogo no Poké Mart; desbloqueia itens e eventos no jogo.

Mais Informações

Os dados do Wonder Card são salvos em jogos salvos por Pokémon (.SAV arquivos), que são criados em cartuchos Nintendo DS. Pokesav e o Pokémon Mystery Gift Editor podem editar arquivos PCD, e Pokesav pode importá-los em arquivos SAV existentes.

Arquivo de dados de componente puro

Desenvolvedor ChemSep
Categoria Dados
Formato Binário 

O que é um arquivo .pcd? (Arquivo de dados de componente puro)

Arquivo de dados químicos criado pela ChemSep, um simulador de coluna para operações de destilação, extração e absorção; contém correlações críticas, moleculares, EOS, T e outros dados para componentes, que incluem benzeno, fenol, ciclohexanona, dióxido de carbono, anilina e bromobenzeno.

Mais Informações

Os dados registrados no arquivo PCD aparecem no Sistema Internacional de Unidades (SI), com exceção das quantidades molares, que são registradas em kmol. Você pode visualizar os dados em arquivos PCD ChemSep através do pcDmanager, que é acessível selecionando Databanks → Pure components data (PCD/PCT).

NOTA: Os dados em arquivos PCD são armazenados em formato binário, enquanto .PCT arquivos armazenam dados em formato de texto.
