O que é um arquivo .mdl? (Arquivo modelo de meia-vida)

Um arquivo MDL é um modelo tridimensional usado para personagens e criaturas no videogame Half-Life. É criado usando polígonos e mapeamento de texturas para detalhes. Os modelos MDL podem ser vistos a partir de diferentes pontos de vista (wireframe, flatshaded, smoothshaded e texturizado) usando o Half-Life Model Viewer.

O que é um arquivo .mdl? (Arquivo de design de roupas lectra modaris)

Um arquivo MDL é um arquivo CAD criado pela Lectra Modaris, um aplicativo de design de roupas. Ele salva a lista (ou cartão de direção do cortador) e a geometria das peças da peça. Os arquivos MDL são usados para salvar novos design de moda em roupas masculinas, femininas e infantis.

Mais Informações

Quando um design é concluído, ele pode ser enviado para um cortador de máquinas que corta os padrões projetados em um rolo de tecido ou outro material.

O que é um arquivo .mdl? (Arquivo modelo 3D da Mansão de Luigi)

Desenvolvedor MathWorks

O que é um arquivo .mdl? (Modelo MathWorks Simulink)

Um arquivo MDL é um modelo de simulação criado usando o Simulink, um aplicativo de simulação e design baseado em modelos. Contém o diagrama do bloco e bloqueia as propriedades de uma simulação.

Mais Informações

Os arquivos MDL são salvos em um formato de texto simples e normalmente incluem uma lista de objetos. Cada objeto contém um conjunto de propriedades e valores que são delimitados por quebras de linha.

NOTA: O formato MDL foi substituído pelo formato de modelo .SLX com o lançamento do Simulink 2012.

O que é um arquivo .mdl? (Warcraft 3 3D Model Text File)

Um arquivo MDL é um arquivo modelo usado por Warcraft III, um jogo de estratégia em tempo real desenvolvido pela Blizzard Entertainment. Ele armazena um modelo 3D, incluindo objetos, malhas e texturas para edifícios e unidades. Os arquivos MDL permitem a edição baseada em texto de propriedades do modelo antes de gerar um arquivo binário de .MDX usado pelo Warcraft III.

Mais Informações

Arquivos MDL usam uma estrutura aninhada para criar propriedades de modelo. Por exemplo, um elemento material pode incluir uma propriedade de camada que, por sua vez, define uma propriedade transparente para a camada.

NOTA: Os arquivos MDL são a versão de texto dos arquivos binários do modelo MDX 3D. Eles podem ser convertidos para o formato MDX usando o programa Conversor MDLX. A partir daí, você pode usar várias ferramentas de modelagem para editar o modelo.
