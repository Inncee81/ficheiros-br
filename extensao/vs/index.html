O que é um arquivo .vs? (Arquivo da lista do Verso GreekToMe)

Arquivo criado pelo GreekToMe, um programa de estudo do Novo Testamento grego; contém uma lista de versículos que foram selecionados na lista de versos e depois exportados através da opção File → Export Verse List... menu; podem ser importados posteriormente pela opção File → Import Verse List... para restaurar as seleções.

O que é um arquivo .vs? (Arquivo sombreador vertex)

Arquivo usado por aplicativos e jogos OpenGL e DirectX; salva propriedades de iluminação para vértices, que são pontos em espaço 3D; permite que um modelo 3D tenha efeitos de sombreamento; salvo em um formato de texto simples.

Mais Informações

NOTA: Os arquivos de sombreador da Vertex usam mais frequentemente a extensão .VSH. Os arquivos do sombreador vertex com a extensão "vs" podem não usar o mesmo formato dependendo do mecanismo gráfico de destino.
