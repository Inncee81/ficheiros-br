O que é um arquivo .stm? (Arquivo de mídia de streaming de exchange)

Arquivo de dados salvo pelo Microsoft Exchange Server em conjunto com .EDB arquivos; usado pela loja de informações exchange, que é composta por:

  1. Priv1edb
  2. Priv1stm
  3. Pub1edb
  4. Pub1stm
STM arquivos armazenam conteúdo nativo da Internet. Novas mensagens são salvas no arquivo STM no servidor de e-mail até serem acessadas pelo cliente de e-mail de um usuário. Quando as mensagens são acessadas por um cliente MAPI, elas são transferidas para o arquivo EDB e convertidas para o formato exchange nativo.

O que é um arquivo .stm? (Página de Web SSI)

Web página que contém lado do servidor inclui (SSI); referências funções no servidor que gera conteúdo para a página Web; processada pelo servidor quando acessada, que faz a saída do HTML para o navegador Web do cliente.

O que é um arquivo .stm? (O arquivo de carimbo de brinquedo em pó)

Um arquivo STM contém um selo criado pelo The Powder Toy, um jogo gratuito de sandbox de simulação de física 2D. Ele armazena um selo, que é uma área personalizada selecionada e salva em The Powder Toy durante a jogabilidade. Os arquivos STM são considerados "mini-saves" que podem ser usados para compor uma simulação maior no jogo, que é armazenado no arquivo .CPS.

Mais Informações

Captura de tela de um arquivo stm em hardWIRED The Powder Toy 925

O Brinquedo em Pó permite que você brinque com diferentes substâncias para simular os efeitos de diferentes interações imobiliárias. O jogo vem com uma variedade de líquidos, gases, componentes eletrônicos e materiais de construção que você pode usar para construir terrenos, explosivos, edifícios e muito mais. Durante a jogabilidade, você pode definir uma área especificada para ser salva e carregada em um momento diferente. Os elementos ou objetos na área especificada estão incluídos na salvação. Essas áreas salvas são chamadas "stamps" e são armazenadas no arquivo STM.

Para criar um arquivo STM, digite "S" durante a jogabilidade e desenhe a área que deseja salvar como um selo. Para carregar um arquivo STM, digite "K" durante a jogabilidade e escolha o selo que deseja carregar.
