O que é um arquivo .gtp? (Arquivo de dados da pasta gerber top solder)

Arquivo de saída de dados criado por ferramentas CAD de placa de circuito impresso (PCB) como Autodesk EAGLE; contém dados de layout para a camada superior da pasta do PCB para criar estêncil para aplicar pasta na placa; escrito em RS-274X, que é o formato Gerber mais comum e contemporâneo.

Mais Informações

O arquivo GTP age semelhante a um arquivo .PDF. É uma versão exportada do layout PCB e usada para criar estêncil para aplicar pasta em placas para adicionar dispositivos de montagem de superfície (SMD), como almofadas. É para ser visto e referenciado, mas não editado.

NOTA: O formato Gerber foi originalmente desenvolvido pela Gerber Systems, mas agora é mantido pela Ucamco.

O que é um arquivo .gtp? (Arquivo do pacote temático GNOME)

Arquivo de pacote temático usado pelo GNOME, um programa que permite que os usuários personalizem suas configurações de desktop em Linux; contém um diretório de theme_name que inclui um arquivo de arquivo indexado e diretórios gtk e metacity; salvo em um formato gzip-compacted-tar.

Mais Informações

ARQUIVOS GTP podem ser instalados clicando-os duas vezes em um ambiente GNOME.

O que é um arquivo .gtp? (Arquivo Pro de guitarra)

Arquivo de dados Guitar Pro usado pelas versões 1 e 2 do Guitar Pro; contém tablatures para instrumentos de guitarra ou baixo; usado para gravar e salvar partituras que podem ser impressas ou reproduzidas.
