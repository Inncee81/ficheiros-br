O que é um arquivo .lng? (Arquivo de idioma AVG)

Arquivo de idioma usado pelo AVG, um software antivírus para Windows; armazena informações de idiomas que permitem que o programa seja localizado em diferentes idiomas; incluído como parte da instalação do software.

Mais Informações

O idioma pode ser alterado no AVG AntiVirus selecionando Tools → Advanced settings..., escolhendo "Aparência"," e, em seguida, selecionando um novo idioma da lista.

O que é um arquivo .lng? (Arquivo de suporte a aplicativos de idioma)

Arquivo de idioma genérico usado por aplicativos para adicionar suporte ao idioma; muitas vezes usado para permitir a localização da interface do usuário; geralmente referenciado pelo programa e não aberto manualmente pelo usuário.

Mais Informações

NOTA: Arquivos de idiomas baseados em texto podem ser abertos com qualquer editor de texto. No entanto, nem todos os arquivos de GNL usam um formato de texto simples.

O que é um arquivo .lng? (Arquivo de idioma Winamp)

Um arquivo de GNL é um arquivo de pacote de idiomas usado pelo Winamp, um music player para Windows e OS X. Ele contém o texto em um idioma específico para uma parte da interface Winamp. Os arquivos de GNL são usados para personalizar o Winamp para diferentes idiomas e são comumente encontrados dentro de .WLZ arquivos plugin do pacote de idiomas.

Mais Informações

Winamp é um media player que pode reproduzir áudio, vídeo e outros conteúdos multimídia. O software suporta uma grande variedade de formatos de arquivos de áudio e vídeo e fornece ferramentas para organizar arquivos de música e vídeo em listas de reprodução e bibliotecas pesquisáveis.
