O que é um arquivo .ora? (Arquivo de imagem OpenRaster)

Arquivo de imagem salvo no formato OpenRaster e formato de gráficos raster aberto que suporta várias camadas; suporta várias camadas e contém dados de imagem e metadados, incluindo tags XMP e EXIF.

Mais Informações

OpenRaster foi projetado para ser um formato padrão de edição de imagens entre plataformas que pode ser usado por qualquer aplicativo de edição de gráficos. Os arquivos ORA podem conter várias camadas de imagem, bem como camadas de ajuste, efeitos de camada, grupos de camadas e opções de mistura de camadas. Eles também podem incluir texto, caminhos, seleções salvas, máscaras de camada e informações de perfil de cores.

NOTA: O formato OpenRaster foi desenvolvido como uma alternativa ao formato .PSD da Adobe, que tem diretrizes de uso mais rigorosas.

O que é um arquivo .ora? (Arquivo de configuração do banco de dados Oracle)

Arquivo de texto usado pelo Oracle Database, um sistema de gerenciamento de banco de dados relacional de nível corporativo (RDBMS); contém parâmetros de configuração de banco de dados; usado para armazenar padrões em todo o sistema e para ajustar o desempenho do banco de dados.

Mais Informações

Todos os parâmetros nos arquivos ORA são opcionais, uma vez que o banco de dados Oracle já contém valores padrão para os parâmetros necessários. Os parâmetros podem ser especificados em qualquer ordem.

Os nomes de arquivos ORA comuns incluem init.ora (parâmetros de inicialização) e tnsnames.ora (arquivo de configuração de conexão SQL*Net).
