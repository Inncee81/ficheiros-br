O que é um arquivo .vmap? (Arquivo de mapa não-compensado da fonte da válvula 2)

Um arquivo VMAP é um arquivo de mapa criado pelo Valve Hammer Editor, uma ferramenta incluída no SDK de origem e usada para criar mapas para jogos usando o mecanismo Source 2. Contém um mapa, que pode incluir pistas, antigos, fontes, quartéis, runas, lojas, campos neutros, torres e outros elementos do mapa. Os arquivos VMAP são compilados em arquivos binários .VMAP_C, que são usados em um jogo usando o motor Fonte 2.

Mais Informações

Os arquivos VMAP são normalmente vistos apenas por jogadores do DOTA 2 que querem modificar mapas jogáveis. Os arquivos VMAP foram introduzidos com a liberação do motor Fonte 2 e substituídos .VMF arquivos, que são usados por jogos com o motor Fonte 1. Os arquivos VMAP são mais comumente usados pelo DOTA 2, mas também podem ser usados para outros jogos que usam o motor Fonte 2.

Você pode criar e editar arquivos VMAP com o Valve Hammer Editor, que está incluído nas Ferramentas de Oficina de Origem SDK e DOTA 2. Depois que você terminar de modificar um arquivo VMAP, você deve construí-lo em um mapa compilado (VMAP_C arquivo), que é embalado em um arquivo .VPK. Selecione File → Build Map, escolha suas configurações e clique em Build para compilar o arquivo VMAP.
