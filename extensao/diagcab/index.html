O que é um arquivo .diagcab? (Arquivo de armário de embalagem de solução de problemas)

Um arquivo DIAGCAB é um arquivo usado pela Ferramenta de Diagnóstico de Suporte da Microsoft (também conhecido como WTP Wizard), um programa Windows Plataforma de Solução de Problemas (WTP) usado para localizar e corrigir problemas de hardware e software em Windows. Ele contém um arquivo .CAB e é usado especificamente para diagnosticar e reparar problemas de computador.

Mais Informações

WTP inclui um assistente que o guia através de uma série de etapas que ajudam você a localizar e corrigir problemas em seu computador. Quando o diagnóstico estiver completo, você tem a opção de enviar informações de diagnóstico pela Internet para uma fonte confiável para que eles possam ajudar a diagnosticar e reparar seu computador.

WTP é usado por fabricantes de equipamentos originais (OEMs) e fornecedores independentes de software (ISVs) para solucionar problemas experimentados pelos clientes. Eles também são usados por administradores do sistema. Os arquivos DIAGCAB às vezes são postados em sites e baixados pelos usuários. Se distribuídos dessa forma, os criadores dos arquivos geralmente os assinam digitalmente para verificar se são de uma fonte confiável.

Os arquivos DIAGCAB podem ser criados com as ferramentas Makecab.exe e Cabarc.exe. Eles podem ser assinados digitalmente usando Signtool.exe. Eles são executados usando a Ferramenta de Diagnóstico de Suporte da Microsoft (msdt.exe). Para executar o arquivo DIAGCAB com o MSDT, pesquise "msdt" no menu Iniciar e selecione o programa na lista de resultados. Em seguida, digite a senha (muitas vezes fornecida por um profissional de suporte da Microsoft), em seguida, siga as instruções na tela para executar o arquivo DIAGCAB.

NOTA: Os arquivos DIAGCAB foram usados pela primeira vez em Windows Vista, então se você estiver executando uma versão Windows anterior, como Windows XP, Windows não reconhecerá o arquivo.
