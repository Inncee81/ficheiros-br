O que é um arquivo .dsf? (Arquivo de ativos do DAZ Studio)

3D formato de arquivo de modelagem introduzido com software de modelagem DAZ Studio 4; pode conter caractere, pose, modelagem, sombreador, materiais ou outros dados para um modelo criado no software; usado para trocar informações do modelo DAZ com outros programas de modelagem 3D que suportam o formato DSF.

Mais Informações

DSF os arquivos usam a formatação JSON para que os arquivos sejam legíveis e facilmente trocados. Os arquivos DSF também podem incorporar .ZIP compactação e, portanto, não estar em um formato JSON de texto simples (a opção de compressão é especificada pelo usuário na caixa de diálogo salvar o DAZ Studio). Os arquivos DSF compactados têm um tamanho de arquivo significativamente reduzido, uma vez que o texto JSON compacta em altas proporções.

NOTA: Os arquivos DSF compactados podem ser descomprimidos com qualquer utilitário compatível com Zip.

O que é um arquivo .dsf? (Arquivo de designer)

Arquivo criado pela Corel Designer, um programa de desenho anteriormente de propriedade da Micrografx; contém um gráfico vetorial, que pode incluir texto, imagens rasteres e objetos desenhados; usado para criar ilustrações, como logotipos da empresa, materiais de marketing e outros gráficos impressos e digitais.

Mais Informações

NOTA: Os arquivos de designer usam a extensão .DES em versões mais recentes do Corel Designer, que agora é oferecido como um aplicativo no Corel DESIGNER Graphics Suite.

O que é um arquivo .dsf? (Projeto Dramatica Pro)

Um arquivo DSF é um documento criado pela Dramatica Pro, um programa de Windows usado por escritores para pensar e organizar histórias. Contém um projeto de história, que inclui o título, autor, progressão de enredo, tema da história, personagens e relacionamentos de personagens.

Mais Informações

Captura de tela de um arquivo dsf em Write Brothers Dramatica Pro 5

O arquivo DSF é o principal tipo de arquivo associado ao Dramatica Pro. Quando você começa uma nova história e a salva, o programa cria o arquivo DSF para salvar o conteúdo da história. Você pode criar um arquivo DSF selecionando File → Save.

Dramatica Pro está disponível apenas para Windows. No entanto, há uma versão do macOS chamada Dramatica Story Expert. Dramatica Pro foi projetada para ajudá-lo a juntar sua história definindo seu enredo geral, personagens, relacionamentos de personagens e temas. Ele também vem com ferramentas de brainstorming, um mecanismo de história para gerar um enredo, e uma ferramenta de elenco para adicionar personagens.

Dramatica Story Expert salva projetos no arquivo .DR5 em vez do arquivo DSF. No entanto, se você quiser abrir um arquivo DSF no Dramatica Story Expert, você pode renomear a extensão "dsf" para a extensão "dr5".

O que é um arquivo .dsf? (Formato de cenário de distribuição de plano X)

Um arquivo DSF contém informações no Formato de Cenário de Distribuição (DSF) que descreve o aparecimento de cenários em X-Plane, um simulador de voo realista para pilotos e amadores de simuladores. Ele armazena dados de cenário, que incluem as coordenadas de longitude e latitude para cada seção da Terra que aparece no jogo e referências a recursos gráficos, como imagens de bitmap e .OBJ arquivos de objeto 3D, usados para preencher as seções. Os arquivos DSF podem ser descompactados ou compactados com .7Z compactação.

Mais Informações

Você só encontrará arquivos DSF formato de cenário de distribuição se você usar o simulador X-Plane e navegar para os arquivos de recursos embalados com o aplicativo. Os arquivos não devem ser abertos manualmente por um usuário. Em vez disso, eles são referência por X-Plane ao cenário de textura apresentado na jogabilidade.

No entanto, alguns jogadores gostam de modificar o cenário que aparece no jogo. Estes "modders" muitas vezes usam o WorldEditor (WED) para modificar cenários 2D e aeroportos em X-Plane. O WED está disponível gratuitamente para Windows, macOS e Linux.

NOTA: Os arquivos DSF são usados pelo X-Plane 7 e posteriormente. No entanto, apenas x-plane 10 e posteriormente suportam arquivos DSF compactados.
