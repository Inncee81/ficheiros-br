O que é um arquivo .iaa? (Arquivo de áudio INTUS)

Arquivo usado pelos terminais INTUS, que são usados para gerenciamento de cartões de ponto, controle de acesso e coleta de dados dos funcionários; salva um arquivo de arquivos de áudio que são usados para a saída de sons do terminal; usados para agregar todos os arquivos de áudio juntos para um terminal.

Mais Informações

Os arquivos IAA são armazenados em um formato compactado e são compactados com o algoritmo Deflate.
