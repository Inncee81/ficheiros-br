O que é um arquivo .odf? (Arquivo de definição de objeto battlezone)

Arquivo de configurações de jogos usado pelo Battlezone, um jogo 3D onde os jogadores lutam com veículos de combate futuristas; contém especificações de texto simples para objetos de jogo, chamados "classes;" podem definir nomes de unidades, objetivos de missão, propriedades do veículo, jogadores de inteligência artificial, referências de arquivos de efeito sonoro e outras configurações.

Mais Informações

Uma vez que os arquivos ODF são salvos em um formato de texto, eles podem ser abertos e editados com qualquer editor de texto. Os arquivos ODF são normalmente armazenados dentro do diretório Activision\Battlezone\addon. Eles são um dos vários outros arquivos que compõem um mapa de jogo.

NOTA: Battlezone foi lançado em 1998.

O que é um arquivo .odf? (Arquivo de documento aberto)

Arquivo genérico criado usando o padrão ODF (Open Document Format, formato de documento aberto) pode conter um dos vários tipos de dados de Documento Aberto, como dados de planilha, texto ou apresentação; salvo no Formato de Documento Aberto OASIS baseado em XML e suportado por vários aplicativos de produtividade.

Mais Informações

NOTA: Os arquivos ODF são mais comumente vistos com um tipo específico de extensão, como .ODT para documentos de processamento de texto, .ODS para planilhas ou .ODP para apresentações.

O que é um arquivo .odf? (Fórmula OpenDocument)

Um arquivo ODF é um documento criado pelo Apache OpenOffice Math, um programa usado para criar fórmulas matemáticas. Contém uma ou mais equações matemáticas e também pode conter macros. Os arquivos ODF são baseados no padrão OASIS OpenDocument.

Mais Informações

Captura de tela de um arquivo odf em OpenOffice Math 413

O arquivo ODF é o principal tipo de arquivo usado pela Math para salvar equações. O arquivo é mais comumente aberto em Matemática para criar ou editar fórmulas, mas também pode ser importado para outras aplicações, particularmente programas OpenOffice, como Calc ou Impress. Quando é importada para outra aplicação, a equação funciona como um objeto que pode ser colocado em um trabalho de pesquisa, apresentação, etc.

A matemática permite que você digite fórmulas em um arquivo ODF de três maneiras diferentes. Você pode adicionar marcação no editor de equações, selecionar um símbolo na caixa de ferramentas "Elementos" ou clicar com o botão direito do mouse no editor da equação e selecionar um símbolo no menu de contexto. Quando você terminar sua equação, você pode compartilhá-la por e-mail ou exportá-la como um arquivo .PDF.

NOTA: O Apache OpenOffice era anteriormente conhecido como OpenOfficeorg.
