O que é um arquivo .cxf? (Arquivo de formato de troca química)

Arquivo criado no Chemical Exchange Format (CXF), formato usado para armazenar dados moleculares; projetado para troca de dados químicos em formato padrão; menos comum do que outros formatos moleculares, como .CML e .MOL.

O que é um arquivo .cxf? (Coordena o formato de exportação)

Um arquivo CXF também pode ser um arquivo de dados salvo no Formato eXport coordenadas (CXF). É um arquivo de texto simples que armazena dados de imagem vetorial em uma lista estruturada. O formato CXF pode ser usado para armazenar dados GIS ou gráficos vetoriais, como desenhos.

Mais Informações

Arquivos CXF normalmente contêm coordenadas de mapa vetorial (VMAP), que representam localizações geográficas. Exemplos de objetos de mapa vetorial incluem dados de terreno, informações topográficas e rótulos de cidades, pontos turísticos e edifícios.

NOTA: A partir de 2020, o aplicativo CXeditor não está disponível para download público.

O que é um arquivo .cxf? (Arquivo de formato estendido cuttlefish)

Arquivo criado pela Cuttlefish, uma ferramenta de visualização de rede; salva os vértices, bordas, pesos e outros valores para um gráfico não direcionado ou direcionado; usado para pesquisa e visualização de rede.

Mais Informações

Cuttlefish possui vários algoritmos avançados de layout de rede que traçam automaticamente a visualização.

O que é um arquivo .cxf? (Arquivo de colagem picasa)

Arquivo criado pelo Picasa, editor gratuito de imagens e programa de compartilhamento de imagens desenvolvido pelo Google; criado junto com os caminhos para as imagens que foram usadas para criar a colagem, bem como as posições das imagens na colagem.

Mais Informações

Se você mover um arquivo JPG de colagem, você também deve mover o arquivo CRX associado ou a colagem não será reconstruída adequadamente no Picasa a partir do novo local. Portanto, certifique-se de copiar o arquivo CRX ao mover suas colagens.
