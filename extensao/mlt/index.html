O que é um arquivo .MLT? (Projeto Shotcut)

Um arquivo MLT é um projeto criado pela Meltytech Shotcut, uma editora de vídeo de código aberto. Ele contém configurações de projeto no formato XML, que inclui informações sobre mídia importada no projeto, mídia colocada na linha do tempo, efeitos aplicados à mídia e propriedades de vídeo e áudio. Os arquivos MLT não contêm vídeo, áudio ou imagens reais, apenas referências aos arquivos de mídia importados no projeto.

Captura de tela de um arquivo mlt em Meltytech Shotcut 19

Para criar um novo projeto no Shotcut, selecione File → New. Depois de modificar o projeto, o usuário pode salvar as informações em um arquivo MLT selecionando File → Save ou Save As.... Depois de salvar um projeto, o usuário pode fechar o projeto e reabri-lo mais tarde para mais edições.

Os usuários também podem exportar projetos Shotcut salvos em arquivos MLT para vários formatos de mídia, incluindo .MP4 e .WMV vídeo, áudio .MP3 e .AAC, e imagens de .JPG e .PNG. Para exportar um projeto para um formato diferente, selecione File → Export Video... ou Export Frame....

NOTA: Enquanto um arquivo de projeto MLT só deve ser aberto com o Shotcut, ele também pode ser aberto com um editor XML para visualizar as configurações no formato XML. No entanto, o arquivo MLT não deve ser modificado com um editor XML, uma vez que a edição do conteúdo provavelmente fará com que o arquivo se torne corrupto.
