O que é um arquivo .pex? (Arquivo de dados Merak Peep)

Armazena modelos de dados e relatórios criados com o Merak Peep, um programa de avaliação econômica do petróleo; usado para criar cenários de "E se", analisar riscos e sensibilidade ao mercado, prever níveis de produção e estimar reservas.

O que é um arquivo .pex? (ProBoard Executable File)

Programa que pode ser executado dentro do sistema de boletins do ProBoard (BBS); escrito em C/C++ usando o kit de desenvolvimento de software fornecido com o ProBoard; muitas vezes com menos de 50K de tamanho.

O que é um arquivo .pex? (Arquivo de exibição portátil do Editor Ocidental)

Documento criado pela West Publisher, um programa usado para capturar artefatos digitais de processos legais; armazena uma exposição para uso em uma audiência de tribunal; pode ser distribuído autônomo ou empacotado em um arquivo .PTZ para distribuição.
