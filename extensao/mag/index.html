O que é um arquivo .mag? (Arquivo de atalho de diagrama de acesso)

Atalho criado pelo Microsoft Access, um aplicativo RDBMS (Relational Database Management System) da área de trabalho; armazena um atalho para um diagrama ou gráfico criado usando um banco de dados access; usado para carregar diagramas no Access sem antes ter que abrir o arquivo do banco de dados.

Mais Informações

NOTA: Como os arquivos MAG contêm links para conteúdo ativo do banco de dados, eles são desconfiados por padrão em vários programas de colaboração de e-mail e arquivos, incluindo algumas versões do Microsoft Outlook e do servidor Microsoft SharePoint.

Desenvolvedor Open Circuit Design
Categoria Dados
Formato Binário 

O que é um arquivo .mag? (Arquivo de layout do circuito mágico)

Design de circuito criado pela Magic, uma ferramenta usada para criar layouts VLSI (Very Large Scale Integration, integração em larga escala) e contém um design de circuito, que incorpora ônibus de dados e componentes, bem como informações de verificação de design que garantem que o circuito seja projetado corretamente.

Mais Informações

A magia é comumente usada para fins educacionais nas áreas de Engenharia Elétrica e de Computação (ECE) e Ciência da Computação (CS).
