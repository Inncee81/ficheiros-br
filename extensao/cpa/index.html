O que é um arquivo .cpa? (Arquivo de arquivo CADSTAR PCB)

Arquivo CAD criado pelo CADSTAR, um aplicativo usado para criar projetos pcb (printd circuit board, placa de circuito impresso) ; pode conter um design PCB ou uma biblioteca de componentes PCB; usado para projetos de exportação, importação e arquivamento.

Mais Informações

Os arquivos CPA geralmente são exportados de arquivos .PCB criados no CADSTAR. Os arquivos CPA também podem ser importados e exportados usando o Altium Designer, outro aplicativo de design PCB.

Programas que abrem o arquivo .cpa (Arquivo de arquivo CADSTAR PCB)
