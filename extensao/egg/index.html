O que é um arquivo .egg? (Arquivo ALZip)

Um arquivo EGG é um arquivo comprimido criado pela ALZip, um programa de compressão e descompressão de arquivos. Ele contém um ou mais arquivos compactados no formato EGG proprietário. Os arquivos EGG podem incorporar criptografia AES-256 e proteção por senha para proteger o arquivo.

Mais Informações

Captura de tela de um arquivo de ovo em ESTSoft ALZip 8

Você pode descomprimir arquivos EGG usando ALZip e Bandizip em Windows. Você também pode usar o UnEgg em Linux.

O formato EGG incorpora técnicas inteligentes de compressão para criar tamanhos de arquivo menores. Ele também suporta Unicode e, portanto, é útil para arquivos multilíngues.

ALZip é uma ferramenta simples de compressão para Windows que é especialmente popular na Coreia. Possui dois formatos proprietários: o formato EGG, que foi lançado na versão 8 e o formato Zip. O programa também pode comprimir e descomprimir arquivos em outros formatos, como .TAR e .ZIP.

O que é um arquivo .egg? (Arquivo de música personalizado beat saber)

Um arquivo EGG é uma canção para um mapa personalizado usado por Beat Saber, um jogo de ritmo de realidade virtual (VR) onde um jogador corta blocos com sabres ao ritmo de uma música. É simplesmente um arquivo .OGG renomeado, que é um arquivo de áudio compactado que usa compactação de áudio Ogg Vorbis gratuita e não patenteada.

Mais Informações

Você provavelmente só encontrará arquivos EGG se jogar Beat Saber. Os jogadores normalmente vêem arquivos EGG se criam suas próprias músicas e mapas personalizados ou instalam mapas e músicas personalizadas de outros jogadores no Beat Saber.

EGG arquivos foram introduzidos no Beat Saber em 2019 como o formato de 20 músicas. Os arquivos acompanham mapas personalizados da dat Beat Saber, que contêm informações sobre a música e o conteúdo do mapa. Exemplos de conteúdo de mapa são a colocação de blocos (notas musicais) que um jogador deve cortar ao ritmo da música, efeitos de iluminação e obstáculos que um jogador precisa para evitar. O formato de 20 músicas substituiu o formato de 10 músicas, que consistia em arquivos de música OGG e .JSON mapas personalizados.

NOTA: O Beat Saber só está disponível para sistemas que suportam fones de ouvido VR, como Windows VR, Playstation VR e Oculus.

O que é um arquivo .egg? (Arquivo modelo Panda3D)

Um arquivo EGG é um modelo 3D ou animação usado pelo Panda3D, um motor 3D projetado principalmente para o desenvolvimento de jogos. Contém um modelo Panda3D, ator, animação ou ator e par de animação. Os arquivos EGG são criados exportando modelos e animações de outros programas de modelagem 3D, como Autodesk Maya ou Blender, e convertendo-os para uso em Panda3D.

Mais Informações

Captura de tela de um arquivo de ovo em PView

Em Panda 3D, o termo "model" refere-se a um modelo 3D estático, enquanto "ator" refere-se a um modelo 3D animado. Atores salvos em arquivos EGG são animados usando animações salvas em arquivos EGG separados. Embora os desenvolvedores possam salvar um ator e sua animação associada dentro do mesmo arquivo EGG, o Panda3D recomenda que eles não o façam, para evitar confusão durante a codificação.

Para converter modelos e animações 3D criados em outros aplicativos para arquivos EGG, os desenvolvedores devem usar um dos vários utilitários separados. Qual utilitário eles devem usar depende do programa que eles usaram para criar seu modelo ou animação.

NOTA: Depois que um desenvolvedor importa um arquivo EGG em Panda3D, ele é convertido em um arquivo BAM otimizado para desempenho para acesso mais rápido.

O que é um arquivo .egg? (Pacote de distribuição python)

Um arquivo EGG é um pacote comprimido por Zip que armazena uma distribuição Python. Ele contém todos os arquivos de origem para uma instalação Python, juntamente com metadados sobre a distribuição. Os arquivos EGG são simplesmente .ZIP arquivos renomeados com a extensão de arquivo .egg.

Mais Informações

O formato de distribuição egg é usado pelos desenvolvedores Python como uma alternativa a um Python Windows executável .EXE arquivo ou distribuição de origem. O formato é mais utilizado para distribuições mais antigas do Python desde que foi substituído pelo formato Wheel .WHL no início da década de 2010.

OS arquivos EGG podem ser abertos com python usando o pacote "SetupTools". Os arquivos também podem ser descomprimidos com um programa de descompressão zip, como o Microsoft File Explorer, Corel WinZip ou o Apple Archive Utility.
