O que é um arquivo .fm? (Documento do FrameMaker)

Um arquivo FM é um documento criado com o Adobe FrameMaker, um aplicativo usado para autor, enriquecer, gerenciar e publicar documentação técnica. Armazena informações sobre a estrutura de uma página que está programada para ser publicada para impressão ou web. Os arquivos FM podem armazenar páginas que devem ser publicadas ou publicadas individualmente como parte de um livro FrameMaker (.BOOK arquivo).

Mais Informações

Documentos do FrameMaker suportam textos, gráficos raster e gráficos vetoriais, incluindo imagens .SVG. Páginas podem ser salvas à medida que .PDF arquivos para melhor compatibilidade com outros sistemas.

NOTA: O FrameMaker só suporta documentos .FM criados no FrameMaker 3 ou posterior.
