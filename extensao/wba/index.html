O que é um arquivo .wba? (Pele Compactada windowblinds)

Pele personalizada ou tema criado com o WindowBlinds, um programa usado para personalizar a interface do sistema operacional Windows; pode alterar o visual de bordas de janelas, barras de título, barras de tarefa, animações de progresso, ícones de barra de ferramentas, o menu Iniciar e outros aspectos de Windows.

Mais Informações

Os arquivos WBA são basicamente compactados .ZIP arquivos que são descompactados quando abertos com windowblinds. Eles também podem ser descomprimidos usando outro programa Unzip como WinAce ou WinZip se a extensão do arquivo for alterada de "wba" para "zip"
