O que é um arquivo .cdt? (Arquivo de texto de CD)

Arquivo de metadados de disco de áudio criado por aplicativos de rasgar, escrever e gravar discos; contém dados de CD-Text, que podem descrever o nome do álbum e informações da música; muitas vezes salvos com um arquivo .CUE e .BIN que juntos compõem os dados de uma imagem de disco de áudio.

Mais Informações

CD-Text faz parte do padrão de CD red book, que foi desenvolvido pela Sony e Philips.

NOTA: O reprodução de mídia VideoLAN VLC pode abrir arquivos de CD-Text em plataformas Windows e Linux com suporte da biblioteca libcdio.

O que é um arquivo .cdt? (Modelo ConceptDraw PRO)

Modelo de diagrama criado pela ConceptDraw PRO, um aplicativo técnico de diagramas; armazena formas, linhas, texto e símbolos predefinidos que podem ser usados para criar novos diagramas de .CDD; podem ser projetados pelo usuário ou incluídos na instalação do software.

O que é um arquivo .cdt? (CorelDRAW Modelo de imagem)

Um arquivo CDT é um modelo de imagem criado pelo CorelDRAW, um programa de edição de imagens e layout de página. Ele contém propriedades para uma nova imagem, incluindo configurações de preenchimento gráfico e objeto e propriedades de página, régua, grade, diretriz e estilo. Os arquivos CDT são usados para criar documentos como cartas, cartões postais, imagens, ícones, páginas da Web, banners da Web e folhetos.

Mais Informações

Captura de tela de um arquivo CDT no CorelDRAW 2020

Os arquivos CDT são usados para armazenar configurações padrão para criar vários documentos ou imagens .CDR. Você pode criar um modelo a partir de um arquivo CDR ao salvá-lo como um modelo selecionando File → Save as Template.... Você também pode editar um dos muitos modelos pré-fabricados do Corel disponíveis online. Você pode acessar esses modelos diretamente do aplicativo selecionando File - > New from Template....

Você pode exportar o arquivo CDT no CorelDRAW para outros formatos mais populares. Dependendo de suas necessidades, você pode exportar o arquivo CDT para um documento de .PDF ou como uma imagem .JPG, .PNG ou .BMP. Você também pode exportar arquivos CDT para o formato .AI Adobe Illustrator.

CorelDRAW está incluído no CorelDRAW Graphics Suite juntamente com os seguintes aplicativos:

  • Corel PHOTO-PAINT - Uma edição de imagem aplicativo de fotos digitais
  • Corel Font Manager - Um utilitário para gerenciamento de fontes
  • CorelDRAWapp - Um programa de ilustração de vetores on-line acessível via navegador da Web
  • Corel CAPTURE - Um utilitário de captura de tela de um único clique (disponível apenas com Windows de)

NOTA: Os modelos de CDT foram substituídos por arquivos .CDRT em CorelDRAW.
