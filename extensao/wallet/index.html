O que é um arquivo .wallet? (Arquivo de carteira multibit)

Um arquivo WALLET é um arquivo criado pela Multibit, uma carteira internacional de Bitcoin usada para realizar transações de Bitcoin. Ele contém transações passadas e chaves privadas usadas para ativar transações BitCoin.

Mais Informações

Para criar um arquivo WALLET no Multibit, selecione File → New Wallet, nomeie seu arquivo, escolha um local de salvamento, selecione "MultiBit (*carteira)" no menu "Formato de arquivo" e clique em Save.

Para abrir um arquivo WALLET no Multibit, basta clicar duas vezes no arquivo ou selecionar File → Open Wallet, navegar e selecionar seu arquivo WALLET e clicar em Abrir.

O que é um arquivo .wallet? (Arquivo de carteira MultiDoge)

Um arquivo WALLET é um arquivo criado pela MultiDoge, uma porta do cliente MultiBit para Bitcoin usada para realizar transações de Bitcoin. Ele contém uma ou mais chaves privadas, que são sequências de caracteres usados para verificar a identidade do usuário e ativar transações BitCoin.

Mais Informações

Para criar um arquivo WALLET no MultiDoge, clique no botão New Wallet ou selecione File → New Wallet, nomeie seu arquivo, escolha um local de salvamento e clique em Ok.

Para abrir um arquivo WALLET no MultiDoge, selecione File → Open Wallet, navegue e selecione seu arquivo WALLET e clique em Abra.

O que é um arquivo .wallet? (Wallet Ransomware)

Um arquivo WALLET é um arquivo criptografado pelo criptomix, ou CrypMix, vírus, que é ransomware utilizado por cibercriminosos. Ele contém um arquivo do usuário, como um arquivo .PDF ou .DOCX, criptografado com criptografia AES pelo vírus. Os arquivos WALLET são semelhantes aos arquivos .LOCKY, mas se tornaram prevalentes em 2017.

Mais Informações

O vírus CryptoMix também pode ser referido como um vírus CrypMix, CryptFile2 ou Dharma. É um malware perigoso que é comumente conhecido como ransomware, onde o objetivo do vírus é tomar seus arquivos como reféns e forçá-lo a pagar o criminoso (por meio de bitcoin) para desbloquear seus arquivos.

Uma vez que o vírus se infiltra no computador, ele começa a embaralhar seus arquivos, renomeá-los e criptografá-los. O vírus então gera uma nota de resgate .TXT (#_RESTORING_FILES_#.TXT) informando sobre a aquisição e o que você precisa fazer para recuperar seus arquivos.

O vírus anexa a extensão da carteira de ID [email@email.com].ID[victim_16_character_ID].wallet nos seus arquivos. Por exemplo, se você tiver um arquivo sample.docx, ele se torna sample.docx.webmafia@asia.com.ID0SHDGS2SJD5DFOCK.wallet. O endereço [email@emailcom] é provavelmente um dos seguintes endereços:

  • braker@plague.life
  • crannbest@foxmail.com
  • stopper@india.com
  • amagnus@india.com
  • funa@india.com
  • webmafia@asia.com
  • bitcoin143@india.com
  • mkgoro@india.com
  • lavandos@dr.com
  • worm01@india.com
  • pay4help@india.com

NOTA: Não há nenhum programa disponível atualmente para restaurar efetivamente seus arquivos. Se você fizer backup de seus arquivos, você pode executar uma restauração completa do sistema.
