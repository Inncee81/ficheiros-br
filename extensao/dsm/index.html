O que é um arquivo .dsm? (Módulo de Som Digital)

Formato do módulo tracker usado por programas de rastreamento de áudio digital; inclui várias faixas com notas que compõem uma partitura musical; as notas estão ligadas a amostras de som, que também estão incluídas no arquivo.

Programas que abrem o arquivo .dsm (Módulo de Som Digital)

  XMMS
  Mac
  CocoModX
  Windows
  MODPlug Player
  Media Player Classic (MPC) 
  Nullsoft Winamp
