O que é um arquivo .epp? (Arquivo de imagem Canon Easy-PhotoPrint)

Arquivo de imagem usado pela Canon Easy-PhotoPrint EX, um programa de impressão de fotos digital às vezes incluído nas compras da impressora Canon; armazena uma imagem em um formato proprietário e pode incluir alterações como correção de olho vermelho.

Mais Informações

Os arquivos EPP são usados para salvar alterações em fotos digitais antes de imprimi-las. Eles só podem ser abertos pelo software Canon Easy-PhotoPrint.

O que é um arquivo .epp? (Lista de reprodução do Elmedia Player)

Playlist criada e usada pelo Elmedia Player, um media player do OS X; contém uma lista de faixas usadas para compartilhar uma lista de reprodução ou criar um backup.

Mais Informações

Você pode adicionar e remover faixas à sua lista de reprodução no Elmedia Player. Para criar o arquivo EPP na sua lista de reprodução, destaque sua lista de reprodução, selecione File → Export Playlist..., nomeie o arquivo, escolha o local de salvamento e clique em Save.

Para importar um arquivo EPP, selecione File → Import Playlist..., navegue até o seu arquivo e clique em Abra.
