O que é um arquivo .drl? (Arquivo de rack de broca gerber)

Contém dados de perfuração para a fabricação de uma placa de circuito impresso (PCB) com base no padrão de especificações gerber; inclui dados sobre as ferramentas de perfuração e tamanhos de perfuração usados na criação da placa.

Mais Informações

Os arquivos do rack de perfuração Gerber também são chamados "Arquivos de perfuração excellon"

NOTA: Os principais dados de design e fabricação do PCB são armazenados em um arquivo .GBR.
