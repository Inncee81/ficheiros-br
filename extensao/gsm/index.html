O que é um arquivo .gsm? (Arquivo de projeto mapviewer)

Projeto de visualização de mapas criado com o MapViewer, um programa de análise espacial usado para criar mapas geográficos; salva configurações de visualização (rótulos de texto, cores, uma legenda) bem como referências a arquivos de dados de origem que são usados para gerar o mapa.

Mais Informações

Mapasvisores são usados para sobrepor dados específicos do usuário em áreas geográficas. Por exemplo, os usuários podem gerar mapas climáticos, mapas de densidade populacional ou muitos outros tipos de mapas usando outros conjuntos de dados. Visualizações completas podem ser exportadas para formatos gráficos raster para publicação e apresentação.

O que é um arquivo .gsm? (Arquivo de áudio GSM de robótica dos EUA)

Formato de áudio usado para transferir voz sobre um modem; usado pelos modelos GSM QuickLink, VoiceGuide e RapidComm; às vezes usado para faxes de áudio.
