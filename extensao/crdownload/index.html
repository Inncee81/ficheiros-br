O que é um arquivo .crdownload? (Arquivo parcialmente baixado do Chrome)

Um arquivo CRDOWNLOAD é um arquivo parcialmente baixado criado pelos navegadores Web do Google Chrome, Microsoft Edge e Chromium. Ele armazena o conteúdo de um arquivo como está sendo recebido pelo navegador e usa a extensão .crdownload para marcar o download como incompleto até que ele seja concluído, momento em que a extensão .crdownload é removida. Os arquivos CRDOWNLOAD são semelhantes aos arquivos .PART, que são usados pelo navegador Mozilla Firefox para downloads parciais.

Mais Informações

Você provavelmente só encontrará um arquivo CRDOWNLOAD se usar Chrome, Edge ou Chromium para navegar na Internet e baixar arquivos. Esses navegadores criam um arquivo CRDOWNLOAD na pasta "Downloads" no seu computador quando você começa a baixar um arquivo. Você pode ver o arquivo CRDOWNLOAD se o arquivo está baixando no momento ou se o download foi interrompido antes da conclusão. Uma interrupção de download é muitas vezes devido a uma conexão de Internet perdida ou devido a você pausar o download.

À medida que o Chrome baixa o arquivo, um arquivo CRDOWNLOAD aparece na pasta "Downloads" como Unconfirmed [set of numbers].crdownload. Por exemplo, o arquivo myDoc.docx apareceria como Unconfirmed 47128.crdownload durante o download, mas depois seria alterado para myDoc.docx uma vez que o download esteja concluído.

O Chrome também pode anexar a extensão .crdownload aos arquivos à medida que são baixados. Por exemplo, o arquivo myMovie.avi teria o nome myMovie.avi.crdownload quando ele está sendo baixado. Uma vez que o download esteja concluído, o Chrome remove a extensão .crdownload e o arquivo está pronto para ser usado.
