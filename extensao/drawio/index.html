O que é um arquivo .drawio? (Arquivo diagrama diagrama de diagrama de diagrama)

Um arquivo DRAWIO contém um diagrama criado pela diagrama drawio, um programa de diagramação de código aberto. Ele armazena o conteúdo do diagrama, que pode incluir texto, formas, ícones e imagens, juntamente com informações de formatação. Os arquivos DRAWIO são usados para salvar vários tipos de diagramas, como gráficos de organização, diagramas de processo e fluxogramas.

Você pode encontrar arquivos DRAWIO se você usar a ferramenta drawio para criar diagramas ou você pode receber um arquivo DRAWIO de um colega de trabalho ou amigo utilizando o programa. drawio está disponível on-line para uso em vários navegadores da Web, como Chrome, Edge, Safari e Firefox. Ele também pode ser baixado e instalado em Windows, macOS e Linux sistemas.

Nas versões desktop e online do drawio, selecione File → Open... para abrir o arquivo DRAWIO. Se você estiver usando a versão online, você precisará carregar o arquivo DRAWIO no prompt "Open" se ele for salvo no seu computador.

Para criar um diagrama, selecione File → New... e selecione File → Save ou Save as.... Se você estiver usando a versão web, você pode salvar o diagrama no Google Drive, OneDrive ou no armazenamento local em seu computador. Clique em Baixe no prompt "Salvar como" para baixar o arquivo DRAWIO para o seu computador.

NOTA: Para exportar um arquivo DRAWIO para outro formato, como .JPG, .PNG, .PDF ou .VSDX, selecione File → Export as em drawio.
