O que é um arquivo .mba? (MailNavigator Mail Box Archive)

Um arquivo de MBA é um arquivo de dados criado pelo MailNavigator, um cliente de e-mail e notícias para Windows. Ele contém todas as informações sobre uma caixa de correio, que podem incluir mensagens enviadas e recebidas, configurações de filtro de mensagens e contatos de catálogo de endereços. Os arquivos de MBA são usados para armazenar um backup de caixas de correio que podem ser compartilhadas com outros usuários do MailNavigator ou usadas para restaurar caixas de correio corruptas.

Mais Informações

Você provavelmente só encontrará um arquivo de MBA do MailNavigator se você usar o programa MailNavigator para enviar e receber e-mails ou gerenciar informações pessoais. Para abrir um arquivo MBA no MailNavigator, selecione File → Load External Mailbox..., selecione "Arquivos MailNavigator (*mba)" do menu suspenso "Arquivos do tipo".

O que é um arquivo .mba? (Arquivo do Boletim Martus)

Um arquivo de MBA é um arquivo criado pela Benetech Martus, uma ferramenta de código aberto usada por ativistas de direitos humanos para coletar e gerenciar dados confidenciais sobre pessoas com segurança. Armazena registros de dados criptografados sobre uma ou mais pessoas exportadas da Martus.

Mais Informações

Para exportar registros criptografados da Martus em um arquivo MBA, selecione os registros a serem exportados e clique no ícone "Exportar"," na barra de tarefas. Para exportar registros não criptografados do Martus em um arquivo .XML, selecione os registros que deseja exportar, clique no ícone "Exportar" na barra de tarefas e desmarque "Criptografado" Para importar registros criptografados armazenados em um arquivo MBA, clique no ícone "Importar" na barra de tarefas.

Martus é projetado especificamente para profissionais, como jornalistas e ativistas de direitos humanos, que trabalham com pessoas vulneráveis, como refugiados ou vítimas de abuso. A Martus permite que os usuários insiram informações sobre uma pessoa e, em seguida, criptografa automaticamente os dados inseridos e os envia para uma rede de servidores protegidos. Os usuários do Martus podem então visualizar, analisar, relatar e visualizar os dados dentro do aplicativo. Os dados em arquivos de MBA só podem ser exportados e importados pela Martus.

O que é um arquivo .mba? (Arquivo de atividade do MyBoard)

Um arquivo de MBA contém uma atividade criada pela Inclusive Technology MyBoard, uma ferramenta educacional interativa para salas de aula. Ele armazena informações sobre uma atividade de aprendizagem, como um livro de colorir, jogo de alfabetização, esboço de traçado ponto a ponto ou jogo de tabuleiro. Os arquivos de MBA são compactados com compactação zip.

Mais Informações

Você provavelmente só encontrará um arquivo myboard activity MBA se você ensinar em uma sala de aula formal e usar o software MyBoard. Você pode criar uma atividade do zero no MyBoard e salvá-la como um arquivo de MBA, baixar um arquivo de MBA da Internet ou receber um arquivo de MBA de outro professor por e-mail.

Como os arquivos de MBA são compactados por Zip, você pode renomear a extensão de arquivo .mba para .zip e, em seguida, extrair o conteúdo no arquivo .ZIP. Os conteúdos no arquivo normalmente incluem .XML, .JPG e .SWF arquivos que compõem a atividade.
