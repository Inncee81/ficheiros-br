O que é um arquivo .wtv? (arquivo de Windows série de TV gravada)

Arquivo de vídeo gravado pelo Windows Media Center, um programa usado para reproduzir conteúdo multimídia e programas de TV de gravação; codifica vídeo usando compactação MPEG-2 e áudio usando compactação MPEG-1 Layer II ou Dolby Digital AC-3; também pode incluir informações dedescritivas de metadata e gerenciamento de direitos digitais (DRM).

Mais Informações

Os arquivos da WTV são criados pelo Stream Buffer Engine (SBE) quando Windows Media Center grava um programa de televisão. Eles são salvos no diretório \Users\public\Recorded TV\ por padrão. Shows gravados podem ser abertos e reproduzidos dentro Windows Media Center.

O formato de arquivo WTV substituiu o formato de arquivo .DVR-MS anterior na versão Windows 7 do Media Center Windows. Os arquivos WTV podem ser convertidos de volta em arquivos DVR-MS clicando-os com o botão direito do mouse Windows Media Center e selecionando a opção "Convert to dvr-ms Format".
