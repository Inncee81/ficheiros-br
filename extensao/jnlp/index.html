O que é um arquivo .jnlp? (Arquivo iniciar Web Java)

Um arquivo JNLP é criado no formato Java Network Launching Protocol (JNLP). Ele contém informações como o endereço remoto para baixar um programa Java (classe a ser executada. Os arquivos JNLP são usados para lançar e gerenciar programas Java em uma rede ou na web.

Mais Informações

Os arquivos JNLP são salvos em um formato XML e podem ser visualizados ou editados com um editor de texto. Eles podem ser clicados duas vezes para executar o programa se o Java Runtime Environment (JRE) estiver instalado. O JRE inclui a tecnologia Java Web Start, que abre arquivos JNLP e executa as instruções especificadas.

NOTA: A Oracle preteriu o Java Web Iniciar na Plataforma Java Standard Edition (JSE) 9 e removeu-a completamente em Java SE 11. No entanto, o OpenWebStart foi desenvolvido como uma implementação de código aberto para permitir que os usuários ainda utilizem aplicativos Java Web Start e JNLP.
