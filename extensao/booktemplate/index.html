O que é um arquivo .booktemplate? (Modelo BookSmart)

Um arquivo BOOKTEMPLATE é um modelo de livro criado pela BookSmart, uma ferramenta gratuita usada para criar livros de fotos, cadernos e planejadores. Ele contém um livro, incluindo as páginas, texto, imagens e configurações de layout, que foi projetado para uso repetido.

Mais Informações

Captura de tela de um arquivo booktemplate em Blurb BookSmart 34

O arquivo BOOKTEMPLATE é semelhante ao arquivo .BOOK, uma vez que é criado para armazenar o conteúdo de um projeto. O arquivo pode ser criado a partir de uma variedade de layouts de página, como álbum de casamento, livro de receitas, livro de visitas, notebook e anuário. Você também pode escolher entre vários temas que incluem bordas de fotos e fundos de página com cores coordenadas.

A diferença entre os arquivos BOOKTEMPLATE e BOOK é que os modelos são projetados especificamente para projetos que você deseja usar várias vezes. O BookSmart pode ser usado para criar e abrir arquivos BOOKTEMPLATE:

Para criar um arquivo BOOKTEMPLATE, selecione File → Export → Export Template....

Para abrir um arquivo BOOKTEMPLATE, selecione File → New → Project from Template....

O que é um arquivo .booktemplate? (Modelo de documento do autor do iBooks)

Modelo de design de livros criado pelo iBooks Author, um aplicativo de autoria de livros digitais; inclui predefinições de design para o layout da página, configurações de texto e outras propriedades do livro; usado como ponto de partida para um projeto de livro (arquivo .BOOK).

Mais Informações

Os modelos de documentos usam com mais frequência a extensão .IBATEMPLATE. A instalação do autor do iBooks inclui vários modelos que usam a extensão "ibatemplate", não a extensão "booktemplate".
