O que é um arquivo .u8? (Mario Kart Wii Archive)

Um arquivo U8 é um formato de arquivo usado por Mario Kart Wii, um jogo de corrida de kart com tema de Mario. Ele armazena arquivos de dados de jogos, como um arquivo de imagem Wii .TPL ou .BRLYT de texturas e listas de material wii. Os arquivos U8 são frequentemente modificados pelas comunidades homebrew e modding para criar gráficos personalizados de Mario Kart.

Mais Informações

Os arquivos U8 são salvos no formato de arquivo U8 e são compactados com o método de compactação Yaz0. Além dos arquivos U8, o formato U8 é normalmente armazenado como .SZS arquivos.
