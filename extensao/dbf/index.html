O que é um arquivo .dbf? (Arquivo de banco de dados)

Um arquivo DBF é um arquivo padrão de banco de dados usado pelo dBASE, um aplicativo do sistema de gerenciamento de banco de dados. Ele organiza dados em vários registros com campos armazenados em um tipo de dados de matriz. Os arquivos DBF também são compatíveis com outros programas de banco de dados "xBase", que surgiram devido à popularidade do formato de arquivo.

Mais Informações

Devido à adoção precoce na comunidade de banco de dados e uma estrutura de arquivos relativamente simples, os arquivos DBF têm sido amplamente aceitos como um formato padrão de armazenamento para dados estruturados em aplicativos comerciais.
