O que é um arquivo .evr? (Arquivo de áudio de taxa variável aprimorada)

Arquivo de som comprimido usando o Código de Taxa Variável Aprimorada (EVRC); projetado principalmente para comprimir a fala com uma taxa de bits baixa, mantendo alta fidelidade.

Mais Informações

O codec EVR pode ser aplicado a vários tipos de arquivos de áudio, incluindo o formato de contêiner 3GPP2 (.3G2). Ele é usado por alguns dispositivos de hardware da Qualcomm e pode ser reproduzido usando o software RealPlayer.
