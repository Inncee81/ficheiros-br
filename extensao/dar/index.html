O que é um arquivo .DAR? (Arquivo de disco DAR)

Um arquivo DAR é um arquivo de um disco ou grupo de arquivos compactados com compactação DAR (Disk ARchive). Ele é usado para criar arquivos divididos e backups incrementais. Os arquivos DAR são projetados como um substituto para o formato de arquivo .TAR.

O que é um arquivo .DAR? (Projeto arquiteto de DVD)

Um arquivo DAR é um projeto usado pelo DVD Architect, um programa para autoria de DVDs. Ele contém vários componentes de um projeto de DVD, incluindo localizações de arquivos de origem, pontos de inserção, texto e capítulos. Os arquivos DAR não contêm as imagens reais, vídeo e áudio exibidos no projeto, mas são usados para criar o produto final quando o usuário é feito a edição.

Mais Informações

Você pode criar um arquivo de projeto DAR selecionando File → New, em seguida, escolha o tipo de projeto que deseja fazer. Os diferentes tipos incluem:

  • Menu Baseado em menu - Produz um DVD que exibe um menu quando reproduzido
  • Reprodução de música/vídeo - Produz um DVD que contém uma compilação de arquivos de vídeo e áudio
  • Picture Compil Divulgação - Produz um DVD que exibe uma apresentação de slides de imagens
  • Single Movie - Produz um DVD que contém um único filme sem menus

>Ao escolher o projeto, você também pode escolher as configurações para o formato do disco, formato de vídeo, e formato de áudio.
