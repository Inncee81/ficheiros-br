O que é um arquivo .dwf? (Arquivo de formato de Web design)

Um arquivo DWF é um desenho 2D/3D salvo no Formato Web de Design (DWF) desenvolvido pela Autodesk. Contém dados de design, que incluem gráficos e texto. Os arquivos DWF são salvos em um formato compactado por Zip para reduzir o tamanho do arquivo.

Mais Informações

O Design Web Format foi desenvolvido como um formato de desenho independente e independente de resolução para distribuição de designs. Não se destina a substituir formatos CAD populares, mas, em vez disso, é usado para salvar designs em um formato mais acessível e aberto que pode ser aberto sem a necessidade de comprar software CAD caro. O formato é baseado no formato HOOPS Stream (HSF) 2D/3D, mas também é comprimido por Zip.
