O que é um arquivo .cel? (Arquivo de resultados da sonda Affymetrix)

Um arquivo CEL é um arquivo de dados criado pelo software de análise de imagens de microarray de DNA da Affymetrix. Ele contém os dados extraídos de "probes" em um Affymetrix GeneChip e pode armazenar milhares de pontos de dados, o que pode torná-lo grande em tamanho de arquivo. Os arquivos CEL podem ser processados por algoritmos de software e visualizados em uma grade 2D como parte de um experimento geral de genoma.

Mais Informações

Um Affymetrix GeneChip é um chip biológico que contém uma microarray que é válida para um experimento. Para criar esses chips, um slide de vidro ou silício é equipado com sondas, que relatam níveis de expressão (intensidades) como valores perfeitos de correspondência (PM) e incompatibilidade (MM), com base em se são complementares à amostra de DNA original. Esses valores podem ser usados para estudar alterações de DNA, chamados polimorfismos de nucleotídeos únicos (SNPs), que são usados em pesquisas de doenças humanas e animais.

O formato de arquivo CEL tem várias versões, que usam formatação diferente. Por exemplo, a versão 3 usa formatação de texto ASCII, enquanto a versão 4 usa um formato binário.

NOTA: Os arquivos CEL exigem um arquivo .CDF correspondente, que é o dicionário para os dados brutos de nível de teste armazenados no arquivo CEL. O MATLAB tem uma função incorporada chamada affyread que pode ser usada para ler arquivos CEL na versão Windows do software.

O que é um arquivo .cel? (Loop de Audição)

Arquivo de áudio usado pela Adobe Audition, um aplicativo de produção de áudio; salva um loop de áudio que estava originalmente no formato .MP3, mas foi modificado pelo software, por isso é adequado para looping.

Mais Informações

A Adobe Audition inclui vários loops gratuitos de royalties com a instalação do software.

Desenvolvedor Celestia Development Team

O que é um arquivo .cel? (Arquivo de script celestia)

Script usado pela Celestia, um aplicativo de navegação espacial gratuita; escrito usando uma sintaxe celestia específica e inclui comandos para mover a perspectiva de visualização em todo o universo; usado para criar passeios no universo 3D.

Mais Informações

Arquivos CEL também podem incluir recursos mais avançados, como instruções para carregar modelos 3D personalizados no passeio espacial. Eles também podem incluir legendas para fornecer fatos interessantes ao observador.

Se você baixar ou receber um arquivo CEL, você pode abri-lo clicando duas vezes no arquivo ou selecionando File → Run Script... no menu do aplicativo e, em seguida, escolhendo o arquivo CEL.

O formato CEL foi substituído pelo formato .CELX nas versões recentes da Celestia. No entanto, celestia ainda suporta arquivos CEL. O mais novo formato CELX usa a linguagem de script lua, que é mais expressiva e poderosa que a linguagem CEL.

NOTA: Os arquivos CEL são salvos como arquivos de texto simples e são editados usando um editor de texto.

Animação criada no formato FLIC; armazena uma série de quadros, mas não armazena partes de quadros anteriores que não são alterados; mais comumente usa as extensões .FLC ou .FLI.

Desenvolvedor Bentley Systems
Categoria CAD
Formato Binário 

O que é um arquivo .cel? (Biblioteca de células de microestação)

Biblioteca celular criada pela MicroStation, um programa CAD usado para criar rascunhos de design 2D e 3D; armazena a geometria, símbolos e outras informações de rascunho para um objeto; usado para armazenar os blocos de construção para gerar projetos de maior ordem.

Mais Informações

Os arquivos CEL podem ser exportados para formatos .DWG e .DXF usando o MicroStation.
