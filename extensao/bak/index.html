O que é um arquivo .BAK? (ACT! Arquivo de backup)

Arquivo de backup criado pelo ACT!, um aplicativo usado para gerenciamento de relacionamento com o cliente (CRM); contém um backup de dados de CRM, que inclui informações do cliente, como nomes, informações de endereço e histórico.

O que é um arquivo .BAK? (Arquivo de backup)

Um arquivo BAK é um backup de outro documento ou arquivo, comumente criado automaticamente por programas de software ou pelo sistema operacional. Ele normalmente contém uma cópia do arquivo original e pode ser restaurado ao original substituindo a extensão .bak pela extensão original.

Mais Informações

Muitas vezes os arquivos BAK são criados simplesmente anexando .bak ao nome de arquivo original. Por exemplo, document.doc pode ter um arquivo de backup document.doc.bak.

NOTA: Um programa de exemplo que usa a extensão "bak" é o AutoCAD. Windows também pode criar arquivos BAK para os arquivos System.ini e Win.ini.

O que é um arquivo .BAK? (Arquivo de backup de marcadores)

Um arquivo BAK é um backup do arquivo de marcadores Google Chrome ou Mozilla Firefox, que pode ser usado para substituir marcadores salvos no caso de o arquivo principal de marcadores ficar corrompido. Ele é gerado automaticamente pelo navegador Web Chrome ou Firefox e é tipicamente chamado de bookmarks.bak.

Mais Informações

Se o Chrome criar um arquivo bookmarks.bak, ele está localizado nos seguintes diretórios com um arquivo de texto Bookmarks em Windows e macOS:

  • Windows:<<C:Users\[username]\AppData\Local\Google\Chrome\User Data\Default
  • macOS:/Users/[username]/Library/Application Support/Google/Chrome/Default

Se o Firefox criar um arquivo de bookmarks.bak, ele está localizado nos seguintes diretórios com um arquivo de texto Bookmarks em Windows e macOS:

  • Windows:C:\Documents and Settings\[username]\Application Data\Mozilla\Firefox\Profiles\[random-name]
  • macOS:/Users/[username]/Library/Application Support/Firefox/Profiles/[random-name].default

O que é um arquivo .BAK? (Arquivo de backup finale)

Arquivo criado pelo Finale, um aplicativo de notação de música para tocar instrumentos, vocais, etc; substituído pelo arquivo .BAKX, que foi lançado no Finale 2014; contém dados de backup para .MUS arquivos; criado quando os arquivos MUS são salvos pelo usuário.

Mais Informações

Para criar um arquivo BAK:

  1. Você deve primeiro ativar a opção de criação de backup.
  2. Selecione Finale [version] → Preferences....
  3. Selecione "Salvar e imprimir" na lista à esquerda da janela "Preferências", verifique a caixa de seleção "Faça backups ao salvar arquivos" e clique em OK.
  4. Agora, quando você salvar seu arquivo MUS, um arquivo BAK será criado.

O que é um arquivo .BAK? (Arquivo de backup MobileTrans)

Um arquivo BAK é um arquivo de backup criado pela Wondershare MobileTrans, um programa usado para backup e transferência de dados de e para dispositivos. Ele contém dados backup de um dispositivo, como contatos, mensagens, informações de calendário, registros de chamadas, fotos, música, vídeos, aplicativos e dados do aplicativo.

Mais Informações

Arquivos BAK são o principal tipo de arquivo associado ao MobileTrans. O arquivo é criado ao fazer backup de um telefone para um computador. Para fazer isso no MobileTrans, clique em "Faça backup do seu telefone" na página inicial do programa, selecione o conteúdo para fazer backup e clique em "Iniciar Transfer". O que é um arquivo .BAK? (Arquivo de backup Terraria)

Um arquivo BAK é um arquivo de backup criado pela Terraria, um jogo de construção de ação 2D e sandbox. Ele contém uma cópia de um mundo (arquivo .WLD) ou um jogador (arquivo .PLR) em Terraria. Os arquivos BAK são usados para armazenar um backup de um WLD ou PLR no caso de o arquivo ficar corrompido ou perdido.

Mais Informações

Arquivos BAK criados pela Terraria são simplesmente arquivos WLD ou PLR, mas com a extensão de arquivo .bak adicionada ao arquivo. Por exemplo, um backup do arquivo example.wld aparece como example.wld.bak. Se você precisar abrir o arquivo de backup no Terraria, remova a extensão .bak e coloque o arquivo na pasta apropriada.

Os arquivos WLD devem ser colocados no seguinte diretório:

[user]\Documents\My Games\Terraria\Worlds\

Os arquivos PLR devem ser colocados no seguinte diretório:

[user]\Documents\My Games\Terraria\Players\
