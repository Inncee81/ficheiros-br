O que é um arquivo .inca? (Arquivo de atribuição incopy CS3)

Um arquivo INCA contém visualizações de elementos de página usadas pelo InCopy, um processador de texto que se integra ao programa de layout de página do Adobe InDesign. Ele armazena visualizações de elementos de página a partir de um subconjunto de um documento Dodesign (arquivo .INDD), bem como permissões de acesso para modificar arquivos de conteúdo vinculados .INCX associados. Os arquivos INCA são usados para dividir o trabalho necessário para a edição de uma publicação.

Mais Informações

Os arquivos de atribuição são criados usando o InDesign e, em seguida, atribuídos a um editor específico, que tem permissão para editar conteúdo de página usando o InCopy. Uma vez que a página esteja completa, o editor notifica o remetente. Esse processo é útil de várias maneiras:

  1. Dividindo tarefas de edição entre vários editores.
  2. Impedindo alterações indesejadas em partes protegidas de documentos do InDesign, como o layout mestre.
  3. Reduzindo o tamanho do arquivo e do projeto enviados aos editores.
  4. Usando um sistema de check-in/check-out para que apenas um trabalhador tenha acesso ao conteúdo da página por vez.
  5. Rastreamento de quais editores modificaram quais partes de um documento.
Os arquivos INCA são exportados com arquivos de conteúdo vinculados ao INCX, que contêm o texto, gráficos e quadros a serem modificados. Os editores normalmente acessam arquivos INCA e INCX em uma rede.

NOTA: Os arquivos INCA foram substituídos por arquivos .ICMA no InDesign e no InCopy CS4 e posteriormente.
