O que é um arquivo .pgm? (Arquivo do programa Akai MPC)

Um arquivo PGM contém um programa usado por vários dispositivos MPC (Music Production Controller, controlador de produção de música), como MPC1000, MPC2000 e MPC3000. Ele armazena informações de configuração sobre as almofadas (botões) no dispositivo MPC e as amostras mono ou estéreo que são designadas para reproduzir quando tocados no dispositivo MPC. Os arquivos PGM também incluem parâmetros para o programa, como atribuições de som, configurações SIMULT e parâmetros de ajuste.

Mais Informações

Os arquivos PGM podem ser salvos no formato padrão do programa MPC ou no formato revisado que foi desenvolvido para suportar dispositivos MPC mais complexos. Os MPCs fabricados antes do dispositivo MPC1000 não suportam o formato PGM mais novo. No entanto, os MPCs fabricados após o MPC1000 suportam ambas as versões do formato PGM.

Enquanto os arquivos PGM foram projetados para serem carregados em dispositivos MPC, eles também podem ser carregados por software MPC desenvolvido para imitar dispositivos MPC em computadores. Alguns exemplos de programas que podem abrir arquivos PGM são 2KXL, Blue Box e MPC Editor.

O que é um arquivo .pgm? (Imagem do mapa cinza portátil)

Um arquivo PGM é um arquivo de imagem em escala de cinza salvo no formato pgm (mapa cinza portátil) e codificado com um ou dois bytes (8 ou 16 bits) por pixel. Contém informações de cabeçalho e uma grade de números que representam diferentes tons de cinza de preto (0) a branco (até 65.536). Os arquivos PGM são normalmente armazenados no formato de texto ASCII, mas também têm uma representação binária.

Mais Informações

Os arquivos PGM incluem um cabeçalho que define o tipo de formato PGM ("P2" para texto ou "P5" para binário), largura e altura de imagem e o número máximo de tons. Embora os arquivos PGM binários possam conter várias imagens, os arquivos ASCII PGM podem incluir apenas uma imagem.

O formato PGM é um dos vários formatos de imagem definidos pelo projeto Netpbm, que é um pacote de código aberto de programas gráficos. Outros formatos incluem o formato de bitmap portátil (.PBM) e o formato pixmap portátil (.PPM).

NOTA: Os arquivos PGM também podem ser chamados de arquivos "greymap" portáteis.
