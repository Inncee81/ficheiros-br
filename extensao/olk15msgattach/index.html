O que é um arquivo .olk15msgattach? (Arquivo de anexo de mensagens de e-mail do Outlook 2016)

Um arquivo OLK15MSGATTACH é um arquivo de dados criado pelo Microsoft Outlook 2016, um programa de e-mail e calendário. Ele contém informações sobre anexos a e-mails enviados e recebidos por um usuário no Outlook. Os arquivos OLK15MSGATTACH são semelhantes aos arquivos .OLK14MSGATTACH, que são criados pelo Outlook 2011.

Mais Informações

Ao visualizar uma mensagem de e-mail na versão de 2016 do Outlook para macOS, os dados do arquivo OLK15MSGATTACH são carregados. O arquivo permite que o software opere de forma mais eficiente, carregando as informações somente quando o usuário precisar.

Os arquivos OLK15MSGATTACH são armazenados no seguinte diretório:[user]/Documents/Microsoft User Data/Office 2011 Identities/Main Identity/Data Records/Message Attachments/
