O que é um arquivo .sid? (Imagem mrsid)

Imagem criada usando compactação de banco de imagem sem emendas multi-resolução (MrSID), que foi desenvolvida pela Extensis para comprimir imagens grandes; comumente usada para particionar e armazenar grandes imagens de Sistemas de Informação Geográfica (SIG).

O que é um arquivo .sid? (Documento ScanIt)

Documento criado e usado pelo ScanIt, um aplicativo usado para extrair dados de gráficos; contém a imagem do gráfico e seus metadados; inclui definições de eixos, janelas de documentos e outras opções específicas de documentos.

Mais Informações

O arquivo SID permite que você salve seu progresso e continue-o posteriormente. O arquivo pode ser salvo em uma variedade de formatos de imagem, como .JPEG, .PNG, .TIF e .BMP.

O que é um arquivo .sid? (Arquivo de áudio SID)

Arquivo de áudio originalmente usado pelos sistemas de computador Commodore, mas agora mais frequentemente usado para reproduzir os sons em emuladores commodore e jogadores de software SID; muitas vezes usado para música de fundo de videogame e pode usar formatos de áudio mono ou estéreo; criado para o processador MOS Technology 6502, que pode ser emulado por softwares desktop mais recentes para reproduzir o áudio.

Mais Informações

Os arquivos SID podem usar o formato PSID ou o formato RSID (Real SID) mais novo. Embora o formato PSID seja mais compatível com o software geral de emulação Commodore, o formato RSID requer emulação mais completa do hardware C64.

Arquivo de backup de dados do steam game

Desenvolvedor Valve

O que é um arquivo .sid? (Arquivo de backup de dados do steam game)

Arquivo de backup criado pelo Steam, um aplicativo usado para baixar e gerenciar jogos de vídeo da Valve em computadores desktop; contém dados de jogos compactados criados pelo recurso de backup Steam (steambackup.exe programa); usado para fazer backup de jogos para DVDs ou outras mídias externas.

Mais Informações

Os arquivos SID são criados com um arquivo .SIS e um arquivo .SIM, que fornecem metadados necessários para restaurar dados do jogo de backup. Os arquivos SID usam a convenção de nomeação {game}_disk{X}_{Y}.sid, onde "game" é o nome do jogo de backup, "X" é o número do disco, e "Y" é o número do arquivo de backup no disco.

NOTA: Os backups no Steam não incluem jogos salvos, mapas multiplayer personalizados ou arquivos de configuração. Você deve fazer backup e restaurar esses arquivos manualmente.
