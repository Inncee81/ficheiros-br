O que é um arquivo .hsh? (Arquivo do índice de pesquisa do catálogo Windows)

Arquivo de sistema criado pelo Indexing Service, um componente do sistema operacional Windows usado para criar um índice de sistema de arquivos pesquisável; comumente criado no diretório C:\System Volume Information\catalog.wci\.

Mais Informações

Os arquivos HSH são salvos juntamente com vários outros arquivos de índice que juntos compõem o índice de pesquisa. Outros tipos de arquivos incluem arquivos .000, .BK1, .BK2, .PS1, .PS2 e .FID.
