O que é um arquivo .GXF? (Arquivo geral de fonte CADD Pro)

Arquivo de fonte usado pelo General CADD Pro, uma ferramenta de design CAD para Windows; armazena uma fonte que pode ser usada para inserir texto e outros símbolos em um desenho CAD; deve estar disponível no software para que um desenho CAD que use a fonte seja exibido corretamente.

O que é um arquivo .GXF? (Arquivo de formato geral eXchange)

Formato geral eXchange usado para transferir clipes audiovisuais entre sistemas de armazenamento de programas de televisão; pode conter vídeo, áudio, código de tempo e metadados.

Mais Informações

O formato GXF foi originalmente desenvolvido para o intercâmbio de imagens de câmera por redes de dados e armazenamento de arquivos sobre tipo de dados. À medida que a tecnologia progrediu, o formato do contêiner evoluiu para suportar a compactação de vídeo baseada em MPEG, M-JPEG e DV, bem como áudio, código de tempo e outros dados do usuário. Agora é um padrão da Sociedade de Filmes & De Televisão (SMPTE).
