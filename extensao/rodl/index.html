O que é um arquivo .RODL? (Arquivo de idioma de definição de remObjects)

Arquivo de desenvolvedor criado pela RemObjects SDK, um aplicativo de desenvolvimento de software usado para a construção de software de nível intermediário; usa e formato XML e define os serviços fornecidos pelo software desenvolvido, incluindo a API e formatos de dados suportados; usado como interface, ou especificação do usuário, para interagir com os serviços.

Mais Informações

Ao implementar os serviços RemObjects, os arquivos RODL são usados para gerar automaticamente esqueletos de código-fonte. Em seguida, o desenvolvedor pode preencher o código funcional que executa a lógica dos serviços.

NOTA: O SDK RemObjects é usado como uma extensão de IDEs de desenvolvimento comum, como Microsoft Visual Studio e Delphi.
