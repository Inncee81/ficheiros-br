O que é um arquivo .pjs? (Arquivo de legendas phoenix)

Arquivo de legendas de filme criado no formato Phoenix Japanimation Society (JPS) ; usa um formato de texto simples indicando o tempo de início e fim de cada legenda, por exemplo:

1234, 2345, "Meu filme legenda"

O que é um arquivo .pjs? (Arquivo do pacote de projeto testComplete)

Configurações do arquivo usado pelo TestComplete, um aplicativo que permite que os usuários testem uma variedade de aplicativos desktop, mobile e web; armazena configurações comuns, como quais informações são exibidas nos painéis TestComplete, para um conjunto de projetos.

Mais Informações

Entre outros recursos, projetos testComplete e suítes de projetos permitem testar diferentes aplicativos usando as mesmas configurações. Um arquivo PJS é semelhante a um arquivo .MDS, mas contém informações para um conjunto de projetos em vez de um único projeto. Um arquivo TCLS também é semelhante a um arquivo PJS, mas é usado para armazenar configurações relacionadas ao testador para um conjunto de projetos.

NOTA: Você pode editar o arquivo PJS com o Project Suite Editor, que está incluído no aplicativo.
