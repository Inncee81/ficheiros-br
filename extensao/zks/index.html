O que é um arquivo .zks? (Pathfinder Kingmaker Game Save File)

A ZKS file is a game save data file created by Pathfinder Kingmaker, a fantasy role-playing game (RPG). It is a Zip-compressed archive that contains multiple .JSON files that store the game save information. ZKS files include all the game save information, including the attributes for the main character and companions and acquired weapons, spells, and gold.

Mais Informações

Você provavelmente só encontrará arquivos ZKS se você estiver procurando mover ou modificar arquivos de salvar pathfinder Kingmaker. Os arquivos ZKS não devem ser abertos manualmente, em vez disso, os arquivos são referenciados pelo jogo para carregar o progresso do jogo.

Embora os arquivos ZKS não sejam feitos para serem abertos manualmente, você pode abrir e extrair informações dos arquivos alterando a extensão do arquivo de .zks para .zip. Um utilitário zip-descompressão, como o Microsoft File Explorer ou o Corel WinZip, pode ser usado para extrair o conteúdo (player.json, party.json, etc), dos arquivos .ZIP. Tenha cuidado ao modificar o conteúdo, pois as edições podem fazer com que os dados de salvamento do jogo se tornem corruptos.

Depois de fazer edições no conteúdo, salve os arquivos modificados e comprima os arquivos em um arquivo ZIP. Em seguida, altere a extensão do arquivo .zip para .zks e coloque o arquivo de volta no local de salvamento original.

Os arquivos ZKS são criados e salvos pelo Pathfinder Kingmaker no seguinte diretório:

C:\Users\[username]\AppData\LocalLow\OwlcatGames\Pathfinder Kingmaker\Saved Games
