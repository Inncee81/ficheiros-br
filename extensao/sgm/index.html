O que é um arquivo .sgm? (Arquivo SGML)

Arquivo escrito na Linguagem de Marcação Generalizada Padrão, uma linguagem de geração de documentos usada como meio padrão de compartilhamento de informações digitalmente; inclui tags personalizadas que descrevem a estrutura e o conteúdo das informações incluídas no documento; muitas vezes faz referência a um arquivo de Definição de Tipo de Documento (.DTD), que define os objetos que podem ser incluídos no arquivo SGML.

O que é um arquivo .sgm? (Pacote Starpoint Gemini 2 Mod)

Um arquivo SGM é um pacote mod usado pelo Starpoint Gemini 2, um jogo de comércio espacial e combate. Ele contém um pacote mod, que inclui todo o conteúdo de um mod. Os arquivos SGM são usados para modding vários aspectos do jogo, incluindo naves espaciais, estações, planetas, armas, regiões do mundo do jogo, mercenários e equipamentos.

Mais Informações

Os arquivos SGM são criados a partir de arquivos de definição mod SGS, que são usados para definir todas as partes de um mod. Você pode usar o gerente mod do jogo para embalar todos os arquivos em um mod no pacote mod SGM, a fim de publicá-lo e compartilhá-lo com outros jogadores do Starpoint Gemini.

NOTA: Os arquivos SGM estão comumente localizados na pasta "Mods" do diretório de instalação Starpoint Gemini.

O que é um arquivo .sgm? (VisualBoyAdvance Jogo Salvo)

Arquivo estadual salvo criado pelo VisualBoyAdvance, um emulador do Game Boy; contém um instantâneo do estado do jogo, que pode ser salvo a qualquer momento durante a jogabilidade; usado para salvar e carregar o progresso no jogo.

Mais Informações

No VisualBoyAdvance, os arquivos SGM podem ocupar um dos dez slots de salvamento por jogo. Os arquivos são nomeados usando o prefixo de nome de arquivo ROM e o número do slot de salvamento. Por exemplo, o MyGame.gba rom .GBA poderia ter salvo o slot 1 criado com o nome de arquivo MyGame1.sgm.
