O que é um arquivo .ls? (Arquivo LightWave LScript)

Script criado na linguagem LScript, que é usado para automatizar ações no software de renderização LightWave 3D da NewTek; pode ser usado para desenhar automaticamente pontos e polígonos, criar movimento de objeto ou para uma variedade de outras ações; permite que os usuários adicionem novos recursos ao software e automatizem tarefas repetitivas.
