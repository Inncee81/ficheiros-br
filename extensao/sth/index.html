O que é um arquivo .sth? (Arquivo Lotus Domino Stash)

Arquivo seguro que armazena a senha de um arquivo de chaveiro Lotus Domino (.KYR); gerado automaticamente quando o chaveiro do servidor é criado; salvo com o mesmo nome do arquivo do teclado; deve ser salvo tanto no diretório de dados Notes da máquina cliente quanto no diretório Domino do servidor.

O que é um arquivo .sth? (Arquivo de dados predefinidos do usuário do SampleTank)

Arquivo de dados predefinido criado pelo SampleTank, um aplicativo que permite criar amostras de som; contém o nome do instrumento e a descrição do arquivo .STIP predefinido do usuário; criado com .STI e .STW arquivos; deve permanecer na mesma pasta que os arquivos STI, STW e STIP relacionados; não deve ser aberto manualmente.

Mais Informações

Para cada arquivo STIP criado, um arquivo STH, STI e STW será criado. Cada um dos três arquivos contém dados dos que o STIP retira.

Para criar um arquivo STH, edite seu som no aplicativo, clique em SAVE na área Predefinição do usuário, nomeie seu som, clique em OK, isso cria um arquivo STIP que também cria o arquivo STH e será localizado na pasta de seus "Parent Sound"

Os arquivos STH, STI, STW e STIP relacionados serão todos nomeados após o mesmo instrumento. Seu STH vai aparecer assim: [instrument_name].sth.
