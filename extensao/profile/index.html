O que é um arquivo .profile? (Perfil bash Shell)

Perfil shell usado por Linux e Mac programas de terminais OS X; contém definições para um ambiente shell, como variáveis de ambiente, scripts para execução e outras instruções; usado para armazenar configurações pré-definidas que são carregadas quando um programa shell é iniciado.

Mais Informações

Ao abrir o Apple Terminal em Mac OS X, o programa busca automaticamente um arquivo PROFILE e executa-o linha por linha como um script shell. Para executar manualmente um arquivo PROFILE, use o comando fonte ~/profile.

ARQUIVOS PROFILE são arquivos ocultos que não têm um prefixo de nome de arquivo. Eles são sempre chamados .profile e estão localizados no diretório doméstico do usuário.

NOTA: O Apple Terminal é um programa de shell bash.

O que é um arquivo .profile? (Perfil de aplicativo Citrix)

Arquivo de configuração usado pelo Citrix XenApp, um pacote de software usado para fornecer software virtualizado aos sistemas clientes; armazena os dados necessários para o lançamento de um aplicativo transmitido ao usuário.

Mais Informações

ARQUIVOS PROFILE podem ser editados usando o programa Citrix Streaming Profiler incluído no software XenApp.

O que é um arquivo .profile? (Arquivo de perfil de recurso do mecanismo godot)

Um arquivo PROFILE é um arquivo de configurações criado pela Godot Engine, um mecanismo de jogo de código aberto. Ele armazena uma lista de recursos que configuram como o Editor aparece no Godot Engine, como painéis, menus, botões e requisitos de motor 2D ou 3D. Os arquivos PROFILE são salvos em um formato de texto semelhante ao JSON.

Mais Informações

Captura de tela de um arquivo de perfil no Godot Engine 321

Godot Engine é um motor de jogo de código aberto usado para criar jogos 2D e 3D para várias plataformas, como Windows, macOS, Linux, Android, iOS e web. Os usuários godot podem exportar perfis de recursos do Editor para salvar as configurações do Editor para os vários tipos de projetos em que trabalham. Os usuários podem criar um PERFIL com o Godot Engine selecionando Editor → Manage Editor Features... e clicando em Export. Eles podem então importar o PERFIL para configurar o Editor com suas predefinições favoritas.

NOTA: Arquivos PROFILE foram introduzidos com o lançamento do Godot Engine 321.

O que é um arquivo .profile? (Perfil do jogador de jogos do Pandemic Studios)

Perfil do jogador salvo por um dos jogos de tiro em terceira pessoa da Pandemic Studios; contém informações do jogador, como nome do perfil, preferências do jogador e progresso em missões de campanha; salvo no diretório \GameData\SaveGames\ de uma instalação de jogo.

Mais Informações

ARQUIVOS PERFIL aplicam-se a jogos da Pandemic Studios como Star Wars: Battlefront 2, Mercenaries 2 e The Sabateur.
