O que é um arquivo .vsl? (Arquivo de modelo de velocidade)

Arquivo desenvolvedor usado pelo Velocity, um mecanismo de modelo Java usado para gerar texto e código fonte automaticamente; contém texto, bem como declarações escritas no Velocity Template Language (VTL); frequentemente usado para gerar código-fonte de página Web.

Mais Informações

O Velocity permite que os dados de um programa Java sejam mesclados com um modelo Velocity para produzir um arquivo de texto de saída. Por exemplo, um arquivo HTML que é um modelo Velocity pode ter uma tabela não povoada que é preenchida com dados quando executada com um programa Java usando velocity.

NOTA: Os modelos velocity usam mais comumente a extensão .VM.

O que é um arquivo .vsl? (Complemento visio)

Addon, ou biblioteca, arquivo usado pelo Microsoft Visio, um aplicativo de diagramação de negócios; salva componentes de software que ampliam a funcionalidade do programa; normalmente criados e distribuídos por desenvolvedores de terceiros.

Mais Informações

NOTA: Os arquivos VSL são, na verdade, .DLL arquivos que estão especificamente configurados para serem executados com o Visio.
