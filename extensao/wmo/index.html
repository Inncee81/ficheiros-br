O que é um arquivo .wmo? (Objeto do Mapa Mundial de Warcraft)

Formato de arquivo usado para armazenar dados de objetos 3D para World of Warcraft, um jogo de aventura multiplayer online; comumente referido como um "objeto de mapa do mundo" ou WMO; pode ser salvo como um arquivo raiz ou um arquivo de grupo.

Mais Informações

Arquivos root listam texturas (.BLP arquivos), materiais, modelos (.M2 arquivos), grupos e outros dados. Os arquivos de grupo contêm os dados reais do modelo 3D para uma única unidade no WMO.
