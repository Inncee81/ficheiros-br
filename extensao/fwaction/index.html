O que é um arquivo .fwaction? (Arquivo de ação freeway)

Arquivo de ação criado e usado pelo Freeway Pro, um aplicativo que permite que os usuários criem sites sem ter que escrever qualquer código; contém uma ação que pode ser adicionada a um documento .FREEWAY para adicionar funcionalidade ao site.

Mais Informações

No Freeway Pro, as ações são semelhantes aos plugins, pois podem ser instaladas para adicionar funcionalidade. Você pode usar ações pré-existentes ou pode escrever o seu próprio para adicionar conteúdo em seu site que não pode ser criado através da interface do Freeway Pro.

Uma ação é composta por vários componentes:

  • HTML ou JavaScript que serão adicionados ao HTML que a saída da Freeway quando você visualizar ou publicar seu documento FREEWAY.
  • "Script de ação" que determina a colocação do JavaScript ou HTML.
  • Script que determina como as opções aparecem na paleta Actions.
  • Definição de todas as variáveis utilizadas quando a Ação é executada.
