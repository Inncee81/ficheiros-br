O que é um arquivo .e2p? (Arquivo do dispositivo PonyProg)

Um arquivo E2P é um arquivo de dispositivo usado pela PonyProg, um programador universal gratuito para dispositivos seriais. Ele contém uma configuração de um dispositivo, que inclui o tipo do dispositivo, bits de segurança e notas associadas ao dispositivo.

Mais Informações

O arquivo E2P é o arquivo principal usado pela PonyProg para armazenar informações de dispositivos seriais. PonyProg cria o arquivo E2P quando você seleciona File → Save Device File. Você também pode salvar informações do dispositivo em um arquivo .BIN, .HEX, .CSM, .MOT, .ROM ou .EEP.

PonyProg permite que você programe dispositivos seriais como Microwire, Microchip PIC micro, Atmel AVR e SPI eeprom. É usado em aplicações de consumo, automotivo, telecomunicações e industriais com armazenamento de memória não involatile.
