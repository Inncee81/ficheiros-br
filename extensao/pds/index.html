O que é um arquivo .pds? (Arquivo do sistema de dados planetário)

Formato de dados científicos criado pela NASA (National Aeronautics and Space Administration); usado para armazenar dados planetários, solares e lunares coletados de missões terrestres e espaciais.

Mais Informações

Arquivos PDS geralmente contêm rótulos que estão escritos na Linguagem de Descrição do Objeto (ODL). Esses rótulos usam palavras-chave que descrevem os dados contidos no arquivo.

O que é um arquivo .pds? (Arquivo de script do PowerDirector)

Arquivo de projeto de vídeo ou apresentação de slides criada com o PowerDirector, um programa de edição de vídeo digital; armazena referências a arquivos de áudio e vídeo e inclui configurações para menus, efeitos, transições e outras informações de produção de vídeo caseiro; usado para criar CDs de vídeo, DVDs, Blu-ray e discos AVCHD.

Mais Informações

NOTA: Os arquivos PDS são frequentemente construídos a partir de um modelo fornecido com o software.

O que é um arquivo .pds? (O Projeto De Impressão)

Um arquivo PDS é um projeto criado pela Broderbund The Print Shop, um programa usado para criar o layout de impressões pessoais e comerciais. Ele contém um projeto para uma página em branco ou layout de placa que pode incluir texto, manchetes, formas, linhas, imagens, tabelas e outros recursos de design.

Mais Informações

Os arquivos PDS têm sido usados por várias versões da Print Shop para salvar projetos ao longo dos anos. No entanto, o formato de arquivo PDS não foi usado após a Print Shop 10. Para abrir um arquivo PDS na Loja de Impressão 10 ou posterior, selecione File → Open, escolha "Todos os arquivos (**)" do menu suspenso do tipo de arquivo, navegue até a localização do arquivo PDS e clique em Abra. Você pode precisar reformatar alguns dos elementos no layout de impressão no arquivo PDS.

NOTA: Arquivos PDS criados pela Print Shop Ensemble I, II ou III não podem ser abertos por nenhuma outra versão da The Print Shop. Além disso, os arquivos PDS criados pela Print Shop 5 e 6 podem ser abertos por quaisquer versões posteriores da Print Shop.
