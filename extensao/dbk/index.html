O que é um arquivo .dbk? (dBASE Database Backup)

Arquivo de backup de banco de dados criado pelo software de desenvolvimento de aplicativos de banco de dados dBASE; salva uma cópia de backup do arquivo .DBF ou .DB2 de trabalho; pode ser usado para restaurar o banco de dados caso ele fique corrompido; pode ser gerado automaticamente pelo software.

O que é um arquivo .dbk? (Arquivo de backup do celular Sony Ericsson)

Um arquivo DBK é um arquivo de backup de telefone celular criado pela Sony Ericsson PC Suite, um programa usado para gerenciar celulares Sony Ericsson. Ele contém vários arquivos zip-compression que armazenam várias informações telefônicas, como nomes e números de telefone de contatos. Os arquivos DBK são usados para fazer backup de informações telefônicas no caso de o telefone ser perdido ou quebrado.

Mais Informações

DBK arquivos podem ser abertos com vários programas, como Sony Ericsson PC Suite, FJ Software Development MyPhoneExplorer e Amrak Software phoneMiner. Os arquivos também podem ser descomprimidos com um utilitário de descompressão zip, como o Microsoft File Explorer, Corel WinZip e o Apple Archive Utility.

Para descomprimir um arquivo DBK, renomeie a extensão de arquivo .dbk para .zip e, em seguida, descomprima e exorte o conteúdo com o programa de descompressão ZIP. O conteúdo, que pode incluir um arquivo de contatos .VCF chamado contacts.vcf pode ser aberto com um programa que suporta o tipo de arquivo. Depois de extrair o conteúdo, comprima os arquivos como um arquivo .ZIP e, em seguida, renomeie a extensão de arquivo .zip para .dbk.
