O que é um arquivo .sam? (Arquivo de amostra LMHOSTS)

Amostra do arquivo de configuração LMHOSTS usado pela Microsoft Windows; contém mapeamentos de endereços IP para nomes NetBIOS de servidores remotos com o computador que um computador se comunicará com o TCP/IP;

Mais Informações

Windows carrega automaticamente as primeiras 100 entradas do LMHOSTS marcados com "#PRE" na inicialização. Esse número pode ser alterado no registro do sistema, mas pode ser limitado pelo desempenho do computador.

Tanto o arquivo LMHOSTS quanto o arquivo de amostra estão localizados em %windir% em 95, 98 e Windows Millenium Edition. Eles estão localizados em %windir%system32driversetc. na Windows NT 40 a 7 e Windows Server 2003, 2008 e R2 2008.

NOTA: O arquivo LMHOSTS não é usado por Windows 8.

O que é um arquivo .sam? (Pacote Samurize)

Arquivo usado pelo Samurize, um programa usado para personalizar o Windows desktop; semelhante a um arquivo Windows .GADGET e salva imagens, informações de configuração e scripts; usado para distribuir pacotes Samurize.

Mais Informações

Os arquivos SAM estão .ZIP arquivos com a extensão "zip" renomeada para "sam" Você pode instalá-los clicando-os duas vezes (assumindo que o Samurize está instalado).

NOTA: O Samurize só suporta sistemas operacionais Windows 2000, XP e 2003.
