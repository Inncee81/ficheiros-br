O que é um arquivo .msix? (Pacote instalador MSIX)

Um arquivo MSIX é um pacote compactado por Zip usado para distribuir e instalar um aplicativo em Windows versão 10 1809 e posterior. Ele contém os arquivos de dados do aplicativo e .XML arquivos de configuração, que são necessários para instalar o aplicativo. O formato de arquivo MSIX é baseado nas tecnologias de instalação .MSI, .APPX, ClickOnce e App-V.

Mais Informações

A MSIX foi projetada para ser um formato de pacote universal para desenvolvedores distribuirem seus programas aos usuários através da Microsoft Store. Foi introduzido em 2018 com o lançamento de Windows versão 10 1809. Ele suporta aplicativos clássicos do Win32 e foi projetado para eventualmente substituir o AppX, o MSI e outros formatos de instalador Windows. O MSIX oferece segurança aprimorada, atualizações simplificadas de aplicativos através de Windows e desinstalação de aplicativos mais limpas.

Vários programas podem criar pacotes MSIX. Estes incluem o Microsoft Visual Studio, o MSIX Packaging Tool e o Caphyon Advanced Installer. Se você tem um aplicativo existente que deseja salvar no formato MSIX, você pode usar a Ferramenta de Embalagem MSIX para reempacotar seu aplicativo.

Para abrir um arquivo MSIX e iniciar a instalação do aplicativo embalado em Windows 10, basta clicar duas vezes no arquivo. O App Installer abre o arquivo MSIX e exibe informações sobre o aplicativo, como nome, editor e versão, bem como os recursos solicitados pelo aplicativo. Em seguida, siga as instruções no App Installer para concluir a instalação. Os pacotes MSIX são extraídos para o diretório ProgramFiles\WindowApps.

Se você quiser examinar o conteúdo de um arquivo MSIX em vez de instalar o aplicativo embalado, você pode extrair o conteúdo usando um utilitário de descompressão Zip, como WinZip ou WinRAR. Para extrair o conteúdo, altere a extensão do arquivo .msix para .zip e, em seguida, extraia os arquivos usando o programa de descompressão do arquivo. Certifique-se de alterar a extensão do arquivo .zip de volta para .msix antes de instalar o aplicativo.
