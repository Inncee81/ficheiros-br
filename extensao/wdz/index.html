O que é um arquivo .wdz? (Arquivo Compactado WDZip)

Um arquivo WDZ é um arquivo compactado criado pelo WDZip, uma ferramenta incluída no WINDEV usada para comprimir e descomprimir arquivos em arquivos. Ele contém um ou mais arquivos dos mesmos ou diferentes formatos. Os arquivos WDZ são usados para agrupar e comprimir arquivos para organizar vários arquivos e economizar espaço.

Mais Informações

WDZip pode ser usado para criar arquivos em vários formatos diferentes. O formato padrão que o programa usa é o formato WDZ, mas também pode economizar nos formatos .ZIP, .RAR, .CAB e .7Z.

Para criar um arquivo no WDZip, selecione "WDZIP" no grupo "Utilitários", que está localizado no painel "Tools". Em seguida, clique em "Novo" ou selecione File → New archive... e escolha o formato do arquivo.

Ao criar um arquivo, você pode optar por dividi-lo em vários sub-arquivos se for um grande arquivo. Esses sub-arquivos podem ser mesclados ao abrir o arquivo. Se você dividir um arquivo WDZ, o primeiro sub-arquivo é um arquivo WDZ e o próximo sub-arquivo é um arquivo .001, depois um arquivo .002 e assim por diante.

Para abrir um arquivo no WDZip, selecione File → Open an archive. Em seguida, você pode extrair ou excluir um ou mais arquivos no arquivo.
