O que é um arquivo .gvsp? (Visualizador geocortex para projeto Silverlight)

Um arquivo GVSP contém um projeto criado pelo Geocortex Viewer for Silverlight, um programa GIS usado para aprimorar o software ArcGIS para usuários web-GIS. Ele armazena um projeto, que inclui um mapa com coordenadas plotadas e linhas desenhadas, áreas, pontos e texto. Os arquivos GVSP também contêm a simbolização da camada e a ordem de desenho de camadas do mapa.

Mais Informações

O Visualizador Geocortex para projetos Silverlight permite que um usuário personalize um mapa para ser exibido de uma determinada maneira, sejam diferentes camadas habilitadas ou uma exibição ampliada em um local específico. Quando você salva seu mapa como um projeto, o arquivo GVSP é criado, o que permite compartilhar com outros usuários. Quando o arquivo GVSP for aberto, a exibição de salvamento será exibida.

Para salvar um arquivo GVSP no Geocortex Viewer for Silverlight, clique no ícone do disco "Salvar projeto", escolha um local de salvamento e clique em Save.

Para abrir um arquivo GVSP no Geocortex Viewer for Silverlight, clique no ícone da pasta "Open Project", navegue até o local do arquivo GVSP e clique em Open. Você precisa fazer login no WEB-GIS para verificar suas credenciais.

Geocortex Viewer for Silverlight é um dos vários programas geocortex incluídos com a Geocortex Essentials usado para melhorar a plataforma ArcGIS na web. Foi lançado em 2011 com a Geocortex Essentials 33.

NOTA: O visualizador geocortex para Silverlight foi descontinuado.
