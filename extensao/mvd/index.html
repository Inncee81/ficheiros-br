O que é um arquivo .mvd? (Documento MindView OS X)

Documento criado e usado pelo MindView, um programa de mapeamento mental usado para pensar visualmente e organizar ideias; armazena um projeto, que pode incluir texto, imagens, formas, ramos de conexão e outros elementos do mapa da mente.

Mais Informações

NOTA: A extensão de arquivo MVD é usada pela versão OS X do MindView para salvar documentos. No entanto, ele pode ser aberto pela versão Windows do MindView. A versão MindView for Windows salva documentos com a extensão de arquivo .MVDX.

O que é um arquivo .mvd? (Arquivo de filme edit pro filme)

Projeto de filme criado com MAGIX Movie Edit Pro ou o aplicativo anterior Video Deluxe; contém referências a arquivos de vídeo e áudio, bem como pontos de edição de filmes e cenas detectadas; também pode incluir uma variedade de efeitos personalizados.

Mais Informações

Arquivos MVD podem ser exportados para o YouTube, um DVD ou outros formatos de arquivo de vídeo. Eles também podem ser compilados em arquivos .MVP como projetos de vários filmes.
