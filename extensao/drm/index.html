O que é um arquivo .drm? (Arquivo do mapa do tambor de Cubase)

Contém configurações de mapa de bateria para um teclado MIDI específico; mapeia diferentes sons de bateria para diferentes teclas no teclado; usado para tocar um instrumento de bateria de software dentro de Cubase da Yamaha, Roland, Korg e outros teclados.

O que é um arquivo .drm? (Arquivo de dados Deus Ex)

Arquivo de dados de jogos usado por Deus Ex: Human Revolution (DXHR), um jogo de RPG de ação; pode conter vários tipos de dados de jogos, incluindo modelos de jogo, texturas, áudio e vídeo; pode ser modded com as ferramentas Deus Ex HR da Gibbed.

Mais Informações

Os arquivos DRM são armazenados dentro dos arquivos de instalação do jogo bigfile.000, bigfile.001 e bigfile.002.
