O que é um arquivo .icd? (Arquivo de driver cliente instalável)

Um driver OpenGL que é responsável pela implementação de todo o pipeline OpenGL para um Windows jogo ou aplicativo; destinado a placas de vídeo high-end que incluem a maior parte do pipeline OpenGL dentro do hardware.

O que é um arquivo .icd? (Arquivo de desenho 3D IronCAD)

Desenho tridimensional criado pelo IronCAD, um programa CAD 3D; contém objetos, tipos de linha, estilos, camadas e símbolos definidos pelo usuário.

Mais Informações

Para criar um arquivo ICD, selecione "Criar um novo documento de desenho" quando a caixa de diálogo "Bem-vindo" for exibida após a abertura do programa IronCAD. Em seguida, você será solicitado a escolher o modelo de desenho desejado e clique em OK. Uma vez que o modelo é aberto, você pode ir para o ícone IronCAD no canto superior direito, em seguida, File → Save As....

O que é um arquivo .icd? (Programa criptografado SafeDisc)

Programa executável criptografado com SafeDisc, um programa de prevenção de cópias de disco para Windows jogos e outros aplicativos; normalmente salvo junto com um arquivo .EXE que carrega o arquivo ICD protegido.

Mais Informações

Para que um programa criptografado safeDisc seja executado, o autenticador safeDisc deve verificar a assinatura do SafeDisc no disco. Após a verificação do disco, o arquivo ICD pode ser descriptografado e executado.

NOTA: Começando pelo SafeDisc 2, o carregador foi integrado ao arquivo EXE principal, removendo a necessidade do arquivo CID.
