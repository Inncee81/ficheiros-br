O que é um arquivo .w? (Arquivo de histórico de pesquisa da barra de ferramentas do Google)

Arquivo criado pela Barra de Ferramentas do Google, um addon de navegador Web que usa pesquisa do Google Web; salva uma lista de pesquisas históricas realizadas com o Google Toolbar; às vezes usada na perícia ao descobrir o que o usuário estava procurando pela última vez no Web.

Mais Informações

Arquivos W são salvos no seguinte diretório depois de instalar o Google Toolbar:

C:\Users\[user]\AppData\Roaming\Google\Local Search History\

NOTA: A barra de ferramentas do Google não está disponível para o Chrome, já que o Chrome já usa o Google por padrão para Web pesquisas. Além disso, a barra de ferramentas do Google para o Firefox foi descontinuada.

O que é um arquivo .w? (Arquivo de código fonte do arquiteto openEdge)

Arquivo de código-fonte usado pelo OpenEdge Architect, um IDE usado para desenvolver programas de software; contém código de programa, procedimentos e formulários que são usados para compilar um aplicativo; pode ser aberto e editado na perspectiva do AppBuilder.
