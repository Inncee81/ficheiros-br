O que é um arquivo .c? (Arquivo de código-fonte C/C++)

Um arquivo C é um arquivo de código fonte para um programa C ou C++. Ele pode incluir o código-fonte de um programa inteiro, ou pode ser um dos muitos arquivos de origem referenciados dentro de um projeto de programação. Os arquivos C podem ser editados usando um editor de texto básico, mas não mostrarão destaques de sintaxe como a maioria dos programas de desenvolvimento de software fazem.

C a linguagem de programação foi originalmente desenvolvida para criar utilitários Unix por Dennis Ritchie no final dos anos 1960 e início dos anos 1970 no Bell Labs. Desde a década de 1970, a popularidade do C cresceu e seu uso aumentou. A linguagem influenciou muitas linguagens de programação, incluindo C++, PHP, Perl, Python, Java e JavaScript.

C++ é uma extensão da linguagem de programação C e foi lançado em meados da década de 1980. Ele é usado para criar muitos dos programas de software atuais e pode ser armazenado em arquivos de código-fonte C, .CC, .CPP, .CXX e .C++.

NOTA: Alguns compiladores Unix usam uma extensão minúscula "c" para C e uma extensão maiúsdia "C" para C++, embora esta convenção não seja comum.

O que é um arquivo .c? (Lite-C Script)

Script escrito em lite-C, uma linguagem de programação projetada para o desenvolvimento fácil de aplicativos de gráficos e realidade virtual; usa uma sintaxe semelhante ao C/C++ e é compilado on-the-fly por lite-C.

Mais Informações

Lite-C inclui um motor de renderização ABT (Adaptive Binary Tree), efeitos de partículas programáveis, uma física corporal e motor de colisão, e desempenha funções para arquivos de som, música e filme.
