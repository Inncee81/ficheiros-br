O que é um arquivo .odb? (Banco de dados OpenDocument)

Um arquivo ODB contém um banco de dados criado pelo Apache OpenOffice Base (OOBase), um programa de banco de dados relacional de área de trabalho de código aberto. Ele armazena um arquivo de arquivos de dados que compõem o banco de dados e é formatado de acordo com o padrão OpenDocument. Os arquivos ODB são usados para salvar informações estruturadas, como tabelas e registros.

Os bancos de dados OpenDocument contêm uma coleção de arquivos e pastas que são compactados usando compactação zip. Eles incluem settings.xml, context.xml e arquivos manifest.xml, assim como uma pasta database que contém os seguintes arquivos:

  • properties: armazena as configurações do banco de dados
  • script: contém definições de tabela
  • data: contém o estado atual do banco de dados
    • <
  • backup: armazena um backup compactado do último estado de banco de dados consistente
  • log: registra alterações no banco de dados
Para examinar o conteúdo de um arquivo ODB manualmente, renomeie a extensão do arquivo "odb" para "zip" e de comprimir o arquivo com qualquer utilitário de descompressão Zip.
