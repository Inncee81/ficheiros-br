O que é um arquivo .game? (Jogo salvo de xadrez da Apple)

Jogo de xadrez salvo com o Apple Chess, o programa de xadrez incluído com Mac OS X; salva o progresso da partida para que o jogo possa ser aberto e retomado posteriormente; também pode ser usado para testar diferentes estratégias de um determinado ponto do jogo.

O que é um arquivo .game? (Arquivo de jogo exportado gameSalad)

Arquivo de jogos criado pelo GameSalad Creator, um programa usado para construir jogos para iOS, Android e Web (HTML5); salva um jogo de vídeo embalado exportado no software a partir de um arquivo de projeto .GAMEPROJ; criado pela escolha de File → Export to iPhone do menu de aplicativos Dom.

Mais Informações

Os arquivos GAME são usados para carregar projetos do GameSalad no site da GameSalad, que converte o arquivo em um projeto Xcode (.XCODEPROJ). Como a Apple exige que os aplicativos sejam enviados usando o Xcode, este é um passo necessário antes que os desenvolvedores possam enviar aplicativos do GameSalad para as lojas de aplicativos do iTunes ou Mac. Os arquivos GAME em si não podem ser executados, e são usados apenas para conversão para Xcode.

NOTA: Os arquivos DO JOGO são .ZIP arquivos que usam a extensão "game" .
