O que é um arquivo .ldf? (Arquivo de origem do dicionário Lingoes)

Arquivo de origem usado para especificar informações de dicionário para Lingoes, um dicionário e programa de tradução de texto; salvo em um formato de texto simples e contém informações de títulos de dicionário, bem como termos e suas definições.

Mais Informações

Os arquivos LDF são compilados em arquivos .LD2, que podem ser importados para o software Lingoes. Atualmente, os usuários não podem compilar seus próprios dicionários, mas o Lingoes Project planeja lançar o software Lingoes Dictionary Creator, que permitirá que os usuários criem arquivos LD2.

O que é um arquivo .ldf? (Arquivo de registro de transações do servidor SQL)

Um arquivo LDF é um arquivo de registro criado pelo SQL Server, um sistema de gerenciamento de banco de dados relacional (RDBMS) desenvolvido pela Microsoft. Ele contém um registro de ações recentes executadas pelo banco de dados e é usado para rastrear eventos para que o banco de dados possa se recuperar de falhas de hardware ou outros desligamentos inesperados.

Mais Informações

Arquivos LDF são registros de transações. Esses registros contêm um histórico de atividade para transações totalmente comprometidas e parcialmente comprometidas no banco de dados (o arquivo .MDF). Após um desligamento inesperado, o SQL Server pode usar o registro de transações para restaurar o banco de dados ao estado exato antes da falha.

Os registros de transações podem ser truncados assim que o banco de dados criar um ponto de verificação, que é um estado estável do banco de dados e do arquivo de registro. Os sistemas RDBMS podem, então, excluir com segurança todos os dados de log até o ponto de verificação, reduzindo o tamanho do arquivo de registro.

NOTA: Os arquivos LDF podem se tornar muito grandes em tamanho. Isso se deve ao fato de que nenhum posto de controle foi criado recentemente. No entanto, também pode ser devido a transações abertas que ainda não foram totalmente comprometidas, ou outros bloqueios nos registros do banco de dados.
