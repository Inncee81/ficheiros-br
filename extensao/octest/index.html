O que é um arquivo .octest? (Pacote de teste da unidade Xcode Objective-C)

Arquivo de desenvolvedor criado pela Xcode, um Mac OS X e iOS desenvolvimento de aplicativos IDE; salva um pacote de testes unitários, que são pequenos testes localizados usados para verificar o comportamento de funções individuais dentro do código fonte geral do aplicativo; permite que todos os arquivos de teste sejam embalados juntos.

Mais Informações

Os testes dentro de um pacote OCTEST podem ser um dos dois tipos:

  1. Testes lógicos que são executados em um ambiente isolado de testes fora do aplicativo em execução.
  2. Testes de aplicativos que são executados dentro do aplicativo em execução. Estes são frequentemente usados para testes de interface de usuário.
