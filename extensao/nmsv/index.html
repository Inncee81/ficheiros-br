O que é um arquivo .nmsv? (Arquivo de som maciço de instrumentos nativos)

Arquivo de som criado pela Native Instruments Massive, um aplicativo sintetizador (DAW) que permite criar e editar sons "massivos", como linhas de graves profundos e perfurados; contém um som pré-gravado ou criado pelo usuário com diferentes configurações personalizadas para recursos como filtros, amplificadores, fx, ruído e osciladores.

Mais Informações

Para criar o arquivo NMSV, selecione File → Save or Save As..., nomeie o arquivo, escolha o local de salvamento e clique em Salve.

Para abrir o arquivo NMSV, selecione File → Open Sound..., escolha seu arquivo NMSV e clique em Abra.

NOTA: A extensão do arquivo NMSV substituiu a extensão .KSD na versão 13 do Massive. No entanto, o Massive ainda abrirá o arquivo KSD, selecionará File → Import KSD Sound..., escolherá seu arquivo KSD e clicará em Open.
