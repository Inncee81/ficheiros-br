O que é um arquivo .svp? (Projeto espectador subsuperficial)

Um arquivo SVP é um projeto criado pelo INSIGHT Subsuperficial Viewer, um programa geológico usado para visualizar e analisar dados sísmicos subsuperficial em 3D. Ele contém um modelo geológico subsuperficial, que pode incluir seções, mapas, dados de unidades geológicas, troncos de furos, camadas, modelos VOXEL, falhas e propriedades rochosas.

Captura de tela de um arquivo svp no INSIGHT Subsuperficial Viewer Reader 6

Você pode abrir e criar arquivos SVP em Windows usando o Subsuperficial Viewer. Você também pode abrir arquivos SVP usando o Subsuperficial Viewer Reader, que está disponível para Windows e macOS.

Para abrir arquivos SVP no Subsuperficial Viewer e Subsurface Viewer Reader, basta selecionar File → Open project e navegar para o arquivo SVP.

NOTA: O software INSIGHT Subsuperficial Viewer foi descontinuado.
