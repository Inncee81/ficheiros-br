O que é um arquivo .qss? (Folha de estilo Qt)

Arquivo de folha de estilo usado pelos programas Qt (pronuncia-se "cute"); contém definições para a aparência e a sensação dos elementos de GUI, incluindo fontes, tamanhos, cores, layouts e comportamentos de mouse-over; armazena uma biblioteca de estilos que podem ser incorporados em uma interface de usuário.

Mais Informações

Qt é uma estrutura de desenvolvimento multiplataforma usada para criar aplicativos em vários ambientes sem recompilar o código do programa. É usado pelo Google Earth, bem como pelo leitor de mídia VLC.

Os arquivos QSS são semelhantes aos arquivos .CSS. Eles podem ser criados com qualquer editor de texto.

NOTA: Qt SDK foi desenvolvido anteriormente pela Nokia antes da Digia adquirir o software.
