O que é um arquivo .VDI? (Arquivo de recursos StepMania Debug)

Arquivo usado pelo StepMania, um programa de dança rítmica que inclui suporte para almofadas de dança; contém informações de depuração usadas para relatar falhas do programa em Windows; armazenado em um formato binário e geralmente tem o nome de arquivo StepMania.vdi.

O que é um arquivo .VDI? (Imagem de disco virtual do VirtualBox)

Formato de unidade virtual usado pelo Oracle VM VirtualBox, um programa de virtualização de desktop de código aberto; pode ser montado como um disco rígido nas plataformas Mac, Windows e Unix; permite que os usuários executem programas escritos para diferentes sistemas operacionais em um ambiente virtual de alta velocidade.

Mais Informações

A Sun Microsystems adquiriu o xVM VirtualBox do innotek no início de 2008. Posteriormente, a Oracle adquiriu a Sun Microsystems em 2010 e agora desenvolve o VM VirtualBox.
