O que é um arquivo .m4a? (Arquivo de áudio MPEG-4)

Um arquivo M4A é um arquivo de áudio salvo no formato MPEG-4 e codificado com o codec AAC (Advanced Audio Coding, codificação de áudio avançado) ou com o Código de Áudio Sem Perdas da Apple (ALAC). Ele pode armazenar vários tipos de conteúdo de áudio, como músicas, podcasts e audiobooks.

MPEG-4 é um formato de contêiner multimídia usado para armazenar dados de áudio e vídeo compactados. O formato é usado por .MP4 arquivos, que armazenam dados de áudio e vídeo. Os arquivos M4A contêm apenas dados de áudio, o que levou à extensão .m4a que significa "MPEG-4 Audio" Essa alteração de nome ajudou a distinguir entre arquivos de vídeo MPEG-4 e arquivos somente de áudio MPEG-4.

A iTunes Store fornece músicas no formato M4A. Essas músicas são codificadas usando compactação AAC, o que reduz significativamente o tamanho de cada arquivo. Músicas que incluem proteção de direitos autorais têm uma extensão .M4P.

NOTA: Windows Media Player 11 requer o K-Lite Codec Pack para reproduzir arquivos M4A. Windows Media Player 12 (incluído com Windows 7) fornece suporte nativo para o formato M4A.
