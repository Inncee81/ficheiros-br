O que é um arquivo .frameset? (Pintor de quadros para arquivo conjunto de quadros LGS)

Um arquivo FRAMESET contém uma animação de iluminação criada por J⁄•Softcode Frame Painter para LGS, um programa usado para criar animações para teclados Logitech compatíveis. Ele armazena informações de um conjunto de quadros, que é uma coleção de um ou mais quadros que compõem uma animação. Os arquivos FRAMESET incluem a posição e o tamanho de uma .PNG, .JPG ou .GIF sobreposição do teclado de imagem para cada quadro da animação.

Mais Informações

Para abrir um arquivo FRAMESET no Frame Painter para LGS, clique no botão Load e escolha o arquivo FRAMESET que deseja abrir. Para criar um arquivo FRAMESET no Frame Painter para LGS, clique no botão New e modifique a animação de iluminação do teclado e clique em Save As..

ARQUIVOS FRAMESET podem ser exportados como .EFT arquivos que podem ser importados no Logitech Gaming Software, que é um programa usado para personalizar mouses de jogos Logitech G, fones de ouvido, alto-falantes, rodas selecionadas e teclados.
