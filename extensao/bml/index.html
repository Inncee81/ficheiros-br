O que é um arquivo .bml? (Arquivo de linguagem de marcação de feijão)

Web página escrita na Linguagem de Marcação de Feijão (BML); com base no Bean Scripting Framework (BSF), que é um conjunto de classes Java que fornecem suporte ao idioma de script dentro de aplicativos Java.

Mais Informações

BSF permite que .JSP arquivos sejam escritos em outros idiomas além do Java, com acesso à biblioteca da classe Java. Os arquivos BML usam BSF para processar feijões Java em manipuladores de eventos e outros scripts incorporados.

O que é um arquivo .bml? (Braille 2000 Braille File)

Arquivo braille criado pela Computer Application Specialties Company (CASC) Braille 2000, um editor de texto para braille; contém linguagem de marcação em braile, que é semelhante ao XML, mas aparece em braile.

Mais Informações

Braille 2000 converte ortografias de impressão para ASCII Braille em arquivos BML. As propriedades XML, como tags, são representadas como propriedades BML, como tags BML. Isso permite que pessoas cegas que podem ler e digitar braille criem páginas XML em braile.

Para criar um arquivo BML no CASC Braille 2000, selecione New → Braille File, nomeie o arquivo, selecione "Braille2000 Braille File (bml)" no menu de retirada do tipo de arquivo e clique em OK.

Para abrir um arquivo BML no CASC Braille 2000, selecione Open → Braille File, navegue até o arquivo e clique em Abra.
