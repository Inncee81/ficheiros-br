O que é um arquivo .mmp? (Arquivo de projeto LMMS)

Projeto criado pelo LMMS (Linux MultiMedia Studio), um programa de criação de música; contém dados de projetos LMMS, como edições de faixas, tempo, pitch, volume e tipos de instrumentos incluídos; pode ser exportado como .WAV e .OGG arquivos de áudio.

Mais Informações

Para criar o arquivo MMP, clique no ícone salvar no canto superior esquerdo do aplicativo ou selecione Project → Save or Save As..., nomeie seu projeto, escolha o local de salvamento e clique em Save.

Para abrir o arquivo MMP, basta clicar duas vezes no arquivo. Você também pode abrir o projeto dentro do aplicativo, selecionar Project → Open..., navegar para o seu arquivo, selecioná-lo e clicar em Abrir.

NOTA: O LMMS substituiu a extensão do arquivo MMP pela extensão .MMPZ.

O que é um arquivo .mmp? (Arquivo de especificação de projeto símbio)

Arquivo de projeto usado por ferramentas de desenvolvimento do Symbian OS; contém propriedades de projeto usadas para construir o projeto independentemente da plataforma ou compilador que está sendo usado; usado pelas ferramentas makmake ou abld para converter o arquivo em um arquivo para uma plataforma específica.

Mais Informações

Arquivos MMP podem conter referências a alvos, arquivos de código-fonte a serem compilados, incluir diretórios e recursos de aplicativos.

NOTA: O Nokia Symbian SDK não está mais disponível.
