O que é um arquivo .prg? (Arquivo do programa Visual FoxPro)

Arquivo de programa de texto usado pela Visual FoxPro, uma ferramenta de desenvolvimento usada para criar aplicativos de banco de dados Windows; contém um ou mais comandos muitas vezes instruindo a FoxPro a carregar um formulário ou janela projetado dentro do software; muitas vezes contém um comando "DO".

Mais Informações

ARQUIVOS PRG permitem que os programas sejam executados dentro do ambiente Visual FoxPro. No entanto, eles não são Windows arquivos executáveis. Portanto, você deve primeiro compilar o programa em uma Windows executável, a fim de executá-lo fora do Visual FoxPro.

NOTA: A FoxPro, antecessora da Visual FoxPro, é uma variante xBase, então os arquivos FoxPro PRG podem ser semelhantes aos encontrados em outros ambientes de programação do xBase, como Clipper, dBFast e versões mais antigas do dBASE.

O que é um arquivo .prg? (Arquivo do programa xBase)

Arquivo de programa usado pelas versões dos DOS mais antigas das variantes dBASE e xBase; salvo como um script que pode ser executado com o intérprete dBASE ou xBase; pode ser executado sem compilação, o que fornece aos desenvolvedores reviravoltas mais rápidas ao testar.
