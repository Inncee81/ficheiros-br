O que é um arquivo .xbm? (Gráfico de Bitmap X11)

Imagem bitmap usada pelo X Window System (X11), uma interface gráfica de usuário (GUI) usada para gerenciar computadores de servidor cliente em rede; armazena uma imagem monocromática que é escrita como texto simples na linguagem de programação C.

Mais Informações

As imagens XBM são compiladas por um compilador C em vez de um programa de imagem. Eles são usados para armazenar bitmaps de ícones e cursor para X11.
