O que é um arquivo .sqlproj? (Projeto visual studio SQL Server)

Projeto de desenvolvimento de banco de dados criado pelo Visual Studio, um IDE de desenvolvimento de software Windows; armazena o esquema do banco de dados e pode incluir código-fonte SQL; usado para criar bancos de dados para o SQL Server, o software RDBMS (Relational Database Management System, sistema de gerenciamento de banco de dados) da Microsoft.
