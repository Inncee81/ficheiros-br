O que é um arquivo .aab? (Pacote de aplicativos Android)

Um arquivo AAB é um Android App Bundle que os desenvolvedores usam para carregar aplicativos para o Google Play. Após o upload, o Google Play usa um processo chamado Dynamic Delivery para fornecer versões otimizadas de pacotes de aplicativos (.APK arquivos) para dispositivos de usuário para que eles contenham apenas as partes específicas do aplicativo que cada dispositivo precisa executar. Com o Dynamic Delivery, os tamanhos de instalação de aplicativos podem ser reduzidos significativamente em dispositivos de usuário.

Mais Informações

Tradicionalmente, os desenvolvedores têm carregado diretamente arquivos APK no Google Play para distribuição. No entanto, com o lançamento do novo formato Android App Bundle, o Google incentiva os desenvolvedores a mudar para o novo formato usando Android Studio 32 ou posterior. Uma vez que um arquivo AAB tenha sido carregado no Google Play, o Dynamic Delivery cria automaticamente e os sinais de código otimizados arquivos APK.

NOTA: Os pacotes de aplicativos não podem ser instalados como aplicativos Android em dispositivos de usuário. Em vez disso, eles devem ser usados para gerar arquivos APK para configurações específicas de dispositivos. Se você quiser gerar arquivos APK a partir de um arquivo AAB, você pode usar uma ferramenta chamada bundletool. O Google fornece esse utilitário para que os desenvolvedores possam realizar testes locais de dispositivos antes de enviar pacotes de aplicativos para o Google Play.
