O que é um arquivo .disc? (Roxio Toast Document)

Um arquivo DISC é um arquivo de projeto criado pela Roxio Toast, um aplicativo multimídia de autoria e gravação de discos. Ele contém as configurações salvas para um projeto de disco de dados, áudio, vídeo ou, cópia ou conversão. Os arquivos DISC também incluem referências aos locais de dados ou arquivos multimídia incluídos no projeto e configurações de queima.

Mais Informações

Captura de tela de um arquivo de disco em Roxio Toast 18 Titanium

Arquivos DISC são o formato nativo para projetos Toast. Esses arquivos de projeto podem ser usados para gravar discos ópticos, bem como arquivos de imagem de disco. A extensão nativa usada para imagens de disco Toast é .TOAST.

Para salvar um projeto como arquivo DISC com Toast, selecione File → Save. Para abrir um projeto salvo em um arquivo DISC, selecione File → Open.

Os seguintes tipos de projetos os arquivos DISC podem armazenar:

  • Data - Gravar qualquer arquivo ou pasta em um disco para arquivar, transferir, fazer backup de propósitos.
  • Áudio - Grave áudio em um CD, DVD de música ou disco MP3 a ser reproduzido em um computador, estéreo doméstico ou de carro, ou leitor de DVD.
  • Vídeo - Criar VCDs, SVCDs, discos de DVD-Vídeo, BLU-ray ou DVDs HighDef para serem usados em um computador ou leitor de DVD.
  • Copiar - Copiar um CD, DVD, BD ou arquivo de imagem de disco.
  • Conversão - Converta um arquivo de vídeo para outro formato de vídeo.
