O que é um arquivo .anx? (Arquivo de formato de troca annodex)

Arquivo multimídia armazenado no formato de troca annodex, que permite a indexação e anotação de vídeo e áudio contidos; destinado a ser amplamente utilizado na web para facilitar a busca por conteúdo de vídeo e áudio.

Mais Informações

O formato Annodex é construído no formato de contêiner Ogg. Ele inclui uma faixa de esqueleto, que fornece informações codec que ajudam no processo de análise e multiplexação, e pelo menos um bitstream cmml (Continuous Media Markup Language, linguagem de marcação de mídia contínua). O formato usa CMML, que é semelhante ao XML, para fornecer metadados para diferentes seções de tempo do arquivo multimídia. Isso permite que o arquivo seja pesquisado texticamente, assim como uma página da Web é pesquisada pelo Google, mesmo que seja um arquivo binário. O formato permite executar funções relacionadas à pesquisa, como acessar diretamente um clipe dentro de um vídeo em um servidor web ou avançar rapidamente para um determinado momento dentro de uma música.

NOTA: O plugin Annodex para o navegador Mozilla Firefox não está mais disponível.

O que é um arquivo .anx? (Arquivo de resposta hotDocs)

Arquivo de dados criado pelo HotDocs, um aplicativo de preparação de documentos; salva "answers," que são usados para preencher os campos em um documento de formulário HotDocs (.HFD ou .HPD arquivo); salvo usando formatação XML e permite o reaproveitamento de respostas para vários documentos.

Mais Informações

Os arquivos ANX são criados usando o modo de entrevista hotDocs, que orienta os usuários através de uma série de formulários digitais a fim de coletar as informações para o formulário HotDocs. Os arquivos são salvos no seguinte diretório por padrão:
\My Documents\HotDocs\Answers\.

NOTA: "Respostas" são quaisquer respostas necessárias para preencher um campo em branco em um formulário HotDocs. Por exemplo, um formulário de seguro pode exigir o nome, endereço e outras informações pessoais de uma pessoa.
