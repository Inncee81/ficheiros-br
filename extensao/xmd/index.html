O que é um arquivo .xmd? (Arquivo plugin de defesa bit)

Arquivo usado pelo software de proteção contra a Internet e antivírus Bitdefender; salva informações de assinatura de arquivos que ajudam o software a reconhecer ameaças para explorações de arquivos comuns, como .ZIP e .RAR.

O que é um arquivo .xmd? (Arquivo de resposta examSoft)

Arquivo de resposta usado pelo ExamSoft, um programa online de gerenciamento de avaliação; contém respostas de um exame administrado através do ExamSoft.

Mais Informações

O arquivo XMD é criado pelo SofTest, um aplicativo ExamSoft para Windows e OS X. Ao concluir um exame usando O SofTest, um arquivo XMD é gerado para conter as respostas.

O ExamSoft é uma ferramenta online que permite controlar todo o processo de teste. Ele permite que você crie exames e fornece recursos de entrega, pontuação, administração e análise. É comumente utilizado nas áreas de estudo K-12, graduação, enfermagem, médica, farmácia, direito e odontologia.

NOTA: O arquivo XMD foi substituído pelo formato de arquivo .XMDX.

Imagem estendida do disco de mídia

Desenvolvedor Engelmann Media
Categoria Imagens de Disco
Formato Binário 

O que é um arquivo .xmd? (Imagem estendida do disco de mídia)

Arquivo de imagem de disco originalmente criado pelo GameJack, um programa usado para fazer backup de jogos em CDs e DVDs; contém uma cópia exata dos dados brutos no disco e permite que cópias exatas de discos sejam feitas; também suporta opções de criação de rótulos e proteção de discos.

Mais Informações

GameJack foi substituído pela Engelmann Media 5-Star GameCopy. O formato XMD agora é suportado por outros softwares de imagem de disco, como o software de CDRWIN e H+H Software.
