O que é um arquivo .si? (Arquivo de imagem softimage)

Arquivo criado pela Softimage, um software de animação de personagens e efeitos; armazena os gráficos para o personagem em um formato proprietário; usado para criar animações 2D ou 3D.

Mais Informações

A Softimage pertencia anteriormente à Softimage, Co. O software agora é de propriedade e desenvolvido pela Autodesk.

NOTA: A Autodesk descontinuou o suporte ao Softimage em 2016.

O que é um arquivo .si? (Arquivo de interleave de som)

Um arquivo SI é um arquivo salvo no formato Si (Sound Interleave, na prática), que é uma variação do Formato de Intercâmbio de Recursos (RIF). Contém ativos de jogo usados pela LEGO Island, um jogo de ação e aventura com tema LEGO para Windows lançado em 1997. Os arquivos SI normalmente armazenam arquivos de música .WAV e arquivos gráficos .BMP.

Mais Informações

Você provavelmente só encontrará arquivos SI se você possui uma cópia da LEGO Island e está procurando extrair os arquivos do diretório de instalação do jogo. Os arquivos SI não devem ser abertos por um usuário, em vez disso, o arquivo é referenciado pelo jogo para reproduzir música e mostrar gráficos durante a jogabilidade.

No entanto, alguns usuários gostam de modificar a jogabilidade e querem acesso aos ativos apresentados no jogo. Vários programas permitem que os usuários extraam arquivos dos arquivos LEGO Island SI, incluindo LEGO Island Rebuilder e LIME (LEGO Island Music Extractor).
