O que é um arquivo .u3p? (U3 Smart Application)

Um arquivo U3P é um aplicativo portátil armazenado em uma Unidade Inteligente U3, um tipo especial de unidade flash USB que inclui a plataforma U3 Launchpad. Contém executáveis do programa e dados de configuração. Os arquivos U3P permitem que um programa seja executado a partir da unidade flash em qualquer computador Windows e pode instalar alguns arquivos no computador local, se necessário.

Mais Informações

Os arquivos U3P são salvos em um formato de .ZIP compactado. Eles podem conter quatro diretórios: data\, device\, host\ e manifest\. O diretório host\ contém o arquivo .EXE. O diretório manifest\ salva as informações de configuração do aplicativo (.U3I arquivo) e o ícone do programa.

Muitas vezes o U3 Launchpad é instalado em unidades inteligentes U3 por fabricantes como SanDisk, Verbatim, Kingston, Memorex e Intuix. No entanto, para instalar o U3P Launchpad manualmente, baixe e execute o U3 Launchpad Installer, que é disponibilizado livremente da SanDisk. O programa U3 Launchpad pode ser removido de unidades flash USB usando a ferramenta de remoção do Launchpad da SanDisk.

O programa U3 Launchpad é executado quando a unidade flash USB é inserida na porta USB do computador. Os programas U3P são então disponibilizados a partir de uma interface de usuário semelhante ao menu Windows Iniciar.

NOTA: O Launchpad SanDisk U3 foi descontinuado e não está mais mantido.
