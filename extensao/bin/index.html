O que é um arquivo .bin? (Imagem do disco binário)

Imagem de disco criada a partir de um disco óptico; armazena os dados binários copiados de um CD ou DVD; pode ser montado pelo computador e reconhecido como um disco físico; semelhante ao formato padrão de imagem do disco .ISO.

Mais Informações

Os arquivos BIN são frequentemente salvos com outros arquivos de metadados que descrevem o conteúdo do disco. Por exemplo, eles são muitas vezes salvos com um arquivo .CUE que descreve o pedido de dados na imagem binária.

Se um arquivo BIN não for aberto, ele poderá ser aberto por alguns utilitários de disco alterando a extensão de "bin" para "iso"

O que é um arquivo .bin? (Arquivo de política de TI blackberry)

Arquivo de configuração nomeado policy.bin e usado para fazer atualizações administrativas para o BlackBerry Desktop Software e o BlackBerry Wireless Handheld; muitas vezes distribuído pelos administradores do sistema de TI para computadores da empresa; pode ser armazenado na área de trabalho ou em outros locais especificados pelo administrador do sistema ao distribuir atualizações de configuração.

Mais Informações

O arquivo policy.bin é criado pelo IT Policy Manager, um programa incluído no BlackBerry Enterprise Server. Normalmente, apenas os administradores do sistema de TI da empresa interagem com este software para criar e distribuir os arquivos BIN.

Quando o Software BlackBerry Desktop é aberto, ele procura o arquivo policy.bin para fazer quaisquer alterações de política no software e em dispositivos BlackBerry conectados. A localização do arquivo é especificada no registro de Windows com a tecla HKEY_CURRENT_USERSoftwareResearch In MotionBlackBerryPolicy ManagerPath.

Se essa chave não existir, o diretório C:\Program Files\Research In Motion\BlackBerry\ é pesquisado em vez disso.

O que é um arquivo .bin? (Arquivo binário genérico)

Um arquivo BIN é um arquivo que armazena dados em um formato binário. É diferente de um arquivo baseado em texto, que pode ser editado em um editor de texto. Os arquivos BIN podem ser criados por uma variedade de programas diferentes, mas normalmente não podem ser editados manualmente.

Mais Informações

As imagens em disco são tipicamente arquivos binários, embora muitas vezes usem outras extensões de arquivo. Os arquivos BIN também podem ser usados para atualizadores de firmware.

O que é um arquivo .bin? (Arquivo codificado macbinário)

Arquivo compactado que é codificado como binário ou MacBinary; armazena um cabeçalho localizador, garfo de dados e garfo de recursos; garante que o garfo de recursos do Macintosh Hierarchical File System (HFS) e o garfo de dados sejam mantidos em um arquivo; comumente usados para transferir arquivos Macintosh pela Internet.

Mais Informações

Existem vários formatos MacBinary: MacBinary, MacBinary II e MacBinary III. Os arquivos MacBinary foram largamente substituídos por arquivos .DMG.

Desenvolvedor Nintendo

O que é um arquivo .bin? (Arquivo de dados do Nintendo Wii)

Um arquivo BIN é um arquivo de dados criado pelo Nintendo Wii, um console de videogame. Contém informações sobre um canal Wii ou jogo salvo. Os arquivos BIN armazenam parcialmente informações em texto simples, mas a maioria das informações é armazenada em um formato criptografado que só pode ser lido por um Wii.

Mais Informações

Os arquivos Nintnedo Wii BIN são salvos no seguinte local em um cartão SD inserido no slot SD frontal do Wii:

F:/private/wii/title/xxxx/

O xxxx é a abreviação de quatro caracteres do jogo ou canal Wii para o qual o arquivo contém dados.

NOTA: Não há um programa conhecido que possa abrir os arquivos BIN de dados do Nintendo Wii. No entanto, os arquivos BIN podem ser abertos em um Wii, desde que seja o Wii original usado para copiar o arquivo BIN para o cartão SD.

O que é um arquivo .bin? (Arquivo firmware do roteador)

Para obter instruções específicas para atualizar o firmware do roteador, consulte a documentação do fabricante.

Sega Genesis Jogo ROM

Desenvolvedor N/A

O que é um arquivo .bin? (Sega Genesis Jogo ROM)

Um arquivo BIN é uma imagem ROM de um jogo de vídeo Sega Genesis. Ele armazena dados despejados de um cartucho de jogo Sega Genesis, que é uma cópia exata do jogo. Os arquivos BIN podem ser reproduzidos em um computador usando um programa de emulação Sega Genesis.

Mais Informações

Sega Genesis game ROMs são tipicamente usados por entusiastas de jogos, especificamente jogadores nostálgicos que gostaram do console na década de 1990. Enquanto os jogos foram desenvolvidos para serem jogados pelo console de jogos real, vários emuladores foram desenvolvidos para jogar os jogos em computadores.

Existem uma variedade de emuladores sega genesis para Windows, macOS e Linux. Alguns dos emuladores são Kega Fusion, Genesis Plus, HazeMD, Gensplus e DGen.

NOTA: As ROMs da Sega Genesis também podem ser armazenadas em arquivos .SMD.
