O que é um arquivo .idap? (Pacote de atribuição do Adobe InDesign)

Um arquivo IDAP é um pacote usado pelo InDesign, um aplicativo de publicação de desktop usado para criar layouts de página profissionais. Ele contém um subconjunto de um documento InDesign (arquivo .INDD) que foi modificado por um editor. Os arquivos IDAP são usados para importar o conteúdo da página editada de volta para o documento original do InDesign.

Mais Informações

Antes de criar um arquivo IDAP, um editor recebe primeiro um arquivo .ICAP, que foi exportado do InDesign. O editor modifica o arquivo ICAP usando o Adobe InCopy, exporta o arquivo IDAP e, em seguida, envia o arquivo IDAP para o remetente InDesign original.

NOTA: Os arquivos IDAP são usados pelo InCopy e indesign CS4 e posteriormente. A versão CS3 usa a extensão .INDP.
