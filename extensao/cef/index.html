O que é um arquivo .cef? (Pacote criptografado centurionmail)

Arquivo criptografado criado pelo CenturionMail, um programa usado para criptografar e-mails e anexos; usa compressão e criptografia, bem como uma senha necessária para extrair o arquivo.

Mais Informações

Os arquivos CEF são usados para enviar arquivos com segurança por e-mail. Você precisa do software CenturionMail para abri-los.

NOTA: Por padrão, o CenturionMail cria arquivos com as extensões .ZIP, .CAB ou .EXE. No entanto, você também pode usar a opção "Extensão de arquivo personalizado" para criar arquivos CEF.

O que é um arquivo .cef? (Arquivo Elementar do Class Action Gradebook)

Arquivo criado pela Edição Elementar do Class Action Gradebook, programa utilizado por professores para o registro de notas do ensino fundamental; salva a lista de aula e atribuições, bem como as pontuações avaliadas; pode ser usado para gerar boletins.
