O que é um arquivo .pro? (Arquivo de design do Adobe Proto)

Arquivo criado pelo Adobe Proto, um Aplicativo Adobe Touch usado para criar wireframes interativos e protótipos de sites; pode armazenar vários layouts de página, cada um com elementos interativos; suporta tecnologias CSS, WebKit e jQuery.

Mais Informações

Os projetos do Adobe Proto podem ser renderizados e visualizados dentro dos navegadores Web padrão. Projetos salvos podem ser carregados no Adobe Creative Cloud e depois baixados no Dreamweaver para uma edição mais extensa.

O que é um arquivo .pro? (Arquivo de código fonte do IDL)

Um arquivo PRO contém o Interactive Data Language (IDL), uma linguagem de programação usada para análise de dados em campos científicos. Armazena funções e procedimentos em texto simples. Os arquivos PRO são normalmente criados usando o aplicativo de ambiente de desenvolvimento IDL.

Mais Informações

Você pode criar um arquivo PRO no IDL selecionando File → Save As.... Você também pode selecionar File → Generate .pro para gerar um arquivo PRO a partir de definições de interface GUIBuilder. Esta opção só está disponível em Windows. Você pode abrir o arquivo no IDL ou em outro editor de texto, uma vez que a linguagem de programação é armazenada em texto simples.

IDL é usado principalmente em áreas científicas, como imagem médica, física atmosférica e astronomia. É usado para visualizar dados numéricos complexos, a fim de ajudar cientistas e engenheiros a interpretar mais facilmente dados.

NOTA: O IDL é desenvolvido por David Stern e ITT Visual Information Solutions.

O que é um arquivo .pro? (Projeto KiCad)

Arquivo de projeto criado pela KiCad, um programa gratuito de automação de design eletrônico (EDA) usado para criar projetos de PCB e outros esquemas eletrônicos; pode ser aberto dentro da versão Windows, Linux ou Mac do software.

Mais Informações

Os arquivos do projeto KiCad armazenam alguns parâmetros comuns para todos os esquemas (.SCH arquivos) e layouts de PCB (.BRD arquivos) no projeto.

O que é um arquivo .pro? (Arquivo do projeto QT)

Arquivo de projeto criado usando o Qt software development toolkit (SDK), que permite aos desenvolvedores criar aplicativos multiplataforma; contém todas as informações necessárias para compilar um aplicativo Qt com o comando qmake.

Mais Informações

ARQUIVOS PRO incluem referências a bibliotecas de projetos, ativos e arquivos de código fonte, bem como outros arquivos, como recursos de aplicativos (arquivos .QRC), projeto inclui (.PRI arquivos), fontes de tradução (arquivos .TS), livros de frases (arquivos .QPH) e folhas de estilo (.QSS arquivos).

Os projetos Qt são usados para criar aplicativos executados no framework Qt. Eles são comumente usados para criar aplicativos móveis.

NOTA: Qt SDK foi desenvolvido anteriormente pela Nokia antes da Digia adquirir o software.

O que é um arquivo .pro? (Arquivo de definição de processo turbointegrator)

Um arquivo PRO define um processo TurboIntegrator. TurboIntegrator é uma ferramenta usada para criar scripts automatizados para executar várias tarefas de gerenciamento de dados dentro do sistema IBM TM1. Os arquivos PRO são frequentemente encontrados no diretório de dados de um servidor TM1.

Mais Informações

OS arquivos PRO são um dos arquivos de definição usados dentro do sistema de planejamento corporativo IBM Cognos TM1, agora parte do IBM Planning Analytics a partir de novembro de 2016. Os processos TurboIntegrator podem ser criados e editados no Espaço de Trabalho do Planning Analytics.
