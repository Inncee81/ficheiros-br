O que é um arquivo .eddx? (Edraw XML FIle)

Um arquivo EDDX é um arquivo criado por Edraw Max e Network Diagram Maker, aplicativos de diagrama profissional. Ele contém um diagrama, que pode incluir texto, formas, setas, conectores ou imagens. Os arquivos EDDX são usados para criar vários tipos de diagramas, como fluxogramas, mapas mentais, planos de piso, infográficos e gráficos organizacionais.

Captura de tela de um arquivo eddx em EdrawSoft Edraw Max 86

Quando você cria um novo diagrama e o salva no Edraw Max, o programa o salva no formato EDDX por padrão. Você também pode optar por salvá-lo como um formato XML. Para criar um arquivo EDDX, File → New, adicione suas informações e selecione File → Save ou Save As.

Quando você cria um novo diagrama, você pode começar com um desenho em branco ou você pode começar a partir de um dos muitos modelos incluídos com Edraw Max. Algumas das categorias de modelos incluem Diagrama de Negócios, Engenharia, Banco de Dados, Designer de Moda, Automação Industrial, Software, Ciência e Gerenciamento de Projetos.

NOTA: O formato EDDX foi introduzido em 2013 com o lançamento da versão 73 do Edraw Max. Ele possui uma relação de compressão melhor, o que leva a um tamanho de arquivo menor do que .EDX arquivos.
