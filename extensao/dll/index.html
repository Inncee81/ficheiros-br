O que é um arquivo .dll? (Página de Web Dinâmica DLL)

Web página gerada dinamicamente quando acessada por um usuário; pode incorporar scripts do lado do servidor, como scripts Perl ou VB que geram o HTML para a página Web.

Mais Informações

Web páginas com extensões de arquivo DLL são normalmente hospedadas em servidores Web Microsoft IIS. A extensão DLL deve ser mapeada para o mecanismo de scriptagem correto no servidor para que a página seja processada corretamente.

O que é um arquivo .dll? (Biblioteca de Links Dinâmicos)

Um arquivo DLL é uma biblioteca compilada que contém um conjunto de procedimentos e/ou drivers que são referenciados e executados por um programa de Windows. Permite que vários programas acessem funções compartilhadas através de bibliotecas comuns. Os arquivos DLL são "dinamicamente ligados" em um programa em tempo de execução, o que significa que eles só são carregados quando necessário.

Mais Informações

Muitos arquivos DLL são fornecidos com o sistema operacional Windows, enquanto outros são incluídos com programas Windows. Eles permitem várias funções do programa, como comunicação com dispositivos externos e leitura e escrita de arquivos. Em alguns casos, os DLLs são referenciados por aplicativos multiplataforma em execução no macOS.

Excluir ou modificar o conteúdo de um arquivo DLL pode causar erros dentro do programa que faz referência ao arquivo. Por essa razão, os arquivos DLL geralmente não devem ser abertos ou editados. No entanto, se um aplicativo tentar acessar um arquivo DLL que foi corrompido ou está faltando, esse programa não funcionará mais. Se isso acontecer, você pode precisar reinstalar o programa ou baixar e instalar uma nova versão do DLL.
