O que é um arquivo .obml15? (Página de Web Mini Salva da Ópera)

Página web salva criada pelo Opera Mini, um navegador web para plataformas móveis; salvo no formato Opera Binary Markup Language (OBML), que só pode ser aberto com o navegador Opera Mini.

Mais Informações

O formato de arquivo OBML15 foi projetado para reduzir o tamanho do conteúdo da Web baixado, o que é útil para usuários que desejam salvar e revisitar páginas da Web sem uma conexão com a Internet. Também é útil se você estiver lidando com baixa potência ou memória em seu dispositivo ou baixando conteúdo em uma rede lenta.

O arquivo OBML15 substituiu a extensão .OBML com o lançamento da versão 5 do navegador Opera Mini. Foi então substituído pela extensão .OBML16 com o lançamento da versão 7 do navegador Opera Mini.

NOTA: O arquivo OBML15 não é suportado pelo navegador Opera Mini 7 ou posterior.
