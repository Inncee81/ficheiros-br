O que é um arquivo .bnk? (Banco de Instrumentos Adlib)

Bancos de instrumentos acessados pelo Adlib Visual Composer, um programa de composição musical; projetado para serem carregados com .ROL arquivos; contém dados de instrumentos personalizados para acompanhar os arquivos ROL que contêm as notas para tocar os instrumentos que compõem as partituras.

Mais Informações

O arquivo bancário usado pelo Adlib Tracker II tem o programa orientado ao DOS e pode ser executado em Windows com um plugin DOS.

O que é um arquivo .bnk? (Arquivo do banco de perguntas do ExamView)

Um arquivo BNK é um arquivo de dados criado pela Turning Technologies ExamView, um programa usado para criar atribuições e perguntas de avaliação para avaliar o desempenho dos alunos. Ele contém um banco de uma ou mais perguntas criadas pela ferramenta Gerador de Teste incluída no ExamView. Os arquivos BNK podem ser abertos pelo Editor do Banco de Perguntas, que faz parte do ExamView.

Mais Informações

Para abrir um arquivo BNK no ExamView, você deve estar usando o módulo Editor do Banco de Perguntas no ExamView. Você pode identificar qual módulo você está usando olhando para o nome do módulo ativo exibido na barra de título da janela ExamView. Para mudar para o módulo Editor do Banco de Perguntas, selecione File → Switch to Question Bank Editor.

O que é um arquivo .bnk? (Arquivo de lista de reprodução do Spotify)

Playlist criada pelo Spotify, um programa de streaming de música; contém seus dados de lista de reprodução, que são as faixas que você adicionou à lista de reprodução; você normalmente nunca as vê a menos que você cave através do código do programa.

Mais Informações

Em Windows, esses arquivos estão localizados no seguinte diretório C:\Users\.CONFIG\AppData\Roaming\Spotify\Users\USER_ID.

Para estar seguro e proteger sua lista de reprodução contra edições indesejadas ou exclusão acidental, você deve localizar seu arquivo específico de lista de reprodução no diretório e copiá-lo para outra pasta em seu computador para segurança.

O que é um arquivo .bnk? (Pacote de dados ilimitados do Test Drive)

Formato de pacote usado pelo Test Drive Unlimited (TDU), um jogo de corrida de carros esportivos; contém arquivos de dados individuais do jogo, como modelos 3D, efeitos sonoros e arquivos de música.

Mais Informações

O conteúdo de um arquivo BNK pode ser extraído usando um utilitário de terceiros, como o TDU Modding Tools. Os arquivos BNK editados podem ser usados para modificar a jogabilidade do Test Drive Unlimited.
