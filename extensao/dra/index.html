O que é um arquivo .dra? (Arquivo de desenho Dolphin PartMaster)

Arquivo criado pela Dolphin PartMaster, um conjunto de aplicativos CAD/CAM usado para projetar e fabricar componentes; contém instruções de controle numérico para dirigir uma ferramenta de máquina, como uma broca, durante a produção.

Mais Informações

O programa Dolphin PartMaster CAD é usado para projetar a peça, que é então passada para o programa PartMaster CAM para gerar caminhos de ferramentas e suas coordenadas.

NOTA: A PartMaster na verdade inclui vários programas no pacote de software para concluir projetos e gerar caminhos de ferramentas.

O que é um arquivo .dra? (Arquivo de gravação de voz do Dragão nuance)

Arquivo de áudio salvo pelo Dragon, um programa usado para gerar documentos de texto a partir de gravações de voz; armazena o áudio como foi ditado pelo usuário; salvo para que o texto gerado do software possa ser corrigido mais tarde se não corresponder ao ditado.

Mais Informações

Dragon usa tecnologia de reconhecimento de voz para traduzir dados de áudio falado em documentos de texto.

O que é um arquivo .dra? (Arquivo de desenho OrCAD)

CAD drawing criado pelo PCB Editor, um aplicativo de design de placa de circuito impresso (PCB) incluído no pacote de software OrCAD PCB Designer; armazena o esquema para um PCB desenhado, que especifica o layout dos componentes do circuito.

O que é um arquivo .dra? (Arquivo scriptor script)

Um arquivo DRA é um arquivo de script criado pelo Scriptor, um aplicativo de roteiro popular nas décadas de 1980 e 1990. Contém texto formatado em um roteiro, que inclui cenas, ator e notações de adereços.

Mais Informações

Scriptor é usado para formatar texto salvo em um documento, como um arquivo .DOC ou .DOCX, em formato de roteiro. A formatação inclui cena, personagem, adereço e organização de localização de tiro. A formatação também inclui a porcentagem do roteiro que uma cena ocupa.

Depois de formatar um documento no Scriptor, o documento é convertido em um arquivo DRA. O único programa que pode abrir o arquivo DRA é o Scriptor. No entanto, o programa não pode editar o arquivo DRA, apenas abri-lo. Infelizmente, como o Scriptor não é mais desenvolvido, existem muitos arquivos de script DRA que são essencialmente "bloqueados"para sempre.
