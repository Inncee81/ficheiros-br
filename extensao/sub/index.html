O que é um arquivo .sub? (Arquivo subcanal cloneCD)

Arquivo de imagem de disco criado no formato subcanal do CloneCD; usado com um arquivo .CCD e .IMG para representar a imagem de um disco copiado de um CD ou DVD; normalmente usa o mesmo prefixo de nome de arquivo que os arquivos CCD e IMG (por exemplo, image.ccd, image.img e image.sub).

O que é um arquivo .sub? (Arquivo de nível SubTerra)

Nível de jogo para SubTerra, um jogo de quebra-cabeça 2D que exige que os jogadores usem o planejamento atencioso para coletar joias e completar cada nível; contém informações de layout de nível, bem como objetos e eventos incluídos no nível.

O que é um arquivo .sub? (Arquivo de legenda)

Arquivo de texto contendo legendas usadas por MicroDVD, DivX e outros formatos de filme; inclui texto de legenda e pontos de sinalização para quando cada legenda deve ser exibida; muitas vezes salvo junto com um arquivo de vídeo correspondente e referenciado durante a reprodução.

Mais Informações

Outro formato de legenda comumente usado é o formato SubRip (.SRT).
