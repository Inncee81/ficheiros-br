O que é um arquivo .EX_? (Arquivo executável compactado)

Windows arquivo executável (.EXE) compactado usando alguma forma de compressão e renomeado de "exe" para "ex_;" muitas vezes encontrados em CDs de instalação de software, onde o instalador deve primeiro renomear o arquivo e descompactá-lo antes da execução; útil para diminuir o tamanho de um arquivo EXE ao empacotar instaladores em mídia externa ou dentro de arquivos para download.

Mais Informações

Arquivos EX_ são comumente criados usando o Makecab, um utilitário de compactação de arquivos incluído com Windows. Aqui está a sintaxe de exemplo para criar o arquivo program.ex_ do arquivo program.exe:

makecab programexe

EX_ arquivo criado com o Makecab pode ser descomprimido de volta à sua forma original usando o programa expand.exe incluído com Windows. Ele usa a seguinte sintaxe:

expandir programex_ programexe

O que é um arquivo .EX_? (Arquivo executável Windows renomeado)

Um arquivo EX_ é um arquivo Windows executável renomeado de "exe" para "ex_". Ele contém o conteúdo exato de um arquivo .EXE. Arquivos EX_ são comumente encontrados em CDs de instalação, onde o instalador deve primeiro renomear o arquivo antes de executá-lo.

Mais Informações

Os arquivos EX_ renomeados são úteis para evitar execuções acidentais ou indesejadas em Windows. Por exemplo, se o usuário acidentalmente clicar duas vezes no arquivo, ele não será executado por padrão.

Para tornar um arquivo EX_ executável, renomeie a extensão "ex_" para "exe". Agora você pode clicar duas vezes no arquivo como qualquer outro arquivo EXE.
