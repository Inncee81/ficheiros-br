O que é um arquivo .dmo? (Cubo 2: Arquivo de demonstração sauerbraten)

Arquivo de jogo criado por Cube 2: Sauerbraten, um jogo de tiro em primeira pessoa gratuito; contém uma gravação de uma única ou sessão multiplayer; inclui os movimentos do jogador, eventos de jogo e registro de bate-papo; permite que Cube 2: Sauerbraten replique um replay exato do jogo.

Mais Informações

Cube 2: Os clientes do jogo Sauerbraten podem baixar arquivos DMO do servidor host que gravou a demonstração. Uma vez baixados, eles podem ser reproduzidos no modo local.

NOTA: Os arquivos DMO não contêm dados de vídeo. Se você quiser gravar uma sessão de vídeo, use a funcionalidade de gravação de filme do Cube 2: Sauerbraten para salvar um arquivo .AVI.

O que é um arquivo .dmo? (Duke Nukem 3D Arquivo de demonstração)

Arquivo criado por Duke Nukem 3D, um jogo de tiro em primeira pessoa desenvolvido com o motor do jogo Build; contém uma gravação de jogos de ações de jogadores e eventos mundiais; permite que os jogadores reproduzam sessões multiplayer.

O que é um arquivo .dmo? (Arquivo de banco de dados do movienizer)

Banco de dados de filmes criado com o Movienizer, um programa de organização cinematográfica; inclui informações sobre cada filme, como atores, produtores e pôsteres de cores; também mantém o controle de quais filmes são de propriedade, emprestados a outros ou em uma lista de desejos.
