O que é um arquivo .lsf? (Arquivo do sistema de biblioteca de logotipos)

Arquivo de dados usado pelo Libronix Digital Library System (DLS), um programa usado por outros aplicativos para gerenciamento de livros digitais; salvo ao lado de um arquivo .LIX, ambos devem estar presentes para ler o livro.

Mais Informações

Libronix DLS é uma subsidiária da Logos Bible Software e é usada no software Logos para baixar traduções da Bíblia e outros livros religiosos.
