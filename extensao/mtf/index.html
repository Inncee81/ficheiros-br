O que é um arquivo .mtf? (Arquivo de tabela de mortalidade)

Arquivo de dados usado pela XLActuary, um complemento do Excel usado para lidar com cálculos de fatores atuariais para empresas de consultoria e provedores de administração de aposentadoria; contém taxas de mortalidade para tabelas padrão e personalizadas usadas pelo complemento; usado para fazer cálculos estatísticos.

O que é um arquivo .mtf? (Arquivo temático motorola)

Arquivo temático usado por celulares da Motorola; armazena configurações de toque, papel de parede e protetor de tela e pode ser usado para personalizar a aparência e a sensação do telefone; pode ser criado usando o Motorola Media Manager.

Mais Informações

NOTA: Motocast USB era anteriormente conhecido como Media Manager e Media Link.

Formato Binário 

O que é um arquivo .mtf? (Arquivo multi tracker)

Arquivo de áudio usado pelo Mult Tracker, um programa que permite que os músicos toquem músicas removendo vocais ou instrumentos; salva uma "backing track"," que é um remake de uma música existente que divide diferentes partes musicais em várias faixas.

Mais Informações

Ao reproduzir um arquivo MTF, os usuários podem ajustar o volume de instrumentos e vocais individualmente.

  Song Galaxy Multi Tracker

Desenvolvedor SDL
Categoria Diversos
Formato Não Disponível 

O que é um arquivo .mtf? (Arquivo de traduções de memória de tradução)

Support file for a translation memory used by Trados translation software; suporta traduções rápidas de palavras e frases contidas em uma memória de tradução.

Mais Informações

Uma memória de tradução (TM) inclui cinco tipos de arquivos, incluindo arquivos MTF, .TMW, .IIX, .MDF e .MWF.
