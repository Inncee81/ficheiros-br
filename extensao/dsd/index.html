O que é um arquivo .dsd? (Arquivo de descrição do conjunto de desenho autocad)

Arquivo de propriedades usado pelo AutoCAD, um aplicativo de produtividade CAD; contém configurações para publicar um desenho em um arquivo .DWF ou em outro formato plotter; usado para salvar configurações de publicação reutilizáveis para diferentes tipos de saída.

Mais Informações

DSD arquivos salvar as seleções de caixa de diálogo para o processo de publicação usado pelo comando PUBLISH. Ao ler um arquivo DSD, o processo de publicação preenche automaticamente a caixa de diálogo com as seleções pré-salvas para que nenhuma interação do usuário seja necessária.

O que é um arquivo .dsd? (Arquivo morph objeto do estúdio DAZ)

3D arquivo de configurações de imagem criado pelo DAZ Studio, um aplicativo de modelagem 3D; salva dados de morfina (alteração de forma/posição) para objetos em uma cena 3D; referenciado por um arquivo de cena .DAZ e é usado com .DSO e .DSV arquivos para compor a cena.

O que é um arquivo .dsd? (Arquivo de dados do Diet Studio)

Arquivo de dados criado pelo Diet Studio, um programa de pesquisa e desenvolvimento de dieta; contém informações sobre planos de dieta e dados relacionados; salvo em um formato XML, que permite que os dados sejam facilmente convertidos em diferentes formatos.
