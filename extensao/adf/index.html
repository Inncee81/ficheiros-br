O que é um arquivo .adf? (ACT! Arquivo de dados)

Um arquivo ADF é um arquivo de banco de dados criado pelo ACT!, um aplicativo de CRM (Customer Relationship Management, gerenciamento de relacionamento com o cliente). Ele contém dados de contato do cliente, como nomes, endereços de e-mail, informações de negócios e histórico de relacionamento. Os arquivos ADF são usados para informações de relacionamento com o cliente em um formato estruturado.

Mais Informações

Os arquivos ADF são salvos com um arquivo de pesquisa .ALF, que contém metadados necessários para navegar, fazer backup e restaurar o banco de dados ACT!. Os arquivos ADF também podem ser arquivados nos backups do banco de dados ACT!, que usam a extensão .BAK. Os arquivos ADF usam a formatação do Microsoft SQL Server e são semelhantes aos arquivos .MDF.

NOTA: Sage ACT! tornou-se Swiftpage Act! em 2013.

Desenvolvedor PY Software

O que é um arquivo .adf? (Arquivo de desenho real)

Um arquivo ADF contém um design de página web criado pela PY Software Actual Drawing, uma ferramenta visual de autoria da Web. Ele armazena um "drawing" de uma página web, que inclui a estrutura da página web, juntamente com cores, formas, imagens, caixas de texto, tabelas e animações incluídas na página. Os arquivos ADF podem ser exportados para .HTML arquivos para publicação on-line quando concluídos.

Mais Informações

Você provavelmente só encontrará arquivos ADF de desenho real se você usou o Desenho Real para criar um site no início dos anos 2000. Como o Desenho Real não é mais desenvolvido e seus recursos estão desatualizados, os arquivos ADF são incomuns.

Se você encontrar um arquivo ADF e precisar abri-lo, você pode abri-lo com o Desenho Real selecionando File → Open.... Em seguida, você pode exportar o design para um arquivo HTML selecionando File → Export to HTML file....

O que é um arquivo .adf? (Arquivo de disco Archimedes)

Um arquivo ADF é uma imagem de disco usada pelo sistema Archimedes incluída em computadores domésticos Acorn Archimedes durante as décadas de 1980 e 1990. Ele contém uma cópia exata de um disquete físico formatado de Arquimedes, que foi usado para distribuir software, como jogos ou outros programas.

Mais Informações

A família de computadores Archimedes foi descontinuada em meados da década de 1990. Mas se você possui um computador Archimedes ou é apenas um técnico retrô curioso, você pode querer montar um arquivo ADF com um emulador archimedes para usar o software.

Existem vários emuladores de Arquimedes disponíveis que podem montar um arquivo ADF. Alguns exemplos são Arculator, ArcEm e Archie.

O que é um arquivo .adf? (Documento Expresso ARIS)

Um arquivo ADF é um documento criado pelo Software AG ARIS Express, um programa gratuito usado para modelar análise e gerenciamento de processos de negócios. Contém um diagrama, que consiste em símbolos usados para modelar e organizar um processo relacionado aos negócios.

Mais Informações

Captura de tela de um arquivo adf no Software AG ARIS Express 24

Os arquivos ADF podem ser usados para armazenar uma variedade de tipos de modelos, como gráficos organizacionais, modelos de dados, quadros brancos, paisagens de sistemas, paisagens de processos, processos de negócios e infraestruturas de TI. Você pode escolher entre uma variedade de símbolos para organizar melhor o diagrama. Alguns dos símbolos que podem ser adicionados são eventos, tarefas, subprocessos, objetos de dados, mensagens, anotações de texto, conexões e gateways.

O arquivo ADF é salvo em um formato proprietário e é o tipo de arquivo principal usado para exportar diagramas no ARIS Express. No entanto, o programa também pode exportar diagramas para outros formatos, incluindo .JPEG, .PNG, .PDF e .EMF. Para exportar um arquivo ADF para um desses formatos, selecione File → Export e escolha o formato.

Para criar um arquivo ADF, selecione File → New..., escolha o tipo de modelo e selecione File → Save.

Para abrir um arquivo ADF, selecione File → Open... e navegue até a localização do arquivo ADF. Você também pode escolher File → Model history para abrir um modelo recém-aberto.

O que é um arquivo .adf? (Formato da grade binária ESRI ArcInfo)

Formato interno de dados raster usado por produtos ESRI como ArcGIS, ArcView e ArcInfo Workstation; armazena dados espaciais como uma grade binária (linhas e colunas de células) e é um dos vários arquivos ADF que juntos compõem a grade total; usados para representar objetos geográficos ou outros espaciais, como mapas e características do mapa.

Mais Informações

Uma grade binária ArcInfo pode conter várias camadas raster, formando uma pilha de camadas. Isso pode ser usado para sobrepor diferentes pontos de vista sobre superfícies e realizar análises estatísticas sobre os dados espaciais.

Os arquivos ADF podem armazenar metadados sobre o conjunto raster ou dados rasteres reais. Quando você cria uma grade binária, os arquivos chamados dblbnd.adf, hdr.adf, sta.adf, vat.adf e w001001x.adf fornecem metadados sobre a grade raster, enquanto o arquivo w001001.adf contém os dados reais raster. Esses arquivos são normalmente separados em sua própria pasta para que sejam fáceis de gerenciar como um conjunto inteiro de cobertura de grade.

O que é um arquivo .adf? (Grand Theft Auto: Vice City Audio Data File)

Um arquivo ADF contém dados de áudio usados por Grand Theft Auto: Vice City (GTAVC), um videogame onde jogadores roubam carros e missões completas. Ele armazena entrevistas de DJ de música ou rádio tocadas por uma estação de rádio específica que pode ser sintonizada e reproduzida ao dirigir um veículo durante a jogabilidade.

Mais Informações

Você provavelmente só encontrará arquivos GTAVC ADF se você estiver olhando no diretório de instalação do jogo para modificar a música reproduzida pela estação de rádio no jogo ou converter as músicas em um formato de áudio diferente e mais amplamente suportado. Há um arquivo ADF para cada estação de rádio.

Os arquivos ADF não devem ser abertos manualmente. Em vez disso, eles são referenciados pelo jogo GTAVC. No entanto, os jogadores que procuram converter arquivos ADF podem usar o utilitário Vice City Audio Converter desenvolvido pela Raptor para convertê-los em arquivos de áudio .MP3.
