O que é um arquivo .ips? (Arquivo de correção interna do sistema de patches)

Patch (ou atualização) para uma ROM de jogo comum que pode conter gráficos, modelos e outros dados; usado principalmente para pequenos patches com menos de 16MB de tamanho; pode ser modificado e aplicado a jogos usando vários utilitários de software IPS.
