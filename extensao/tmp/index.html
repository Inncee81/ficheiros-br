O que é um arquivo .tmp? (Arquivo temporário finale)

Arquivo criado pelo Finale, um programa de notação de música para instrumentos, vocais e muito mais; contém dados de backup para o documento em que você está trabalhando; atua semelhante a um arquivo de cache; só existe enquanto você estiver trabalhando no Finale e excluído quando o programa é fechado; não pode ser aberto manualmente.

Mais Informações

NOTA: Você não pode criar manualmente um arquivo TMP no Finale e provavelmente nunca verá um.

O que é um arquivo .tmp? (Arquivo Temporário)

Um arquivo TMP é um arquivo temporário criado automaticamente por um programa de software que geralmente serve como um arquivo de backup ou cache. Às vezes é criado como um arquivo invisível e muitas vezes é excluído quando o programa é fechado. Arquivos TMP também podem ser criados para conter informações temporariamente enquanto um novo arquivo está sendo gerado.

Mais Informações

Muitas vezes um arquivo temporário é composto por dados brutos que são usados como uma etapa intermediária no processo de conversão do conteúdo de um formato para outro. Alguns programas que criam e usam arquivos TMP incluem o Microsoft Word e o Apple Safari.

NOTA: Os arquivos TMP são frequentemente chamados "arquivos temporários"
