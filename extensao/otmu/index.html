O que é um arquivo .otmu? (Modelo de matriz Unicode de Origem)

Um arquivo OTMU é um modelo matricial criado pela Origin 2018 e, posteriormente, um aplicativo de análise e gráfico de dados usado por cientistas e engenheiros em vários campos. Contém configurações de matriz, que são usadas para criar gráficos .OGMU. Os arquivos OTMU armazenam uma ou mais folhas, o número de linhas e colunas para cada folha, funções matemáticas e o formato e estilo.

Mais Informações

Os arquivos OTMU substituíram .OTM arquivos, que foram usados para salvar modelos de matriz no Origin antes do lançamento do Origin 2018. Os arquivos OTMU são salvos em um formato unicode que apresenta vantagens de desempenho sobre o formato de arquivo OTM mais antigo.

Você só pode abrir arquivos OTMU com o Origin, que podem ser usados para criar, abrir e editar arquivos OTMU. O programa só está disponível para Windows.

Para abrir um arquivo OTMU na Origem, selecione File → Open..., selecione o tipo de arquivo "Modelo de origem otmu", navegue até o local do arquivo OTMU e clique em Open.
