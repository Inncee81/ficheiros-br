O que é um arquivo .dna? (Arquivo de backup e migração do CA)

Arquivo criado pelo CA Backup and Migration, um programa usado para fazer backup do "DNA"de um computador; contém configurações de rede, impressora, sistema e aplicativos, bem como arquivos, pastas, catálogos de endereços e fotos; usado para fazer backup das configurações do computador e transferir configurações de um computador para outro.

Mais Informações

Arquivos de DNA são auto-extrativáveis. Quando abertos, eles migram automaticamente as configurações e arquivos de backup para o novo computador.

O que é um arquivo .dna? (Arquivo de DNA genepool)

Arquivo de dados usado por GenePool, um jogo de simulação de vida artificial; contém o DNA de uma criatura que pode ser simulado com outras criaturas em um experimento virtual; às vezes usado para competições para ver quais criaturas personalizadas do jogador sobrevivem melhor do que outras quando simuladas juntas.

Mais Informações

NOTA: Arquivos de DNA também são usados por Darwin Pond, o antecessor de GenePool. No entanto, os arquivos de DNA de Darwin Pond usam um formato de texto simples, enquanto os arquivos de DNA genepool usam formatação binária. Os arquivos de DNA da Lagoa Darwin contêm uma lista fixa de parâmetros separados por quebras de linha, e cada arquivo tem o mesmo tamanho de arquivo.

O que é um arquivo .dna? (Arquivo de DNA SnapGene)

Arquivo de dados criado pela SnapGene, um programa de biologia molecular usado para analisar dados de DNA; armazena uma sequência de DNA e metadados XML e binários associados; também inclui o autor, uma descrição, comentários e outras informações escritas.

Mais Informações

Arquivos de DNA são usados para documentar, estudar e visualizar sequências de DNA. O SnapGene e o SnapGene Viewer gratuito fornecem a capacidade de visualizar os dados em várias visualizações diferentes para a realização de análises de DNA.
