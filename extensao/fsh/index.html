O que é um arquivo .fsh? (Arquivo shader fragmento)

Arquivo gráfico usado por aplicativos que usam a biblioteca de gráficos OpenGL; armazena uma função gráfica usada para sombrear um fragmento, que representa um pixel na imagem 3D; usado por GPUs para processamento de cores, sombras e outros efeitos.

Mais Informações

NOTA: Os sombreadores de fragmentos diferem dos sombreadores de vértices (.VSH arquivos), que representam os dados necessários para sombrear os vértices 3D.
