O que é um arquivo .webarchive? (Arquivo Web Safari)

Um arquivo WEBARCHIVE é uma página da Web salva pelo navegador Safari. Ele contém IMAGENS HTML e vinculadas para que a página possa ser carregada totalmente posteriormente, mesmo que o computador esteja offline (não conectado à Internet). Os arquivos WEBARCHIVE também podem conter CSS ou JavaScript, dependendo do conteúdo da página web.

Mais Informações

Os arquivos web do Safari podem ser vistos no Safari, bem como nos navegadores web iCab e Cruz. Eles também podem ser abertos com o Apple TextEdit, mas o conteúdo pode não aparecer corretamente.

NOTA: O safari foi desenvolvido para Windows até a versão 517, mas depois descontinuado.
