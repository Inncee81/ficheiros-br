O que é um arquivo .rex? (Arquivo de loop de reciclagem)

Formato de loop de áudio usado pela ReCycle, um programa de criação e edição de loop amostrado; armazena dados de áudio e informações de corte de loop; Os arquivos REX podem ser quantizados, ter o tempo alterado e podem ser sintonizados ou desajustados.

O que é um arquivo .rex? (Arquivo de origem rexx)

Arquivo de código-fonte escrito na linguagem de programação Desarmado Estendido Executor (Rexx), uma linguagem de script e macro usada principalmente para gerar relatórios e dados de processo; contém variáveis, funções e outras afirmações que informam o programa como funcionar.

Mais Informações

Se você alterar as permissões do arquivo REX para "executável", você poderá executar o arquivo. Os arquivos REX são colocados na biblioteca de origem REXX no local do Grupo de Trabalho por padrão.

NOTA: Existem vários intérpretes gratuitos do Rexx disponíveis, como Regina, Reginald e BRexx, que podem abrir o REX fie.
