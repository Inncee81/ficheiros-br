O que é um arquivo .GRF? (Gráfico de transformação de dados cloverdx)

Arquivo de transformação de dados criado pela Javlin CloverDX Designer, um aplicativo de integração de dados baseado no Eclipse; salvo usando formatação XML e contém um diagrama de leitores de dados, escritores, transformadores e outros manipuladores de dados; usado para integração de dados, limpeza, migração e armazenagem; pode ser executado em um computador desktop para trabalhos menores ou em um grupo de servidores CloverDX para trabalhos maiores.

Mais Informações

O pacote CloverDX inclui várias edições do software, incluindo versões Community, Desktop, Enterprise, Cluster e OEM. Os arquivos GRF são criados usando o CloverDX Designer, que está incluído a cada edição de software.

O Eclipse IDE é um aplicativo gratuito de desenvolvimento Java e o CloverDX Desktop é um conjunto de plugins eclipse proprietários que permitem que projetos CloverDX e arquivos GRF sejam criados.

NOTA: Javlin CloverDX era anteriormente conhecido como Javlin CloverETL.

O que é um arquivo .GRF? (Arquivo gráfico DPlot)

gráfico 2D ou 3D criado pela DPlot, um programa usado para a plotagem de dados científicos; salva o tipo de gráfico e dados associados; usado para gráficos de distribuição, malhas de wireframe, projeções 3D, gráficos polares, gráficos de probabilidade, gráficos de triângulo, escalas hidráulicas e muitos outros tipos de gráficos.

Mais Informações

DPlot é usado principalmente por cientistas, engenheiros e outros profissionais técnicos. Os gráficos são frequentemente incluídos em apresentações e artigos de revistas científicas.

NOTA: Os arquivos GRF podem ser visualizados com o visualizador DPlot gratuito.

O que é um arquivo .GRF? (Arquivo gráfico do grapher)

Arquivo criado pelo Grapher, um aplicativo usado para fazer gráficos 2D e 3D; pode armazenar um dos muitos tipos diferentes de gráficos, como uma linha, dispersão ou gráfico de barras; também suporta tipos avançados de plotagem, como um ploter ternary, box-whisker e gráficos de alta-proximidade.

Mais Informações

Os gráficos criados no Grapher são usados para criar gráficos técnicos para publicação e apresentações.

NOTA: Os arquivos GRF não contêm os dados de origem usados para gerar o gráfico. Eles armazenam apenas o gráfico renderizado e os parâmetros necessários para o dimensionamento. Isso difere de .GPJ arquivos, que contêm os dados, bem como o gráfico renderizado.

O que é um arquivo .GRF? (Arquivo de Clientes Ragnarok)

Arquivo de arquivos compactado usado pelo Ragnarok Online, um jogo multiplayer coreano; inclui imagens compactadas, sons e outros arquivos de dados carregados por Ragnarok.

O que é um arquivo .GRF? (Arquivo de recursos gráficos do magnata do transporte)

Arquivo de recursos usado pelos jogos de simulação de negócios ferroviários Do Magnata do Transporte (TT) e Transport Tycoon Deluxe (TTD); contém uma coleção de sprites, que armazenam os gráficos 2D para objetos de jogo; usados para exibir e animar veículos ferroviários, cenários e outros objetos de jogo.

Mais Informações

A versão original dos arquivos GRF suportava sprites simples. No entanto, as novas variantes de código aberto do Transport Tycoon, incluindo OpenTTD e TTDPatch, suportam novos dados gráficos em arquivos GRF. O novo formato da OpenTTD, chamado NewGRF, pode incluir dados para veículos, cargas, edifícios, tipos ferroviários e indústrias.

OpenTTD fornece pacotes de jogos GRF, que são arquivos NewGRF para download que podem ser aplicados ao jogo para exibir gráficos personalizados e alterar propriedades do jogo.
