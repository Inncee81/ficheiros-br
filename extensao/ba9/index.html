O que é um arquivo .ba9? (TaxAct 2019 Tax Return Backup File)

Um arquivo BA9 contém um backup de uma declaração de imposto criada pela versão 2019 do TaxAct, um programa de preparação fiscal para o ano fiscal de 2019. Armazena dados fiscais de 2019, que incluem dados da Receita Federal federal e do fisco estadual, além de informações pessoais básicas, como o nome e ocupação do usuário. Os arquivos BA9 são usados para fins de arquivamento e como backup do arquivo de declaração de imposto de renda .TA9 no caso de ele ser perdido ou se tornar corrupto.

Mais Informações

Você provavelmente só encontrará um arquivo BA9 se você arquivar seus impostos com o software TaxAct 2019. Cada versão anual do TaxAct usa uma extensão diferente para arquivos de backup. Por exemplo, um arquivo de backup de declaração de imposto criado pelo TaxAct 2017 usa a extensão .ba7, enquanto o TaxAct 2018 cria arquivos de backup com a extensão de arquivo .ba8. Os arquivos BA9 podem ser criados manualmente por um usuário ou gerados automaticamente pelo TaxAct 2019.

Para salvar manualmente um backup da sua declaração de imposto com o TaxAct 2019, selecione File → Backup... e nomeie seu arquivo. Você deve salvar sua declaração de imposto como um arquivo TA9 antes de criar seu backup. Para restaurar um backup no TaxAct 2019, selecione File → Restore Backup... e navegue até a localização do seu arquivo BA9.

TaxAct 2019 gera automaticamente três tipos de arquivos de backup de declaração de imposto BA9 e os armazena na pasta "AUTO-BACKUP". Os três tipos são o backup atual, um backup diário e um backup semanal. Você pode dizer o tipo a partir do nome do arquivo. Por exemplo, o arquivo de backup ba9 diário para uma declaração de imposto de renda de 2019 é .ta9_daily.ba9.
