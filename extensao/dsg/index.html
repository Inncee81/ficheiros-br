O que é um arquivo .dsg? (Documento do estúdio de controle)

Arquivo criado pelo Control Studio, um programa usado para simular e protótipo de sistemas eletrônicos; armazena componentes eletrônicos e conexões em uma tela de design; também pode armazenar texto e imagens escritos; usado para salvar e carregar projetos, que podem ser simulados no software.

Mais Informações

A janela Control Studio Gallery contém componentes eletrônicos para entradas, processos, amplificadores, drivers, saídas e medidas. Os usuários podem selecionar componentes dentro dessas categorias e colocá-los na tela de design.

O que é um arquivo .dsg? (Jogo salvo de Decker)

Arquivo salvo pelo Decker, um jogo onde os jogadores assumem o papel de um hacker que invade computadores corporativos; contém informações de jogo, como trabalhos e inventários aceitos e concluídos; usado para armazenar o progresso do jogo.

Mais Informações

O que é um arquivo .dsg? (Doom Saved Game)

Save file criado e usado por Chocolate Doom, uma versão feita por fãs do Windows 95 Doom; contém o progresso do jogo salvo, que inclui a versão do jogo, habilidade do jogo, episódio do jogo, nível atual, tempo desde que o nível começou, quantidade de armaduras possuídas, tipo armadura, poderes e armas alcançados, se o jogador está vivo, morto ou respawning, e outras informações do jogo.

Mais Informações

NOTA: Há seis slots disponíveis para jogos salvos e cada slot usa um arquivo DSG numerado de 0 a 5. O arquivo se chama doomsave[slot number].dsg.

O que é um arquivo .dsg? (Arquivo de design da era de ponto)

Um arquivo DSG contém um design de bordado criado pela Sierra Stitch Era, um programa usado para projetar bordados costurados com uma máquina de costura. Ele armazena um design vetorial que foi extraído do zero ou de uma imagem digitalizada. Os arquivos DSG também incluem informações avançadas de pontos, como padrões de preenchimento e cores de rosca.

Mais Informações

Você pode criar um arquivo DSG na Era do Ponto selecionando File → New ou New na barra de ferramentas e, em seguida, clicando File → Save. Você pode abrir um arquivo DSG na Era do Ponto selecionando File → Open ou Open na barra de ferramentas.
