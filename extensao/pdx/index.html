O que é um arquivo .pdx? (Arquivo do índice Adobe Acrobat)

Arquivo de índice criado pela Adobe Acrobat, um programa de criação de documentos e visualizador; contém um índice de documentos e diretórios; permite a capacidade de pesquisar em vários documentos .PDF; útil para empresas que usam grandes conjuntos de documentos PDF, como advogados.

Mais Informações

Os índices PDX podem ser criados sob o Choose Advanced → Document Processing → Full Text Index with Catalog option in Adobe Acrobat. Outros arquivos podem ser adicionados a um índice PDX existente usando a mesma opção.

O Adobe Acrobat está disponível nas versões Standard, Pro e Pro Extended.

O que é um arquivo .pdx? (Arquivo eXchange de dados do produto)

Arquivo de dados eXchange (PDX) de dados de dados do Produto (PDX) usado para facilitar as interações na cadeia de suprimentos; contém uma definição total do produto de um produto fabricado; inclui dados como bill of materials (BOM), configuração original do produto, lista de fabricantes aprovada (ALM) e histórico do produto.

Mais Informações

Os arquivos PDX ajudam a comunicar informações técnicas entre fabricantes de equipamentos originais (OEM), provedores de serviços de fabricação de eletrônicos e fornecedores de componentes. Eles usam um formato XML padronizado especificado pelo padrão IPC-2570 Supply Chain Communication.
