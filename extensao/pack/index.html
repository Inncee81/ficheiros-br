O que é um arquivo .pack? (Pacote temático de ferramentas custopack)

Arquivo usado pelo CustoPack Tools, um programa usado para personalizar o tema Windows; contém estilos visuais temáticos, sons e gráficos para Windows ícones e o fundo da área de trabalho; pode ser baixado no site do CustoPack Tools.

Mais Informações

Os temas do CustoPack Tools são distribuídos como arquivos .EXE auto-instalados. O arquivo PACK para o tema está contido no arquivo EXE, de modo que os usuários raramente vêem o arquivo PACK real. Se você quiser acessar o arquivo PACK, use um programa como 7-Zip para abrir o conteúdo do arquivo EXE.

NOTA: O CustoPack Tools está descontinuado e não está mais disponível.

O que é um arquivo .pack? (Arquivo de jarra embalado Pack200)

Arquivo criado pelo Pack200, um utilitário incluído no Java 150 que compacta .JAR arquivos; contém um arquivo JAR que foi compactado em até sessenta por cento; usado por Java Web Clientes Start para recuperar e desempacotar arquivos JAR para programas Java.

Mais Informações

Os arquivos PACK são frequentemente compactados em arquivos .PACK.GZ. Eles também podem ser vistos com a extensão de "jarpack".
