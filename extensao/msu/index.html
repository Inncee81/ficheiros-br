O que é um arquivo .msu? (pacote de atualização do Windows Vista)

Um arquivo MSU é um pacote de atualização usado pelo Windows Update, um aplicativo incluído no Windows Vista, Windows 7 e Windows Server. Ele contém uma ou mais atualizações para aplicativos e arquivos no sistema Windows. Os arquivos MSU são instalados pelo Windows Update Stand-alone Installer (Wusa.exe), que é executado pelo Windows Update automaticamente (o usuário geralmente não precisa instalar manualmente arquivos MSU).

Mais Informações

Cada arquivo MSU contém quatro elementos:

  1. Windows Atualizar metadados, que descreve o pacote de atualização
  2. Um ou mais .CAB arquivos que armazenam os dados de atualização do pacote
  3. Um arquivo .XML que descreve o conteúdo do arquivo MSU
  4. A arquivo de propriedades que é lido por Wusaexe

Você pode extrair manualmente o conteúdo de um arquivo MSU no prompt de comando de Windows digitando:

expand -f:* "example.msu" output_directory

Você pode então instalar manualmente o pacote de atualização digitando o seguinte comando:

pkgmgr.exe /n:output_directory\example.xml

(example.msu e example.xml têm o mesmo prefixo de nome de arquivo).

NOTA: Versões anteriores do sistema operacional Windows usavam arquivos .MSI para atualizações.
