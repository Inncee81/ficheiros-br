O que é um arquivo .pkg? (CoCreate Arquivo de pacote de modelagem do OneSpace)

O que é um arquivo .pkg? (Mac pacote instalador OS X)

Um arquivo PKG é um pacote de arquivos instalados compactados usados para instalar um programa de software. É comumente usado para instalar software em Mac OS X e pode ser instalado individualmente ou referenciado por um script instalador incluído no software.

Mais Informações

Clique duas vezes no pacote para instalar os arquivos. Clique com o botão direito do mouse (Controle-clique) e selecione "Mostrar conteúdo do pacote" para visualizar os arquivos armazenados no pacote. O Apple Installer é o utilitário integrado para abrir arquivos PKG incluídos com Mac OS X.

O que é um arquivo .pkg? (Pacote baixado da PlayStation Store)

Um arquivo PKG é um pacote baixado da Sony's PlayStation Store, uma loja online com jogos, filmes e demonstrações para dispositivos PS3 e PSP. Ele é salvo em um formato criptografado que só é reconhecido por dispositivos compatíveis da Sony e é usado principalmente para transferir jogos para download, atualizações de jogos e demonstrações.

Mais Informações

Pacotes baixados na PlayStation Store podem ser transferidos para um PSP ou PS3 via USB, Memory Stick ou uma conexão de rede.

O que é um arquivo .pkg? (Arquivo do pacote Symbian)

Um arquivo PKG é um arquivo de texto necessário para construir um arquivo instalador .SIS, que é usado para instalar aplicativos em dispositivos que executam o sistema operacional Symbian (OS). Ele contém informações de instalação em texto simples que define como construir o arquivo instalador SIS. Os arquivos PKG geralmente incluem informações sobre os idiomas suportados pelo aplicativo, nome do fornecedor, dependências de software, arquivos de aplicativos para copiar e detalhes para assinar digitalmente o arquivo para verificar seu conteúdo.

Mais Informações

Você provavelmente só encontrará arquivos PKG Symbian se estiver desenvolvendo um aplicativo para o Sistema operacional Symbian. O arquivo PKG deve ser definido pelos desenvolvedores do Symbian para construir um único arquivo instalador SIS distribuível. Quando o aplicativo está pronto para ser construído, os desenvolvedores normalmente usam o comando "makesis", que faz parte do utilitário CreateSIS. Os desenvolvedores também podem usar os aplicativos Sisar ou Easy SIS Creator, que fornecem uma GUI para a construção de arquivos SIS.

Como os arquivos PKG são salvos em texto simples, eles podem ser abertos e modificados com um editor de texto, como o Microsoft Notepad ou o Apple TextEdit, que são empacotados com Windows e macOS. Se você não puder abrir o arquivo PKG com um editor de texto, provavelmente você terá um tipo diferente de arquivo PKG. Isso se deve a vários tipos de arquivos usando a extensão de arquivo .pkg, que inclui os formatos macOS Installer Package e PlayStation Store Download Package.

Enquanto os arquivos SIS são usados para instalar aplicativos, você pode desempacotar os arquivos se quiser visualizar seu conteúdo. Você pode usar a ferramenta UnSIS gratuita para desempacotar arquivos SIS.

NOTA: Em 2010, a Fundação Symbian terminou e o Symbian OS foi descontinuado. A Nokia finalmente parou de aceitar software de desenvolvedores em 2014.
