O que é um arquivo .sublime-settings? (Arquivo de configurações de texto sublime)

Configurações do arquivo usado pelo Sublime Text, um editor de texto usado principalmente para editar código-fonte; contém configurações que descrevem como um tipo de arquivo, como um arquivo .CSS, é editado em Texto Sublime.

Mais Informações

O arquivo SUBLIME-SETTINGS é usado para descrever como os tipos de arquivos serão editados em Texto Sublime. Você pode alterar as configurações para um tipo de arquivo específico, de modo que, quando você abrir esse arquivo, as configurações mudarão de acordo. Algumas das preferências que podem alterar são o preenchimento de linha, o comportamento de rolagem, o tema da cor do texto e se o recuo automático está ativado.

Você pode alterar manualmente as configurações de preferência usando o arquivo Preferences.sublime-setttings. Há uma versão padrão e de usuário do arquivo, uma está localizada no diretório Packages/Default e a outra está localizada no diretório Packages/User. Certifique-se de editar o arquivo Preferences.sublime-settings no diretório Packages/User em vez do diretório Packages/Default. Isso impede que suas personalizações sejam substituídas quando uma nova versão do Texto Sublime for instalada. Você também pode acessar esses arquivos através de Preferences → Settings - Default ou Preferences → Settings - User.

NOTA: Na versão 2 do Sublime Text, o arquivo SUBLIME-SETTINGS substituiu o arquivo .SUBLIME-OPTIONS, que foi usado na versão 1.
