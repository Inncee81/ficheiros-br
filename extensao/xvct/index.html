O que é um arquivo .xvct? (Arquivo do projeto VISE XML)

Arquivo de projeto XML criado pelo VISE X, um aplicativo usado para construir Mac arquivos instaladores do OS X; contém referências aos arquivos de instalação e ações necessárias para fazer a instalação; usado para exportar projetos binários .VCT usando formatação XML.

Mais Informações

Os arquivos XVCT são, na verdade, Mac os pacotes OS X. Você pode clicar com o botão direito do mouse neles e selecionar "Mostrar conteúdo do pacote" para ver os arquivos incluídos. Um arquivo de lista de propriedades (.PLIST) salva as configurações do projeto em um formato XML.
