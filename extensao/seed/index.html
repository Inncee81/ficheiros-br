O que é um arquivo .seed? (arquivo preseado Linux)

Arquivo instalador incluído em discos Linux OS, como um CD de instalação de Linux do Ubuntu; salvo em um formato de texto com quatro campos por linha; cada linha inclui o nome do programa que deve executar o comando e o nome, tipo e valor da variável.

O que é um arquivo .seed? (Arquivo de dados SEED)

Um arquivo SEED é um arquivo de dados salvo no formato Padrão para a Troca de Dados de Terremotos (SEED), um formato de dados comum usado no campo da sismologia. Contém valores de séries temporáticas, metadados de instrumentos ou ambos. Os valores das séries temporânsicas são uma coleção de intervalos de tempo de forma de onda registrados por um instrumento sísmico. Os metadados do instrumento incluem informações de resposta ao instrumento e a localização física da estação (onde os instrumentos estavam quando os dados foram registrados).

Mais Informações

Os arquivos SEED são gerados por instrumentos sismológicos usados para monitorar movimentos sísmicos na terra. O formato SEED é mantido pela Federação das Redes De Sismografadas Digitais (FDSN).

OS arquivos SEED que contêm valores de séries temporâneas e metadados são separados em duas partes. A parte que contém os valores da série temporal é conhecida como "miniSEED" A parte que contém metadados é conhecida como "Seed sem data" Se um arquivo SEED contiver apenas valores de séries temporais, ele é chamado de arquivo miniSEED, mas ainda tem a extensão "sementes". Se um arquivo SEED contiver apenas metadados, ele é conhecido como um arquivo SEED sem data, mas também ainda tem a extensão "sementes".

Os arquivos SEED são tipicamente convertidos em outros formatos de dados para processamento. rdseed é um programa popular usado para abrir e converter arquivos SEED para outros formatos.
