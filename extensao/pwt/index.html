O que é um arquivo .PWT? (Publicar AutoCAD para Web modelo)

Desenho do modelo de publicação criado pela AutoCAD, um aplicativo de design e desenho 2D e 3D; armazena configurações para publicar um desenho (arquivo .DWG) em um formato HTML Web; inclui as propriedades para imagens, texto, hiperlinks, cores, títulos de Web páginas e animações de vídeo; usado para publicar desenhos Web com a mesma formatação.

Mais Informações

O AutoCAD fornece vários arquivos PWT pré-instalados que podem ser personalizados para publicação de desenhos.

NOTA: Os arquivos PWT não são os mesmos de .PTW arquivos. Os arquivos PTW são criados a partir do modelo PWT quando um desenho foi publicado em um formato Web.

O que é um arquivo .PWT? (Modelo de palavra de bolso)

Modelo criado pelo Microsoft Pocket Word, uma versão reduzida dos dispositivos móveis Microsoft Word for Pocket PC; contém os estilos padrão e formatação para um documento .PSW; usado para criar vários documentos com o mesmo visual e sensação.
