O que é um arquivo .dmc? (Arquivo de tela de dados do Datamartist)

Arquivo de documentos criado Datamartist, uma ferramenta de perfil de dados usada para analisar, combinar e gerenciar dados de várias fontes; pode conter conjuntos de dados internos, bem como especificações de transformação de dados na forma de blocos de transformação e conexões entre eles.

Mais Informações

Os conjuntos de dados internos para o arquivo DMC são limitados a 250.000 valores por conjunto de dados. Portanto, eles são adequados para pequenos e moderados conjuntos de dados e são frequentemente usados para gerar conjuntos de dados de referência para testes. Conjuntos de dados maiores podem ser referenciados dentro do arquivo DMC e são armazenados em arquivos locais separados de cache e índice para acesso mais rápido e acesso off-line.

O Datamartist pode gerenciar dados de bancos de dados do Microsoft SQL Server, planilhas do Microsoft Excel, bancos de dados do Microsoft Access, arquivos DMC e outras fontes.

O que é um arquivo .dmc? (Arquivo de amostra DPCM)

Arquivo de áudio usado por programas de rastreamento e amostragem de áudio, como o FamiTracker; salva uma amostra de instrumento que pode ser looped, inclinação de tom e utilizada de outras maneiras; muitas vezes usada para fazer sons de videogame.

Mais Informações

FamiTracker refere-se aos arquivos DMC como "Amostras moduladas delta"

O que é um arquivo .dmc? (Script do Gerente Médico)

Script usado para automatizar tarefas dentro do software de gerenciamento de prática sage Medical Manager; pode ser usado para exportar informações, instalar atualizações ou para outras funções.

O que é um arquivo .dmc? (Arquivo de configuração de imitação)

Arquivo criado pela Mimic, um programa usado para fazer sequências de animação facial; usado como formato de projeto e salva informações de animação de sincronia labial; pode ser exportado para o estúdio DAZ ou Smith Micro Poser para completar a modelagem e animação do personagem.
