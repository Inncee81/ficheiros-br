O que é um arquivo .acf? (Arquivo de filtro personalizado do Adobe Photoshop)

Arquivo de kernel de convolução criado pelo Adobe Photoshop, um programa de edição de imagens; contém uma matriz de valores de kernel NxN usado para manipular os pixels vizinhos em torno de um determinado pixel; pode conter valores positivos e negativos, bem como expressões matemáticas.

Mais Informações

As matrizes do kernel de convolução podem ser usadas para desfocar, afiar, embosar ou realizar outras operações gráficas raster para uma imagem; uma vez salvos em um arquivo ACF, eles podem ser repetidamente aplicados a imagens.

O que é um arquivo .acf? (Arquivo de dados de caractere do agente)

Arquivo de dados que descreve um personagem criado para o Microsoft Agent, um programa de animação interativo; contém dados de caracteres que podem ser carregados sobre o protocolo HTTP (ou Web); salvo junto com um arquivo .ACA correspondente, que contém dados de animação para o personagem.

Mais Informações

NOTA: O Agente microsoft foi descontinuado com o lançamento de Windows 7.

O que é um arquivo .acf? (Arquivo de cache de aplicativos steam)

Um arquivo ACF é um arquivo de cache usado pela Valve Steam, um pacote de software usado para baixar jogos. Ele salva informações relacionadas ao software baixado e é usado para o download original, bem como atualizações de jogos.

Mais Informações

Arquivos ACF são usados por versões mais recentes do sistema de entrega de conteúdo Steam. Versões mais antigas usam arquivos .GCF e .NCF.

O que é um arquivo .acf? (Arquivo da aeronave X-Plane)

Arquivo de aeronaves usado pela X-Plane, um simulador de voo realista para pilotos e amadores de simuladores; contém dados que descrevem as propriedades da aeronave, como potência do motor, física de voo, propriedades de massa e limites.

Mais Informações

Você pode criar um arquivo ACF usando o Plane-Maker, que é um utilitário que vem com o X-Plane. No entanto, o Plane-Maker não permite que você crie um modelo 3D, ele é usado apenas para determinar o layout de formas e propriedades aerodinâmicas básicas da aeronave. Você pode usar o aplicativo AC3D com o plugin X-Plane para criar um modelo 3D no formato X-Plane OBJ.

NOTA: Antes da versão 10 do X-Plane, o arquivo ACF continha dados da aeronave em formato binário. Agora contém dados em formato de texto.
