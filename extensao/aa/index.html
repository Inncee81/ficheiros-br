O que é um arquivo .aa? (Arquivo de livro de áudio audível)

Formato de áudio digital usado pela Audible para armazenar livros de áudio; projetado para armazenar áudio de palavras faladas e suporta marcadores de capítulos, marcadores, etc.

Mais Informações

Os arquivos audíveis são suportados pelo Apple iPod, Creative Zen, SanDisk Sansa, vários dispositivos GPS e o Amazon Kindle.

NOTA: Os arquivos de áudio AA podem ser convertidos para o formato .MP3 mais compatível usando um programa como Tunebite, TuneCab ou SoundTaxi Media Suite.

O que é um arquivo .aa? (Imagem de CD genérico)

Imagem de disco genérico criada por um programa de imagem de disco; armazena uma réplica exata de um disco rígido ou outro tipo de mídia; basicamente um arquivo .ISO com uma extensão de arquivo diferente.
