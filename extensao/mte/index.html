O que é um arquivo .mte? (Arquivo de configuração de exportação mp3tag)

Um arquivo MTE contém uma configuração de exportação para Mp3tag, uma ferramenta usada para editar metadados de arquivos de áudio. Ele armazena informações sobre como exportar dados de tag associados a um arquivo de áudio para um formato de arquivo especificado, como .HTML, .CSV, .RTF ou .TXT.

Mais Informações

Captura de tela de um arquivo mte no Microsoft Notepad

Você provavelmente só encontrará um arquivo MTE de configuração de exportação se estiver editando os metadados de arquivos de áudio com o Mp3tag. O programa suporta uma variedade de formatos de áudio, incluindo Codificação de Áudio Avançada (.AAC), Formato de Arquivo de Intercâmbio de Áudio (.AIFF), Matroska (.MKA e .MKV), MPEG Layer 3 (.MP3), WAVE (.WAV), Ogg Vorbis (.OGG) e MPEG-4 (.MP4, .M4A e .M4V).

Metadados é um aspecto importante dos arquivos de áudio porque armazena informações sobre a faixa, artista, gênero, arte do álbum, letras, produtor e muito mais para o arquivo. Essas informações são exibidas pela maioria dos reprodutos de áudio e são referenciadas para organizar os arquivos. Também é essencial para licenciar música e fornecer provas de autoria.

Você pode salvar a configuração das configurações de exportação para dados de tag com o Mp3tag selecionando File → Save configuration.... O Mp3tag cria um ou mais arquivos MTE armazenando a configuração e comprime-a em um arquivo .ZIP.

Para abrir e restaurar uma configuração de exportação no Mp3tag, extrair os arquivos MTE com um utilitário de compactação Zip para o seguinte diretório:

  • Windows XP - C:\Documents and Settings\User]\Application Data\Mp3tag
  • >>Windows Vista - C:\Users\[User]\AppData\Roaming\Mp3tag
  • Windows 7/8/81/10 - C:\Users\[User]\AppData\Roaming\Mp3tag
