O que é um arquivo .dae? (Arquivo de troca de ativos digitais)

Um arquivo DAE é um arquivo de intercâmbio 3D usado para trocar ativos digitais entre uma variedade de programas gráficos. Pode conter uma imagem, texturas ou, provavelmente, um modelo 3D. O formato DAE é baseado no esquema XML collada (COLLAborative Design Activity), que agora é de propriedade e desenvolvido pela Autodesk.

O formato COLLADA foi originalmente desenvolvido pela Sony e agora é apoiado conjuntamente pela Sony e pelo Grupo Khronos. Ele foi projetado para fornecer um esquema padrão que os aplicativos gráficos 3D poderiam usar para salvar arquivos, para que esses arquivos pudessem ser facilmente trocados entre aplicativos.
