O que é um arquivo .zi_? (Arquivo zip renomeado)

Um arquivo ZI_ é um arquivo .ZIP renomeado que usa um sublinhado ("_") em vez do "p" para que o arquivo não possa ser acidentalmente expandido quando clicado duas vezes. Ele também é usado para contornar programas antivírus quando enviado por e-mail. ZI_ arquivos podem ser renomeados de volta para arquivos "zip" e, em seguida, descompactados.

Mais Informações

ZI_ arquivos são criados quando um usuário compacta um ou mais arquivos como no formato Zip, mas depois renomeia a extensão "zip" como "zi_". If you receive a ZI_ file, simply rename the extension as "zip" e abre-a com um programa de compactação Zip, como Corel WinZip, RARLAB WinRAR, Apple Archive Utility ou B1 Free Archiver.
