O que é um arquivo .osdx? (Arquivo de descrição do conector de pesquisa)

Arquivo de instalação de um conector de pesquisa (arquivo .SEARCHCONNECTOR-MS) no Microsoft Windows; contém informações de descrição para um local de armazenamento remoto; instala um Conector de pesquisa em Windows para que os usuários possam navegar em locais remotos no Windows Explorer; útil para configurar ambientes de trabalho de escritório remoto.

Mais Informações

Se você receber um arquivo OSDX, você pode clicar duas vezes nele e Windows instalará um Conector de pesquisa no seguinte diretório: [user]\Searches\. Você pode abrir o Conector de pesquisa instalado para navegar em sua localização remota. Observe que os arquivos SEARCHCONNECTOR-MS realmente aparecem como pastas, uma vez que Windows não exibe sua extensão de arquivo.

NOTA: O "OSD" no OSDX significa OpenSearch Description e o "X" significa XML, a formatação usada para arquivos OSDX.
