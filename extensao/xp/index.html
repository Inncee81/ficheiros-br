O que é um arquivo .xp? (Padrão Fastracker 2)

Padrão de música salvo pelo rastreador de áudio Fasttracker; pode ser usado com outros padrões para criar uma música completa ou .XM módulo.

O que é um arquivo .xp? (Mapa de XPilot)

Arquivo de jogo usado pelo XPilot, um jogo de luta espacial 2D mais antigo; salvo em um formato de texto simples e salva os recursos e obstáculos do mapa como uma grade 2D; também salva propriedades de jogabilidade para o mapa.
