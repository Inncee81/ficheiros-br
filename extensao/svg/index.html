O que é um arquivo .svg? (Arquivo de jogo salvo)

Arquivo de jogo salvo criado por vários jogos, incluindo Quake III: Team Arena, Return to Castle Wolfenstein e Grand Theft Auto 2 (GTA2); salva o progresso do jogador em um jogo e permite que o progresso seja retomado; normalmente salvo pelo jogo sem que o usuário tenha que acessar manualmente o próprio arquivo.

Mais Informações

Como diferentes jogos usam formatos diferentes para arquivos SVG, você pode ter que experimentar vários programas antes de ser capaz de abri-los. Por exemplo, o GTA2 Saved Game Editor pode editar arquivos SVG criados por GTA2, mas não arquivos SVG criados por Quake III: Team Arena.

NOTA: Se você não tiver um programa que abra ou edite arquivos SVG, tente abrir seu arquivo SVG em um editor de texto como o Bloco de Notas.

O que é um arquivo .svg? (Arquivo gráfico vetorial escalável)

Um arquivo SVG é um arquivo gráfico que usa um formato gráfico vetorial bidimensional criado pelo World Wide Web Consortium (W3C). Ele descreve imagens usando um formato de texto baseado em XML. Os arquivos SVG são desenvolvidos como um formato padrão para exibir gráficos vetoriais na web.

Mais Informações

Captura de tela de um arquivo svg em CorelDRAW Graphics Suite X8

O formato SVG é um padrão aberto desenvolvido sob o W3C (World Wide Web Consortium), com a Adobe desempenhando um papel importante. As imagens SVG podem ser criadas e exportadas a partir de programas do Adobe Creative Suite, como Illustrator e GoLive.

O formato SVG não é suportado nativamente pelo Adobe Photoshop, Photoshop Elements e InDesign. No entanto, você pode abrir arquivos SVG nesses programas usando o kit SVG para plug-in Adobe CS, que é desenvolvido pela Scand.

NOTA: Mais informações sobre o formato SVG podem ser encontradas online na página de informações do Vector Graphics escalável da Adobe.
