O que é um arquivo .bcm? (Arquivo de cache da Ponte adobe)

Um arquivo BCM é um arquivo de cache gerado pela Adobe Bridge, um aplicativo de gerenciamento de arquivos Adobe Creative Cloud (CC). Ele contém dados, como etiquetas e classificações, para arquivos que não podem ser armazenados no arquivo. Os arquivos BCM são semelhantes a .BC, .BRIDGECACHE e .BCT arquivos e os dados que os arquivos possuem ajudam a melhorar os resultados quando você pesquisa arquivos.

Mais Informações

Arquivos em cache, como arquivos BCM, podem ocupar espaço de armazenamento para removê-los, navegar até sua pasta e selecionar Tools → Cache → Purge Cache For Folder "foldername".

O que é um arquivo .bcm? (Arquivo do gerente de contato comercial)

Armazena dados do Business Contact Manager para o Outlook; inclui contas, contatos comerciais, projetos e oportunidades de negócios; também armazena o histórico de comunicação de versões anteriores do Business Contact Manager para o Outlook.

Mais Informações

Os arquivos BCM também podem incluir contatos do Business Contact Manager e-mail Marketing Service do Outlook 2007 e dados do Microsoft Sales Leads de versões anteriores do Business Contact Manager para o Outlook.

O Microsoft Office Outlook pode importar arquivos BCM usando o assistente de importação e exportação de dados comerciais. Você pode optar por importar todos os registros ou selecionar registros individuais que deseja importar.

O que é um arquivo .bcm? (Backup do script de comunicações da Microsoft Works)

Backup de um arquivo .WCM que contém um script de comunicações para o Microsoft Works; usado para conexões telefônicas dial-up e para iniciar sessões de terminais TTY, ANSI e VT100.

Mais Informações

A extensão do arquivo "bcm" pode precisar ser alterada para "wcm" para funcionar corretamente.

NOTA: O Microsoft Works foi descontinuado e não está mais disponível.
