O que é um arquivo .zap? (Arquivo FileWrangler)

Arquivo compactado ou criptografado pelo FileWrangler, um programa de gerenciamento de arquivos para Windows; pode ser usado para renomear ou reatar grupos de arquivos, compactar e expandir arquivos, visualizar e extrair arquivos de .CAB, dividir arquivos grandes em vários discos e localizar arquivos em um disco rígido.

O que é um arquivo .zap? (Arquivo do pacote de administração zero)

Windows arquivo de implantação de aplicativos frequentemente usado em ambientes de negócios para publicar programas não-MSI (arquivos .MSI) aos usuários; contém texto simples e especifica o nome do programa, o arquivo do programa para execução, o número da versão, o editor e outras informações opcionais; permite que .EXE programas sejam publicados aos usuários quando nenhum instalador MSI estiver disponível; chamado "Zero Administration Package" (ZAP) porque permite que programas sejam implantados aos usuários sem usar privilégios administrativos em computadores locais.

Mais Informações

Os aplicativos ZAP são implantados em Windows usando um GPO (Group Policy Object, objeto de política de grupo), que é um objeto de controle de acesso que permite acesso a determinados arquivos e programas. Os usuários para os quais o administrador forneceu acesso podem usar os aplicativos ZAP publicados localmente em seus computadores. Para usuários permitidos, os aplicativos aparecem na lista programas de adicionar/remover do usuário. A gestão de políticas de grupo está incluída como parte do software Windows Server.

Você pode criar e editar arquivos ZAP com qualquer editor de texto. Por exemplo, você pode criar um arquivo de texto simples (.TXT) no Microsoft Notepad e alterar a extensão do arquivo para "zap" quando concluído.

NOTA: Os arquivos ZAP têm limitações que os instaladores tradicionais de MSI não têm. Por exemplo, os arquivos ZAP não podem assumir privilégios elevados para instalar um aplicativo.

O que é um arquivo .zap? (Arquivo de dados ZoneAlarm Pro)

Arquivo de dados usado pelos utilitários Zone Alarm; contém informações de segurança e configurações referenciadas pelo programa; arquivos ZAP comuns incluem alert.zap, email.zap, filter.zap, firewall.zap, programs.zap e security.zap.

Mais Informações
