O que é um arquivo .sbd? (Arquivo DNASTAR SeqBuilder)

Arquivo SeqBuilder criado pela Lasergene, um conjunto de aplicações de biologia estrutural, genômica, clonagem e biologia molecular; contém uma sequência de DNA em texto simples e pode ser visto pela maioria dos programas de sequenciamento de DNA e editores de texto.

O que é um arquivo .sbd? (Arquivo de dados da empresa de contabilidade do escritório)

Banco de dados contábeis da empresa criado pelo Microsoft Office Accounting, um programa usado para bancos online; contém dados de contas bancárias da empresa, folha de pagamento, contas e outras informações financeiras; usado para rastrear dados financeiros para empresas online.

Mais Informações

NOTA: A partir de 2009, o Escritório de Contabilidade não está mais em desenvolvimento.

O que é um arquivo .sbd? (Arquivo de definição de dados do Superbase)

Arquivo de definição de dados usado pelo SuperBase, um programa que permite que os usuários criem programas de cliente/servidor em uma rede; contém definições de dados para os dados armazenados em um arquivo .SBF; salvos em um formato de texto.
