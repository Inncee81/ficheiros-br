O que é um arquivo .pkf? (Arquivo de pico de audição)

Arquivo peak criado pelo Adobe Audition, um aplicativo de produção de áudio; contém dados de forma de onda relacionados a um arquivo .WAV específico; gerado automaticamente quando um arquivo de áudio é aberto ou importado.

Mais Informações

Os arquivos PKF fornecem uma exibição visual da forma de onda dentro de um arquivo de áudio. Uma vez que um arquivo PKF tenha sido gerado, a forma de onda não precisa ser reaugurada cada vez que o arquivo de áudio ou o arquivo de projeto correspondente for aberto.

NOTA: O formato PKF foi introduzido no Adobe Audition CS55. É semelhante ao formato .PK usado por versões anteriores do Audition, mas contém dados de ponto flutuante e pode exibir dados de forma de onda para sinais de áudio acima de 0dB. Uma vez que os arquivos PKF usam um formato de arquivo mais recente do que arquivos PK, eles só são reconhecidos pela Audição CS55 e posteriormente.
