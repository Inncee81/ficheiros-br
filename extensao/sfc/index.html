O que é um arquivo .SFC? (Criaturas Salvas Jogo)

Arquivo de jogo salvo criado por Criaturas 1 e 2, jogos de vida artificais em que o jogador ensina e controla "Norns;" salva o estado do mundo, incluindo o progresso do jogador no treinamento dos Norns.

O que é um arquivo .SFC? (Imagem do microscópio moético)

Arquivo multimídia criado pelo software de microscopia digital Motic; salva um gráfico bitmap feito por uma câmera de microscópio motic e também pode incluir narração de áudio gravada durante a captura da imagem; usada para revisar dados de laboratório.

Mais Informações

NOTA: Os arquivos SFC podem ser convertidos em imagens de .JPG padrão usando o SFC da Motic para o JPG Converter.

O que é um arquivo .SFC? (Super Nintendo ROM)

Game ROM usado por emuladores SNES como ZSNES e Snes9x; salva um jogo no formato original usado pelo sistema SNES; permite que o jogo seja jogado por um software que emula o SNES original.

Mais Informações

NOTA: As ROMs do Super Nintendo usam mais comumente a extensão .SMC. SFC significa "Super Famicom," o nome dado à versão japonesa e sudeste-asiática do Super Nintendo. Portanto, enquanto os ROMs SMC são normalmente usados para armazenar jogos SNES ocidentais, os arquivos SFC são frequentemente usados para jogos SNES japoneses e do sudeste asiático.
