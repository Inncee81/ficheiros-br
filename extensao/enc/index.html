O que é um arquivo .enc? (Arquivo de dados on-line cabal)

Arquivo de dados usado pela Cabal Online, um jogo de RPG online massivamente multiplayer (MMORPG); comumente localizado no diretório \data\ da instalação do software; pode ser nomeado cabal.enc ou um dos vários outros nomes.

Mais Informações

Arquivos ENC corruptos são conhecidos por causar problemas ao carregar Cabal Online, às vezes após a aplicação de patches de jogo. Alguns usuários tiveram sucesso em resolver esse problema desinstalando e reinstalando o software.

O que é um arquivo .enc? (Arquivo PDF protegido por copysafe)

Arquivo de .PDF protegido por cópias criado com o software CopySafe PDF; impede que imagens e textos sejam copiados e protege contra comandos de captura de tela e impressão; também pode incluir proteção de impressão, proteção por senha e expiração de data.

O que é um arquivo .enc? (Arquivo gráfico de navegação eletrônica)

Formato de gráfico de navegação usado pelo Office of Coast Survey (OCS); também chamado de arquivos NOAA ENC, que significa "Administração Nacional Oceânica e Atmosférica" (NOAA) "Gráfico de Navegação Eletrônica" (ENC).

Mais Informações

O formato do arquivo NOAA ENC é definido pela norma International Hydrographic Organization S-57. Os dados enc são usados para apoiar a infraestrutura de transporte marítimo e o gerenciamento costeiro.

Os arquivos DO NOAA ENC são salvos como um banco de dados de recursos de gráficos. As informações salvas em um arquivo ENC incluem a localização de cada recurso de gráfico, bem como atributos como cor, forma, profundidade, etc.

O que é um arquivo .enc? (Arquivo codificado)

Um arquivo ENC é um arquivo codificado genérico que pode usar um dos vários tipos diferentes de codificações proprietárias ou padrão. Ele é codificado para ajudar a proteger o arquivo contra uso não autorizado ou para ajudar a configurar o arquivo para um uso específico, como a transferência da Internet.

Mais Informações

Os arquivos ENC podem ser codificados no formato .UUE, que pode ser aberto com um programa como o StuffIt Deluxe. No entanto, os arquivos ENC que usam uma codificação proprietária só podem ser abertos pelo programa que codificava o arquivo.

O que é um arquivo .enc? (Arquivo de notação musical encore)

Partitura musical criada pelo Encore, programa usado para compor música; pode conter instrumentos, notas, letras, assinaturas de tempo e assinaturas-chave, bem como as informações de título e compositor; usadas para armazenar composições e imprimir partituras musicais.

Mais Informações

Encore pode importar e exportar arquivos XML padrão para integrar com outros programas de notação musical. O software também suporta impressões de alta qualidade usando tecnologias PostScript e TrueType.
