O que é um arquivo .fcd? (Desenho fastcad dos)

2D desenho vetorial criado com FastCAD para DOS, um programa de software CAD; contém objetos que são usados para representar projetos e layouts arquitetônicos.

Mais Informações

Versões mais recentes do FastCAD salvam arquivos com uma extensão .FCW.

O que é um arquivo .fcd? (Formato de CD virtual)

Imagem não compactada de um CD-ROM criado pelas versões de CD virtual 1 a 3; suportado em versões posteriores do CD Virtual, mas salvo em um formato diferente.
