O que é um arquivo .grp? (Arquivo do grupo build engine de Ken Silverman)

Um arquivo GRP é um arquivo de dados de jogo usado por jogos que são construídos com o Build Engine de Ken Silverman. Exemplos de jogos incluem Duke Nukem 3D e Shadow Warrior. Os arquivos GRP podem armazenar vários tipos de dados do jogo, como gráficos, áudio e informações de mapa. Eles são salvos em um formato simples e não comprimido e, portanto, podem ser facilmente modificados para mudar a jogabilidade.

Mais Informações

O formato de grupo é um formato mais antigo e, portanto, o suporte para ele é limitado. Ainda existem ferramentas como kgroup e kextract que podem ser usadas para extrair o conteúdo dos arquivos e para reempacotar os arquivos com conteúdo atualizado. Esses programas fazem parte do Build Engine de Ken Silverman.

NOTA: Uma vez que existem várias filiais do motor de Ken Silverman, pode haver pequenas variações nos arquivos GRP. Portanto, você deve olhar para a documentação do formato de arquivo de grupo para o jogo específico que você tem.

O que é um arquivo .grp? (Arquivo de grupo SmarterMail)

Arquivo criado pelo SmarterMail, um aplicativo de servidor de e-mail usado para hospedar Web acesso a e-mails; armazena um dia de mensagens de e-mail para uma pasta de caixa de correio; usado para armazenar um banco de dados de mensagens de e-mail para um determinado usuário.

Mais Informações

Os arquivos GRP são criados com nomes de arquivos que refletem a data das mensagens de e-mail. Por exemplo, o arquivo 2011-01-01.grp contém todos os dados de mensagens de e-mail para 1º de janeiro de 2011 para uma pasta de caixa de correio específica. Os arquivos GRP são salvos com um arquivo correspondente "cfg", que tem a lista de todas as mensagens de e-mail e o arquivo GRP onde cada mensagem está localizada.

O que é um arquivo .grp? (Arquivo do grupo gráfico StarCraft)

Coleção de imagens de quadros usadas para unidades, armas, feitiços, wireframes e gráficos de menu dentro do Starcraft; cada objeto pode conter em qualquer lugar de algumas a várias centenas de quadros.

Mais Informações

Os arquivos GRP podem ser extraídos do arquivo .MPQ. Eles são referenciados pelo StarCraft, mas podem ser editados com um editor de terceiros.

O que é um arquivo .grp? (Grupo de gerentes de programas Windows)

Define um grupo de programas listados no Menu iniciar Windows; ou seja, "Principal," "Jogos," "Acessórios," ou outra pasta.

Mais Informações

Se um arquivo GRP ficar corrompido, você pode restaurá-lo para Windows configurações padrão usando o Gerenciador de programas. Selecione File→Executar e digitar "configuração /p" e pressione Enter. Isso irá reconstruir o arquivo GRP usando Windows configurações padrão.
