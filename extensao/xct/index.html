O que é um arquivo .xct? (Arquivo de tabela de conversão de caracteres XVI32)

Um arquivo XCT contém uma tabela de conversão de caracteres criada por XVI32, um editor de hex. Ele armazena uma tabela de caracteres ANSI e ASCII e os caracteres para os quais eles são convertidos quando digitados em XVI32. Os arquivos XCT são armazenados em texto simples.

Mais Informações

O arquivo XCT é usado por XVI32 para salvar conversões de conjunto de caracteres. Você pode editar um arquivo XCT para personalizar quais caracteres são convertidos em outros caracteres do programa. O arquivo permite carregar diferentes conjuntos de conversões de caracteres em momentos diferentes.

Você pode criar um arquivo XCT selecionando Search → Character conversion... e clicando em [b]Salvar tabela[/m].

Você pode carregar um arquivo XCT selecionando Search → Character conversion... e clicando na tabela [b]Carregar[/m].

XVI32 vem com vários arquivos XCT pré-instalados que contêm diferentes conversões de conjunto de caracteres. Os arquivos XCT são DOSWIN.XCT, WINDOS.XCT, WINEBCUS.XCT, EBCUSWIN.XCT, WINEBCDE.XCT e EBCDEWIN.XCT.
