O que é um arquivo .nl? (Arquivo de layout Natron)

Arquivo de layout criado pela Natron, um compositor baseado em nó usado para criar efeitos visuais para filme e televisão; contém dados de layout que especificam como o espaço de trabalho do aplicativo é exibido.

Mais Informações

NOTA: O arquivo de layout é especialmente útil para usuários que procuram manter o mesmo layout sem ter que defini-lo manualmente a cada novo projeto. Para exportar ou importar um layout Natron, selecione [m]Layout[m] do menu e se você deseja exportar ou importar o layout.

O que é um arquivo .nl? (Arquivo de borda de nota)

Arquivo de notas criado pela Kdan Mobile NoteLedge, um aplicativo de gerenciamento de notas para dispositivos móveis; contém notas compostas por texto, desenhos, vídeo ou áudio; inclui data e localização quando o arquivo foi criado; pode ser exportado como .PDF ou .JPEG para fácil compartilhamento.
