O que é um arquivo .DICOM? (Arquivo de imagem DICOM)

Um arquivo DICOM é uma imagem salva no formato Digital Imaging and Communications in Medicine (DICOM). Contém uma imagem de uma tomografia médica, como um ultrassom ou ressonância magnética. Os arquivos DICOM também podem incluir dados de identificação para pacientes para que a imagem esteja ligada a um indivíduo específico.

Mais Informações

O formato DICOM foi desenvolvido pela NEMA (Associação Nacional dos Fabricantes de Elétricos). Foi projetado para a troca e visualização de imagens médicas, como tomografias, ressonâncias magnéticas e imagens de ultrassom.

Vários programas para Windows, macOS e Linux podem visualizar arquivos DICOM. Essas imagens também podem ser visualizadas online com o Jack Imaging através de certos navegadores da Web. Ele só é compatível usando Chrome, Opera, Firefox e Internet Explorer com a extensão Google Chrome Frame instalada.

NOTA: As imagens DICOM usam mais comumente a extensão .DCM.
