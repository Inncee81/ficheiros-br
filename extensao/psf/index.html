O que é um arquivo .PSF? (Arquivo de padrões postscript autocad)

Arquivo de informações de fonte usado pelo AutoCAD, um aplicativo de projeto de engenharia CAD e documentação; armazena informações de substituição de fontes usadas ao exportar um desenho para um arquivo PostScript; permite que os usuários personalizem estilos de fonte em desenhos impressos ou publicados.

Mais Informações

O arquivo padrão postScript padrão do AutoCAD tem o nome de arquivo acad.psf. O arquivo PSF do AutoCAD LT tem o nome aclt.psf.

O que é um arquivo .PSF? (Arquivo de suporte de previsão gps)

Arquivo de suporte usado pelo software GPS; contém os valores quadrados médios de raiz (RMS) dos componentes efêmeros, que definem as posições dos objetos celestes; também inclui o tempo de relógio para cada satélite rastreado nos últimos sete dias.

Mais Informações

Os arquivos de suporte à previsão são usados para determinar como os satélites individuais estão se saindo. Eles também podem ser usados para calcular estimativas estatísticas de precisão gps de longo prazo.

O que é um arquivo .PSF? (Arquivo de especificação do produto HP-UX)

Arquivo de texto que define um pacote de software HP-UX; usado pela sessão SD-UX swpackage e identifica o conteúdo, atributos, compatibilidades e dependências do produto.

Mais Informações

O PSF é usado para os seguintes fins:

  1. Definir informações sobre o fornecedor
  2. Especifique um ou mais produtos
  3. Defina atributos para um ou mais arquivos, arquivos e subprodutos
  4. Definir atributos para a mídia de distribuição
  5. Especifique quais computadores e sistemas operacionais o produto suporta
  6. Defina atributos descritivos dos objetos de software

O que é um arquivo .PSF? (Arquivo de configurações de prova do Photoshop)

Arquivo de preferência que armazena configurações de cores usadas pelo Photoshop para fornecer visualizações precisas de como as imagens ficarão quando impressas; podem ser carregadas selecionando View → Proof Setup → Custom... no Photoshop.

Mais Informações

Em Windows, os arquivos PSF são armazenados no seguinte diretório:
~\Application Data\Adobe\Color\Proofing.

Em Mac OS X, eles são armazenados na seguinte pasta:
~/Library/Application Support/Adobe/Color/Proofing.

O que é um arquivo .PSF? (Arquivo PhotoStudio)

Formato de imagem bitmap usado pelo PhotoStudio, um programa de edição de imagens; suporta texto, formas e várias camadas; comumente usado para salvar fotos digitais editadas.

Mais Informações

ArcSoft PhotoStudio for Canon é comumente incluído com câmeras digitais Canon.

Arquivos PSF também podem ser abertos pela ArcSoft PhotoImpression, que foi descontinuada. Eles podem ser abertos com o Adobe Photoshop alterando a extensão de "psf" para "jpg". No entanto, outros programas de edição de imagens ainda podem não ser capazes de abrir o arquivo.

NOTA: PhotoStudio foi descontinuado e não é mais suportado pelo ArcSoft.

O que é um arquivo .PSF? (Arquivo de script PID)

Script de ID de Parâmetro (PID) usado pelo software de scanner OBD-II; permite que o software recupere códigos de erro e informações de status do dispositivo OBD (On-Board Diagnostic) de um automóvel; pode ser importado com a ferramenta de diagnóstico PMSCAN.

O que é um arquivo .PSF? (Arquivo de som portátil)

Arquivo de som salvo no Formato de Som Portátil (PSF); projetado para trazer formatos de som de jogos anteriores para consoles de próxima geração; usa o código do driver de música do jogo original, que permite que o som seja autenticamente reproduzido.

Mais Informações

PSF inclui os seguintes subformatos:

  • PSF1 - Playstation
  • PSF2 - Reprodução estação 2
  • SSF - Saturno (em desenvolvimento)
  • DSF - Dreamcast (em desenvolvimento)
  • USF - Nintendo 64 (desenvolvido por Adam Gashlin)
  • GSF - GameBoy Advance (desenvolvido por Caitsith2 e Zoopd)
  • QSF - Capcom QSound
Arquivos PSF podem ser reproduzidos em um PC usando programas comuns de áudio com plugins de suporte.

O que é um arquivo .PSF? (Arquivo de saída de proxy SPI)

Arquivo de saída gerado pelo SPI Proxy, uma ferramenta de servidor proxy incluída no SPI Dynamics SPI ToolKit; registros e logs detalhes de tráfego passando pelo servidor proxy; salvo em um formato proprietário legível apenas pelo software SPI.

Mais Informações

A SPI Dynamics foi adquirida pela HP em 1º de agosto de 2007.
