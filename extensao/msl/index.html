O que é um arquivo .msl? (Arquivo de linguagem de script do Magick)

Script escrito para ImageMagick, uma ferramenta de código aberto para conversão e manipulação de imagens; escrito na Linguagem de Scripting Magick (MSL), uma linguagem baseada em XML usada por conjure, biblioteca do ImageMagick para executar tarefas personalizadas de processamento de imagens.

O que é um arquivo .msl? (Arquivo de linguagem de especificação de mapeamento)

Arquivo de desenvolvedor usado por alguns programas desenvolvidos com o Entity Framework da Microsoft, que faz parte do conjunto de ferramentas de desenvolvimento NET; descreve um mapeamento entre um modelo de dados conceitual e o modelo de armazenamento subjacente; fornece uma camada de abstração para maior flexibilidade entre o aplicativo e a tecnologia de armazenamento subjacente.
