O que é um arquivo .HDS? (Arquivo de dados Do Microsoft PlayReady)

Arquivo usado pelo Microsoft PlayReady, um programa drm (Digital Rights Management, gerenciamento de direitos digitais) usado para streaming de vídeo online; chamado mspr.hds (arquivo Microsoft PlayReady HDS) e armazenado no diretório de C:\ProgramData\Microsoft\PlayReady\ em Windows ou /Library/Application Support/Microsoft/PlayReady/ em Mac OS X.

Mais Informações

Os arquivos HDS são mais comumente encontrados ao atualizar o software Microsoft Silverlight, o que pode causar o mau funcionamento do streaming da Netflix. Muitos usuários resolveram esse problema com sucesso, desligando todos os programas, excluindo mspr.hds ou renomeando-os para mspr.hds.bak e, em seguida, reiniciando a Netflix. Ao reiniciar, a Netflix criará uma nova cópia do arquivo mspr.hds, que corrige o problema de streaming.

O que é um arquivo .HDS? (Paralelamente arquivo disco rígido da área de trabalho)

Um arquivo HDS é um disco rígido parcial ou inteiro criado pelo software de virtualização Parallels Desktop. Ele armazena o conteúdo do disco rígido de uma máquina virtual e é montado pela Parallels quando o VM é aberto. Os arquivos HDS podem ser um dos vários arquivos HDS que compõem o disco rígido total.

Mais Informações

Os arquivos HDS estão localizados dentro de um arquivo .HDD disco rígido, que por si só é salvo dentro do arquivo VM .PVM. Para visualizar os arquivos dentro de um arquivo HDS, você deve montar seu arquivo HDD de conexão com o Parallels Mounter, um programa incluído no Parallels Desktop. Você pode conseguir isso seguindo estas instruções:

  1. Abra o arquivo PVM associado clicando nele com o botão direito do mouse e escolhendo "Mostrar conteúdo do pacote"
  2. Localizar o arquivo HDD principal do VM (tem o mesmo nome de arquivo previx do arquivo PVM).
  3. Clique com o botão direito do mouse no arquivo HDD e escolha Open With → Parallels Mounter.app.

Se você não ver o Mounterapp paralelos na lista de programas para abrir o arquivo HDD com, você pode encontrá-lo localizado no diretório /Library/Parallels/ em seu disco rígido.
