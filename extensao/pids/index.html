O que é um arquivo .pids? (Arquivo de dados de varredura snap-on)

Um arquivo PIDS é um arquivo de dados de varredura criado por ferramentas do sistema de diagnóstico e informação Snap-on, como Verus Pro, Zeus e Veredicto. Contém dados em formato XML de uma varredura de um veículo que inclui o estado da bomba de freio e combustível, pressão barométrica, velocidade do motor e tempo da válvula.

Captura de tela de um arquivo pids no Snap-on ShopStream Connect 7

Ferramentas de snap-on, como Veredicto, Verus Pro e Zeus, são usadas para executar varreduras de um veículo para testar seu desempenho. Uma vez que a varredura é concluída, os dados são armazenados no arquivo PIDS, que é usado para compartilhar com outros ou armazenar para registros. Uma varredura coleta uma ampla gama de informações, como a relação de combustível de ar, temperatura do refrigerador do motor, largura do pulso de combustível, temperatura do ar de admissão, pressão absoluta do coletor e tempo da válvula.

Você pode abrir um arquivo PIDS com o ShopStream Connect, um programa desenvolvido pelo Snap-on para visualizar arquivos de dados a partir de varreduras de veículos, como arquivos SCM, SCP e SCS. Uma vez que o arquivo é salvo no formato XML, você também pode visualizar o arquivo PIDS com editores de texto.
