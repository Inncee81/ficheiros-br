O que é um arquivo .ase? (Arquivo de troca de amostra de amostra do Adobe)

Arquivo usado pelos aplicativos de layout de página profissional e gráficos adobe InDesign e Illustrator; também usado pelo Adobe Color, um aplicativo de toque para dispositivos Android e iOS; contém uma amostra de cores, que é uma coleção salva de cores; permite que ilustradores e editores compartilhem amostras de cores.

Mais Informações

NOTA: Os arquivos ASE foram introduzidos com a versão 2 do Creative Suite (CS) da Adobe. O Photoshop salva amostras de cores à medida que .ACO arquivos.

O que é um arquivo .ase? (Arquivo Aseprite Sprite)

Um arquivo ASE contém um sprite criado pela Aseprite, uma editora animada e ferramenta de pixel art. Ele armazena uma animação 2D ou gráfico, que inclui camadas, quadros, paletas, tags e configurações. Os arquivos ASE também podem ser salvos à medida que .ASEPRITE arquivos.

Mais Informações

Captura de tela de um arquivo de ase em Aseprite 12

Os arquivos ASE e ASEPRITE são salvos no mesmo formato e são os arquivos mais comuns associados ao Aseprite. Os arquivos são criados para armazenar o gráfico ou animação sprite e podem ser fechados e reabertos para nova edição em Aseprite.

Você pode criar um arquivo ASE no Aseprite selecionando File → New..., depois File → Save ou Save As... e escolhendo o formato ASE. Você pode abrir um arquivo ASE no Aseprite selecionando File → Open... ou Open Recent e escolhendo o arquivo ASE.

Aseprite também pode ser usado para exportar arquivos ASE para formatos mais populares, como .PNG ou arquivos .GIF. Para fazer isso, abra o arquivo ASE e selecione File → Save Copy As.

O que é um arquivo .ase? (Arquivo de exportação de cena do Autodesk ASCII)

Arquivo de texto contendo informações de cena 2D ou 3D; exportado por vários programas autodesk; salvo em um formato de texto, que é mais compatível com outros programas.

Mais Informações

Os arquivos ASE são semelhantes aos arquivos .ASC autodesk, mas contêm informações adicionais de cena, bem como pontos e formas.
