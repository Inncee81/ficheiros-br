O que é um arquivo .FRM? (Formulário AccountEdge)

Formulário criado pela AccountEdge Pro, um aplicativo de contabilidade empresarial; contém dados em um layout personalizado para cheques, contracheques, recibos, notificações de pagamento, faturas, declarações, ordens de compra, folhas de pagamento, boletos de depósito, etiquetas de e-mail e formulários fiscais.

Mais Informações

Para visualizar seus formulários, selecione Setup → Customize Forms e escolha o formulário que deseja ver. Para personalizar formulários da janela de exibição, selecione "Personalizar," faça suas edições, selecione "Salvar formulário" ou "Salvar formulário como" nomear o formulário e clicar em OK. O formulário personalizado será salvo na pasta "Formulários personalizados".

O que é um arquivo .FRM? (Arquivo gráfico fallout)

Arquivo gráfico usado por Fallout 1 & 2, uma série de jogos de rpg ambientada em um mundo pós-apocalíptico; armazena um ou mais quadros de imagem referenciados para a jogabilidade; inclui animações animais, cabeças de personagens falantes, objetos de mapa, fundos de diálogo e componentes para a interface Pip-Boy.

Mais Informações

Um quadro de imagem, que está contido em um arquivo FRM, é uma imagem de 256 cores não pavimentada. Cada quadro tem um arquivo correspondente .PAL paleta que fornece cor para a imagem ou animação.

Os 3 personagens iniciais do jogo são pré-fabricados e cada personagem tem um arquivo FRM, .GCD e .BIO correspondentes. Esses arquivos estão todos localizados na pasta "premade".

NOTA: Fallout 1 e 2 foram originalmente desenvolvidos pela Interplay e Black Isle Studios, mas a Bethesda Softworks adquiriu os direitos.

O que é um arquivo .FRM? (Arquivo de formato de banco de dados MySQL)

Um arquivo FRM contém as informações de formatação ou os dados da estrutura para uma tabela em um banco de dados MySQL. Ele especifica a tabela armazenada no banco de dados e define os campos e a estrutura da tabela. Os arquivos FRM são gerados quando uma tabela MySQL é criada e eles têm o mesmo nome das tabelas para as quais armazenam as informações de formatação.

Mais Informações

Você provavelmente só encontrará arquivos MySQL FRM se você for um arquiteto de banco de dados ou desenvolvedor configurando o backend de armazenamento de dados MySQL de um aplicativo. Os arquivos são criados no mesmo diretório que representa o banco de dados ao qual as tabelas que estão armazenando informações de formatação pertencem.

ARQUIVOS FRM não devem ser abertos manualmente. Em vez disso, eles são referenciados pelo MySQL para formatar cada tabela em um banco de dados.

FRM os arquivos são salvos juntamente com um arquivo .MYD, que contém os dados reais armazenados no banco de dados. Ambos os arquivos são necessários para que o MySQL reconheça o banco de dados. Se o banco de dados MySQL usar tabelas MyISAM, um arquivo .MYI também será usado e necessário.

Há alguns casos em que os dados em um banco de dados MySQL são perdidos ou se tornam corruptos. Utilitários como O Reparo do Banco de Dados Aryson MySQL podem ser usados para recuperar ou reparar os arquivos FRM que armazenam as informações de formatação perdidas.

NOTA: FRM é abreviação de "FoRMat"

Desenvolvedor Corel
Categoria Imagens Rasterizadas
Formato Binário 

O que é um arquivo .FRM? (Arquivo de pilha de quadros do pintor)

Pilha de quadros criada pelo Corel Painter, um aplicativo de pintura digital; armazena uma série de quadros, incluindo largura, altura, resolução e profundidade de cor dos quadros; usados para criar animações de pinturas.

Mais Informações

Para criar um arquivo FRM, selecione File → New... e escolha a opção "Filme Com". Uma paleta flutuante Frame Stacks aparecerá, que pode ser usada para navegar entre os quadros e para visualizar a animação.

Arquivos FRM podem ser exportados para filmes quicktime (arquivos .MOV) bem como arquivos de .GIF animados.
