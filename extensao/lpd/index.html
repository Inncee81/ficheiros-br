O que é um arquivo .LPD? (Arquivo do driver do protocolo de vigia)

Arquivo de configuração usado pelo LabVIEW, um software de engenharia usado para interligar dispositivos de hardware de medição e controle com software; salva uma configuração de servidor OPC (OLE for Process Control) e fornece os dados de configuração para interligação com um aplicativo PLC, RTU ou um aplicativo HMI/SCADA.

O que é um arquivo .LPD? (Arquivo de permissões de impressão LPD Daemon)

Arquivo usado pelos sistemas IBM AIX e outros sistemas UNIX; tem o nome de arquivo hosts.lpd e especifica permissões de impressão para computadores individuais ou grupos de computadores; permite que os administradores do sistema criem políticas de controle de acesso de grãos finos para impressoras em rede.

Mais Informações

Arquivos LPD são usados pelo lpd daemon, um serviço que processa solicitações de impressão. Quando uma solicitação de impressão chega, o lpd daemon cria um processo que verifica tanto o arquivo hosts.lpd quanto o arquivo hosts.equiv no diretório /etc/. Usando esses arquivos, o daemon determina se a solicitação de impressão é permitida.

NOTA: Arquivos LPD podem ser editados com qualquer editor de texto.
