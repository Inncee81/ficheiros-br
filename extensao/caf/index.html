O que é um arquivo .caf? (Arquivo de animação binária Cal3D)

Arquivo de animação tridimensional usado pelo Cal3D, um kit de ferramentas gratuito usado para modelagem de caracteres e animação; contém configurações de movimento para o esqueleto do modelo; especifica rotações e movimentos para os ossos do modelo ao longo do tempo.

Mais Informações

Os arquivos DA CAF são usados para dar aos personagens movimentos humanos realistas. Eles podem especificar animações para caminhada, corrida ou vários outros movimentos.

NOTA: Os arquivos de animação Cal3D também podem ser expressos em um formato XML usando arquivos .XAF.

Desenvolvedor CalEd Software
Categoria Dados
Formato Binário 

O que é um arquivo .caf? (Arquivo de grade de ação de classe)

Arquivo gradebook criado pela Class Action Gradebook, um programa usado para organizar informações em sala de aula; pode conter informações dos alunos, arranjos de assentos, tarefas de casa, notas premiadas e outros dados de classe.

Mais Informações

Os arquivos DA CAF são usados para gerenciar dados do gradebook e para imprimir relatórios. Os professores podem gerar relatórios como resumos de notas em sala de aula, relatórios individuais de alunos, cartas dos pais e gráficos de classe e atribuição. Cada relatório pode ser personalizado.

NOTA: Os arquivos do CAF são criados pelas edições Secundárias e Universitárias do Class Action Gradebook.

Arquivo de áudio principal

Categoria Áudio
Formato Binário 

O que é um arquivo .caf? (Arquivo de áudio principal)

Formato de áudio digital baseado na tecnologia Core Audio da Apple; armazena dados de áudio, bem como anotações de texto, marcadores, layouts de canal e outras informações; semelhante a um arquivo .AIFF ou .WAVE, mas não tem um limite de tamanho de 4GB e pode armazenar qualquer número de canais de áudio.

Mais Informações

O formato de áudio da CAF é suportado por Mac aplicativos OS X que incorporam a API core audio, incluindo QuickTime Player 7 e posterior.

Arquivo de animação de caractere CryENGINE

Desenvolvedor Crytek
Categoria Desenvolvedor
Formato Binário 

O que é um arquivo .caf? (Arquivo de animação de caractere CryENGINE)

Arquivo criado pelo CryENGINE SDK, um conjunto de aplicativos usado para fazer jogos de PC e console; contém dados de animação de caracteres que foram compilados a partir de um arquivo de caracteres .CHR usando um arquivo de especificação de compilação .CBA.

Mais Informações

Os arquivos DA CAF são usados como arquivos de compilação intermediário antes de construir o arquivo de animação de banco de dados (.DBA), que é adequado para uso no jogo final entregue.
