O que é um arquivo .lit? (Arquivo eBook)

Versão eletrônica de um livro ou outra publicação salva no formato Microsoft LIT; inclui informações de gerenciamento de direitos digitais (DRM), que protege os direitos autorais limitando os sistemas em que cada livro pode ser lido.

Mais Informações

Arquivos LIT usam a tecnologia Microsoft ClearType para melhorar a legibilidade.

O que é um arquivo .lit? (Arquivo de código fonte iluminado)

Arquivo de código fonte escrito no Lit, uma linguagem de programação de alto nível e propósito geral; pode ser editado com qualquer editor de texto simples.

O que é um arquivo .lit? (Roteiro de Haskell alfabetizado)

Código fonte de um programa escrito na linguagem de programação Haskell com formatação alfabetizada; semelhante a um script haskell padrão (.HS arquivo), mas inclui comentários, onde apenas linhas que começam com ">" são consideradas como código de programa.
