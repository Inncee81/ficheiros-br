O que é um arquivo .ppsx? (PowerPoint Open XML Slide Show)

Um arquivo PPSX é uma apresentação de slides criada pelo Microsoft PowerPoint ou outro programa de apresentação, como OpenOffice Impress ou SoftOffice Presentations. Ele contém uma coleção final de slides que são usados para reprodução de apresentação de slides. Os arquivos PPSX são semelhantes aos arquivos .PPTX, mas abertos como uma apresentação de slides quando clicado duas vezes, em vez de um arquivo de apresentação de trabalho.

Captura de tela de um arquivo ppsx no Microsoft PowerPoint 2019

Arquivos PPSX são úteis para enviar apresentações finalizadas para usuários que só precisam visualizar a apresentação em vez de editá-la. No entanto, eles ainda podem ser editados no PowerPoint, mas devem ser abertos diretamente no programa selecionando File → Open e escolhendo o arquivo PPSX.

PPSX arquivos são salvos no formato Open XML, que foi introduzido no Microsoft Office 2007. Eles podem ser abertos pelo PowerPoint 2007 ou posterior, ou por versões anteriores do PowerPoint com o componente Open XML instalado.

NOTA: Embora o OpenOfficeorg não abra nativamente arquivos PPSX, você ainda pode abri-los no software selecionando a opção "Microsoft PowerPoint 2007 XML" da lista de formatos de arquivo suportados.
