O que é um arquivo .fpa? (Arquivo de avaliação de desempenho final)

Arquivo criado pelo Finale, um programa de notação musical; contém uma avaliação de uma performance musical; a crítica inclui notas corretamente tocadas, uma nota da performance e a porcentagem de notas corretas tocadas.

Mais Informações

Para abrir um arquivo FPA no Finale, selecione File → Open, escolha "Finale Performance Assessment" no menu "Tipo de arquivo" e clique duas vezes no arquivo FPA.

NOTA: A partir de 2007, a opção de Avaliação de Desempenho Final para criar arquivos FPA foi descontinuada em Finale.

O que é um arquivo .fpa? (Arquivo de ordem criptografada do painel frontal)

Arquivo criptografado contendo informações para um pedido processado com software de comércio eletrônico do Painel Frontal.
