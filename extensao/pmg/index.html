O que é um arquivo .pmg? (Arquivo de composição panorâmica do Adobe Photoshop Photomerge)

Imagem panorâmica criada pelo Adobe Photoshop, um programa de edição de imagens e fotos; armazena uma composição de imagem que foi mesclada a partir de várias imagens; também salva o arranjo de imagem e as opções de fusão selecionadas pelo usuário; comumente usada para armazenar panoramas de fotos digitais.

Mais Informações

Ao mesclar e misturar as bordas de várias imagens, a tecnologia Adobe Photoshop Photomerge "stitches" fotos digitais em uma única imagem para que elas apareçam como se fossem capturadas como uma única foto de câmera.

Para usar o Photomerge, selecione File → Automate → Photomerge.... Alternativamente, você pode abrir o Adobe Bridge, selecionar vários arquivos de imagem e, em seguida, escolher Tools → Photoshop → Photomerge.... Depois de selecionar este item do menu, o Photoshop abrirá e fornecerá uma caixa de diálogo para selecionar as opções de fusão.

NOTA: Enquanto o Photoshop e o Photoshop Elements suportam arquivos PMG, o Photoshop pode lidar com imagens muito maiores.

O que é um arquivo .pmg? (Arquivo do grupo PageMaker)

Arquivo criado pelo PageMaker, um programa usado para criar layouts de página; contém um gráfico .EPS, que inclui tanto um gráfico raster de baixa resolução quanto dados postScript para impressões de alta resolução; criado usando o recurso "PS Group it", que agrupa objetos e substitui os gráficos da página por um arquivo PMG gerado.

Mais Informações

NOTA: PageMaker foi substituído pelo produto InDesign da Adobe, que suportava arquivos PMG através da versão CS. No entanto, as versões atuais do InDesign não suportam o formato de arquivo PMG.
