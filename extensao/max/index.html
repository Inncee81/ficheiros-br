O que é um arquivo .max? (Arquivo de cena max 3ds)

Um arquivo MAX é um arquivo de cena tridimensional criado pelo 3ds Max, um aplicativo de modelagem, animação e renderização 3D. Pode conter vários modelos que incluem wireframes, texturas, efeitos de iluminação, sombreamento, animação e outros elementos de design 3D.

Mais Informações

MAX arquivos são usados para desenvolver gráficos profissionais para jogos, filmes e televisão. 3ds Max permite exportar o arquivo MAX para muitos formatos diferentes, incluindo .FBX, .AI, .DWG, .DXF, .IGS, .SAT, .STL e muitos outros.

NOTA: 3ds Max era anteriormente conhecido como 3D Studio MAX, que também salvou cenas usando a extensão "max". Você ainda pode exportar cenas para .3DS arquivos na versão mais recente do software 3ds Max.

Formato Binário 

O que é um arquivo .max? (Action Replay Max Save File)

Arquivo salvo criado pelo Action Replay MAX da Datel, um programa de PlayStation 2 (PS2) usado para aplicar cheats em jogos PS2 e criar salvamentos PS2; contém o estado salvo de um videogame PS2; pode ser salvo em um disco rígido ou transferido para um PC usando um pen drive USB.

Mais Informações

MAX arquivos podem ser abertos diretamente no PlayStation 2 usando o Gerenciador de Memória no disco PS2 Action Replay MAX.

O que é um arquivo .max? (Documento digitalizado omnipage)

Documento de imagem ou texto digitalizado salvo pelo software de digitalização OmniPage; reconhece caracteres e palavras e detecta informações de layout de página usando o software OCR.

Mais Informações

OmniPage foi desenvolvido anteriormente pela ScanSoft, que foi adquirida pela Nuance Communications, Inc.
