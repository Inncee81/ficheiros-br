O que é um arquivo .ds_store? (Mac Arquivo de configurações de pastas OS X)

Um arquivo DS_STORE contém preferências de visualização de uma pasta no macOS (Mac OS X). Ele armazena informações sobre a localização da janela da pasta, o tamanho da janela, qual opção de exibição foi selecionada (ícone, lista ou coluna) e o layout dos ícones na janela.

Mais Informações

DS_STORE arquivos geralmente são invisíveis, mas podem se tornar visíveis quando copiados por uma rede, copiados para um computador Windows ou restaurados a partir de um backup. O arquivo DS_STORE pode ser excluído de dentro de uma pasta sem afetar nenhum dos dados dentro da pasta. Se o arquivo DS_STORE for removido, a pasta será aberta usando as configurações de exibição padrão do Finder.

NOTA: DS significa "Desktop Services"
