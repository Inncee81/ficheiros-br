O que é um arquivo .flp? (Arquivo Flipchart ActivPrimary)

Apresentação flipchart criada com programas de planejamento de aulas educacionais promethean, como ActivPrimary e ActivStudio; contém um plano de aula interativo que pode ser interagido com o uso de hardware de quadro eletrônico; usado por professores para armazenar planos de aula para os alunos.

Mais Informações

O produto de software da Promethean ActivInspire usa a extensão .FLIPCHART, mas ainda pode abrir arquivos FLP.

NOTA: Se fruityLoops ou FL Studio forem instalados no mesmo computador que o ActivPrimary ou ActivStudio, a associação de arquivos Flipchart pode ser atribuída incorretamente ao FruityLoops ou ao FL Studio. Você pode abrir os arquivos Flipchart diretamente no software Promethean selecionando File → Open....

O que é um arquivo .flp? (Projeto de Estúdio FL)

Um arquivo FLP é um projeto criado pelo FL Studio, um aplicativo usado para compor músicas de arranjo, gravação e edição. Ele contém todos os dados associados a um projeto, incluindo plugins de demonstração e configurações de faixas de música.

Mais Informações

ARQUIVOS FLP não armazenam o áudio real que aparece no projeto. Em vez disso, os arquivos contêm referências aos arquivos de áudio. Portanto, se você mover um arquivo FLP ou os arquivos que ele faz referência, como um arquivo de amostra .WAV ou simsynth ou arquivo predefinido DrumSynth, o projeto pode não ser reproduzido corretamente no FL Studio.

NOTA: FL Studio era anteriormente conhecido como FruityLoops, que foi originalmente lançado em 1998. FruityLoops tornou-se FL Studio com o lançamento da versão 4 em março de 2003.

O que é um arquivo .flp? (Projeto FlipAnim)

Um arquivo FLP contém um projeto criado por Poklik FlipAnim, um programa online usado para criar animações flipbook. Ele armazena um ou mais desenhos que compõem uma animação flipbook. Os arquivos FLP também contêm as configurações da animação, como velocidade da página, transparência da página e zoom.

Mais Informações

Captura de tela de um arquivo flp em Poklik FlipAnim

Você pode salvar um projeto FlipAnim como um arquivo FLP clicando no ícone de engrenagem e, em seguida, selecionando o ícone de documento com a seta vermelha de download. Seu navegador da Web baixará o arquivo FLP para o seu computador.

Você pode abrir um projeto FlipAnim clicando no ícone de engrenagem, selecionando o ícone do documento com a seta de upload verde e escolhendo o arquivo FLP que deseja carregar. Em seguida, o navegador carregará o arquivo FLP e o programa web FlipAnim abrirá o projeto.

NOTA: Os arquivos FLP foram substituídos por arquivos .FLIPANIM pela FlipAnim para salvar projetos.

O que é um arquivo .flp? (Imagem de disco disquete)

Imagem em disco criada a partir de um disquete de 35%"; muitas vezes criada usando um programa MS-DOS ou um programa a partir de uma versão mais antiga de Windows; pode ser usada para armazenar arquivos bootáveis e inicializar sistemas operacionais convidados em plataformas virtualizadas; também pode ser usado para copiar o conteúdo de um disco para vários disquetes.

Mais Informações

O programa de imagem disquete chamado fdimage.exe é um programa comum do MS-DOS usado para criar arquivos FLP. Também está incluído com CDs de instalação FreeBSD.
