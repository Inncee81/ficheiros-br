O que é um arquivo .csh? (Script de concha c)

Script que pode ser executado dentro de um ambiente de concha C em um sistema BSD Unix; a sintaxe é semelhante ao C/C++.

O que é um arquivo .csh? (Arquivo de forma de onda de Cubase)

Arquivo Waveform criado pela Cubase, um aplicativo usado para produção de áudio; contém informações sobre arquivos de áudio do projeto Cubase; normalmente gerado ao salvar um projeto de áudio (.CPR arquivo); gerado pela edição SX/SL do Cubase.

Mais Informações

NOTA: Os arquivos CSH não armazenam dados de áudio reais.

O que é um arquivo .csh? (Arquivo formas personalizadas do Photoshop)

Coleção de formas personalizadas usadas pelo Photoshop e elementos do Photoshop; acessada pela Ferramenta de Forma Personalizada quando a opção de forma personalizada é selecionada; pode ser carregada ou salva clicando na seta direita na lista suspensa de Formas e selecionando "Formas de carga" ou "Salvar formas"

Mais Informações

Os arquivos Custom Shapes são salvos por padrão no diretório Photoshop/Presets/Custom Shapes.
