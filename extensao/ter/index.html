O que é um arquivo .ter? (Arquivo Terragen Terrain)

Arquivo de terreno criado pela Terragen, um programa de geração de cenários naturais; contém dados de terreno referenciados pela Terragen para determinar a altitude do terreno; inclui valores inteiros ao longo de y, z e x-axes especificando o comprimento, altura, profundidade, etc. no formato de campo de altura da Terragen para fácil transferência de terreno.

Mais Informações

O arquivo TER consiste em 2 8 byte identificando strings e "chunks" de dados. A estrutura será semelhante à seguinte:

  • 8 byte "TERRAGEN" que identifica o programa.
  • 8 byte "TERRAIN" string identificando o terreno.
  • "SIZE" que aparece antes dos dados de altitude identificando o tamanho do terreno.
    • São necessários pedaços
  • "XPTS" e "YPTS" se o terreno não for quadrado.
  • "EOF" string no final do arquivo se for uma versão mais antiga Terragen.

Para importar um arquivo TER, selecione a aba Terrain> clique em Adicionar Terreno, selecionar Heightfield (load file), navegar até o arquivo TER e clicar em Open.
