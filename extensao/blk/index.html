O que é um arquivo .blk? (Modelo de área do Abbyy FineReader)

Modelo usado pelo ABBYY FineReader, um programa que permite aos usuários converter imagens e outros documentos em documentos de texto pesquisáveis; contém dados que designam a área de um documento que deve ser digitalizado pelo FineReader.

Mais Informações

Arquivos BLK podem ser usados para economizar tempo ao digitalizar uma grande quantidade de documentos com o mesmo layout. Você pode criar um modelo BLK que pode ser aplicado aos seus documentos. Siga as instruções abaixo para criar e aplicar modelos.

Para criar um modelo, abra sua imagem e desenhe sua área manualmente ou deixe o FineReader analisar automaticamente o layout, selecionar Area → Save Area Template..., nomear o modelo e clicar em Salvar.

Para aplicar um modelo de área, selecione as páginas a que deseja aplicar o modelo na janela "Páginas". Em seguida, selecione Area → Load Area Template..., selecione o modelo desejado, escolha "Páginas selecionadas" ao lado de "Inscreva-se para" e clique em Abra.

O que é um arquivo .blk? (Arquivo de modelo de bloco AutoCAD)

Modelo de extração de atributos usado pelo AutoCAD, um aplicativo de design e documentação CAD 2D e 3D; armazena elementos de desenho selecionados usados para publicar um desenho em um formato .PDF, .DWF ou .DWFX; armazena especificamente os blocos de desenho e suas propriedades.

Mais Informações

Os arquivos BLK são usados para controlar os dados que são publicados em um arquivo fragmentável. Isso é benéfico em situações onde apenas certos tipos de informações devem ser distribuídas. Os arquivos BLK podem ser criados em AutoCAD com a caixa de diálogo Modelo de bloco ou usando o assistente de extração de atributos.

O que é um arquivo .blk? (Arquivo de dados BLK360)

Um arquivo BLK é um arquivo de dados criado pela Leica Geosystems BLK360, um scanner a laser de imagem usado para coletar informações 3D sobre um ambiente para fins de visualização. Ele contém dados de nuvem de ponto 3D capturados pelo dispositivo BLK360 que descreve objetos e terrenos circundantes. Os arquivos BLK podem ser abertos pelo Leica Geosystems Cyclone Register 360, um programa empacotado com o dispositivo BLK360.

Mais Informações

Os dados em um arquivo BLK foram capturados por uma varredura conduzida por um dispositivo BLK360. Os dados são transferidos por um WLAN para um computador executando o Cyclone Register 360, que é usado para salvar os dados como um arquivo BLK.

Arquivos BLK são comumente usados para renderizar imagens de salas, edifícios e superfícies geológicas. O dispositivo é mais utilizado pelas comunidades de levantamento, construção, levantamento, engenharia e pesquisa.
