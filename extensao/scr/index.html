O que é um arquivo .scr? (Arquivo de script)

Um arquivo SCR é um script executável genérico criado ou usado por uma série de programas possíveis. Quando aberto, o script normalmente executa uma série de comandos na ordem em que são listados.

Mais Informações

Uma vez que os arquivos de script são salvos em um formato de texto, eles geralmente podem ser editados em um editor de texto básico.

A extensão .SCR pode ser dada aos arquivos .EXE para desviar quaisquer restrições colocadas em arquivos EXE por programas antivírus ou malware.

O que é um arquivo .scr? (Windows Screensaver)

Um arquivo SCR é um arquivo de salvamento de tela usado pela Microsoft Windows. Ele contém um gráfico, animação, apresentação de slides ou vídeo que você pode usar como um Windows protetor de tela. Os arquivos SCR são normalmente armazenados no diretório principal da Windows.

Mais Informações

Os protetores de tela foram originalmente criados para evitar que monitores de computador CRT e plasma sofressem queimaduras de tela, uma condição que ocorre quando a tela mostra a mesma imagem por muito tempo. Embora a maioria dos monitores modernos não sofra de queimadura de tela, os protetores de tela ainda são usados para fins de segurança (para ocultar a tela do usuário e exigir um login antes de retomar o trabalho) e para mostrar o estilo do usuário.
