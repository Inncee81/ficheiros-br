O que é um arquivo .rcd? (Arquivo Arcade)

Arquivo usado pelo Arcade, um programa criado para modelar estruturas físicas em tempo real usando métodos computacionais de jogos de computador; usado para criar simulações físicas dinâmicas em ambientes educacionais para ensinar exercícios, lição de casa e laboratórios.

Mais Informações

Arcade foi desenvolvido pela Universidade da Virgínia e é distribuído livremente.

Desenvolvedor The LEGO Group
Categoria CAD
Formato Binário 

O que é um arquivo .rcd? (Lego MINDSTORMS Arquivo de gravação)

Arquivo criado pelo lego MINDstorms NXT software de programação de robôs; registra posições de um robô LEGO MINDSTORMS físico em sequência, coletando amostras em alta frequência; permite que os usuários criem e reproduzam movimentos de robôs; salvos com o nome de arquivo RobotAction.rcd por padrão.

Mais Informações

Arquivos RCD podem ser reproduzidos em Android dispositivos móveis com o aplicativo MINDdroid.

NOTA: LEGO MINDSTORMS NXT "evoluiu" em LEGO MINDSTORMS EV3 em 2013.

pcAnywhere Arquivo de sessão gravado

Desenvolvedor Symantec
Categoria Vídeo
Formato Binário 

O que é um arquivo .rcd? (pcAnywhere Arquivo de sessão gravado)

Gravação criada pelo pcAnywhere remote desktop control software; armazena uma gravação de tela do ambiente de desktop remoto como está sendo usado; pode incluir várias sessões em um único arquivo; pode ser usado para logs de suporte técnico.

Mais Informações

Os arquivos RCD são salvos em um formato proprietário e só podem ser reproduzidos usando o software pcAnywhere.

NOTA: Se você quiser converter um arquivo RCD em um formato de vídeo padrão, como .MPG ou .AVI, uma opção é usar um software de gravação de tela separado e gravar sua tela à medida que o arquivo RCD é reproduzido em pcAnywhere.
