O que é um arquivo .mht? (MHTML Web Archive)

Um arquivo MHT é um arquivo de página web salvo por um navegador da Web, como o Google Chrome ou o Internet Explorer. Ele é formatado usando MIME HTML ou "MHTML," que salva HTML, imagens e outros recursos vinculados em um único arquivo. Os arquivos MHT são mais comumente salvos à medida que .MHTML arquivos.

Mais Informações

A maioria das páginas da Web visualizadas em um navegador da Web contêm apenas imagens HTML e referência e outros arquivos de mídia que são armazenados em outros diretórios no servidor. Como todos os arquivos são codificados com .MIME codificação em um único arquivo em um documento MHTML, os arquivos MHT são comumente usados para arquivar páginas da Web.
