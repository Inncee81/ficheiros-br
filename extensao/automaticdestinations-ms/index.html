O que é um arquivo .automaticdestinations-ms? (Arquivo da lista de salto Windows)

Um arquivo AUTOMATICDESTINATIONS-MS é um arquivo Jump List usado pelo Windows 7, um sistema operacional Windows (OS) da Microsoft. Ele contém informações sobre o aplicativo usado que é fixado na barra de tarefas do usuário, horários e os caminhos para itens (documentos, páginas da web, imagens, etc) acessados recentemente por um programa fixado na barra de tarefas de um usuário.

Mais Informações

O recurso Jump List foi introduzido em Windows 7 e permite que você visualize rapidamente itens recentemente editados por um programa que é fixado na sua barra de tarefas simplesmente clicando com o botão direito do mouse no ícone. Quando você clicar com o botão direito do programa na barra de tarefas, a Lista de Salto aparecerá acima do programa. Por exemplo, se Windows Media Player estiver preso à sua barra de tarefas e você tiver visto recentemente várias imagens .JPG, elas aparecerão quando você exibir a Lista de Saltos para o programa. O mesmo se aplica a navegadores da Web, como o Internet Explorer ou o Google Chrome, mas em vez de imagens, provavelmente mostrará páginas da Web recentemente visualizadas.

O arquivo AUTOMATICDESTINATIONS-MS é criado quando você usa um programa fixado na barra de tarefas. O caminho para esse arquivo específico e o datamp de tempo é armazenado no arquivo AUTOMATICDESTINATIONS-MS, então referenciado pelo Windows 7 OS quando o usuário clica com o botão direito do programa.

Você provavelmente nunca verá o arquivo AUTOMATICDESTINATIONS-MS desde que ele é referenciado pelo SO, mas se você precisar navegar para o arquivo, ele está localizado no diretório a seguir:

Users\%USERNAME%\AppData\Roaming\Microsoft\Windows\Recent\AutomaticDestinations
