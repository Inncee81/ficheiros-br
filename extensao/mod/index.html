O que é um arquivo .mod? (Arquivo do módulo de música amiga)

Formato de módulo de música padrão reconhecido pela maioria dos programas de rastreamento de áudio; pode ser criado por FastTracker, StarTrekker, Noise Tracker ou outro programa de criação mod.

Mais Informações

ARQUIVOS MOD contêm padrões de notas (muito parecidos com um arquivo MIDI) bem como amostras de som usadas para diferentes instrumentos. As amostras de som são atribuídas a diferentes faixas dentro do módulo e são reproduzidas de acordo com as notas da pontuação.

O que é um arquivo .mod? (Arquivo modelo AMPL)

Arquivo modelo usado para calcular problemas matemáticos complexos e em larga escala; contém variáveis (parâmetros e conjuntos), uma função e restrições escritas no código AMPL (A Mathematical Programming Language); aparece como linear, não linear, quadrático, misturado e outros tipos de problemas; inclui comentários, que devem ser precedidos com um "#"

Mais Informações

Para carregar seu arquivo MOD, por exemplo, example.mod, digite model examplemod; se ele estiver localizado em sua pasta "amplcml" com seu arquivo sw.exe. Se o seu arquivo example.mod estiver localizado em um local diferente, você precisará incluir o caminho. Digite algo assim (se usar Windows):modelo C:Usersexamplemod;. Depois de carregar com sucesso seu arquivo MOD, digite solve; para resolver o problema. Em seguida, se você quiser exibir qualquer um de seus resultados, digite display e quaisquer resultados que você deseja exibidos. Lembre-se sempre de terminar sua linha com um ;.

Um exemplo de um arquivo MOD:

var XB;
var XC;
maximizando o lucro: 25 * XB + 30 * XC;
sujeito a B_limit: (1/200# * XB + #1/140) * XC <=>
sujeito a B_limit: 0 <=>
sujeito a C_limit: 0 <=>

O que é um arquivo .mod? (Arquivo de vídeo gravado por filmador)

Modelo Femap

O que é um arquivo .mod? (Modelo Femap)

Arquivo modelo usado pela Femap, um programa de análise de engenharia CAD; armazena um modelo 3D com elementos, materiais, propriedades, superfícies de dados, restrições e outros dados do modelo.

Mais Informações

A Femap significa Modelagem e Pós-Processamento de Elementos Finitos e permite que você construa modelos de elementos finitos de problemas de engenharia e visualize os resultados da solução.

NOTA: A extensão do arquivo MOD foi substituída por .MODFEM com a versão 9 do Femap.

O que é um arquivo .mod? (Arquivo do módulo Fortran)

Arquivo de programa compilado escrito na linguagem de programação fortran; criado para cada declaração de módulo dentro do código-fonte do Fortran; agrupa procedimentos e dados relacionados em um único arquivo, que podem ser referenciados em outras unidades do programa.

Mais Informações

Os arquivos MOD do Fortran são arquivos "intermediários", e não devem ser abertos manualmente. No entanto, eles podem ser referenciados por compiladores fortran, como os listados abaixo.

Módulo GRUB

Desenvolvedor The GNU Project
Categoria Sistema
Formato Não Disponível 

O que é um arquivo .mod? (Módulo GRUB)

Módulo de software usado pelo GRUB (GRand Unified Bootloader), um carregador de inicialização usado para carregar vários sistemas operacionais; armazena um módulo GRUB e é carregado pelo GRUB 2 dinamicamente apenas se necessário para o processo de inicialização específico.

Mais Informações

GRUB 2 foi ligeiramente redesenhado para ser modular. Este design permite uma pegada menor para o carregador de inicialização, uma vez que apenas os módulos necessários são carregados.

OS arquivos GRUB MOD estão localizados nos diretórios /boot/grub/ ou /boot/grub2/.
