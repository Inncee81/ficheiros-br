O que é um arquivo .xlsb? (Planilha Binária excel)

Um arquivo XLSB é uma planilha criada pelo Microsoft Excel, um programa usado para criar e editar planilhas. Contém uma ou mais planilhas com células dispostas por uma grade de linhas e colunas, bem como gráficos, macros e formatação. Os arquivos XLSB são salvos em um formato binário.

XLSB os arquivos diferem das planilhas padrão do formato Open XML (.XLSX) já que armazenam a planilha usando um formato binário (BIFF12). Você provavelmente verá e usará o arquivo XLSX com mais frequência porque ele é comprimido para economizar espaço e é mais fácil de compartilhar com outros usuários. Os arquivos XLSB não são compactados, tornando-os maiores em tamanho do que os arquivos XLSX. No entanto, os arquivos XLSB podem funcionar de forma mais eficiente dentro do Excel porque são salvos no formato binário maior.

Dependendo do sistema operacional que você está usando, o Excel salva arquivos XLSB nos seguintes locais por padrão :

Windows Vista, 7. & 10:
C:\Users\<user name>\AppData\Local\Microsoft\Excel\XLStart

Windows XP:
C:\Documents and Settings\Application Data\Microsoft\Excel\Start

<macOS:
~/Library/Containers/com.microsoft.Excel/Data/Library/Application Support/Microsoft/Roaming/Excel/

NOTA: Os arquivos XLSB podem ser abertos pelo Excel 2007 ou posterior, ou por versões anteriores do Excel com suporte ao Open XML.
