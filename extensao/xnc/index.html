O que é um arquivo .xnc? (Arquivo Exchange NC)

Um arquivo XNC é um arquivo de texto salvo no formato Exchange NC (XNC). Contém instruções numéricas de controle (coordenadas XY) para direcionar uma ferramenta de máquina, como uma broca, para a produção de Placas de Circuito Impresso (PCB). Os arquivos XNC são projetados para trocar informações entre programas CAD e CAM.

Mais Informações

Captura de tela de um arquivo xnc no Ucamco Reference Gerber Viewer

O formato XNC é compacto e agiliza o processo CAD para CAM sem necessidade de arquivos laterais adicionais. O formato armazena vários comandos em linhas separadas, com cada linha no arquivo representando um único comando. Não há limite de comandos que possam ser armazenados em um arquivo XNC, mas os comandos devem ser maiústos. Os arquivos XNC também contêm metadados que podem incluir a extensão dos furos de perfuração, se o furo da broca é um orifício mecânico ou componente, ou se o furo da broca estiver banhado.
