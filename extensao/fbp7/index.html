O que é um arquivo .fbp7? (Projeto FinalBuilder 7)

Arquivo de projeto criado pelo FinalBuilder 7, um aplicativo usado para construir um projeto de desenvolvimento de software em um programa de Windows final; contém propriedades de construção e referências ao arquivo necessário para a construção do programa final; pode ser usado para compilações automatizadas.

Mais Informações

Os projetos do FinalBuilder também são salvos com a extensão .FBZ7, que é a extensão padrão usada para projetos finalbuilder 7. Os arquivos FBZ7 usam compactação para reduzir o tamanho do arquivo.

NOTA: Algumas versões de Embarcadero Delphi incluem FinalBuilder.
