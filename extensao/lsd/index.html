O que é um arquivo .lsd? (Arquivo dicionário Lingvo)

Arquivo de dicionário criado pelo software Lingvo Dictionary, que é usado para traduzir palavras em diferentes idiomas e procurar definições; contém entradas de dicionário compiladas a partir de arquivos de dicionário .DSL; consiste na palavra-cabeça do cartão (palavra definida) e no corpo do cartão (definição); não pode ser editado.

Mais Informações

Um arquivo LSD pode ser um dicionário de sistema fornecido com o Dicionário Lingvo ou criado pelos usuários. Ele suporta diferentes tipos de opções de formatação e layout, como comentários, fontes ousadas e itálicas e exemplos de uso.

Você pode criar um arquivo LSD a partir de um arquivo DSL. Primeiro crie um arquivo .TXT, adicione seu conteúdo e, em seguida, renomeie a extensão do arquivo para "dsl" Você pode usar o Compilador DSL, que está incluído no Dicionário Lingvo, para compilar o arquivo DSL em um arquivo LSD.

O que é um arquivo .lsd? (Arquivo de salvamento de jogo do fabricante de RPG)

Salvar arquivo para um jogo criado com RPG Maker; armazena o estado atual da jogabilidade para que o jogo possa ser retomado a partir do ponto de salvação em um momento posterior; por padrão, os arquivos de salvamento de jogos são chamados "Save01lsd," "Save02lsd," "Save03lsd," etc.
