O que é um arquivo .adi? (Active@ arquivo de backup de imagem de disco)

Imagem em disco criada por Active@ Disk Image, um programa de backup e recuperação; pode conter um backup de um disco rígido, disco de mídia ou outro dispositivo de armazenamento externo; pode fazer parte de um arquivo dividido ou de um backup completo.

Mais Informações

Os arquivos ADI podem ser usados para backups e restaurações completas do sistema, bem como backups parciais do sistema de arquivos. Eles só armazenam os setores ocupados de um disco e usam compactação, o que ajuda a reduzir o tamanho do backup.

Active@ nomes de imagens de disco dividem arquivos usando o nome de backup seguido pelo número na sequência. Por exemplo, um arquivo dividido de dois arquivos para "MyBackup" teria os arquivos MyBackup1.adi e MyBackup2.adi.

O que é um arquivo .adi? (Formato amador de intercâmbio de dados)

Um arquivo ADI é um arquivo de dados salvo no Formato amador de intercâmbio de dados (ADIF). Ele contém dados de contato em texto simples que é trocado entre software de registro de rádio amador. Os arquivos ADI são usados principalmente para transferir diários de bordo.

Mais Informações

A ADIF é utilizada no Serviço de Rádio Amador, que consiste em amadores que usam a tecnologia de rádio como hobby. O formato foi lançado em 1996 e foi adotado por uma variedade de programas amadores como o formato uniforme para a troca de registros entre entusiastas de rádio amadores. Existem três especificações de ADIF. Os arquivos ADI podem ser usados por cada especificação do formato, mas os arquivos ADX (salvos em um formato XML) são usados principalmente pela terceira especificação do ADIF.

NOTA: Os arquivos ADI também são salvos como .ADIF arquivos.

O que é um arquivo .adi? (Arquivo de backup aomei backupper disco)

Imagem em disco criada pelo AOMEI Backupper, um programa de backup e recuperação para usuários Windows; contém um backup de um disco rígido que pode ser usado para restaurar discos rígidos quando ocorre perda de dados ou corrupção.

Mais Informações

OS arquivos ADI podem ser compactados e divididos em vários arquivos, a fim de reduzir o tamanho do arquivo de imagem. Você também pode reduzir o tamanho do arquivo optando por apenas fazer backup dos setores usados de sistemas de arquivos.

NOTA: Active@ Disk Image também usa a extensão do arquivo "adi" para salvar arquivos de backup de imagem em disco.

O que é um arquivo .adi? (Arquivo binário binário independente do dispositivo AutoCAD)

Imagem monocromática gerada pelo AutoCAD, um aplicativo CAD usado para criar designs 2D e 3D; contém um gráfico em preto e branco que foi plotado a partir de um desenho baseado em vetores; usado para armazenar rascunhos de design plotados.

O que é um arquivo .adi? (Arquivo do índice de documentação do desenvolvedor do Dynamics AX)

Contém um índice de arquivos de documentação do desenvolvedor (.ADD) usados pelo software Microsoft Dynamics AX; fornece recursos de acesso e pesquisa mais rápidos para a documentação de ajuda.
