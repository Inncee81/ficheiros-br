O que é um arquivo .zpl? (Lista de reprodução do Zoom Player)

O formato de lista de reprodução da ZPL é usado pelo Zoom Player, um leitor de DVD e mídia para Windows projetado para trabalhar com sistemas de entretenimento doméstico. Ele é salvo em um formato de texto que usa tags personalizadas para definir informações sobre cada item na lista de reprodução.

Mais Informações

O formato de lista de reprodução da ZPL inclui as tags abaixo. Todas as etiquetas, exceto o AC, NM e BR! tags são opcionais.

  1. AC - Arquivo ativo (o último a ser jogado)
  2. NM - Filename
  3. DR - Duração do arquivo em segundos
  4. TT - Descrição do arquivo
  5. TG - Gênero de arquivo
  6. TR - Número da faixa
  7. TL - Linguagem de arquivo
  8. TA - Nome do artista
  9. TB - Nome do álbum
  10. TY - Ano de produção
  11. TH - Nome do autor
  12. TE - Codificador do arquivo
  13. TC - Comentários
  14. TU - URL do arquivo
  15. TO - Informações sobre direitos autorais
  16. FD - Duração forçada (para slides)
  17. BR! - Fim do arquivo

O que é um arquivo .zpl? (Lista de reprodução zune)

Lista de reprodução de áudio ou vídeo criada pela Zune, um aplicativo multimídia para Windows; salvo em formato XML e contém referências a itens de lista de reprodução, como músicas ou filmes; permite que o usuário organize listas de reprodução por gêneros ou favoritos.

Mais Informações

Por padrão, os arquivos ZPL são salvos no seguinte diretório:

[user]\My Music\Playlists\

>NOTA: Zune também se refere a um dispositivo móvel multimídia. O software de desktop Zune pode interagir com dispositivos Zune.
