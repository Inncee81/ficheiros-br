O que é um arquivo .sxd? (Documento ArcScene)

Documento criado pelo ArcScene, um aplicativo usado para visualizar dados GIS em 3D; armazena links para as fontes de dados para a cena; também armazena as propriedades e camadas de visualização da cena; usado para renderizar visualizações 3D dos dados do SIG.

Mais Informações

ArcScene está incluído como parte do pacote De desktop ArcGIS.

O que é um arquivo .sxd? (Desenho do StarOffice)

Desenho criado pelo StarOffice Draw, um programa de edição de gráficos vetoriais; salva elementos vetoriais, como pontos, linhas e curvas, mas também pode incluir gráficos raster; usado para armazenar ilustrações técnicas.

Mais Informações

StarOffice agora é conhecido como Open Office. O software usa a mesma base de código que o Apache OpenOffice, uma versão gratuita do pacote office. Tanto o Apache OpenOffice quanto o Open Office agora usam o padrão OpenDocument e os gráficos vetoriais da loja como .ODG arquivos.

NOTA: O Apache OpenOffice era anteriormente conhecido como OpenOfficeorg.
