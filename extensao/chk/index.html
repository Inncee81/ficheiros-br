O que é um arquivo .chk? (Banco de dados InterCheck Checksum)

Um arquivo CHK é um arquivo de banco de dados que contém definições de vírus para Sophos Anti-Virus. Ele é atualizado com o InterCheck, um serviço que verifica um sistema central de emissão de relatórios de vírus e pode ser modificado alterando o nível de relatórios InterCheck e as configurações de configuração do syslog.

Mais Informações

O banco de dados InterCheck é chamado "interchk.chk" por padrão. É armazenado no diretório /Program Files/Sophos/Sophos Anti-Virus/Config em computadores Windows.

O que é um arquivo .chk? (Arquivo de firmware do Roteador Netgear)

Arquivo usado pelos roteadores Netgear ao atualizar o firmware; armazena dados de sistema de baixo nível que executam o roteador; pode ser usado para aplicar atualizações de roteador usando a interface do navegador Netgear Web, que muitas vezes é acessada no 19216811 endereço de rede local.

O que é um arquivo .chk? (Fragmento de arquivo salvo)

Windows fragmento de arquivo limpo do disco rígido pelo ScanDisk ou CHKDISK; armazenado na pasta FOUND após a realização de uma varredura de disco; normalmente pode ser excluído, uma vez que as informações foram movidas para outro lugar no disco.

Mais Informações

O programa CHK-Mate pode ser usado para examinar o conteúdo de um arquivo CHK antes de ser removido. Além disso, o UnCHK e o FileCHK podem ser usados para recuperar dados de arquivos CHK.
