O que é um arquivo .001? (Arquivo fax)

Arquivo de imagem fax criado por vários dispositivos de fax e programas de software; pode conter a imagem da primeira página de um fax de várias páginas, enquanto páginas subsequentes usam a extensão "002," "003," etc; usada para armazenar os dados de imagem de faxes recebidos.

O que é um arquivo .001? (Arquivo de backup de fusão multimídia)

Backup de um arquivo de projeto (.MFA) criado com o Multimedia Fusion, um programa de desenvolvimento de jogos; gerado pelo recurso Autobackup do programa; permite que o projeto seja restaurado caso o arquivo original seja corrompido ou seja acidentalmente excluído.

Mais Informações

NOTA: A Multimedia Fusion pode criar vários arquivos de backup com as extensões "001," "002," etc, dependendo de quantas cópias de backup foram selecionadas nas preferências do programa.

O que é um arquivo .001? (Backup do disco fantasma do Norton)

Backup de um disco rígido criado com o Norton Ghost; criado quando um backup é dividido em vários arquivos devido ao limite de tamanho de arquivo de 2GB nos DOS; outras partes do backup dividido têm extensões "002," "003," etc.

Mais Informações

Os backups divididos criados com o Norton Ghost também incluem um arquivo .GHO, que contém informações sobre os arquivos de backup e é necessário para restaurar o backup.

NOTA: O Norton Ghost foi renomeado como Ghost Solution Suite.

Snes9x Slot 2 Estado Salvo

O que é um arquivo .001? (Snes9x Slot 2 Estado Salvo)

Saved game state criado pelo Snes9x, um emulador do Super Nintendo Entertainment System (SNES); contém um instantâneo de jogo salvo no segundo de nove slots (slots 1-9); usado para salvar e carregar estados de jogo emulados SNES.

Mais Informações

Os slots de jogo salvos 1-9 correspondem a extensões de arquivo "000" através de "009" na versão Windows do Snes9x. Seus arquivos são nomeados usando o nome ROM seguido pela respectiva extensão de slot. Por exemplo, o slot GameA 2 seria nomeado GameA.001.

Os estados salvos são armazenados no diretório Saves de uma instalação do Snes9x. Em Mac versões do Snes9x, estados salvos usam a extensão .FRZ ("frozen"states).

NOTA: Ao contrário de um estado normal de jogo salvo, que só pode ser criado em determinados pontos do jogo, arquivos 001 podem ser criados a qualquer momento durante a jogabilidade.

O que é um arquivo .001? (Arquivo de arquivamento dividido)

Um arquivo 001 é o primeiro arquivo de um arquivo dividido criado com vários arquivos dividindo e juntando utilitários. Ele é usado para dividir grandes arquivos em arquivos menores e mais gerenciáveis. Arquivos 001 podem ser usados para salvar arquivos em vários dispositivos de armazenamento, como unidades USB ou CDs, ou para criar anexos de e-mail menores.

Mais Informações

O arquivo 001 é uma parte de um arquivo maior, o que significa que há pelo menos mais um arquivo dividido associado ao arquivo. Cada parte de um arquivo dividido recebe uma extensão numédua ascendente, que são .002, .003, etc.

NOTA: Programas como extensões de compostos de uso 7-Zip para arquivos divididos. Por exemplo, o 7-Zip cria arquivos divididos com as extensões .7Z.001, .7Z.002, etc.
