O que é um arquivo .pix? (Imagem alias PIX)

Imagem bitmap criada no formato Alias PIX proprietário; contém um cabeçalho seguido por pixels que são codificados em pacotes; originalmente usado pelo software Alias 3D, como o PowerAnimater, mas não é mais comumente usado.

O que é um arquivo .pix? (Arquivo de imagem bruta brl-CAD)

Formato de imagem bruta usado pelo BRL-CAD, um sistema de modelagem sólido de código aberto multiplataforma; salva dados de imagem em um formato não comprimido de 24 bits; usa o modo de cor RGB e não contém informações de dimensão.

Mais Informações

NOTA: Os arquivos PIX podem ser convertidos em arquivos de .PNG padrão usando o comando pix-png de BRL-CAD.

O que é um arquivo .pix? (Arquivo de banco de dados pci geomatics)

Arquivo PCIDSK (PCI Geomatics Database File) criado pela Geomatica, um programa de geomática usado para processar e analisar imagens geográficas; contém informações espaciais divididas em vários canais de imagem; muitas vezes usadas para armazenar conjuntos de dados geoespaciais pequenos e muito grandes, como imagens de satélite.

Mais Informações

O formato de arquivo PCIDSK foi criado pela PCI Geomatics em 1988. Foi projetado como um substituto para o formato UNIDSK, que dependia de arquiteturas de 16 bits e era incapaz de escalar para sistemas de 32 bits e 64 bits. O formato PCIDSK inclui mais flexibilidade para armazenar novos requisitos de dados.

Geomatica pode ser usado para sensoriamento remoto, análise geoespacial, fotogrametria digital, produção de mapas, mosaico e funções automatizadas do sistema de produtos. Geomatica está disponível nas edições Core e Prime.
