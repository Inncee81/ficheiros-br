O que é um arquivo .tpl? (Arquivo de predefinições da ferramenta Adobe Photoshop)

Um arquivo TPL é um arquivo de configurações usado pelo Adobe Photoshop, um programa usado para editar gráficos raster. Ele contém informações predefinidas para uma ferramenta photoshop, que pode incluir uma configuração de largura de traçado para a ferramenta de escova, um padrão para a ferramenta de preenchimento ou outra configuração. Os arquivos TPL são usados para salvar e reutilizar as configurações da ferramenta.

Mais Informações

As predefinições da ferramenta podem ser acessadas através do painel Predefinições da ferramenta (Window → Tool Presets) ou abrindo o Predefinido Manager selecionando Edit → Preset Manager... e, em seguida, escolhendo "Ferramentas" na lista suspensa.

O que é um arquivo .tpl? (Modelo de documento)

Um arquivo TPL é um arquivo de modelo criado e usado por vários tipos de aplicativos. Ele salva configurações padrão para o layout da página de um documento, texto e outras formatação. Os arquivos TPL são usados para criar documentos com um estilo e estrutura semelhantes.

Mais Informações

Os arquivos TPL são mais notavelmente usados pelo ACT! Word Processor e pela plataforma de comércio eletrônico Shopware. Os arquivos de modelo de exemplo incluem modelos de newsletter, modelos de papel timbrado da empresa e modelos de desenho.

O que é um arquivo .tpl? (Modelo do servidor de arquivos HTTP)

Um arquivo TPL é um modelo usado pelo HTTP File Server (HFS), um programa de compartilhamento de arquivos usado para enviar e receber arquivos em HTTP. Ele contém HTML dividido em seções que descrevem cada parte de uma página HTML servida a um navegador solicitante. Os arquivos TPL são usados pelo HFS para construir dinamicamente páginas HTML.

Mais Informações

Arquivos TPL armazenam principalmente HTML, mas também podem conter CSS e JavaScript. O HFS pega o código HTML (ou CSS ou JavaScript) de arquivos TPL categorizados por seções com suportes quadrados ([]). Algumas das seções incluem estilo, link, comentário e upload. Os arquivos TPL também incluem símbolos cercados por sinais percentuais (%) usado para definir cada seção.

Para abrir e editar arquivos TPL no HFS, você deve estar no "Modo Expert" no HFS. Para entrar no modo de especialista, selecione Menu → Other options → Switch to Expert mode. Uma vez no modo de especialista, selecione Menu novamente e escolha HTML template → Edit... ou Change file....

NOTA: Os modelos são normalmente armazenados na mesma pasta do arquivo hfs.exe.

O que é um arquivo .tpl? (Modelo Phorum)

Um arquivo TPL é um modelo usado pela Phorum, um sistema de placa de mensagem de código aberto baseado em PHP e MySQL. Ele contém código de modelo, que é principalmente HTML simples misturado com linguagem de modelo personalizado Phorum que consiste em declarações simples. Os arquivos TPL são usados para exibir dados dinâmicos gerados pelo Phorum, como resultados de pesquisa, informações de mensagens e listas de mensagens privadas.

Mais Informações

Captura de tela de um arquivo tpl no MacroMates TextMate 2

A linguagem de modelo encontrada em arquivos TPL consiste em instruções de modelo HTML e simples Phorum. Essas declarações suportam quatro tipos de dados, que podem ser inteiros, strings, constantes PHP ou variáveis de modelo. Uma vez que a linguagem de modelo é armazenada em arquivos TPL como texto simples, você pode abrir e editar arquivos TPL com qualquer editor de texto.

A Phorum normalmente usa arquivos TPL para armazenar código de modelo, mas também pode usar .PHP arquivos se eles forem colocados no diretório de modelos e forem nomeados com o nome base apropriado. Por exemplo, se o Phorum estiver exibindo o modelo do rodapé, ele primeiro procura footer.php no diretório de modelos e se o arquivo não existir, então procurará footer.tpl. A inclusão de arquivos PHP permite que os usuários usem apenas código PHP/HTML para modelos se não quiserem usar a linguagem de modelo Phorum exigida pelos modelos TPL.

O que é um arquivo .tpl? (Arquivo de imagem Wii)

Arquivo de imagem utilizado por jogos nintendo Wii como Mario Kart Wii; contém uma ou mais imagens que compõem menus que aparecem em jogos Wii e no próprio console Wii; inclui cabeçalho de paleta e imagem e seções de dados para cada imagem contida no arquivo para atribuir altura, largura, etc; normalmente encontradas em .SZS arquivos.
