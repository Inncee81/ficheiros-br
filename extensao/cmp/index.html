O que é um arquivo .cmp? (Arquivo de componente CADD genérico)

Componente de desenho criado com CADD genérico, um programa CAD mais antigo desenvolvido para Windows; usado para armazenar objetos vetoriais que podem ser inseridos em um arquivo de desenho CADD genérico (.GCD).

Mais Informações

NOTA: O CADD genérico foi substituído pelo General CADD Pro (GCP), que salva componentes à medida que .GXC arquivos. Os arquivos CMP podem ser abertos tanto pelo Generic CADD quanto pelo General CADD Pro.

O que é um arquivo .cmp? (Arquivo de componente da cláusula hotDocs)

Módulo de texto criado usando o software de preparação de formulário digital HotDocs; salva uma "cláusula" que contém texto, bem como componentes HotDocs (por exemplo, campos de formulário); fornece um objeto reutilizável para inserção em modelos HotDocs (.HFT ou .HPT arquivos).

Mais Informações

As cláusulas podem ser específicas para um modelo ou para uma biblioteca de cláusulas (.HDL arquivo). Por exemplo, se você criar uma cláusula durante a edição de um modelo HotDocs, ela será adicionada ao arquivo de componente CMP do modelo.

O que é um arquivo .cmp? (Arquivo de arreios de fio de borda sólida)

O arreio de fio projetado com o Siemens Solid Edge Wire Harness Design, um programa eCAD; contém o roteamento de arame e o design do arreio do cabo usado para organizar fios em um arquivo de montagem 3D (.ASM).

Mais Informações

O Solid Edge fornece feedback ao vivo para violações de design em arquivos CMP, o que impede a fiação incorreta dos dispositivos.

O que é um arquivo .cmp? (Perfil do gerente de conexão Windows)

Configurações do arquivo criado usando o Connection Manager, um conjunto de componentes de software que permite aos administradores do sistema configurar Windows perfis de conexão remota; salva informações como o nome de usuário, senha e o número de telefone para discar.

Mais Informações

Os arquivos CMP permitem que os administradores configurem configurações iniciais para a conexão dial-up do usuário.

NOTA: O Kit de Administração de Gerentes de Conexão (CMAK) é o nome da ferramenta Gerenciador de Conexões usada para configurar perfis.

  Microsoft Connection Manager Administration Kit
