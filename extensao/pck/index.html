O que é um arquivo .pck? (Arquivo godot engine game pack)

Um arquivo PCK contém um pacote de recursos criado pela Godot Engine, um mecanismo de jogo de código aberto usado para criar jogos 2D e 3D. Ele armazena informações sobre um jogo exportado da Godot Engine, que pode incluir texturas, modelos, cenas, sombreadores, scripts, música ou efeitos sonoros.

Mais Informações

Arquivos PCK permitem que os desenvolvedores adicionem funcionalidade aos jogos que já implantaram. Um desenvolvedor pode usar o pacote de recursos de jogos PCK por várias razões, incluindo a distribuição de patches para corrigir bugs, mods para permitir que os jogadores criem conteúdo para o jogo ou conteúdo para download para adicionar recursos a um jogo.

NOTA: Para exportar os recursos de um projeto para um arquivo PCK, selecione Project → Export e clique em Export PCK/Zip.

O que é um arquivo .pck? (Especificação do pacote Oracle e arquivo do corpo)

Um arquivo PCK é um pacote criado com software de banco de dados Oracle PL/SQL, mais notavelmente Allround Automations PL/SQL Developer. Contém uma especificação do pacote e um pacote na linguagem PL/SQL. Os arquivos PCK são usados para definir os procedimentos públicos armazenados, funções, tipos, etc. que podem ser acessados por outros aplicativos para evitar que os dados do pacote sejam modificados involuntariamente.

Mais Informações

Você pode abrir e editar arquivos PCK usando o DESENVOLVEDOR PL/SQL, que é desenvolvido pela Allround Automations. Você também pode visualizar e editar um PCK com um editor de texto, uma vez que ele é armazenado em texto simples.

O arquivo PCK permite que você combine as especificações de um pacote PL/SQL e o corpo do pacote em um arquivo. Normalmente, as especificações e o corpo são armazenados em arquivos separados. A especificação do pacote, que declara as variáveis, exceções, tipos, constantes, cursores e subprogramas acessíveis publicamente a partir de outros aplicativos, é comumente armazenada no arquivo .PKS. O corpo do pacote, que consiste no código que implementa os procedimentos, funções e cursores listados na especificação do pacote, é comumente armazenado no arquivo .PKB.

NOTA: PL/SQL é uma extensão de linguagem processual para a Linguagem de Consulta Estruturada (SQL). Ele está disponível no Banco de Dados Oracle (desde a versão 6), IBM DB2 (desde a versão 97) e no banco de dados timesTen em memória (desde a versão 1121).

Arquivo de dados de jogos usado pela Perfect World, um jogo gratuito de RPG online massivamente multiplayer( MMORPG); pode conter vários dados do jogo, incluindo informações mundiais e modelos de avatar, item e construção; usado para instalações e atualizações de jogos.

O que é um arquivo .pck? (Arquivo do pacote do gerenciador de configuração do centro do sistema)

Arquivo usado pelo System Center Configuration Manager (SCCM), um aplicativo usado para administrar grandes grupos de sistemas de computador Windows; contém um pacote de atualização, que pode armazenar arquivos de sistema e aplicativos; usado para atualizar sistemas clientes.

Mais Informações

Para distribuir arquivos PCK, o software SCCM primeiro constrói um arquivo PCK compactado com os arquivos de origem para a atualização e, em seguida, envia-o para um computador de Ponto de Distribuição (DP). O DP permite que computadores clientes registrados baixem pacotes disponíveis para atualizar seu software.

NOTA: O SCCM foi anteriormente chamado de Servidor de Gerenciamento de Sistemas (SMS) nas versões 2003 e anteriores.
