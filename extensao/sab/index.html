O que é um arquivo .sab? (Arquivo modelo ACIS SAB)

3D modelo criado no formato Standard ACIS Binary (SAB); economiza informações de geometria 3D e estrutura para um componente fabricado; usado para trocar informações de modelos entre vários programas CAD.

Mais Informações

Os arquivos SAB, salvos em formato binário, também podem ser salvos em um formato de texto usando arquivos .SAT.

O que é um arquivo .sab? (Arquivo saber)

Arquivo sabre de luz usado por Star Wars Jedi Knight: Jedi Academy, um jogo onde o jogador escolhe ter seu personagem Jedi seguindo o caminho claro ou escuro do lado escuro; contém dados para sabres de luz usados pelo personagem; inclui informações como a cor, comprimento, raio, do sabre e se é jogável ou duas mãos; referências .WAV arquivos para efeitos sonoros e um arquivo .GLM para o modelo.

Mais Informações

NOTA: O arquivo SAB é semelhante aos arquivos .VWP e .VEH usados por Star Wars Jedi Knight: Jedi Academy.
