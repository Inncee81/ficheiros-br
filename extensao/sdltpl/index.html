O que é um arquivo .sdltpl? (Modelo de projeto do estúdio SDL Trados)

Um arquivo SDLTPL é um modelo criado pelo SDL Trados Studio, um programa usado por empresas para editar, revisar e gerenciar projetos de tradução. Ele contém um ou mais arquivos adicionados por um usuário que pretende ser traduzido para uma linguagem diferente ou revisado por outro usuário. Os arquivos SDLTPL são usados para criar vários projetos com as mesmas configurações.

Mais Informações

Captura de tela de um arquivo sdltpl no SDL Trados Studio 2017

Os arquivos SDLTPL podem conter uma variedade de arquivos, como .DOC, .DOCX ou .XML arquivos, necessários para serem traduzidos e revisados. Os modelos também armazenam configurações de projeto que incluem a descrição do projeto, a data de vencimento do projeto, os usuários que podem acessá-lo e a origem e o idioma-alvo para completar a tradução.

Você pode abrir um modelo SDLTPL selecionando File → New → New Project e escolhendo "Criar um projeto baseado em um modelo de projeto". Você pode criar um modelo SDLTPL selecionando File → Save e escolher o formato de modelo SDLTPL.
