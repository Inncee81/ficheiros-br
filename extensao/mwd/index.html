O que é um arquivo .mwd? (Documento de gravação da Mariner)

Documento criado com o Mariner Write, um processador de texto simplificado para Mac OS X projetado para ser flexível e eficiente; pode conter textos, gráficos, cabeçalhos, rodapés e outros dados.

O que é um arquivo .mwd? (Arquivo de atalho do cliente MyWorkDrive)

Um arquivo MWD contém um atalho criado pelo MyWorkDrive Mapped Drive Client for Windows, um programa que permite que os usuários mapeiem uma unidade para arquivos para que possam acessá-los e editá-los com segurança e controle remoto. Ele armazena um link para um arquivo ou pasta armazenado no MyWorkDrive que pode ser acessado por um usuário com as permissões apropriadas de compartilhamento de arquivos.

Mais Informações

Arquivos MWD podem ser abertos em Windows clicando duas vezes neles. No entanto, o cliente MyWorkDrive Mapped Drive deve ser instalado e definido como o programa padrão para abrir os arquivos MWD. Quando você clicar duas vezes no arquivo MWD, a pasta ou arquivo vinculado no arquivo será aberta no MyWorkDrive Mapped Drive Client.
