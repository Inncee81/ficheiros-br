O que é um arquivo .fpk? (Arquivo de dados Civilization 4)

Arquivo de ativos de jogo usado por Sid Meier's Civilization IV, um jogo de estratégia baseado em turnos; salva dados de arte para gráficos de jogos; armazenado no diretório de instalação de Civilization IV\Assets\; pode ser extraído usando PakBuild para Windows ou Macapaka para Mac.

Mais Informações

Exemplo Civilization IV arquivos de arte incluem Art0.fpk, Assets0.fpk, Assets1.fpk, Assets2.fpk e Assets3.fpk.

NOTA: Arquivos FPK também são usados pelos Piratas de Sid Meier.

O que é um arquivo .fpk? (Arquivo JetForm FormFlow)

Formulários criados por produtos de software desenvolvidos pela JetForm ou Accelio; descontinuados após a aquisição dos produtos de software pela Adobe Systems, que agora usa o formato .PDF para documentos de formulário.
