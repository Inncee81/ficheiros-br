O que é um arquivo .ics? (Arquivo de calendário)

Um arquivo ICS é um arquivo de calendário salvo em um formato de calendário universal usado por vários programas de e-mail e calendário, incluindo Microsoft Outlook, Google Calendar e Apple Calendar. Ele permite que os usuários publiquem e compartilhem informações do calendário na web e por e-mail. Os arquivos ICS são frequentemente usados para enviar solicitações de reunião para outros usuários, que podem importar os eventos em seus próprios calendários.

Mais Informações

Os arquivos do calendário são salvos em um formato de texto simples. Eles contêm informações como título, resumo, hora de início e tempo final para o evento do calendário. O formato Calendário também suporta atualizações de eventos e cancelamentos.

Você pode converter rapidamente seu arquivo ICS em outros formatos de arquivos populares, como .CSV, .PDF, .XLSX e .HTML, com o programa web ICS File Converter.

NOTA: O Apple Calendar era anteriormente conhecido como Apple iCal, mas foi alterado em 2012 com o lançamento do OS X Mountain Lion.

O que é um arquivo .ics? (Arquivo de som do gravador IC)

Arquivo de áudio criado por um gravador Sony IC (por exemplo, o Sony ICD-R100), um dispositivo portátil usado para gravação de mensagens pessoais; comumente usado para armazenar áudio de voz, que é útil para ambientes forenses e legais, bem como para memorandos pessoais gerais.

Desenvolvedor IronCAD

Formato Não Disponível 

O que é um arquivo .ics? (Arquivo de desenho 3D IronCAD)

Modelo tridimensional ou cena criada com IronCAD, um programa CAD 3D; pode conter um ou mais objetos 3D com cores, texturas, efeitos de iluminação e dados de animação.

Mais Informações

NOTA: Os arquivos ICS podem ser abertos automaticamente em um programa de calendário, como o MS Outlook, devido à associação de arquivos iCalendar. Para abrir arquivos ICS no IronCAD, você pode precisar clicar com o botão direito do mouse no arquivo, selecionar "Abrir com" e escolher IronCAD como o programa.
