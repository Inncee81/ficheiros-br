O que é um arquivo .loadtest? (Arquivo de teste de carga do estúdio visual)

Um arquivo LOADTEST contém um teste de carga usado pelo Visual Studio, uma ferramenta de desenvolvimento de software para programas de Windows e aplicativos web. Ele armazena um ou mais testes web que são usados para simular muitos usuários acessando um servidor ao mesmo tempo. Os arquivos LOADTEST também incluem propriedades, como alvos do navegador, padrões de carga e perfis de usuário.

Mais Informações

Os testes de carga são usados pelo Visual Studio para vários tipos de testes. Eles podem ser usados para testar a capacidade de resposta de um aplicativo, como um aplicativo se sai em diferentes capacidades, como um aplicativo funciona sob cargas leves por durações mínimas ou se um aplicativo é executado com sucesso sob uma carga pesada por uma longa duração.

Você pode criar arquivos LOADTEST com o Assistente de Teste de Carga e editar suas propriedades com o Editor de Teste de Carga. Você também pode visualizar testes de carga quando eles são executados usando o Monitor de Teste de Carga e testes concluídos usando o Analisador de Teste de Carga.

O Visual Studio usa diferentes extensões de arquivo para uma variedade de tipos de arquivos. Algumas das diferentes extensões de arquivo de teste incluem LOADTEST, .GENERICTEST e .WEBTEST.
