O que é um arquivo .exl? (Arquivo lister de exportação)

Arquivo usado pelo Export Notes, um programa usado para converter bancos de dados do Lotus Notes em caixas de correio do Microsoft Outlook; contém uma lista de .NSF arquivos Do Lotus Notes criados usando a ferramenta Export Lister, que está incluída em algumas versões do Export Notes; usado para armazenar uma lista de arquivos para converter.

O que é um arquivo .exl? (Lista de Inclusão ou Exclusão do JSDAI)

Arquivo usado pelo JSDAI, uma API usada para manipular dados armazenados no idioma de modelagem de dados EXPRESS; pode conter uma lista de inclusão ou uma lista de exclusão de arquivos para o projeto; usado pelo compilador para incluir ou ignorar determinados arquivos; ajuda os desenvolvedores quando há muitos arquivos de projeto.

Mais Informações

JSDAI está disponível como uma ferramenta autônoma ou como um plugin para o software de desenvolvimento eclipse. Os desenvolvedores também podem baixar o código-fonte ou a biblioteca central compilada .JAR arquivos para usar para o desenvolvimento.
