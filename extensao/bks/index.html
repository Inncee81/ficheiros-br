O que é um arquivo .bks? (Arquivo de backup da planilha da Microsoft Works)

Arquivo de backup de planilha (arquivo .WPS) criado pelo Microsoft Works; gerado automaticamente como backup no caso de o arquivo de planilha original ficar corrompido e não puder ser aberto.

Mais Informações

A extensão do arquivo pode precisar ser alterada de "bks" para "wps" para abrir corretamente.

NOTA: O Microsoft Works foi descontinuado e não está mais disponível.

O que é um arquivo .bks? (Arquivo de configurações NTBackup)

Arquivo usado pelo NTBackup, um programa de Windows usado para criar backups; armazena uma lista de diretórios de backup e nomes de arquivos em um formato de texto simples; pode especificar inclusões e exclusões de diretórios ou arquivos; também pode especificar para backup do estado do sistema.

Mais Informações

Os arquivos BKS são usados para criar backups automatizados de arquivos Windows. Eles podem armazenar arquivos, informações de registro e outras informações de status do sistema. Quando executado, o NTBackup cria .BKF arquivos de backup.
