O que é um arquivo .hid? (Arquivo KeepSafe)

Arquivo de segurança criado pelo KeepSafe, um aplicativo de Android e iOS que oculta suas imagens e vídeos; contém um vídeo ou imagem que está escondido do usuário até que o pino apropriado seja inserido; também aparece com a extensão de arquivo .HID2.

Mais Informações

Quando você escolhe um vídeo ou imagem a ser ocultado, o KeepSafe anexa uma série de números que precedem o nome do arquivo e a extensão HID até o final do arquivo. A extensão oculta o arquivo de ser reconhecido como o que realmente é, como um arquivo .JPG, e impede que outros aplicativos no dispositivo reconheçam o arquivo.

Por exemplo, o KeepSafe pode salvar um arquivo de imagem como 8945715774357453030.IMG_11429.JPG.HID. O "8945715774357453030" é uma sequência aleatória de números gerados pelo KeepSafe e "HID" é a extensão adicionada pelo programa.

NOTA: A extensão HID foi substituída pela extensão .KSD em 2013.

O que é um arquivo .hid? (Arquivo de configurações remotas sony Ericsson)

Armazena informações de configuração para usar um celular Sony Ericsson como controle remoto Bluetooth; mapeia chaves no telefone para determinadas chaves ou comandos no computador.

Mais Informações

Diferentes arquivos HID podem ser carregados para controlar diferentes programas, como jogadores de áudio e vídeo, navegadores de Web, etc.

ARQUIVOS HID podem ser transferidos para o celular usando o utilitário Bluetooth File Exchange em Mac OS X ou o Painel de Controle de Dispositivos Bluetooth em Windows.
