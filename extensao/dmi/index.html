O que é um arquivo .DMI? (Arquivo de ícone do criador de sonhos BYOND)

Imagem bitmap usada pela BYOND, um mecanismo de jogo usado para criar e jogar jogos multiplayer online 2D; salva um ou mais estados gráficos de ícones, bem como informações de animação; usado para gráficos de jogos, como personagens ou objetos que são visíveis em um mapa; semelhante a uma folha de sprite.

Mais Informações

Arquivos DMI podem ser editados no Dream Maker, um programa incluído no conjunto de ferramentas de desenvolvimento de jogos BYOND.

O que é um arquivo .DMI? (Arquivo de configuração de interface de nó estatístico)

Um arquivo DMI é um arquivo de configuração usado pela TIBCO Statistica, um programa de análise utilizado por cientistas de dados e analistas. Ele armazena informações em formato de texto simples que define a interface do usuário (Interface do Usuário) de um nó em um espaço de trabalho. Os arquivos DMI incluem informações sobre a natureza do nó, como o nome do nó, o grupo ao qual pertence, a descrição do nó, o tipo de nó e parâmetros e valores do nó.

Mais Informações

A Statistica fornece espaços de trabalho para os usuários definirem fluxos de trabalho para análise de dados. No espaço de trabalho, os usuários podem adicionar nós do Navegador Node em Statistica e conectá-los a fontes de dados para vários fins, como preparação de dados, análise e gráficos.

Cada nó é definido por dois arquivos: arquivos SVX e DMI. Os arquivos SVX armazenam informações que definem o funcionamento do nó enquanto o arquivo DMI armazena informações que definem a interface de usuário do nó, que é como o nó aparece e quais informações podem ser inseridas no nó. Ambos os arquivos têm o mesmo nome de arquivo. Por exemplo, um nó chamado "DataAnalysis" consiste em um arquivo DataAnalysis.svx e DataAnalysis.dmi arquivo.

A statistica vem com vários nódulos pré-configurados do espaço de trabalho. Os arquivos SVX e DMI que compõem esses nós estão localizados no seguinte diretório:

C:\Program Files\StatSoft\Statistica 13\DataMiner\
