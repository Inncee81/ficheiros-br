O que é um arquivo .vdf? (Arquivo de definições de vírus antivir)

Arquivo de dados usado pelo AntiVir, um aplicativo antivírus para Windows; salva definições de vírus, que contêm assinaturas que são usadas para reconhecer vírus no sistema do computador; muitas vezes baixado durante atualizações de software para que o software possa verificar se há novos vírus.

Mais Informações

Os arquivos VDF são salvos no diretório de instalação do AntiVir e têm os nomes de arquivos vbase000.vdf, vbase001.vdf, etc.

O que é um arquivo .vdf? (Arquivo de dados do jogo gótico)

Arquivo de dados usado pela série gótica de videogames; contém mundos, personagens, modelos 3D, texturas e sons; Os arquivos VDF Góticos II são armazenados na pasta /Gothic II/Data.

Mais Informações

Arquivos VDF podem ser editados com o GoMan ou um editor de hex, mas normalmente não devem ser alterados. Modificar incorretamente arquivos VDF pode torná-los inutilizáveis.

O que é um arquivo .vdf? (Arquivo de dados da válvula)

Um arquivo VDF é um formato de arquivo de dados usado pelo mecanismo de jogo Source da Valve. Ele contém vários tipos de metadados de jogo, incluindo dados para recursos, scripts de instalação, scripts de configuração e elementos de visualização.

Mais Informações

Arquivos VDF podem ser vistos frequentemente ao instalar jogos distribuídos através do cliente do jogo Steam da Valve, como Half-Life 2 e Counter-Strike. O arquivo possui o formato KeyValues da Valve, que é um formato de texto simples que salva parâmetros de valor-chave em uma hierarquia de árvore. O formato é semelhante ao JSON e YAML. Enquanto os arquivos VDF são feitos para serem usados pelo mecanismo de jogo Valve Source, você pode abrir os arquivos com um editor de texto para visualizar o conteúdo.

NOTA: Você pode visualizar mais informações sobre o formato KeyValues da Valve na Página wiki de valores-chave.

O que é um arquivo .vdf? (Filtro de vídeo VirtualDub)

Filtro plugin para VirtualDub, um programa de utilidade de captura e processamento de vídeo para Windows; adiciona recursos extras, como efeitos especiais e suporte adicional de arquivos ao programa VirtualDub.

Mais Informações

Os filtros VDF podem ser carregados no VirtualDub clicando no botão "Carregar" na caixa de diálogo Adicionar filtros. Eles serão carregados automaticamente se os arquivos VDF forem colocados na pasta plugins dentro da pasta do programa VirtualDub.
