O que é um arquivo .arduboy? (Pacote de Jogos Arduboy)

Um arquivo ARDUBOY é um pacote de jogos usado pela Arduboy, uma plataforma de jogos de código aberto de 8 bits. Ele contém um jogo compilado, uma imagem de banner (700px x 192px), uma ou mais capturas de tela (128px x 64px) e um metadados como informações de título, versão, autor e configuração de botões.

Mais Informações

Os pacotes ARDUBOY permitem que os usuários carreguem seus jogos para arduboy, juntamente com outras informações úteis, como banners e capturas de tela. Os pacotes são compactados com compactação Zip, o que significa que é um arquivo .ZIP renomeado com a extensão "arduboy".

Você pode criar um pacote ARDUBOY comprimindo-o como um arquivo ZIP. Você pode abrir um pacote ARDUBOY renomeando o arquivo para "zip" e descompactando o arquivo.

NOTA: Você pode baixar e instalar jogos em seu dispositivo Arduboy com Arduboy Manager ou Team ARG. Game Loader.
