O que é um arquivo .abk? (Arquivo do livro de abertura da arena)

Arquivo usado pela Arena, uma interface de usuário gráfico freeware usada por motores de xadrez; contém movimentos "livro de abertura", que armazenam estratégias de abertura para um jogador de inteligência artificial; podem conter milhões de movimentos predefinidos.

Mais Informações

Arquivos ABK são usados para reduzir a despesa computacional de determinar estratégias ideais para os primeiros vários movimentos de um jogo de xadrez.

Desenvolvedor N/A
Categoria Cópia de Segurança
Formato Binário 

O que é um arquivo .abk? (Arquivo de backup automático)

Arquivo de backup criado por vários programas, incluindo CorelDRAW, Ability Office Write e PrintMaster Gold; contém um backup do documento ou projeto atualmente aberto; usado para recuperar dados do programa no caso de uma falha no computador ou corrupção de arquivos.

Mais Informações

NOTA: Os arquivos ABK normalmente não devem ser abertos diretamente. Normalmente, o programa que os criou orientará os usuários no processo de recuperação de quaisquer dados perdidos.

Programas que abrem o arquivo .abk (Arquivo de backup automático)

  CorelDRAW Graphics Suite 2020
  Broderbund PrintMaster 2020

O que é um arquivo .abk? (Necessidade de arquivo de áudio subterrâneo de velocidade)

Arquivo de áudio usado por Need for Speed Underground (NFSU), um jogo de corrida de carros; contém discurso de jogo, música e outros arquivos de som; usado para reproduzir efeitos sonoros e música durante a jogabilidade.

Programas que abrem o arquivo .abk (Necessidade de arquivo de áudio subterrâneo de velocidade)
