O que é um arquivo .cad? (Arquivo QuickCAD do Autodesk)

Um arquivo CAD é um arquivo CAD criado pelo Autodesk QuickCAD, um programa de desenho vetorial CAD 2D que foi descontinuado em 2004. Contém um desenho 2D técnico, que inclui informações de geometria e medição. Os arquivos CAD são frequentemente usados para armazenar layouts de piso e projetos mecânicos.

Mais Informações

Os arquivos CAD QuickCAD só são suportados pelo QuickCAD. Para abri-los em outro programa, os arquivos devem ser convertidos em um formato mais amplamente suportado, como o formato de .DWG de desenho AutoCAD ou o formato de .DXF do Drawing Exchange. Infelizmente, o único programa capaz de converter arquivos CAD QuickCAD é o Autodesk QuickCAD, que não é mais desenvolvido.

NOTA: Autodesk QuickCAD foi originalmente Softdesk Drafix Quick CAD na década de 1990. Mas quando o Softdesk foi adquirido pela Autodesk, o Drafix Quick CAD foi reembalado como Autodesk QuickCAD, que foi desenvolvido até a versão 8 em 2004.

O que é um arquivo .cad? (Arquivo BobCAD-CAM)

Arquivo criado por produtos de design CAD BobCAD-CAM; armazena dados CAD incluindo geometria e dimensões de peças 2D e 3D; usado para projetar peças e produtos para fabricação.

Mais Informações

BobCAD-CAM está disponível em várias edições, incluindo Mill, Lathe, BobART, Wire EDM e CNC Router, WaterJet & Plasma. BobCAD-CAM pode importar conteúdo para arquivos CAD de arquivos .SLDPRT, .DWG, .DSF, .3DM, .IGES, .STEP, .STL, .SAT, .X_B e .X_T arquivos.

NOTA: Os arquivos CAD são criados pela versão 21 do BobCAD-CAM e anteriormente. Versão 22 e posteriormente usar arquivos .BBCD.
