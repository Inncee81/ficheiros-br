O que é um arquivo .cas? (Arquivo de licença de cascata de autodesk)

Arquivo de licença usado por produtos autodesk como AutoCAD e Revit; contém informações que validam uma cópia de software comprada; usada em uma configuração de licença "em cascata", onde uma instalação de rede do software pode ser usada por vários computadores em rede.

Mais Informações

Arquivos CAS usam o nome de arquivo CascadeInfo.cas. Esses arquivos podem potencialmente se tornar corruptos se o computador falhar inesperadamente, negando acesso ao software. Se isso ocorrer, consulte o suporte do Autodesk para resolver o problema.

Por padrão, o arquivo CascadeInfo.cas está localizado no seguinte diretório:

  • Windows Vista/7: C:\ProgramData\Autodesk\ADLM\
  • <Windows XP: C:\Documents and Settings\All Users\Application Data\Autodesk\ADLM\
  • Mac OS X: /Library/Application Support/Autodesk/Adlm/.config/

Desenvolvedor The Creative Assembly
Categoria Imagens 3D
Formato Binário 

O que é um arquivo .cas? (Arquivo modelo de guerra total)

3D arquivo modelo usado por alguns jogos na série de jogos de estratégia Total War; contém modelos para edifícios, unidades e outros objetos de jogo; salvo com .TGA e imagens .DDS que são usadas para texturização dos modelos.

Mais Informações

Os arquivos CAS podem ser importados e exportados com o software de modelagem 3ds Max usando o script Vercengetorix CAS Importer/Exporter.
