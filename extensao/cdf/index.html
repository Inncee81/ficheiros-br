O que é um arquivo .cdf? (Arquivo de definição de chip Affymetrix)

Arquivo de biblioteca usado pelo software de análise de microarray de DNA Affymetrix; especifica o layout de um array "GeneChip" e contém a definição dos dados armazenados em um arquivo de .CEL Affymetrix; usado para entender e analisar dados CEL.

Mais Informações

A GeneChip é um chip físico contendo dados biológicos de microarray de DNA. GeneChips são fabricados pela Affymetrix.

O que é um arquivo .cdf? (Formato de dados comum)

Criar arquivos no Formato de Dados Comuns (CDF), formato usado para armazenar dados escalares e multidimensionais de forma independente da plataforma-alvo e disciplina; usado como um formato científico de armazenamento e intercâmbio de dados por vários pesquisadores e organizações e, muitas vezes, por aqueles no governo.

Mais Informações

Os arquivos do Formato de Dados Comuns podem ser manipulados usando o kit de ferramentas de software CDF fornecido pelo Space Physics Data Facility (SPDF), uma parte do Goddard Space Flight Center (GSFC). O software é multiplataforma.

O Formato de Dados Comuns é suportado com interfaces para linguagens de programação como C, Fortran, C#, Perl, Java. Ele também é suportado por pacotes de software comercial, como MathWorks MATLAB, ITT Visual Information Solutions' Interactive Display Language #IDL#, e Weisang FlexPro.

O formato CDF foi adotado como um formato padrão por organizações como o projeto International Solar-Terrestrial Physics #ISTP# e o Projeto Central Data Handling Facilities #CDHF#, uma entidade científica de processamento de dados.

NOTA: O formato CDF difere do formato NetCDF, que é baseado no CDF, mas usa convenções ligeiramente diferentes. Às vezes, os arquivos NetCDF usam a extensão "cdf", mas o formato é incompatível.

O que é um arquivo .cdf? (Arquivo de formato de documento computável)

Documento interativo criado no CDF (Computable Document Format, formato de documento computável), um padrão de documento desenvolvido pela Wolfram Research; pode conter gráficos interativos e outros materiais visuais, além de formatação de texto e página estáticas; pode carregar dados de visualização sob demanda dentro do arquivo do documento.

Mais Informações

Os documentos do CDF podem ser usados para vários fins, incluindo apresentações de negócios, artigos de revistas e materiais de e-learning, como livros didáticos e tutoriais. O formato é usado por professores, cientistas, editores e outros.

CDF arquivos podem ser abertos usando o Wolfram CDF Player, que é um aplicativo autônomo e um plug-in de navegador Web fornecido livremente pela Wolfram Research. O jogador suporta plataformas Mac, Windows, Linux e mobile.

O que é um arquivo .cdf? (Arquivo de definição de conteúdo)

Formato de dados baseado em XML projetado para armazenar e compartilhar conteúdo e metadados abstratos do Enterprise Content Management (ECM) e metadados; pode ser usado para trocar conteúdo entre repositórios e organizações.

Mais Informações

O formato CDF foi originalmente definido como parte da estrutura de Serviços de Transformação de Conteúdo (CTS) pelo Enterprise Content Management Group (ECMG).

O que é um arquivo .cdf? (Arquivo de formato NetCDF)

Arquivo de dados criado no formato Common Data Form da NetCDF, que é baseado no formato padrão de dados comuns (CDF), mas difere de várias maneiras e, portanto, não é compatível; usado para armazenar dados científicos de forma independente da plataforma e disciplina para a qual é usado; mais comumente usa a extensão de arquivo .NC.

Mais Informações

O que é um arquivo .cdf? (Arquivo de dados nominal da empresa de contabilidade de pequenas empresas)

Arquivo de dados contábeis criado pela Nominal Small Business Accounting, um aplicativo usado para rastrear finanças da empresa; contém contas, itens de compra e venda, termos de câmbio, métodos de pagamento, códigos fiscais e informações sobre folha de pagamento de funcionários para uma empresa.

Mais Informações

Os arquivos CDF podem ser usados para armazenar todas as informações contábeis de um negócio de pequeno porte.
