O que é um arquivo .rml? (Arquivo de modelo de relatório do Elixir)

Arquivo usado pelo Elixir Repertório, um mecanismo de relatórios que suporta muitos tipos de banco de dados backend; armazena um modelo para um relatório e especifica o layout e os espaços reservados para preencher o conteúdo do documento; pode ser usado para renderizar relatórios nos formatos .PDF, .RTF e .XLS.

Mais Informações

Arquivos RML residem no servidor Elixir Repertório e são analisados pelo mecanismo de relatório quando as solicitações são feitas. O repertório do Elixir é frequentemente referido como "Elixir Report"

NOTA: Elixir Technology Elixir Repertório é descontinuado.

O que é um arquivo .rml? (Arquivo de configuração FTG)

Arquivo De configurações criado pelo aplicativo de configuração FREE-To-Guest (FTG), um programa usado para configurar dispositivos LG Electronics, como caixas de televisão set-top boxes (STBs); salva as configurações do mapa do canal FTG e do menu do instalador; pode ser usado para restaurar a configuração do dispositivo.

Mais Informações

Os dispositivos FMA (Free-To-Guest Management Appliance, aparelho de gerenciamento gratuito) são frequentemente usados em hotéis para mostrar os canais de televisão suportados em televisores de quartos de hóspedes.

Desenvolvedor Autodesk
Categoria CAD
Formato XML 

O que é um arquivo .rml? (Arquivo de linguagem de marcação vermelha)

Arquivo criado pelo Volo View, um programa usado para marcar desenhos CAD; salvo em um formato XML e contém anotações e outras marcações que são exibidas em texto vermelho quando sobrepostas em um desenho; permite que os revisores de design incorporem notas em desenhos CAD.

Mais Informações

Volo View foi descontinuado em 2009 e não é mais suportado pela Autodesk. Em vez disso, os revisores podem marcar desenhos CAD usando um produto como Autodesk Design Reviewer, que salva notas de marcação de linha vermelha em .DWF e .DWFX arquivos.

NOTA: O tipo de arquivo RML foi suportado pelo AutoCAD até o AutoCAD 2005, que incluiu uma ferramenta de linha de comando do importador RML chamada RMLIN. Esta ferramenta importa a marcação em uma camada de desenho bloqueada chamada "_Markup_"
