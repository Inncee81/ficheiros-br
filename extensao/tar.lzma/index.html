O que é um arquivo .TAR.LZMA? (Tarball comprimido LZMA)

"Tarball" (arquivo .TAR) compactado com .LZMA (Lempel-Ziv-Markov chain Algorithm) e, portanto, usa a extensão do arquivo composto "tarlzma;" muitas vezes usado para distribuir pacotes de software linux.

Mais Informações

Os arquivos TARLZMA podem ser descompactados primeiro descomprindo o arquivo e, em seguida, extraindo arquivos do tarball resultante. Este processo de duas etapas é comumente tratado em uma etapa por vários utilitários de descompressão de arquivos. Por exemplo, você pode extrair o conteúdo do arquivo com o programa Unix tar (assumindo lzma está instalado) usando este comando:

tar --lzma -xvpf filetarlzma
