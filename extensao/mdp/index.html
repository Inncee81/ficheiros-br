O que é um arquivo .mdp? (Imagem firealpaca)

Um arquivo MDP é um arquivo de imagem criado pelo FireAlpaca, uma ferramenta de pintura gratuita para Windows e macOS. É o formato nativo usado para salvar imagens no FireAlpaca, que pode incluir várias camadas de imagem, máscaras, estêncil, anotações, filtros e outros elementos.

Mais Informações

Captura de tela de um arquivo mdp no FireAlpaca 2

O arquivo MDP é o tipo de arquivo principal associado ao FireAlpaca. Ele é usado para salvar imagens que podem ser fechadas e reabertas pelo FireAlpaca para mais edição. No entanto, o formato não é amplamente suportado. Portanto, quando você terminar de editar sua imagem no arquivo MDP, você pode exportar o arquivo como um arquivo .PNG, .JPG, .BMP ou .PSD.

Você pode criar um arquivo MDP no FireAlpaca selecionando File → New... e, em seguida, selecionando File → Save ou Save As... e selecionando o formato "FireAlpaca". Você pode abrir um arquivo MDP no FireAlpaca selecionando File → Open... e escolhendo o arquivo MDP.

O que é um arquivo .mdp? (Imagem MediBang Paint Pro)

Um arquivo MDP é uma imagem criada pelo MediBang Paint Pro, um aplicativo gratuito de pintura e criação de quadrinhos para Windows, macOS, Android e iOS. É o formato nativo usado para salvar imagens no MediBang Paint Pro. Os arquivos MDP armazenam uma ou mais camadas, que podem incluir anotações, estêncil, máscaras e outras formas.

Mais Informações

Captura de tela de um arquivo mdp no MediBang Paint Pro 21

O arquivo MDP é o principal tipo de arquivo associado ao MediBang Paint Pro. O formato é suportado apenas pelo MediBang Paint Pro, mas pode ser exportado para formatos mais populares, como .PNG, .JPG, .BMP, .TIFF, .WEBP e .PSD.

Você pode criar um arquivo MDP no MediBang Paint Pro selecionando File → New... e, em seguida, selecionando File → Save ou Save As... e selecionando o formato "MediBang Paint Pro". Você pode abrir um arquivo MDP no MediBang Paint Pro selecionando File → Open... e escolhendo o arquivo MDP.

O que é um arquivo .mdp? (Arquivo visual C++ 5 Workspace)

Configurações do arquivo criado pelo Visual C++ 5, um programa usado para desenvolver programas Windows usando a linguagem de programação C++; armazena as configurações de configuração do espaço de trabalho; usado para salvar e carregar layouts personalizados do espaço de trabalho.

NOTA: Visual C++ está incluído na ferramenta de desenvolvimento de software Visual Studio da Microsoft.
