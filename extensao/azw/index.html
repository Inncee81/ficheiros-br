O que é um arquivo .azw? (Arquivo eBook do Amazon Kindle)

Um arquivo AZW é um eBook digital formatado para o Amazon Kindle, um dispositivo de leitura sem fio. Contém texto de livro formatado e marcadores de capítulos e é semelhante a um arquivo .PRC Mobipocket. Os arquivos AZW podem ser transferidos de um PC para o Kindle através de uma conexão USB padrão e também podem ser lidos usando o software de desktop Kindle.

Mais Informações

A maioria dos arquivos AZW contêm proteção DRM que só permite que eles sejam visualizados no Amazon Kindle ou com o software de leitura Amazon Kindle. No entanto, pode ser possível visualizar eBooks Kindle desprotegidos em um PC com Mobipocket Reader alterando a extensão do arquivo de "azw" para "prc"
