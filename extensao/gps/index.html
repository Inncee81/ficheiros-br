O que é um arquivo .GPS? (Arquivo de configurações genepix)

Arquivo criado pelo GenePix Pro, um programa de análise de imagens de microarray; contém configurações para aquisição (o laser usado, tensões e área de varredura), análise (localização e identificação de características na imagem) e exibição (brilho, contraste e mapeamento de cores) de uma imagem.

O que é um arquivo .GPS? (Arquivo de pele do jogador GOM)

Arquivo de pele usado pelo GOM Player, um programa de reprodução de vídeo para Windows; contém um arquivo de arquivos gráficos e arquivos de especificação XML para a pele; permite que os usuários alterem a aparência do jogador; podem ser instalados clicando duas vezes no arquivo.

Mais Informações

Arquivos GPS são .ZIP arquivos que contêm arquivos de imagem .BMP e tipicamente dois arquivos .XML chamados LIST.XML e SKIN.XML. Esses dois arquivos XML fornecem as informações de como cada imagem de bitmap no arquivo é exibida para a pele. As imagens podem ser para quadros de programa, botões, controles deslizantes, logotipos e outras partes da interface.

Uma vez que as skins do GOM Player são salvas em um formato Zip, você pode alterar sua extensão para "zip" e abri-las com qualquer utilitário de descompressão zip, como Corel WinZip ou ESTsoft ALZip.

O que é um arquivo .GPS? (Arquivo de coordenadas Do Survey Pro)

Arquivo usado pelo Survey Pro, um programa usado com coletores de dados GPS do Tripod Data Systems; contém pontos de controle de coordenadas de latitude e longitude coletados durante o levantamento geográfico; usado para calibrar coordenadas GPS com locais conhecidos.

Mais Informações

Arquivos GPS podem ser importados no Survey Pro selecionando a opção Job → Import Coordinates.
