O que é um arquivo .spiff? (Formato de arquivo de intercâmbio de imagem ainda)

Imagem bitmap armazenada no formato SPIFF, que foi projetada para interligar arquivos de imagem entre ambientes de aplicativos; contém dados de imagem compactados divididos em seções de cabeçalho, diretório, dados de imagem e dados indiretos; usado para armazenar dados de imagem em um disco rígido ou para transmitir dados de imagem em uma rede.

Mais Informações

SPIFF Images suportam imagens bitonal ou de tom contínuo (cor ou escala de cinza). Eles também suportam vários algoritmos de compressão diferentes, incluindo JBIG e JPEG.

NOTA: O formato SPIFF foi projetado para substituir o formato .JFIF. Ainda é retrocompatível com JFIF.
