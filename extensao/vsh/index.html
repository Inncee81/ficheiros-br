O que é um arquivo .vsh? (Arquivo sombreador vertex)

3D arquivo de sombreamento usado por aplicativos que usam a biblioteca gráfica OpenGL; contém uma função gráfica usada para sombrear um vértice, que é um ponto no espaço 3D; executado por uma GPU a fim de mostrar efeitos de sombreamento 3D em um objeto.

ARQUIVOS VSH diferem de arquivos .FSH, que são sombreadores de fragmentos.

O que é um arquivo .vsh? (Arquivo de projeto do VideoStudio HTML5)

Um arquivo VSH é um projeto de vídeo HTML5 criado pelo Corel VideoStudio, um programa usado para criar slideshows e filmes. Ele contém dados do projeto de vídeo, da mesma forma que um arquivo .VSP, mas também pode incluir capítulos e hiperlinks.

Captura de tela de um arquivo vsh no Corel VideoStudio 2019

Para abrir um arquivo de projeto VideoStudio VSH HTML5, siga estas etapas:

  1. Selecione File → Open Project....
  2. Navegue até a localização do seu arquivo VSH e selecione o arquivo.
  3. Clique em Open.

Para criar um arquivo de projeto VideoStudio VSH HTML5, siga estas etapas:

  1. Selecione File → New HTML5 Project.
  2. Edite seu projeto e selecione File → Save ou Save As....
  3. Escolha o local de salvamento, nomeie o projeto e clique em Salve.

NOTA: O VideoStudio, originalmente de propriedade da Ulead, foi adquirido pela Corel em 2006.
