O que é um arquivo .lightkeyproj? (Projeto Lightkey)

Um arquivo LIGHTKEYPROJ contém um projeto criado pela Lightkey, um programa profissional usado para configurar e controlar luminárias compatíveis com DMX para vários tipos de locais. Armazena informações do projeto, que incluem luminárias e mostras de luz configuradas para uma exibição de iluminação. Os arquivos LIGHTKEYPROJ estão funcionando arquivos, o que significa que eles podem ser modificados, salvos, fechados e depois reabertos para edição posterior.

Mais Informações

Captura de tela de um arquivo lightkeyproj em Monospace Lightkey 3

Os projetos Lightkey são comumente criados, editados e salvos pelos usuários do Lightkey. Os usuários podem criar vários projetos para diferentes instalações de iluminação, mas normalmente só usam um, já que o projeto pode armazenar qualquer número de mostras de luz para uma instalação de iluminação.

Em um projeto Lightkey, você pode modificar o layout da instalação de iluminação, adicionar luminárias conectadas e definir predefinições de iluminação, pistas e sequências. Você também pode visualizar os vários programas de luz salvos no arquivo do projeto.

Para abrir um arquivo LIGHTKEYPROJ no Lightkey, selecione File → Open... ou Open Recent. Para criar um arquivo LIGHTKEYPROJ no Lightkey, selecione File → New Project, modifique o projeto e selecione File → Save.
