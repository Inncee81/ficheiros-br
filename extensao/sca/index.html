O que é um arquivo .sca? (Arquivo de varredura do Norton AntiVirus)

Arquivo de dados criado pelo Norton AntiVirus, um programa que protege computadores desktop contra malware; contém uma lista de arquivos no disco rígido que foram digitalizados ou precisam ser digitalizados; usados pelo Norton AntiVirus para dizer quais arquivos precisam ser digitalizados em seguida.

Mais Informações

Os arquivos SCA estão tipicamente localizados no diretório Tasks de uma instalação do Norton AntiVirus. Eles são comumente vistos com o nome de arquivo mycomp.sca.

NOTA: A Symantec introduziu a extensão "sca" com o Norton AntiVirus 2002. O Norton AntiVirus tornou-se Norton Security em 2014.

Arquivo de script scala

Desenvolvedor Scala
Categoria Executável
Formato Não Disponível 

O que é um arquivo .sca? (Arquivo de script scala)

Script escrito com o Scala, um programa de gerenciamento de sinalização digital; inclui instruções que determinam como e quando imagens digitais são exibidas; usadas para controlar outdoors eletrônicos, sinais digitais dinâmicos, publicidade DOOH (Digital-Out-of-Home), TV de funcionários, pôsteres eletrônicos e outros tipos de sinais digitais.

Mais Informações

Os scripts do Scala são escritos na língua ScalaScript, que usa uma sintaxe proprietária. Uma vez que os arquivos SCA são salvos em um formato de texto simples, eles podem ser abertos e editados com um editor de texto básico.
