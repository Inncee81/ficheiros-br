O que é um arquivo .update? (Arquivo de atualização do ISAPI)

Arquivo usado pelo ISAPI Loader, um programa que permite que os DLLs sejam "hot-swapped" em um servidor web Microsoft IIS durante a execução; contém conteúdo de arquivo DLL atualizado para o servidor Web; automaticamente reconhecido e atualizado pelo Carregador ISAPI.

Mais Informações

O carregador ISAPI requer que todos os arquivos DLL atualizados sejam renomeados com o RUN de extensão. Ao descobrir um novo arquivo UPDATE, ele termina as solicitações atuais Web servidor, realiza novas solicitações de servidor Web, altera o arquivo RUN existente para usar o BACKUP de extensão, renomeia o arquivo UPDATE para RUN, recarrega o novo arquivo RUN e, em seguida, retoma Web operações do servidor normalmente.

ISAPI, que significa Interface de Programação de Aplicativos do Servidor de Internet, habilita dois componentes no IIS, extensões e filtros. Extensões são aplicativos executados no IIS, e os filtros são extensões para a funcionalidade do servidor web IIS.
