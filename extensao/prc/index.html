O que é um arquivo .prc? (Arquivo mobipocket eBook)

EBook digital salvo no formato Mobipocket "Mobi"; pode ser salvo como um arquivo protegido ou desprotegido; arquivos de RPC protegidos incluem proteção de direitos autorais DRM que limita os dispositivos nos quais o arquivo pode ser aberto.

Mais Informações

Arquivos DA RPC podem ser visualizados em um PC usando Mobipocket Reader Desktop ou transferidos para um smartphone ou PDA. Arquivos de RPC desprotegidos também podem ser visualizados no dispositivo de leitura portátil Amazon Kindle.

O que é um arquivo .prc? (Arquivo de código de recurso de palma)

Pacote instalador para um aplicativo Palm OS; também pode ser usado para atualizações do programa Palm OS ou do sistema; pode ser instalado em um Palm Pilot a partir de um computador Mac ou Windows usando o HotSync Manager dentro do programa Palm Desktop.

Mais Informações

NOTA: A extensão "prc" às vezes é usada para documentos de texto do Palm OS também.

Arquivo compacto de representação do produto

Desenvolvedor Adobe Systems
Categoria Imagens 3D
Formato Binário 

O que é um arquivo .prc? (Arquivo compacto de representação do produto)

Arquivo criado no formato Compact de Representação de Produtos (RPC), que é usado para representar modelos e conjuntos 3D; contém estrutura do produto, geometria e informações de fabricação de produtos (PMI); armazena dados em um formato altamente compactado.

Mais Informações

Os arquivos da RPC são otimizados para armazenar, carregar e exibir dados 3D. Eles também suportam fluxos de trabalho de documentação baseados em PDF, uma vez que podem ser incorporados em arquivos .PDF.
