O que é um arquivo .XMF? (Arquivo de malha Cal3D XML)

Arquivo de malha de modelo tridimensional criado com Cal3D, uma biblioteca de animação de caracteres de código aberto; contém polígonos compostos de nós e bordas, que compõem a geometria do personagem; dá ao modelo sua estrutura esquelética.

Mais Informações

As malhas Cal3D são mais comumente armazenadas usando o formato binário .CMF.

NOTA: Os modelos Cal3D são usados para aplicativos como o IMVU, um aplicativo de bate-papo on-line 3D que usa arquivos XMF para avatares virtuais.

O que é um arquivo .XMF? (Arquivo de música extensível)

Família de formatos de arquivo relacionados à música criados e administrados pela Associação do Fabricante MIDI; projetado para conter vários tipos de arquivos em um único arquivo, como arquivos de dados MIDI (.MID), arquivos de instrumentos (.DLS) e arquivos de áudio de onda (.WAV).

O que é um arquivo .XMF? (Arquivo de imagem do disco gamejack)

Arquivo de imagem de disco originalmente criado pelo GameJack, um programa usado para fazer backup de CDs e DVDs de jogos; semelhante a um montado com .

Mais Informações

GameJack, um produto alemão, foi substituído pela Engelmann Media 5-Star GameCopy. O formato XMF agora é suportado por programas adicionais de software de imagem de disco, como o HH Software Virtual CD.
