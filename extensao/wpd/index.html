O que é um arquivo .wpd? (Documento de processamento de texto 602text)

Documento de texto criado com o 602Text, um programa de processamento de palavras mais antigo para Windows incluído no 602PC SUITE; contém informações de layout de texto e página formatadas; salvo em um formato projetado para ser compatível com o Microsoft Word.

O que é um arquivo .wpd? (Documento WordPerfect)

Um arquivo WPD é um documento de texto criado pelo Corel WordPerfect, um processador de texto popular. Pode conter textos, tabelas, objetos desenhados e imagens formatados. Os arquivos WPD são armazenados em um formato Corel proprietário, mas podem ser exportados para outros formatos de dentro do WordPerfect.

Mais Informações

Captura de tela de um arquivo wpd no Corel WordPerfect Office X8

Corel descontinuou as versões DOS e Macintosh do WordPerfect em 1997. No entanto, a empresa ainda desenvolve a versão Windows e a inclui como o aplicativo principal dentro do pacote WordPerfect Office.

Os arquivos WPD são usados principalmente pelos usuários do WordPerfect, mas ainda podem ser abertos usando outros processadores de texto. O Microsoft Word pode abrir a maioria das versões de arquivos WPD, enquanto o LibreOffice Writer e o NeoOffice também suportam arquivos WPD.
