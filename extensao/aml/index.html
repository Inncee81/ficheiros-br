O que é um arquivo .aml? (Linguagem de marcação abstrata)

Arquivo usado pelo Gem, um programa associado ao Ruby que cria uma linguagem de marcação XHTML organizada para criar páginas HTML estáticas.

Mais Informações

Toda vez que você salva seu arquivo AML, um arquivo HTML é criado com o mesmo nome. Um exemplo seria quando test.aml é salvo, test.html é criada. Alguns recursos dos arquivos HTML de saída são claramente tabulados linguagem para evitar erros e variáveis simples que são fáceis de referenciar e definir.

NOTA: A linguagem de marcação abstrata de gem requer RubyGems (com Ruby 200+) para ser instalada.

O que é um arquivo .aml? (Arquivo de linguagem da máquina ACPI)

Arquivo do sistema escrito no ACPI (Advanced Configuration and Power Interface) Machine Language (AML), uma linguagem de baixo nível usada para configurar propriedades de hardware; muitas vezes usada para inicializar Mac OS X em sistemas de hardware não-Apple ("Hackintosh"), como placas-mãe P55; também usado para permitir maior compatibilidade de hardware e melhorar o desempenho do sistema.

Mais Informações

Um arquivo AML contém um DSDT (Tabela de Descrição do Sistema Diferenciado), que armazena instruções que notificam o SO sobre determinadas configurações de hardware. Para Mac sistemas OS X, o arquivo AML deve ser colocado no diretório raiz para ser reconhecido.

Vários arquivos AML pré-fabricados para Mac OS X estão disponíveis no tonymacx86. Esses arquivos AML podem ser editados com personalizações usando o DSDTSE, que é desenvolvido pela equipe EVOSX86. Os arquivos AML não são editados diretamente, mas são primeiro descompilados em arquivos "dsl", editados e, em seguida, compilados de volta em arquivos "aml".

NOTA: Os arquivos AML contêm instruções no nível acpi, que é entre o BIOS e o sistema operacional.

O que é um arquivo .aml? (Arquivo de linguagem de macro arc)

Script usado pela ArcInfo Workstation, um aplicativo GIS que foi o antecessor do ArcGIS Desktop da ESRI; escrito na Linguagem Arc Macro (AML), que é uma linguagem interpretada que fornece facilidades para criar aplicativos ArcInfo; pode incluir comandos para executar instruções, gerar componentes de interface do usuário e manipular dados de mapa; salvo em um formato de texto ASCII.

Mais Informações

O que é um arquivo .aml? (Arquivo de linguagem de marcação de automação)

Um arquivo AML é um arquivo de dados Automation Markup Language (AutomationML ou AML). Baseia-se no formato .XML e usado para armazenar e trocar informações de engenharia de plantas. Os arquivos AML são usados em uma variedade de disciplinas modernas de ferramentas de engenharia, como engenharia mecânica de plantas, controle de robôs e controladores lógicos programáveis.

Mais Informações

O formato AutomationML descreve os componentes de engenharia da planta como objetos com seus diferentes aspectos. Os objetos podem incluir uma célula de fabricação inteira com muitas informações descritivas ou informações simplistas sobre um parafuso ou braço robô. Cada objeto normalmente inclui informações sobre a topologia, geometria, cinemática e lógica do objeto.

NOTA: Os arquivos AML podem ser criados, abertos e editados pelo AutomationML Editor.

O que é um arquivo .aml? (Arquivo de linguagem de marcação de assistência da Microsoft)

XML arquivo formatado no Microsoft Assistance Markup Language (MAML), um idioma usado para criar documentação de ajuda Windows; especifica estrutura de documentos de ajuda, estilos e conteúdo; usado para criar páginas de código-fonte e ajuda de aplicativos.

Mais Informações

Arquivos MAML podem ser criados usando o Sandcastle, um conjunto de esquemas .XSL e executáveis do programa. Eles também podem ser criados usando o DocProject, uma ferramenta de criação do Microsoft Visual Studio.

NOTA: O formato MAML foi desenvolvido como uma alternativa ao formato de arquivo .CHM.
