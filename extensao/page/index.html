O que é um arquivo .page? (Página de Web HybridJava)

Página web dinâmica que contém código HTML e Java; semelhante a uma página .JSP, mas mescla a sintaxe formal HTML e Java no mesmo código; compilada em um servidor Web pelo compilador HybridJava, que envia o código resultante para o navegador Web do usuário.

Mais Informações

OS arquivos PAGE devem ser colocados no subdiretório \pages no servidor Web. O HybridJava Compiler foi descontinuado.

O que é um arquivo .page? (Arquivo da página da nota S)

Um arquivo PAGE é um arquivo de dados usado pelo S Note, um aplicativo avançado de anotações desenvolvido pela Samsung para uso com dispositivos móveis Samsung, como Galaxy S e Galaxy Note. Ele contém qualquer texto que apareça na nota .SPD, juntamente com outras informações criptografadas sobre a nota, como quando foi atualizada pela última vez.

Mais Informações

OS arquivos PAGE não devem ser abertos diretamente. Em vez disso, eles são referenciados por um aplicativo, como o Samsung S Note em Android ou Windows, ao abrir um arquivo SPD. No entanto, você pode abrir um arquivo PAGE em um editor de texto para visualizar o texto, juntamente com outras informações criptografadas, que aparece na nota SPD.

PAGE arquivos estão localizados em Notas SPD, que são compactadas com compactação zip. Você pode extrair arquivos PAGE dos arquivos SPD substituindo a extensão "spd" em um arquivo SPD com "zip" para criar um arquivo .ZIP. Em seguida, você pode extrair o conteúdo, que inclui um arquivo PAGE, .DAT e .JPG, com um programa de descompressão Zip, como Corel WinZip ou Apple Archive Utility.
