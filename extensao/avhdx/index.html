O que é um arquivo .AVHDX? (Arquivo de disco rígido virtual automático)

Um arquivo AVHDX é um ponto de verificação de imagem em disco usado pelo Windows Server e sua tecnologia Microsoft Hyper-V, que inicializa máquinas virtuais (VM) usando imagens de disco. Ele contém a configuração de estado, dados e hardware de um VM no momento da criação do arquivo. Os arquivos AVHDX são criados para salvar um estado específico de um disco rígido virtual .VHDX.

Mais Informações

ARQUIVOS AVHDX são conhecidos como discos de referência porque usam outros discos para criar uma cadeia de discos diferente. Cada arquivo AVHDX contém um ponto no tempo, que é usado para compor uma linha do tempo com os outros arquivos AVHDX na cadeia. Esses arquivos permitem que uma máquina virtual retorne a um estado anterior, o que permite criar uma condição específica para solucionar problemas.

Você pode se der em um problema onde um ponto de verificação não pode ser aplicado em Windows Server para reverter para um estado de salvamento. Isso pode ser devido a uma cadeia de controle quebrada ou arquivo corrupto. Você pode mesclar manualmente os arquivos AVHDX para corrigir esse problema, embora, esta deve ser sua última opção. Uma opção é fundir pontos de verificação criando um novo ponto de verificação e, em seguida, diluindo-o, o que força o Hyper-V a classificar automaticamente seus pontos de verificação. Você também pode exportar seu VM e importá-lo de volta para o Hyper-V Manager. Antes de tentar mesclar seus arquivos, você deve criar um backup.

O Microsoft Hyper-V permite criar e gerenciar um ambiente de computação virtualizado usando a tecnologia de virtualização incorporada ao Windows Server. O Hyper-V vem com ferramentas de gerenciamento e quaisquer componentes necessários.
