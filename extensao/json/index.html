O que é um arquivo .JSON? (Arquivo de notação de objeto JavaScript)

Um arquivo JSON é um arquivo que armazena estruturas de dados simples e objetos no formato JSON (JavaScript Object Notation, notação de objetos javascript), que é um formato padrão de intercâmbio de dados. É usado principalmente para transmitir dados entre um aplicativo web e um servidor. Os arquivos JSON são leves, baseados em texto, lêveis em humanos e podem ser editados usando um editor de texto.

Mais Informações

O formato JSON foi originalmente baseado em um subconjunto de JavaScript, mas é considerado um formato independente do idioma, sendo suportado por muitas APIs de programação diferentes. JSON é comumente usado na programação de aplicativos Ajax Web. Está se tornando cada vez mais popular como uma alternativa ao XML.

Embora muitos aplicativos usem o JSON para intercâmbio de dados, eles podem realmente não salvar .json arquivos no disco rígido, uma vez que o intercâmbio de dados ocorre entre computadores conectados à Internet. No entanto, alguns aplicativos permitem que os usuários salvem .json arquivos. Um exemplo é o Google+, que usa arquivos JSON para salvar dados do Perfil. Após o login, você pode escolher a página "Liberação de dados" e selecionar "Baixe seus dados de perfil"

O Mozilla Firefox salva backups de marcadores usando arquivos JSON. Os arquivos são salvos no diretório de perfil do usuário do Firefox dentro de uma pasta chamada bookmarkbackups. Você também pode fazer backups manuais dos marcadores do Firefox e importá-los em uma cópia diferente do Firefox. Para importar, abra a Biblioteca de marcadores, clique no botão de importação/exportação na barra superior, escolha Restore → Choose File... e selecione seu arquivo de backup JSON.
