O que é um arquivo .hbx? (Arquivo codificado BinHex)

Arquivo de texto binário para Hexadecimal criado para Mac computadores; contém dados compactados e codificados que estavam originalmente em um formato binário de 8 bits, mas agora está em um formato de texto de 7 bits; normalmente usado para transferir imagens binárias, documentos e dados multimídia usando um formato textual em anexos de e-mail.

Mais Informações

A extensão "hbx" é uma alternativa às extensões .BHX ou .HQX mais usadas.
