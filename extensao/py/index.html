O que é um arquivo .py? (Python Script)

Um arquivo PY é um arquivo de programa ou script escrito em Python, uma linguagem de programação interpretada orientada a objetos. Ele pode ser criado e editado com um editor de texto, mas requer um intérprete Python para ser executado. Os arquivos PY são frequentemente usados para programar servidores web e outros sistemas de computador administrativos.

Python foi projetado para ser fácil de ler e simples de implementar. É de código aberto e usado para desenvolver uma grande variedade de aplicações gratuitas e comerciais, como Bazar, Liquidificador, Pilãos e Panda3D.

Você pode abrir e editar scripts PY com um editor de texto, como Microsoft Notepad ou Apple TextEdit. No entanto, você também pode abrir e editar arquivos PY com editores de código-fonte e IDEs que fornecem recursos de edição de linguagem Python. MacroMates TextMate e Sublime Text são editores de código-fonte úteis, bem como JetBrains PyCharm, que é um IDE Python para desenvolvedores profissionais.

NOTA: A linguagem de programação foi originalmente criada no final dos anos 1980 por Guido van Rossum com versões subsequentes lançadas em 2000 (Python 20) e 2008 (Python 30).
