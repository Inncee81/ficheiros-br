O que é um arquivo .md5? (Arquivo MD5 Checksum)

Um arquivo MD5 é um arquivo de cheques usado para verificar a integridade de um disco, imagem de disco ou um arquivo baixado. Ele armazena um cheque, que é um valor criado a partir de um algoritmo baseado no número de bits no arquivo. Os arquivos MD5 às vezes são gerados quando uma imagem de disco é criada.

Mais Informações

MD5 arquivos funcionam de forma semelhante às impressões digitais humanas. Eles são únicos e são usados para verificar a identidade. Isso é especialmente importante por razões de segurança no caso de um arquivo ser alterado para fins maliciosos ou corrompido de um erro de disco ou transferência de arquivo incompleta.

NOTA: Os cheques padrão MD5 têm 32 caracteres de comprimento, o que requer 16 bytes ou 128 bits.
