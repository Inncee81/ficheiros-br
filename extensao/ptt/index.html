O que é um arquivo .ptt? (Modelo de mapa do MapPoint)

Modelo para criar um novo arquivo de mapa (.PTM) no Microsoft MapPoint, um programa de mapeamento de negócios; pode incluir dados de viagens de negócios e território de vendas, bem como informações de documentos do Microsoft Office.

O que é um arquivo .ptt? (Modelo de sessão de ferramentas pro)

Modelo de sessão criado e usado pelo Pro Tools, um programa de edição e produção de áudio; contém dados de sessão, que podem incluir faixas de instrumentos de áudio e software; também inclui parâmetros de sessão, como Taxa de Amostra e Profundidade de Bit.

Mais Informações

Os modelos Pro Tools permitem criar um arranjo personalizado que pode ser importado para a sua sessão para definir configurações e efeitos. Você pode importar um modelo selecionando File → Import → Session Data....

NOTA: O formato PTT foi substituído pelo mais novo formato de modelo .PTXT na versão 10 do Pro Tools.
