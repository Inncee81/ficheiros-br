O que é um arquivo .scn? (Arquivo de cenário da Era da Mitologia)

Arquivo de cenário usado pela Age of Mythology, um jogo de estratégia em tempo real onde os jogadores participam de campanhas mitológicas controlando exércitos e unidades; contém um mapa de campanha com terreno, unidades, locais de partida dos jogadores e condições de vitória.

Mais Informações

SCN armazena conteúdo de cenário pré-instalado e personalizado. Eles são encontrados na pasta "scenario" dentro das instalações da Age of Mythology. Cenários personalizados podem ser construídos dentro do editor de cenários da Era da Mitologia.

A Microsoft Age of Empires também usa arquivos SCN.

O que é um arquivo .scn? (Arquivo de cena binária do motor godot)

Um arquivo SCN contém uma cena criada pela Godot Engine, um mecanismo de jogo de código aberto e gratuito usado para criar jogos 2D e 3D. Ele armazena dados binários sobre uma única árvore de cena dentro do Godot Engine, que inclui caminhos para quaisquer recursos externos, e informações sobre recursos internos, conexões e nódulos.

Mais Informações

Para abrir uma cena em um arquivo SCN em Godot Engine, selecione Scene → Open Scene... ou Open Recent. Para criar uma cena e salvá-la como um arquivo SCN no Godot Engine, selecione Scene → New Scene ou New Inherited Scene.... Em seguida, selecione Scene → Save Scene ou Save Scene As....

O que é um arquivo .scn? (Arquivo de cenário OpenTTD)

Arquivo de jogo usado pelo OpenTTD, o projeto de follow-on de código aberto para Transport Tycoon; salva um cenário de jogo, que define as condições de partida para um mapa jogado pelo usuário; pode incluir a paisagem, edifícios, cidades, trilhos de trem e outros recursos.

Mais Informações

NOTA: Arquivos SCN também são usados pelo Magnata do Transporte original.

O que é um arquivo .scn? (Arquivo de dados de varredura TD)

Um arquivo SCN é um arquivo de dados criado pelo TD Scan, um conjunto de software usado para visualizar, analisar e criar relatórios de dados ultrassônicos. Contém uma varredura de dados ultrassônicos de componentes industriais, como soldas de tubulação, forjas e fundições, e soldas de vasos de pressão.

Mais Informações

TD Scan é usado para monitorar e analisar o desempenho de instalações industriais, como usinas nucleares. Ele oferece recursos de coleta de dados em tempo real, rastreamento de vídeo e de difração de tempo de voo (ToFD), Pulse Echo e mapeamento de corrosão. O programa armazena varreduras no arquivo SCN, que é usado para varreduras completas ou para varreduras divididas que podem ser unidas quando o tamanho dos dados se torna muito grande.

NOTA: Para abrir um arquivo SCN no TD Scan, selecione File → Open Scan File.

O que é um arquivo .scn? (Imagem treePaint)

Arquivo de imagem criado pelo TreePaint Plus, um programa usado para criar desenhos de impressão para fotoengraving, processos de tela de seda e outros tipos de meios de impressão; suporta 256 cores, 65.536 cores ou formatos de cores verdadeiras de 24 bits; pode conter informações de separação colorway e tela de seda; usado para soluções têxteis e de moda.
