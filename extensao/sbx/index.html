O que é um arquivo .SBX? (Arquivo Adobe Illustrator Tsume)

Arquivo criado pelo Illustrator, um programa de edição de gráficos vetoriais; contém dados de kerning proporcionais para caracteres chineses, japoneses e coreanos (CJK); acompanha fontes CJK que usam espaçamento proporcional incorporado; não devem ser abertos manualmente).

Mais Informações

NOTA: A maioria dos caracteres CJK são monoespaciados (cada caractere tem o mesmo comprimento horizontal e vertical). No entanto, ao usar a opção Tsume, o monoespaçamento padrão mudará para uma largura e altura proporcionais para o texto.

O que é um arquivo .SBX? (Arquivo de índice espacial ESRI)

Arquivo usado pelo software de mapeamento ESRI ArcGIS Desktop GIS; contém um índice espacial para um arquivo de forma .SHP; usado para otimizar consultas e reduzir o tempo total gasto acessando informações multidimensionais de forma de objeto.

Mais Informações

Os arquivos SBX são opcionais ao especificar um arquivo de forma. No entanto, os arquivos de forma sem índices espaciais podem sofrer de tempos de carregamento e renderização lentos.

O que é um arquivo .SBX? (Arquivo de projeto ScratchX)

Um arquivo SBX é um projeto criado pela ScratchX, uma plataforma de desenvolvimento usada para testar extensões experimentais do JavaScript Scratch. Ele contém um projeto de extensão Scratch desenvolvido pelos usuários do Scratch para permitir que a linguagem de programação Scratch interaja com hardware, como o vácuo iRobot ou robôs LEGO EV3, ou fontes de informações da Web, como Amazon Text to Speech ou Google Translate.

Mais Informações

Scratch é uma linguagem de programação fácil de aprender que ajuda os jovens a desenvolver jogos, animações e histórias interativas. Existem várias versões de Scratch, incluindo arquivos scratch 30 (.SB3), arquivos Scratch 20 (.SB2) e o formato original Scratch (.SB arquivos). O ScratchX permite que os usuários criem, modifiquem e compartilhem extensões experimentais que permitem ao Scratch se conectar e interagir com fontes externas de hardware ou informações da Web.

Para abrir um arquivo SBX no editor online do MIT ScratchX:

  1. Abra o editor do projeto Scratch na página do MIT ScratchX.
  2. Selecione File → Load Project na janela do navegador.

Para criar um arquivo SBX no editor online do MIT ScratchX:

  1. Abra o editor do projeto Scratch na página do MIT ScratchX.
  2. Crie seu projeto e selecione File → Save Project.
  3. Nomeie seu arquivo, escolha o local de salvamento e clique em Salve.

NOTA: ScratchX é um projeto do Grupo de Jardim de Infância ao longo da vida no Massachusetts Institute of Technology (MIT) Media Lab.

O que é um arquivo .SBX? (Arquivo de contêiner SeqBox)

Um arquivo SBX é um arquivo de contêiner SeqBox (Sequenced Box) criado pela SBXEnc, um programa usado para proteger e fazer backup dos arquivos de um usuário em caso de falhas no sistema, perda de informações de partição ou quaisquer outros problemas de disco. Ele contém um arquivo que foi codificado pelo programa SBXEnc. Os arquivos SBX armazenam informações em uma coleção de blocos com cada bloco que consiste em um cabeçalho, identificador de arquivo exclusivo, checkum, número de sequência de blocos e versão.

Mais Informações

O formato SeqBox foi projetado para fornecer uma solução segura de arquivamento para proteger e armazenar os arquivos de um usuário. O programa SBXEnc é usado para codificar um arquivo de dados em um contêiner SBX e o programa SBXDec é usado para decodificar o arquivo SBX de volta para o arquivo original.
