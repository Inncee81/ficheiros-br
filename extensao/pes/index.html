O que é um arquivo .pes? (Formato de Bordado irmão)

CAM (fabricação auxiliada por computador) que contém instruções para criar bordados com uma máquina de costura; inclui códigos como "stop," "jump," e "trim;" também inclui as cores para o design da paleta de cores Brother suportada; pode ser aberto e visto por vários programas de costura.

Mais Informações

PES fles são criados pelo software PE-Design incluído com máquinas de costura Brother, mas também podem ser usados com máquinas de bordados domésticos Babylock e Bernina.

O que é um arquivo .pes? (Arquivo da folha do Editor de Física)

Arquivo de dados criado pelo PhysicsEditor, um programa usado para editar propriedades físicas para objetos; contém um ou mais sprites, bem como a imagem traced e propriedades físicas para cada imagem; pode ser publicado para uso em videogames baseados em física.
