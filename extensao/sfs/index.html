O que é um arquivo .sfs? (Arquivo de salvamento do programa de espaço Kerbal)

Arquivo criado pelo Kerbal Space Program (KSP), um jogo onde os jogadores constroem e gerenciam um programa espacial; salva o estado do jogo como um arquivo chamado persistent.sfs; permite que o jogo seja retomado.

Mais Informações

Kerbal Space Program permite que os usuários criem "quicksaves" de jogos e voltem aos estados anteriores do jogo no caso do foguete falhar. Quando o jogo cria a rapidez, ele armazena o conteúdo no arquivo quicksave.sfs.

O que é um arquivo .sfs? (Arquivo de amostra SFX)

Arquivo criado pelo sfxr, um programa usado para fazer efeitos sonoros; armazena configurações sonoras como tempo de ataque, tempo de sustentação, frequências, cortes e filtros; pode ser usado para fazer efeitos sonoros de videogame.

Mais Informações

Os arquivos SFS podem ser exportados para .WAV arquivos no software.

NOTA: Existe uma versão Mac do sfxr que é chamada de cfxr ("Cacau sfxr"). No entanto, o cfxr não pode abrir arquivos SFS. Ele usa seus próprios arquivos de .CFXR formatados XML.

O que é um arquivo .sfs? (Arquivo de arquivo squashfs)

Formato de arquivo somente leitura desenvolvido para sistemas operacionais baseados em Linux; muitas vezes usado para criar arquivos de instaladores e distribuições de software; tipicamente compactado usando .GZIP compactação.

Mais Informações

Os arquivos SFS também são usados para instalar atualizações em dispositivos GPS baseados em Navigon.

NOTA: Alguns usuários relataram que o 7-zip é incapaz de extrair arquivos SFS. Se você estiver tendo problemas, desinstale 7-zip e instale 7-zip 925 alfa. Esta versão abrirá arquivos SFS.
