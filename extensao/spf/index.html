O que é um arquivo .spf? (ShadowProtect Arquivo de backup completo)

Arquivo de backup criado pelo ShadowProtect, um programa usado para gerenciamento de backup e recuperação de desastres; contém arquivos de backup compactados; usado para armazenar todos os arquivos de backup em um único arquivo, em vez de usar um arquivo estendido ou um backup incremental.

Mais Informações

ShadowProtect está disponível nas seguintes edições:

  • Desktop
  • Server
  • Small Business Server (SBS)
  • IT
NOTA: Os backups incrementais do ShadowProtect são armazenados em arquivos .SPI.

O que é um arquivo .spf? (Arquivo favoritos do slingplayer)

Arquivo criado pelo Slingplayer, um programa que faz interface com um dispositivo Slingbox DVR para gravar e reproduzir programas de TV em um computador; contém uma coleção de programas de TV e canais favoritos; usado para armazenar atalhos para programas de TV favoritos.
