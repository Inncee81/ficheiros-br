O que é um arquivo .dcm? (Imagem DICOM)

Um arquivo DCM é um arquivo de imagem salvo no formato de imagem Digital Imaging and Communications in Medicine (DICOM). Armazena uma imagem médica, como uma tomografia computadorizada ou ultrassom. Os arquivos DCM também podem incluir informações do paciente para emparelhar a imagem com o paciente.

Mais Informações

O formato DICOM foi criado pela Associação Nacional dos Fabricantes de Elétricos (NEMA) como padrão para distribuição e visualização de imagens médicas, como ressonâncias magnéticas, tomografias e imagens de ultrassom. Você pode abrir arquivos DICOM com uma variedade de programas para Windows, macOS e Linux, como XnViewMP, GIMP e MeVisLab.

NOTA: Arquivos DCM também podem ser salvos como .DICOM arquivos.

O que é um arquivo .dcm? (Arquivo de catálogo do DiskCatalogMaker)

Um arquivo DCM é um arquivo salvo no formato de catálogo usado pelo DiskCatalogMaker, um programa de catalogação e organização de arquivos para o Mac. Ele contém um banco de dados de arquivos armazenados em vários discos que podem ser pesquisados por nome de arquivo e outros atributos. Os arquivos DCM também podem ser salvos à medida que .DCMF arquivos, que são mais comuns.

Mais Informações

Se você receber ou encontrar um arquivo DCM DiskCatalogMaker, você só poderá abri-lo no DiskCatalogMaker. Não há outro programa disponível que suporte o formato.

Os arquivos DCM são semelhantes aos arquivos .DCMD, mas os arquivos DCMD suportam imagens em miniatura. Você pode converter um arquivo DCM em um arquivo DCMD abrindo o arquivo no DiskCatalogMaker, selecionando File → Save As... e, em seguida, optando por salvar como um arquivo DCMD.
