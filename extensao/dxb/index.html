O que é um arquivo .DXB? (Binário de troca de desenho)

Formato de imagem vetorial usado pela Autodesk AutoCAD; armazena desenhos 3D CAD criados pela AutoCAD, como projetos de construção e paisagismo.

Mais Informações

DXB é a versão binária de um arquivo .DXF, que é baseado em texto. Os arquivos DXB são menores e carregam mais rápido que os arquivos DXF, mas não são tão compatíveis com outros programas como os arquivos DXF são.

O que é um arquivo .DXB? (Arquivo Duxbury Braille)

Arquivo criado por Duxbury Braille Translator, um tradutor em braile; contém braille que pode ser traduzido para Impressão; pode ser visualizado em diferentes fontes, como braile, simbraille ou fonte de impressão; pode conter tabelas, códigos e outros elementos de documento.

Mais Informações

Para criar um arquivo DXB com o Duxbury Braille Translator, selecione File → Save or Save as..., nomeie o arquivo, escolha o local de salvamento e clique em Save.

Para abrir um arquivo DXB com o Duxbury Braille Translator, selecione File → Open..., navegue até o seu arquivo e clique em Abra.

NOTA: Semelhante à extensão de arquivo .DXP que contém impressão em vez de braile. Duxbury MegaDots foi descontinuado.
