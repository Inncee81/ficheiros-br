O que é um arquivo .mif? (Arquivo de formato de intercâmbio framemaker)

Arquivo criado pelo FrameMaker, um programa usado para gerar e manipular documentos grandes e estruturados; salvo em texto ASCII simples e armazenando todos os dados originalmente de um arquivo .FM ou .BOOK; descreve as informações de texto, gráficos e layout do documento.

Mais Informações

Os arquivos MIF são usados para exportar documentos do FrameMaker em um formato que pode ser manipulado por aplicativos e scripts de terceiros. Os arquivos MIF exportados podem ser convertidos de volta aos seus arquivos FM ou BOOK originais sem perder nenhuma informação.

O que é um arquivo .mif? (Formato de informações de gerenciamento)

Define arquiteturas personalizadas do sistema que podem ser testadas usando o Microsoft Systems Management Server; também conhecido como arquivo IDMIF.

O que é um arquivo .mif? (Arquivo de formato de intercâmbio MapInfo)

Formato de arquivo usado pelo software de mapeamento e análise geográfica do MapInfo; salva uma visualização de mapa em um formato que pode ser reconhecido por aplicativos de terceiros; usado como formato de troca entre aplicativos GIS.

Mais Informações

MapInfo é usado para uma geração de visualizações de uma variedade de conjuntos de dados, incluindo estatísticas demográficas, informações de engenharia civil (estradas, rede de água, etc) e dados de inteligência de negócios.

NOTA: Os arquivos MIF às vezes são acompanhados por um arquivo .MID (dados MapInfo) que contém dados espaciais adicionais.
