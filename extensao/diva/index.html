O que é um arquivo .diva? (Arquivo de música do Projeto DIVA)

Arquivo de jogo usado pelo Project DIVA, um jogo de ritmo japonês criado pela SEGA; salva informações sobre músicas; inclui uma referência à música .MP3 arquivo e os tempos para os quais os eventos devem ser acionados para marcar pontos no jogo.

Mais Informações

Pode haver vários arquivos DIVA para cada música que são usados para os diferentes níveis de dificuldade. Por exemplo, pode haver modos fáceis, normais e difíceis para cada música.

NOTA: Os arquivos DIVA são salvos no diretório \songs\ da instalação do Projeto DIVA.
