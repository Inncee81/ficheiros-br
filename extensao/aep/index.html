O que é um arquivo .aep? (Projeto Activ E-Book)

Livro eletrônico, ou e-book, criado com o compilador Activ E-Book; inclui páginas que podem ser lidas como um livro físico; podem incluir texto e imagens, bem como animações, navegação de hipertexto (links), JavaScript e applets Java.

Mais Informações

Um projeto Activ E-Book pode ser salvo como um arquivo executável padrão (.EXE), que será executado em qualquer computador com Windows 95 ou mais.

O que é um arquivo .aep? (Arquivo criptografado do pacote de criptografia avançada)

Arquivo criptografado pelo Advanced Encryption Package (AEP), um programa usado para proteger arquivos para armazenamento e transferência; pode ser protegido por senha e protegido usando um dos vários métodos de criptografia fornecidos pelo software.

O que é um arquivo .aep? (Projeto After Effects)

Um arquivo AEP é um projeto de edição de vídeo criado com o Adobe After Effects. Ele contém uma composição de vídeo que inclui áudio, imagens e clipes de vídeo, uma linha do tempo e vários efeitos que podem ser aplicados ao filme. Os arquivos AEP são usados para criar produções de vídeo de qualidade profissional.

Mais Informações

Captura de tela de um arquivo aep no Adobe After Effects 2020

O arquivo AEP é um dos dois tipos de arquivos usados para salvar projetos de edição de vídeo em After Effects. O formato principal é o arquivo AEP, que é salvo em formato binário. Os outros projetos de formato podem ser salvos é o arquivo .AEPX, que é um arquivo de projeto XML baseado em texto.

Após os projetos de efeitos podem ser salvos como modelos. Um tipo de arquivo usado para armazenar modelos é o projeto binário .AET e o projeto XML .AETX.

Para criar um projeto After Effects, selecione File → New → New Project clique em File → Save ou Save As → Save As....

Para abrir um projeto After Effects, selecione File → Open Project....
