O que é um arquivo .shd? (Windows imprimir arquivo sombra de trabalho)

Arquivo criado por Windows quando um usuário envia um trabalho de impressão; armazena informações de cabeçalho para um trabalho de impressão, mas não armazena os dados reais do documento de bobina; permite que as configurações de trabalho de impressão sejam "sombreadas" para que o trabalho possa ser reapresentado novamente se houver uma interrupção.

Mais Informações

Os arquivos SHD são criados ao lado de arquivos .SPL, que contêm os dados de impressão em bobe. Ambos os arquivos SHD e SPL são escritos para o seguinte diretório quando um trabalho de impressão é submetido: C:\Windows\System32\spool\PRINTERS\.
