O que é um arquivo .layoutdesigner? (Arquivo de designer de layout quickbooks)

Arquivo criado pela QuickBooks, um aplicativo de gerenciamento financeiro; contém um modelo personalizado de uma fatura, recibo de venda, estimativa, memorando de crédito, declaração ou pedido de compra que pode ser usado com outro arquivo da empresa QuickBooks; criado pela ferramenta Layout Designer em QuickBooks.

Mais Informações

Para criar um arquivo LAYOUTDESIGNER:

  1. Designer de layout aberto. Na parte superior do formulário escolha um modelo existente, selecione New Template... ou Edit Current Template....
  2. Personalize seu modelo. Você pode escolher quais tipos de informações aparecem no formulário e formatar o modelo adicionando um logotipo, alterando fontes, adicionando uma imagem de fundo, etc.
  3. >Uma vez que você terminar de editar o modelo, selecione File → Export e escolha o formato para o qual deseja exportar o modelo para: .TIFF, .PDF ou o arquivo LAYOUTDESIGNER.
