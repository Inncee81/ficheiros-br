O que é um arquivo .RAW? (Dolphin Wii e Gamecube Emulator Game Save File)

Arquivo de salvamento de jogos para arquivos do Cartão de Memória Wii e Gamecube; contém dados de salvamento de jogos.

Mais Informações

Dolphin é um emulador de código aberto usado para rodar jogos nintendo wii e gamecube nos sistemas operacionais Windows, Linux e Mac OSX.

O emulador de consoles Dolphin Wii e GameCube usa o tipo de arquivo RAW como um arquivo de salvamento para arquivos Wii e Gamecube Memory Card. Embora o emulador dolphin não possa abrir o tipo de arquivo RAW, ele pode usá-lo como uma salvação para jogos.

O que é um arquivo .RAW? (Dados de áudio brutos)

O PCM não compactado (modulação de código de pulso) assinou o arquivo de dados de áudio; contém dados de áudio bruto armazenados em formato binário; não contém nenhuma informação de cabeçalho ou usa um codec de áudio.

O que é um arquivo .RAW? (Arquivo de dados de imagem bruta)

Um arquivo RAW é uma imagem gerada por câmeras digitais como câmeras Panasonic, Leica e Casio. Contém dados de imagem bruta e não comprimidos que podem ser ajustados para exposição e balanço de branco usando um software que suporta o formato. Os arquivos RAW são usados para armazenar dados de imagem não-alteradas como capturados pelo CCD de uma câmera digital.

Mais Informações

Captura de tela de um arquivo bruto no Adobe Photoshop CC 2019

Os arquivos RAW são usados para produzir imagens de maior qualidade, uma vez que o formato armazena dados capturados a partir do sensor de uma câmera em um formato não comprimido. Os arquivos RAW não perdem dados como os arquivos JPEG fazem quando são compactados. No entanto, os arquivos RAW podem ser muito grandes em tamanho porque não são compactados. Esta é a troca entre imagens de baixa qualidade, menores e imagens maiores e de maior qualidade.

NOTA: Arquivos RAW também podem ser criados por software de geração de imagens, embora isso não seja tão comum.
