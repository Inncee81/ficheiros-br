O que é um arquivo .xpr? (Gráfico de design de expressão da Microsoft)

Gráfico criado pela Microsoft Expression Graphic Design (EGD), o antecessor do Microsoft Expression Design (ED); armazena um gráfico que pode ser usado como um mockup de interface de usuário; substituído por .DESIGN arquivos no Expression Design.

Mais Informações

Expression Design faz parte do Microsoft Expression Studio. No entanto, a Microsoft descontinuou o Expression Studio em 2012.

O que é um arquivo .xpr? (Arquivo acelerador de instância de peça pro/engenheiro)

Arquivo criado pelo Pro/ENGINEER, um programa usado para o design de produtos 3D; contém uma instância de um arquivo de peça (.PRT), que define parâmetros específicos para uma parte genérica.

Mais Informações

Os arquivos XPR aceleram o carregamento de uma instância de peça, uma vez que o Pro/ENGINEER não precisa carregar todas as informações da peça de geometria "árvore genealógica" e "genérico" primeiramente.

NOTA: Os arquivos XPR podem ser carregados pelo Adobe Acrobat Pro Extended, que inclui suporte para vários tipos de gráficos 3D. Eles podem não ser suportados pela versão padrão do Acrobat Pro.

O que é um arquivo .xpr? (SyncRO Soft oXygen XML Editor Project)

XML project criado com oXygen XML Editor (escrito "<oXygen/> Editor XML"), um programa usado para criar e editar .XML arquivos e aplicativos baseados em XML; contém referências a arquivos XML, configurações de espaço de trabalho do usuário, informações de conexão do sistema de controle de versão e outras propriedades do projeto.

Mais Informações

Os arquivos XPR são usados para desenvolver e empacotar programas baseados em XML, como Web, banco de dados, aplicativos de troca de dados. Durante o desenvolvimento, os projetos oXygen podem ser usados para validação XML, bem como depuração de transformações .XSLT e consultas XQuery.
