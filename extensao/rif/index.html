O que é um arquivo .rif? (Arquivo de imagem do pintor Raster)

Um arquivo RIF é uma imagem rasterada criada por Corel Painter (ex-MetaCreations Painter), um programa usado para pintura digital. Contém uma imagem que pode incluir pinceladas, esboços de lápis, gráficos de mangueira de imagem e vários outros elementos pintados. Os arquivos RIF também podem ser vistos com a extensão de arquivo .RIFF menos comum.

Captura de tela de um arquivo rif em Corel Painter 2020

Corel Painter é usado por artistas casuais e profissionais para criar pinturas digitais. O programa inclui vários tipos diferentes de pincéis, telas e ferramentas de desenho e também se integra com displays de caneta e tabletes de desenho para desenho à mão livre.

O arquivo RIF é o tipo de arquivo principal usado pelo Painter para armazenar as imagens criadas no Painter. Os arquivos RIF retêm todas as informações do projeto, o que normalmente as torna maiores do que os formatos de imagem rasteres, como .GIF ou .JPG arquivos. Além disso, os arquivos RIF são salvos separados da tela, o que permite ao pintor experimentar diferentes texturas de tela mais tarde.
