O que é um arquivo .sbf? (Arquivo do sistema Android)

Arquivo de sistema usado pelo Google Android dispositivos, como celulares Motorola DROID 2 e DROID X; contém todos os dados do telefone como foi originalmente configurado a partir da fábrica; permite que o telefone seja definido para as configurações originais de fábrica.

Mais Informações

NOTA: O programa RSD Lite pode ser usado para "flash" um arquivo SBF para um telefone Motorola DROID 2 ou DROID X. Geralmente, os usuários não devem restaurar manualmente arquivos SBF, uma vez que o uso inadequado dos arquivos pode causar o mau funcionamento do telefone.

O que é um arquivo .sbf? (Arquivo Binário Septentrio)

Um arquivo SBF é um arquivo de dados GPS criado pelos receptores Septentrio GNSS, que são dispositivos eletrônicos que recebem e processam sinais de satélites globais do sistema de satélite de navegação (GNSS). Ele contém um registro de dados sobre a posição gps, tempo e velocidade de um dispositivo equipado com um receptor GNSS. Os arquivos SBF são salvos em formato binário para reduzir o tamanho do arquivo.

Mais Informações

Os receptores Septentrio GNSS são usados em uma grande variedade de indústrias, que incluem sistemas de informação geográfica, veículos aéreos científicos e não tripulados, marinha, construção, agricultura e levantamento e mapeamento. Os arquivos SBF são usados para armazenar os registros de dados de dispositivos usados nessas indústrias.

Os arquivos SBF podem ser abertos por uma variedade de ferramentas da suíte Septentrio RxTools, que incluem "SBF Analyzer" e "SBF Converter". You can convert SBF files using the "SBF Converter" para vários formatos, como ASCII, KML, GPX e RINEX. As ferramentas podem ser encontradas no subdiretório rools\binary\win32 no caminho de instalação.

O que é um arquivo .sbf? (Dados do SmartBroker)

Formato de arquivo de dados usado pelo software de rastreamento de ações SmartBroker; pode conter listas de ações, dados de tendência, manchetes de notícias, relatórios de rastreamento ou outros dados.

O que é um arquivo .sbf? (Arquivo de banco de dados superbase)

Arquivo de banco de dados usado pelo SuperBase, um programa usado para criar aplicativos de cliente/servidor de rede; armazena dados em registros, que podem conter de 1 a 999 campos; salvos em um formato binário.
