O que é um arquivo .milk? (Predefinição do plugin MilkDrop Winamp)

Armazena predefinições para o plugin de visualização MilkDrop para Winamp; usa a placa gráfica do computador para criar visualizações 3D enquanto a música está sendo reproduzida; as animações são baseadas nas batidas e frequências do som.
