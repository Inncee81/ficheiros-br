O que é um arquivo .iba? (Documento autoral do iBooks)

Livro digital criado pelo iBooks Author, programa usado para criar livros para dispositivos macOS e iOS; salva os capítulos, seções e páginas do livro, bem como as informações de layout da página; também salva o título do livro, mídia de introdução, tabela de conteúdo e glossário; usado para livros didáticos, livros de imagens, livros multimídia e muitos outros tipos de livros digitais.

Mais Informações

iBook Os documentos do autor podem ser exportados para o formato .IBOOKS para upload direto em um iPad (através do software do iTunes) ou para compartilhar com outros. Os arquivos IBA também podem ser exportados para .PDF ou .TXT (texto simples) para leitura no computador local ou em outros dispositivos.

IBA arquivos são salvos em um formato de .ZIP comprimido. Portanto, você pode abri-los manualmente com qualquer programa de descompressão zip para ver o conteúdo do livro.

Para vender um livro na Apple Book Store, você pode publicar o arquivo IBA no formato .ITMSP (pacote iTunes Store) selecionando File → Publish.... Em seguida, envie o pacote para a Apple Book Store para revisão usando o software iTunes Producer, que está previsto para download com o programa iTunes Connect (iTC) Content Provider.

NOTA: Os livros eram anteriormente conhecidos como "iBooks", mas renomeados com o lançamento do macOS Mojave e iOS 12.

O que é um arquivo .iba? (Arquivo de peças de design de roupas lectra)

ARQUIVO CAD usado pela Lectra Modaris, um aplicativo de design de roupas e moda; descreve a geometria das peças que compõem um design de roupa; salvo com um arquivo .VET que fornece a folha de "cortador deve".

Mais Informações

Arquivos VET-IBA podem incluir um design para uma camisa, vestido ou calças. As informações da peça de vestuário podem ser enviadas para uma máquina de corte de padrão que corta as peças de um rolo maior de tecido.

NOTA: A Lectra Modaris agora usa arquivos .MDL em vez de arquivos VET e IBA.
