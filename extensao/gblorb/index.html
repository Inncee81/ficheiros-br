O que é um arquivo .gblorb? (Arquivo de jogo Glulx Blorb)

Arquivo de jogo usado por intérpretes de Ficção Interativa (IF), como Spatterlight ou Zoom; contém uma história de Glulx no formato Blorb, que é um formato usado para agrupar diferentes arquivos de recursos de jogo em um único arquivo; diferentes arquivos de recursos incluem música, gráficos e informações de nível.

Mais Informações

Os arquivos GBLORB são semelhantes aos arquivos .BLORB, mas são usados apenas para jogos Glulx. Um arquivo GBLORB é um arquivo .ULX contido em um invólucro Blorb, o que lhe dá mais recursos, como armazenar arquivos de recursos. .ZBLORB arquivos, que são semelhantes aos arquivos GBLORB também, são usados por máquinas Z.
