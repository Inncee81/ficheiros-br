O que é um arquivo .spq? (Arquivo de consulta de banco de dados SPSS)

Arquivo criado pelo software de mineração de dados e análise estatística spss; salva uma consulta a um banco de dados; especifica o tipo de banco de dados ODBC (por exemplo, MS Access, dBASE, Arquivo Excel), os campos de dados para extrair e transformações que escrevem os dados para o formato desejado.

Mais Informações

Arquivos SPQ permitem que os usuários do SPSS salvem consultas para que possam ser executadas repetidamente. Isso é útil para atualizar o conjunto de dados quando os dados no banco de dados de origem mudam.

SPSS os arquivos de consulta podem ser executados usando o diálogo File → Open Database → Run Query.... Eles podem ser criados no Assistente de Banco de Dados, que você pode acessar a partir da opção File → Open Database → New Query... menu de aplicativos.
