O que é um arquivo .wsdl? (Arquivo de idioma de descrição de serviços Web)

Arquivo de informações escrito no formato WSDL XML, que é mantido pela W3Corg; fornece descrições de "portas," ou pontos finais de rede, na Internet ou em outra rede; muitas vezes usado para publicar feeds de dados e especificações para acesso remoto a aplicativos.

Mais Informações

Arquivos WSDL descrevem informações como os tipos (definições de tipo de dados), mensagens (os dados comunicados), operações (ações) e protocolos utilizados pelo serviço. No entanto, eles frequentemente descrevem essas informações em um nível abstrato. Portanto, os arquivos WSDL geralmente requerem outro arquivo que forneça as vinculações para os parâmetros do serviço. Isso permite que os arquivos WSDL sejam reutilizados com várias ligações diferentes.

Qualquer número de tecnologias pode ser usada para interagir com serviços WSDL. Por exemplo, os arquivos WSDL são usados pelos aplicativos ASPNET, C/C++e Java.

NOTA: O Microsoft Visual Studio inclui um programa chamado Web Services Description Language Tool (Wsdlexe), que pode ser usado para criar arquivos WSDL.
