O que é um arquivo .ipr? (Instalar arquivo de projeto profissional doShield)

Arquivo de projeto criado pelo InstallShield Professional, um programa usado para criar pacotes de instalação baseados em InstallScript; contém informações do projeto, incluindo configurações, estruturas de diretório, scripts e configurações de diálogo do instalador.

Mais Informações

Arquivos IPR armazenam as informações necessárias para construir um pacote de instalação.

NOTA: Os arquivos IPR são criados usando o InstallShield Professional 3x - 7x.

O que é um arquivo .ipr? (Arquivo de projeto IntelliJ IDEA)

Arquivo de projeto criado pela IntelliJ IDEA, um Java IDE; contém configurações XML para o projeto, incluindo a estrutura do diretório, arquivos de código fonte, configurações de compilador e módulos incluídos (.IML arquivos); usados para salvar e carregar projetos de desenvolvimento.

O que é um arquivo .ipr? (Arquivo de dados iProcrastinado)

Arquivo de dados usado pelo software organizador pessoal iProcrastinate; salva as informações inseridas pelo usuário, como listas de tarefas, eventos e prazos; permite que os dados sejam ressarcidos e também contém dados usados para sincronização entre os dispositivos Mac, iOS e o serviço Dropbox.
