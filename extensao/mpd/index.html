O que é um arquivo .mpd? (Lista de reprodução do criador da lista de arquivos)

Arquivo usado pelo File List Creator (FLC), um utilitário que ajuda os usuários a gerenciar e rastrear seus arquivos de computador; armazena uma lista de arquivos; comumente usado para armazenar uma lista de .MP3 arquivos; também pode armazenar listas de fotos digitais, arquivos compactados e outros tipos de arquivos.

Mais Informações

Enquanto os arquivos MPD rastreiam listas de arquivos, eles contêm apenas referências de arquivos, não os próprios arquivos reais.

NOTA: MPD também é abreviação de "Music Player Daemon," um music player para Unix que pode reproduzir arquivos de áudio de sistemas remotos.

O que é um arquivo .mpd? (Banco de dados do Microsoft Project)

Arquivo de banco de dados criado pelo Microsoft Project, um programa usado para gerenciamento de projetos; contém dados de projeto armazenados em um formato compatível com o Microsoft Access; usado para salvar e carregar um projeto inteiro em um formato de banco de dados baseado em registros relacionais.

Mais Informações

NOTA: Arquivos MPD não podem ser salvos pelo Microsoft Project 2007 ou posteriormente. No entanto, os arquivos MPD existentes ainda podem ser usados como base para um novo projeto.
