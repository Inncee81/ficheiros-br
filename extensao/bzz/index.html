O que é um arquivo .bzz? (Projeto BuzzBundle)

Um arquivo BZZ é um projeto criado pelo BuzzBundle, uma ferramenta de gerenciamento de mídia social. Contém informações de configuração para um ou mais fluxos online, como redes sociais, blogs e fóruns. Os arquivos BZZ também podem armazenar palavras-chave rastreadas personalizadas e relatórios de menções on-line.

Captura de tela de um arquivo bzz no Link-AssistantCom BuzzBundle 2

Você provavelmente só encontrará um arquivo BZZ se você usar o BuzzBundle para rastrear a presença e atividade on-line relacionadas à sua marca. O arquivo BZZ é gerado pelo BuzzBundle quando você cria um projeto e o salva selecionando File → New Project... e, em seguida, clicando em File → Save Project ou Save Project As....

Ao configurar um projeto, você pode criar um perfil para sua marca e conectar canais sociais, como Twitter, Facebook, LinkedIn e YouTube. Você também pode escolher palavras-chave para rastrear e gerar relatórios de vários fatores que influenciam a presença online ou a reputação da sua marca.

Os arquivos do Projeto BuzzBundle só podem ser abertos com o BuzzBundle. Para abrir um arquivo BZZ no seu computador, selecione File → Open Project.... Você também pode abrir um arquivo BZZ SEO PowerSuite Cloud, Dropbox ou um link compartilhado selecionando File → Open Project (SEO PowerSuite Cloud)..., Open Project (SEO Dropbox)... ou Open Project by Shared Link....
