O que é um arquivo .sup? (Arquivo bitmap da legenda)

Imagem de bitmap de legendas extraída de um FILME DVD ou Blu-ray; contém gráficos de texto legendados que são exibidos durante a reprodução do filme; pode ser usado para adicionar legendas de DVD a filmes com programas de autoria como o IfoEdit.

Mais Informações

Os arquivos SUP podem ser convertidos em texto simples usando programas OCR (Optical Character Recognition), como o SupRip.

O que é um arquivo .sup? (Arquivo de definição de projeto super)

Contém uma lista de arquivos Super Project Data (.SUD); pode ser carregado pelo Super Project Analyzer, um programa de análise e emissão de relatórios de desenvolvimento de software; contém referências aos arquivos de dados, mas não contém os dados reais do projeto em si.
