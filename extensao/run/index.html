O que é um arquivo .RUN? (arquivo executável Linux)

Um arquivo RUN é um arquivo executável normalmente usado para instalar programas Linux. Contém dados do programa e instruções de instalação. Arquivos RUN são frequentemente usados para distribuir drivers de dispositivos e software entre Linux usuários. Você pode executar arquivos RUN no terminal do Ubuntu.

O que é um arquivo .RUN? (Arquivo de mapa runa)

Um arquivo RUN é um mapa de nível para Rune, um jogo de ação 3D que usa uma perspectiva em terceira pessoa e é baseado na mitologia nórdica e alta fantasia. Contém informações do mapa, mas não inclui texturas ou dados de música.

Mais Informações

Os arquivos do mapa Rune podem ser criados e editados usando o RunEd, um editor de mapas baseado no editor UnrealEd para Unreal 3D.

O que é um arquivo .RUN? (Arquivo de varredura docano de corrida)

Um arquivo RUN contém informações coletadas em uma varredura de sistema realizada pelo Runscanner, um utilitário analisador de inicialização e sequestro. Ele lista programas de execução, locais de inicialização automática, drivers, serviços e pontos de sequestro encontrados durante a varredura.

Mais Informações

Os arquivos RUN podem ser compartilhados entre usuários (via e-mail, fóruns online, etc) para ajudar a localizar processos maliciosos. Um especialista pode marcar itens em um arquivo RUN que precisam ser corrigidos e enviar o arquivo RUN editado de volta para o usuário.
