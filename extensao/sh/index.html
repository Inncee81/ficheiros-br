O que é um arquivo .sh? (Bash Shell Script)

Um arquivo SH é um script programado para bash, um tipo de shell Unix (Bourne-Again SHell). Ele contém instruções escritas no idioma Bash e pode ser executado digitando comandos de texto dentro da interface de linha de comando da shell.

Mais Informações

Bash é uma linguagem de comando e shell Unix que substitui a concha Bourne, que é como ele conseguiu o nome Bourne-again. É amplamente utilizado por muitas distribuições Linux e os os X.

NOTA: Arquivos SH também podem ser usados para armazenar scripts para outros programas de shell, como Bourne Shell, C-Shell e Korn Shell. Apple Terminal é uma concha Bash.

O que é um arquivo .sh? (Arquivo Unix Shell)

Auto-extração "shar" arquivo que contém arquivos e um script shell com instruções sobre como extrair o conteúdo; criado com o programa de utilidades Unix shar; normalmente apenas utilizável em computadores Unix com o shell Bourne instalado.

Mais Informações

Os arquivos Shar também usam a extensão de arquivo .SHAR.
