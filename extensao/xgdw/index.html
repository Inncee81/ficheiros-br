O que é um arquivo .xgdw? (Arquivo de extensão GDevelop)

Um arquivo XGDW é um arquivo de extensão usado pelo GDevelop, um aplicativo que permite aos usuários criar jogos sem conhecimento de programação. Ele contém uma extensão selecionada pelo usuário ao compilar um projeto .GDG em um executável nativo. O arquivo de extensão permite adicionar funcionalidade ao seu jogo.

Mais Informações

As extensões fornecem uma variedade de recursos para o GDevelop, como ações, novos objetos, comportamentos e novos eventos. Você pode visualizar as extensões disponíveis no GDevelop clicando duas vezes em "Extensões" no painel do Project Manager. Algumas das extensões disponíveis são objeto de texto, objeto sprite de ladrilhos, desenho primitivo, objetos ligados e eventos de função.

O número de extensões que você selecionar para adicionar ao seu jogo determina o número de arquivos XGDW que estarão em sua pasta de projeto quando você compilar seu projeto para um executável nativo. Por exemplo, se você selecionar 3 extensões, haverá 3 arquivos XGDW em sua pasta de projeto, juntamente com o arquivo .EXE e .DLL arquivos. Os arquivos XGDW serão nomeados após as extensões correspondentes escolhidas.

O aplicativo GDevelop foi projetado para todos os tipos de usuários que desejam criar um jogo 2D sem amplo conhecimento de programação. Você pode ilustrar cenas, adicionar e animar objetos e definir eventos que determinam a lógica de sua jogabilidade.
