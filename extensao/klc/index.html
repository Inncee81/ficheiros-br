O que é um arquivo .klc? (Arquivo de origem do criador do layout do teclado)

Um arquivo KLC contém uma configuração de teclado criada pelo Microsoft Keyboard Layout Creator (MSKLC), um programa usado para criar ou modificar layouts de teclado. Ele armazena um layout de um teclado e as funções associadas às teclas. Os arquivos KLC podem incluir cores para teclas atribuíveis, chaves mortas e chaves não atribuídas.

Captura de tela de um arquivo klc no Microsoft Keyboard Layout Creator 14

Para abrir o arquivo KLC no Keyboard Layout Creator, siga estas etapas:

  1. Selecione File → Load Source File....
  2. Navegue até a localização do arquivo KLC e selecione-o.
  3. Clique em Open.

Para criar o arquivo KLC no Keyboard Layout Creator, siga estas etapas:

  1. Configure o teclado selecionando as teclas e atribuindo funções a elas.
  2. Clique Keyboard → Save Source File ou Save Source File As....
  3. Escolha o local de salvamento, nomeie o arquivo e clique em Save.

MSKLC é usado para criar novos layouts de teclado, modificar layouts de teclado e empacotar layouts de teclado para outros usuários Windows. O programa foi projetado para uso em Windows 2000, Windows XP e Windows Server 2003.

No entanto, o MSKLC também pode ser usado em Windows 10 com o Microsoft NET 20 ou NET 35 Framework instalado. A opção NET 35 Framework exige que você ligue "Net Framework 35 (inclui NET 20 e 30)" em Windows Recursos.
