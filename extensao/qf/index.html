O que é um arquivo .qf? (Arquivo do banco de perguntas de diploma)

Arquivo usado pela Wimba Diploma, um programa de criação de perguntas de teste; contém bancos de perguntas usados para criar testes; inclui formatos de perguntas como múltipla escolha, verdadeiro/falso, correspondência, preenchimento em branco e resposta curta; pode ser exportado como um arquivo .RTF para ser visualizado por editores de texto; semelhante ao arquivo .QB.

Mais Informações

NOTA: Quando Wimba foi comprado pela Blackboard em 2010, o desenvolvimento do Diploma cessou.

O que é um arquivo .qf? (Arquivo de versão do Nokia Maps)

Contém informações de versão e dados de configuração para o Nokia Maps, um programa de mapeamento de telefones celulares; normalmente armazenado no diretório raiz do cartão de memória do telefone.

Mais Informações

Em geral, o arquivo QF do Maps não deve ser modificado. No entanto, para desinstalar o Maps 30, o arquivo QF e a pasta "cities" podem precisar ser excluídos.

O que é um arquivo .qf? (Modelo de página de Web Qompose)

Web modelo de página criado com Qompose (pronuncia-se "compor"), uma linguagem templating HTML; permite que várias páginas sejam construídas a partir do mesmo layout básico; usa "tokens" para especificar diferentes seções de cada página.

Mais Informações

Páginas de Web publicadas também terminam em "qf" (em vez de "html"). Eles são processados pelo script Qompose CGI instalado no servidor Web.
