O que é um arquivo .tx_? (Arquivo de texto compactado)

Um arquivo TX_ é um arquivo .TXT compactado normalmente criado pelo Makecab, um utilitário Windows usado para comprimir arquivos e gerar .CAB arquivos. É frequentemente usado para salvar um arquivo TXT em um CD para que ele use menos espaço em disco.

Mais Informações

Para descomprimir um arquivo TX_ em Windows com o Microsoft Expand instalado, digitar o seguinte comando:

expanda myfiletx_ myfiletxt

>NOTA: A sintaxe sublinhada também é usada para muitos outros tipos de arquivos comprimidos, incluindo .EX_, .DL_ e .SY_.
