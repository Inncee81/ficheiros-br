O que é um arquivo .mdd? (Arquivo de recursos MDict)

Arquivo usado pelo MDict, um programa de dicionário de vários idiomas para Windows e plataformas móveis; pode incluir imagens, sons e outros conteúdos que melhorem as definições no dicionário.

O que é um arquivo .mdd? (Arquivo de dados de deformação do forno point)

Modelo de arquivo de dados usado pela Point Oven, um conjunto comercial de plugins usados para transferir dados de vértices e fcurve entre programas de modelagem; salva "baked" dados de vértices para cada quadro de animação de uma malha especificada; permite que posições reais de malha vértice, ou deformações, sejam trocadas entre outros programas que tenham suporte mDD de forno point.

Mais Informações

Como os arquivos MDD armazenam dados de vértices assados em vez de curvas de animação e informações de movimento, eles podem ser muito grandes em tamanho, dependendo do tamanho da malha original e da quantidade de quadros de animação escolhidos.
