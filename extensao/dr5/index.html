O que é um arquivo .dr5? (Projeto Especialista em Histórias Dramáticas)

Um arquivo DR5 é um documento criado pela Dramatica Story Expert, um aplicativo macOS (OS X) usado por escritores para formular histórias. Ele contém um projeto de história, que inclui o título, autor, enredo de história, personagens e relacionamentos de personagens.

Mais Informações

Captura de tela de um arquivo dr5 em Write Brothers Dramatica Story Expert 5

O arquivo DR5 é o principal tipo de arquivo associado ao Dramatica Story Expert. Quando você começa uma nova história e a salva, o programa cria o arquivo DR5 para salvar o conteúdo. Você pode criar um arquivo DR5 selecionando File → Save.

DR5 arquivos também são usados para armazenar modelos. Estes são úteis se você não quiser começar do zero e quiser reutilizar elementos da história do modelo. Para começar a partir de um modelo, selecione File → New Document from Template....

Dramatica Story Expert é a versão macOS do Dramatica Pro, que só está disponível para Windows. Dramatica Story Expert foi projetado para ajudá-lo a juntar sua história definindo seu enredo geral, personagens, relacionamentos de personagens e temas. Ele também vem com ferramentas de brainstorming, um mecanismo de história para gerar um enredo, e uma ferramenta de elenco para adicionar personagens.

Dramatica Pro salva projetos no arquivo .DSF em vez do arquivo DR5. No entanto, se você quiser abrir um arquivo DR5 no Dramatica Pro, você pode renomear a extensão "dr5" para a extensão "dsf".
