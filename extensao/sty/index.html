O que é um arquivo .sty? (Arquivo estilos de legenda aegisub)

Arquivo criado pelo Aegisub, um editor de legendas de plataforma cruzada; salvo em um formato de texto simples e armazena as propriedades da fonte para um ou mais estilos em uma categoria de legenda; inclui o tipo de fonte, cores, efeitos e margens; usado para salvar e carregar fontes criadas no programa.

Mais Informações

Os estilos Aegisub são gerenciados através da janela Styles Manager, que é acessada selecionando Subtitles → Styles Manager... no menu do aplicativo. Os arquivos STY são salvos automaticamente e não são acessados diretamente pelo usuário.

A versão Windows da Aegisub armazena arquivos STY no seguinte diretório: [user]\AppData\Roaming\Aegisub\catalog\. A versão Mac salva para este diretório: [user]/Library/Application Support/Aegisub/catalog/.

O que é um arquivo .sty? (Arquivo de estilos de banda em uma caixa)

Arquivo de estilo musical usado pela Band-in-a-Box, (BiaB) um programa que cria automaticamente acompanhamentos completos da banda; contém um estilo de reprodução, como Jazz, Rock, Hip-Hop ou Country; usado para gerar sons estilizados para um determinado gênero musical.

Mais Informações

Os estilos de BiaB podem ser MIDI ou podem usar instrumentos e tambores reais, chamados RealStyles. RealStyles usam gravações reais de músicos para as amostras de áudio.

Os arquivos STY atuam como plug-ins para o software Band-in-a-Box. A instalação do software normalmente contém muitos estilos diferentes. Estilos adicionais podem ser comprados e baixados no site da PG Music Band-in-a-Box.

O que é um arquivo .sty? (Estilo LaTeX)

Arquivo de definição de estilo usado para formatar um documento LaTeX (.TEX ou .LTX); define estilos de linha, rótulos, como as fórmulas devem ser alinhadas e outras informações; salvo em um formato de texto no qual as definições começam com uma barra invertida () e os comentários começam com um sinal percentual (%).

Mais Informações

Os arquivos de estilo LaTeX são frequentemente referenciados por documentos LaTeX.

Arquivo de história do NewsMaker

Desenvolvedor EyePower Games
Categoria Dados
Formato XML 

O que é um arquivo .sty? (Arquivo de história do NewsMaker)

Arquivo criado pelo NewsMaker, um aplicativo que permite que os alunos criem notícias; salvo em um formato XML e contenha o texto escrito para a história; referenciado dentro de um projeto NewsMaker (.NMP arquivo).

Mais Informações

Os projetos do NewsMaker podem conter uma ou mais histórias, cada uma das quais armazenadas em um arquivo STY separado. Os usuários não abrem arquivos STY diretamente, uma vez que são salvos dentro do diretório do projeto NewsMaker por padrão e podem ser acessados a partir do projeto aberto no software.

NOTA: EyePower Games NewsMaker não está mais desenvolvido.

O que é um arquivo .sty? (Estilo de teclado Yamaha/Korg)

Estilo musical criado em um sintetizador Yamaha ou Korg; normalmente contém um ritmo musical que inclui múltiplos instrumentos; pode ser salvo em um disco e carregado em teclados compatíveis.

Mais Informações

NOTA: Yamaha e Korg salvam arquivos STY em diferentes formatos, o que significa que eles não são multiplataformas. Os estilos Yamaha só podem ser usados nos teclados Yamaha e os estilos Korg só podem ser usados nos teclados Korg. Os estilos Yamaha podem ser editados usando o Style Fixer, enquanto os estilos Korg podem ser editados com o programa GNIStyle.
