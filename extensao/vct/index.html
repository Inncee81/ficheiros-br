O que é um arquivo .vct? (Imagem vetorial de IDRISI)

IMAGEM GIS criada por IDRISI Taiga, um aplicativo GIS usado para analisar e visualizar imagens geoespaciais; armazena dados vetoriais usando pontos conectados em vez de dados raster usados pelos arquivos .RST da IDRISI.

Mais Informações

Arquivos VCT são úteis para armazenar informações mapeadas, como estradas e limites de propriedade. Os dados vetoriais permitem que os recursos da imagem sejam ampliados sem perder a qualidade da imagem.

O que é um arquivo .vct? (Projeto instalador VISE)

Arquivo de projeto criado pelo VISE, um aplicativo usado para criar instaladores de software Windows e Mac; salva referências a arquivos de instalação, bem como parâmetros de configuração, scripts e ações necessárias para fazer a instalação; pode ser incorporado em um pacote de instalador padrão.

Mais Informações

VISE está disponível em várias edições: VISE X (Mac OS X), Instalador VISE para Macintosh (Mac OS Classic e Mac OS X) e Installer VISE para Windows.
