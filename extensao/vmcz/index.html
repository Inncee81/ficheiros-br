O que é um arquivo .vmcz? (Máquina virtual compactada hyper-V)

Um arquivo VMCZ contém uma máquina virtual compactada (VM) criada pelo Microsoft Hyper-V, um programa de virtualização Windows. Ele armazena a configuração de estado, dados e hardware de um VM no momento da criação do arquivo em um pacote compactado por Zip. Os arquivos VMCZ são exportados pela Hyper-V para compartilhar e copiar máquinas virtuais para outros computadores ou servidores.

Mais Informações

O Microsoft Hyper-V permite que os usuários executem vários sistemas operacionais como máquinas virtuais em Windows, bem como criem e gerenciem componentes de software e hardware virtualizados. O software Hyper-V também vem com o Hyper-V Manager, que é uma ferramenta que fornece uma interface de usuário amigável (UI) para interagir com o Hyper-V. O Hyper-V Manager está disponível como um programa opcional incorporado em sistemas Windows que suportam Hyper-V.

A Microsoft introduziu o tipo de arquivo VMCZ quando adicionou o recurso de exportação de máquina virtual ao Hyper-V Manager com o lançamento do Windows 10 Insider Build de 16226 em junho de 2017. O recurso foi introduzido através da adição de um botão "Share" à interface do usuário do Hyper-V Manager que permite aos usuários exportar um Hyper-VM como um arquivo VMCZ compactado.

O recurso Compartilhar facilita a cópia de máquinas virtuais e o transporte para outros locais. Depois de mover um arquivo VMCZ para outro sistema com o Hyper-V instalado, o usuário pode simplesmente clicar duas vezes no arquivo para importá-lo em Hyper-V.

Uma vez que um arquivo VMCZ é um pacote comprimido por Zip, o usuário também pode extrair manualmente o conteúdo do pacote com um utilitário de descompressão zip, como Microsoft File Explorer, Corel WinZip, 7-Zip ou WinRAR. O conteúdo normalmente inclui uma unidade de disco rígido VM .VHD ou .VHDX arquivo, uma configuração VM .VMCX arquivo e um arquivo VM run state VMRS.

Para exportar uma máquina virtual no Hyper-V Manager como um arquivo VMCZ, selecione o botão "Compartilhar" na barra de ferramentas ou clique com o botão direito do mouse em uma máquina virtual e selecione Export.

NOTA: Hyper-V foi lançado pela primeira vez com o Windows Server 2008 e foi disponibilizado gratuitamente a partir de Windows 8 e Windows Server 2012. Está disponível em versões de 64 bits de Windows 10 Pro, Enterprise e Education.
