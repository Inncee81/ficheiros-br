O que é um arquivo .m? (Arquivo de entrada matemático)

Contém funções ou um conjunto de instruções usadas pelo Mathematica, um programa para criar e executar funções matemáticas; salvo em um formato de texto, que pode ser visualizado e editado com um editor de texto básico.

Mais Informações

Arquivos chamados "initm" são arquivos de inicialização que contêm configurações carregadas quando o programa Mathematica é aberto.

O que é um arquivo .m? (Arquivo de código-fonte DO MATLAB)

Um arquivo M é um arquivo de texto usado pelo MATLAB, um aplicativo usado para cálculos matemáticos. Ele pode armazenar um script, classe ou uma função individual no idioma MATLAB. Arquivos M são usados para executar algoritmos, traçar gráficos e realizar outras operações matemáticas.

O que é um arquivo .m? (Arquivo de código fonte de Mercúrio)

Arquivo de código fonte escrito usando Mercúrio, uma linguagem de programação funcional que foi desenvolvida na Universidade de Melbourne, Austrália; salvo usando formatação de texto simples e usa sintaxe semelhante ao Prolog.

Mais Informações

Mercury foi escrito para fornecer uma linguagem robusta para grandes aplicações do mundo real. Ele inclui vários recursos de verificação de erros não fornecidos por outras linguagens de programação lógica. Por exemplo, o idioma incorpora análises estáticas que podem capturar erros de verificação de tipo antes do tempo de execução. Isso permite maior garantia sobre a correção do programa e também permite que o compilador gere um código mais eficiente. No entanto, o desenvolvedor deve especificar que cada um dos construtos "type," "mode" e "determinism" é durante a programação, o que não é necessariamente exigido por outros idiomas.

NOTA: A versão de desenvolvimento do compilador Mercury está atualmente distribuída em formato de código-fonte e não inclui binários de compilador. No entanto, uma Windows porto de Mercúrio, chamada winmercury, inclui binários. Ele também inclui arquivos de destaque de sintaxe vim para edição de código-fonte.

O que é um arquivo .m? (Arquivo de implementação objetivo-C)

Um arquivo M é um arquivo de implementação de classe usado por programas escritos em Objective-C. Começa com a diretiva @implementation e inicializa variáveis e funções que podem ser referenciadas por outros arquivos de origem Objective-C. Os arquivos M também podem referenciar arquivos de cabeçalho (.H).

Mais Informações

Objective-C é mais comumente usado para implementar aplicativos para os sistemas operacionais Mac OS X e iOS da Apple. A API cacau da Apple fornece a interface de programação Objective-C para Mac OS X, e a iOS API fornece a interface de programação para a plataforma iOS.
