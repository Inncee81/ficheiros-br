O que é um arquivo .erd? (Arquivo de dados da Divisão de Pesquisa de Engenharia)

Um arquivo ERD é um arquivo de dados usado por vários programas usados para analisar os perfis de estradas, como ProVAL e EP (Engineering Plotter). Pode conter perfis rodoviários, saída de simulação de veículos, dados de teste do veículo e medições do comportamento dos componentes do veículo.

Mais Informações

O formato ERD consiste em duas partes:

  • Header - Armazena informações sobre como as ferramentas pós-processamento devem ler os dados.
  • Dados - Armazena dados numerados, que podem estar em texto ou formulário binário.

Se a parte de dados for armazenada em forma de texto, tanto o cabeçalho quanto as partes de dados estiverem contidos em um arquivo ERD. Se a parte de dados for armazenada em forma binária, a parte do cabeçalho será armazenada como texto simples em um arquivo ERD separado da parte de dados, que é armazenada em um arquivo .BIN. Tanto o arquivo ERD quanto o BIN devem ter o mesmo nome e serem colocados na mesma pasta para serem lidos por um programa.

O formato ERD foi desenvolvido pela Divisão de Pesquisa em Engenharia (ERD) do Instituto de Pesquisa de Transportes da Universidade de Michigan (UMTRI). Foi originalmente usado apenas pelo RoadRuf, um programa de análise de perfil rodoviário. No entanto, acabou sendo adotado como um formato de dados padrão para análise e medição de perfis rodoviários.

O que é um arquivo .erd? (Diagrama de Relação de Entidade)

Um arquivo ERD é um diagrama criado pela DBeaver, uma ferramenta de banco de dados de código aberto usada por desenvolvedores, administradores de bancos de dados, analistas e programadores SQL. Ele contém um diagrama de um banco de dados ou esquema, que consiste em tabelas de dados dispostas em uma grade e setas conectando tabelas relacionadas. Os arquivos ERD podem ser gerados automaticamente ou personalizados construídos por um usuário.

Mais Informações

Os arquivos ERD só são suportados pelo DBeaver. Se você quiser compartilhar seu diagrama em um formato mais amplamente suportado, você pode salvá-lo como um arquivo .PNG, .GIF ou .BMP. Você também pode salvá-lo no formato .GRAPHML. Para salvar seu diagrama em um desses formatos, selecione File → Save As e escolha o formato.
