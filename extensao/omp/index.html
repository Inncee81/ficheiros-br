O que é um arquivo .omp? (Arquivo de documentos do gerente do escritório)

Banco de dados criado com o Office Manager, um sistema de gerenciamento de banco de dados configurável e de propósito geral; suporta catalogação de dados e indexação automática de documentos; permite acesso rápido a documentos Windows.

Mais Informações

Os arquivos OMP são salvos em um formato de banco de dados padrão que também pode ser aberto pelo Microsoft Access.

Programas que abrem o arquivo .omp (Arquivo de documentos do gerente do escritório)

  Windows
  Krekeler Office Manager Pro 
  Krekeler Office Manager Enterprise 
  Microsoft Access 2019

O que é um arquivo .omp? (Documento de Windows do OpenMind)

Documento do mapa mind criado e usado pelo MindView (anteriormente OpenMind), um aplicativo usado para pensar visualmente e apresentar ideias; contém um mapa com uma ou mais ideias, que inclui texto, formas, ramos e outros elementos do mapa da mente.

Mais Informações

NOTA: O arquivo OMP foi usado pela versão Windows do OpenMind antes que a extensão do arquivo .MVDX o substituísse e o nome do aplicativo mudasse para MindView.
