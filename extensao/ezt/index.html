O que é um arquivo .ezt? (Arquivo worm malicioso EZT)

Arquivo de worm malicioso que se espalha de computador para computador através de redes de compartilhamento de arquivos e e-mail; contém dados que permitem que o worm baixe conteúdo malicioso e se replique; instala-se ao processo de inicialização Windows quando um computador é infectado.

Mais Informações

Para remover o worm EZT, use um programa antivírus. Além disso, embora não seja recomendado, pode ser possível remover manualmente worms EZT matando todos os processos de worm, excluindo o arquivo worm e removendo todas as entradas de worms do registro do computador.

EZT worm files use o nome WormWin32AutoRunezt.

O que é um arquivo .ezt? (Arquivo de legendas EZTitles)

Formato de legenda usado pelo software de legendagem de filmes EZTitles; suporta formatos de DVD, Blu-ray e HD-DVD, bem como legendas 3D; após edição, pode ser exportado para formatos de legendagem padrão sevral, incluindo .SRT, .SUB, .PAC, .890, EBU (.STL) e .SMI.
