O que é um arquivo .m5p? (Arquivo predefinido machfive)

Um arquivo M5P é um predefinido usado por MachFive, um sampler para produção musical. Contém uma predefinição, que é um instrumento ou efeito usado pela MachFive como a unidade principal para a criação de composições.

Mais Informações

MachFive vem com predefinições incluídas, mas você também pode carregar predefinições no programa. Você pode usar as predefinições para produzir performances, que são armazenadas em arquivos M5M.

M5P predefinições que vêm com o aplicativo estão localizadas na pasta "Preset". Você pode ver o arquivo M5P nesta pasta, mas provavelmente você vai se deparar com o arquivo ao receber o arquivo de outro usuário ou site para carregar no programa.
