O que é um arquivo .igg? (Arquivo de mapa de cores do SuiteProfiler)

Arquivo de dados do Mapa de Cores criado pelo SuiteProfiler, um programa que permite que os usuários criem mapas de cores independentes do dispositivo e, em seguida, usem os mapas de cores para produzir perfis de mapa de cores ICC dependentes do dispositivo para prova suave e impressão de imagem; pode incluir vários perfis de mapa de cores que juntos compõem um conjunto de perfis para um determinado ambiente de perfil.

Mais Informações

SuiteProfiler e arquivos IGG permitem que os usuários criem perfis de mapa de cores para alterações na escala de cores sem editar a imagem original. Diferentes perfis de mapa de cores podem ser aplicados à mesma imagem para alcançar diferentes efeitos, como aparências em preto e branco ou esquemas de cores personalizados para um dispositivo de impressão de destino.
