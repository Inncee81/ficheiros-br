O que é um arquivo .smod? (Arquivo Stonehearth Mod)

Um arquivo SMOD é um arquivo comprimido zip que contém um mod usado pela Radiant Entertainment Stonehearth, um jogo onde você defende e constrói uma aldeia para sobreviver com outros colonos pioneiros. Ele armazena um diretório com todos os arquivos de modificação (arquivos de imagem, modelo ou dados) e um arquivo .JSON (manifest.json) que descreve o mod. Arquivos SMOD são usados para alterar a jogabilidade, como adicionar novos móveis, receitas de alimentos, equipamentos e traduções de idiomas.

Mais Informações

A criação de modificações é incentivada em Stonehearth e muitos usuários criam e compartilham arquivos SMOD com a comunidade Stonehearth. Para adicionar um mod a Stonehearth, coloque o arquivo SMOD em um dos seguintes diretórios:

  • Se você instalou o StoneHearth através do Humble Bundle - C:\Program Files (x86)\Stonehearth\mods
  • Se você instalou o StoneHearth através do Steam - C:\Program Files (x86)\Steam\SteamApps\common\Stonehearth\mods

>>Uma vez que o arquivo SMOD é colocado na mods pasta, open Stonehearth e o mod é acessível e ativo.

Como os arquivos SMOD são compactados com compactação zip, eles podem ser expandidos usando um utilitário de descompressão Zip. Basta renomear a extensão .smod para .zip e usar um programa de descompressão zip para expandir os conteúdos e visualizá-los. Alguns programas zip incluem Corel WinZip, 7-Zip, Apple Archive Utility e ES File Explorer.
