O que é um arquivo .iip? (Instalar arquivo de projeto Pro creator)

Arquivo criado pelo Install Creator Pro, um programa usado para fazer Windows instaladores de programas; contém configurações como diretório de destino, arquivos para instalar, versão de Windows de destino e Windows local do Menu Iniciar; usado para criar um instalador de Windows com propriedades personalizadas.

Mais Informações

NOTA: Os arquivos IIP usados pela versão padrão do Install Creator têm uma extensão .IIT.

O que é um arquivo .iip? (Arquivo de configuração ThinAnywhere)

Contém configurações usadas para se conectar a um servidor ThinAnywhere; pode incluir o endereço IP remoto, informações de login, tamanho da janela e outros parâmetros; pode ser salvo pelo ThinAnywhere uma vez que uma conexão bem sucedida tenha sido estabelecida.

Mais Informações

Os arquivos IIP são salvos no diretório "Meus documentosMinhas conexões" por padrão. Arquivos IIP salvos podem ser carregados pelo ThinAnywhere para conectar rapidamente a servidores ativos.

"iIP" também é o nome do protocolo que ThinAny onde usa para se comunicar entre o servidor e o cliente. Ele suporta criptografia AES de 128 bits para fins de segurança.
