O que é um arquivo .ba7? (TaxAct 2017 Tax Return Backup File)

Um arquivo BA7 é um arquivo de backup de uma declaração de imposto .TA7 de 2017 criada com a versão de 2017 do TaxAct, um programa de preparação fiscal. Ele contém dados fiscais para o ano fiscal de 2017, que é usado para fins de arquivamento e no caso do arquivo TA7 ser perdido ou se tornar corrupto. Os arquivos BA7 armazenam dados federais da Receita Federal e formulários fiscais estaduais, juntamente com as informações básicas do usuário.

Mais Informações

Os usuários do TaxAct 2017 provavelmente usarão apenas o arquivo TA7 para salvar, imprimir ou carregar seus impostos. No entanto, o arquivo BA7 é útil para fazer backup de um arquivo TA7 no caso de ele ficar corrompido ou perdido.

Para salvar um backup da sua declaração de imposto de renda selecione File → Backup... e nomeie seu arquivo. Você deve salvar sua declaração de imposto como um arquivo TA7 antes de criar seu backupPara restaurar um backup no TaxAct, selecionar File → Restore Backup... e navegar até a localização do seu arquivo BA7.

Como as declarações fiscais anuais são arquivadas no ano anterior, o arquivo BA7 é criado em 2018 normalmente entre o início de janeiro e 15 de abril. Uma vez que você tenha concluído uma declaração de imposto, você pode imprimir e enviar nos formulários fiscais ou arquivar eletronicamente.
