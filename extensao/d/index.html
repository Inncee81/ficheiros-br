O que é um arquivo .d? (Arquivo de código fonte D)

Código fonte escrito na linguagem de programação D; semelhante ao C++, mas também influenciado por C#, Java e Eiffel; pode ser aberto e editado em um editor de texto, mas requer um compilador D para ser compilador em um programa executável.

O que é um arquivo .d? (Arquivo de dependência de origem)

Um arquivo D é um arquivo de dependência de origem gerado pelo GCC, um compilador GNU C. Ele contém dependências em texto simples que descrevem os arquivos que foram usados para criar objetos compilados (.O arquivos) por um compilador C. Os arquivos D são gerados automaticamente quando o sinalizador -MMD é ativado ao compilar com GCC.

Mais Informações

D arquivos são normalmente usados pelo GNU Make, um utilitário usado para controlar a criação de executáveis e outros arquivos não-fonte de um programa. Os arquivos D também podem ser abertos por qualquer editor de texto, uma vez que armazenam texto simples.
