O que é um arquivo .ufr? (Arquivo Ultra Fractal)

Imagem fractal criada pela Ultra Fractal, um programa usado para criar arte fractal; salva parâmetros usados para gerar uma imagem fractal; inclui configurações de cores e perspectiva ampliada do usuário dentro da imagem; pode incluir sobreposições de cores fluorescentes, gradientes e outras configurações de cores; usado para fundo de desktop, bem como arte impressa.

Mais Informações

Os arquivos UFR podem ser exportados para vários formatos de imagem raster, incluindo .JPG, .PNG, .TIFF e .BMP.

O que é um arquivo .ufr? (Arquivo de upfiring)

Um arquivo UFR é um arquivo de dados usado pelo Upfiring, um aplicativo de compartilhamento de arquivos peer-to-peer (P2P), para baixar um ou mais arquivos carregados por outros usuários. Ele não armazena o conteúdo real, mas contém metadados criptografados, o que inclui informações sobre o (s) arquivo(s) sendo compartilhado, o preço dos arquivos e o proprietário do arquivo(s). Os arquivos UFR são semelhantes aos arquivos .TORRENT, que são usados pelo BitTorrent e outros programas de compartilhamento de arquivos P2P.

Mais Informações

Upfiring é um programa P2P baseado em blockchain do Ethereum que fornece tokens UFR para usuários que carregam arquivos para a rede e "semente" os arquivos, que é quando os usuários ficam conectados à rede para ajudar a distribuir arquivos para outros usuários. Ao incentivar os usuários a se tornarem "semeadores"," o Upfiring é capaz de oferecer uma rede mais robusta de usuários, o que resulta em tempos de download mais rápidos e mais arquivos disponíveis para download.

A Upfiring administra todas as transações entre usuários usando contratos inteligentes, o que permite que os usuários enviem o Ether (a moeda digital Ethereum semelhante ao Bitcoin) para outros usuários. Os usuários devem adicionar uma carteira Ethereum no aplicativo Upfiring para concluir corretamente as transações.

Para criar um arquivo UFR no Upfiring, selecione o ícone "Uploads" no painel esquerdo e clique em Upload. Após o upload do arquivo UFR, outros usuários do Upfiring podem acessar o arquivo UFR para baixar arquivos compartilhados.

Para importar um arquivo UFR no Upfiring, selecione o ícone "Downloads" no painel esquerdo e arraste e solte o arquivo UFR ou navegue arquivos no seu computador para selecionar o arquivo UFR.

NOTA: Você pode visualizar o conteúdo de um arquivo UFR usando o Simpio UFR Viewer.
