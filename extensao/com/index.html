O que é um arquivo .com? (Arquivo de comando DOS)

Um arquivo COM é um programa executável capaz de ser executado pelo MS-DOS e pelo Windows. Ele é salvo em um formato binário e é semelhante a um metadados. Os arquivos COM são comumente usados para executar um conjunto de instruções, enquanto os arquivos EXE são usados para programas totalmente desenvolvidos.

Mais Informações

A maioria dos usuários Windows não encontrará arquivos executáveis com, uma vez que os arquivos são usados principalmente no MS-DOS. No entanto, você pode adquirir um arquivo COM antigo que armazena um programa ou comandos e precisa executá-lo em Windows.

Você pode executar um arquivo COM na versão de 32 bits de Windows mas será solicitado a instalar o componente NT Virtual DOS Machine (NTVDM) quando iniciar o arquivo pela primeira vez. Você também pode executar um arquivo COM na versão de 64 bits de Windows com uma máquina virtual que suporta o ambiente MS-DOS.

NOTA: Se uma pasta incluir arquivos EXE e COM com o mesmo nome de arquivo (por exemplo, run.exe e run.com), o prompt de comando DOS ou Windows executará o arquivo COM se você digitar o nome do arquivo sem a extensão.
