O que é um arquivo .rvl? (Arquivo da biblioteca Adobe Revel)

Biblioteca de fotos criada pela Adobe Revel, um programa de gerenciamento de fotos digital; salva cópias de fotos digitais importadas, bem como quaisquer metadados criados pelo usuário (eventos, favoritos, etc); usado para armazenar a biblioteca de imagens no computador local e para manter a biblioteca em sincronia com a nuvem.

Mais Informações

Os arquivos RVL são criados automaticamente quando você cria uma nova biblioteca no Revel, mas você não os salva ou os carrega manualmente. Da próxima vez que você iniciar o Revel, o software detecta automaticamente a localização do arquivo RVL e reabre a biblioteca.

NOTA: A Revel foi descontinuada pela Adobe.

O que é um arquivo .rvl? (muvee revelar arquivo do projeto X)

Projeto de filme criado pelo muvee Reveal X, um programa usado para fazer vídeos caseiros; salva uma lista de fotos, vídeos e música para o filme; também inclui o estilo de apresentação que mostra diferentes transições e efeitos.

Mais Informações

Filmes criados pelo muvee Reveal X podem ser exportados para Facebook, YouTube, DVD, dispositivos móveis e formatos personalizados de arquivos de vídeo, incluindo .WMV, .MOV, .MPG, .AVI e .MP4.

NOTA: Os arquivos RVL não armazenam dados reais de vídeo e áudio. Eles só armazenam referências a arquivos de áudio e vídeo incluídos no projeto. Portanto, se você quiser compartilhar um projeto, você precisará compartilhar os arquivos de áudio e vídeo separadamente também. Os arquivos RVL usam um formato XML de texto simples.
