O que é um arquivo .sph? (Arquivo de mapeamento da esfera de MikuMikuDance)

3D arquivo de efeitos de modelo usado pelo MikuMikuDance, um jogo onde os jogadores controlam personagens de dança virtual vocalóides; contém uma imagem .BMP, mas usa a extensão "sph" em vez disso; pode ser usado para adicionar um visual brilhante a recursos de modelo 3D, como cabelo ou roupas.

Mais Informações

Arquivos SPH também podem incorporar transparência (por exemplo, .PNG ou .TGA imagens), mas são mais comumente criados a partir de imagens bitmap não transparentes.

NOTA: Para criar um arquivo SPH, desenhe uma imagem de bitmap ("bmp") com qualquer editor gráfico e renomeie a extensão do arquivo para "sph"

O que é um arquivo .sph? (Arquivo de áudio NIST SPHERE)

Arquivo de áudio Waveform criado no formato NIST SPHERE, que muitas vezes é usado em pesquisas de reconhecimento de fala; muitas vezes contém uma gravação de uma pessoa falando; pode ser fornecido como entrada para sistemas de reconhecimento de fala.

Mais Informações

Formas de onda SPHERE usam PCM linear de 16 bits e uma taxa de amostra de 16KHz. Eles têm um cabeçalho 1024 byte de leitura humana que usa formatação de texto ASCII.

NOTA: Os arquivos SPH podem ser convertidos para o formato .WAV usando um utilitário chamado sph_convert ou SoX.
