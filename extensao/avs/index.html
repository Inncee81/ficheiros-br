O que é um arquivo .avs? (Arquivo de variações do Adobe Photoshop)

Arquivo de configurações usado pelo Adobe Photoshop, um aplicativo de edição de imagens; contém ajustes que podem ser aplicados a uma imagem; pode ser acessado selecionando a seguinte opção de menu: Image → Adjustments → Variations....

Mais Informações

NOTA: Algumas versões de 64 bits do Photoshop não incluem o submenu Variações. A Adobe fornece um download manual de plug-in para corrigir esse problema em algumas versões de software.

O que é um arquivo .avs? (Arquivo de script AviSynth)

Script criado para a AviSynth, uma ferramenta de pós-produção de vídeo usada para edição e processamento de arquivos de vídeo; usada para automatizar funções de vídeo, como cortar seções, redimensionar clipes de vídeo, dublar áudio, ajustar brilho, etc.

Mais Informações

O software AviSynth é distribuído como um arquivo .DLL, que funciona como um "frameserver" que serve quadros de vídeo para o media player sob demanda quando um arquivo AVS é aberto. Dessa forma, o media player reconhece os quadros servidos como dados de vídeo padrão .AVI. Portanto, para reproduzir vídeo usando Avisynth, crie um script Avisynth e, em seguida, abra-o com um leitor de mídia AVI suporte.

O que é um arquivo .avs? (Arquivo predefinido AVS)

Arquivo de configuração usado pelo Advanced Visualization Studio (AVS), um visualizador de áudio para Winamp; contém configurações para animações criadas com base na música reproduzida dentro do Winamp.
