O que é um arquivo .capx? (Arquivo Capella CapXML)

Partitura musical criada pela Capella, um programa de composição usado para criação e edição de partituras musicais; contém anotações e outros dados de músicas, como texto e gráficos.

O que é um arquivo .capx? (Construir arquivo de projeto compactado)

Projeto de desenvolvimento de jogos criado pela Construct, um programa usado para fazer jogos 2D para iOS, Android, HTML5, Chrome e outras plataformas; salva todos os arquivos do projeto em um formato de .ZIP comprimido; inclui gráficos, sons, modelos de física e eventos de jogos; usado para salvar e transportar todo o projeto como um único arquivo.

Mais Informações

ARQUIVOS CAPX também incluem o arquivo de projeto .CAPROJ XML que é usado como ponto de entrada para um projeto não compactado.

NOTA: Para salvar um projeto Construct como um arquivo CAPX, escolha File → Save As Single File... no menu do aplicativo.
