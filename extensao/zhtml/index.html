O que é um arquivo .zhtml? (Página da RabbitWeb)

Página da Web analisada pela RabbitWeb, um servidor de Web que integra tags HTML; contém comandos "ZHTML" incorporados na página da Web que são usados para gerar conteúdo dinâmico para a página da Web; às vezes podem ser vistos como "zhtml" em uma URL de Web navegador.

O que é um arquivo .zhtml? (Arquivo HTML com zíper seguro)

Web página salva pelo Secure IE, um navegador Web semelhante ao Internet Explorer, mas inclui recursos adicionais de segurança; armazena o HTML e as imagens da página Web em um único arquivo compactado.

Mais Informações

NOTA: O Secure IE requer que o Internet Explorer seja instalado para ser executado.
