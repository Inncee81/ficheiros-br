O que é um arquivo .wsc? (Componente de script Windows)

XML formatos de scripts objeto contendo propriedades e/ou métodos; pode ser referenciado por um ou mais scripts; suporta várias linguagens de desenvolvimento de software, incluindo VBScript (.VBS), Páginas do Servidor Ativo (.ASP) e uma variedade de outros idiomas.

Mais Informações

Arquivos WSC permitem que os desenvolvedores criem componentes com (Component Object Model, modelo de objeto de componentes) reutilizáveis. Esses componentes permitem que os componentes de software se comuniquem entre si e com serviços prestados pelo sistema operacional Windows.

O que é um arquivo .wsc? (WonderSwan Color Game ROM)

Arquivo de jogo usado por emuladores para o sistema de console portátil japonês WonderSwan, que é semelhante ao Game Boy Color; contém um jogo ROM, que armazena as mesmas informações do cartucho original do jogo; pode ser usado para jogar jogos no PC com software de emulação como Oswan ou Cygne.
