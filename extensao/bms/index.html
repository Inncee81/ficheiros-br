O que é um arquivo .bms? (Arquivo de música BeatMania)

Arquivo de música usado pela BeatMania, um jogo de vídeo rítmico com interface controladora de DJ; salvo em formato de texto e armazena dados referenciados durante a reprodução da música; inclui o título da música, artista, gênero, BPM (batidas por minuto), número de jogadores, posições de notas, referências a arquivos de efeito sonoro e outras informações.

Mais Informações

Os arquivos BMS são nativos do BeatMania, um videogame baseado em console, mas podem ser abertos pela StepMania em um PC.

O que é um arquivo .bms? (Arquivo de segurança multicamadas BIOWRAP)

Arquivo criptografado gerado no formato BMS (Multi-Layer Security, segurança multicamadas) do BIOWRAP; pode incorporar mecanismos de autenticação biométrica e digital; pode conter qualquer tipo de dados digitais, como um documento de texto ou um grande arquivo de dados de negócios; usado para enviar arquivos digitais com segurança usando anexos de e-mail, discos, pen drives e outros meios.

Mais Informações

Os arquivos BMS podem ser abertos com o aplicativo de desktop BIOWRAP Reader gratuito. Eles também podem ser visualizados através do aplicativo do site online BIOWRAP, que é executado em um navegador Web. Além disso, eles podem ser acessados usando o Serviço de Web de Leitores BIOWRAP, que permite que documentos BMS sejam integrados às plataformas de fluxo de processos de documentos existentes.

Os arquivos BMS são gerados usando o software BIOWRAP da Nation Technologies (edições Profissionais, Empresariais e Industriais#. Este software e a interface Web BIOWRAP permitem que os usuários especifiquem direitos de retenção e restrições finas de controle de acesso em grãos, que limitam o acesso aos arquivos com base em critérios específicos.

O Sistema BIOWRAP utiliza criptografia baseada em identidade para certificar a identidade e as credenciais #company, grupo, indústria, etc. de um indivíduo que cria arquivos criptografados. Os arquivos criptografados são carimbados com o nível de autenticação usado para executar a criptografia. Os mecanismos de identificação suportados incluem biometria, certificados PKI x509 e nomes de usuário e senhas do BIOWRAP.

NOTA: BioWRAP Reader foi descontinuado.
