O que é um arquivo .vsi? (Arquivo instalador de conteúdo do estúdio visual)

Pacote de instalação do Visual Studio que pode conter kits iniciais, modelos de itens, controles, trechos de código, complementos ou outros tipos de arquivos; usado para trocar conteúdo do Visual Studio dentro da comunidade de desenvolvedores; também referido como um "componente comunitário"

Mais Informações

Os pacotes de instalação de conteúdo do Visual Studio são compactados usando compactação ZIP. Eles são semelhantes aos arquivos .ZIP, mas usam a extensão "vsi". Os arquivos VSI podem ser criados e assinados usando o programa MakeZipExe.
