O que é um arquivo .3d? (Arquivo de imagem CAD-3D estéreo)

Imagem tridimensional usada pelo Atari Stereo CAD-3D, um programa para sistemas Atari ST; armazena objetos 3D usando a biblioteca Motorola Fast Floating Point (LIBF) para coordenadas de vértices; usado como um formato anterior para animações 3D.

O que é um arquivo .3d? (Arquivo de caverna 3D da Survex)

3D arquivo criado pela Survex, um programa usado para gravar e visualizar dados do sistema de cavernas; contém dados que permitem que o software visualize um sistema de cavernas 3D; foi gerado a partir de um arquivo de .SVX usando o programa cavern incluído no Survex.
