O que é um arquivo .pst? (Arquivo de armazenamento de informações pessoais do Outlook)

Um arquivo PST é um arquivo de armazenamento de dados que contém informações pessoais usadas pelo Microsoft Outlook e exchange. Também pode incluir pastas de e-mail, contatos, endereços e outros dados.

Mais Informações

Os arquivos PST têm um limite de 2 GB e farão com que o Outlook desacelere quando o arquivo estiver perto do limite. A Microsoft fornece um programa chamado Exchange 2000 Tool: PST2GB que reduzirá o tamanho do arquivo de um arquivo PST. No entanto, o utilitário pode truncar o arquivo, o que significa que alguns dados serão perdidos, então você deve ter certeza de fazer backup do arquivo PST primeiro.

Ao contrário da versão completa do Outlook, o Outlook Express não usa arquivos PST. No entanto, o Outlook Express pode importar dados PST usando o recurso de importação do programa. Mac usuários podem usar a Ferramenta de Importação PST para Entourage para importar dados PST em Entourage ou no PST para MBOX Converter para converter arquivos PST em arquivos .MBOX.
