O que é um arquivo .emf? (Metafile de Windows aprimorado)

Windows formato gráfico vetorial reconhecido por muitos aplicativos de edição de imagens; basicamente uma versão de 32 bits do formato metafile Windows original (formato RGB e não suporta dados CMYK.

Mais Informações

EMF também pode ser usado como um idioma gráfico para drivers de impressora.

Desenvolvedor Jasspa
Categoria Texto
Formato Texto 

O que é um arquivo .emf? (Arquivo Macro Jasspa MicroEmacs)

Arquivo macro contendo comandos para automatizar tarefas na versão Jasspa do MicroEmacs, um editor de texto extensível; usado para automatizar tarefas repetitivas que de outra forma seriam muito demoradas e monótonas para fazer à mão.

Mais Informações

Os usuários podem escrever macros personalizadas e criar novos comandos usando a extensa linguagem macro da Jasspa MicroEmac e salvá-las como arquivos EMF.

NOTA: Os arquivos EMF não são compatíveis com outras versões do MicroEmacs.
