O que é um arquivo .WMF? (Arquivo de mapa de Widelands)

Arquivo mapa usado pela Widelands, um jogo de estratégia onde os jogadores controlam tribos e constroem assentamentos; contém um mapa de jogo, que inclui o tamanho do mapa, terreno, recursos iniciais e locais de partida da tribo; pode ser usado para criar mapas personalizados.

O que é um arquivo .WMF? (metafilia Windows)

Formato gráfico Windows proprietário desenvolvido pela Microsoft; contém uma série de operações de desenho, incluindo comandos para linhas de desenho, círculos e retângulos; também pode conter operações de desenho bitmap; usado principalmente para imagens vetoriais, mas também pode incluir gráficos raster.

Mais Informações

Arquivos WMF apareceram originalmente em 1988 com o lançamento de Windows 20. Mais tarde, a Microsoft publicou a especificação do formato de arquivo WMF com a versão Windows 31 em 1992, e depois publicou uma especificação subsequente em 2006.

Os arquivos WMF foram largamente substituídos por formatos mais padrão, como imagens .JPG e .GIF. O formato WMF ainda é suportado por vários programas, incluindo o Windows Photo Viewer da Microsoft, que está incluído em Windows 7.
