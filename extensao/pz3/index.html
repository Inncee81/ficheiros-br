O que é um arquivo .PZ3? (Arquivo de cena poser)

Um arquivo PZ3 é um formato de cena 3D usado pelo Bondware Poser, um programa interativo de design de figuras 3D e animação. Contém informações sobre figuras, adereços, iluminação, ângulos da câmera e movimento incluídos em uma cena 3D.

Mais Informações

Captura de tela de um arquivo pz3 em Bondware Poser 113

Bondware Poser é um programa tridimensional de design de personagens que permite aos designers criar personagens e animações 3D realistas para fins de produção de arte digital, arquitetura, cinema e videogame. Você provavelmente só encontrará um arquivo PZ3 se você usar Renderosity Poser para criar cenas 3D. Você também pode ser enviado um arquivo PZ3 de outro usuário poser tentando compartilhar uma cena.

O formato de cena PZ3 só é suportado pelo software Poser. Para abrir um arquivo PZ3 no Poser, selecione File → Open. Para criar um arquivo PZ3 no Poser, selecione File → Save ou Save As....

Se você receber um arquivo PZ3 e não tiver acesso ao Poser, peça ao remetente para exportar o PZ3 em um formato que você possa abrir, como um arquivo .OBJ, .DXF ou .VRML. Para converter um arquivo PZ3 no Poser, selecione File → Export.

NOTA: Poser foi originalmente desenvolvido pela Smith Micro, mas foi vendido para a Bondware em junho de 2019.
