O que é um arquivo .JWL? (Arquivo do caso Roxio Jewel)

Arquivo de rótulo criado pela Easy CD Creator, um programa usado para criar e imprimir CDs; contém os locais de todos os recursos de modelo, tema, imagem e ícone associados a uma imagem de caixa de joias; usado para imprimir uma etiqueta de CD ou capa de caixa de joias.

Mais Informações

Os arquivos JWL são substituídos por formatos mais novos usados no software Roxio CD, mas podem ser usados pela versão 10 ou anterior do Easy Media Creator.
