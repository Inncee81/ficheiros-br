O que é um arquivo .17t? (Arquivo de retorno do StudioTax 2017)

Um arquivo 17T é uma declaração de imposto criada pela versão de 2017 do StudioTax, um programa de Windows e macOS usado para preparar declarações de imposto de renda pessoal no Canadá. Contém todas as informações fiscais federais e províncias canadenses necessárias e dados pessoais inseridos pelo usuário.

Captura de tela de um arquivo 17t em BHOK IT Consulting StudioTax 2017

Quando você salva sua declaração de imposto no StudioTax 2017, o arquivo 17T é criado para salvar seu progresso. É o principal tipo de arquivo usado pela versão 2017 do StudioTax e é usado principalmente para três fins:

  • Salvar uma declaração de imposto de 2017 a ser arquivada eletronicamente ou impressa para ser enviada.
  • Apoiar uma declaração de imposto em caso de perda ou corrupção de uma declaração de imposto de renda.
  • Importar dados fiscais de 2017 para a versão 2018 do StudioTax.

As prorrogações de arquivos de declaração de impostos usadas pelo StudioTax refletem o ano dos dados fiscais que contêm. As declarações fiscais do ano de 2016 utilizam a prorrogação .16t, as devoluções para o ano de 2015 utilizam a prorrogação .15t, e assim por diante.
