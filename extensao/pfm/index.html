O que é um arquivo .pfm? (Arquivo de métricas de fonte da impressora)

Fonte Tipo 1 usada pelo sistema operacional Microsoft Windows; inclui medições de caracteres dentro da fonte; requer um arquivo .PFB associado para a fonte; não contém os glifos de fonte reais, mas ajuda a fonte no arquivo PFB a ser exibida corretamente.

Mais Informações

Para instalar fontes PFM/PFB na Windows 7, você pode copiar os arquivos PFM e PFB para o diretório C:\Windows\Fonts\, clicar com o botão direito do mouse no arquivo PFM e selecionar "Instalar" na lista. Em seguida, você pode visualizar a fonte com o Microsoft Windows Font Viewer, que pode ser aberto navegando para o diretório de \Windows\Fonts\ no Windows Explorer ou digitando "font" na caixa de pesquisa do menu Windows Iniciar e, em seguida, clicando em "Exibir fontes instaladas" da lista de pesquisa preenchida.

NOTA: Se o arquivo PFB não estiver no mesmo diretório do arquivo PFM, ele pode estar localizado na pasta \PSFONTS\ e o arquivo PFM pode estar localizado na pasta \PSFONTS\PFM\.
