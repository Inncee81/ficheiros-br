O que é um arquivo .spk? (Arquivo de origem artspeak)

Arquivo de origem escrito no Artspeak, uma linguagem de computador usada para criar uma saída gráfica; contém declarações simples baseadas em texto que são referenciadas a formas de desenho, linhas, padrões, etc.

Mais Informações

Artspeak foi originalmente desenvolvido em 1970 por Jacob Schwartz. A língua só funcionava no CDC 6600, que era um supercomputador criado por Seymour Cray. Foi usado para produzir um design gráfico em uma folha de papel. A linguagem acabou desaparecendo, mas foi trazida de volta em 2012 por Ron Schnell. Agora pode ser usado para produzir um gráfico .PNG em vez de uma folha de papel.

NOTA: O arquivo SPK pode ser criado e carregado para exibir a saída gráfica usando o programa web Do Intérprete artspeak.

O que é um arquivo .spk? (Documento de faísca brilhante)

Arquivo criado pela Bright Spark, um aplicativo educacional usado para ensinar princípios eletrônicos; contém um ou mais projetos eletrônicos, bem como texto escrito que ensina o leitor sobre o design; salva as informações necessárias para apresentar a lição no software.

Mais Informações

Os usuários podem interagir com diagramas no software. Por exemplo, um usuário pode clicar em um interruptor que conecta um circuito e acende uma lâmpada. O software anima o novo fluxo de corrente do circuito e mostra o circuito recém-conectado.

NOTA: Os usuários podem navegar por uma lista de projetos em arquivos SPK usando os botões "Ir para o próximo tópico" e "Ir para tópicos anteriores" no aplicativo. Alguns arquivos SPK de amostra estão incluídos na instalação do software.

O que é um arquivo .spk? (Arquivo de mapa de caçador de veados)

Mapa de jogo usado por Deer Hunter, um jogo de caça virtual; inclui o terreno, plantas, locais de caça e animais; pode ser um mapa de jogo pré-instalado ou um mapa personalizado criado pelo jogador.

O que é um arquivo .spk? (Pacote de Sinatologia)

Pacote de instalação usado pelo DiskStation Manager (DSM), um sistema operacional desenvolvido pela Synology para uso com seus dispositivos NAS; pode conter programas ou outros componentes de software que estendem o software DSM base; podem ser instalados usando o Gerenciador de Pacotes da DSM, que está disponível com o firmware DSM 722 e posteriormente.

Mais Informações

Os arquivos SPK são salvos no formato .TAR. Portanto, você também pode abri-los com qualquer extrator de arquivo TAR.
