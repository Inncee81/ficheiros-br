O que é um arquivo .std? (Modelo de desenho do Apache OpenOffice)

Modelo criado pelo Apache OpenOffice Draw, um programa de desenho baseado em vetores; contém elementos de desenho padrão que podem ser usados como base para a criação de novos arquivos .SXD; permite que os usuários criem vários desenhos com uma aparência e sensação semelhantes.

Mais Informações

Arquivos STD também podem ser abertos pelo Open Office Draw, o sucessor do StarOffice que compartilha a mesma base de código que o Apache OpenOffice Draw.

NOTA: O Apache OpenOffice era anteriormente conhecido como OpenOfficeorg. Além disso, o Apache OpenOffice agora armazena gráficos vetoriais usando arquivos .OTG.

O que é um arquivo .std? (Documento de tradutor promt)

Documento criado e usado pelo PROMT Translator, uma ferramenta de tradução para o Microsoft Word e Outlook; contém informações sobre uma sessão de tradução, que inclui texto de origem e os idiomas de onde é traduzido; também inclui configurações de tradução e dicionários habilitados.

Mais Informações

NOTA: O tradutor promt está disponível para download como parte do pacote PROMT Office.
