O que é um arquivo .book? (Arquivo do livro BookSmart)

Um arquivo BOOK é um livro criado pela BookSmart, uma ferramenta gratuita usada para criar livros de fotos, cadernos e planejadores. Ele contém as páginas de um livro, que inclui texto, imagens e configurações de layout. Os arquivos BOOK são salvos em um formato XML.

Mais Informações

Captura de tela de um arquivo de livro em Blurb BookSmart 34

O arquivo BOOK é o principal tipo de arquivo associado ao BookSmart. Quando você salva um projeto de livro, o programa cria o arquivo BOOK para armazenar o conteúdo. Você pode escolher livros com diferentes layouts de página, como álbum de casamento, livro de receitas, livro de visitas, caderno e anuário. Você também pode escolher entre diferentes temas que incluem bordas de fotos e fundos de página com cores coordenadas. Se você quiser fazer backup ou compartilhar seu arquivo BOOK, você pode exportar o projeto como um arquivo .BOOKEXPORT.

Para criar um arquivo BOOK, selecione File → New → Book Project. Você pode salvar um arquivo BOOK selecionando File → Save. Depois de salvar o arquivo BOOK, o BookSmart salvará automaticamente o arquivo e quaisquer edições que você fizer daqui para frente.

O que é um arquivo .book? (Orçamento em breve arquivo de livro)

Arquivo usado pelo Budget In Brief, um programa usado para gerar documentos orçamentários; salvo usando formatação XML e contém propriedades de geração de documentos; também inclui referências a arquivos de dados de orçamento de entrada e modelos de documentos de saída; pode ser usado para gerar arquivos .DOCX e documentos de .PDF do Microsoft Word.

Mais Informações

Budget In Brief suporta vários tipos de documentos diferentes para a criação dos relatórios orçamentários de produção. Esses formatos incluem .DOC, .DOCX, .RTF, .HTML e .ODT. O programa também pode aceitar fontes de dados de entrada do Microsoft Excel .XLSX arquivos e arquivos .XML.

NOTA: Arquivos BOOK são atualmente criados usando um editor de texto simples, como o Microsoft Notepad, ou um editor XML. Budget In Brief não está mais disponível para download.

O que é um arquivo .book? (Arquivo do livro framemaker)

Arquivo de livros criado pelo FrameMaker, um programa usado para gerar livros e outros documentos grandes; armazena referências a documentos frameMaker (.FM e arquivos .MIF), que podem conter capítulos, apêndices, tabela de conteúdo e outros livros; permite que os usuários compilem um livro a partir de subpartas de documentos menores.

Mais Informações

NOTA: Quando você renomeia arquivos referenciados em um arquivo BOOK, o FrameMaker também renomeia os arquivos referenciados em disco. Isso ocorre porque os livros framemaker contêm links para os documentos framemaker reais no disco rígido do computador.

Desenvolvedor Apple
Categoria Dados
Formato Binário 

O que é um arquivo .book? (Documento autoral do iBooks)

Projeto de design de livros criado pelo iBooks Author, um programa usado para autoria de livros digitais para o Apple iPad; armazena as seções e capítulos de um iBook; pode ser exportado para o formato .IBOOKS e carregado diretamente para um iPad, ou pode ser publicado na Apple iBookstore usando o formato .ITMSP.

Mais Informações

iBook Os documentos do autor também podem ser exportados para formatos .PDF e .TXT (texto simples) para leitura no computador local ou em outros dispositivos.

NOTA: iBooks O autor usa a extensão de arquivo .IBA como extensão padrão para designs de iBooks. No entanto, ele também suporta a extensão "book" para projetos.
