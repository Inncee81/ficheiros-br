O que é um arquivo .sci? (Imagem gráfica ColoRIX)

Imagem bitmap usada pela ColoRIX, um aplicativo de pintura MS-DOC desenvolvido pela RIX Software nas décadas de 1980 e 1990; contém um gráfico bitmap que suporta 256 cores e resolução 320x200; também pode ser visto com a extensão .RIX.

O que é um arquivo .sci? (Roteiro scilab)

Um arquivo SCI é um script usado pelo Scilab, um programa de computação numérica de código aberto. Ele contém várias funções Scilab que podem ser executadas com o comando exec .

Mais Informações

Você pode criar o arquivo SCI com o editor scilab. Quando você executa os comandos no arquivo SCI, as funções são carregadas no ambiente Scilab, mas o programa não os executa. O arquivo SCI é semelhante ao arquivo .SCE, pois ambos são usados pela Scilab para conter vários comandos. No entanto, o arquivo SCE também pode conter declarações executáveis, não apenas funções Scilab.
