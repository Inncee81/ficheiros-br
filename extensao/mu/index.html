O que é um arquivo .mu? (Arquivo de malha do programa espacial Kerbal)

3D arquivo de imagem usado pelo Kerbal Space Program (KSP), um jogo que permite que os jogadores construam um programa espacial; usa o nome de arquivo model.mu e salva o ônibus espacial e outros dados de objetos; usa um formato proprietário suportado apenas pelo jogo.

Mais Informações

O arquivo MU é acompanhado pelo arquivo part.cfg que o faz referência como uma malha. Há também .MBM arquivos referenciados pelo arquivo MU que contêm as texturas para o modelo.

NOTA: Os modelos KSP são construídos usando o software de modelagem Unity.
