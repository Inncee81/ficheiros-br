O que é um arquivo .li3d? (Documento 3D do Interior Vivo)

Projeto de design de interiores criado pela Live Interior 3D, um programa 3D de design de casas e interiores para Mac; economiza plantas de piso, projetos de iluminação, móveis e outras informações de design de casa; pode ser renderizado para imagens de alta qualidade e panorâmicas.

Mais Informações

O software Live Interior 3D pode exportar visualizações interiores para formatos padrão raster, como .JPG, .TIFF e .PNG.

NOTA: Live Interior 3D foi sucedido pelo Live Home 3D. Os arquivos LI3D ainda são suportados pelo Live Home 3D.
