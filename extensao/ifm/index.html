O que é um arquivo .ifm? (Documento de dados de formulário informado)

Um arquivo IFM contém dados para um formulário criado pelo InformEr, um programa usado para gerenciar formulários eletrônicos que podem ser preenchidos e enviados. Ele armazena dados para um formulário preenchido que foi preenchido por um usuário. Os arquivos IFM são normalmente usados por organizações governamentais, empresas e universidades.

Mais Informações

Os arquivos IFM são projetados para armazenar os dados de um formulário (arquivo ITP) criado pelo Informed Designer. Os formulários podem ser preenchidos, aprovados, assinados e enviados eletronicamente usando o Preenchimento Informado. Alguns exemplos de formulários incluem notas fiscais, pedidos e preferências para o serviço militar.

Os dados do arquivo IFM são usados para preencher os registros apropriados no formulário ITP. Um arquivo IFM não pode armazenar dados para vários formulários, apenas um. Se você excluir o arquivo IFM, os dados no formulário serão apagados.

NOTA: Designer e Enchimento informados foram usados com destaque na década de 1990, mas desde então foram descontinuados.
