O que é um arquivo .crash? (Mac arquivo de registro de acidentes do OS X)

Arquivo Log criado pelo CrashReporter, um programa de Mac OS X incorporado que registra dados de falhas do programa; armazena informações sobre o estado do computador no momento de um acidente; inclui informações de timetamp, programa e despejo de memória; usado para relatar e depurar problemas com aplicativos.

Mais Informações

ARQUIVOS CRASH podem ser enviados à Apple selecionando a opção Report. após um acidente. Eles também podem ser vistos usando o aplicativo Mac O CONSOLE OS X. No Mac aplicativo OS X 106 Console, os arquivos CRASH podem ser encontrados navegando até o seguinte local: Log List → DIAGNOSTIC INFORMATION → User Diagnostic Reports.

OS arquivos CRASH usam a convenção de nomeação "[program]_[date]_[number]_[machine name].crash" Por exemplo, um acidente do Apple Safari pode ser chamado Safari_2009-01-01_11223_my-iMac.crash.

Os arquivos de falha são salvos em um formato de texto simples e são armazenados no diretório a seguir:
/Library/Logs/DiagnosticReports

>Eles também têm um arquivo de atalho neste diretório:
/Library/Logs/CrashReporter
