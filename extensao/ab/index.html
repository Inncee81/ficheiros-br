O que é um arquivo .ab? (Arquivo Ambling BookPlayer MP3)

Arquivo de áudio usado pelo Ambling BookPlayer, um reprodudor de áudio para telefones celulares baseados no Android; adiciona a extensão "ab" a um arquivo de .MP3 normal (ou outro arquivo de áudio) e faz a extensão do composto "abmp3;" usados para que os arquivos Ambling BookPlayer baixados não sejam misturados com outros arquivos de áudio reproduzidos por outros music players.

Mais Informações

Ambling BookPlayer tem uma preferência chamada "Adicionar extensão ab" que controla se a extensão "ab" é adicionada ou não a arquivos de áudio. Você pode desmarcar esta opção para desativar a renomeação da extensão do arquivo.

NOTA: O Ambling BookPlayer foi descontinuado.

O que é um arquivo .ab? (Android arquivo de backup ADB)

Arquivo de backup criado pela adb (Android Debug Bridge), uma ferramenta de linha de comando incluída no Android SDK; pode incluir os dados de aplicativos, sistema e usuário de um dispositivo de 40 ou mais Android; usado para salvar e restaurar instantâneos de dispositivos.

Mais Informações

Muitas vezes os backups AB não incluem os aplicativos .APK em si, mas apenas os dados de suporte ao aplicativo e sistema do dispositivo. Por exemplo, você pode fazer backup dos dados de um dispositivo Android conectado sem os arquivos APK digitando o seguinte comando:

adb backup -all

>Isso cria o arquivo backup.ab no diretório atual. Para incluir os arquivos APK também, use o seguinte comando:

adb backup -apk -all

Para restaurar um arquivo de backup AB no dispositivo, digite o seguinte comando:

adb restore backup.ab

O programa adb também inclui opções para incluir ou excluir outros tipos de dados e especificar o nome do arquivo de backup. Consulte a documentação do ADB para obter mais informações.

O que é um arquivo .ab? (Arquivo de sequência de DNA)

Formato de dados científicos usado para armazenar informações de sequência de DNA; pode ser visto graficamente com um visualizador de cromatógrafo, que permite a detecção rápida de mutações em sequências de DNA e outras análises de DNA.

Mais Informações

A maioria dos arquivos de sequência de DNA usa a extensão .ABI.
