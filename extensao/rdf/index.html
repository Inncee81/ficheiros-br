O que é um arquivo .rdf? (Modelo ReDIF)

Um arquivo RDF é um modelo salvo no Formato de Informações de Documentos de Pesquisa (ReDIF). Contém metadados que descrevem uma ou mais publicações de pesquisa, como revistas, artigos, livros e software. Arquivos RDF armazenam informações sobre a publicação, como o autor, e-mail para contato com o autor, título da publicação, editor, data publicada e local de trabalho do autor da publicação.

Mais Informações

RDF arquivos são usados pela Research Papers in Economics (RePEc), um grupo dedicado a melhorar a disseminação de pesquisas em Economia e outras ciências relacionadas. A RePEc usa arquivos RDF e .REDIF para armazenar metadados sobre publicações, o que permite que as publicações sejam mais facilmente organizadas e acessadas. Os arquivos RDF são usados para conter metadados com codificação de caracteres ISO-LATIN-1 ou Windows-1252, enquanto os arquivos REDIF são usados para armazenar metadados com codificação de caracteres UTF-8.

Os arquivos RDF são tipicamente criados com software ReDIF, mas uma vez armazenados em texto simples podem ser abertos e editados por qualquer editor de texto. Os metadados armazenados no arquivo RDF são organizados em linhas separadas de field_name: valor de campo. Um exemplo de nome e valor de campo é Autor-Local de Trabalho-Nome: Universidade de Minnesota.

O que é um arquivo .rdf? (Arquivo de estrutura de descrição de recursos)

Um arquivo RDF é um documento escrito na linguagem RDF (Resource Description Framework, framework de descrição de recursos), que é usado para representar informações sobre recursos na web. Contém informações sobre um site em um formato estruturado chamado metadados. Os arquivos RDF podem incluir um mapa do site, um registro de atualizações, descrições de páginas e palavras-chave.

Mais Informações

Os arquivos RDF também são usados pela Mozilla Web navegadores, como o Firefox, para armazenar configurações para configurações de janelas e barras de ferramentas.
