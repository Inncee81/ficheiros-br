O que é um arquivo .SPT? (Arquivo treinador AQW)

"Autoer" usado pelo Hell Quest, um programa de treinador usado para nivelar automaticamente personagens no jogo MMORPG Adventure Quest Worlds (AQW); salva instruções para a entrada do usuário; usado para carregar diferentes perfis de treinador para diferentes tarefas de jogo.

Mais Informações

Os programas de treinadores são usados para gerar automaticamente a entrada do jogador em jogos sem que o jogador humano tenha que estar presente no computador. Esta técnica é às vezes usada para jogos em que o tempo é necessário para aumentar o nível dos personagens do jogo. No entanto, os treinadores são geralmente desencorajados e são contra a maioria das políticas de contrato de licença de usuário final (EULA). Nem os arquivos Hell Quest nem SPT são suportados pela Artix Entertainment, os criadores de Adventure Quest Worlds.

NOTA: Programas diferentes do Hell Quest também podem abrir arquivos SPT, incluindo Cross Hell, um aplicativo Flash (.FLA).

O que é um arquivo .SPT? (Script semicondutor cypress)

Script criado pelo Cypress USB Console, um programa usado para criar scripts que podem baixar automaticamente firmware para dispositivos compatíveis com USB; inclui informações de configuração e referências a binários de firmware, bem como instruções para carregar o firmware no dispositivo.

Mais Informações

Usando o Cypress USB Console, os desenvolvedores podem criar scripts SPT gravando suas ações enquanto clicam na interface do usuário. O script gravado pode ser executado para carregar automaticamente o firmware no dispositivo compatível com USB.

NOTA: O Cypress USB Console é um componente do software Cypress' CyConsole, que muitas vezes é incluído com aplicativos cypress Development Kit.

O que é um arquivo .SPT? (Arquivo de dados da árvore SpeedTree)

Contém dados de árvores processuais usados pelo SpeedTreeCAD, um programa de modelagem de árvores; inclui informações necessárias para a construção da geometria da árvore, bem como dados para implementação de LOD, animação eólica e clonagem.

Mais Informações

Arquivos SPT também podem ser lidos por outros programas de modelagem 3D que têm os plugins SpeedTreeMAX ou SpeedTreeMAYA instalados.

O que é um arquivo .SPT? (Arquivo compactado do empacotador de arquivos TM)

Arquivo compactado ou grupo de arquivos criados com o TM File Packer, um programa de compactação de arquivos desenvolvido pela TMAgency; pode ser usado para dividir, compactar e descomprimir arquivos.

Mais Informações

NOTA: A TMAgency é uma patrocinadora paga por download. TM File Packer é um programa de adware, portanto, o uso do software não é recomendado.
