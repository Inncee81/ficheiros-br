O que é um arquivo .int? (Roteiro compilado por Fallout)

Roteiro compilado escrito para Fallout 1 & 2 para animar o ambiente Fallout; armazena a versão compilada de um arquivo .SSL, que dita o comportamento dos objetos nos mapas, a existência do personagem NPC, o diálogo entre personagens e outros elementos de jogabilidade.

Mais Informações

Um arquivo SSL deve ser compilado em um arquivo INT para que o jogo o use. Você pode usar o Fallout Script Editor para compilar arquivos SSL. Depois de compilar o script, o arquivo INT terá o mesmo nome do SSL de que é compilado, por exemplo, example.ssl se torna example.int.

Para usar um arquivo INT, ele deve ser colocado no diretório \data\scripts do seu diretório de jogos.

NOTA: Se você quiser editar um arquivo INT, abra o arquivo SSL correspondente, faça suas alterações e, em seguida, compile-o novamente.

O que é um arquivo .int? (Arquivo ROM do jogo intellivision)

Um arquivo INT é um arquivo ROM de jogo usado por emuladores da Intellivision, como MESS, Nostalgia e Bliss. Ele contém uma cópia exata dos dados do jogo do cartucho original do jogo, que é usado por um emulador para jogar o jogo.

Mais Informações

Os emuladores da Intellivision são usados para jogar jogos da Intellivision como se fossem executados no sistema original. Os dados do jogo do cartucho original do jogo são despejados em um arquivo INT, que é usado para armazenar o jogo em um formato que pode ser jogado por um emulador da Intellivision.

Intellivision é um console de videogame lançado em 1979 pela Mattel Electronics como concorrente do Atari 2600. Foi eventualmente descontinuado em 1990. Alguns títulos de jogos da Intellivision incluem Boxing, Auto Racing, Armor Battle, Night Stalker, Sub Hunt e Tron: Solar Sailer.

O que é um arquivo .int? (Imagem SGI Integer)

Formato básico de imagem em preto e branco (monocromático) desenvolvido pela Silicon Graphics; armazena dados de imagem usando inteiros que representam o conteúdo de cada pixel.

Mais Informações

Color SGI images frequentemente usam a extensão de arquivo .SGI.

O que é um arquivo .int? (Arquivo de gerenciamento de imagem da filmadora sony)

Arquivo criado para filmadoras digitais da Sony; contém metadados que ajudam a gerenciar vídeos e imagens gravados; localizados na pasta AVF_INFO juntamente com .BNP e .INP arquivos; não devem ser modificados.

Mais Informações

Não exclua este arquivo, se você fizer isso, você não será capaz de gravar ou reproduzir vídeos ou imagens corretamente. Os arquivos INT são padrão como arquivos ocultos e geralmente não são exibidos.

O que é um arquivo .int? (Arquivo de localização internacional irreal)

Arquivo usado pelo Unreal Engine para desenvolver jogos como Rune e Unreal Tournament; salvo no mesmo formato que .INI arquivos de configuração; contém dados que referências irreales para exibir texto, como mensagens ou diálogos, no idioma correto; acompanha um pacote Irreal.

Mais Informações

Arquivos de localização irreal são armazenados em subpastas específicas do idioma do diretório Engine\Localization\ de um projeto. Diferentes extensões são atribuídas para diferentes idiomas:

  • int - Internacional (Inglês)
  • det - Alemão
  • est - Espanhol
  • frt - Francês
  • itt - Italiano
  • kot - Coreano
  • tct - Tcheco
  • rut - Russo
