O que é um arquivo .db? (Arquivo de banco de dados)

Um arquivo DB é um arquivo de banco de dados genérico que armazena dados em um formato estruturado, normalmente com um arranjo de tabelas, campos de tabela, tipos de dados de campo e valores de campo. Ele pode ser criado por vários programas de banco de dados e exportado para diferentes formatos, como .CSV arquivos. Os arquivos DB também podem ser importados por vários programas de banco de dados e são usados para arquivos de jogos também.

O que é um arquivo .db? (Arquivo de banco de dados de dispositivos móveis)

Um arquivo DB é um arquivo de banco de dados usado em dispositivos móveis como Android, iOS e Windows telefones celulares Phone 7. Muitas vezes é usado para armazenar contatos e informações de SMS, mas pode armazenar qualquer tipo de dados de dispositivo ou aplicativo. Os arquivos DB são comumente armazenados em um formato de banco de dados SQLite, mas também podem ser bloqueados ou criptografados para que o usuário não possa ver os dados diretamente.

Mais Informações

Arquivos DB em telefones celulares geralmente não devem ser abertos ou editados manualmente, uma vez que são arquivos de suporte que incluem informações para o dispositivo ou para um aplicativo.

Um exemplo de um arquivo DB é sms.db, um arquivo de suporte iOS que armazena as mensagens de texto do usuário (localizadas no diretório de /private/var/mobile/Library/SMS/ no dispositivo). Outro exemplo de um arquivo DB é consolidated.db, um arquivo de banco de dados em iOS dispositivos que rastreia automaticamente as informações de localização.

O que é um arquivo .db? (Arquivo de banco de dados de contas QQ)

Arquivo de dados criado pela QQ, um programa de bate-papo on-line chinês; pode armazenar a lista de contatos ou o registro de sessões de bate-papo on-line para uma conta de usuário; usado para armazenar dados de aplicativos localmente e carregá-los através da interface do programa.

Mais Informações

Os arquivos QQ DB são normalmente salvos no diretório \Program Files\Tencent\QQ\[Account number]\.

O que é um arquivo .db? (Arquivo de histórico de bate-papo do Skype)

Arquivo de histórico de bate-papo criado e usado pelo Skype, um aplicativo que permite fazer chamadas telefônicas, mensagens e chats de vídeo; contém seu histórico de conversas, que inclui todas as mensagens instantâneas e detalhes de chamada que um usuário enviou e recebeu de contatos.

Mais Informações

NOTA: Você pode transferir seu histórico de bate-papo do Skype para outro dispositivo substituindo o arquivo main.db no novo dispositivo pelo arquivo main.db que contém o histórico de bate-papo que você deseja manter.

O que é um arquivo .db? (Cache de miniaturas Windows)

Cache de imagens em miniatura que representam imagens maiores em uma pasta Windows; armazena pequenas versões de imagens armazenadas em uma pasta para navegação mais rápida; normalmente não é vista a menos que a opção "Mostrar arquivos invisíveis" seja verificada dentro das opções de exibição Windows.

Mais Informações

Windows arquivos do Cache em miniatura são nomeados Thumbs.db por padrão. Para obter mais informações sobre Windows arquivos Thumbsdb, incluindo como evitar que eles sejam criados, siga estas instruções.
