O que é um arquivo .rbf? (LEGO MINDSTORMS EV3 Arquivo de tijolos robô)

Um arquivo RBF é um arquivo executável usado pelo EV3 Intelligent Brick, que é um dispositivo usado para controlar robôs LEGO MINDSTORMS EV3. Ele armazena um programa EV3 Basic, que é usado para fazer um robô executar uma função. Os arquivos RBF são compilados pelo compilador embutido no EV3 Explorer e executados diretamente no EV3 Intelligent Brick.

Mais Informações

EV3 Os programas básicos são usados para fazer com que os robôs LEGO MINDSTORMS EV3 executem uma variedade de tarefas, seja movendo uma determinada direção ou pegando um objeto. Esses programas podem ser executados no modo PC e no modo tijolo. No modo PC, os programas são executados no Microsoft Small Basic, enquanto retransmite informações para o EV3 Intelligent Brick para operar o robô.

No entanto, no modo tijolo, os programas são executados no EV3 Intelligent Brick, que é um dispositivo portátil que possui um sistema operacional baseado em Linux e é usado para interagir com os robôs. Os arquivos RBF são usados para executar programas no EV3 Intelligent Brick.

Os arquivos RBF são compilados a partir de arquivos .SB, que são arquivos que armazenam o código do programa Small Basic, do EV3 Explorer, que é um programa Windows usado para visualizar e organizar arquivos armazenados no EV3 Intelligent Brick. O EV3 Explorer também pode ser usado para baixar arquivos RBF, .RSF e .RGF para o EV3 Intelligent Brick.

O que é um arquivo .rbf? (Arquivo Binário Redatam)

Arquivo criado pela Retadam (Recuperação de Dados para Pequenas Áreas pela Microcomputador), um software de mapeamento usado para analisar dados de censos e pesquisas locais e regionais; salva dados como registros estruturados em formato binário comprimido; usado para analisar áreas geográficas do tamanho de alguns blocos urbanos para um país inteiro.

Mais Informações

Os arquivos da RBF podem armazenar dados censitários, como sexo, idade e informações domiciliares. Os dados são armazenados na forma de "entidades" (uma hierarquia lógica de objetos) e "variáveis" (informações sobre entidades), que são definidas ainda pela documentação retadam.

NOTA: O software Retadam é apoiado pela Comissão Econômica para a América Latina e o Caribe (CELADE).

O que é um arquivo .rbf? (Arquivo de backup retrospecto)

Conjunto de backup contendo dados com backup de dados com software de backup Retrospect; contém arquivos e pastas reais apoiados pelo Retrospect; pode ser restaurado com base nas informações salvas no arquivo de catálogo de backup .RBC.

Mais Informações

Retrospect foi desenvolvido anteriormente pela Dantz software. O programa agora é de propriedade e distribuído pela EMC Insignia.

O que é um arquivo .rbf? (Imagem de backup do Insta Estelar)

Imagem de backup criada pelo Insta Backup Gold, um programa de Windows usado para backup e restauração de arquivos; contém os dados de backup de um banco de dados de arquivo, pasta, volume ou e-mail.

Mais Informações

Há uma série de configurações que o Insta Backup Gold fornece para criar um arquivo de backup RBF. Você pode proteger seu backup com uma senha, selecionar o nível de compressão do backup e agendar backups a serem criados posteriormente e em uma frequência personalizada. Além disso, você pode escolher se quiser um backup completo, incremental ou diferenciado.

NOTA: O Stellar Insta Backup Gold foi descontinuado.

Arquivo de reversão do instalador Windows

Desenvolvedor Microsoft
Categoria Cópia de Segurança
Formato Não Disponível 

O que é um arquivo .rbf? (Arquivo de reversão do instalador Windows)

Arquivo de arquivos criados durante um processo de instalação de software executado pelo Windows Installer; contém arquivos que podem ser restaurados se a instalação não for concluída com sucesso; se a instalação for bem sucedida, o arquivo de reversão será automaticamente excluído.

Mais Informações

O arquivo RBF é salvo juntamente com um arquivo de script .RBS correspondente e armazenado em um arquivo chamado "ConfigMsi"
