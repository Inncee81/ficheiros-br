O que é um arquivo .qvf? (Arquivo do aplicativo Qlik Sense Desktop)

Um arquivo QVF é um aplicativo criado pelo Qlik Sense Desktop, um programa de análise usado para criar relatórios personalizados e interativos e visualizações de dados. Ele contém uma ou mais visualizações de dados, como gráficos, mapas, tabelas e parcelas. Os arquivos QVF também incluem configurações que configuram a aparência das visualizações.

Captura de tela de um arquivo qvf em Qlik Sense Desktop 32

Os arquivos QVF são usados para armazenar visualizações de dados, como números de vendas ou informações pessoais. No entanto, provavelmente você só verá o arquivo QVF real ao compartilhar seu aplicativo de visualização de dados com outro usuário do Qlik Sense Desktop. Eles são frequentemente compartilhados como um anexo de e-mail ou carregados na nuvem, mas é importante notar que os arquivos QVF geralmente se tornam grandes em tamanho, tornando-os difíceis de anexar a um e-mail.

Ao criar um arquivo QVF, você importará dados de um ou mais arquivos ou conectará uma fonte de dados. Se você compartilhar o arquivo QVF, o aplicativo não pode ser recarregado a menos que o outro usuário tenha acesso à fonte de dados também.

Os arquivos QVF são tipicamente localizados no diretório a seguir:

C:\Users\[userid]\Documents\Qlik\Sense\Apps
