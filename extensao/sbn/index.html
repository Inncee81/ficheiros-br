O que é um arquivo .sbn? (Arquivo de firmware de telefone IP cisco)

Arquivo Firmware, ou "image" arquivo, usado por telefones Cisco IP (Internet Protocol) ; contém dados que controlam o funcionamento do telefone; usado durante o processo de atualização do firmware do telefone IP; geralmente distribuídos com alguns outros arquivos em um pacote de firmware.

Mais Informações

Os dados nos arquivos SBN geralmente estão em conformidade com o padrão IETF (Internet Engineering Task Force) (Session Engineering Task Force). Muitas vezes, os arquivos SBN são usados ao atualizar um telefone IP do Skinny Call Control Protocol (SCCP) para o protocolo SIP.

O firmware em telefones IP Cisco pode ser gerenciado usando o software Unified Communications Manager, que antes era conhecido como CallManager.

NOTA: As imagens do Cisco IP phone SBN são "assinadas", pela Cisco, para que a fonte emissora possa ser validada e o arquivo não possa ser facilmente adulterado antes de instalar no telefone. As imagens não assinadas são distribuídas com a extensão "bin".

O que é um arquivo .sbn? (Arquivo binário espacial ESRI)

Arquivo usado pelo software de mapeamento espacial ESRI ArcGIS; armazena uma representação binária das informações de forma armazenadas em um arquivo .SHP; salvo com um arquivo de índice .SBX, que juntos compõem o índice de forma; contém informações que aceleram consultas espaciais em arquivos de forma.

Mais Informações

Arquivos SBN armazenam informações de forma em muitos retângulos menores. Cada retângulo tem suas próprias propriedades que podem ser combinadas com critérios de consulta espacial. As informações de forma dividida e o formato binário permitem consultas mais rápidas. No entanto, uma vez que todos os dados de forma são armazenados em um arquivo SHP separado, os arquivos SBN não são necessários para consulta e análise de informações de forma.

NOTA: O formato SBN é usado apenas pelo software ESRI.

O que é um arquivo .sbn? (Arquivo de dados binário GT-31)

Arquivo de dados salvo por um gt-31 ou BGT-31 dispositivo de navegação GPS ao ar livre e impermeável; contém dados de posição e velocidade registrados durante a navegação com o dispositivo; usado para registrar caminhos de viagem e revisá-los mais tarde.

Mais Informações

Os arquivos SBN são salvos no cartão SD no cartão GT-31. Eles são armazenados em um formato binário.

NOTA: A sigla SBN significa "SiRF binary logging (não-NMEA)" O SiRF é um tipo de chip de dispositivo GPS.
