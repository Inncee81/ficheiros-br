O que é um arquivo .dtd? (Arquivo de definição do tipo de documento)

Um arquivo DTD é um arquivo de texto que especifica elementos e atributos usados em um documento XML, SGML, HTML ou XHTML. Também pode definir o pedido e o aninhamento de elementos e seus atributos. Os arquivos DTD são comumente referenciados dentro de páginas da Web usando uma URL.

Mais Informações

Arquivos DTD também são usados para validar se um arquivo representa um documento válido. Por exemplo, antes de analisar um arquivo XML, um programa pode primeiro validar as declarações de marcação usando um arquivo DTD para garantir que o XML possa ser analisado corretamente.
