O que é um arquivo .DLX? (Documento DIALux)

Arquivo de projeto de projeto de iluminação criado pela DIALux, um aplicativo gratuito de planejamento de luz profissional; armazena um plano de iluminação interna ou externa; pode incluir luminárias (unidades de luz elétrica), plantas de piso, layouts de ruas e outros fatores de iluminação ambiental.

Mais Informações

Os projetos DA DIALux podem importar dados de luminárias de arquivos .LDT, que usam o formato de luminária Elumdat.

O que é um arquivo .DLX? (Arquivo DLL renomeado)

Arquivo DLL (Dynamic Link Library, biblioteca de links dinâmicos) que foi renomeado de "dll" para "dlx;" pode ser usado para fazer backup .DLL arquivos ao substituí-los ou para renomear temporariamente arquivos DLL ao reparar instalações de programas de software.

Mais Informações

NOTA: Programas de desenvolvimento de aplicativos como o Microsoft Visual Studio podem ser usados para compilar os arquivos DLL originais.

O que é um arquivo .DLX? (Arquivo de vídeo Sony VDU)

Arquivo de vídeo gravado por alguns dispositivos VDU (Sony Video Disk Unit), incluindo o Sony VDU DSR-DU1; contém dados de vídeo digital (DV) capturados pela filmadora Sony; usado para arquivar vídeo, bem como importar vídeo gravado em ferramentas de edição, como o Adobe Premiere Pro.
