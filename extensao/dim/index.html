O que é um arquivo .dim? (Active@ arquivo de backup de imagem em disco bruto)

Backup de imagem em disco bruto criado pela Active@ Disk Image, um programa de backup e recuperação de Windows; armazena uma imagem em disco bruto, que inclui dados de configuração que descrevem a geometria do disco ou da partição.

Mais Informações

Quando você cria uma imagem de disco bruto com Active@ você cria uma cópia idêntica de cada setor a partir de um disco rígido físico selecionado ou partição lógica, que inclui todo o espaço usado e não utilizado. A imagem de disco bruto DIM é usada principalmente para recuperação futura de dados ou backup de sistemas operacionais não Windows.

Você pode criar uma imagem bruta selecionando "Criar imagem bruta" em "Operações de disco" ou "Ferramentas" na barra de menu. Dois arquivos são criados quando você cria uma imagem de disco bruto. Um arquivo DIM contendo a descrição da imagem é criado enquanto outro arquivo é criado para conter os dados. Este arquivo de dados tem o nome da imagem do disco bruto e tem "001" anexado à sua extensão "dim". Se sua imagem de disco bruto for nomeada "DiskImage", o arquivo de dados será nomeado DiskImage.dim.001. Se for necessário mais de um arquivo de dados, outro arquivo será criado com "002" anexado à extensão.

O que é um arquivo .dim? (Arquivo DIME)

Formato de mensagem binária usado para enviar dados pela Internet; pode encapsular uma ou mais cargas de vários tipos e tamanhos em uma única mensagem; contém comprimento de carga, tipo de carga útil e um identificador de carga útil.

Mais Informações

Web servidores e servidores de e-mail que reconhecem mensagens DIME traduzirão o arquivo encapsulado no anexo original para que ele possa ser aberto pelo usuário final.
