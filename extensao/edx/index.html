O que é um arquivo .edx? (Edraw XML FIle)

Um arquivo EDX é um diagrama criado por Edraw Max, um aplicativo de diagrama profissional. Ele contém um diagrama, que inclui formas de gráfico, setas e texto. Os arquivos EDX foram substituídos por arquivos .EDDX, que também são usados para armazenar diagramas, mas são menores em tamanho.

Mais Informações

Algumas categorias de modelos incluem Engenharia, Planos de Piso, Mapas, Software, diagrama de Web e Rede. Em cada uma dessas categorias, você pode escolher o modelo específico que deseja usar.

Para criar um arquivo EDX, você deve primeiro abrir um modelo que você gostaria de usar. Você pode fazer isso abrindo edraw Max e, em seguida, sob a guia New à esquerda, você pode escolher entre as categorias "Categorias de modelo," e escolher um dos "Modelos" Depois de abrir seu modelo, vá para File → Save As.... Depois de abrir o programa Edraw Max, você pode realmente abrir diferentes arquivos EDX e eles serão exibidos como guias no programa.

NOTA: A versão compactada deste arquivo é a extensão .EDXZ.
