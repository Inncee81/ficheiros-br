O que é um arquivo .dgn? (Arquivo de design de microestação)

Um arquivo DGN é um desenho 2D/3D criado por vários softwares CAD de construção, como MicroStation e Intergraph Interactive Graphics Design System. Ele pode ser salvo em um dos dois formatos, seja o Intergraph Standard File Format (ISFF) ou o mais novo padrão V8 DGN. Os arquivos DGN são normalmente usados para salvar projetos para projetos de construção de rodovias, pontes e edifícios.

Mais Informações

DGN os arquivos podem ser abertos pela Bentley Systems MicroStation e pelo programa gratuito Bentley View. O formato DGN também é suportado por vários outros programas de terceiros, como Autodesk AutoCAD e IMSI TurboCAD. Os arquivos DGN podem ser convertidos para o formato de .DWG mais universal pelo AutoCAD LDT.

NOTA: A extensão do DGN é abreviação de "design"
