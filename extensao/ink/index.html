O que é um arquivo .ink? (Imagem inkML)

Desenho de caneta digital salvo usando o Inke Markup Language (InkML), um formato de arquivo XML usado para salvar imagens e notas digitalmente desenhadas; contém um gráfico vetorial armazenado como uma ou mais sequências de pontos conectados.

Mais Informações

Os arquivos INK são usados para armazenar traços de caneta e redesenhá-los usando funções de rastreamento. Eles podem ser redimensionados sem perder a qualidade das curvas originais de traçado da caneta, uma vez que as funções de rastreamento interpolam entre os pontos.

O que é um arquivo .ink? (Arquivo de dados de tinta mimio)

"O formato "quadro eletrônico", salvo pelo Mimio, um programa que permite ao usuário desenhar em uma tela de computador tablet usando uma caneta digital; grava notas, diagramas, gráficos e ilustrações desenhadas no quadro branco no modo Digital Ink Recording; salva dados em um formato vetorial baseado em objeto.

Mais Informações

Os arquivos INK suportam a edição completa, incluindo dimensionamento e rotação de objetos e alterando a espessura de cores ou linhas de formas. Páginas e objetos dentro de páginas também podem ser reordenados. Os arquivos INK também suportam copiar e colar dados selecionados para outros aplicativos.

O que é um arquivo .ink? (Arquivo de referência Pantone)

Contém informações de cores relacionadas a um arquivo de imagem específico; salvo em um formato padrão definido pela Pantone, Inc; deve ser armazenado no mesmo diretório do arquivo de imagem correspondente.
