O que é um arquivo .adame? (Arquivo criptografado do Adame Ransomware)

Um arquivo com uma extensão .adame é um arquivo que foi criptografado pelo vírus Adame, que é uma variante do ransomware Phobos que se tornou predominante no início de 2019. Ele é criptografado para que não seja possível abrir o arquivo simplesmente alterando a extensão de arquivo .adame.

Mais Informações

Adame é um tipo de malware utilizado por cibercriminosos que toma os arquivos de um usuário como reféns e o obriga a pagar o criminoso para desbloquear os arquivos. Na maioria das vezes é introduzido no computador de uma vítima através de e-mails de spam com anexos de arquivos maliciosos que contêm macros e são baixados e executados por usuários desavisados. O vírus também pode ser introduzido quando um usuário clica em anúncios on-line maliciosos ou baixa arquivos maliciosos de sites de torrent.

Quando o ransomware é executado no computador de um usuário, ele criptografa arquivos no computador e adiciona a extensão .adame aos nomes dos arquivos. Os arquivos alvo são tipicamente documentos, imagens, vídeos e arquivos de backup, como janela pop-up explicando a aquisição hostil dos arquivos do usuário. O arquivo info.txt é a versão de texto simples do programa HTML. As informações nos dois arquivos incluem uma explicação sobre o que ocorreu no computador, a criptografia usada nos arquivos e como o usuário pode recuperar seus arquivos pagando um pagamento de resgate.

Atualmente, existem várias opções para remover o vírus Adame, que inclui backup de seus arquivos, ele pode realizar uma restauração do sistema para remover o vírus, mas quaisquer alterações feitas nos arquivos após o backup ser feito serão perdidas.
