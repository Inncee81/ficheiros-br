O que é um arquivo .cps? (Arquivo Estilos cativante)

Um arquivo CPS é um arquivo de estilos criado e usado pelo Adobe Captivate, um programa de criação de conteúdo eLearning. Ele contém um estilo para um objeto, que consiste em atributos visuais, como fonte ou cor.

Mais Informações

Os arquivos CPS permitem exportar e importar estilos personalizados para objetos de um projeto Captivate para outro. O aplicativo vem com estilos predefinidos, mas eles podem ser modificados. Além disso, objetos em um projeto herdam automaticamente os estilos definidos no tema do projeto, mas as teses também podem ser modificadas.

Para exportar um arquivo CPS, selecione Edit → Object Style Manager... e clique em Export..

Para importar um arquivo CPS, selecione Edit → Object Style Manager... e clique em Import..

O que é um arquivo .cps? (Arquivo corel photo house)

Raster image file salvo pela Corel Photo House, um programa de edição de imagens mais antigo semelhante ao Corel Photo Paint; normalmente usado para alterar e adicionar efeitos às fotos digitais.

Mais Informações

NOTA: Tanto a Photo House quanto a Photo Paint foram substituídas pela Paint Shop Pro.

O que é um arquivo .cps? (O arquivo de salvamento de brinquedos em pó)

Um arquivo CPS contém uma simulação salva criada pelo The Powder Toy, um jogo de física 2D usado para simular partículas, pressão de ar, gravidade e calor. Armazena os dados para a simulação em formato compactado e pode ser distribuído a outros para compartilhar simulações.

Mais Informações

Captura de tela de um arquivo cps em hardWIRED The Powder Toy 925

O Brinquedo em Pó permite que você brinque com diferentes substâncias para simular os efeitos de diferentes interações imobiliárias. O jogo vem com uma variedade de líquidos, gases, componentes eletrônicos e materiais de construção que você pode usar para construir terrenos, explosivos, edifícios e muito mais. Durante a jogabilidade você pode optar por salvar a simulação, que armazenará a simulação nesse momento, que pode ser carregada mais tarde.

Para salvar uma simulação e criar um arquivo CPS, clique no ícone de arquivo e seta na barra de ferramentas inferior e nomeie o arquivo. Para abrir uma simulação armazenada em um arquivo CPS, clique no ícone de arquivo e seta no lado esquerdo da barra de ferramentas inferior e escolha o arquivo CPS que deseja abrir.
