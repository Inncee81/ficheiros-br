O que é um arquivo .fpt? (Arquivo de memorando de tabela Alfa Cinco)

Arquivo de memorando criado pela Alpha Five, um IDE de desktop, web e aplicativo móvel; contém quaisquer campos de memorando que são criados no arquivo .DBF; só existe quando um campo de memorando é definido em um arquivo DBF; vinculado a um arquivo DBF, quaisquer alterações feitas no arquivo DBF alterarão o arquivo FPT.

Mais Informações

O arquivo FPT é criado pela Alpha Five porque as tabelas em arquivos DBF têm limitações de largura sobre quantos caracteres podem estar em uma coluna. O formato de memorando em arquivos FPT não é limitado, eles podem ter qualquer comprimento alfanumérico (uma sequência de dados com números e letras). Assim, o Alpha Five cria um campo de dez caracteres no arquivo DBF que faz referência ao local do memorando no arquivo FPT vinculado que contém todos os dados para esse registro na tabela.

NOTA: Em 2013, a Alpha Five tornou-se Alpha Anywhere.

O que é um arquivo .fpt? (Arquivo de memorando do Banco de dados FileMaker Pro)

Arquivo de banco de dados criado pelo FileMaker Pro, um programa de banco de dados relacional; contém informações textuais representando um "memo;" usado para descrever um banco de dados com texto bruto em vez de usar um campo de banco de dados padrão, que normalmente é limitado a menos número de caracteres.

Mais Informações

Arquivos FPT não contêm as informações reais do banco de dados. Eles só armazenam os comentários do memorando.
