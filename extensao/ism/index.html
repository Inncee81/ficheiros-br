O que é um arquivo .ism? (Arquivo manifesto de streaming suave do IIS)

Arquivo usado pela extensão Smooth Streaming do servidor IIS Web da Microsoft; usa formatação XML e contém taxa de bits e informações de faixa para áudio (streaming; salvo no mesmo diretório de servidor que os arquivos multimídia; permite que o servidor saiba quais bits as taxas mapeiam para quais arquivos multimídia, o que é útil para determinar a qualidade do streaming depois de negociar a velocidade de conexão com um cliente.

Mais Informações

Os arquivos ISM são frequentemente referenciados em Silverlight e outros aplicativos de streaming para acessar fluxos de vídeo ou áudio. Na API de streaming suave, o SmoothStreamingSource é usado para especificar uma fonte de streaming para um SmoothStreamingMediaElement (SSME) que transmite o conteúdo multimídia.

NOTA: Os arquivos ISM são gerados usando o Expression Encoder, um programa que cria o conteúdo multimídia além dos manifestos.

O que é um arquivo .ism? (Instalar arquivo de projetoShield)

Arquivo de projeto criado pelo InstallShield, um programa usado para criar Windows instaladores; contém todas as configurações e códigos do projeto, incluindo estruturas de diretório, nomes de arquivos, scripts personalizados e diálogos instaladores; usado para construir um pacote de instalação.

Mais Informações

Arquivos ISM podem ser salvos em um formato binário ou em um formato XML.

NOTA: Os arquivos ISM são criados pelas edições Premiere e Professional do InstallShield.
