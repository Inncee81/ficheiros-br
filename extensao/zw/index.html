O que é um arquivo .zw? (Arquivo de texto chinês)

Documento de texto contendo caracteres chineses; usa codificação de caracteres ASCII/Chinês; semelhante a um arquivo .HZ, mas usa um estilo de codificação diferente; também conhecido como um arquivo HXWZ.

Mais Informações

O formato ZW foi usado principalmente para o envio de texto chinês através de programas de comunicação de terminais. Como a maioria dos sistemas operacionais suportam vários idiomas agora, os arquivos ZW não são comumente usados.

O que é um arquivo .zw? (Modelo de widget zooper)

Arquivo de .ZIP comprimido criado e usado pelo Zooper Widget Pro, um aplicativo para Android usado para criar widgets personalizados Android; armazena um modelo com todos os recursos do widget, como bitmaps e fontes.

Mais Informações

Quando você salva um modelo, o arquivo ZW é salvo na pasta "ZooperWidget" no cartão SD, juntamente com uma captura de tela .PNG do widget. Você pode compartilhar o modelo colocando-o na pasta "ZooperWidget" no cartão SD do dispositivo do destinatário.

NOTA: A versão gratuita do Zooper Widget não pode abrir o arquivo ZW, apenas o Zooper Widget Pro pode abrir o arquivo ZW. Zooper Widget Pro foi descontinuado e não está mais disponível para download.
