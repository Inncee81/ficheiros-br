O que é um arquivo .msc? (Arquivo de controle snap-in do console de gerenciamento da Microsoft)

Arquivo adicionado ao Microsoft Management Console (MMC), um programa usado para configurar e monitorar Windows sistemas de computador; contém um "snap-in," que é um módulo que fornece recursos adicionais de administração; salvo em um formato XML que pode ser visualizado e editado com um editor de texto básico.

Mais Informações

Os snap-ins são comumente usados pelos administradores do sistema para gerenciamento de dispositivos, desfragmentação de disco e monitoramento de eventos do sistema. Os snap-ins personalizados podem ser desenvolvidos com a API (Application Programming Interface, interface de programação de aplicativos do MMC), que é publicada pela Microsoft Developer Network (MSDN).

MMC está incluído com sistemas operacionais Windows a partir de Windows 2000. Os atalhos abaixo podem ser acessados dentro de Control Panel → Administrative Tools.

O que é um arquivo .msc? (Arquivo de composição do MuseScore)

Arquivo criado pelo MuseScore, um programa gratuito usado para composição e anotação de música; contém a composição, que inclui a assinatura chave, instrumentos, assinatura de tempo e notas musicais; também contém metadados de partituras, incluindo informações de título, legenda, compositor e direitos autorais.

Mais Informações

Os arquivos MSC são usados por compositores para armazenar partituras musicais. Eles podem ser traduzidos para vinte idiomas diferentes no MuseScore.

NOTA: Os arquivos MSC aplicam-se às versões do MuseScore 094 e anteriores. MuseScore versão 095 e posteriormente usar a extensão de arquivo .MSCX.
