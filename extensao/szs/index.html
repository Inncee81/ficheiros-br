Arquivo de jogos usado por alguns jogos para o sistema de console GameCube, como Super Mario Sunshine; pode incluir vários dados gráficos de modelo e bitmap para o jogo; às vezes modificado por usuários de homebrew.

O que é um arquivo .szs? (Nintendo Switch Game Data Archive)

Um arquivo SZS é um arquivo de dados de jogos usado por jogos do Nintendo Switch, como Mario Kart 8 Deluxe e Splatoon 2. Ele normalmente contém arquivos de modelo 3D e textura usados por um jogo do Switch e é comprimido usando o formato de compressão do Nintendo Yaz0. Os arquivos SZS são frequentemente modificados pelas comunidades homebrew e modding para alterar a aparência de personagens e objetos em jogos do Switch.

Mais Informações

Você pode extrair o conteúdo de arquivos SZS usando Uwizard em Windows. Você também pode usar mk8Snitch para descomprimir, editar e recompilar arquivos SZS Mario Kart 8 Deluxe SZS.

NOTA: Os arquivos SZS também são comumente usados para armazenar dados de jogos para jogos nintendo Wii e GameCube. No entanto, os arquivos Switch SZS podem não ser compatíveis com programas que abrem os formatos Wii ou GameCube SZS.

O que é um arquivo .szs? (Arquivo de dados do jogo Wii)

Arquivo de dados usado por jogos para o Nintendo Wii; mais conhecido por armazenar dados de modelo e textura 3D para o jogo Wii Mario Kart, mas também usado por outros jogos para os mesmos tipos de dados, bem como outros tipos de dados; às vezes modificado pelas comunidades de homebrew e modding para criar gráficos personalizados de Mario Kart.

Mais Informações

Arquivos SZS para Mario Kart são armazenados em um formato chamado formato de arquivo U8 e são compactados com um método chamado compressão Yaz0.
