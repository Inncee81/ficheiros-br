O que é um arquivo .cdoc? (Documento clicker)

Um arquivo CDOC é um documento criado pelo Clicker, uma ferramenta de alfabetização usada pelos professores para criar recursos curriculares adaptados às necessidades dos alunos. Pode conter texto, imagens e notas de voz gravadas.

Mais Informações

O documento CDOC é o arquivo principal usado pelo processador de texto Clicker. O programa oferece vários recursos de escrita, como a capacidade de ouvir texto falado em vozes realistas, receber previsões de palavras com base no que é digitado, receber sugestões ortográficas que podem ser ilustradas e faladas, e gravar pensamentos e ideias iniciais antes de escrever.

O que é um arquivo .cdoc? (Arquivo DigiDoc criptografado)

Um arquivo CDOC é um documento digital criptografado (DigiDoc) criado com qDigiDoc, um software de código aberto usado para autenticar cartões de identificação emitidos pelo governo estoniano e finlandês e documentos assinados. Ele contém um arquivo de dados (um metadados.

Mais Informações

O arquivo CDOC é salvo no formato ENCDOC-XML. É desenvolvido por Riigi Infosüsteemi Amet (RIA), que é a Autoridade do Sistema de Informação da Estônia. CDOC é um dos três arquivos de assinatura digital e criptográfica da família DigiDoc, juntamente com os arquivos .DDOC e .BDOC.

NOTA: O software qDigiDoc pode ser usado para assinar documentos digitalmente usando o cartão de identificação ou rg do móvel de um funcionário e pode abrir, salvar e verificar documentos com assinaturas digitais. O software é de código aberto e pode ser baixado gratuitamente através dos links abaixo.
