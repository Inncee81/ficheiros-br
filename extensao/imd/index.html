O que é um arquivo .imd? (Arquivo de metadados de imagem GIS)

Arquivo GIS que contém informações de georreferenciamento para uma imagem rasteretiva correspondente, como um GeoTIFF (.TIF); pode armazenar informações de localização coordenadas, bem como a imagem de raster referenciado; mapeia imagens GIS na geografia do mundo real.

O que é um arquivo .imd? (Arquivo de imagem de disco ImageDisk)

Um arquivo IMD contém uma imagem de disco de um disquete, que também é conhecido como disquete. Ele armazena uma cópia exata da imagem do disco do disquete e inclui informações sobre o formato do disco. Os arquivos IMD são tipicamente criados pelo ImageDisk, um utilitário de imagem de disco baseado em MS-DOS usado para arquivar e recriar vários formatos de disquete.

Mais Informações

ImageDisk pode criar arquivos IMD a partir de uma variedade de formatos de disquete, mas eles devem ser compatíveis com o controlador de disquetes e drives do PC 765. Embora os arquivos IMD sejam tipicamente criados pelo ImageDisk, eles também podem ser criados e manipulados pelo pyImageDisk.

O que é um arquivo .imd? (Arquivo Microsoft VisioModeler)

Arquivo criado pelo VisioModeler, um programa mais antigo da Microsoft usado para criar diagramas; usado para armazenar modelos de informações de Modelagem de Papel de Objeto (ORM); usado para modelagem de dados de negócios e fins de documentação.

  Microsoft VisioModeler
