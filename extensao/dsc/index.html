O que é um arquivo .dsc? (Arquivo do catálogo do espaço de celestia)

Arquivo de dados usado pela Celestia, um aplicativo gratuito usado para simular visualizações 3D do universo; salva informações sobre entidades do espaço profundo, como nebulosas e galáxias; fornece localização e coleta de informações para o programa.

O que é um arquivo .dsc? (Arquivo de controle de fonte do Debian)

Arquivo de origem criado para o Debian, um sistema operacional gratuito; contém dados descrevendo os outros arquivos no pacote de origem; formatados como um único parágrafo composto por uma série de campos.

Mais Informações

O arquivo DSC é produzido pela ferramenta dpkg-source quando o arquivo de origem é construído e é baseado nos outros arquivos do pacote de origem. Quando descompactado, o arquivo DSC é referenciado contra os outros arquivos e diretórios no pacote de origem.

O arquivo DSC faz parte do pacote de origem Dobian, juntamente com um arquivo upstream tar ball com a extensão .TAR.GZ e quaisquer outros arquivos criados para o pacote Debian. Um pacote de origem é criado com a ferramenta de pacote dpkg-build ferramenta.

O que é um arquivo .dsc? (Arquivo de equação da ciência do design de Kingsoft)

Um arquivo DSC é um arquivo de configurações usado pelo WPS Office (anteriormente conhecido como Kingsoft Office), um pacote de escritório que inclui os aplicativos Writer, Planilhas e Apresentação. Ele armazena as configurações para o Design Science Equation Editor, que inclui o tamanho da janela, estilo de interface e ampliação de zoom.

Mais Informações

O arquivo DSC é referenciado pelo Editor de Equações que está disponível nas fitas do aplicativo Writer, Planilhas e Apresentação. Ele permite que o editor seja reaberto pelos usuários e retome as mesmas configurações, como a ampliação do zoom da janela e o tamanho da janela que foi definida quando foi aberta anteriormente.

O que é um arquivo .dsc? (Arquivo de identificação de disco Nikon)

Contém informações sobre fotos tiradas com uma câmera digital Nikon (como uma câmera CoolPix); salvas no cartão de memória junto com os arquivos de imagem; normalmente apenas alguns kilobytes de tamanho, já que não contém nenhum dado de imagem.

Mais Informações

DSC arquivos podem estar escondidos no cartão, uma vez que são referenciados pelo software de câmera digital da Nikon e não devem ser abertos manualmente.

Muitas câmeras digitais salvam fotos com nomes de arquivos começando com "DSC,", mas esses arquivos de imagem normalmente têm uma extensão .JPG.

NOTA: O Nikon ViewNX não está mais disponível para download.

O que é um arquivo .dsc? (Arquivo de descrição de texto)

Arquivo de texto simples usado por vários aplicativos de software para armazenar informações de descrição; pode descrever um aplicativo de software associado como um arquivo readme; também pode conter parâmetros de dados ou outros tipos de informações de texto simples.
