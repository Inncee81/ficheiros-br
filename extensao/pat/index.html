O que é um arquivo .pat? (Arquivo de patch 3D)

3D arquivo de modelo tipicamente criado por programas CAD; contém patches, que armazenam a geometria tridimensional como malha retangular; difere de outros formatos que usam vértices únicos para armazenar formas 3D.

Mais Informações

Os arquivos PAT usam um formato semelhante aos arquivos .3DA.

O que é um arquivo .pat? (Arquivo padrão autoCAD hatch)

ARQUIVO CAD usado pelo AutoCAD, um aplicativo profissional de desenho CAD; armazena um padrão hatch que pode ser usado para texturizar ou preencher uma área; ajuda a dar uma aparência material (por exemplo, tijolo, alumínio, vidro ou plástico) a objetos desenhados.

Mais Informações

Os padrões hatch são baseados em vetores e são escritos em um formato de texto simples. Consulte a documentação do AutoCAD para saber a sintaxe usada para escrever padrões de escotilha.

O que é um arquivo .pat? (Arquivo padrão CorelDRAW)

Padrão vetorial usado pelo CorelDRAW Graphics Suite, um conjunto de programas que inclui aplicativos de edição de gráficos raster e vetorial; salva um padrão que pode ser repetido dentro de um documento de layout de imagem vetorial ou página; muitas vezes usado para aplicar texturas de fundo.

Mais Informações

Corel CONNECT, um aplicativo de navegação de conteúdo gráfico incluído no CorelDRAW Graphics Suite, fornece um pacote de conteúdo gráfico que inclui vários padrões vetoriais, que podem ser incluídos em imagens ou documentos.

O que é um arquivo .pat? (Arquivo de instalação do gerenciador de discos)

Um arquivo PAT é um arquivo de sistema usado pelo DiskStation Manager (DSM), o sistema operacional usado para dispositivos NAS de Sinologia. Ele contém arquivos necessários para instalar ou atualizar o sistema operacional de baixo nível que executa o dispositivo.

Mais Informações

Os arquivos PAT também são usados pelo Synology Assistant, um programa usado para instalar firmware em dispositivos DSM. Durante o processo de configuração do dispositivo DSM, o Synology Assistant solicita um arquivo de instalação PAT para o dispositivo. Você pode usar o arquivo PAT localizado no CD de instalação da Synology ou um arquivo PAT baixado do Synology Download Center.

O que é um arquivo .pat? (Arquivo padrão de som ketron)

O formato de padrão de som Ketron usado para tocar diferentes estilos para os teclados Ketron, incluindo os modelos SD1, SD3, SD5 e Audya; contém sons, níveis e efeitos de estilo; pode ser usado para uma grande variedade de gêneros como Rock, Jazz, Pop, R&B, Latim, Dance e Gospel.

Mais Informações

Nos modelos SD, os arquivos PAT estão localizados na pasta ESTILO dentro da PASTA MASTER, que é acessada quando você aperta o botão MASTER FOLDER no teclado. Dependendo da versão do teclado que você está usando, você pode carregar o arquivo PAT através de um disquete ou unidade USB. Você pode criar um arquivo PAT com a função "Pattern Edit" em modelos SD. Nos modelos Aduya, o arquivo PAT é comumente acompanhado por um arquivo .BIN que contém os parâmetros para o estilo contido no arquivo PAT.

NOTA: Você pode abrir arquivos PAT com software Ketron para o modelo apropriado do seu teclado.

O que é um arquivo .pat? (Arquivo padrão)

Imagem bitmap usada como um padrão que pode ser importado em múltiplos programas gráficos; tipicamente um arquivo quadrado que pode ser de 8x8 pixels a 256x256 pixels de tamanho; repete para preencher uma área com o padrão; muitas vezes usado para criar um fundo texturizado.
