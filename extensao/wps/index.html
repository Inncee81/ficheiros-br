O que é um arquivo .wps? (Documento do Escritor Kingsoft)

Um arquivo WPS é um documento de processamento de texto criado pelo Kingsoft Writer, um processador de texto que faz parte do pacote Kingsoft WPS Office. Contém texto, imagens e formatação de páginas. Os arquivos WPS são semelhantes aos documentos do Microsoft Word (.DOC ou .DOCX arquivos).

Captura de tela de um arquivo wps no Kingsoft WPS Office 2019

O arquivo WPS é o arquivo principal associado ao Writer. Ele é criado quando você salva um documento para armazenar o conteúdo do documento. Você pode criar um documento WPS do zero ou usar o modelo .WPT para iniciá-lo e salvá-lo como um arquivo WPS.

Writer está incluído no pacote WPS Office, juntamente com os programas Apresentação e Planilhas. Além do arquivo WPS, o Writer suporta outros documentos, incluindo .DOCX, .DOC, .DOTX, .PDF e documentos .RTF. O writer possui recursos ricos de edição de texto e formatação de documentos e vem com várias ferramentas, incluindo verificação ortom ortomal, contagem de palavras, dicionário e complementos.

O que é um arquivo .wps? (Documento do processador de texto do Microsoft Works)

Um arquivo WPS é um documento criado com o software de processamento de texto Microsoft Works. Ele contém elementos de documentos semelhantes a um documento do Microsoft Word (.DOC), mas não inclui opções avançadas de formatação e macros que um documento do Word pode conter.

Mais Informações

Captura de tela de um arquivo wps em LibreOffice Writer 5

O Microsoft Works 9 foi a versão final do Works e o software foi descontinuado em 2010. No entanto, Windows usuários ainda podem abrir arquivos WPS criados pelas versões Works 6 a 9 com aplicativos microsoft office e "Microsoft Works 6-9 File Converter" instalados. O arquivo WPS também pode ser aberto no LibreOffice.

A versão macOS (Mac OS X) do Microsoft Works foi descontinuada após a versão 40. Portanto, a maioria dos documentos de Obras modernos não pode ser aberta usando obras para Mac. No entanto, os arquivos WPS podem ser abertos no macOS usando o NeoOffice 22 ou posterior, o que inclui a biblioteca "libwps".

Nos sistemas Linux, os arquivos WPS podem ser abertos usando o OxygenOffice Professional ou outra variação do conjunto OpenOfficeorg que inclui a biblioteca "libwps". Algumas distribuições Linux também podem incluir um programa chamado wps2odt, que usa libwps e pode converter arquivos WPS em arquivos .ODT. Um pacote chamado "writerperfect" também pode incluir wps2odt se a versão apropriada do libwps estiver disponível.

Além disso, o programa Writer incluído no pacote gratuito LibreOffice é outro programa que suporta arquivos WPS. É uma ótima alternativa para usuários de Windows, macOS e Linux.

NOTA: O formato WPS foi descontinuado pela Microsoft em 2006 e foi substituído pelo formato padrão do Microsoft Word "DOC" .

O que é um arquivo .wps? (Arquivo do projeto de bancada de trabalho do tradutor)

Arquivo de configurações de projeto salvo pela Bancada de Trabalho do Tradutor, um programa de tradução e localização de idiomas; armazena configurações usadas para traduzir texto de aplicativo para outro idioma; permite que os usuários armazenem e reutilizem propriedades do sistema de memória de tradução.
