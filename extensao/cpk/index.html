O que é um arquivo .cpk? (Pacote CRI Middleware)

Arquivo de pacote de jogos usado por jogos criados com o software de embalagem CRI Middleware; pode armazenar gráficos, áudio e outros dados do jogo; usado como um contêiner para arquivos de jogos.

Mais Informações

CRI A tecnologia Middleware é usada para embalar dados para vários sistemas de consoles de jogos, incluindo os da Microsoft, Nintendo, Sega e Sony.

O que é um arquivo .cpk? (Arquivo de registro de teste DTM)

Arquivo log gerado pelo Driver Test Manager (DTM), um componente do software de certificação de dispositivos Windows Logo Kit (WLK) da Microsoft; contém um relatório de teste, que registra informações sobre um dispositivo de hardware; usado para enviar resultados de teste à Microsoft para que um dispositivo possa ser certificado para Windows; também usado para depurar testes fracassados.

Mais Informações

Arquivos CPK podem ser visualizados com o DTM Log Viewer. O espectador mostra informações como a data do teste, o sistema operacional de teste e testes aprovados e reprovados.

O software WLK é disponibilizado pelo Windows Hardware Quality Labs (WHQL), um grupo da Microsoft que certifica Windows dispositivos de hardware.
