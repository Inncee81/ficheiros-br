O que é um arquivo .cfs? (Arquivo de console CFS)

Programa usado pelo CFS Console, um programa de console usado para instalar programas desenvolvidos pela Zaplots; contém um programa que pode ser executado no console CFS; comumente usado para carregar jogos Zaplots.

O que é um arquivo .cfs? (Arquivamento compacto do conjunto de arquivos)

Arquivo de contêiner comprimido criado no formato CFS (Compact File Set, conjunto de arquivos compactos), um formato aberto desenvolvido pela Pismo Technic; armazena dados usando uma implementação reduzida do padrão de imagem de disco .ISO; usado para armazenar imagens de discos de CD e DVD, bem como para a criação de distribuições de software e grandes arquivos de arquivos compactados.

Mais Informações

Os arquivos CFS foram projetados para serem mais fáceis de ler do que os arquivos ISO, o que pode ser difícil de ler devido à grande variedade de opções de armazenamento suportadas na especificação ISO. Uma vez que os arquivos CFS são compatíveis com o padrão de arquivo ISO, eles podem ser lidos por muitos aplicativos ISO existentes.

NOTA: O formato CFS é baseado nos seguintes padrões: ISO-9660, Joliet Extensions, ISO-9660:1999, Compact ISO.

O que é um arquivo .cfs? (Arquivo de índice composto de Lucene)

Arquivo usado por Lucene, uma biblioteca de mecanismos de busca de texto escrita em Java; usada para salvar vários arquivos de índice em um único arquivo composto; permite um gerenciamento mais fácil do índice, mas pode ter um desempenho ligeiramente mais lento.

Mais Informações

NOTA: Começando pela versão 14, Lucene usa arquivos CFS por padrão.
