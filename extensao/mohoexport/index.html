O que é um arquivo .mohoexport? (Documento do perfil de exportação de Moho)

Um arquivo MOHOEXPORT é um documento de perfil de exportação criado com o Anime Studio 12 e, posteriormente, um programa de desenho e animação. Contém uma configuração para um trabalho de exportação em lote em Moho. Arquivos MOHOEXPORT substituíram .ANIMEEXPORT arquivos pela versão 12 do Moho Debut e Pro.

Mais Informações

Os arquivos MOHOEXPORT foram projetados para ajudar os usuários a compartilhar as configurações de exportação com outros usuários e acessar rapidamente a configuração de configuração para futuras exportações repetidas de lotes. Os arquivos armazenam várias configurações que compõem o perfil de exportação do lote. Algumas das opções de configuração incluem o local onde os arquivos serão exportados, opções de renderização e o formato para o qual um item é exportado.

Você pode criar e abrir arquivos MOHOEXPORT no Moho Exporter, que está incluído no Moho Pro e na Debut. Para acessar o Exportador Moho, selecione File → Moho Exporter....

NOTA: Moho era anteriormente conhecido como Anime Studio antes do lançamento da versão 12.
