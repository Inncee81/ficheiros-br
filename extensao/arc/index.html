O que é um arquivo .ARC? (Arquivo de dados do Safari Africano)

Contém música, imagens e outros dados usados pelo Safári Africano de Cabela; chamado "dataarc" e está localizado no diretório /PSP_GAME/USRDIR/ em discos Sony PSP; O Safári Africano de Cabela está disponível para Windows PC, Xbox, Playstation 2 e PSP.

Mais Informações

O arquivo DATAARC é referenciado pelo jogo Safári Africano e não deve ser aberto manualmente pelo usuário.

O que é um arquivo .ARC? (Arquivo de arquivo compactado)

Um arquivo ARC é um arquivo de arquivo compactado criado como um arquivo HFS (Hierarchical File System, sistema de arquivos hierárquicos macintosh) ou um arquivo MS-DOS usando compactação padrão MS-DOS. Contém uma coleção de arquivos comprimidos, cada um sendo comprimido potencialmente com uma técnica de compressão diferente.

Mais Informações

Arquivos ARC eram comumente usados no final da década de 1980 e início dos anos 1990. No entanto, eles raramente são usados agora e foram substituídos por algoritmos de compressão mais novos e eficientes.

NOTA: Enquanto os arquivos ARC eram usados principalmente para MS-DOS e versões mais antigas de computadores Macintosh, vários programas atuais ainda podem abri-los, como StuffIt e WinZip.

O que é um arquivo .ARC? (Norton Backup Archive)

Um arquivo ARC é um arquivo de backup criado por produtos de software norton, como Norton 360 e Norton Online Backup. Ele contém um backup de arquivos de computador criados automaticamente pelo software, que pode incluir arquivos pessoais do usuário, bem como arquivos do sistema.

Mais Informações

Os arquivos ARC são usados para armazenar e recuperar arquivos em caso de exclusão acidental ou falha no sistema. Eles podem ser apoiados até um local fora do local usando o Norton Online Backup.

NOTA: Como os arquivos ARC podem incluir todos os arquivos pessoais e do sistema, eles podem ser muito grandes em tamanho.

O que é um arquivo .ARC? (Arquivo de backup do Symbian OS)

Arquivo de backup criado para dispositivos móveis que usam o sistema operacional de código aberto Symbian; contém dados de backup pessoais e do sistema do dispositivo móvel; usado para restaurar dados caso eles se percam.

Mais Informações

Exemplos de arquivos ARC incluem backup.arc, Messages.arc, Calendars.arc e Bookmarks.arc.
