O que é um arquivo .plantuml? (Arquivo PlantUML)

Arquivo criado pelo PlantUML, um criador de diagramas uml usado dentro de programas como Sublime Text Editor ou linguagem de programação como HTML; mesmo que o arquivo .PU; contém código referenciado pelo PlantUML para criar imagens de diagrama que podem ser geradas como .PNG ou .SVG arquivos; cria sequência, caso de uso, classe, atividade, componente, estado e diagramas de objetos.

Mais Informações

Um exemplo da sintaxe usada pelo arquivo PLANTUML para um diagrama de objeto:

@startuml
object Chair
object Table

Chair --|> Tabela : móveis
<@enduml
Este exemplo criaria um diagrama de dois objetos chamados Cadeira e Mesa com uma linha e seta apontando da cadeira para a mesa. Além disso, a palavra mobília apareceria como um rótulo no meio da linha que conecta os dois objetos. A linha @startuml começa o comentário e a linha @enduml encerra o comentário.

NOTA: PlantUML é baixado como um arquivo de .JAR, consulte o link plantuml abaixo para saber como executar a ferramenta.
