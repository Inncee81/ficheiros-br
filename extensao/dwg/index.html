O que é um arquivo .dwg? (Arquivo de banco de dados de desenho autocad)

Um arquivo DWG é um banco de dados de desenhos 2D ou 3D criados com o AutoCAD, um programa CAD profissional. Ele contém dados de imagem vetorial e metadados que descrevem o conteúdo do arquivo. Os arquivos DWG estão relacionados com versões ASCII (texto) de arquivos DWG.

Mais Informações

Captura de tela de um arquivo dwg no CorelCAD 2019

ARQUIVOS DWG podem ser visualizados em Windows usando o programa gratuito DWG TrueView da Autodesk. Eles também podem ser abertos por outros aplicativos de terceiros que suportam o formato DWG, como CorelCAD, IMSI TurboCAD e Adobe Illustrator.

Várias variações do formato DWG foram publicadas pela AutoCAD. Desde 2000, foram introduzidos os formatos DWG 2000, DWG 2004, DWG 2007, DWG 2010, DWG 2013 e DWG 2018. Embora esses formatos sejam semelhantes, suas estruturas de arquivos são ligeiramente diferentes. Portanto, programas CAD de terceiros podem ser capazes de abrir alguns arquivos DWG, mas não outros.
