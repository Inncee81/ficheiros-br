O que é um arquivo .glb? (Arquivo de formato de transmissão GL binário)

Um arquivo GLB contém um modelo 3D salvo no GL Transmission Format (glTF). Armazena informações sobre um modelo 3D, como hierarquia de nó, câmeras e materiais, animações e malhas em formato binário. Os arquivos GLB são a versão binária de .GLTF arquivos.

Mais Informações

OS arquivos GLB podem ser usados para salvar e compartilhar ativos digitais entre diferentes ferramentas de modelagem 3D, da mesma forma que aqui.

O que é um arquivo .glb? (Arquivo de jogo Glulx Blorb)

Um arquivo GLB é um arquivo de jogo criado no formato Glulx Blorb, um formato usado para agrupar diferentes arquivos de recursos de jogo em um único arquivo. Pode conter gráficos, informações de nível, música ou outros dados. Os arquivos GLB são usados por jogos como DemonStar, um jogo de tiro de rolagem vertical que acontece no espaço sideral.

Mais Informações

OS arquivos GLB foram originalmente projetados para uso com jogos de aventura de texto "Z-machine"," mas foram adaptados para outros tipos de jogos.

O que é um arquivo .glb? (Arquivo STK Globe)

Um arquivo GLB é um arquivo de globo usado pelo AGI Systems Tool Kit (STK), um programa usado para simular missões aeroespaciais, sistemas de defesa e sistemas de telecomunicações. Ele armazena um globo 3D que modela terrenos reais ou simulados. Os arquivos GLB são usados para compartilhar globos entre os usuários de STK, permitindo que eles usem os globos dos outros dentro de suas próprias simulações.

Mais Informações

Para criar um arquivo GLB fragmentável, os usuários de STK podem exportar seu globo atual usando o Globe Manager da STK. Eles podem então carregar esse arquivo para o AGI Globeserver, um servidor web de armazenamento de globo, permitindo que outros usuários de STK o acessem.

NOTA: Arquivos GLB são arquivos de texto que registram quais texturas, imagens e outros arquivos são usados em um globo, bem como onde esses arquivos são armazenados. Como tal, quando os usuários compartilham um arquivo GLB, eles também devem fornecer acesso às texturas, imagens e outros arquivos referenciados dentro desse arquivo GLB. Caso contrário, o arquivo GLB não será carregado corretamente.
