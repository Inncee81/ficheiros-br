O que é um arquivo .MAT? (Arquivo de materiais Max 3ds)

Arquivo usado pela 3ds Max, um pacote de software de modelagem 3D e animação; armazena um material, que define a forma como texturas e iluminação são aplicadas a uma superfície; pode ser acessado a partir do componente Editor de Materiais em 3ds Max.

O que é um arquivo .MAT? (MATLAB MAT-File)

Formato de contêiner de dados binário usado pelo MATLAB; pode incluir matrizes, variáveis, funções e outros tipos de dados; pode ser salvo em diferentes formatos escolhendo uma versão dentro das Preferências DO MATLAB (selecione General → MAT-Files).

Mais Informações

Nível 4 MAT-Files suportam matrizes bidimensionais e strings de caracteres, enquanto o MAT-Files nível 5 inclui suporte adicional para matrizes numéricas multidimensionais, matrizes de caracteres, matrizes de células, matrizes esparsas, objetos e estruturas.

Os arquivos MAT também podem ser usados para representar áudio em pontos flutuantes de 64 bits, inteiros assinados de 16 bits e formatos inteiros não assinados de 8 bits.

Formato Binário 

O que é um arquivo .MAT? (Arquivo de atalho de tabela de acesso do Microsoft)

Arquivo de atalho da Microsoft Access Table (MAT) criado pelo Microsoft Access, um programa de banco de dados relacional para computadores desktop; contém um atalho para uma tabela dentro de um banco de dados de acesso; abre uma tabela de banco de dados relacional sem antes ter que abrir o Microsoft Access.

Mais Informações

Os arquivos MAT podem ser criados no Access clicando e arrastando uma tabela para a área de trabalho.

O que é um arquivo .MAT? (Arquivo de materiais v-ray)

Contém materiais, ou estilos de textura, que podem ser carregados em programas de design 3D usando o plugin V-Ray; V-Ray é um plugin de geração de material de superfície e edição desenvolvido pelo Chaos Group.

O que é um arquivo .MAT? (Arquivo de material vue)

Arquivo criado por programas de software E-on Vue, que são usados para criar visualizações 2D e 3D, bem como materiais gráficos de computação; contém imagens e texturas que podem ser aplicadas a objetos 3D; usados como componentes, ou blocos de construção, para criar uma cena geral de Vue.
