O que é um arquivo .tmlanguage? (Arquivo de gramática de linguagem textmate)

Arquivo de gramática de linguagem usado pelo TextMate, um editor de código-fonte Mac OS X; contém uma especificação para a sintaxe de uma linguagem de programação; usado para adicionar suporte de destaque de sintaxe para idiomas adicionais.

Mais Informações

Os arquivos TMLANGUAGE são salvos usando a formatação XML, mas podem ser codificados com uma representação binária dos dados XML.

NOTA: Os arquivos de gramática da linguagem TextMate são frequentemente salvos como arquivos binários da Lista de Propriedades (.PLIST). Por exemplo, alguns dos arquivos .TMBUNDLE incluídos na instalação TextMate incluem um arquivo PLIST em seu diretório de /Syntaxes/.
