O que é um arquivo .cdp? (Arquivo PRO CD/Spectrum)

Arquivo de informações usado pelo CD/Spectrum Pro, um reprodutor de áudio de CD que inclui um analisador de espectro gráfico; contém informações de artista de CD, título e faixa; pode ser importado em CD/Spectrum Pro para carregar automaticamente informações para um CD específico.

O que é um arquivo .cdp? (Arquivo ConceptDraw PROJECT)

Arquivo de projeto criado pelo ConceptDraw PROJECT, um aplicativo de gerenciamento de projetos; armazena cronogramas de projetos, recursos, prioridades e informações de rastreamento; usado pelo programa para exibir "dashboards", que resumem os principais indicadores de desempenho.

O que é um arquivo .cdp? (Projeto Sony CD Architect)

Um arquivo CDP é um projeto criado pela Sony CD Architect, um programa de masterização e gravação de CD de áudio. Ele contém informações do projeto, como a localização de arquivos de origem, cortes, pontos de inserção, edições e propriedades do projeto. Os arquivos CDP não armazenam o áudio real, apenas as informações necessárias para a criação do CD após a edição ser concluída.

Mais Informações

Arquivos CDP permitem que você faça edições em seu CD sem modificar os arquivos de origem reais. Isso significa que você pode experimentar as configurações e edições e não precisa se preocupar em alterar o áudio.

Quando você executa pela primeira vez o CD Architect, um novo projeto é criado automaticamente, mas não é até que você escolha salvar o projeto que um arquivo CDP real é criado para armazenar o conteúdo do projeto. Escolha File → Save para salvar o projeto.

Os arquivos CDP só são suportados pelo CD Architect. Para abrir um arquivo CDP, selecione File → Open..

Um projeto de CDP é salvo automaticamente a cada 5 minutos pelo CD Architect no arquivo .AUTOSAVE.CDP. No entanto, o arquivo é excluído quando o CD Architect é fechado. Você pode desativar a função de autosave desmarcando a caixa de seleção Enable autosave.

NOTA: Sony CD Architect é descontinuado.

O que é um arquivo .cdp? (Pacote de despachante de conteúdo do simulador trainz)

Arquivo usado pelos jogos de simulação ferroviária do Trainz Simulator; contém conteúdo complemento do jogo, como rotas de trem, projetos de locomotivas e cenário; pode ser importado usando as ferramentas de Despachante de Conteúdo e Gerente de Conteúdo Plus dentro dos produtos Trainz Simulator; instalado na pasta WorldDispatcherDownloads das instalações do Trainz.

Mais Informações

Os arquivos CDP são desenvolvidos por provedores de conteúdo de terceiros, como a Auran, e não fazem parte do conteúdo original do jogo. Existem várias ferramentas para gerenciar pacotes de CDP.
