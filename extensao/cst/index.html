O que é um arquivo .cst? (Arquivo de conjunto personalizado de tela)

Configurações do arquivo criado pelo Canvas, um programa de ilustração técnica; contém atalhos de teclado de aplicativos usados para acessar funções do programa; permite que os usuários salvem preferências personalizadas do espaço de trabalho.

Mais Informações

Os arquivos CST podem incluir combinações-chave para seleções de menus, ferramentas, tintas, canetas e fontes. O conjunto personalizado padrão é salvo no arquivo DefaultKeyboard.cst, que está localizado no diretório /Extras/ da instalação Do Canvas.

Desenvolvedor ContentServ
Categoria Desenvolvedor
Formato Não Disponível 

O que é um arquivo .cst? (Modelo contentserv)

Modelo para a criação de um novo documento ContentServ; pode ser usado por um projeto emms (Enterprise Marketing Management Solutions) para gerenciar processos de marketing, vendas e comunicação.

Mais Informações

Os modelos do ContentServ incluem CSTags, que definem o conteúdo que é substituído pela saída do mecanismo de modelo quando o documento é acessado.

Formato Não Disponível 

O que é um arquivo .cst? (Arquivo de elenco externo do diretor)

Arquivo cast criado pelo Adobe Director, um programa de criação de filmes originalmente desenvolvido pela Macromedia; armazena propriedades para gerar a saída de vídeo; permite que um elenco seja reutilizado para vários vídeos.

Mais Informações

Arquivos CST contêm moldes externos, que estão totalmente desvinculados do filme-alvo. Elencos internos, por outro lado, não podem ser desvinculados do filme e só podem ser usados para o filme ao que estão ligados.

Desde que o Adobe Director foi substituído pelo Adobe Flash, os arquivos CST não são mais comumente usados.

Programas que abrem o arquivo .cst (Arquivo de elenco externo do diretor)

  Windows

O que é um arquivo .cst? (Gary Gadget Plug-in)

Arquivo plugin usado por Gary Gadget: Building Cars, um carro projetando, construindo e dirigindo jogo para crianças; adiciona recursos ao jogo, como peças de carro ou jogos bônus.

Mais Informações

CST arquivos de plugin estão disponíveis para download no site Gary Gadget: Building Cars. Quando o download estiver concluído, coloque os arquivos CST no diretório Gary Gadget\Building Cars\Plugins do local de instalação do jogo. Se a pasta Plugins não existir, você precisará criá-la. Depois de colocar o arquivo na pasta, reinicie o jogo e o plugin#s estará disponível.

NOTA: O programa Viva Media Gary Gadget: Building Cars não está mais disponível para download.

Desenvolvedor CELSYS

Formato Não Disponível 

O que é um arquivo .cst? (Arquivo mangá Studio Story)

Arquivo de layout de página criado pelo Manga Studio, um programa usado para criar histórias em quadrinhos; salva o enredo para o gibi e pode incluir vários capítulos, várias páginas (.CPG arquivos) por capítulo e layouts para cada página; usado para especificar a estrutura do gibi.

Mais Informações

Manga Studio tornou-se Clip Studio Paint em 2015. Também é conhecido como ComicStudio no Japão.
