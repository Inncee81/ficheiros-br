O que é um arquivo .layout? (Código::Bloqueia arquivo de layout do espaço de trabalho)

Arquivo de suporte ao aplicativo criado por Code::Blocks, um IDE de desenvolvimento de software de código aberto para C/C++; salva configurações de layout para o espaço de trabalho do desenvolvedor, como quais arquivos estavam abertos e onde estavam posicionados; salvo com um arquivo .CBP (Code::Blocks project) e usado para restaurar o layout do espaço de trabalho para o projeto.

O que é um arquivo .layout? (Documento LayOut)

Arquivo compactado criado pelo Layout, um programa de desenho de arquitetura apresentado no SketchUp Pro; contém um arquivo .XML que se vincula a quaisquer recursos referenciados em seu documento; também contém todos os arquivos para os links de arquivo XML, como arquivos de texto, modelos e imagens.

Mais Informações

Você pode localizar arquivos layout padrão seguindo estas etapas:

  1. Abra a janela Finder e clique na pasta SketchUp no local que você salvou (provavelmente em Aplicativos).
  2. Em seguida, clique com o botão direito do aplicativo LayOut
  3. Selecione Show Package Contents → Contents → Resources → English.lproj, os arquivos LAYOUT estarão localizados em pastas Scrapbooks ou Modelos.

Você também pode criar seus próprios arquivos LAYOUT:

  1. Com o aplicativo LayOut aberto, selecione File → Save As... e escolha onde você quer que ele seja salvo e nomeado.
  2. Você também pode ir para File → New, em seguida [Arquivo → Salvar As[/m].
