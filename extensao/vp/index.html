O que é um arquivo .vp? (Arquivo de publicação ventura)

Arquivo criado por Corel Ventura, um aplicativo profissional de layout de página; armazena uma publicação digital, que pode incluir gráficos de texto, raster e vetor, e informações de formatação de página; usado para armazenar uma publicação e publicá-la em mídia digital ou impressa.

Mais Informações

As publicações criadas em Ventura são automaticamente baseadas no modelo de publicação, que é armazenado no arquivo Default.vp. Este modelo contém o layout padrão para os capítulos, quadros, objetos gráficos e folhas de estilo de uma nova publicação.

NOTA: Corel Ventura não está mais desenvolvido.

O que é um arquivo .vp? (Arquivo de código fonte criptografado Verilog)

Arquivo criptografado originalmente escrito no Verilog, um idioma de descrição de hardware (HDL) usado para especificar projetos eletrônicos do sistema; armazena a versão criptografada de um arquivo de código fonte Verilog, que usa a extensão .V; usada para proteger a propriedade intelectual armazenada em arquivos de código fonte de texto simples.

Mais Informações

Arquivos VP permitem que os desenvolvedores ofereçam modelos de sistemas eletrônicos de forma criptografada. Os arquivos são criptografados com uma chave pública para que eles só possam ser descriptografados com a chave privada associada. Uma vez descriptografados, eles podem ser lidos como arquivos Verilog normais.

O que é um arquivo .vp? (Arquivo do programa Vertex)

Arquivo usado por aplicativos que usam a tecnologia OpenGL; especifica propriedades de exibição para vértices dentro de um modelo tridimensional; salvo em um formato de texto e usa sintaxe de baixo nível chamada linguagem de montagem.

Mais Informações

NOTA: Os arquivos VP também são referidos como "ARB Vertex Programs" ARB significa OpenGL Architecture Review Board, um comitê que supervisiona a especificação OpenGL.

Desenvolvedor Descent Network Team

Formato Binário 

O que é um arquivo .vp? (Arquivo do pacote de volição)

Arquivo de arquivos não comprimido criado no formato VP (Volition Package) contém uma coleção de arquivos, como gráficos de jogos e outros recursos; permite que os desenvolvedores de jogos juntem vários arquivos enquanto mantêm uma estrutura de diretório acessível.

Mais Informações

O formato do Pacote de Volição foi criado como parte da suíte Ferramentas de Gerenciamento de Descida (DMTOOLS), que é um conjunto de utilitários para Descent 1-3 e FreeSpace 1-2. Descida e FreeSpace são jogos de computador onde os jogadores controlam naves no espaço sideral.
