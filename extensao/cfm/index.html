O que é um arquivo .cfm? (Arquivo de marcação coldfusion)

Um arquivo CFM é uma página web que contém código ColdFusion ou CFML (ColdFusion Markup Language). Ele é executado dinamicamente por um servidor web ColdFusion quando a página é acessada por um usuário. Os arquivos CFM podem executar aplicativos e scripts do ColdFusion escritos no CFScript e podem referenciar informações do banco de dados, gerar formulários e criar relatórios PDF em tempo real.

Mais Informações

Você provavelmente só encontrará arquivos CFM se você for um desenvolvedor web e integrar a funcionalidade coldFusion em suas páginas web. Os desenvolvedores normalmente usam o Adobe ColdFusion Builder para criar aplicativos no CFML. No entanto, eles podem usar uma variedade de outros IDEs e editores CFML, como Adobe Dreamweaver, CFEclipse (plugin for Eclipse) e CFBrackets (extensão para o editor de código fonte Brackets).

Arquivos CFM podem ser abertos e visualizados com um editor de código-fonte, como Sublime Text ou Visual Studio Code. Os arquivos também podem ser abertos com um editor de texto simples, como o Micorost Notepad ou o Apple Textedit, que são empacotados com Windows e macOS.

CFML é uma linguagem baseada em tags que apresenta uma sintaxe semelhante a HTML e XML. Ele inclui tags exclusivas e suporte para tags personalizadas para criar aplicativos web dinâmicos. Todas as tags CFML começam com cf.

NOTA: ColdFusion foi originalmente desenvolvido pela Allaire, depois comprado pela Macromedia, e agora é distribuído pela Adobe Systems após adquirir a Macromedia em 2005.
