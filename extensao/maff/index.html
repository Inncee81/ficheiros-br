O que é um arquivo .maff? (Arquivo de formato de arquivo da Mozilla)

Arquivo de Web comprimido criado pelos aplicativos Firefox e SeaMonkey da Mozilla; salvo em um formato de .ZIP compactado e contém uma ou mais páginas Web, incluindo os metadados necessários para reproduzir as páginas offline; também contém conteúdo para download, como áudio ou vídeo.

Mais Informações

Arquivos MAFF são usados para navegar Web páginas off-line. Como o formato suporta várias guias do navegador, os usuários também podem navegar entre guias salvas offline.

MAFF files require the Mozilla plug-in called MAF (baixado como um arquivo .XPI). Uma vez instalado este plug-in, você pode salvar arquivos MAFF no Firefox selecionando File → Save Tabs As... ou File → Save All Tabs As....
