O que é um arquivo .esp? (Arquivo de plugin de pergaminhos mais velhos)

Arquivo Plugin usado pelos RPGs em primeira pessoa do The Elder Scrolls (TES), como Oblivion e Skyrim; semelhante a um arquivo .ESM, mas é usado especificamente para adicionar recursos de plugin, como novos gráficos, armas e conteúdo mundial; usado para criar alterações personalizadas do jogo.

Mais Informações

Para aplicar modificações personalizadas de jogos, os arquivos ESP interagem com arquivos ESM (Master) e substituem comportamentos e dados padrão. Embora os arquivos ESP possam ser usados para plugins, muitas vezes os usuários criam mods de arquivos ESM e os distribuem para alterações personalizadas do jogo sem usar arquivos ESP.

O que é um arquivo .esp? (Arquivo plugin Fallout 3)

Arquivo Plugin usado pelos jogos de RPG pós-guerra nuclear futuristas de Nova Vegas; contém um plugin que modifica o conteúdo do jogo; pode incluir objetos de jogo, diálogo, configurações de jogabilidade, configurações de IA, propriedades mundiais e outros conteúdos.

Mais Informações

Arquivos ESP podem ser usados para modificar os dados do jogo existentes ou para adicionar novos dados do jogo. Eles podem ser editados usando a GECK. (o Kit de Criação garden of Eden), que é disponibilizado pela Bethesda Softworks, a desenvolvedora da série Fallout.

Os arquivos de plugin ESP são complementares aos arquivos mestres .ESM e não serão carregados sem um arquivo ESM correspondente. Além disso, as configurações em arquivos ESP substituem as dos arquivos ESM, uma vez que os dados do jogo ESP são sempre carregados após os dados do ESM. Portanto, você pode usar arquivos ESP para substituir as propriedades padrão do jogo com propriedades personalizadas.
