O que é um arquivo .sc4? (Arquivo de cenário do magnata da montanha-russa)

Arquivo de jogo usado pela RollerCoaster Tycoon, um jogo onde os jogadores criam e gerenciam parques de diversões; salva as condições iniciais para jogar um cenário; pode ser um cenário incluído com o jogo original ou incluído com um pacote de expansão.

O que é um arquivo .sc4? (Arquivo salvo simCity 4)

Arquivo de jogo criado pelo SimCity 4, um jogo de simulação onde os jogadores constroem cidades; armazena o terreno, edifícios e estradas para uma nova cidade que é desenvolvida pelo jogador no jogo; pode ser uma cidade pequena, média ou de grande porte.

Mais Informações

Arquivos SC4 podem ser portados para o Sims 2 e usados como novos bairros. No entanto, apenas cidades de pequeno porte (ou seja, não médias ou grandes) podem ser utilizadas no Sims 2. Além disso, há algumas incompatibilidades com os tipos de estradas e edifícios que podem ser importados.

O que é um arquivo .sc4? (Arquivo de projeto do SuperCard 4)

Arquivo de projeto criado pelo SuperCard, um ambiente de desenvolvimento de alto nível para a plataforma Mac OS X; contém um documento de tempo de execução do SuperCard, que inclui componentes da interface do usuário, recursos de aplicativos e scripts.

Mais Informações

Os projetos do SuperCard são usados para desenvolver aplicativos autônomos do OS X. Eles também podem ser usados para desenvolver apresentações, serviços de ensino e outras ferramentas.

NOTA: A extensão "sc4" significa SuperCard 4. A extensão .SC45 substituiu a extensão "sc4" e é usada por versões subsequentes do software.
