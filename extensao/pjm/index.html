O que é um arquivo .pjm? (Modelo xPlan)

Modelo criado pelo xPlan, um aplicativo de gerenciamento de projetos; semelhante ao arquivo .PJ2; possui uma pasta de documentos e dois arquivos .XML; contém entrada de dados do projeto, como relatórios, tarefas de projeto, recursos e eventos.

Mais Informações

Os dois arquivos XML são metadata.xml e project.xml. O arquivo metadata.xml contém dados sobre o programa, como sua compatibilidade com iPads ou sua versão atual. O arquivo project.xml contém informações sobre o projeto real, como o orçamento e o cronograma semanal.

Para criar um arquivo PJM:

  1. Abra o aplicativo xPlan, selecione Crie um novo projeto..
  2. Uma vez aberto o projeto, selecione File → Save....
  3. Com a caixa de diálogo Salvar aberta, nomeie seu projeto, escolha o local de salvamento, selecione "xPlan Model" no menu drop down do Formato de arquivo e clique em Save.
