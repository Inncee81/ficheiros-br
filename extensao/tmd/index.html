O que é um arquivo .tmd? (Arquivo de banco de dados temporário MySQL)

Arquivo de banco de dados intermediário criado pelo MySQL, um programa de banco de dados relacional de código aberto; contém informações que ajudam o MySQL a completar os processos de recuperação ou conversão de banco de dados; criado no diretório de banco de dados onde os arquivos .MYI, .MYD e .FRM estão localizados.

Mais Informações

Os arquivos TMD podem ser criados quando o comando myisampack MySQL é executado. Se o processo for encerrado antes da conclusão, os arquivos TMD temporários podem não ser excluídos. Os arquivos TMD também podem ser criados por outras operações de banco de dados MySQL.

O que é um arquivo .tmd? (Arquivo metadados do título da Nintendo)

Um arquivo TMD contém metadados de título usados para verificar a integridade e executar jogos nintendo 3DS, Wii e WiiU. Ele armazena metadados sobre o arquivo que contém os dados reais do jogo, como um arquivo APP, 3DS ou CIA, e é referenciado para descriptografar e verificar o conteúdo do arquivo de dados do jogo para iniciar o jogo. Os arquivos TMD são normalmente distribuídos pelo Nintendo Update Servers (NUS).

Mais Informações

Os arquivos TMD podem armazenar metadados variados dependendo do console que o jogo é criado para ser executado. Por exemplo, um arquivo TMD para um jogo WiiU contém a versão WiiUOS necessária, enquanto um arquivo TMD para um jogo 3DS não incluirá essas informações.

A maioria dos jogadores de 3DS, Wii e WiiU não encontrará um arquivo TMD, uma vez que é referenciado pelo console nintendo em segundo plano quando um jogo é lançado. No entanto, os jogadores que quiserem emular um jogo em seus computadores provavelmente verão o arquivo TMD (juntamente com um arquivo TIK e CERT) acompanhando o arquivo principal de dados do jogo.

Desenvolvedor N/A

O que é um arquivo .tmd? (Arquivo modelo de jogo do PlayStation)

3D modelo usado por alguns jogos para os sistemas de consoles PlayStation; salva a geometria (vértices e formas) para a imagem tridimensional; usado para personagens de jogos, cenários, monstros e outros objetos de jogo.

Mais Informações

Um exemplo de jogo para PlayStation que usa arquivos TMD é Jurassic Park: Operation Genesis. Alguns arquivos TMD podem ser visualizados com MilkShape 3D, mas sem a coloração.

O que é um arquivo .tmd? (Documento textmaker)

Documento criado pelo TextMaker, um aplicativo de processamento de texto que faz parte do SoftMaker Office; suporta texto, imagens, objetos desenhados e propriedades de formatação de página; usado para armazenar documentos autorais.

Mais Informações

Os arquivos TMD foram substituídos por arquivos .TMDX.
