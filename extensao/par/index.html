O que é um arquivo .par? (Arquivo de parâmetros da aeronave FMS)

Um arquivo PAR é um arquivo de parâmetros de aeronave usado pela FMS (Flying-Model-Simulator), um jogo de simulação de voo para Windows. Economiza a geometria do avião e propriedades físicas, como massa, centro de gravidade e momentos de inércia. Os arquivos PAR são usados para criar aeronaves personalizadas para voar no jogo.

Mais Informações

PAR arquivos são um dos vários arquivos que compõem o total de dados da aeronave para o jogo. Por exemplo, pode haver outros arquivos de modelo 3D, bitmaps de textura e sons.

O que é um arquivo .par? (Arquivo de índice de pergaminho)

Arquivo de índice contendo um grupo de volumes de paridade em um arquivo Parchive (ou "parity volume set") pode ser usado para recuperar um arquivo de arquivo quando um dos arquivos contidos no arquivo é perdido.

Arquivo de peça de borda sólida

Desenvolvedor Siemens PLM Software
Categoria Imagens 3D
Formato Binário 

O que é um arquivo .par? (Arquivo de peça de borda sólida)

Arquivo de peça criado pelo Solid Edge, um programa usado para criar projetos de produtos 3D; contém um design de peças, incluindo geometria, aparência e informações estruturais; usado para armazenar informações de peças no processo de design do produto.

Mais Informações

A Solid Egde pode simular peças para estudar seu desempenho antes de fabricá-las.
