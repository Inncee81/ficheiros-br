O que é um arquivo .ast? (Modelo de planilha de habilidade)

Modelo de planilha criado pela AbilitySheet, um aplicativo que faz parte do Escritório de Capacidade do pacote de produtividade; usado para criar várias planilhas de habilidade (.AWS arquivos) a partir do mesmo modelo.

O que é um arquivo .ast? (Tabela de separações de cores da Adobe)

Contém opções de separação de cores que podem ser carregadas por vários programas da Adobe; pode ser usado para definir separações de cores para impressão, exportação para PDF ou exportação para outro programa; pode ser usado para imagens usando RGB, CMYK e outros modos de cores.

Mais Informações

Outro formato de separação de cores comum é um arquivo "Separações de cores de desktop" (.DCS), que foi desenvolvido pela Quark.

O que é um arquivo .ast? (Arquivo assistente clarisWorks)

Assistente para a criação de um novo documento na ClarisWorks; semelhante a um modelo, mas oferece assistência adicional na composição do arquivo; pode ser usado para criar cartões de visita, calendários, boletins informativos, apresentações, etc.
