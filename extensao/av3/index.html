O que é um arquivo .av3? (Arquivo de vigilância de vídeo do árbitro)

Um arquivo AV3 é um arquivo de vídeo salvo no formato AV3, que é usado para armazenar imagens de vigilância capturadas por câmeras corporais BWC do Árbitro Panasonic. Contém vídeo gravado compactado no formato H264 e criptografado para proteger contra adulteração. Os arquivos AV3 também armazenam metadados criptografados.

Mais Informações

ARQUIVOS AV3 são normalmente usados para armazenar vídeo gravado pela câmera corporal do Árbitro BWC 3 usada por policiais. A filmagem é usada como evidência para casos criminais que vão ao tribunal. Uma vez que as imagens devem ser protegidas para evitar qualquer adulteração, os arquivos AV3 só podem ser abertos usando o Panasonic SafeServ, que faz parte do Sistema Unificado de Gerenciamento de Evidências (UEMS) da Panasonic.

NOTA: Os arquivos AV3 são semelhantes aos arquivos AV.
