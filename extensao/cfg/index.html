O que é um arquivo .cfg? (Arquivo de configuração do modelo Cal3D)

Arquivo de configuração de modelo tridimensional usado pelo Cal3D, um kit de ferramentas de animação de caracteres de código aberto; salvo em um formato de texto simples e contém referências a componentes do modelo; usado para juntar todos os componentes do modelo.

Mais Informações

Arquivos CFG contêm referências a arquivos esqueleto Cal3D (extensão .CSF ou .XSF), arquivos de animação (extensão .CAF ou .XAF), malhas (.CMF ou arquivos .XMF) e materiais (.CRF ou .XRF arquivos). Eles também armazenam o diretório raiz para os arquivos referenciados e a escala do modelo.

Desenvolvedor Celestia Development Team

Formato Texto 

O que é um arquivo .cfg? (Arquivo de configuração celestia)

Configurações do arquivo usado pela Celestia, um programa de simulação de universo 3D; salvo em um formato de texto simples e especifica os arquivos de dados para carregar quando o programa é iniciado; pode ser editado pelo usuário, mas deve ser feito com cuidado, uma vez que alterações inadequadas no arquivo podem fazer com que o processo de carregamento da Celestia falhe.

Mais Informações

Os arquivos de configuração celestia contêm referências a .STC, .SSC, arquivos de dados .DSC, scripts de .CEL, definições de fonte .TXF e outros parâmetros de inicialização.

O que é um arquivo .cfg? (Arquivo de configuração)

Um arquivo CFG é um arquivo de preferência genérico que armazena informações de configuração e configuração. É usado por vários programas, o que significa que diferentes arquivos CFG podem armazenar dados em diferentes formatos. Os arquivos CFG normalmente não devem ser abertos manualmente, mas podem ser salvos em um formato de texto que pode ser visualizado em um editor de texto.

Mais Informações

Arquivos de configuração genéricos são frequentemente chamados "configurações padrão config.cfg". Arquivos CFG baseados em texto geralmente contêm uma variável e valor por linha no seguinte formato:

var1=low
var2=med
var3=high

Para testar se um programa gera automaticamente um arquivo CFG padrão, mova o arquivo de configuração para fora do diretório padrão quando o programa não estiver em execução. Em seguida, abra o programa e verifique o diretório para ver se um novo arquivo CFG foi criado. Além disso, um arquivo CFG às vezes pode ser movido ou excluído para reverter para o arquivo de um programa.

NOTA: Arquivos de configuração do Linux normalmente usam a extensão .CONF em vez de CFG.

O que é um arquivo .cfg? (Arquivo de configuração MAME)

XML arquivo de configuração de teclado usado por emuladores de jogos de arcade MAME; contém mapeamentos de teclado e hotkeys usados para jogar um jogo; usado para configurar controles de teclado com base nas preferências do jogador.

Mais Informações

Arquivos CFG separados podem ser salvos para diferentes jogos. Isso dá aos jogadores a capacidade de personalizar controles separadamente, dependendo do jogo que está sendo jogado.

O que é um arquivo .cfg? (Arquivo de linguagem de marcação de Wesnoth)

Game file used by The Battle for Wesnoth, a free turn-based strategy game for Mac, Windows, and Linux; contains source code written in the Wesnoth Markup Language (WML), a proprietary language used for creating scenarios, campaigns, units, and other game content for The Battle for Wesnoth.

Mais Informações

Arquivos CFG podem ser editados usando qualquer editor de texto. Alguns plugins de destaque de sintaxe WML foram desenvolvidos para editores de texto como Emacs e Vi.
