O que é um arquivo .rad? (Arquivo de entrega rápida de aplicativos citrix)

Arquivo de configuração usado pelo software de virtualização Citrix XenApp (antigo Citrix Presentation Server); salvo em um formato de texto simples e contém uma referência a um perfil de aplicativo hospedado; usado para lançar aplicativos transmitidos, que fornecem software aos usuários sob demanda.

Mais Informações

Os arquivos RAD são abertos automaticamente pelo programa Raderun.exe incluído no XenApp. Este programa também pode ser executado manualmente na linha de comando.

O que é um arquivo .rad? (Arquivo de cena de radiance)

Arquivo de cena usado pela Radiance, um aplicativo gratuito de ray tracing; salva objetos, materiais e propriedades de iluminação; serve como uma entrada para gerar um arquivo octree (.OCT) para que a cena possa ser renderizada.

Mais Informações

RAD os arquivos são salvos em um formato de texto, para que possam ser editados facilmente com um editor de texto antes de serem convertidos para renderização.

O que é um arquivo .rad? (Módulo de rastreador Adlib de realidade)

Módulo criado pelo Reality AdLib Tracker, um aplicativo de rastreamento de áudio; contém informações de notas e instrumentos para uma música; contém até 32 padrões e 31 instrumentos; o tamanho comum é de 10k ou menos.

Mais Informações

NOTA: Reality AdLib Tracker e Adlib Tracker II são apenas aplicativos dos, mas podem ser usados através do DOSBox, um emulador dos DOS, em PCs contemporâneos.

Categoria Imagens 3D

O que é um arquivo .rad? (Arquivo luzes VRAD)

3D arquivo de iluminação usado pelo VRAD, um programa de linha de comando que adiciona propriedades de luz a um arquivo de mapa de jogo de tiro em primeira pessoa .BSP; armazena propriedades de iluminação para materiais, que definem a aparência superficial de objetos 3D.

Mais Informações

RAD os arquivos são especificados em um formato XML. Eles podem incluir configurações para cor e intensidade normal de LDR e HDR (alta gama dinâmica).

Para compilar propriedades de iluminação em um arquivo de mapa BSP existente, use o seguinte comando: vrad -luzes bspmapnamerad
