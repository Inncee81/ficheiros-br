O que é um arquivo .PRF? (Arquivo de preferência clarisWorks)

As preferências das lojas para ClarisWorks, como posicionamento da barra de ferramentas e opções de tela de boas-vindas; devem ser nomeadas "CLWORKSPRF" e colocadas no diretório CLARIS; podem ser excluídas para restaurar as preferências padrão do programa.

Mais Informações

NOTA: ClarisWorks é uma suíte de produtividade Macintosh desenvolvida pela Apple que mais tarde também foi disponibilizada para Windows computadores. Mais tarde foi substituído pelo AppleWorks, que foi então descontinuado.

O que é um arquivo .PRF? (Arquivo de perfil do Outlook)

Um arquivo PRF é um perfil de usuário referenciado pelo Microsoft Outlook que contém informações e preferências do usuário. Ele inclui configurações de conta de e-mail, assinaturas, referências a pastas de e-mail e outras configurações.

Mais Informações

Quando o Outlook importa um arquivo PRF, as configurações do perfil do usuário são imediatamente atualizadas. Os arquivos PRF também podem ser criados usando o Kit de Recursos do Office da Microsoft. Os arquivos podem então ser implantados em vários sistemas usando o ASSISTENTE de Instalação Personalizada (CIW) ou outro assistente de manutenção incluído no Kit de Recursos do Office.

Arquivo de regras PICS

Desenvolvedor W3C
Categoria Web
Formato Texto 

O que é um arquivo .PRF? (Arquivo de regras PICS)

Web perfil criado no formato Plataforma de Seleção de Conteúdo na Internet (PICS); contém regras de filtragem que permitem ou bloqueiam o acesso a determinadas URLs; usadas para implementar políticas de segurança na Internet, como controles parentais ou professores, que limitam o acesso a determinados sites para alunos e crianças; também usadas para compartilhar perfis PICS com outras pessoas.

Mais Informações

O formato PICS foi proposto em meados da década de 1990, e foi adotado por algumas tecnologias Web. No entanto, ele não é amplamente utilizado hoje e foi substituído pelo Protocolo para Web Recursos de Descrição (POWDER).

NOTA: Os arquivos da PRF foram suportados por Web alguns navegadores e servidores Web, como o Internet Explorer e o Microsoft IIS.

O que é um arquivo .PRF? (Arquivo de solicitação de parcela da estação de plot)

Formulário que contém solicitações de plot para a Estação de Enredo do eQuorum, um programa para gerenciar trabalhos de plotagem; salvo em um formato de texto ASCII e inclui instruções e parâmetros para gerar a saída ploted; pode ser criado pelo Cliente da Estação de Parcelas e é processado pelo Plot Station Server.

O que é um arquivo .PRF? (Arquivo do sistema Windows)

Contém configurações e preferências usadas pelo sistema operacional Windows; também pode ser usado por outros aplicativos para armazenar as configurações do programa; não deve ser aberto manualmente.
