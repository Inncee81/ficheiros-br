O que é um arquivo .ccgame? (XNA Creators Club Game Package)

Pacote de jogos criado pelo XNA Studio, um kit de desenvolvimento de software de jogos usado para fazer jogos Windows, Xbox 360 e Windows Phone; inclui todos os ativos de desenvolvimento (código fonte, recursos, etc) e incorpora proteção digital para garantir que os conteúdos estejam protegidos contra acesso indesejado.

Mais Informações

O formato CCGAME é o formato usado para o upload de jogos para o Xbox LIVE Marketplace da Microsoft e para o App Hub da Microsoft. Portanto, quando os desenvolvedores terminam o desenvolvimento, eles embalam jogos como arquivos CCGAME e os enviam para revisão antes que os jogos possam ser vendidos aos usuários. Os arquivos CCGAME também podem ser usados para compartilhar projetos de desenvolvimento entre cópias do XNA Game Studio.

Você pode criar arquivos CCGAME selecionando Build → Package → as Creators Club Game no menu XNA Game Studio dentro do Visual Studio, ou usando XnaPack.exe, um utilitário de linha de comando incluído no XNA Game Studio. Para desempacotar um arquivo CCGAME, você pode clicar duas vezes no arquivo (assumindo que o XNA Game Studio esteja instalado) ou usar o programa XnaPack.exe com a opção unpack .
