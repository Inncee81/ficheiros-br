Imagem do veículo Garmin

Desenvolvedor Garmin
Categoria Imagens Rasterizadas
Formato Binário 

O que é um arquivo .srf? (Imagem do veículo Garmin)

Um arquivo SRF contém uma imagem de avatar do veículo que é usada para representar o veículo na tela de vários dispositivos GPS Garmin. Ele armazena várias imagens de bitmap que compõem um avatar 3D e inclui dados de imagem para várias perspectivas do veículo, como as perspectivas 3D e overhead. Os arquivos SRF podem ser criados pela Garmin ou por um usuário de GPS garmin para compartilhar com outros usuários.

Mais Informações

Você provavelmente só encontrará um arquivo avatar de veículo SRF se você usar um dispositivo GPS Garmin Nuvi para fins de navegação e estiver procurando alterar o veículo que aparece na tela do dispositivo. Vários sites permitem que os usuários compartilhem avatares personalizados de veículos SRF, com alguns arquivos SRF contendo imagens de veículos reais e outros armazenando imagens de veículos fictícios, como o ciclo de luz Tron, o dróide Star Wars BB-8 e o campista breaking bad.

A maioria dos usuários que encontram um arquivo de avatar de veículo SRF o baixaram de um site compactado em um arquivo .ZIP. Se foi assim que você adquiriu o arquivo, você pode usar o utilitário Zip empacotado com Windows (Microsoft File Explorer) ou macOS (Apple Archive Utility) para extrair a imagem.

Você pode usar o software Garmin embalado com o dispositivo GPS para abrir a imagem do veículo SRF para visualizá-la e importá-la para a pasta /Garmin/Vehicle/ em seu dispositivo GPS Garmin. Você também pode usar o Garmin Express para importar arquivos SRF para o seu dispositivo GPS.

Existem vários programas feitos pelo usuário que podem converter imagens SRF em outros formatos de imagem. O programa on-line "nüvi utilities" pode converter imagens SRF em .PNG formato ou imagens PNG em formato SRF.

O que é um arquivo .srf? (Arquivo lightwave surface)

3D arquivo de aparência criado pela LightWave 3D, um aplicativo de modelagem e renderização; armazena as opções selecionadas na janela Surface Editor, que é usada para especificar a aparência superficial de um objeto 3D; usado para salvar e reutilizar predefinições.

Mais Informações

Arquivos de superfície salvam informações como cor, sombreamento, luminosidade, especulação, brilho e transparência.

O que é um arquivo .srf? (Arquivo de gravação de Smart TV da Samsung)

Um arquivo SRF é um arquivo de gravação de vídeo criado por Smart TVs selecionadas da Samsung. Contém conteúdo, como um programa de TV ou filme, gravado na TV. Os arquivos SRF são criptografados e só podem ser abertos pela Smart TV da Samsung que os criou.

Mais Informações

As Smart TVs da Samsung são usadas por alguns usuários como gravadores de vídeo pessoais (PVRs) para gravar vídeo digital em uma unidade flash USB. As TVs criptografam arquivos SRF devido a problemas de gerenciamento de direitos digitais (DRM) a fim de proteger contra o compartilhamento ilegal de conteúdo gravado da TV.

O que é um arquivo .srf? (Arquivo de resposta do servidor)

Arquivo usado por aplicativos Visual Studio NET e processado pelo mecanismo ATL Server; pode conter html estático e tags de script que produzem conteúdo gerado dinamicamente quando a página é carregada.

Mais Informações

O arquivo "loginsrf" é comumente usado por sites seguros. Ele processa o nome de usuário e senha do usuário, envia um cookie identificando o usuário de volta para o navegador e redireciona o usuário para a página segura.

Uma página da SRF também pode ser chamada de "estêncil"," uma vez que pode conter tags que podem ser processadas por CHtmlStencil.

Desenvolvedor Rational Acoustics
Categoria Dados
Formato Não Disponível 

O que é um arquivo .srf? (Arquivo de espectro Smaart)

Um arquivo SRF é um arquivo de dados criado pela Rational Acoustics Smaart, um programa usado por técnicos e engenheiros de áudio para medir e analisar a acústica de um ambiente. Contém um "snapshot" estático de medições de espectro vivo, que consistem em um sinal de medição (a saída de um dispositivo) e um sinal de entrada usado para referência. Os arquivos SRF também armazenam configurações de espectro, como a escala de frequência do gráfico, a altura da fatia do espectrógrafo e as configurações de exibição RTA.

Mais Informações

Captura de tela de um arquivo srf em Rational Acoustics Smaart v8

As medições de espectro permitem que os usuários analisem um ou mais canais de entrada. As medidas são exibidas como um gráfico em tempo real. Esses gráficos podem ser capturados selecionando Capture ou Capture All na barra de dados na exibição "Spectrum". As medições de espectro capturados são salvas no arquivo SRF.

Por padrão, os arquivos SRF são colocados no diretório a seguir:

Documents\Smaart v8\Traces\Spectrum\Session Data

Você pode movê-los deste local. No entanto, as medições de espectro armazenadas nos arquivos SRF que são movidos não serão mais exibidas no Smaart.

A Acústica Racional Smaart permite que técnicos de áudio analisem sistemas de som e diagnosticem problemas. É usado em uma variedade de locais, como estádios, salas de concerto e igrejas. O nome do programa é um acrônimo, que significa System Measurement Acoustic Analysis Real-Time Tool (SMAART).

NOTA: Os arquivos SRF foram introduzidos com a versão 8 do Smaart e substituídos por arquivos REF usados por versões anteriores do Smaart.

Arquivo de recursos Steinberg

Desenvolvedor Steinberg Media Technologies
Categoria Configurações
Formato Binário 

O que é um arquivo .srf? (Arquivo de recursos Steinberg)

Arquivo de pele usado por Steinberg Cubase, um aplicativo de gravação de várias faixas; armazena um arquivo de imagens bitmap, bem como as localizações dos bitmaps na interface de usuário cubase; usado para alterar a aparência do aplicativo e plug-ins associados.

Mais Informações

Para Windows instalações, os arquivos SRF são normalmente armazenados no diretório de skins dentro do diretório de instalação de aplicativos cubase. Para Mac OS X, eles são comumente encontrados no diretório Contents/Resources do pacote de aplicativos. Os arquivos SRF são salvos em um formato proprietário e muitas vezes têm o nome skin.srf.
