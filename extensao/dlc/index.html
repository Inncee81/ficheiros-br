O que é um arquivo .dlc? (Arquivo de controle de luz DIALux)

Arquivo criado pela DIALux, um aplicativo profissional que permite modelar e planejar a iluminação para área interna e externa; contém informações sobre cenas de luz e grupos de controle usados pelo programa DIALux para controlar o escurecimento e o valor de cor das luminárias; exportado para ser lido por um sistema DALI (Digital Addressable Lighting Interface) e programar as configurações para um sistema de luz.

Mais Informações

Para criar o arquivo DLC, selecione File → Export → Save DIALux light scene file..., escolha o local de salvamento, nomeie o arquivo e clique em Salve.

NOTA: Atualmente não existem sistemas DALI que usam o arquivo DLC.

O que é um arquivo .dlc? (Baixe Arquivo de contêiner link)

Formato de contêiner criptografado usado para armazenar conteúdo para download; usa criptografia cliente-servidor na qual links são processados localmente e as chaves são distribuídas e rescriptografia por um serviço de Web; podem ser processadas pelo JDownloader, um programa de gerenciamento de download baseado em Java.
