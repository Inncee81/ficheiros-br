O que é um arquivo .p8? (Arquivo de código fonte PICO-8)

Um arquivo P8 contém código-fonte escrito no PICO-8, que é uma linguagem usada para programar jogos de estilo retrô de 8 bits que podem ser jogados com o console de jogos de fantasia virtual PICO-8. O código-fonte PICO-8 usa a sintaxe da linguagem de script lua e pode incluir referências a sprites, efeitos sonoros, mapas e dados musicais que compõem o jogo.

Você provavelmente só encontrará um arquivo P8 se estiver desenvolvendo um jogo no PICO-8 e salvar o código-fonte como um "cartucho" Ao salvar o "cartucho"," PICO-8 cria o arquivo P8 para armazenar o código-fonte. Você também pode adquirir o arquivo P8 se baixar o código-fonte de um jogo PICO-8 ou recebê-lo de outro desenvolvedor PICO-8.

P8 arquivos podem ser editados com PICO-8, que permite executar o jogo localmente e fazer builds do jogo. O PICO-8 vem com vários editores, incluindo Code, Sprite, Map, SFX e Music, que permitem modificar partes específicas do jogo armazenadas no arquivo P8.

Você também pode modificar um arquivo P8 com um editor de texto, como Microsoft Notepad ou Apple TextEdit. No entanto, os editores de texto simples não fornecem os mesmos recursos de destaque de sintaxe que o PICO-8.

P8 também pode ser compilado com PICO-8 no formato binário .P8.PNG. Este formato é usado para armazenar jogos que podem ser compartilhados entre outros jogadores e jogados no PICO-8.

NOTA: O PICO-8 é um programa multiplataforma que permite criar, jogar e compartilhar pequenos jogos no estilo retrô. Os jogos podem ser salvos como "cartuchos virtuais", com extensões de arquivo .p8 ou .p8.png.
