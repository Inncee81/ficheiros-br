O que é um arquivo .md? (Arquivo de documentação de marcação)

Um arquivo MD é um arquivo de texto criado usando um dos vários dialetos possíveis da linguagem Markdown. Ele é salvo em formato de texto simples, mas inclui símbolos de texto inline que definem como formatar o texto (por exemplo, negrito, recuos, cabeçalhos, formatação de tabela). Os arquivos MD são projetados para criar documentação de texto simples que pode ser facilmente convertida em HTML.

Mais Informações

Captura de tela de um arquivo md no Microsoft Notepad

Arquivos MD não são úteis apenas para sistemas de documentação HTML, mas também para controle de versão de código-fonte. Isso ocorre porque os arquivos podem ser comparados com revisões históricas em textos legívels por humanos (enquanto um formato binário não pode ser facilmente comparado). Projetos criados com o GitHub, um popular sistema de controle de versão on-line, muitas vezes usam um arquivo chamado README.md, que contém o readme para o projeto.

John Gruber, o desenvolvedor da Markdown, criou um programa Perl para converter Markdown em HTML. O programa, chamado "Markdown"," está disponível no site do Daring Fireball.

NOTA: Os arquivos de markdown também usaram a extensão .MARKDOWN.

O que é um arquivo .md? (Arquivo de dados financeiros moneydance)

O que é um arquivo .md? (SEGA Mega Drive ROM)

Game ROM usado pelo software de emulação para o SEGA Mega Drive, um console de jogos mais antigo (chamado de SEGA Genesis na América do Norte); armazena o conteúdo de um cartucho de jogo; usado para rodar jogos mais antigos no PC.

Mais Informações

NOTA: SBWin pode ser usado para converter arquivos MD para o formato ROM de jogo .BIN SEGA, que pode ser jogado por muitos emuladores diferentes da SEGA Master Drive/Genesis.

  SBWin

O que é um arquivo .md? (Sharkport Saved Game File)

Arquivo criado pela Sharkport, um acessório de hardware para o PlayStation 2; contém um jogo salvo do PS2 criado pelo dispositivo Sharkport; usado para copiar jogos salvos para o PC e compartilhá-los com outros jogadores; pode ser convertido em outros formatos de jogo salvos, como .PSU, usando o PS2 Save Builder.

Mais Informações

NOTA: Sharkport também agora usa a extensão .PS2 para jogos salvos.

Programas que abrem o arquivo .md (Sharkport Saved Game File)

  Windows
  PS2 Save Builder
