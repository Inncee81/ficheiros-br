O que é um arquivo .SMF? (Serif Metafile)

Imagem vetorial criada no formato Serif Metafile (SMF), um formato proprietário que estende o formato Windows Metafile (WMF) para incluir melhores definições de linha, preenchimento e texto; usado para compartilhar gráficos vetoriais entre aplicativos Serif, incluindo DrawPlus e PagePlus.

Mais Informações

Serif descontinuou o PagePlus e o substituiu pela Affinity Publisher em 2019. E a Serif descontinuou o DrawPlus em 2015. A última versão do software foi X8.

O que é um arquivo .SMF? (Arquivo skinmagic toolkit)

Arquivo de projeto criado pelo SkinMagic Toolkit, um programa de desenvolvedor usado para criar skins de interface de aplicativos; pode conter diálogos de interface do usuário e gráficos bitmap que são usados para sobreposição de componentes (botões, bordas, etc) da interface.

Mais Informações

Os arquivos SMF são criados por um programa chamado SkinMagicBuilder, que está incluído na instalação do SkinMagic Toolkit.

O que é um arquivo .SMF? (Arquivo MIDI padrão)

Especificação formal de arquivo MIDI desenvolvida pela MIDI Manufacturers Association; semelhante a um arquivo .MID.

O que é um arquivo .SMF? (Arquivo de fórmula StarMath)

Fórmula matemática criada pela OpenOffice Math e outras variantes do pacote office, como LibreOffice Math; armazena a sintaxe e formatação de uma equação ou expressão matemática; substituída pelo formato mais recente .ODF, mas ainda é apoiada pelo Apache OpenOffice.

Mais Informações

NOTA: O formato SMF foi originalmente usado pela StarMath através da versão 5 do StarMath. StarMath 6 e Apache OpenOffice versão 1 adotaram o formato .SXM, que foi então substituído pelo formato OpenDocument ODF. O Apache OpenOffice ainda suporta o formato SMF, embora tenha sido preterido.

NOTA: O Apache OpenOffice era anteriormente conhecido como OpenOfficeorg.
