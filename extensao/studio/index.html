O que é um arquivo .studio? (Documento do Estúdio de Silhueta)

Um arquivo STUDIO é um projeto criado pelo Silhouette Studio, um programa usado para projetar itens que podem ser cortados com uma máquina de corte eletrônico Silhueta. Ele contém um design, que pode incluir formas, imagens, texto, linhas, cores, gradientes, padrões e efeitos. Os arquivos STUDIO também armazenam configurações de página de design, configurações de corte e marcas de registro.

Captura de tela de um arquivo de estúdio em Silhouette Studio 3

O arquivo STUDIO foi o arquivo principal associado às versões 1 e 2 do Silhouette Studio. Quando você criaria e salvaria um projeto de design, o arquivo STUDIO era o formato padrão usado para armazenar o design.

No entanto, com o lançamento da versão 3 do Silhouette Studio, o arquivo STUDIO foi substituído pelo arquivo .STUDIO3. O arquivo STUDIO ainda é suportado pela versão 3 do Silhouette Studio. Você também pode salvar seu design como um arquivo STUDIO, embora isso possa deixar alguns recursos avançados não salvos.
