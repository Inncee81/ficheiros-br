O que é um arquivo .AV? (Redação FINAL AV)

Um arquivo AV é um documento criado pelo Final Draft AV (versão 1), um processador de texto usado para criar roteiros para comerciais, vídeos corporativos e documentários. Ele contém um script, que inclui diálogo, informações de personagens, cenas e colunas de vídeo e áudio. Os arquivos AV também incluem informações sobre o script, como o título, comprimento, escritor, diretor e versão de rascunho.

Mais Informações

O arquivo AV é o arquivo principal usado pela versão 1 do Final Draft AV para armazenar scripts. Os arquivos AV foram substituídos por arquivos XAV com a versão 2 do Rascunho Final AV.

Para salvar o arquivo AV no Redendo Final AV, selecione File → Save ou Save As e escolha o tipo de arquivo AV. Para abrir o arquivo AV no Redendo Final AV, selecione File → Open e escolha o tipo de arquivo AV.

NOTA: O Draft final AV foi descontinuado e não está mais disponível para download.

O que é um arquivo .AV? (Arquivo de vigilância de vídeo)

Um arquivo AV é um arquivo de vídeo usado para armazenar vídeos de vigilância criados por várias câmeras, como câmeras de segurança IP sem fio e câmeras de painel. Contém vídeo gravado criptografado compactado no formato H264. Os arquivos AV podem ser reproduzidos por Yoosee CMSClient e SAMTECH GPlayer.

Mais Informações

ARQUIVOS AV são normalmente usados para armazenar vídeo gravado por câmeras IP sem fio eRobot, que são usadas para vigilância remota de vídeo. O software CMSClient permite que os usuários reprodurem arquivos AV gravados e os convertam em formato .MP4.

ARQUIVOS AV também podem ser gravados por câmeras de painel em veículos. Essas câmeras são usadas para capturar imagens de condução, o que é útil para documentar acidentes e infrações de trânsito. As câmeras do painel também são usadas por algumas forças policiais.
