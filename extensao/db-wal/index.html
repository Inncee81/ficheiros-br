O que é um arquivo .db-wal? (Arquivo de registro de gravação com antecedência do banco de dados SQLite)

Arquivo temporário de banco de dados criado pelo SQLite 370 ou posterior, um programa de banco de dados compacto comumente incorporado em aplicativos móveis e desktop; salva informações de log de gravação antecipada (WAL) para o banco de dados; automaticamente criado e gerenciado pelo banco de dados.

Mais Informações

Como o SQLite é uma tecnologia de banco de dados embarcada popular, você pode ver DB-WAL, .DB-SHM e outros arquivos temporários de banco de dados com aplicativos que usam SQLite.

NOTA: Em um esquema de registro com gravação com antecedência do SQLite, todas as modificações de banco de dados são escritas no arquivo DB-WAL antes de serem comprometidas com o banco de dados. O WAL foi introduzido como um esquema alternativo à política padrão "jornal de reversão" do SQLite.
