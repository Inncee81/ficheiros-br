O que é um arquivo .sle? (Arquivo do pacote de sisulizer)

Arquivo de pacote compactado salvo pelo Sisulizer, uma ferramenta de localização de software visual; contém o arquivo do projeto de troca e outros arquivos necessários pelo tradutor; semelhante a um arquivo .SLP, mas é compactado usando .ZIP compactação.

O que é um arquivo .sle? (Unidade criptografada segura Steganos)

Um arquivo SLE contém dados criptografados pelo software de criptografia Steganos Safe para armazenar dados com segurança. Ele contém um "Safe," que é uma unidade que armazena dados criptografados com criptografia AES-XEX de 384 bits. Os arquivos SLE só podem ser descriptografados pelo software Steganos Safe.

Captura de tela de um arquivo de trenó em Steganos Safe 21

Para criar um arquivo SLE com o Steganos Safe, selecione Safe → Create... e siga as instruções para especificar a localização, o tamanho máximo do arquivo e a senha do arquivo. Para ajustar as configurações do Cofre, selecione Safe → Settings....

Para abrir um arquivo SLE com o Steganos Safe, clique em Safe → Import e digite a senha para abrir o arquivo. Após a abertura do arquivo, a unidade armazenada no arquivo é aberta e torna-se acessível, permitindo que os arquivos sejam adicionados ou removidos. Uma vez que o arquivo SLE é fechado, a unidade é criptografada novamente e não mais acessível.
