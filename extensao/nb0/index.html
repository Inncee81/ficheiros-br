O que é um arquivo .nb0? (Imagem do carregador de inicialização do dispositivo)

Arquivo do sistema usado para "flash" uma imagem de carregador de inicialização em um dispositivo móvel; contém uma imagem binária bruta que pode ser carregada diretamente na RAM do dispositivo alvo; permite que o dispositivo inicializa e, em seguida, carregue uma imagem formatada separada; pode ser usada para alterar dados do sistema, como o sistema operacional padrão e aplicativos.

Mais Informações

NB0 arquivos contêm dados que inicializam o dispositivo alvo, baixam uma nova imagem em tempo de execução e, em seguida, "pisca" a nova imagem de tempo de execução, que é então carregada pelo dispositivo. O formato NB0 é útil porque permite que a MEMÓRIA RAM do dispositivo alvo seja escrita diretamente, uma vez que é armazenada no formato nativo da RAM. O processo inicial de pisca-pisca é frequentemente feito usando uma conexão JTAG (Joint Test Action Group), que se conecta diretamente à placa de circuito impresso (PCB) através de uma sonda.

NB0 os arquivos são usados por dispositivos móveis Android e Windows CE. Windows dispositivos CE geralmente usam arquivos NB0 para carregar uma imagem formatada de dispositivo alvo que tem a extensão "bin" .
