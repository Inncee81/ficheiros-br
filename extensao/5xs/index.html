O que é um arquivo .5xs? (Arquivo setlist de edição da linha 6 POD HD500X)

Set list criado pela Linha 6 POD HD500X Edit, um aplicativo que permite editar efeitos de instrumento (DAW); contém até 64 predefinições que são usadas para alterar as configurações para "FX," "AMPS," "Misturador," e "Controllers"," que personalizam o som do seu instrumento; quando carregado, substitui a lista de conjuntos atualmente carregada dentro do aplicativo.

Mais Informações

Para criar o arquivo 5XS, personalize suas configurações, selecione File → Save Setlist As..., nomeie o arquivo, escolha o local de salvamento e clique em Salve.

Para abrir o arquivo 5XS, clique duas vezes no arquivo ou selecione File → Open Setlist..., navegue até o arquivo e clique em Abra.

Em Mac, o local de salvamento padrão para arquivos 5XS está no seguinte diretório:
/Documents/Line 6/Tones/POD HD500X Edit/Setlists

Em Windows, o local de salvamento padrão para arquivos 5XS está no seguinte diretório:
\My Documents\Line 6\Tones\POD HD500X Edit\Setlists

NOTA: Quando o POD HD500X Edit foi lançado, a extensão 5XS substituiu a extensão .H5S, que foi usada pelo POD HD500 Edit.
