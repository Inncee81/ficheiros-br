O que é um arquivo .mid? (Arquivo de dados de ritmo de guitarra)

Um arquivo MID também pode conter dados de músicas usados por vários jogos de ritmo de guitarra, como Guitar Hero, Clone Hero, Phase Shift e Rock Band. Ele não contém dados de áudio, mas usa dados MIDI para marcar eventos durante a reprodução da música. Os eventos incluem notas de guitarra, seções de modo de prática, frases de poder estelar e outros eventos.

Mais Informações

Arquivos MID são normalmente armazenados em uma pasta com um ou mais arquivos de áudio. Por exemplo, uma faixa de áudio Guitar Hero (salva como um arquivo .OGG) pode ter um arquivo MID correspondente que contém notas de guitarra a serem reproduzidas para essa música.

A maioria dos jogadores de ritmo de guitarra não precisará abrir ou editar arquivos MID a menos que eles estejam procurando modificar notas ou criar seus próprios arquivos de música. Se você quiser modificar notas, você pode abrir e editar arquivos MID usando o FeedBack e o Moonscraper Chart Editor. Você pode importar arquivos MID diretamente para Guitar Hero III usando Guitar Hero Three Control Panel (GHTPC).

NOTA: Os arquivos de ritmo de guitarra também são salvos com uma extensão de arquivo .CHART para evitar confusão com arquivos MIDI padrão.

O que é um arquivo .mid? (Arquivo MIDI)

Um arquivo MID é um arquivo MIDI (Musical Instrument Digital Interface) usado por programas de autoria e mixagem de música, bem como dispositivos de hardware MIDI. Ele contém dados de música, como quais notas são reproduzidas, quando são reproduzidas, quanto tempo cada nota é mantida e quão alto cada nota é reproduzida.

Mais Informações

Arquivos MID são usados para codificar dados MIDI em um formato padrão interoperável entre diferentes tipos de programas de software e dispositivos de hardware. Eles também podem conter dados do controlador para um dispositivo. Por exemplo, um arquivo MID pode ser usado para alterar instrumentos em um dispositivo de hardware MIDI ou modificar informações de pé de suporte.

ARQUIVOS MID contêm atributos de dados de música em vez de dados de música em si. Eles podem ser reproduzidos usando instrumentos de software que interpretam esses dados.
