O que é um arquivo .ftn? (Arquivo de código fonte fortran)

Um arquivo FTN é um arquivo de código fonte escrito no Fortran, uma linguagem de programação frequentemente usada para computação numérica. Ele é salvo em um formato de texto simples e pode ser aberto e editado com qualquer editor de texto. Os arquivos FTN podem armazenar código para um programa inteiro ou um módulo de programa.

Você provavelmente só encontrará um arquivo de código fonte FTN se você for um programador científico. Embora os arquivos FTN possam ser abertos e editados com qualquer editor de texto, os programadores normalmente editam arquivos FTN com editores de código-fonte que fornecem recursos de sintaxe fortran, como Photran e Apple Xcode.

Depois de modificar arquivos FTN, os programadores compilam o código-fonte com um compilador Fortran, que o traduz para código de máquina que pode ser executado em um computador como um programa. Alguns exemplos de compiladores fortran incluem Absoft Pro Fortran e Intel Parallel Studio XE.

NOTA: A extensão do arquivo .ftn é abreviatura para Fortran. Outros arquivos que armazenam o código fonte do Fortran incluem .F, .F90 e arquivos .FOR.
