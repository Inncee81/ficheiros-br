O que é um arquivo .nut? (Arquivo de vídeo NUT)

Arquivo de vídeo salvo no Formato de Contêiner Aberto NUT, um formato de contêiner semelhante a .AVI, .ASF e .MKV; suporta vários codecs de áudio e vídeo diferentes, bem como informações de legendas.

Mais Informações

O formato NUT foi desenvolvido pelas equipes MPlayer, FFmpeg e Libav. No entanto, não ganhou grande popularidade e, portanto, é um formato de vídeo incomum.

O que é um arquivo .nut? (Roteiro de Esquilo)

Um arquivo NUT é um arquivo de texto que contém código Esquilo escrito na linguagem de programação Squirrel. É um script usado como plugin para estender a funcionalidade de um programa e para eventos roteirizado em videogames.

Mais Informações

Squirrel é uma linguagem de programação orientada a objetos semelhante a Lua, C++, e JavaScript. foi projetada para ser leve, adaptando os requisitos de memória, tamanho e tempo real de programas, principalmente videogames, como Left 4 Dead 2, Simutrans, Shadow Warrior, SuperTux e Portal 2.

Como existe outro tipo de arquivo com a extensão "nut", você pode olhar para o tamanho do arquivo para uma indicação do tipo de arquivo NUT que você tem. Os arquivos NUT contendo um script Squirrel raramente atingem 1MB, enquanto os arquivos de vídeo NUT são maiores que 1MB.
