O que é um arquivo .quid? (Documento QuidProQuo)

Um arquivo QUID é um documento de texto criado pelo Scratchware QuidProQuo, um simples processador de texto para Windows. Contém texto que pode ser formatado em vários estilos diferentes, como tipo negrito, itálico, cor da página e diferentes fontes e tamanhos de fonte. Os arquivos QUID são semelhantes aos arquivos .RTF.

Mais Informações

Você provavelmente só encontrará arquivos QUID se criar documentos de texto com o processador de texto QuidProQuo ou se alguém que usa QuidProQuo compartilhar um documento com você por e-mail ou serviço de armazenamento em nuvem. Os arquivos QUID podem ser usados para salvar vários tipos de documentos, como relatórios, cartas, ensaios e currículos.

Os arquivos QUID são normalmente abertos com o QuidProQuo, mas você também pode abri-los com qualquer editor de texto ou processador de texto que suporte texto formatado. Algumas alternativas ao QuidProQuo incluem Microsoft WordPad, Microsoft Word, AbiSource AbiWord, Apple TextEdit e Apple Pages.
