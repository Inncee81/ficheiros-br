O que é um arquivo .sti? (Arquivo de dados predefinidos do usuário do SampleTank)

Arquivo de dados predefinido criado pelo SampleTank, um aplicativo de criação de amostras de som; contém os dados do programa de instrumentos para o arquivo de predefinição do usuário de instrumentos .STIP arquivo; inclui caminhos para arquivos de áudio referenciados, como .AIF; criados com .STH e arquivos .STW; devem permanecer na mesma pasta que os arquivos STH, STW e STIP relacionados; não devem ser abertos manualmente.

Mais Informações

Para cada arquivo STIP criado, será criado um arquivo STH, STI e STW. Cada um dos três arquivos contém dados dos que o STIP retira.

Para criar um arquivo STI, edite seu som no aplicativo, clique em SAVE na área Predefinição do usuário, nomeie seu som, clique em OK, isso cria um arquivo STIP que também cria o arquivo STI e será localizado na pasta de seus "Parent Sound"

Os arquivos STI, STH, STW e STIP relacionados serão todos nomeados após o mesmo instrumento. Se o seu instrumento for chamado de "Trompete", então seu arquivo STI aparecerá assim: Trumpet.sti.

O que é um arquivo .sti? (Modelo de apresentação do StarOffice)

Modelo de apresentação originalmente criado pelo StarOffice, o antecessor do Apache OpenOffice; usado para criar apresentações com um design semelhante; muitas vezes inclui um cabeçalho e rodapé pré-formatados para o slide de título e slides de conteúdo.

Mais Informações

Apache OpenOffice agora usa .OTP arquivos OpenDocument para modelos de planilha. No entanto, o Apache OpenOffice ainda pode abrir arquivos STI mais antigos.

NOTA: O Apache OpenOffice era anteriormente conhecido como OpenOfficeorg.
