O que é um arquivo .td? (Arquivo de definição de destino)

Um arquivo TD é um arquivo de definição usado pelo TableGen, uma ferramenta incluída com o compilador LLVM usado para escrever backends LLVM. Ele contém classes (class) e definições (def) escritas na linguagem TableGen que especificam como o LLVM traduz o código-fonte compilado de um programa em código que o dispositivo alvo entende.

Captura de tela de um arquivo td no Apple Xcode 10

LLVM é usado para compilar códigos de desenvolvedores, como código Objective-C ou C++, para código que pode ser executado em uma máquina de destino. Para fazer isso, o LLVM primeiro pega o código do desenvolvedor e o compila em sua própria representação interna (IR), que é uma linguagem intermediária que permite que o compilador LLVM seja flexível ao traduzir código compilado em código para diferentes ambientes de destino.

O backend llvM traduz o IR para o código de destino, mas o backend LLVM exige regras de como traduzir corretamente o IR para o código de destino. Essas regras são definidas e armazenadas no arquivo TD usando tableGen, com cada arquivo TD armazenando regras para ambientes de destino separados.

O que é um arquivo .td? (Arquivo de download incompleto thunder)

Um arquivo TD é um arquivo parcialmente baixado criado pelo Thunder, um gerenciador de downloads da Internet localizado em chinês. Ele armazena o conteúdo de um arquivo como está sendo recebido e usa a extensão "td" para marcar o download como incompleto até que ele seja concluído, momento em que a extensão "td" é removida.

Mais Informações

A extensão do arquivo "td" é uma extensão temporária anexada a um arquivo que está sendo baixado pelo Thunder. Por exemplo, o arquivo sample.mp4 teria o nome sample.mp4.td quando ele está sendo baixado. Quando o download estiver concluído, a extensão "td" é removida. Se a extensão "td" não for removida, o download provavelmente será interrompido antes de terminar e o arquivo estiver incompleto. Você deve retomar o download ou re-baixar o arquivo.

NOTA: O trovão também é chamado de "Xunlei", " que é 迅雷 em chinês. O software é desenvolvido por uma empresa chinesa chamada Thunder Network Technologies.
