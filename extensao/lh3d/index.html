O que é um arquivo .lh3d? (Projeto Live Home 3D)

Um arquivo LH3D é um projeto criado pelo Live Home 3D, um aplicativo usado para projetar e visualizar o interior e o exterior de uma casa. Contém um projeto, que inclui plantas de piso, materiais de construção, projetos de iluminação, luminárias e móveis. Os arquivos LH3D permitem que os usuários fechem um design e o reabrem para mais edição posteriormente.

Captura de tela de um arquivo lh3d em BeLight Live Home 3D 3

O arquivo LH3D é o principal tipo de arquivo associado ao BeLight Live Home 3D. Os arquivos são usados para armazenar informações para uma variedade de tipos de projetos, como banheiro, cozinha, quarto, sala, escritório ou o exterior de uma casa. Você pode começar a partir de um dos modelos Live Home 3D ou do zero para personalizar completamente o design. Em seguida, você pode adicionar luminárias, móveis, cores de parede e muito mais ao seu design, que é tudo armazenado no arquivo LH3D quando você optar por salvar (File → Save...). Você também pode salvar o projeto como um Mail, Messages ou AirDrop diretamente do Live Home 3D.
