O que é um arquivo .z7? (Arquivo Z-code Z-code da máquina Z)

Arquivo de jogo de código Z usado pelo motor de jogo Z-machine , que foi usado para jogar jogos de aventura de texto no final dos anos 1970 e 80; contém código fonte para o jogo; inclui gráficos referenciados no jogo; comumente executado por um intérprete de máquina Z, exemplos de intérpretes são listados na seção programas; semelhante ao arquivo .ULX.

Mais Informações

O arquivo Z7 usa uma linguagem de programação chamada Zork Implementation Language (Zork foi o primeiro jogo da Infocom). Todas as extensões de arquivos de código Z usam um valor numérico para as diferentes versões da máquina Z que são usadas para executá-las. Exemplos incluem as extensões .Z1, .Z2, .Z3, .Z4, .Z5, .Z6 e .Z8.

NOTA: Juntamente com arquivos Z4 e Z6, as extensões de arquivo Z7 raramente são usadas.
