O que é um arquivo .opj? (Arquivo de projeto OrCAD)

ARQUIVO DE PROJETO CAD criado pela OrCAD, um programa usado para projetar circuitos elétricos, conhecidos como PCBs (placas de circuito impresso); salva referências a arquivos incluídos no projeto, bem como configurações de projeto; geralmente faz referência a pelo menos um arquivo de design .DSN; usado para armazenar informações de design do PCB em conjunto.

Mais Informações

Os arquivos OPJ são salvos em um formato de texto simples, mas só devem ser modificados usando a interface de usuário OrCAD em vez de um editor de texto.

O que é um arquivo .opj? (Projeto Origem)

Um arquivo OPJ é um projeto criado pela Origin, um aplicativo de análise e gráfico de dados usado por vários cientistas e engenheiros. Ele contém todos os conteúdos do projeto, que inclui dados, gráficos 2D/3D e configurações.

Mais Informações

Captura de tela de um arquivo opj no OriginLab Origin 2018

O arquivo OPJ é o tipo de arquivo mais comum associado à Origin, uma vez que é usado como um contêiner para projetos. Você pode abrir arquivos OPJ com o Origin e também pode abrir arquivos OPJ com o Origin Viewer, um programa gratuito e leve que permite visualizar e copiar informações em projetos OPJ.

Para abrir um arquivo OPJ no Origin, selecione File → Open..., selecione o tipo de arquivo "opj", navegue até o local do arquivo OPJ e clique em Abra.

Para salvar um arquivo OPJ no Origin, selecione File → Save project ou Save Project As..., nomeie o arquivo, escolha o local de salvamento e clique em Save.

NOTA: O lançamento do Origin 2018 incluiu um formato unicode atualizado para salvar projetos (.OPJU), que apresenta vantagens de desempenho em relação ao formato de arquivo OPJ mais antigo e não-unicode.
