O que é um arquivo .atomsvc? (Documento de Serviço Átomo)

Um arquivo ATOMSVC é um documento de serviço Atom gerado pelo SQL Server Reporting Services (SSRS), que é usado para criar, publicar e gerenciar relatórios sobre dados do SQL Server. Ele contém informações de conexão sobre um ou mais feeds de dados usados para povoar regiões em um relatório com dados do SQL Server. Os arquivos ATOMSVC são salvos no formato XML.

Mais Informações

Os arquivos ATOMSVC são usados para integrar feeds de dados do SQL Server em aplicativos compatíveis, como o Microsoft Power BI e o Microsoft Excel com o complemento Power Pivot. A integração do feed de dados permite que os usuários acessem dados atuais sem a necessidade de inserir manualmente os dados.

Os dados são fornecidos pelo SQL Server para um relatório com o objetivo de povoar as diferentes regiões do relatório, como tabelas, listas, gráficos, gráficos, matrizes e medidores. Cada região de um relatório tem seu próprio feed de dados, mas algumas regiões podem ter múltiplos feeds de dados dependendo de sua estrutura. Esses feeds de dados são armazenados no arquivo ATOMSVC.

É importante notar que os arquivos ATOMSVC não armazenam os dados reais vistos em cada região. Eles armazenam informações para conexão com um ou mais feeds de dados, que podem ser o endereço URL do servidor web ou o endereço de uma pasta local ou compartilhada em um computador que aponta para a localização dos dados.

Para criar um arquivo ATOMSVC, selecione o ícone "Export to Data Feed" ao lado de um relatório no portal "Reporting Services" e salve o documento.

Para abrir o arquivo ATOMSVC no Power Pivot em Excel:

  1. Selecione o botão "Gerenciar" na guia Power Pivot.
  2. Clique em "De outras fontes", e escolha "Outros Feeds".
  3. Navegue até a localização do arquivo ATOMSVC e clique em Conexão de teste.
  4. Selecione as tabelas e visualizações das quais deseja importar dados, nomeie-os e clique em Finish.
  5. Power Pivot começará a importar os dados, o que pode levar vários minutos, e depois notificá-lo se a importação foi bem sucedida.
  6. Clique em Close.

NOTA: Uma vez que os arquivos ATOMSVC são salvos no formato XML, eles também podem ser abertos por um editor xml ou editor de texto.
