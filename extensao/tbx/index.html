O que é um arquivo .tbx? (ArcGIS para arquivo caixa de ferramentas de desktop)

Um arquivo TBX contém uma caixa de ferramentas de um ou mais scripts, ou "ferramentas," que são usados para executar várias funções no ESRI ArcGIS for Desktop, um programa de mapeamento GIS para Windows. Ele pode ser usado para realizar uma variedade de operações, como calcular estatisitics, criar buffers e realizar sobreposições. Os arquivos TBX podem conter caixas de ferramentas do sistema embaladas com ArcGIS para Desktop ou caixas de ferramentas personalizadas criadas por um usuário.

Mais Informações

Para criar um arquivo TBX, navegue até a geodadosbase (uma estrutura de armazenamento de dados usada pelo ArcGIS para gerenciar e editar dados GIS) ou pasta na janela "Catalog", na qual deseja que a caixa de ferramentas seja criada, clique com o botão direito do mouse na base ou pasta de geodados e selecione New → Toolbox. Você pode deixar o nome padrão como "Toolbox" ou digitar um novo nome. Você também pode criar uma caixa de ferramentas clicando no ícone da caixa de ferramentas ao salvar um novo modelo no ModelBuilder.

Você pode navegar pelas ferramentas armazenadas nas caixas de ferramentas pesquisando seus nomes na janela "Pesquisa" ou usando a janela "Catalog" ou "ArcToolbox". Algumas caixas de ferramentas tbx do sistema que vêm com ArcGIS para Desktop são ferramentas de analista 3D, ferramentas de cartografia, ferramentas de conversão, ferramentas de gerenciamento de dados, ferramentas de geocodificação, ferramentas de esquema, ferramentas de analista espacial e ferramentas de analista de rastreamento.

OS arquivos TBX são salvos por padrão no seguinte local:

C:\Documents and Settings\[username]\Application Data\ESRI\ArcToolbox\My Toolboxes

O que é um arquivo .tbx? (Arquivo eXchange do TermBase)

Um arquivo TBX é um arquivo de dados salvo no formato de código aberto TermBase eXchange (TBX), que é usado para representar e trocar informações de terminologia (o estudo dos termos e seu uso). Contém dados de terminologia, que inclui termos, palavras e dialetos, em formato XML. Os arquivos TBX são usados para tornar os bancos de dados de terminologia mais fáceis de distribuir e manter.

Mais Informações

Os arquivos TBX são usados principalmente para arquivar informações em uma base de termo (banco de dados contendo informações de terminologia) ou trocar informações entre sistemas. Os arquivos TBX podem ser importados por uma variedade de ferramentas de tradução assistidas por computador, como memoQ tradutor pro, Wordfast Pro e JiveFusion.

NOTA: A TBX é um padrão internacional e foi desenvolvida pela Associação de Padrões da Indústria de Localização (LISA) e publicada em 2008.
