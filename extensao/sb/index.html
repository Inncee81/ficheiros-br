O que é um arquivo .sb? (Arquivo de projeto de arranhão)

Um arquivo SB é um projeto criado pela Scratch, uma plataforma de desenvolvimento de aplicativos criada no MIT (Massachusetts Institute of Technology). Ele contém todas as informações do projeto, incluindo qualquer mídia importada, como sprites, sons e imagens. Os arquivos SB foram substituídos por arquivos .SB2, que são salvos no formato Scratch 20.

Mais Informações

Captura de tela de um arquivo sb no MIT Scratch 2

O arquivo SB é o principal arquivo associado ao MIT Scratch 1. O arquivo é salvo no formato de arquivo Scratch, que consiste em código escrito na linguagem de programação Scratch. Arquivos de projeto scratch podem ser usados para criar pequenos jogos e outros tipos de programas interativos.

Scratch foi desenvolvido pelo Media Lab do MIT e é usado principalmente por professores e alunos para criar animações e jogos. O programa está disponível online ou como editor offline para Windows, macOS e Linux.

NOTA: Um projeto Scratch pode ser compilado como um arquivo Java (arquivo .JAR), que pode ser executado em um navegador da Web.

O que é um arquivo .sb? (Pacote Slax)

Pacote de software usado pela Slax, uma variante portátil do Linux OS; pode armazenar um aplicativo de software, papéis de parede, um jogo ou qualquer outro pacote destinado a ser instalado em um computador Slax; pode ser pré-instalado com Slax ou pode ser um módulo baixado.

Mais Informações

Se você baixar um módulo Slax no site do Slax, você pode copiar o arquivo SB para o diretório /slax/modules/ e ele será lido automaticamente durante a próxima inicialização do sistema.

O que é um arquivo .sb? (Pequeno arquivo de código de código-fonte básico)

Arquivo de código fonte de texto criado usando a linguagem de programação Microsoft Small Basic; contém código de programa Small Basic; pode ser compilado em um arquivo de .EXE executável usando o Small Basic; comumente usado para criar jogos simples.

Mais Informações

Ao fornecer uma simples sintaxe e interface gráfica do usuário, a Small Basic ajuda a introduzir os alunos à programação em ambientes educacionais. Pequenos programas Básicos podem ser publicados no Web e jogados usando o jogador Microsoft Silverlight.
