O que é um arquivo .edf? (Projeto Edificius)

Um arquivo EDF é um projeto criado pela ACCA Edificius, um programa CAD que ajuda os usuários a criar projetos arquitetônicos 2D e 3D. Contém informações sobre vários aspectos de um projeto inteiro, que inclui cada elemento do projeto do edifício (material do telhado, largura da porta, modelo de fixação de janela, inclinação da laje da varanda, etc), terreno e um ou mais gráficos GANTT. Os arquivos EDF também podem armazenar renderizações fotorealistas e passeios de vídeo do design.

Captura de tela de um arquivo edf em ACCA Edificius BIM 2

Você provavelmente só encontrará um arquivo EDF do Projeto Edificius se você for um profissional de Arquitetura, Engenharia e Construção (AEC) usando o Edificius. O arquivo EDF é criado pelo Edificius quando um usuário salva seu projeto de construção selecionando File → Save ou Save As.

EDF arquivos são o tipo de arquivo principal associado ao Edificius e só são suportados por Edificius. Você pode abrir um arquivo EDF com o Edificius selecionando File → Open.

Se você receber um arquivo EDF e não tiver acesso ao Edificius, você pode baixar e instalar o teste de Edificius em seu site para abrir o arquivo. Ou você pode pedir ao remetente para exportar o design em um formato diferente, como Autodesk Drawing Exchange (.DXF), AutoCAD Drawing Database (.DWG), Wavefront 3D Object (.OBJ), SketchUp (.SKP), COLLADA (.DAE), Stereolithography (.STL), Polygon Model (.PLY) ou Industry Foundation Classes (.IFC).

Junto com as informações de design, os arquivos do FED também contêm dados que compõem o "Bill of Quantities"," que é uma lista de tarifas, descrições de trabalho, dimensões e quantidade financeira da construção do projeto. Esses dados podem ser exportados para um arquivo .DCF.

O que é um arquivo .edf? (Arquivo de dados ESRF)

Arquivo de dados ESRF (EDF) usado como contêiner para armazenar texto e dados binários; contém informações de cabeçalho e dados binários associados; comumente usado em aplicativos de pesquisa científica para armazenar coleções de imagens.

O que é um arquivo .edf? (Arquivo de dados estimador)

Armazena estimativas criadas com o Dunstad Estimator, um programa de software freeware para gerenciar estimativas financeiras; salvo em um formato básico delimitado por vírgula que contém dados estimadores.
