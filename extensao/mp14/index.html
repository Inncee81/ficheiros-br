O que é um arquivo .mp14? (Projeto Multisim 14)

Um arquivo MP14 é um projeto criado pela versão 14 do Multisim, um programa usado para analisar, projetar e simular circuitos. Ele contém resultados de simulação, relatórios, documentos e referências a um ou mais projetos de circuito armazenados em .MS14 arquivos. Os arquivos MP14 são usados para organizar projetos MS14 em um projeto.

Captura de tela de um arquivo mp14 em Instrumentos Nacionais Multisim 14

O arquivo MP14 é um dos principais tipos de arquivos associados ao Multisim, juntamente com o arquivo MS14. Para criar um arquivo MP14, selecione File → Projects and packing → New project... e escolha o local para salvar o arquivo MP14. O MP14 será criado e armazenado em uma pasta de projeto no local de salvamento escolhido. Para abrir um arquivo MP14, basta clicar duas vezes no arquivo MP14 ou selecionar File → Projects and packing → Open project....

Multisim anexa um número para projetar extensões de arquivos, o que indica a versão Multisim que criou o projeto. Por exemplo, o Multisim 12 cria a extensão de arquivo .MP12 e o Multisim 13 cria a extensão de arquivo .MP13. Além disso, a Multisim suporta formatos de projeto mais antigos, o que significa que a versão 14 do Multisim ainda pode abrir um projeto .MP8 criado pela versão 8 do Multisim.
