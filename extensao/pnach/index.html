O que é um arquivo .pnach? (Arquivo de patch PCSX2)

Um arquivo PNACH é um arquivo de código de trapaça usado pelo PCSX2, um emulador de jogos para jogos do PlayStation 2. ele contém dados que permitem códigos GameShark ou Code Breaker, ou outros tipos de cheats. Os arquivos PNACH incluem o título do jogo, a versão do jogo (por exemplo, SLUS) e o código de trapaça.

Mais Informações

Os arquivos PNACH são salvos em um formato de texto simples e podem ser editados com qualquer editor de texto. No entanto, você pode precisar renomear a extensão "pnach" para "txt" para abrir o arquivo e, em seguida, renomear a extensão de volta para "pnach".

O código de trapaça é um código hexadecimal que é usado pelo emulador para inserir dados modificados no jogo emulado em um determinado ponto de dados. Aqui estão os conteúdos de um exemplo de arquivo PNACH:

gametitle=Meu Jogo [SLUS 12345] (U)
comment=Um comentário<
patch=1,EE,0032C220,word,000000FF

O prefixo de nome de arquivo PNACH deve usar o código CRC dado ao jogo do PlayStation. Também deve ser armazenado no diretório \patches\ da instalação PCSX2. O PCSX2 carrega automaticamente arquivos PNACH que possuem um código CRC que corresponde ao código CRC do jogo.

NOTA: O termo "pnach" é pronunciado "patch"
