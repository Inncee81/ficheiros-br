O que é um arquivo .toc? (Arquivo de tabela de conteúdo brasero)

Arquivo de dados criado pela Brasero, um programa de autoria de discos para Unix; define a tabela de conteúdo (ou estrutura) dos dados armazenados dentro de uma imagem de disco; salvo junto com um arquivo .BIN correspondente, que contém os dados reais.

Mais Informações

NOTA: Ambos os arquivos TOC e BIN são obrigados a gravar uma imagem de disco em um CD ou DVD com Brasero.

O que é um arquivo .toc? (Tabela de Conteúdo de Eudora)

Contém uma tabela de conteúdo para mensagens armazenadas em uma determinada pasta de correio eudora; lista todas as mensagens que são exibidas quando uma pasta de e-mail é aberta dentro de Eudora.

Mais Informações

NOTA: Eudora não está mais ativamente desenvolvida.

O que é um arquivo .toc? (Arquivo de tabela de conteúdo de World of Warcraft)

Arquivo usado pela World of Warcraft, um jogo de RPG on-line massivamente multiplayer (MMORPG); contém informações de tabela de conteúdo para um complemento de interface de usuário; usado para descrever o nome de complemento, descrição, variáveis salvas e instruções de como o complemento deve ser carregado.

Mais Informações

TOC arquivos contêm linhas de texto com tags "##" seguidos por nomes de arquivos de série. Primeiro, as tags descrevem informações para o cliente. Por exemplo, "## Versão: 12" pode fornecer a versão add-on. Em segundo lugar, uma lista de nomes de arquivo descreve a ordem em que os arquivos adicionais são carregados.

NOTA: Arquivos TOC frequentemente fazem referência a arquivos Blizzard Texture (.LUA) e arquivos de configuração .XML.
