O que é um arquivo .svn-base? (Arquivo base de subversão)

Arquivo criado pela Subversion, um sistema de controle de versão de código aberto; armazena uma cópia base de um arquivo que é verificado no repositório de Subversão; usado para comparar a cópia de trabalho de um arquivo de projeto com sua versão mais recente verificada.

Mais Informações

A subversão cria arquivos SVN-BASE localmente no computador do desenvolvedor quando os arquivos são verificados no repositório. Isso permite que a Subversão compare alterações aos arquivos localmente sem ter que re-buscá-las do repositório remoto. Desta forma, os arquivos SVN-BASE são como um cache.

Os arquivos SVN-BASE são armazenados dentro de um diretório oculto .SVN de um projeto de Subversão. Eles normalmente são ocultos por padrão dentro de IDEs que se integram com a Subversão.
