O que é um arquivo .lgc? (Arquivo de banco de dados SimpleK)

Formato de banco de dados usado pelo SimpleK, um sistema de chaves mestre e programa de software de emissão de chaves; usado por serralheiros, bem como gerentes de segurança e instalações.

Mais Informações

SimpleK é usado para projetar teclas mestres, construir cilindros de chaves e fixação de núcleos, rastrear a emissão de chaves e teclados e criar relatórios. Para obter mais informações, visite o site da SimpleK.

O que é um arquivo .lgc? (WordPerfect DTD Compiler Logic File)

Arquivo criado pelo WordPerfect DTD Compiler, um programa usado para criar arquivos .DTD compilados e usá-los para validar documentos XML; armazena a estrutura de documentos legais, elementos e atributos permitidos em um arquivo XML; pode ser reutilizado para garantir que vários documentos XML estejam em conformidade com um determinado padrão.

Mais Informações

WordPerfect DTD Compiler está incluído no WordPerfect Office.
