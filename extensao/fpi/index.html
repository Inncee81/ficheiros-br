O que é um arquivo .fpi? (Plugin foxit leitor)

Um arquivo FPI é um plugin usado pelo Foxit Reader, um programa para visualização, assinatura e impressão de arquivos PDF. Ele contém informações que adicionam recursos ao programa, como login automático, assistência de carregamento de plugin e atualizações automáticas.

Mais Informações

Se você quiser desativar/remover um plugin do programa, você pode excluir ou mover o arquivo FPI de seu diretório:Foxit Software/Foxit Reader/plugins

Foxit Reader foi projetado para ser seu programa de saída para lidar com .PDF arquivos. Ele oferece ferramentas para criar, visualizar, anotar, proteger, imprimir e compartilhar o formato PDF.

O que é um arquivo .fpi? (Script de inteligência do criador do FPS)

Script ou programa criado com FPS Creator, um programa de desenvolvimento de jogos de tiro em primeira pessoa 3D (FPS); usado para desenvolver a inteligência artificial (IA) para personagens dentro do jogo.

Mais Informações

Os arquivos FPI são salvos em um formato de texto que pode ser visualizado e editado com um editor de texto. Eles podem ser compilados em um programa executável usando FPS Creator.
