O que é um arquivo .DFTI? (Documento FlexiWrite)

Um arquivo DFTI é um documento salvo pelo FlexiWrite, um processador de texto projetado para melhorar a experiência de leitura na tela de arquivos de texto. Pode conter texto, imagens e várias seções de documentos. Os arquivos DFTI armazenam dados de documentos no formato proprietário "Texto e Imagens Formatados Dinâmicos "(DFTI) do FlexiWrite.

Captura de tela de um arquivo dfti em FlexiWrite

O arquivo DFTI é o principal tipo de arquivo associado ao FlexiWrite. Ele é criado quando você salva seu documento e pode ser fechado e aberto posteriormente. O documento é usado pelo FlexiWrite da mesma forma que o Word usa os arquivos .DOCX e .DOC.

FlexiWrite está disponível para download gratuito como um aplicativo nativo do OS X para Macs e uma versão de arquivo Java (.JAR) está disponível para Windows.
