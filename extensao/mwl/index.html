O que é um arquivo .mwl? (Nível Mundial de Super Mario)

Super Mario World formato de nível usado pela Lunar Magic, um editor de nível para o jogo Super Nintendo Entertainment System (SNES) Super Mario World; contém um ou mais níveis personalizados salvos de Super Mario World.

Mais Informações

Lunar Magic requer a versão 10 Super Mario World SNES ROM, ou a versão americana 10 Mario All Stars + World SNES ROM.

O que é um arquivo .mwl? (Arquivo de layout de impressão XMS)

Um arquivo MWL contém um layout de impressão criado pela Aquaveo SMS e GMS, que são programas de hidrogeologia usados para modelar recursos de águas superficiais e subterrâneas. Ele armazena informações da Janela Gráfica Principal Do Aquaveo XMS, como um mapa, texto, seta norte, barra de escala, retângulo ou bitmap, colocado em um layout personalizado para impressão. Os arquivos MWL são usados para gerar relatórios e apresentações.

Mais Informações

Arquivos MWL são criados e abertos usando o Editor de Layout incluído nos programas Aquaveo SMS e GMS. Você pode exportar um layout de impressão como um arquivo MWL selecionando File → Export.... Você pode importar um arquivo MWL selecionando File → Import....
