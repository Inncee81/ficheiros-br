O que é um arquivo .fxp? (Arquivo do projeto Adobe Flex)

Flex arquivo de projeto criado pelo Flash Catalyst e pelo Flash Builder, que são programas usados para construir aplicativos ricos da Internet; armazena código-fonte de projeto e ativos; também armazena o design da interface do usuário, que pode incluir componentes como objetos desenhados, botões, barras de rolagem, caixas de texto e imagens; também pode armazenar linhas temporais de usuário, transições e comportamentos para interações do usuário.

Mais Informações

Arquivos FXP criados no Flash Catalyst podem ser importados para o Flash Builder. Ambos os programas são comumente usados para desenvolver aplicativos Flash, que são executados em Web navegadores que têm o plug-in do Adobe Flash Player instalado.

NOTA: O Adobe Flash Catalyst foi descontinuado em 2012.

O que é um arquivo .fxp? (Arquivo predefinido FX)

Arquivo usado por aplicativos de produção de áudio compatíveis com VST; contém uma configuração de parâmetro predefinido ou armazenado, para um efeito específico; muitas vezes usado para armazenar predefinições para plugins VST.

Mais Informações

Para importar um arquivo FXP em Steinberg Cubase, carregue seu efeito da pasta "Plug-ins" anterior e clique no botão SoundFrame, que abrirá o menu pop-up Preset Management. Em seguida, selecione Import FXB/FXP..., navegue até o seu arquivo FXP e clique em Abra. Você pode converter um arquivo FXP para o formato .VSTPRESET mais novo selecionando "Converter lista de programas para predefinições VST" As predefinições convertidas serão armazenadas na pasta "VST 3 Preset".

Os arquivos FXP são mais comumente usados pela Steinberg Cubase, mas também são suportados por outros programas de áudio. Os arquivos FXP são usados pelas edições padrão, VST e SL/SX do Cubase. As predefinições FXP também são suportadas pelo plugin Sylenth1 VST desenvolvido pela Lennar Digital e disponível para plataformas Windows e OS X.

NOTA: Os arquivos FXP e .FXB foram substituídos por predefinições VST 3 com a extensão "vstpreset".
