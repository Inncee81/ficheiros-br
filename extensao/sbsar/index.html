O que é um arquivo .sbsar? (Arquivo de Substâncias)

Um arquivo SBSAR é um arquivo criado pela Allegorithmic Substance Designer, um programa usado para criar materiais para uso em conteúdo 3D, como animações, efeitos especiais e videogames. Contém um material 3D, como concreto, tapete, vidro, mármore, aço, ardósia, ferro, metal, madeira e vidro. Os arquivos SBSAR são usados para compartilhar materiais que foram feitos por usuários do Substance ou estão disponíveis para download com uma assinatura de Fonte de Substâncias.

Mais Informações

Um arquivo SBSAR é a versão publicada de um arquivo .SBS. Ele é otimizado para armazenar apenas as informações necessárias para renderizar a substância que armazena e não pode ser editado. Para publicar uma substância no Substance Designer, que cria um arquivo SBSAR, siga estas etapas:

  1. Selecione o arquivo SBS no Explorer.
  2. Clique no ícone "Publicar" ou clique com o botão direito do mouse e selecione Export.
  3. Nomeie o arquivo.
  4. Escolha as opções de exposição de compressão e parâmetros.
  5. Clique em Ok.

Os arquivos SBSAR podem ser abertos por uma variedade de programas, incluindo Substance Designer, Substance Painter, Unity, Unreal Engine 4, Maya, 3ds Max, Cinema 4D, CATIA e TouchDesigner.

Você pode abrir um arquivo SBSAR no Substance Designer selecionando File → Open.... Você pode adicionar arquivos SBSAR ao Substance Painter selecionando File → Import resources..., arrastando e soltando os arquivos no painel "Shelf" no Substance Painter, ou colocando-os na pasta "materiais" no diretório a seguir:

Documents/Substance Painter/shelf/

Os materiais armazenados nos arquivos SBSAR recém-adicionados estarão disponíveis no painel "Shelf" no Substance Painter.
