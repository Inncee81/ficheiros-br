O que é um arquivo .ico? (Arquivo ícone)

Um arquivo ICO contém um ícone, que normalmente é usado para um programa, arquivo ou pasta Windows. Ele armazena uma ou mais imagens em vários tamanhos para que possam ser dimensionadas adequadamente dependendo de seu uso. Os arquivos ICO são semelhantes aos arquivos .CUR, que também são usados em Windows, e .ICNS arquivos, que são usados no macOS.

Todos os ícones para arquivos executáveis em Windows são salvos no formato ICO. Windows arquivos ICO normalmente armazenam imagens que variam de 16x16 a 256x256 pixels. Arquivos ICO mais antigos incluem apenas imagens até 48x48 que anteriormente eram as dimensões recomendadas por Windows. No entanto, a maioria dos ícones mais novos incluem imagens de até 256x256 para acomodar telas com DPI mais alto.

Cada imagem em um arquivo ICO com menos de 32 bits de profundidade de cor armazena dois bitmaps: 1) o Bitmap E - a máscara de imagem (que determina qual parte do ícone é transparente) e 2) o bitmap XOR - contém o ícone que é mapeado na máscara de imagem. Imagens com 32 bits de cor são imagens de 24 bits com canais de 8 bits para fins de composição alfa.
