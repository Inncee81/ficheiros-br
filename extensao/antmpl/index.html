O que é um arquivo .antmpl? (Arquivo de modelo de animação do Adobe Edge)

Arquivo de modelo criado pelo Adobe Edge Animate, um programa usado para tornar Web conteúdo animado; contém um modelo original ou predefinido, que pode conter imagens, texto, fonte, símbolos e outras alterações personalizadas; pode ser distribuído livremente e armazenado no sistema de arquivos.

Mais Informações

Para criar um arquivo ANTMPL:

Com seu arquivo de projeto .AN aberto, selecione File → Save As Template..., nomeie seu arquivo, selecione o local de salvamento e clique em Save.

Existem duas maneiras de usar um arquivo ANTMPL já criado:

  1. Ao abrir o aplicativo Edge Animate, selecione a opção "Criar a partir do modelo",, selecione um de seus modelos ou clique em Import para importar um modelo, clique em Abra.
  2. Outra maneira é selecionar "Criar novo" ao abrir o aplicativo, selecionar File → Create From Template..., selecionar um de seus modelos ou clicar em Importar para importar um modelo, clique em Abra.

NOTA: A partir de novembro de 2015, o Edge Animate não está mais sendo desenvolvido ativamente.
