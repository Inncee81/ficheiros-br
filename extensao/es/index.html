O que é um arquivo .es? (Arquivo de experimento e-studio 1x)

Arquivo de projeto criado pelo E-Studio, uma ferramenta de design de experimentos psicológicos incluído no conjunto de programas E-Prime; salva os passos de um experimento de pesquisa comportamental; inclui etapas de configuração, bem como estímulos injetados (por exemplo, vídeo ou sons) e procedimentos de registro de dados.

Mais Informações

Quando um design estiver completo, você pode compilá-lo escolhendo Run → Generate no menu do aplicativo. Para executar o experimento, escolha Run → Run.

NOTA: A versão 2 do E-Studio salva projetos de experimentos usando arquivos .ES2.

O que é um arquivo .es? (Arquivo de script SageCRM)

Script usado pela SageCRM, uma solução de CRM (Customer Relationship Management, gerenciamento de relacionamento com o cliente) para gerenciar processos de negócios; contém chamadas de API baseadas em JavaScript e eWare; usada para script ou instalar alterações de personalização de interface do usuário em um aplicativo SageCRM.

Mais Informações

ES os arquivos são tipicamente chamados "COMPONENTES" e são salvos no diretório "CRM<installname>inf" juntamente com os arquivos .ECF correspondentes.
