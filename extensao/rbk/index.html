O que é um arquivo .rbk? (Windows arquivo de backup de registro de 95)

O arquivo de backup do registro Windows 95 ou 98, que armazena configurações de sistema e aplicativos, associações de arquivos e outros dados; pode ser usado para restaurar o registro se ele ficar corrompido.

Mais Informações

Arquivos RBK podem ser criados usando a Ferramenta de Backup de Configuração (Cfgback.exe) incluída com Windows 95. Geralmente, os arquivos são chamados Regback#.rbk, onde "#" é um dígito de 1 a 9.

Os backups de registro também são usados por programas como o SQL Server 70 para salvar entradas de registro específicas do SQL Server. O SQL Server 70 fornece um programa chamado Regrebld.exe que pode ser usado para os backups.
