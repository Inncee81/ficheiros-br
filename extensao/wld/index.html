O que é um arquivo .wld? (Arquivo Mundial ESRI)

ARQUIVO GIS criado no formato ESRI World (WLD), que é um formato de texto ASCII que especifica informações coordenadas para uma imagem raster; armazena seis linhas de dados e inclui o tamanho do pixel, rotação e localização de coordenadas.

Mais Informações

NOTA: Ao saber o tamanho do pixel, a coordenada do pixel superior esquerdo e as informações de rotação de uma imagem raster, a localização de uma foto aérea pode ser colocada precisamente em um mapa.

O que é um arquivo .wld? (Arquivo Mundial SMBX)

Arquivo mundial usado pelo SMBX (Super Mario Bros. X), um clone criado por fãs de Super Mario Bros. para PC; salva o mapa aéreo de um mundo, que inclui caminhos para diferentes níveis jogáveis (especificados em .LVL arquivos); pode ser personalizado usando o programa Level Editor incluído no SMBX.

O que é um arquivo .wld? (Arquivo Mundial terraria)

Arquivo de jogo usado pela Terraria, um jogo de ação 2D e construção de caixas de areia; salva um mundo, que contém terreno, água, monstros, itens e outros objetos; usado para jogabilidade única ou multiplayer e pode ser carregado pelo servidor dedicado à Terraria, que permite que os jogadores hospedem mundos terraria em uma rede.

Mais Informações

Os arquivos mundiais podem ser um mapa de jogo pré-instalado, ou podem ser um mapa personalizado criado pelo usuário. Os arquivos WLD são salvos no seguinte diretório em Windows: [user]\Documents\My Games\Terraria\Worlds\.

Arquivos WLD podem ser editados usando o Editor de Mapas terraria (TEdit). Eles podem ser vistos sem o jogo Terraria usando MoreTerra (Terraria World Viewer).
