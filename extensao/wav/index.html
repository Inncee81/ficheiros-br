O que é um arquivo .wav? (Arquivo DTS-WAV)

Arquivo criado por programas de áudio DTS, que são usados para autor de música estéreo e CDs de música sonora surround; contém dados de áudio para vários canais; só pode ser reproduzido por programas que suportam codificação DTS-WAV.

O que é um arquivo .wav? (Arquivo de áudio WAVE)

Um arquivo WAV é um arquivo de áudio salvo no formato WAVE, que é um formato de arquivo de áudio digital padrão utilizado para armazenar dados de forma de onda. Os arquivos WAV podem conter gravações de áudio com diferentes taxas de amostragem e bitrates, mas muitas vezes são salvos em um formato estéreo de 441 KHz, 16 bits, que é o formato padrão usado para áudio de CD.

Captura de tela de um arquivo wav no Microsoft Windows Media Player

O formato WAVE é baseado no RIFF (Resource Interchange File Format, formato de arquivo), que é um formato de contêiner de arquivos usado principalmente para salvar vídeo e som. A Microsoft e a IBM desenvolveram conjuntamente o formato WAVE no início da década de 1990 adotando arquivos WAV como a principal opção para salvar áudio bruto e não comprimido na Microsoft Windows.

OS arquivos WAV são semelhantes aos arquivos .AIF, que são salvos no Formato de Arquivo de Intercâmbio de Áudio (AIFF) e são mais comuns em sistemas Macintosh. No entanto, os arquivos WAV são mais frequentemente salvos e reproduzidos em computadores Windows em vez de Macs.

Há um grande número de programas que suportam arquivos WAV em sistemas Windows, macOS e Linux. Alguns deles incluem programas empacotados, como o Microsoft Windows Media Player e o Apple iTunes, e aplicativos de terceiros, como videolan VLC media player e Eltima Elmedia Player.

NOTA: WAV é abreviação de "Waveform Audio File Format," e é pronunciado "wave"
