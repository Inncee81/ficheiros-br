O que é um arquivo .upr? (Dicionário de usuário do FileMaker)

Arquivo de dados usado pelo FileMaker, um aplicativo de banco de dados de desktop; contém uma lista de palavras definida pelo usuário que é usada para exceções de verificador ortom ortomedeiro ao verificar um banco de dados ou formulário; difere de um arquivo de .MPR, que é um dicionário de idiomas fileMaker padrão.

Mais Informações

O diretório padrão do usuário é User.upr, que está localizado no diretório /Dictionaries/ de uma instalação do FileMaker. Você pode acessar a lista de palavras neste dicionário selecionando Edit → Spelling → Edit User Dictionary... no FileMaker.

O que é um arquivo .upr? (Arquivo de parâmetros fractal)

Arquivo de dados usado por programas de matemática fracttal e arte como Apophysis e Ultra Fractal; salvo em formato de texto simples e contém os parâmetros usados para gerar uma imagem fractal; muitas vezes usado para fractais de chama; também usado para compartilhar conjuntos fractais com outros.

Mais Informações

Como o formato UPR economiza parâmetros fractais em vez de um bitmap, permite que a imagem fractal seja renderizada matematicamente e ampliada sem perda de qualidade. Programas que renderizam arquivos UPR geralmente podem salvar qualquer exibição dentro da imagem fractal como um gráfico raster, como um arquivo .PNG, .JPG ou .BMP.

NOTA: Apóofise está disponível em formato de código-fonte.
