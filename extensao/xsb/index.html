O que é um arquivo .xsb? (Banco de Som XACT)

Coleção de sons e sugestões criadas com o Microsoft XACT, a Ferramenta de Criação de Áudio Multiplataforma; usada para criar sons de videogame e definir quando devem ser reproduzidos no jogo.

Mais Informações

Arquivos XSB não contêm dados de áudio, mas, em vez disso, referências de dados de onda armazenados em .XWB arquivos. Os arquivos XSB são usados por vários videogames, como programas gas powered games, e são armazenados por padrão no diretório /Sounds.

O que é um arquivo .xsb? (Arquivo binário de esquema XML)

Um arquivo XSB é um arquivo XML Schema Binary criado pela XMLBeans, uma biblioteca de desenvolvedores de código aberto usada para ler e escrever XML usando a linguagem de programação Java. O arquivo contém dados que o XMLBeans gera a partir de um arquivo .XSD correspondente. Os dados resultantes permitem que um programa Java leia, escreva e valide dados XML descritos no arquivo XSD durante o tempo de execução.

Mais Informações

Quando o XMLBeans gera código, ele cria .JAVA arquivos de código-fonte com os tipos definidos no arquivo XSD. Ele também cria arquivos XSB, que funcionam como equivalentes compilados ao arquivo XSD. Quando o programa Java final é compilado e seus arquivos .CLASS são gerados, os arquivos XSB são embalados com os arquivos CLASS. Eles são usados para ler, escrever e validar XML durante a execução. O arquivo XSD original não é necessário durante o tempo de execução.

O que é XMLBeans?

XMLBeans é uma tecnologia de ligação Java e XML que fornece uma maneira conveniente para os desenvolvedores lerem e escreverem dados XML usando código-fonte Java gerado automaticamente. O código gerado automaticamente pelo XMLBeans não só proporciona conveniência, mas também pode validar xML quando o programa é executado. Os tipos Java gerados correspondem aos elementos XML, facilitando a leitura e gravação do código. Por exemplo, um elemento "Cliente" em um arquivo XML pode corresponder a uma classe "Cliente" no código Java gerado.

NOTA: O atributo classgendir para a ferramenta XMLBeans Ant Task dita onde os arquivos XSB são gerados. A ferramenta de linha de comando está localizada no diretório bin da instalação XMLBeans.
