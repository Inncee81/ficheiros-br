O que é um arquivo .wdb? (Arquivo de banco de dados Blizzard World of Warcraft)

Arquivo de cache de banco de dados criado por World of Warcraft, um jogo de RPG massivamente multiplayer (MMORPG); contém dados de jogos baixados de servidores de jogos; pode armazenar informações sobre itens, criaturas, personagens não-jogadores (NPCs) e missões; usado para baixar e armazenar conteúdo de jogos.

Mais Informações

Exemplos de arquivos WDB incluem creaturecache.wdb, gameobjectcache.wdb, itemcache.wdb, npccache.wdb e questcache.wdb.

O que é um arquivo .wdb? (Banco de dados microsoft works)

Banco de dados criado com o programa de banco de dados Microsoft Works; semelhante a um Banco de Dados de Acesso (arquivo .MDB), mas com mais limitações; usa um formato de dados proprietário que não pode ser aberto pelo MS Access.

Mais Informações

Para abrir um banco de dados de Obras no Access, abra primeiro o arquivo em MS Works. Em seguida, exporte o banco de dados como um arquivo .DBF. Em seguida, importe o arquivo DBF para o MS Access.

NOTA: Algumas versões de arquivos WDB podem ser convertidas em arquivos de .CSV e planilhas do Excel usando o Conversor de Banco de Dados de Obras. No entanto, alguns campos podem não se converter corretamente.
