O que é um arquivo .dsw? (Arquivo visual C++ 6 Workspace)

Contém configurações de espaço de trabalho do usuário para o software de desenvolvimento Microsoft Visual C++ 6; salva o estado atual da interface registrando o tamanho e a posição de janelas e paletas abertas; pode ser carregado posteriormente para recorrer à interface salva.

NOTA: O Microsoft Visual C++ agora está incluído como parte do Microsoft Visual Studio.
