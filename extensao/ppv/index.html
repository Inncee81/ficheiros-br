O que é um arquivo .ppv? (Apresentação do Pocket PowerPoint)

Um arquivo PPV contém uma apresentação de slides criada pelo Microsoft Pocket PowerPoint, uma versão do PowerPoint para o Microsoft Pocket PC. Ele armazena uma apresentação, que pode incluir texto formatado, pontos de bala, imagens, transições e efeitos sonoros. Os arquivos PPV são semelhantes aos arquivos .PPT, mas não suportam tantos recursos de apresentação.

Mais Informações

O arquivo PPV é usado para salvar e abrir apresentações no Pocket PowerPoint, que é um aplicativo desenvolvido para ser executado no Pocket PC, um assistente digital pessoal que executa um sistema operacional baseado em Windows CE. Pocket PowerPoint foi lançado em 2000 como parte da versão móvel do Microsoft Office para o Pocket PC 2000. Ao longo dos anos, o apoio ao formato PPV cessou. Não existem programas conhecidos que possam abrir o formato, o que significa que os arquivos PPV só podem ser abertos por Windows dispositivos CE.
