Desenvolvedor Kubotek

O que é um arquivo .prt? (Arquivo de peça CADKEY)

Arquivo CAD criado pelo CADKEY, um programa usado para projetar peças 2D e 3D; contém layouts de modelo e desenho; inclui geometria de modelo em wireframe, superfície e sólidos (baseado em ACIS), além de dimensionamento, anotação, vetor, plano e outros tipos de entidades em um espaço de modelagem.

Mais Informações

O modo de layout de desenho no CADKEY contém visualizações definidas da geometria do modelo para dimensionamento associativo e anotação, bem como visualizações de seção. Também permite a criação de geometria de wireframe 2D.

NOTA: KeyCreator, o sucessor do CADKEY, usa arquivos .CKD para armazenar designs.

Desenvolvedor PTC
Categoria CAD
Formato Binário 

O que é um arquivo .prt? (Arquivo de peça paramétrica de Creo)

Um arquivo PRT é um arquivo de peça criado pelo PTC Creo Parametric, um programa CAD parte da suíte Creo que é usado para projetar peças fabricadas e processos de montagem. Contém informações de peças, incluindo um modelo 3D e estrutura de peças. Os arquivos PRT podem ser combinados com outras partes em um arquivo de montagem (.ASM arquivo).

Mais Informações

PTC Creo Parametric era anteriormente conhecido como Pro/ENGINEER, mas foi renomeado em 2010. PTC significa Corporação de Tecnologia Paramétrica.

O que é um arquivo .prt? (Arquivo de saída da impressora)

Saída de um trabalho de impressão enviado à impressora e salvo como um arquivo; contém informações de formatação que informam à impressora como o documento deve ser impresso.

Mais Informações

Arquivos PRT podem ser criados em sistemas Unix usando a função FILEUTIL com um parâmetro filename. Eles também podem ser abertos pelo FILEUTIL e impressos usando o parâmetro LP (impressora de linha).

O que é um arquivo .prt? (Arquivo de peça unigráficos)

Um arquivo PRT é um arquivo de peça modelo salvo em um formato de modelo tridimensional usado pela Unigraphics, um programa CAD 3D. Contém informações superficiais e sólidas para uma única parte. Os arquivos PRT são salvos em um formato binário, semelhante a um modelo de fio Alias e um modelo SDRC (Structural Dynamics Research Corporation).

Arquivo do portal de origem da válvula

Desenvolvedor Valve
Categoria Jogos
Formato Texto 

O que é um arquivo .prt? (Arquivo do portal de origem da válvula)

Um arquivo PRT contém informações do portal usadas pela ferramenta de linha de comando Valve Visible Information Set (VVIS) incluída no Hammer Editor, que é um programa usado para construir mapas para jogos da Valve, como Half-Life. Ele armazena uma lista de portais em formato ASCII que são usados pela VVIS para realizar cálculos de visibilidade para determinar quais objetos em um mapa podem ser vistos por um jogador e devem ser renderizados.

Mais Informações

Captura de tela de um arquivo prt no Apple TextEdit 1

Os arquivos PRT são usados para otimizar mapas/níveis em jogos de mecanismo de origem. Os arquivos PRT são gerados pelo VBSP, que é uma ferramenta usada para compilar arquivos raw Valve Map .VMF para o formato De Partição espacial binária (BSP) para que possam ser usados em motores de jogo baseados em BSP. Esses mapas .BSP devem ser completamente "selados", o que significa que eles não podem se conectar à área infinita negra ao redor do mapa. Se os mapas não estiverem completamente selados, eles não funcionarão corretamente no jogo.

Se o nível estiver selado, então o VBSP cria um arquivo [mapname].prt, que é então usado pelo VVIS para calcular a visibilidade dos objetos e melhorar o desempenho do mapa comunicando quais objetos devem ser renderizados. Se houver um "leak" no mapa, então o VBSP o informa e não gera o arquivo PRT, o que faz com que o VVIS não realize os cálculos de visibilidade.

NOTA: Uma vez que os arquivos PRT são salvos em um formato de texto, você pode abri-los com um editor de texto, como o Microsoft Notepad ou o Apple TextEdit. No entanto, você pode ter que alterar a extensão de arquivo .prt para .txt para abri-la no editor de texto.
