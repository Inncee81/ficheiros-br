O que é um arquivo .tt17? (Declaração de Imposto turbotax Canada 2017)

Um arquivo TT17 é uma declaração de imposto criada pela versão 2017 do TurboTax Canada, um programa usado para apresentar declarações fiscais. Contém informações fiscais federais e províncias canadenses e dados pessoais inseridos pelo usuário. Os arquivos TT17 são mais comumente criados e usados em 2018 entre janeiro e abril.

Mais Informações

Quando você salva sua declaração de imposto no TurboTax Canada 2017, o arquivo TT17 é criado para salvar seu progresso. É o principal tipo de arquivo usado pela TurboTax Canada e é utilizado principalmente para três fins:

  1. Salvar uma declaração de imposto de 2017 a ser arquivada ou impressa eletronicamente para ser enviada.
  2. Apoiar uma declaração de imposto em caso de perda ou corrupção de uma declaração de imposto de renda.
  3. Importar dados fiscais de 2017 para a versão 2018 do TurboTax Canada.

As extensões de arquivos de declaração de imposto usadas pela TurboTax Canada refletem o ano dos dados fiscais que contêm. As declarações fiscais do ano de 2016 utilizam a prorrogação .TT16, as devoluções para o ano de 2015 utilizam a extensão .TT15, e assim por diante.

NOTA: A versão dos EUA do TurboTax 2017 salva arquivos com uma extensão de arquivo .TAX2017.
