O que é um arquivo .fig? (Arquivo de figura max 3ds)

3D arquivo modelo usado pela 3ds Max, um programa de modelagem e animação 3D; armazena a estrutura de figuras para um biped, que é uma figura de dois pés; armazena todas as posições de link, links de torção e as balanças para a pose atual; permite que os modeladores salvem a estrutura de um biped e, em seguida, rig personagens semelhantes com as mesmas poses.

Mais Informações

Os arquivos FIG são frequentemente salvos com arquivos .CAL, que adicionam informações de ajuste de pose.

NOTA: Os arquivos FIG não contêm dados reais de malha esquelética ou de animação.

O que é um arquivo .fig? (Arquivo de figura MATLAB)

Um arquivo FIG contém uma figura criada pelo MathWorks MATLAB, um aplicativo usado para cálculos matemáticos. Armazena um ou mais gráficos ou gráficos, que são usados para visualizar dados.

Mais Informações

No MATLAB, uma figura é a janela que contém saída gráfica de dados matemáticos. A função figure cria uma figura (janela), que permite que os usuários insiem gráficos e gráficos. Você pode abrir um arquivo FIG usando a função openfig(filename) e salvar uma figura como um arquivo FIG com a função saveas(fig,filename). Na interface do aplicativo MATLAB, você pode salvar um arquivo FIG selecionando File → Save ou pode exportá-lo para uma imagem, como .JPG ou .PNG, selecionando File → Export.

O que é um arquivo .fig? (Desenho Xfig)

Um arquivo FIG é um desenho de linha criado usando os aplicativos Xfig ou WinFIG. É um arquivo de imagem vetorial que pode incluir formas, arcos, splines, setas e objetos de texto. Os usuários também podem adicionar imagens, cores e padrões aos seus desenhos FIG.

Mais Informações

Captura de tela de um arquivo de figo no WinFIG

O formato FIG foi originalmente desenvolvido para o Xfig, um programa de desenho baseado em Linux lançado em 1985. Muitos outros desenvolvedores expandiram as capacidades da Xfig e lançaram suas próprias versões do programa, que estão disponíveis para Windows e Mac PCs. Desses aplicativos mais novos, apenas o WinFIG ainda permite que os usuários abram, editem e criem desenhos FIG.
