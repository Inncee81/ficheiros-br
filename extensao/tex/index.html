O que é um arquivo .tex? (Documento de origem laTex)

Um arquivo TEX é um documento de origem preparado pelo LaTeX, um sistema de digitação de alta qualidade. Pode incluir texto, símbolos, expressões matemáticas e gráficos. Os arquivos TEX são comumente usados em ciência da computação, engenharia elétrica, estatística, economia e física para digitar artigos, livros e outras publicações.

Mais Informações

Captura de tela de um arquivo tex em TeXworks 06

OS arquivos TEX contêm texto simples ao contrário dos programas WYSIWYG, que contêm texto formatado. O arquivo TEX normalmente contém convenções de marcação que definem a estrutura do documento, como uma carta, livro ou artigo.

Uma vez que os arquivos TEX contêm texto simples, eles podem ser abertos com qualquer editor de texto. No entanto, os arquivos são mais comumente usados pelos programas TeXworks, LaTeX Editor, LyXMac e Texmaker.

NOTA: Os arquivos TEX podem os arquivos LaTeX também usar a extensão .LTX.

O que é um arquivo .tex? (Arquivo de textura)

Imagem bitmap usada para armazenar texturas; pode criar a aparência de diferentes superfícies e pode ser aplicada a objetos 2D e 3D; muitas vezes usados em videogames (como atiradores em primeira pessoa) que suportam mapeamento de textura 3D.

Exemplos de jogos de vídeo que usam arquivos TEX incluem Avalon Heroes (que realmente usa o formato .DDS para seus arquivos TEX), Serious Sam e Dead Rising 2. Para abrir os arquivos TEX usados por Dead Rising 2, primeiro renomeie-os para ter a extensão "grande"," e, em seguida, abri-los com as Ferramentas DeadRising2 do Gibbed.

OS arquivos TEX podem ser abertos por alguns programas que fazem referência a texturas ou usam mapeamento de texturas. Eles podem possivelmente ser abertos com um editor de imagens genérico.
