O que é um arquivo .iso? (Arquivo de imagem de disco)

Um arquivo ISO é um formato de imagem de cd ou dvd comum baseado no padrão ISO-9660. Ele contém uma cópia exata dos dados do disco original, que inclui os dados salvos no disco, bem como as informações do sistema de arquivos, como estruturas de diretório, atributos de arquivo e código de inicialização. Os arquivos ISO são frequentemente usados para fazer cópias de CDs e DVDs.

Mais Informações

Arquivos ISO são úteis para duplicar ou fazer backup de CDs e DVDs porque copiam cada bit de dados exatamente do disco. Isso é diferente de copiar todos os arquivos de um disco porque as informações do cabeçalho do disco são perdidas no processo de cópia. Um arquivo ISO pode ser usado para restaurar um disco exatamente à versão original.

Windows 7 inclui um utilitário chamado Windows Disc Image Burner que pode gravar imagens ISO em CDs e DVDs. O utilitário está localizado no diretório WindowsSystem32 e tem o nome de arquivo isoburn.exe.

NOTA: Alguns arquivos ISO podem ser carregados por um PSP Sony usando o utilitário Fast Loader.

O que é um arquivo .iso? (Arquivo ROM do PlayStation 2)

ARQUIVO ROM que contém uma cópia completa de dados extraídos de um jogo do PlayStation 2; muitas vezes usado para fazer cópias de backup de jogos PS2; pode ser aberto com um emulador de PlayStation 2 (PS2) como PCSX2.

Mais Informações

Para abrir um arquivo ISO com o PCSX2, você pode simplesmente arrastar e soltar o arquivo para o aplicativo sem montar a imagem do disco. Você também pode selecionar CDVD → Iso Selector → Browse, navegar para o seu arquivo e clicar em Abrir. Para usar o PCSX2, você deve "dump" os arquivos BIOS (Basic Input Output System, sistema de saída de entrada básica) do console PS2. É ilegal usar dados bios de outros consoles PS2.

NOTA: Devido ao seu grande tamanho de arquivo, os arquivos do jogo PS2 ISO podem ser compactados. Os arquivos ISO compactados podem ter uma extensão de arquivo diferente, como .7Z, o que indica o tipo de compactação usada para reduzir o tamanho do arquivo.

O que é um arquivo .iso? (Arquivo ROM do jogo Wii)

Um arquivo ISO é um arquivo ROM que contém uma cópia completa dos dados extraídos de um jogo nintendo Wii. Ele pode ser usado por um emulador do Nintendo Wii, como Dolphin Emulator, para jogar um jogo. Os arquivos ISO também podem ser usados para armazenar uma cópia de backup de um jogo.

Mais Informações

Como os arquivos ISO normalmente armazenam uma cópia exata e não compactada de um jogo Wii, eles são muitas vezes muito grandes em tamanho. Muitos arquivos ISO são compactados e podem ter uma extensão de arquivo diferente, como .ZIP ou .7Z, o que indica o tipo de compactação usada para reduzir o tamanho do arquivo. Você pode usar um utilitário de descompressão, como 7-Zip, Corel WinZip ou Apple Archive Utility para extrair o arquivo ISO do arquivo.

NOTA: Baixar arquivos ISO contendo jogos Wii é ilegal. Você só deve despejar jogos Wii de discos que você possui para arquivos ISO.
