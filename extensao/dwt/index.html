O que é um arquivo .dwt? (Modelo de desenho AutoCAD)

Modelo criado pelo AutoCAD, um programa de produtividade CAD usado para o desenvolvimento de projetos 2D e 3D; contém configurações padrão de desenho, incluindo tipos de unidades e precisão, blocos de títulos, nomes de camadas, tipos de linhas e propriedades de desenho de snap e grade; usado para desenvolver vários desenhos .DWG que usam as mesmas convenções e configurações.

Mais Informações

O AutoCAD 2011 fornece vários modelos padrão, que estão localizados em um diretório de instalação oculto chamado \UserDataCache\Template\.

O que é um arquivo .dwt? (Modelo de página da Web de Dreamweaver)

Um arquivo DWT é um modelo usado para criar várias páginas da Web no Dreamweaver, um programa popular de desenvolvimento web. Ele contém uma página web com regiões editáveis definidas pelo usuário e regiões não editáveis que afetam todas as páginas projetadas a partir do modelo.

Mais Informações

Captura de tela de um arquivo dwt no Adobe Dreamweaver 2020

Os arquivos DWT são úteis para os desenvolvedores porque permitem que eles economizem tempo e trabalhem apenas tendo que fazer uma edição uma vez para ser refletida nas outras páginas. As regiões editáveis designadas em um modelo são as áreas que você deseja ser editada individualmente em páginas da Web que não afetarão outras páginas da Web que vêm do modelo. As regiões não editáveis são as áreas que você deseja permanecer as mesmas em todas as páginas, como menus, barras de navegação, logotipos e rodapés.

Você pode salvar uma página da Web como modelo DWT selecionando File → Save as Template.... Quando você criar o arquivo DWT, uma pasta chamada "Templates" será criada na pasta do seu site e o arquivo DWT será colocado nele.

NOTA: O Dreamweaver foi originalmente desenvolvido pela Macromedia. A Adobe adquiriu a Macromedia em 2005 e o programa agora é desenvolvido e distribuído pela Adobe.
