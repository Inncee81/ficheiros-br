O que é um arquivo .vox? (Arquivo de áudio de voz dialógico)

Arquivo de áudio codificado no formato ADPCM (VOX) da Dialogic, um formato usado para salvar áudio de fala humana; salva dados de voz digitalizados amostrados a uma taxa baixa; permite que o tamanho dos dados de áudio seja pequeno enquanto ainda mantém a clareza; usado por sistemas de telefonia de computador.

O que é um arquivo .vox? (Arquivo modelo Voxlap Voxel)

3D modelo criado no formato Voxel ("VOX" para abreviar), um formato 3D blocky usado pelo motor de jogo Voxlap; salva a geometria para um objeto de jogo, como um personagem, construção ou uma arma; para ser usado para criar modelos de videogame e pode ser editado com o programa Slab6.

Mais Informações

O motor Voxlap foi criado por Ken Silverman e está disponível livremente. Ele tem sido usado para criar jogos como Ace of Spades, que usa .KV6 arquivos que são baseados no formato VOX.

NOTA: Um voxel (que significa "pixel volumoso") é um pixel representado como um cubo no espaço 3D. Portanto, os modelos voxel parecem ter sido construídos com blocos LEGO quando renderizados. O formato Voxlap VOX é um dos muitos formatos voxel diferentes.
