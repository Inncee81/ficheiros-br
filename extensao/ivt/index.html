O que é um arquivo .ivt? (Além do arquivo de tabela 20/20)

Tabela de dados criada pela Beyond 20/20 software de análise estatística e visualização de dados; pode ser compartilhada através do Web usando o software Beyond 20/20; usa um sistema OLAP compatível com tecnologias de Web da Microsoft.

Mais Informações

OS arquivos IVT podem ser visualizados em um computador desktop usando o aplicativo de desktop Professional Browser, ou podem ser visualizados através de um navegador Web usando os aplicativos Perspective e Web Data Server (WDS) Web.

O que é um arquivo .ivt? (Arquivo de dados do Voleibol Internacional 2006)

Arquivo de dados de jogos referenciado pelo International Volleyball 2006, um jogo de vôlei 3D; pode incluir dados como informações de jogadores, regras de jogo, ações ou gráficos usados pelo jogo.
