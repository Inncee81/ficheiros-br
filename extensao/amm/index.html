O que é um arquivo .amm? (Todos os meus arquivos de banco de dados de filmes)

Arquivo criado por All My Movies, um organizador de filmes pessoais para Windows; salva um catálogo de informações sobre arquivos de vídeo, DVDs, filmes blu-ray e fitas VHS; usado para manter um inventário de uma coleção pessoal de filmes.

Mais Informações

All My Movies recupera informações sobre filmes baixando automaticamente informações de bancos de dados online.

NOTA: Os arquivos AMM são salvos no formato .MDB (Microsoft Access).
