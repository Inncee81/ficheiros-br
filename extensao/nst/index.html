O que é um arquivo .nst? (Módulo NoiseTracker)

Um arquivo NST é um módulo usado pelo NoiseTracker, um programa de rastreamento de áudio freeware criado em 1989. Contém uma ou mais faixas de áudio, que incluem padrões de notas e amostras de som.

Mais Informações

O arquivo NST é o principal tipo de arquivo associado ao NoiseTracker. Ele salva um módulo, que consiste em amostras de som que são reproduzidas de acordo com as notas nos módulos.

NoiseTracker era comumente usado por músicos de jogos amigas para criar música. Amiga foi uma plataforma de computador pessoal fabricada pela Comodoro nas décadas de 1980 e 1990. NoiseTracker foi descontinuado e não está mais disponível para download. Se você tiver um arquivo NST que precisa ser reproduzido, você pode usar o Nullsoft Winamp em Windows.

O que é um arquivo .nst? (Conector do Outlook para arquivo de notas)

Um arquivo NST é um arquivo de armazenamento criado pelo Outlook Connector para IBM Lotus Domino, um complemento para o Microsoft Outlook que permite que os usuários acessem o Lotus Notes através do Outlook. Ele contém mensagens Notes e outros dados para acesso off-line que são armazenados localmente em seu computador. Os arquivos NST são semelhantes aos arquivos de .OST off-line do Outlook.

Mais Informações

Os arquivos NST podem crescer grande em tamanho, o que pode retardar o desempenho do seu computador e fazer com que o Outlook falhe. Se você excluir arquivos NST, eles ão automaticamente recriados pelo complemento do Connector. Para remover permanentemente o arquivo NST, você deve excluir o perfil do Outlook e criar um novo perfil.

NST arquivos estão localizados na mesma pasta com outros arquivos de configurações de perfil locais, como arquivos OST e .PST, nos seguintes diretórios:

  • Windows XP - C:\Documents and Settings\%username%\Local Settings\Application Data\Microsoft\Outlook\
  • Windows Vista, 7 Windows de Windows, e 8 de Windows - C:\Users\%username%\AppData\Local\Microsoft\Outlook\
