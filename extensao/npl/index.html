O que é um arquivo .npl? (Arquivo da Biblioteca de Cubase)

Audio library criada pelos aplicativos DAW (Digital Audio Workstation, estação de trabalho de áudio digital) da Cubase e Nuendo; salva um "pool" autônomo de clipes de áudio que podem ser incluídos em um projeto de áudio (arquivo .CPR); usado para salvar e carregar grupos de clipes de áudio.

O que é um arquivo .npl? (Arquivo de lista de reprodução nokia)

Lista de reprodução de arquivos de mídia criados com o Nokia Multimedia Player, um programa de reprodução de mídia incluído no Nokia PC Suite; pode incluir referências a arquivos de áudio e vídeo, bem como arquivos de mensagens multimídia, mas não contém os próprios arquivos referenciados.

Mais Informações

Para criar uma lista de reprodução com o Nokia Multimedia Player, selecione File → Save playlist.

NOTA: O Nokia Multimedia Player foi descontinuado e não está mais disponível para download.

O que é um arquivo .npl? (Arquivo do projeto Xilinx ISE 5-6)

Arquivo de projeto criado pelas versões 5 e 6 do software de design eletrônico ISE; inclui referências a arquivos de origem do projeto, bem como propriedades de projeto; usado para salvar projetos de design FPGA (Field-programable Gate Array).

Mais Informações

NOTA: O formato NPL é um formato de projeto mais antigo. As versões 7-10 do software ISE usam a extensão .ISE, e o formato .XISE foi introduzido com a versão ISE 11. Você pode converter arquivos NPL em arquivos XISE em versões recentes do software. Selecione File → Open Project e escolha seu arquivo NPL.
