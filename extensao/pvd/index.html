O que é um arquivo .pvd? (Instalit Script)

Script que contém uma lista de comandos executados pelo kit de ferramentas de instalação Installit DOS/Windows; usado para instalar programas de software e arquivos relacionados; não mais desenvolvido ou suportado.

O que é um arquivo .pvd? (Descritor de vetor polygen3D)

Arquivo de projeto criado pela Polygen3D, uma ferramenta Java usada para gerar superfícies 3D; contém dados do projeto, como gráficos, dados de superfície e quaisquer outras informações específicas do projeto;

Mais Informações

O arquivo PVD é criado selecionando o botão Save project... na caixa de diálogo Polygen3D. No próximo prompt, você pode nomear seu arquivo, escolher seu local de salvamento e, em seguida, selecionar Save.

Este arquivo é criado no formato GCSKV (Group-able- seccionado de valor-chave) separado por cólons (GCSKV), que armazena pares/grupos de valor chave/valor em texto simples. Aqui está um exemplo simples do formato:

pvdversion:1
generator:Polygen3D
dimension:width/353:height/366
surface:{

    name:Surface 1
    border:#ff6600
    pos:x/2.0:y/5.0
    count:x/3:y/6
    size:x/12.0:y/13.0
    shift:x/0.1:y/0.4
}

Nomes de chaves de propriedade e nomes de propriedade que aparecem no arquivo incluem:

  • pvdversion = Nome da tecla de propriedade que indica a versão do formato PVD usada pelo arquivo. O valor desta propriedade é um número, ex. 1.
  • generator = Nome da chave de propriedade indicando o criador do aplicativo. O valor é o nome do aplicativo que gerou o arquivo, ex. Polygen3D.
  • dimension = Nome da tecla de propriedade especificando a altura e largura do gráfico. O valor deve ser um número, ex. dimension:width/353:height/366
  • surface = Conjunto de propriedades que define as características de cada superfície incluída no projeto, pode ocorrer várias vezes em um arquivo.
  • name = A propriedade de nome de superfície exibe o nome da superfície.
  • border = A propriedade fronteiriça define a cor da fronteira de todos os polígonos incluídos em uma superfície. Ele aparecerá no formato de cores #rrggbb.
  • texture = A propriedade textura dá a localização da textura aplicada à superfície. Pode ser um caminho absoluto ou relativo.
  • pos = A propriedade da posição indica a posição da superfície no gráfico. O valor deve ser um número positivo, pode incluir decimais, e será padrão para 0 para x e y se nenhum valor for dado.
  • count = A propriedade count dá a quantidade de polígonos em uma superfície. O valor deve ser um número positivo, ex. count:x/3:y/6.
  • size = A propriedade size indica o tamanho de todos os polígonos da superfície. O valor deve ser um número positivo, ex. size:x/12.0:y/13.0.
  • shift = A propriedade de mudança de borda declara o quão longe para a parte inferior e direita cada polígono deve ser deslocado em relação aos lados superior e esquerdo. O valor pode ser um número positivo ou negativo, ex. shift:x/0.1:y/0.4.

NOTA: O programa Polygen3D é um arquivo .JAR que requer java 7 ou posteriormente para ser executado.
