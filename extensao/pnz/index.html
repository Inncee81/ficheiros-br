O que é um arquivo .pnz? (Conjunto de banco de dados Panorama)

Arquivo criado pelo Panorama, um rápido programa de banco de dados e planilha; salva referências a arquivos de banco de dados (.PAN arquivos), mas não armazena os bancos de dados reais em si; permite que vários bancos de dados sejam abertos de uma só vez.
