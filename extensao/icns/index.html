O que é um arquivo .icns? (Arquivo de recurso do ícone macOS)

Um arquivo ICNS é um arquivo de ícone usado por aplicativos macOS. Armazena uma ou mais imagens, tipicamente que eram originalmente docas.

Mais Informações

Captura de tela de um arquivo icns no Apple Preview 10

Os arquivos ICNS geralmente estão localizados dentro do diretório Contents/Resources/ dentro do pacote de aplicativos. Eles também são referenciados pelo arquivo .PLIST (Info.plist) do aplicativo usando a propriedade CFBundleIconFile propriedade.

arquivos de ícones macOS podem ser visualizados com o programa Apple Preview incluído no macOS. Os arquivos ICNS suportam imagens de ícones de 16x16, 32x32, 48x48, 128x128, 256x256 e 512x512 pixels, bem como 1024x1024 pixels a partir de Mac OS X 108 Mountain Lion.
