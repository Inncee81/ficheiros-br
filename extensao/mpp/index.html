O que é um arquivo .mpp? (Arquivo de projeto da Microsoft)

Um arquivo MPP é um projeto criado pelo Microsoft Project, um programa para organizar, rastrear e manter projetos. Ele contém uma linha do tempo, uma lista de tarefas, orçamentos, recursos atribuídos a tarefas e outras informações do projeto. Os arquivos MPP são usados para controlar cargas de trabalho, horários, finanças, etc.

Mais Informações

Os arquivos MPP são o principal tipo de arquivo associado ao Microsoft Project e são criados quando você salva um projeto. Você pode abrir e editar um arquivo MPP com o Microsoft Project ou você pode escolher entre um dos muitos espectadores disponíveis para download.

NOTA: Diferentes versões do Projeto MS podem abrir alguns, mas não todos os arquivos do projeto. Veja a lista a seguir para obter mais informações.

  • MS O Projeto 98 pode abrir os arquivos project 4x e Project 98.
  • MS Project 2000 pode abrir arquivos do Projeto 98 e do Projeto 2000.
  • MS Project 2002 pode abrir arquivos do Projeto 98, Projeto 2000 e Projeto 2002.
  • MS Project 2003 pode abrir os arquivos do Projeto 98, 2000, 2002 e 2003.
  • MS Project 2007 pode abrir arquivos do Projeto 98, 2000-2003 e 2007.
  • MS Project 2010 pode abrir o Projeto 98, 2000-2003, 2007 e 2010 arquivos.
  • MS Project 2013 pode abrir arquivos do Projeto 2000-2003, 2007, 2010 e 2013.
  • MS Project 2016 pode abrir o Projeto 2000-2003, 2007, 2010, 2013 e 2016 arquivos.
  • MS Project 2019 pode abrir os arquivos do Projeto 2000-2003, 2007, 2010, 2013, 2016 e 2019.

O que é um arquivo .mpp? (Arquivo editor do projeto MobileFrame)

Arquivo criado pelo MobileFrame Desktop, uma ferramenta administrativa usada para gerenciar projetos e implantações de aplicativos móveis MobileFrame; contém um ou mais projetos de MobileFrame e todas as dependências do projeto; usado para salvar todos os dados do projeto em um único arquivo.

Mais Informações

Os arquivos de projeto mpp podem incluir dados para desenvolvimento de aplicativos móveis, gerenciamento de dispositivos móveis, administração de projetos e rastreamento gps, bem como dados de configuração para integração com sistemas de negócios.
