O que é um arquivo .VPK? (Pacote de aplicativos do PlayStation Vita)

Um arquivo VPK é um pacote de aplicativos usado para instalar aplicativos de terceiros em um console de jogos Sony PlayStation Vita. Ele contém várias pastas e arquivos que compõem um jogo, que podem incluir .BIN, .XML e .PNG arquivos, que são compactados com compactação zip.

Mais Informações

Arquivos VPK só podem ser instalados em consoles PlayStation Vita que foram "jailbroken," ou "hackeados", com Henkaku. Este ativador Homebrew permite que os usuários personalizem melhor o conteúdo que pode ser usado ou reproduzido no console. Você pode instalar aplicativos embalados em um arquivo VPK usando o VitaShell, que é um gerenciador de arquivos que pode ser instalado no console PlayStation Vita.

Como os arquivos VPK são compactados usando compactação zip, você pode extrair o conteúdo dos arquivos usando um utilitário de descompressão Zip, como WinZip ou 7-Zip. Você pode querer renomear a extensão de arquivo .vpk para .zip primeiro e depois abri-la com o utilitário de descompressão.

O que é um arquivo .VPK? (Valia Pak)

Um arquivo VPK é um arquivo Valve Pak usado por alguns jogos desenvolvidos com o Source Engine, que incluem Half-Life, Left 4 Dead, DOTA 2 e Portal 2. Pode ser um arquivo de diretório mestre para uma coleção de arquivos VPK ou pode ser um dos arquivos de dados VPK brutos. Arquivos VPK são usados para armazenar mapas, modelos, materiais, cenas coreográficas e outros conteúdos originais do jogo, mas também podem ser usados para fazer mods de jogo.

Mais Informações

Arquivos VPK são salvos como parte de um pacote VPK. Cada pacote VPK tem um arquivo de diretório VPK mestre chamado archive_dir.vpk (onde "archive" é o nome do pacote VPK) e arquivos de arquivo VPK chamados archive_0.vpk, archive_1.vpk, etc.

Você pode criar arquivos VPK usando a ferramenta de linha de comando vpk.exe localizada no diretório do jogo. Para garantir a compatibilidade, você deve usar a ferramenta vpk.exe localizada no diretório do jogo para o que você está criando o arquivo VPK. Em Windows, a ferramenta vpk.exe provavelmente estará localizada no seguinte diretório:
C:\Program Files (x86)\SteamApps\common\[game title]\bin

>NOTA: O formato VPK é o sucessor do formato .GCF.
