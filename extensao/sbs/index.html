O que é um arquivo .sbs? (Arquivo do sistema de backup seguro)

Arquivo de backup criado pelo Secure Backup System (SBS), um aplicativo de backup e recuperação; armazena arquivos de backup em um formato criptografado proprietário; usado para fazer backup de dados confidenciais.

Mais Informações

Os arquivos SBS exigem o login e a senha corretos, bem como o software Secure Backup System para desbloquear o conteúdo.

O que é um arquivo .sbs? (SPSS Script)

Script usado pelo SPSS, um aplicativo para análise estatística; armazena instruções escritas usando a linguagem Sax Basic; usada para automatizar tarefas repetitivas, como abrir e salvar arquivos e realizar transformações de dados.

Mais Informações

Cada script SPSS é categorizado em um dos três tipos:

  • Script inicial: um script de modelo incluído no software que fornece segmentos de código úteis para os usuários aprenderem scripts.
  • Roteiro global: um script de biblioteca que pode ser referenciado por qualquer outro script.
  • Script script: um script que é executado automaticamente quando certas condições de gatilho são atendidas no software.

NOTA: Versões mais recentes do SPSS usam arquivos .SPS para scripts.

O que é um arquivo .sbs? (Pacote de substâncias)

Um arquivo SBS contém um pacote de substâncias criado pela Allegorithmic Substance Designer, um programa usado para criar materiais para uso em conteúdo 3D, como animações, efeitos especiais e videogames. Armazena um gráfico com nódulos que representam visualmente as propriedades do material 3D. Os arquivos SBS também incluem todos os recursos que compõem o material 3D.

Mais Informações

Captura de tela de um arquivo sbs em Allegorithmic Substance Designer 2018

Os arquivos SBS só podem ser abertos pelo Substance Designer e Pelo Tocador de Substâncias. Eles são frequentemente referidos como "arquivos de origem", que devem ser publicados, ou "compilados," em arquivos .SBSAR (substâncias) para uso em outros programas, como Substance Painter, Unity, Unreal Engine 4, Maya, 3ds Max ou Cinema 4D. Para publicar uma substância no Substance Designer, siga estes passos:

  1. Selecione o arquivo SBS no Explorer.
  2. Clique no ícone "Publicar" ou clique com o botão direito do mouse e selecione Export.
  3. Nomeie o arquivo.
  4. Escolha as opções de exposição de compressão e parâmetros.
  5. Clique em Ok.
