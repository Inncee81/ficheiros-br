O que é um arquivo .animset_ingame? (Arquivo de conjunto de animação no jogo FaceFX)

Arquivo de animação usado pelo FaceFX, um aplicativo usado para projetar expressões faciais para videogames; salva uma versão despojada de um arquivo .ANIMSET adequado para uso no jogo-alvo; remove todos os dados não essenciais para que a animação seja executada eficientemente pelo FaceFX.

Mais Informações

ANIMSET_INGAME arquivos são executados usando o FaceFX SDK, um componente de software que permite que os motores de jogo executem animações FaceFX em seus respectivos jogos.
