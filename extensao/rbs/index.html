O que é um arquivo .rbs? (Arquivo de toque MP3)

Arquivo de música criado no formato .MP3; contém dados de áudio e é frequentemente codificado em 64Kbps; comumente usado para distribuir tons de toque de celular pela Internet; pode ser reproduzido como um arquivo MP3 regular renomeando a extensão do arquivo para "mp3"

Mais Informações

NOTA: Os arquivos de toque rbs não devem ser confundidos com arquivos de músicas do Rebirth. Os toques RBS são codificados em um formato MP3, enquanto as músicas Rebirth são codificadas no formato de sintetizador proprietário do Propellerhead Software.

O que é um arquivo .rbs? (Arquivo de canção de renascimento)

Arquivo de áudio criado por um sintetizador de software Rebirth RB-338; pode incluir sons de bateria e outros instrumentos eletrônicos chamados de vozes; usados por músicos para salvar composições musicais.

Mais Informações

A Propellerhead Software, desenvolvedora do Rebirth RB-338, descontinuou o produto em 1º de setembro de 2005. O software ainda está disponível gratuitamente no site do Museu Rebirth. No entanto, nenhum suporte de software é fornecido.

O que é um arquivo .rbs? (Script de reversão do instalador Windows)

Script criado durante uma instalação de software executado pelo Windows Installer; contém um conjunto de instruções usadas para restaurar o estado do sistema de pré-instalação se a instalação não for concluída; se a instalação for bem sucedida, o arquivo será excluído automaticamente.

Mais Informações

O script de reversão restaura arquivos salvos no arquivo de reversão correspondente .RBF. Ambos os arquivos são armazenados em um único arquivo chamado "ConfigMsi"
