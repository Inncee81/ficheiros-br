O que é um arquivo .g3d? (Arquivo GOM 3D)

ARQUIVO CAD criado por um scanner 3D óptico ATOS, um dispositivo que usa métodos ópticos para construir digitalmente a geometria de um objeto do mundo real; suporta dados como malhas e nuvens de ponto; usado para visualização e análise dimensional.

Mais Informações

Os arquivos G3D podem ser visualizados com o software gom inspect gratuito. Escolha File → Import → File... e selecione seu arquivo G3D.

NOTA: O formato GOM 3D é semelhante ao formato .STL (Estereotipografia), mas é mais compacto. Ambos os formatos podem armazenar milhões de pontos e podem ser grandes em tamanho. Portanto, é necessário um computador adequado para analisá-los.
