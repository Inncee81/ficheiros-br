O que é um arquivo .brn? (Arquivo de dados BrainStorm)

Arquivo criado pelo BrainStorm, um programa usado para capturar ideias de forma estruturada; contém um título para uma ideia e subpontos, que juntos compreendem uma árvore de informações; pode armazenar referências a arquivos, imagens, hiperlinks e outras informações; pode ser visto de diferentes perspectivas no software.

Mais Informações

Os arquivos BRN brainstorm podem ser publicados no HTML para compartilhar no Web.

O que é um arquivo .brn? (Arquivo gráfico do Cubo)

Arquivo gráfico usado pelos celulares HTC Touch para exibir animações do Cubo; salva uma coleção de imagens bitmap que são lidas pelo telefone para exibir a animação do cubo 3D; pode ser convertido entre o formato .BMP e BRN usando o BRN Converter; às vezes substituído por gráficos personalizados para uso no telefone.

O que é um arquivo .brn? (Arquivo do celeiro de Gabriel Knight)

Arquivo de dados de jogo usado por Gabriel Knight 3: Blood of the Sacred, Blood of the Damned, um jogo de aventura lançado em 1999; referido como um arquivo de celeiro porque abriga vários tipos de dados para o jogo; pode ser extraído para os arquivos de origem fechado usando o GK3 Barn Browser.
