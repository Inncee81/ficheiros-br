O que é um arquivo .xfd? (Arquivo de dados de formulário Do FormFlow 99)

Arquivo criado no formato FormFlow 99, que foi desenvolvido pelo JetForm; contém informações do elemento de formulário de página, como nomes e valores dos campos de texto; usado para armazenar dados de formulários que podem ser importados em um documento PDF.

Mais Informações

Os arquivos XFD são semelhantes aos arquivos .XFDF, exceto que os arquivos XFDF são mais comuns e têm um formato mais novo que usa formatação XML.

NOTA: Os arquivos XFD do FormFlow 99 não devem ser confundidos com arquivos XFDL (Forms Description Language, linguagem de descrição de formulários extensíveis), que também usam a extensão "xfd".

O que é um arquivo .xfd? (Arquivo XFD)

Arquivo que usa o padrão XFDL (Extensible Forms Description Language, linguagem de descrição de formulários extensíveis) para representar um formulário de negócios estruturado e legalmente vinculativo; contém dados de formulários de negócios, incluindo layout, assinaturas digitais, expressões inline e detalhes de transações comerciais; aborda as necessidades de transações comerciais para auditabilidade, não repúdio, preservação do contexto e segurança.

Mais Informações

Arquivos XFD são os mesmos de .XFDL arquivos. Eles foram projetados para criar formas de negócios padronizadas e são usados em transações governamentais e comerciais.

NOTA: A Força Aérea dos Estados Unidos descontinuou o uso do IBM PureEdge Viewer. Em vez disso, você deve usar o Lotus Forms Viewer.
