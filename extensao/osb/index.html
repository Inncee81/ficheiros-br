O que é um arquivo .osb? (Arquivo de mapa binário openstreetmap)

Arquivo usado pelo Merkaartor, um editor do OpenStreetMap (OSM); salva os dados XML de um arquivo .OSM em formato binário; permite tamanhos de arquivos mais compactos e manipulação mais rápida, mas é mais difícil de ler, já que não é armazenado em texto simples.

O que é um arquivo .osb? (Osu! Arquivo de storyboard)

Um arquivo OSB contém um storyboard criado por osu!, um jogo de ritmo onde os jogadores tocam pistas visuais em sincronia com a música. Ele armazena o caminho do diretório de uma imagem, tipicamente um arquivo .JPG ou .PNG, que é exibido com um arquivo Beatmap .OSZ durante a jogabilidade. Ele também inclui informações de texto simples que descrevem quando e onde a imagem é exibida em segundo plano, dependendo de como o jogador executa.

Mais Informações

OSB arquivos estão localizados na mesma pasta "Beatmaps" da instalação do jogo osu! como o arquivo OSZ com o que estão associados. Normalmente, o diretório de instalação é C:\Program Files\osu!\Beatmaps\, mas você pode ter que criar o diretório manualmente.
