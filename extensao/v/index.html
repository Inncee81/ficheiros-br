O que é um arquivo .v? (Arquivo de código fonte coq)

Arquivo de desenvolvedor usado pela Coq, um aplicativo usado para desenvolver provas matemáticas; salvo em formato de texto e contém código-fonte construído na língua Gallina, que é baseado em uma linguagem formal expressiva chamada Cálculo de Construções Indutivas (CIC); usada para armazenar provas matemáticas.

Mais Informações

Os desenvolvedores especificam expressões em arquivos V usando o Gallina "vernáculo," que é a sintaxe, ou linguagem de comandos, para a língua Gallina. Uma vez que os arquivos V são escritos, eles podem ser compilados e executados com o Coq Proof Assistant IDE, que tem o nome do programa CoqIDE.

NOTA: O desenvolvimento da Coq, que antes era mantido pela Equipe de Desenvolvimento do Coq, agora é mantido pela ADT Coq. ADT significa "Ação para o Desenvolvimento Tecnológico"

O que é um arquivo .v? (Imagem YUV bruta subsamplada)

Raw "V" arquivo de imagem criado por ppmtoyuvsplit, um programa baseado em Unix usado para dividir imagens pixmap portáteis (arquivos .PPM) em arquivos .Y, .U e V; armazena o crominância "V" ou cor, componente para a imagem; pode ser restaurado de volta ao arquivo PPM original usando o utilitário yuvsptoppm.

Mais Informações

V arquivos armazenam dados de imagem em um formato exigido pelo codec MPEG de Stanford.

NOTA: Os utilitários de linha de comando yuvsplittoppm e ppmtoyuvsplit fazem parte do pacote gráfico baseado em netpbm Unix.

O que é um arquivo .v? (Arquivo de código fonte verilog)

Um arquivo com uma extensão de arquivo V pode ser um arquivo de código fonte escrito na linguagem de descrição de hardware verilog (HDL). Ele especifica um modelo de um sistema eletrônico. Os arquivos V normalmente contêm código-fonte Verilog 2005, mas eles também podem usar um dos padrões verilog mais antigos, como Verilog-95 ou Verilog 2001.

Mais Informações

Verilog 2005 foi substituído pelo SystemVerilog, que constrói o padrão anterior, mas adiciona novos recursos de modelagem de design. Os arquivos de origem do SystemVerilog têm uma extensão de arquivo .SV.

NOTA: Os arquivos de origem verilog são salvos em um formato de texto simples e podem ser abertos em um editor de texto como o Bloco de Notas++.
