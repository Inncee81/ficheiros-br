O que é um arquivo .ivd? (Arquivo de definições de vírus incrementais do bitdefender)

Arquivo usado pelo software de proteção contra a Internet e antivírus Bitdefender; salva atualizações de definição de vírus que ajudam o software a reconhecer os vírus mais recentes; usado para complementar o arquivo de definições de vírus .CVD base com novas atualizações.

Mais Informações

Periodicamente, as definições no arquivo CVD base são atualizadas. Quando isso ocorre, as definições de IVD são redefinidas, mas podem ser atualizadas novamente com as definições mais recentes do vírus até a próxima atualização do DCV.
