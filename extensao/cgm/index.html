O que é um arquivo .cgm? (Metafile de Computação Gráfica)

Um arquivo CGM é uma imagem salva em um formato gráfico vetorial projetado para suportar as necessidades de muitos aplicativos gráficos. Foi originalmente usado para bibliotecas de arte de clipe, mas revisões mais novas (baseadas no padrão ISO/IEC 8632) são agora comumente usadas para desenhos CAD.

Mais Informações

Arquivos CGM são codificados usando um dos três métodos diferentes:

  1. Codificação binária - Codificação padrão de arquivos que permite que programas de software leiam o arquivo de forma rápida e eficiente.
  2. Codificação baseada em caracteres - Produz o menor tamanho de arquivo possível, que economiza espaço em disco e permite transferências de dados mais rápidas.
  3. Codificação de texto claro - Permite que os usuários leiam e compreendam o conteúdo do arquivo e modifiquem o arquivo usando um editor de texto básico.
