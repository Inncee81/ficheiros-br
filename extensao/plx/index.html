O que é um arquivo .plx? (Módulo de plugin IDA de Linux de 32 bits)

32 bits Linux módulo de plugin usado pelo IDA (The Interactive Disassembler), uma ferramenta que permite aos usuários desmontar arquivos executáveis em código de linguagem de montagem de baixo nível; contém um plugin escrito em C++ que amplia os recursos da ferramenta IDA.

Mais Informações

Os arquivos PLX são bibliotecas carregáveis, semelhantes a .DLL, que ampliam a funcionalidade do IDA. O plugin PLX deve ser colocado no diretório /plugin/plx/.

NOTA: Cada plugin IDA usa uma extensão diferente dependendo do tipo do sistema operacional. A extensão .PLW é usada para Windows extensão de 32 bits, a extensão .P64 é usada por Windows 64 bits, e a extensão .PLX64 é usada por plugins de Linux de 64 bits.

O que é um arquivo .plx? (Arquivo execuvível perl)

Programa executável escrito na linguagem de programação Perl; muitas vezes usado para Web aplicativos que são executados em um servidor Web, mas podem ser executados em qualquer sistema que inclua um intérprete Perl.

Mais Informações

Os scripts perl podem ser executados em sistemas IIS Windows usando Perl para ISAPI (Internet Server Application Programming Interface) ou com o ActivePerl.

A maneira padrão de nomear arquivos Perl é .PM para módulos Perl, .PL para bibliotecas Perl e PLX para scripts perl executáveis.
