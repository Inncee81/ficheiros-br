O que é um arquivo .gsd? (Arquivo de descrição da estação geral)

Arquivo de configuração de texto ASCII usado para dispositivos PROFIBUS DP (Decentralized Periféricos) ou PA (Automação de Processos); contém palavras-chave e informações de configuração separadas por quebras de linha; pode ser criado com um editor de texto, o PROFIBUS GSD Editor ou outra ferramenta que suporte PROFIBUS.

Mais Informações

PROFIBUS (Process Field Bus) é um conjunto de protocolos de rede de computadores usados por dispositivos em rede na fabricação automatizada. Os arquivos GSD descrevem os recursos de comunicação do dispositivo, que permitem o controle e a interoperabilidade de sensores, motores, switches, atuadores e outros componentes em uma rede PROFIBUS.

Arquivos de descrição da Estação Geral também usam essas extensões específicas do idioma:

  • "gse" - Inglês
  • "gsf" - Francês
  • "gsg" - Alemão
  • "gsi" - Italiano
  • <<0>>
  • "gsp" - Portugese
  • "gss" - Espanhol
NOTA: Os arquivos GSD também são usados para dispositivos PROFINET IO, mas eles usam a extensão "xml" e são baseados no formato GSD XMLML. GSD anteriormente representava "Generic Station Description"

O que é um arquivo .gsd? (Arquivo gráficos de vetor graphtec)

Formato de imagem gráfica vetorial desenvolvido pela Graphtec; usado para criar modelos de corte para Craft ROBO e Wishblade e cortadores de artesanato; pode criar projetos usados para artesanato, como adesivos, laminados, telas de seda de camiseta e outros desenhos impressos.

Mais Informações

Os arquivos GSD também são suportados por máquinas de corte de scrapbook Accugraphic Klic-N-Kut. O plugin Robo Cut Master para o Adobe Illustrator permite que o programa plote diretamente para o cortador ou plotter, mas não permite que o Illustrator leia arquivos GSD.
