O que é um arquivo .con? (Arquivo de configuração battlefield 1942)

Arquivo usado por Battlefield 1942, um jogo de tiro em primeira pessoa da Segunda Guerra Mundial; salvo em texto simples e contém informações de configuração para o jogo, incluindo configurações de controle de vídeo e entrada; muitas vezes usado para o jogo de ajuste fino "mods"

Mais Informações

Os arquivos CON comuns incluem video.con, videoDefault.con, maplist.con e maplist.con.

O que é um arquivo .con? (Arquivo de origem do aplicativo conceito)

Arquivo fonte de um aplicativo escrito para o Concept Application Server; permite a comunicação bidirecional entre o usuário e o servidor.

Mais Informações

Pode ser acessado com um navegador Web se o Cliente Conceito (incluído no pacote Concept Server) estiver instalado. Os URLs conceituais começam com "concept://" como:
concept://myservercom/my_application/maincon.

O que é um arquivo .con? (Arquivo de configurações do sistema COW)

Um arquivo CON é um arquivo de configurações usado pelo sistema operacional COW, que é um sistema operacional baseado em Linux para dispositivos móveis selecionados. Ele contém parâmetros de configuração que configuram vários elementos do sistema operacional COW, como o caminho para o toque de .MP3 que é reproduzido durante uma chamada telefônica recebida. Os arquivos CON são salvos em formato de texto simples e podem ser abertos usando um editor de texto.

Mais Informações

Você provavelmente não encontrará arquivos CON porque eles são tipicamente escondidos no dispositivo e não devem ser abertos.
