O que é um arquivo .idc? (Arquivo conector do banco de dados da Internet)

Componente do Servidor de Informações da Internet (IIS) da Microsoft Web servidor que permite o acesso a um banco de dados compatível com ODBC; contém consultas enviadas ao banco de dados, que muitas vezes são geradas pela entrada do usuário através de uma página de Web.

Mais Informações

Os resultados da consulta de banco de dados são colocados em um arquivo de extensão HTML (.HTX) e enviados para o navegador Web do usuário.
