O que é um arquivo .jsxbin? (Arquivo de script binário extendScript)

Script binário criado pelo ExtendScript Toolkit (ESTK), uma ferramenta usada para adicionar extensões aos aplicativos do Adobe Creative Suite (CS); contém a versão binária de um JavaScript; usado para distribuir extensões CS e para proteger scripts, tornando-os não-humanos legíveis.

Mais Informações

Para exportar um arquivo JSXBIN usando o ExtendScript Toolkit, selecione File → Export as Binary... enquanto tiver um arquivo JSX aberto.

NOTA: Os arquivos JSXBIN não são necessariamente arquivos JavaScript, uma vez que o ExtendScript suporta funcionalidade extra que o JavaScript não.
