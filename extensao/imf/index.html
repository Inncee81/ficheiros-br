O que é um arquivo .imf? (Arquivo de música de id)

Id Music File (FMI) criado para uso em jogos de vídeo id Software; contém dados semelhantes a arquivos MIDI em que os arquivos não contêm áudio digital amostrado, mas sim contêm notas musicais; usados para enviar dados de música para reprodução em cartões de som AdLib.

Mais Informações

O formato de arquivo do FMI foi criado pela id Software para seus videogames. Devido à natureza de baixa qualidade do som e do formato proprietário (somente placas de som AdLib), o formato do FMI não é mais utilizado.

Alguns programas, como o AdPlug, emulam placas de som do AdLib e podem reproduzir arquivos do FMI.
