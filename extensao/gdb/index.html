O que é um arquivo .gdb? (Arquivo de banco de dados GPS)

Um arquivo GDB é um arquivo de banco de dados criado pela MapSource, um aplicativo de edição de rotas GPS e planejamento de viagens. Contém pontos de passagem, rotas e faixas que podem ser transferidas para um dispositivo de navegação Garmin. Os arquivos GDB são semelhantes aos arquivos .GPX universalmente transferíveis.

Mais Informações

Para criar um arquivo GDB no MapSource, basta selecionar File → Save As, escolher seu local de salvamento, nomear seu arquivo, selecionar o formato GDB e clicar em Save.

Para criar/exportar um arquivo GDB no programa BaseCamp:

  1. Escolha uma lista ou dispositivo de qual deseja exportar dados ou clique em My Collection se quiser exportar todos os dados na Biblioteca.
  2. Selecione File → Export, clique Export [Name] para exportar todos os itens em uma lista ou pasta ou clique em Export Selection para exportar itens selecionados.
  3. Escolha um local que deseja salvar seu arquivo, selecione o formato de arquivo GDB, nomeie seu arquivo e clique em Save.

Para importar um arquivo GDB no programa BaseCamp:

  1. Vá ao painel biblioteca e dispositivos e escolha o local do qual deseja importar o arquivo GDB.
  2. Selecione File → Import.
  3. Escolha seu arquivo GDB e clique em Abra.

O que é um arquivo .gdb? (Arquivo de banco de dados InterBase)

Arquivo de banco de dados criado com Borland InterBase; extensão padrão usada por versões do InterBase antes do InterBase 70; bancos de dados criados com o InterBase 65 e anteriormente têm um limite de tamanho de arquivo de 2GB.

Mais Informações

InterBase 70 e, posteriormente, salva arquivos de banco de dados com uma extensão de arquivo .IB.
