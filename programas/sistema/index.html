Software de sistema

O software do sistema é um software projetado para fornecer uma plataforma para outro software. Exemplos de software de sistema incluem sistemas operacionais como macOS, GNU / Linux e Microsoft Windows, software de ciência computacional, motores de jogos, automação industrial e aplicativos de software como serviço.
