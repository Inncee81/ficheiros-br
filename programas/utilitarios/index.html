Software Utilitário

O software utilitário é um software desenvolvido para ajudar a analisar, configurar, otimizar ou manter um computador. Ele é usado para dar suporte à infraestrutura do computador - em contraste com o software aplicativo, que visa realizar diretamente tarefas que beneficiam os usuários comuns.
